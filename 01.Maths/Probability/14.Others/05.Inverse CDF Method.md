The Inverse CDF Method (also called Inverse Transform Sampling) is a technique to generate random samples from any probability distribution, as long as you can write down or compute its CDF.

The core idea: if $U \sim \text{Uniform}(0,1)$, then $X = F^{-1}(U)$ has CDF $F$.

This is not a trick. It is a theorem.

---

## The Theorem

Let $U \sim \text{Uniform}(0,1)$ and let $F$ be any valid CDF. Define:

$$X = F^{-1}(U)$$

where $F^{-1}(u) = \inf{x : F(x) \geq u}$ is the quantile function.

Then $X \sim F$.

**Proof (continuous case):**

$$P(X \leq x) = P(F^{-1}(U) \leq x)$$

Since $F$ is monotone increasing:

$$= P(U \leq F(x)) = F(x)$$

The last step uses the fact that $U$ is Uniform(0,1), so $P(U \leq t) = t$ for $t \in [0,1]$.

QED. It is a one-liner once you see it.

---

## Why This Works Intuitively

Imagine $F(x)$ maps the real line into $[0,1]$. A Uniform draw $U$ picks a random height on $[0,1]$. Inverting $F$ at that height gives back a point on the real line. The probability mass assigned to any interval by $F$ is exactly the length of the corresponding interval in $[0,1]$, which $U$ hits proportionally. So the distribution is preserved exactly.

---

## Algorithm

```
1. Compute or derive F^{-1}(u) analytically
2. Draw U ~ Uniform(0,1)    [use any standard RNG]
3. Return X = F^{-1}(U)
4. Repeat for n samples
```

For discrete distributions, replace step 3 with: find the smallest $x_k$ such that $F(x_k) \geq U$.

---

## Continuous Example: Exponential Distribution

$X \sim \text{Exp}(\lambda)$, so $F(x) = 1 - e^{-\lambda x}$ for $x \geq 0$.

Solve $u = 1 - e^{-\lambda x}$ for $x$:

$$x = -\frac{1}{\lambda} \ln(1 - u)$$

Since $1 - U \sim \text{Uniform}(0,1)$ whenever $U \sim \text{Uniform}(0,1)$, we can simplify:

$$X = -\frac{1}{\lambda} \ln(U)$$

```python
import numpy as np

def sample_exponential(lam, n):
    U = np.random.uniform(0, 1, n)
    return -np.log(U) / lam
```

---

## Discrete Example: Geometric Distribution

$X \sim \text{Geom}(p)$, $P(X = k) = (1-p)^{k-1} p$ for $k = 1, 2, 3, \ldots$

$F(k) = 1 - (1-p)^k$

Invert: find smallest $k$ such that $1 - (1-p)^k \geq u$:

$$(1-p)^k \leq 1 - u$$

$$k \geq \frac{\ln(1-u)}{\ln(1-p)}$$

So:

$$X = \left\lceil \frac{\ln(U)}{\ln(1-p)} \right\rceil$$

(using $\ln(1-U) \overset{d}{=} \ln(U)$)

```python
def sample_geometric(p, n):
    U = np.random.uniform(0, 1, n)
    return np.ceil(np.log(U) / np.log(1 - p)).astype(int)
```

## When Inverse CDF Is Not Tractable

For many distributions, $F^{-1}$ has no closed form. Examples:

- Normal distribution: $\Phi^{-1}(u)$ has no elementary form
- Beta, Gamma with non-integer parameters

In these cases:

|Problem|Alternative|
|---|---|
|No closed form inverse|Rejection sampling, Ziggurat algorithm|
|High-dimensional|MCMC (Metropolis-Hastings, Gibbs)|
|Known up to constant|Rejection sampling|

The Normal case is handled in practice by the Box-Muller transform or rational approximations to $\Phi^{-1}$.

## Connection to the Probability Integral Transform

The flip side of the inverse CDF method is the **Probability Integral Transform**:

If $X \sim F$ (continuous), then $F(X) \sim \text{Uniform}(0,1)$.

This is the forward direction. The inverse CDF method is the backward direction. Together they say:

$$U \xrightarrow{F^{-1}} X \xrightarrow{F} U$$

This has major consequences in statistics: it is the basis for the KS test, copula theory, and probability plots (QQ plots).

---

## Connection to Stat 110 and Simulation

In Stat 110, this appears in the context of:

- **Universality of the Uniform**: the theorem above is what Blitzstein calls universality. Every continuous distribution is a transformation of Uniform.
- **Simulation proofs**: if you want to show $X$ has CDF $F$, one clean method is to exhibit a function of $U$ that equals $X$ in distribution.
- **Order statistics**: $F^{-1}(U_{(k)})$ gives the $k$-th order statistic of $F$, where $U_{(k)}$ is the $k$-th order statistic of Uniform samples.

---

## Limitations

- Requires knowing $F^{-1}$ analytically or numerically. Numerical inversion is expensive if done naively.
- For discrete distributions, the floor/ceiling operation must be handled precisely to avoid off-by-one errors.
- In high dimensions, the method does not generalize directly (use copulas or MCMC instead).

## Personal Insight

The deepest thing here is not the algorithm. It is the universality principle: the Uniform distribution is the master distribution. Every other distribution is just a reshaping of it via the quantile function. When I think about why Monte Carlo methods work, why importance sampling is even conceivable, why QQ plots detect non-normality — it all traces back to this one theorem.

Whenever I see a complicated distribution and need samples from it, the first question I ask is: can I invert the CDF analytically? If yes, this is always the fastest and cleanest method.
