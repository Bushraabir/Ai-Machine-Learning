## Ordinary LOTP

If ${A_1, A_2, \ldots, A_n}$ partition $\Omega$, then:

$$P(B) = \sum_{i=1}^{n} P(B \mid A_i), P(A_i)$$

Now ask: what if everything is already conditioned on some background event $E$?

---

##  The Extended Statement
Let ${A_1, A_2, \ldots, A_n}$ partition $\Omega$, and let $P(A_i \cap E) > 0$ for all $i$. Then:

$$\boxed{P(B \mid E) = \sum_{i=1}^{n} P(B \mid A_i, E) P(A_i \mid E)}$$

> [!tip] _Read this as: LOTP, but every probability wears $E$ as a subscript. The structure is identical — $E$ is just the new ambient universe._

---

## Proof

$$P(B \mid E) = \frac{P(B \cap E)}{P(E)}$$

Partition $B \cap E$ using the $A_i$'s:

$$= \frac{\displaystyle\sum_i P(B \cap A_i \cap E)}{P(E)}$$

Expand each term:

$$P(B \cap A_i \cap E) = P(B \mid A_i, E), P(A_i \mid E), P(E)$$

So:

$$P(B \mid E) = \frac{\displaystyle\sum_i P(B \mid A_i, E), P(A_i \mid E), \cancel{P(E)}}{\cancel{P(E)}}$$

$$= \sum_{i=1}^{n} P(B \mid A_i, E), P(A_i \mid E) \qquad \blacksquare$$

> [!note] _The $P(E)$ factors cancel perfectly — same trick as in Bayes with extra conditioning. Everything reduces to ordinary definitions._

---

# Worked Example: Hierarchical Bag Problem

**Setup.**

- Bag 1: 3 red, 7 blue
- Bag 2: 6 red, 4 blue

A bag is chosen uniformly at random. Two balls drawn **with replacement**. First is red ($R_1$). What is $P(R_2 \mid R_1)$?

Let $B_1$, $B_2$ = events that Bag 1 or Bag 2 was chosen.

**Apply LOTP** conditioning on the bag, inside the universe $E = R_1$:

$$P(R_2 \mid R_1) = P(R_2 \mid R_1, B_1),P(B_1 \mid R_1) + P(R_2 \mid R_1, B_2),P(B_2 \mid R_1)$$

First, compute updated bag probabilities via Bayes (see [[05.Bayes' Rule with Extra Conditioning]]):

$$P(B_1 \mid R_1) = \frac{1}{3}, \qquad P(B_2 \mid R_1) = \frac{2}{3}$$

Since draws are with replacement, $R_2 \perp R_1 \mid \text{Bag}$:

$$P(R_2 \mid R_1, B_1) = P(R_2 \mid B_1) = 0.3$$ $$P(R_2 \mid R_1, B_2) = P(R_2 \mid B_2) = 0.6$$

Therefore:

$$P(R_2 \mid R_1) = (0.3)!\left(\tfrac{1}{3}\right) + (0.6)!\left(\tfrac{2}{3}\right) = 0.1 + 0.4 = \boxed{0.5}$$

**Compare:** $P(R_2) = (0.3)(0.5) + (0.6)(0.5) = 0.45 \neq 0.5$

> [!warning] **$R_1$ and $R_2$ are not independent.** Seeing $R_1$ shifts belief about which bag you're in — which shifts the probability of $R_2$. The dependence is entirely mediated through the **latent variable** (the bag). This is the prototype of how hidden structure creates marginal dependence — you'll see this constantly in statistics and ML.

---

## The Key Structural Point


| Ordinary LOTP                        | LOTP with Extra Conditioning                          |
| :----------------------------------- | :---------------------------------------------------- |
| $P(B) = \sum_i P(B \mid A_i),P(A_i)$ | $P(B \mid E) = \sum_i P(B \mid A_i, E),P(A_i \mid E)$ |
| Partition of $\Omega$                | Partition of $\Omega$, restricted to $E$              |
| $P(A_i)$ are weights                 | $P(A_i \mid E)$ are the updated weights               |

> [!tip] _Think of $E$ as shrinking the sample space to $\Omega_E = E$. In that smaller universe, LOTP holds with exactly the same structure — just every probability inherits $E$ as background._
