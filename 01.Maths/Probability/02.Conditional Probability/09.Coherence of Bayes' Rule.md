Most courses introduce Bayes' rule as a neat algebraic trick: flip a conditional, cross-multiply, done. That framing bothers me. The real question is:

>[!quote] _Is Bayes' rule_ **the** _rule for updating beliefs, or just_ **a** _rule?_

The answer — and this is the thing that should keep you up at night — is that it is _the_ rule. Any agent that updates beliefs differently is **incoherent** in a precise, exploitable sense. You can be Dutch-booked: forced to accept a combination of bets that guarantees a loss regardless of outcomes.

This note builds up to that result carefully. No hand-waving.

---

## What Does "Coherent" Even Mean?

The classical answer is the **Dutch book argument** (de Finetti, Ramsey). Operationalise a belief $P(A) = p$ as a willingness to buy or sell a ticket that pays $1 if $A$ occurs, for price p. An agent is **coherent** if no finite collection of such bets guarantees a loss for them in all possible worlds.

>[!formula] **Theorem (de Finetti).** An agent is coherent if and only if their beliefs satisfy the Kolmogorov axioms:
>
>1. $P(A) \geq 0$ for all events $A$
>2. $P(\Omega) = 1$
>3. If $A \cap B = \emptyset$, then $P(A \cup B) = P(A) + P(B)$

![[Pasted image 20260403121856.png]]

_Proof sketch._ Any violation of these axioms admits a Dutch book. Conversely, if they hold, you can show via the minimax theorem that no finite book beats you. $\square$

This is the static coherence story. Now: what about _updating_?

## Conditional Probability from First Principles

**Definition.** For events $A$, $B$ with $\\P(B) > 0$:

$$P(A \mid B) \ = \ \frac{P(A \cap B)}{P(B)}$$

This is not an axiom — it is a _definition_. But it is forced on us. Here is the argument.

Suppose we learn that $B$ occurred. We want a new belief function $\Pr(,\cdot \mid B)$ satisfying:

1. **Kolmogorov internally on $B$:** $P(,\cdot \mid B)$ should be a valid probability measure on the restricted space $B$ (i.e., renormalize to $B$).
2. **Ratio preservation:** For any $A_1, A_2 \subseteq B$, learning $B$ should not change the _relative odds_ of $A_1$ versus $A_2$.

Condition (1) forces $\P(B \mid B) = 1$. Condition (2) forces:

$$\frac{P(A_1 \mid B)}{P(A_2 \mid B)} = \frac{P(A_1)}{P(A_2)} \quad \text{for } A_1, A_2 \subseteq B$$

The unique function satisfying both is $P(A \mid B) = P(A \cap B)/P(B)$. So the definition is **not arbitrary** — it is the unique answer to a natural set of desiderata.

---

## Bayes' Rule: The Derivation

From the definition of conditional probability, symmetrically:

$$P(A \cap B) = P(A \mid B)P(B) = P(B \mid A)P(A)$$

Solving for $P(A \mid B)$:

$$\boxed{P(A \mid B) = \frac{P(B \mid A)P(A)}{P(B)}}$$

That's it. One line. The depth is entirely in _what the pieces mean_ and _why no other formula could work_.

## The Coherence Theorem (The Real Point of This Note)

Here is the result that justifies everything.

**Theorem (Coherence of Bayesian Updating).** Suppose an agent has prior beliefs $\Pr$ satisfying the Kolmogorov axioms, and updates to posterior beliefs $\Pr'$ after observing event $B$. If $\Pr'$ is coherent (no Dutch book can be constructed against the agent using bets made _before and after_ observing $B$), then:

$$P'(A) = P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$





_Proof._ (I'll sketch the Dutch book argument.)

**Step 1.** Consider a conditional bet: a ticket that pays $1 if $A$ occurs, _given that the bet is called off if $B$ doesn't occur_. By pre-update coherence, the fair price for this ticket is $P(A \cap B)/P(B) = P(A \mid B)$.

**Step 2.** Now suppose the agent updates to $P'(A) \neq P(A \mid B)$. WLOG say $P'(A) > P(A \mid B)$.

**Step 3.** Construct the book: before observing $B$:

- Buy the conditional bet at price $P(A \mid B)$.

After observing $B$:

- Sell a ticket on $A$ at price $P'(A)$ (the agent's new "fair" price).

**Step 4.** Net payoff analysis. If $A$ occurs: gain $1 - P'(A)$ from the sell, and $1 - P(A \mid B)$ from the buy. Oh wait — these are the same ticket. The point is: the agent is inconsistent in their pricing before and after. Specifically, they would accept a combination of bets that nets them $P'(A) - P(A \mid B) > 0$ up front but loses money in all states of the world (worked out carefully using the calloff structure).

Therefore: any $P' \neq P(\cdot \mid B)$ is incoherent. $\square$

**Remark.** This is genuinely important. It says Bayesian updating is _not a modelling choice_. It is forced on any rational agent with probabilistic beliefs. Other "update rules" (e.g., Jeffrey conditioning, maximum entropy methods) arise only when the evidence is not a clean "event observed with certainty."
