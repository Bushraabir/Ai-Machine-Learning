
> _"Before asking where a chain goes in the long run, I have to ask what kind of states it even has. Some states the chain visits forever. Some it eventually abandons and never returns to. The classification of states is the taxonomy that makes the convergence theory possible."_

---

## Why Classify States?

The convergence theorem ŌĆö that $\mathbf{t}^{(n)} \to \mathbf{s}$ ŌĆö doesn't hold for every Markov chain. It requires conditions: **irreducibility** and **aperiodicity**. Both of these are properties of the chain's states. Before I can state or prove the convergence theorem, I need a precise vocabulary for describing how states relate to each other and how the chain revisits them.

The classification of states answers two fundamental questions:

1. **Will the chain ever return to state $i$?** (Recurrence vs. transience)
2. **How does the chain's graph structure look?** (Communicating classes, irreducibility)
3. **Is the chain's return behavior periodic?** (Aperiodicity)

---

## Accessibility and Communication

### Definition: Accessibility

State $j$ is **accessible** from state $i$, written $i \to j$, if:

$$q_{ij}^{(n)} = P(X_n = j \mid X_0 = i) > 0 \quad \text{for some } n \geq 0$$

There exists some $n$ for which there is positive probability of reaching $j$ starting from $i$.

**Convention:** $i \to i$ always, since $q_{ii}^{(0)} = 1$.

### Definition: Communication

States $i$ and $j$ **communicate**, written $i \leftrightarrow j$, if:

$$i \to j \quad \text{and} \quad j \to i$$

Both states are accessible from each other.

### Lemma: Communication is an Equivalence Relation

Communication satisfies:

- **Reflexivity:** $i \leftrightarrow i$ (trivially, $i \to i$)
- **Symmetry:** $i \leftrightarrow j \Rightarrow j \leftrightarrow i$ (by definition)
- **Transitivity:** $i \leftrightarrow j$ and $j \leftrightarrow k$ $\Rightarrow$ $i \leftrightarrow k$

**Proof of transitivity:** If $i \to j$ in $m$ steps and $j \to k$ in $n$ steps, then by Chapman-Kolmogorov:

$$q_{ik}^{(m+n)} = \sum_{\ell} q_{i\ell}^{(m)} q_{\ell k}^{(n)} \geq q_{ij}^{(m)} q_{jk}^{(n)} > 0$$

So $i \to k$. Similarly $k \to i$. $\blacksquare$

Because communication is an equivalence relation, it **partitions** the state space $\mathcal{S}$ into equivalence classes.

### Definition: Communicating Classes

The equivalence classes under $\leftrightarrow$ are called **communicating classes**. Within a class, every state can reach every other. Between different classes, communication is broken in at least one direction.

> _I think of the communicating classes as the strongly connected components of the directed graph on $\mathcal{S}$, where there's an edge $i \to j$ whenever $q_{ij} > 0$. The Chapman-Kolmogorov transitivity argument is exactly graph reachability ŌĆö if I can get from $i$ to $j$ and from $j$ to $k$, I can get from $i$ to $k$ in the concatenated number of steps._

## Irreducibility

### Definition

A Markov chain is **irreducible** if there is only one communicating class ŌĆö i.e., every state communicates with every other state:

$$i \leftrightarrow j \quad \text{for all } i, j \in \mathcal{S}$$

Equivalently: from any state, every other state is reachable in some finite number of steps.

### Why It Matters

An irreducible chain cannot get "stuck" in a subset of states. The long-run behavior is not sensitive to which communicating class the chain is initialized in, because there's only one. This is the condition that guarantees the stationary distribution is **unique**.

Without irreducibility, different communicating classes might have different stationary distributions ŌĆö or some classes might have none ŌĆö and the long-run behavior depends entirely on which class the chain starts in.

> _The simplest failure of irreducibility: two absorbing states $a$ and $b$, with no way to move between them. If the chain starts near $a$ it gets absorbed there; near $b$ it gets absorbed there. There are two stationary distributions, $\delta_a$ and $\delta_b$, and every convex combination is also stationary. Irreducibility kills all of this pathology._

## Recurrence and Transience

This is the deeper classification ŌĆö not about the graph structure, but about whether the chain **returns** to each state.

### Definition: First Return Time

Define the **first return time** to state $i$:

$$T_i = \min{n \geq 1 : X_n = i}$$

with the convention $T_i = \infty$ if the chain never returns. Then:

$$f_{ii} = P(T_i < \infty \mid X_0 = i) = P(\text{chain ever returns to } i \mid \text{starts at } i)$$

### Definition: Recurrent and Transient States

- State $i$ is **recurrent** if $f_{ii} = 1$ ŌĆö the chain returns to $i$ with probability 1.
- State $i$ is **transient** if $f_{ii} < 1$ ŌĆö there is positive probability of never returning.

### The Canonical Theorem: Return Count Criterion

Let $N_i = \sum_{n=1}^{\infty} \mathbf{1}[X_n = i]$ be the total number of visits to state $i$ after time 0.

**Theorem.**

$$i \text{ is recurrent} \iff E[N_i \mid X_0 = i] = \infty \iff \sum_{n=1}^{\infty} q_{ii}^{(n)} = \infty$$

$$i \text{ is transient} \iff E[N_i \mid X_0 = i] < \infty \iff \sum_{n=1}^{\infty} q_{ii}^{(n)} < \infty$$

**Proof.**

Write $N_i = \sum_{n=1}^{\infty} \mathbf{1}[X_n = i]$. By linearity of expectation:

$$E[N_i \mid X_0 = i] = \sum_{n=1}^{\infty} P(X_n = i \mid X_0 = i) = \sum_{n=1}^{\infty} q_{ii}^{(n)}$$

Now I need to connect this to $f_{ii}$.

Given $X_0 = i$, the number of returns $N_i$ is a geometric-type random variable. The chain returns to $i$ for the first time with probability $f_{ii}$. Each time it returns, the future is independent of the past (Markov property), so the chain again returns with probability $f_{ii}$. Therefore $N_i \mid X_0 = i$ has a **Geometric** distribution on ${0, 1, 2, \ldots}$ with success probability $f_{ii}$:

$$P(N_i \geq k \mid X_0 = i) = f_{ii}^k$$

So:

$$E[N_i \mid X_0 = i] = \sum_{k=1}^{\infty} P(N_i \geq k \mid X_0 = i) = \sum_{k=1}^{\infty} f_{ii}^k = \frac{f_{ii}}{1 - f_{ii}}$$

when $f_{ii} < 1$ (transient), this is finite. When $f_{ii} = 1$ (recurrent), this is $\infty$. $\blacksquare$

> _The geometric structure here is the Markov property in action again. Each return to state $i$ "resets" the chain ŌĆö from $i$'s perspective, the chain has no memory of how many times it has been there before. So the number of visits is exactly geometric. This is the cleanest proof I know of the recurrence criterion._

### Corollary: Transient States Are Visited Finitely Often

If $i$ is transient, then $N_i < \infty$ almost surely ŌĆö the chain visits $i$ only finitely many times, and eventually leaves and never comes back.

If $i$ is recurrent, then $N_i = \infty$ almost surely ŌĆö the chain returns to $i$ infinitely often.

This is a 0-1 law: not "sometimes finite, sometimes infinite" ŌĆö it's always one or the other, depending only on whether $f_{ii} = 1$.

## Recurrence and Transience Are Class Properties

### Theorem

If $i \leftrightarrow j$ (they communicate), then $i$ is recurrent if and only if $j$ is recurrent.

**Proof.**

Suppose $i \leftrightarrow j$. Then there exist $m, n \geq 1$ such that $q_{ij}^{(m)} > 0$ and $q_{ji}^{(n)} > 0$. Let $\alpha = q_{ij}^{(m)} q_{ji}^{(n)} > 0$.

For any $k \geq 0$, by Chapman-Kolmogorov:

$$q_{ii}^{(m+k+n)} \geq q_{ij}^{(m)} \cdot q_{jj}^{(k)} \cdot q_{ji}^{(n)} = \alpha \cdot q_{jj}^{(k)}$$

Summing over $k$:

$$\sum_{k=0}^{\infty} q_{ii}^{(m+k+n)} \geq \alpha \sum_{k=0}^{\infty} q_{jj}^{(k)}$$

If $j$ is recurrent, $\sum_k q_{jj}^{(k)} = \infty$, so $\sum_k q_{ii}^{(m+k+n)} = \infty$, hence $\sum_k q_{ii}^{(k)} = \infty$, so $i$ is recurrent. By symmetry, $j$ recurrent $\Rightarrow$ $i$ recurrent. $\blacksquare$

### Consequence

Every state in a communicating class has the same recurrence/transience classification. So I can speak of **recurrent classes** and **transient classes** without ambiguity.

## Closed and Open Classes

### Definition

A communicating class $C$ is **closed** if no state outside $C$ is accessible from any state inside $C$:

$$i \in C, , j \notin C \Rightarrow q_{ij}^{(n)} = 0 \text{ for all } n$$

A class that is not closed is **open**.

### Theorem

Every recurrent class is closed. Every finite closed class is recurrent.

**Proof sketch (recurrent $\Rightarrow$ closed):** Suppose $C$ is recurrent but not closed. Then some $i \in C$ can reach $j \notin C$. Since $j \notin C$ and $i \in C$, we have $j \not\to i$ (otherwise $i \leftrightarrow j$, contradicting $j \notin C$). So once the chain reaches $j$, it never returns to $i$ ŌĆö contradicting recurrence of $i$. $\blacksquare$

**Proof sketch (finite closed class $\Rightarrow$ recurrent):** In a finite closed class, the chain can never escape. It must visit some state infinitely often (by pigeonhole ŌĆö finitely many states, infinite time). By the class property, all states in the class are visited infinitely often, so all are recurrent. $\blacksquare$

> _The pigeonhole argument for finite closed classes is the reason finite irreducible chains are always recurrent ŌĆö there's nowhere to escape to, and with finitely many states, the chain must keep coming back. This is why the theory simplifies so much in the finite case._

## Aperiodicity

### Definition: Period

The **period** of state $i$ is:

$$d(i) = \gcd{n \geq 1 : q_{ii}^{(n)} > 0}$$

the greatest common divisor of all return times. If ${n \geq 1 : q_{ii}^{(n)} > 0} = \emptyset$ (i.e., $i$ is transient and never returns), set $d(i) = \infty$ by convention.

State $i$ is **aperiodic** if $d(i) = 1$ and **periodic** with period $d > 1$ if $d(i) = d$.

### Lemma: Period Is a Class Property

If $i \leftrightarrow j$, then $d(i) = d(j)$.

**Proof.** Since $i \leftrightarrow j$, there exist $m, n$ with $q_{ij}^{(m)} > 0$ and $q_{ji}^{(n)} > 0$. Note $q_{ii}^{(m+n)} \geq q_{ij}^{(m)} q_{ji}^{(n)} > 0$, so $d(i)$ divides $m + n$.

For any $k$ with $q_{jj}^{(k)} > 0$:

$$q_{ii}^{(m+k+n)} \geq q_{ij}^{(m)} q_{jj}^{(k)} q_{ji}^{(n)} > 0$$

so $d(i)$ divides $m + k + n$. Since $d(i)$ divides $m + n$, it divides $k$. So $d(i)$ divides every element of ${k : q_{jj}^{(k)} > 0}$, hence $d(i) \leq d(j)$. By symmetry $d(j) \leq d(i)$, so $d(i) = d(j)$. $\blacksquare$

So I can speak of the **period of a communicating class**.

### What Periodicity Looks Like

A chain with period $d > 1$ cycles through $d$ groups of states in a fixed order. The canonical example:

**Cycle graph on ${0, 1, \ldots, d-1}$:** $q_{i, i+1 \bmod d} = 1$. The chain moves deterministically around the cycle, returning to any state only at times $d, 2d, 3d, \ldots$. Period = $d$.

For such a chain, $q_{ii}^{(n)} = 0$ whenever $d \nmid n$, and the chain does not converge in distribution ŌĆö it oscillates. This is why aperiodicity is a hypothesis in the convergence theorem.

### Sufficient Conditions for Aperiodicity

- If $q_{ii} > 0$ for some $i$ (self-loop), then $d(i) = 1$, so $i$ is aperiodic. If the chain is irreducible, the whole chain is aperiodic.
- If the chain can return to $i$ in both $k$ and $k+1$ steps (for some $k$), then $\gcd(k, k+1) = 1$, so $d(i) = 1$.

> _In practice, whenever I see a self-loop in the transition matrix, I immediately know the chain is aperiodic (if irreducible). Many applied chains ŌĆö random walks with laziness, MCMC with rejection ŌĆö are made aperiodic precisely by adding a self-loop: stay with probability $1/2$, move with probability $1/2$. This is a standard trick._

## The Full Picture: State Classification Tree

```
States of a Markov Chain
Ōöé
Ōö£ŌöĆŌöĆ Recurrent (return with prob. 1; visited infinitely often)
Ōöé   Ōöé
Ōöé   Ōö£ŌöĆŌöĆ Positive recurrent: E[T_i | X_0 = i] < Ōł×
Ōöé   Ōöé   ŌööŌöĆŌöĆ These states have a stationary distribution
Ōöé   Ōöé
Ōöé   ŌööŌöĆŌöĆ Null recurrent: E[T_i | X_0 = i] = Ōł×
Ōöé       ŌööŌöĆŌöĆ Return a.s., but expected return time is infinite
Ōöé           (occurs only in countably infinite state spaces)
Ōöé
ŌööŌöĆŌöĆ Transient (positive prob. of never returning; visited finitely often)
```

**For finite state spaces:** Null recurrence cannot occur. Every recurrent state is positive recurrent. So the classification simplifies to just recurrent vs. transient.

**For countably infinite state spaces** (e.g., random walks on $\mathbb{Z}$): null recurrence is possible and important. The simple random walk on $\mathbb{Z}$ is null recurrent ŌĆö it returns to 0 with probability 1, but the expected return time is infinite.

> _Null recurrence is the strange middle ground: the chain keeps coming back, but it takes so long on average that there's no stationary distribution. I find this the most counterintuitive phenomenon in the classification ŌĆö "recurrent but no stationary distribution" feels paradoxical. It only happens in infinite state spaces, which is why the finite theory is cleaner._

## Putting It Together: The Convergence Theorem

**Theorem (Convergence to Stationarity).** Let $(X_n)$ be an **irreducible** and **aperiodic** Markov chain on a finite state space. Then:

1. There exists a unique stationary distribution $\mathbf{s}$.
2. For all $i, j \in \mathcal{S}$:

$$q_{ij}^{(n)} \to s_j \quad \text{as } n \to \infty$$

3. For any initial distribution $\mathbf{t}^{(0)}$:

$$\mathbf{t}^{(n)} \to \mathbf{s} \quad \text{as } n \to \infty$$

The role of each condition:

- **Irreducibility:** Ensures there is only one communicating class, so the stationary distribution is unique. Without it, the limit could depend on the starting class.
- **Aperiodicity:** Ensures $q_{ij}^{(n)}$ actually converges (doesn't oscillate). Without it, $Q^n$ might not converge even if a stationary distribution exists.
- **Finite state space:** Ensures positive recurrence (no null recurrence). In infinite state spaces, need to additionally assume positive recurrence.

> _This theorem is the payoff for all the classification work. Every condition in the hypothesis corresponds to exactly one pathology that it rules out. Irreducibility rules out multiple stationary distributions. Aperiodicity rules out oscillation. Finiteness rules out null recurrence. It's a beautifully tight set of hypotheses._

## Examples

### Example 1: Irreducible and Aperiodic

$$Q = \begin{pmatrix} 1/2 & 1/2 \ 1/3 & 2/3 \end{pmatrix}$$

Both states communicate ($q_{12}, q_{21} > 0$), so irreducible. Self-loops ($q_{11} = 1/2 > 0$), so aperiodic. Convergence theorem applies. Unique stationary distribution: solve $\mathbf{s}Q = \mathbf{s}$, get $\mathbf{s} = (2/5, 3/5)$.

### Example 2: Irreducible but Periodic

$$Q = \begin{pmatrix} 0 & 1 \ 1 & 0 \end{pmatrix}$$

States communicate, so irreducible. But $q_{11}^{(n)} = 1$ if $n$ even, $0$ if $n$ odd. Period $d = 2$. The chain oscillates forever ŌĆö $Q^n$ does not converge. The stationary distribution $(1/2, 1/2)$ exists but $\mathbf{t}^{(n)}$ oscillates around it rather than converging to it.

### Example 3: Reducible ŌĆö Absorbing States

$$Q = \begin{pmatrix} 1 & 0 & 0 \ 1/2 & 0 & 1/2 \ 0 & 0 & 1 \end{pmatrix}$$

States ${1}$ and ${3}$ are absorbing (closed recurrent classes). State ${2}$ is transient ŌĆö it can reach 1 or 3 but neither can reach it back. The chain gets absorbed in 1 or 3; the long-run distribution depends on the starting state.

## One-Line Summary for Future Me

> States are classified by two independent axes: **recurrence** (does the chain return?) and **period** (does the return time have a GCD $> 1$?). Both are class properties. An irreducible (one class), aperiodic ($d = 1$), positive recurrent chain on a finite state space is the "nice" case ŌĆö unique stationary distribution, convergence from anywhere.
