

> _The joint PDF is not a probability — it's a density. The probability lives in the volume under the surface, and that's everything. Internalize this and the rest follows._

## Motivation

In the discrete case, the joint PMF assigns actual probabilities to individual points: $P(X = x, Y = y)$. In the continuous case, individual points have probability zero. The right object is instead a function $f_{X,Y}$ such that probabilities of _regions_ are computed by integrating $f_{X,Y}$ over those regions.

The passage from discrete to continuous is just: **sums become integrals, PMF becomes PDF**. The conceptual structure is identical.

---

## Definition

A pair of continuous random variables $(X, Y)$ has **joint PDF** $f_{X,Y} : \mathbb{R}^2 \to \mathbb{R}$ if for every (reasonable) region $A \subseteq \mathbb{R}^2$,

$$P\bigl((X,Y) \in A\bigr) = \iint_A f_{X,Y}(x,y), dx, dy.$$

In particular, for a rectangle:

$$P(a \leq X \leq b,; c \leq Y \leq d) = \int_a^b \int_c^d f_{X,Y}(x,y), dy, dx.$$

---

## Valid Joint PDF: The Two Conditions

$f_{X,Y}$ is a valid joint PDF if and only if:

1. **Nonnegativity:** $f_{X,Y}(x,y) \geq 0$ for all $(x,y) \in \mathbb{R}^2$.
    
2. **Normalisation:** $\displaystyle\int_{-\infty}^{\infty}\int_{-\infty}^{\infty} f_{X,Y}(x,y), dx, dy = 1.$
    

That's it. The "probability" of any event is then recovered by integrating over the corresponding region. Note that $f_{X,Y}(x,y)$ itself can exceed 1 — it's a _density_, not a probability. What matters is that it integrates to 1.

---

## Marginal PDFs

Exactly parallel to the discrete case, marginalizing means integrating out the unwanted variable:

$$\boxed{f_X(x) = \int_{-\infty}^{\infty} f_{X,Y}(x,y), dy}$$

$$f_Y(y) = \int_{-\infty}^{\infty} f_{X,Y}(x,y), dx.$$

The proof that $f_X$ is a valid PDF: nonnegativity is clear, and

$$\int_{-\infty}^{\infty} f_X(x), dx = \int_{-\infty}^{\infty}\int_{-\infty}^{\infty} f_{X,Y}(x,y), dy, dx = 1$$

by Fubini (all terms nonneg, so interchange is justified). Exactly the same argument as in the discrete case, just with Fubini instead of Tonelli.

---

## Independence

$(X,Y)$ are independent (as continuous r.v.'s) iff

$$f_{X,Y}(x,y) = f_X(x), f_Y(y) \quad \text{for all } (x,y).$$

**Practical check:** can you write $f_{X,Y}(x,y) = g(x) \cdot h(y)$ for some functions $g,h$, _and_ the support is a rectangle (does not depend on $x$ or $y$ jointly)? If yes, $X \perp Y$, and then $g$ and $h$ are automatically proportional to $f_X$ and $f_Y$.

The support condition is easy to forget. If the support is a triangle like ${0 < x < y < 1}$, the joint does _not_ factor no matter what $f$ looks like — knowing $Y$ tells you something about the range of $X$, so they cannot be independent.

---

## Worked Example 1: Uniform on the Unit Square

$$f_{X,Y}(x,y) = 1, \quad 0 < x < 1,; 0 < y < 1.$$

**Marginals:**

$$f_X(x) = \int_0^1 1, dy = 1, \quad 0 < x < 1.$$

So $X \sim \text{Uniform}(0,1)$, and by symmetry $Y \sim \text{Uniform}(0,1)$.

**Independence:** $f_{X,Y}(x,y) = 1 = f_X(x) \cdot f_Y(y)$, and the support is the square (rectangular). So $X \perp Y$. ✓

---

## Worked Example 2: Uniform on the Triangle

$$f_{X,Y}(x,y) = 2, \quad 0 < x < y < 1.$$

_(The factor of 2 ensures normalisation: the triangle has area $1/2$, so $2 \cdot 1/2 = 1$.)_

**Marginals:** For $0 < x < 1$:

$$f_X(x) = \int_x^1 2, dy = 2(1-x).$$

For $0 < y < 1$:

$$f_Y(y) = \int_0^y 2, dx = 2y.$$

**Check:** $\int_0^1 2(1-x),dx = 2[x - x^2/2]_0^1 = 1$. ✓

**Independence?** $f_X(x) \cdot f_Y(y) = 4y(1-x) \neq 2 = f_{X,Y}(x,y)$ in general. Also the support is a triangle, not a rectangle. So $X$ and $Y$ are _dependent_.

**Intuition:** If I tell you $Y = 0.1$, then $X$ must be less than $0.1$. Knowing $Y$ constrains $X$. Of course they're dependent.

---

## Worked Example 3: Exponential Joint

$$f_{X,Y}(x,y) = e^{-x-y}, \quad x > 0,; y > 0.$$

This factors immediately: $e^{-x-y} = e^{-x} \cdot e^{-y}$, and the support is the first quadrant (a rectangle). So $X \sim \text{Expo}(1)$, $Y \sim \text{Expo}(1)$, and $X \perp Y$. No integration needed — the factorisation is the proof.

---

## Computing Probabilities: The Region Matters

**Example.** With $(X,Y)$ uniform on $[0,1]^2$, find $P(X + Y \leq 1)$.

The region is a right triangle with vertices $(0,0),(1,0),(0,1)$, area $1/2$.

$$P(X+Y \leq 1) = \int_0^1 \int_0^{1-x} 1, dy, dx = \int_0^1 (1-x), dx = \frac{1}{2}.$$

Or just: area of region times constant density $= 1/2 \cdot 1 = 1/2$. **Always draw the region first.**

**Example.** With $(X,Y)$ uniform on the triangle ${0 < x < y < 1}$, find $P(Y - X > 1/2)$.

$$P(Y - X > 1/2) = \int_0^{1/2}\int_{x+1/2}^{1} 2, dy, dx = \int_0^{1/2} 2!\left(\tfrac{1}{2} - x\right)dx = \bigl[x - x^2\bigr]_0^{1/2} = \frac{1}{4}.$$

## The Joint CDF

$$F_{X,Y}(x,y) = P(X \leq x, Y \leq y) = \int_{-\infty}^x \int_{-\infty}^y f_{X,Y}(s,t), dt, ds.$$

Recovering the PDF: if $F_{X,Y}$ is sufficiently smooth,

$$f_{X,Y}(x,y) = \frac{\partial^2}{\partial x, \partial y} F_{X,Y}(x,y).$$

I rarely need to go through the CDF directly — setting up the double integral over the region is usually more transparent. But the mixed partial formula is worth knowing cold.

## Non-Rectangular Supports: The Dangerous Case

When the support of $(X,Y)$ is not a rectangle, the limits of integration in the marginals depend on the other variable, and the order of integration matters.

General strategy: draw the support region. For $f_X(x)$: fix $x$, ask "what values can $y$ take?" → those are the inner limits. Then integrate $x$ over its full marginal range.

**Example.** $f_{X,Y}(x,y) = c \cdot x$ on ${0 < x < 1, 0 < y < x}$.

_Find $c$:_

$$1 = \int_0^1 \int_0^x cx, dy, dx = \int_0^1 cx^2, dx = \frac{c}{3} \implies c = 3.$$

_Marginal of $X$:_

$$f_X(x) = \int_0^x 3x, dy = 3x^2, \quad 0 < x < 1. \qquad [\text{Beta}(3,1)]$$

_Marginal of $Y$:_ Fix $y \in (0,1)$; $x$ ranges from $y$ to $1$ (since we need $x > y$):

$$f_Y(y) = \int_y^1 3x, dx = \frac{3}{2}(1-y^2), \quad 0 < y < 1.$$

**The critical step:** when switching to $f_Y$, the lower limit of the $x$-integral is $y$, not $0$. Getting this wrong is the single most common mistake on these problems.

## A Note on Existence

Not every pair of random variables admits a joint PDF. It is possible for both marginals to be continuous yet the joint to be singular. The canonical example: $X \sim N(0,1)$ and $Y = X$. Then $(X,Y)$ is concentrated on the line $y = x$ in $\mathbb{R}^2$, which has 2D Lebesgue measure zero, so no joint density exists. The formal statement is that a joint PDF exists iff the joint distribution is absolutely continuous with respect to 2D Lebesgue measure.

The Bivariate Normal is the canonical well-behaved example:

$$f_{X,Y}(x,y) = \frac{1}{2\pi\sigma_X\sigma_Y\sqrt{1-\rho^2}} \exp!\left(-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_X)^2}{\sigma_X^2} - \frac{2\rho(x-\mu_X)(y-\mu_Y)}{\sigma_X\sigma_Y} + \frac{(y-\mu_Y)^2}{\sigma_Y^2}\right]\right).$$

Takeaway for now: both marginals are normal, and $X \perp Y \iff \rho = 0$. The bivariate normal is the unique distribution where uncorrelated implies independent — this is special to the Gaussian family and fails in general.
