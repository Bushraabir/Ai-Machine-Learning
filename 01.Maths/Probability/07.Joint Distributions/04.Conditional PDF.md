

> _The conditional PDF is where the continuous theory gets genuinely subtle. In the discrete case, conditioning on $X = x$ is unambiguous — it's an event with positive probability. In the continuous case, $P(X = x) = 0$ for every $x$, so you can't condition the usual way. The definition that saves us is elegant, but you should understand why it's forced on us, not just accept it as a decree._

---

## The Problem with Continuous Conditioning

Recall: for discrete RVs, the conditional PMF is just Bayes' rule applied to point events:

$$p_{Y|X}(y \mid x) = \frac{p_{X,Y}(x,y)}{p_X(x)}$$

This works because $P(X = x) = p_X(x) > 0$ (assuming $x$ is in the support).

For continuous RVs, $P(X = x) = 0$ for every $x$. So writing

$$P(Y \leq y \mid X = x) = \frac{P(Y \leq y,; X = x)}{P(X = x)} = \frac{0}{0}$$

is meaningless. We need a different approach.

---

## The Right Definition, Derived

The idea: replace the zero-probability event ${X = x}$ with a thin strip ${x < X \leq x + \varepsilon}$, which has positive probability, do ordinary conditioning there, and take $\varepsilon \to 0$.

For fixed $y$, define the conditional CDF we want as:

$$P(Y \leq y \mid X = x) := \lim_{\varepsilon \to 0} P(Y \leq y \mid x < X \leq x + \varepsilon)$$

Expanding the right side:

$$P(Y \leq y \mid x < X \leq x + \varepsilon) = \frac{P(Y \leq y,; x < X \leq x + \varepsilon)}{P(x < X \leq x + \varepsilon)}$$

**Numerator:** $$P(Y \leq y,; x < X \leq x+\varepsilon) = \int_x^{x+\varepsilon} \int_{-\infty}^y f_{X,Y}(s,t), dt, ds \approx \varepsilon \int_{-\infty}^y f_{X,Y}(x,t), dt$$

**Denominator:** $$P(x < X \leq x + \varepsilon) = \int_x^{x+\varepsilon} f_X(s), ds \approx \varepsilon, f_X(x)$$

Dividing and taking $\varepsilon \to 0$:

$$P(Y \leq y \mid X = x) = \frac{\int_{-\infty}^y f_{X,Y}(x,t), dt}{f_X(x)}$$

Differentiating in $y$ gives the conditional density:

$$\boxed{f_{Y|X}(y \mid x) = \frac{f_{X,Y}(x,y)}{f_X(x)}}$$

defined for all $x$ with $f_X(x) > 0$.

> _The formula looks identical to the discrete case, with densities in place of PMFs. That's not a coincidence — the derivation above shows it's the natural limit of ordinary conditioning. The notation hides the limiting argument that justifies it._

---

## Verification: It's a Valid PDF

For each fixed $x$ with $f_X(x) > 0$, the function $y \mapsto f_{Y|X}(y \mid x)$ is a genuine PDF:

**Non-negativity:** $f_{X,Y}(x,y) \geq 0$ and $f_X(x) > 0$, so $f_{Y|X}(y \mid x) \geq 0$. ✓

**Normalisation:** $$\int_{-\infty}^\infty f_{Y|X}(y \mid x), dy = \int_{-\infty}^\infty \frac{f_{X,Y}(x,y)}{f_X(x)}, dy = \frac{1}{f_X(x)} \int_{-\infty}^\infty f_{X,Y}(x,y), dy = \frac{f_X(x)}{f_X(x)} = 1 \checkmark$$

---

## The Factorisation Identity

Rearranging the definition gives the **chain rule for densities**:

$$f_{X,Y}(x,y) = f_{Y|X}(y \mid x), f_X(x) = f_{X|Y}(x \mid y), f_Y(y)$$

This is the continuous analogue of $P(A \cap B) = P(A|B)P(B)$. Both factorisations are valid; choosing which to use depends on what's natural to specify first.

**Bayes' rule for densities** falls out immediately by equating the two factorisations:

$$f_{X|Y}(x \mid y) = \frac{f_{Y|X}(y \mid x), f_X(x)}{f_Y(y)}$$

This is exactly [[09.Laplace's Rule of Succession]]'s mechanism, now stated for general densities. The denominator $f_Y(y) = \int f_{Y|X}(y|x) f_X(x), dx$ is the normalising constant (total probability).

---

## Conditional Expectation from the Conditional PDF

Given the conditional PDF, the **conditional expectation** is just the expected value under that density:

$$\mathbb{E}[g(Y) \mid X = x] = \int_{-\infty}^\infty g(y), f_{Y|X}(y \mid x), dy$$

As $x$ varies, this is a function of $x$. Setting $h(x) = \mathbb{E}[Y \mid X = x]$, we get the random variable $\mathbb{E}[Y \mid X] = h(X)$.

**Adam's Law** then says $\mathbb{E}[Y] = \mathbb{E}[h(X)]$, which in integral form is:

$$\mathbb{E}[Y] = \int_{-\infty}^\infty \mathbb{E}[Y \mid X = x], f_X(x), dx = \int_{-\infty}^\infty \int_{-\infty}^\infty y, f_{Y|X}(y|x), f_X(x), dy, dx$$

which reduces to $\int\int y, f_{X,Y}(x,y), dy, dx = \mathbb{E}[Y]$ by Fubini. Consistent. ✓

## Worked Examples

### Example 1: Uniform on a Triangle

Let $(X, Y)$ be uniform on the triangle $\mathcal{T} = {(x,y) : 0 < x < 1,; 0 < y < x}$.

The area of $\mathcal{T}$ is $\frac{1}{2}$, so $f_{X,Y}(x,y) = 2$ on $\mathcal{T}$.

**Marginal of $X$:** $$f_X(x) = \int_0^x 2, dy = 2x, \quad 0 < x < 1$$

**Conditional PDF of $Y$ given $X = x$:** $$f_{Y|X}(y \mid x) = \frac{f_{X,Y}(x,y)}{f_X(x)} = \frac{2}{2x} = \frac{1}{x}, \quad 0 < y < x$$

So $Y \mid X = x \sim \text{Uniform}(0, x)$. This makes sense: given the "width" of the triangle at height $x$, the $y$-value is uniform over $[0,x]$.

**Conditional mean:** $\mathbb{E}[Y \mid X = x] = x/2$, so $\mathbb{E}[Y \mid X] = X/2$.

**Marginal mean via Adam's law:** $$\mathbb{E}[Y] = \mathbb{E}[X/2] = \frac{1}{2}\mathbb{E}[X] = \frac{1}{2} \cdot \frac{2}{3} = \frac{1}{3}$$

(using $\mathbb{E}[X] = \int_0^1 x \cdot 2x, dx = 2/3$ since $X \sim \text{Beta}(2,1)$.)

---

### Example 2: Exponential Conditioning

Let $X \sim \text{Expo}(\lambda)$ and $Y \mid X = x \sim \text{Uniform}(0, x)$.

**Joint PDF:** $$f_{X,Y}(x,y) = f_{Y|X}(y|x), f_X(x) = \frac{1}{x} \cdot \lambda e^{-\lambda x}, \quad 0 < y < x$$

**Marginal of $Y$:** integrate out $x$ over ${x > y}$: $$f_Y(y) = \int_y^\infty \frac{\lambda e^{-\lambda x}}{x}, dx$$

This integral doesn't have a closed form in elementary functions (it involves the exponential integral $\text{Ei}$). That's fine — the marginal of $Y$ need not be a standard distribution.

**Conditional mean of $Y$:** $$\mathbb{E}[Y \mid X = x] = \frac{x}{2}$$

**Marginal mean via Adam's law:** $$\mathbb{E}[Y] = \mathbb{E}!\left[\frac{X}{2}\right] = \frac{1}{2\lambda}$$

using $\mathbb{E}[X] = 1/\lambda$.

> _Notice: we computed $\mathbb{E}[Y]$ without ever writing down $f_Y(y)$. Adam's law lets you short-circuit the marginalisation entirely. This is one of the most useful computational tricks in the subject._

---

### Example 3: Recovering a Known Distribution

Let $f_{X,Y}(x,y) = \lambda^2 e^{-\lambda y}$ for $0 < x < y$.

**Marginal of $X$:** $$f_X(x) = \int_x^\infty \lambda^2 e^{-\lambda y}, dy = \lambda^2 \cdot \frac{e^{-\lambda x}}{\lambda} = \lambda e^{-\lambda x}$$

So $X \sim \text{Expo}(\lambda)$.

**Conditional PDF of $Y$ given $X = x$:** $$f_{Y|X}(y \mid x) = \frac{\lambda^2 e^{-\lambda y}}{\lambda e^{-\lambda x}} = \lambda e^{-\lambda(y-x)}, \quad y > x$$

This is $\text{Expo}(\lambda)$ shifted to start at $x$: $Y \mid X = x \sim x + \text{Expo}(\lambda)$.

Interpretation: $Y$ is the time of the second arrival in a Poisson process, and $X$ is the first. Given the first arrival at time $x$, the second arrival is $x$ plus an independent $\text{Expo}(\lambda)$ waiting time — exactly the memoryless property.

## The Continuous Bayes' Rule in Practice

The factorisation $f_{X|Y}(x|y) \propto f_{Y|X}(y|x), f_X(x)$ is the engine of Bayesian inference:

- $f_X(x)$ is the **prior** on parameter $x$.
- $f_{Y|X}(y|x)$ is the **likelihood** of data $y$ given parameter $x$.
- $f_{X|Y}(x|y)$ is the **posterior** — what we believe about $x$ after seeing $y$.

The denominator $f_Y(y) = \int f_{Y|X}(y|x), f_X(x), dx$ is the **marginal likelihood** (or evidence), which is just the normalising constant ensuring the posterior integrates to 1.

In the Beta-Binomial model from [[09.Laplace's Rule of Succession]]], $x = p$ plays the role of the parameter, $y = s$ (observed successes) is the data, and the prior $f_X = \text{Beta}(\alpha,\beta)$ is conjugate to the Binomial likelihood — making the posterior analytically tractable.

---

## Conditional PDF vs Conditional Probability of an Event

A point of notation that causes confusion: $f_{Y|X}(y|x)$ is a **density**, not a probability. To get a probability, integrate:

$$P(Y \in B \mid X = x) = \int_B f_{Y|X}(y \mid x), dy$$

In particular, $f_{Y|X}(y|x)$ can exceed 1 — it's a density, not a probability mass. What must integrate to 1 over $y$ is the conditional density, not its pointwise values.

---

## Independence Restated

$X$ and $Y$ are independent if and only if the conditional PDF equals the marginal:

$$f_{Y|X}(y \mid x) = f_Y(y) \quad \text{for all } x, y$$

This says: conditioning on $X = x$ gives you no information about $Y$ — the distribution of $Y$ is the same regardless of what $X$ is. Formally:

$$f_{Y|X}(y|x) = f_Y(y) \iff \frac{f_{X,Y}(x,y)}{f_X(x)} = f_Y(y) \iff f_{X,Y}(x,y) = f_X(x)f_Y(y)$$

which is exactly the definition of independence. Full circle. ✓

---

## Pitfalls

**1. The conditioning value must be in the support.**  
$f_{Y|X}(y \mid x)$ is only defined where $f_X(x) > 0$. Conditioning on a value outside the support of $X$ is undefined.

**2. Non-product supports break independence even when the formula "looks" factored.**  
If $f_{X,Y}(x,y) = c$ on ${0 < x < y < 1}$, the density is constant — but the support is triangular, not a product set. The conditional $f_{Y|X}(y|x) = 1/(1-x)$ on $(x,1)$ depends on $x$, confirming dependence.

**3. Conditional densities are defined pointwise in $x$, but the conditional expectation $\mathbb{E}[Y \mid X]$ is defined up to a.e. equivalence.**  
At a single point $x$ with $f_X(x) = 0$, the conditional density is undefined. This rarely matters in calculations but matters in the measure-theoretic treatment (where $\mathbb{E}[Y \mid X]$ is defined as a Radon-Nikodym derivative — see Billingsley if you want the full story).

## Summary of Key Formulas

$$f_{Y|X}(y \mid x) = \frac{f_{X,Y}(x,y)}{f_X(x)}$$

$$f_{X,Y}(x,y) = f_{Y|X}(y \mid x), f_X(x)$$

$$f_{X|Y}(x \mid y) = \frac{f_{Y|X}(y \mid x), f_X(x)}{\int f_{Y|X}(y \mid x'), f_X(x'), dx'} \quad \text{(Bayes)}$$

$$\mathbb{E}[g(Y) \mid X = x] = \int g(y), f_{Y|X}(y \mid x), dy$$

$$\mathbb{E}[Y] = \int \mathbb{E}[Y \mid X = x], f_X(x), dx \quad \text{(Adam's law)}$$
