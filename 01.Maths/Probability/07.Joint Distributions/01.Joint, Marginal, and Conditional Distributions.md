
> _Most of probability is about one random variable at a time. Real problems involve many. The moment you have two RVs living on the same probability space, you need the joint distribution — and everything else (marginals, conditionals, independence) is derived from it. The joint is the ground truth; everything else is a projection or a slice._

---

## The Joint Distribution

**Definition (discrete).** For RVs $X$ and $Y$ on the same probability space, the **joint PMF** is:

$$p_{X,Y}(x, y) = P(X = x,, Y = y)$$

defined for all $(x, y)$ in the support. It satisfies: $$\sum_x \sum_y p_{X,Y}(x,y) = 1, \qquad p_{X,Y}(x,y) \geq 0$$

**Definition (continuous).** The **joint PDF** $f_{X,Y}(x,y)$ satisfies:

$$P((X,Y) \in A) = \iint_A f_{X,Y}(x,y), dx, dy$$

for all Borel sets $A \subseteq \mathbb{R}^2$. Normalisation: $$\int_{-\infty}^\infty \int_{-\infty}^\infty f_{X,Y}(x,y), dx, dy = 1$$

**Definition (joint CDF).** For any $(X, Y)$ (discrete, continuous, or mixed):

$$F_{X,Y}(x,y) = P(X \leq x,, Y \leq y)$$

The joint CDF always exists and uniquely determines the joint distribution. The PDF/PMF may not exist (e.g. if there's a point mass).

---

## Marginal Distributions

The **marginal distribution** of $X$ is simply the distribution of $X$ alone, recovered by summing/integrating out $Y$.

**Discrete:** $$p_X(x) = \sum_y p_{X,Y}(x, y), \qquad p_Y(y) = \sum_x p_{X,Y}(x,y)$$

**Continuous:** $$f_X(x) = \int_{-\infty}^\infty f_{X,Y}(x,y), dy, \qquad f_Y(y) = \int_{-\infty}^\infty f_{X,Y}(x,y), dx$$

> **Crucial point: you can always recover the marginals from the joint, but not vice versa.** The joint contains strictly more information. Two distributions can have identical marginals but completely different joint distributions — independence is a special additional structure.

### Example

Joint PMF of $(X, Y)$:

||$Y=0$|$Y=1$|$p_X$|
|---|---|---|---|
|$X=0$|$1/4$|$1/4$|$1/2$|
|$X=1$|$1/4$|$1/4$|$1/2$|
|$p_Y$|$1/2$|$1/2$|$1$|

Marginals: $X \sim \text{Bern}(1/2)$, $Y \sim \text{Bern}(1/2)$.

But the same marginals are consistent with many different joints. For instance:

||$Y=0$|$Y=1$|$p_X$|
|---|---|---|---|
|$X=0$|$1/2$|$0$|$1/2$|
|$X=1$|$0$|$1/2$|$1/2$|
|$p_Y$|$1/2$|$1/2$|$1$|

Here $X = Y$ with probability 1 — as far from independent as possible — yet the marginals are identical.

## Conditional Distributions

### Discrete Case

**Definition.** The **conditional PMF** of $Y$ given $X = x$ is:

$$p_{Y \mid X}(y \mid x) = P(Y = y \mid X = x) = \frac{P(X = x, Y = y)}{P(X = x)} = \frac{p_{X,Y}(x,y)}{p_X(x)}$$

defined for all $x$ with $p_X(x) > 0$.

This is just Bayes' rule / the definition of conditional probability, applied to events ${Y = y}$ and ${X = x}$.

For each fixed $x$, the function $y \mapsto p_{Y|X}(y \mid x)$ is a valid PMF: it sums to 1 over $y$.

### Continuous Case

Here we must be careful. The event ${X = x}$ has probability zero, so we can't naively write $P(Y \leq y \mid X = x) = P(Y \leq y, X = x)/P(X = x)$ — that's $0/0$.

**Definition.** If $f_{X,Y}(x,y)$ is the joint PDF and $f_X(x) > 0$, the **conditional PDF** of $Y$ given $X = x$ is:

$$f_{Y \mid X}(y \mid x) = \frac{f_{X,Y}(x,y)}{f_X(x)}$$

**Why is this the right definition?** It's the density version of the limit:

$$P(Y \leq y \mid x < X \leq x + dx) = \frac{P(Y \leq y,, x < X \leq x+dx)}{P(x < X \leq x+dx)} \xrightarrow{dx \to 0} \frac{\int_{-\infty}^y f_{X,Y}(x,t),dt}{f_X(x)}$$

Differentiating in $y$ gives $f_{X,Y}(x,y)/f_X(x)$.

## The Fundamental Factorisation

The definitions immediately give:

$$f_{X,Y}(x,y) = f_{Y \mid X}(y \mid x), f_X(x) = f_{X \mid Y}(x \mid y), f_Y(y)$$

This is the **chain rule** for joint distributions. It extends to $n$ variables:

$$f_{X_1, \ldots, X_n}(x_1, \ldots, x_n) = f_{X_1}(x_1), f_{X_2 \mid X_1}(x_2 \mid x_1), f_{X_3 \mid X_1, X_2}(x_3 \mid x_1, x_2) \cdots$$

> _This factorisation is the backbone of graphical models, Bayesian networks, and HMMs. Every time you specify a generative model by saying "sample $X_1$, then sample $X_2 \mid X_1$, then sample $X_3 \mid X_1, X_2$...", you are implicitly specifying a joint via this chain rule._

## Independence

**Definition.** $X$ and $Y$ are **independent** if:

$$f_{X,Y}(x,y) = f_X(x), f_Y(y) \quad \text{for all } (x,y)$$

(or $p_{X,Y}(x,y) = p_X(x), p_Y(y)$ in the discrete case).

**Equivalent characterisations:**

1. $f_{Y \mid X}(y \mid x) = f_Y(y)$ for all $x, y$ — knowing $X$ tells you nothing about $Y$.
2. $F_{X,Y}(x,y) = F_X(x), F_Y(y)$ for all $x, y$.
3. $\mathbb{E}[g(X)h(Y)] = \mathbb{E}[g(X)], \mathbb{E}[h(Y)]$ for all bounded Borel $g, h$.
4. The joint factors as a function of $x$ alone times a function of $y$ alone.

Characterisation 4 is a practical test: if you can write $f_{X,Y}(x,y) = g(x)h(y)$ with the support a product set, then $X \perp Y$.

### LOTUS and Independence

A fact used constantly: if $X \perp Y$, then $\mathbb{E}[XY] = \mathbb{E}[X]\mathbb{E}[Y]$, so $\text{Cov}(X,Y) = 0$. **But the converse fails:** uncorrelated does not imply independent in general.

**Standard counterexample.** $X \sim \text{Uniform}(-1, 1)$, $Y = X^2$. Then $\text{Cov}(X,Y) = \mathbb{E}[X^3] - \mathbb{E}[X]\mathbb{E}[X^2] = 0 - 0 = 0$, yet $Y$ is a deterministic function of $X$ — they are as dependent as possible.

## Conditional Expectation

**Definition.** The **conditional expectation** $\mathbb{E}[Y \mid X = x]$ is:

$$\mathbb{E}[Y \mid X = x] = \begin{cases} \sum_y y, p_{Y \mid X}(y \mid x) & \text{(discrete)} \ \int_{-\infty}^\infty y, f_{Y \mid X}(y \mid x), dy & \text{(continuous)} \end{cases}$$

This is a function of $x$. When we write $\mathbb{E}[Y \mid X]$, we mean this function evaluated at the random variable $X$ — so $\mathbb{E}[Y \mid X]$ is itself a random variable.

### Adam's Law (Law of Total Expectation)

$$\mathbb{E}[Y] = \mathbb{E}\left[\mathbb{E}[Y \mid X]\right]$$

**Proof (continuous).** $$\mathbb{E}[\mathbb{E}[Y \mid X]] = \int \mathbb{E}[Y \mid X = x], f_X(x), dx = \int \left(\int y, f_{Y|X}(y|x), dy\right) f_X(x), dx$$ $$= \int\int y, f_{X,Y}(x,y), dy, dx = \int y, f_Y(y), dy = \mathbb{E}[Y] \qquad \square$$

### Eve's Law (Law of Total Variance)

$$\text{Var}(Y) = \mathbb{E}[\text{Var}(Y \mid X)] + \text{Var}(\mathbb{E}[Y \mid X])$$

_Unexplained variance_ + _variance of explained mean_.

**Proof.** Write $\text{Var}(Y) = \mathbb{E}[Y^2] - (\mathbb{E}[Y])^2$. Apply Adam's law to each term:

$$\mathbb{E}[Y^2] = \mathbb{E}[\mathbb{E}[Y^2 \mid X]] = \mathbb{E}[\text{Var}(Y|X) + (\mathbb{E}[Y|X])^2]$$

$$(\mathbb{E}[Y])^2 = (\mathbb{E}[\mathbb{E}[Y|X]])^2$$

So: $$\text{Var}(Y) = \mathbb{E}[\text{Var}(Y|X)] + \mathbb{E}[(\mathbb{E}[Y|X])^2] - (\mathbb{E}[\mathbb{E}[Y|X]])^2$$ $$= \mathbb{E}[\text{Var}(Y|X)] + \text{Var}(\mathbb{E}[Y|X]) \qquad \square$$

> Eve's law decomposes variance into two parts: the average residual variance after conditioning (unexplained noise), and the variance of the conditional mean (signal). This shows up constantly — in ANOVA, mixed models, random effects, and any time you're partitioning variability.

## Covariance and Correlation

**Definition.** $$\text{Cov}(X,Y) = \mathbb{E}[(X - \mathbb{E}X)(Y - \mathbb{E}Y)] = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y]$$

$$\rho(X,Y) = \frac{\text{Cov}(X,Y)}{\sqrt{\text{Var}(X)\text{Var}(Y)}} \in [-1, 1]$$

**Properties:**

- $\text{Cov}(X,X) = \text{Var}(X)$
- $\text{Cov}(aX + b, cY + d) = ac,\text{Cov}(X,Y)$
- $\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X,Y)$
- More generally: $\text{Var}!\left(\sum_i X_i\right) = \sum_i \text{Var}(X_i) + 2\sum_{i < j} \text{Cov}(X_i, X_j)$

**The covariance matrix.** For a random vector $\mathbf{X} = (X_1, \ldots, X_n)^T$:

$$\Sigma = \text{Cov}(\mathbf{X}) = \mathbb{E}[(\mathbf{X} - \mathbb{E}\mathbf{X})(\mathbf{X} - \mathbb{E}\mathbf{X})^T]$$

with $\Sigma_{ij} = \text{Cov}(X_i, X_j)$. This matrix is always **positive semi-definite**: for any $\mathbf{v} \in \mathbb{R}^n$:

$$\mathbf{v}^T \Sigma \mathbf{v} = \text{Var}(\mathbf{v}^T \mathbf{X}) \geq 0$$

## The Bivariate Normal

The most important joint continuous distribution. $(X, Y)$ is **bivariate normal** with parameters $\mu_X, \mu_Y, \sigma_X^2, \sigma_Y^2, \rho$ if:

$$f_{X,Y}(x,y) = \frac{1}{2\pi\sigma_X\sigma_Y\sqrt{1-\rho^2}} \exp!\left(-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_X)^2}{\sigma_X^2} - \frac{2\rho(x-\mu_X)(y-\mu_Y)}{\sigma_X\sigma_Y} + \frac{(y-\mu_Y)^2}{\sigma_Y^2}\right]\right)$$

Key properties:

1. **Marginals are normal:** $X \sim N(\mu_X, \sigma_X^2)$, $Y \sim N(\mu_Y, \sigma_Y^2)$.
    
2. **Conditionals are normal:** $$Y \mid X = x \sim N!\left(\mu_Y + \rho\frac{\sigma_Y}{\sigma_X}(x - \mu_X),; \sigma_Y^2(1-\rho^2)\right)$$
    
3. **For bivariate normal, uncorrelated $\iff$ independent.** This is the one case where the converse of $\text{Cov} = 0 \implies$ uncorrelated holds.
    
4. **Linear combinations are normal:** $aX + bY \sim N(a\mu_X + b\mu_Y,; a^2\sigma_X^2 + b^2\sigma_Y^2 + 2ab\rho\sigma_X\sigma_Y)$.
    

> The conditional mean formula $\mathbb{E}[Y \mid X = x] = \mu_Y + \rho\frac{\sigma_Y}{\sigma_X}(x-\mu_X)$ is exactly the **linear regression** of $Y$ on $X$. The bivariate normal is the distribution where the regression function is linear — and the conditional variance $\sigma_Y^2(1-\rho^2)$ is constant (homoskedastic). That's not a coincidence; it's a theorem.

## Computing Joint Distributions: Worked Examples

### Example 1: Discrete joint to conditional

$X \sim \text{Uniform}{1,2,3}$, and $Y \mid X = x \sim \text{Uniform}{1, \ldots, x}$. Find $P(Y = 1)$.

By Adam's law: $$P(Y=1) = \mathbb{E}[P(Y=1 \mid X)] = \sum_{x=1}^3 P(Y=1 \mid X=x) P(X=x) = \frac{1}{3}\left(1 + \frac{1}{2} + \frac{1}{3}\right) = \frac{11}{18}$$

### Example 2: Continuous — finding the marginal

$f_{X,Y}(x,y) = e^{-y}$ for $0 < x < y < \infty$. Find $f_X(x)$.

$$f_X(x) = \int_x^\infty e^{-y}, dy = e^{-x}, \quad x > 0$$

So $X \sim \text{Expo}(1)$.

### Example 3: Independence check via factorisation

$f_{X,Y}(x,y) = 6e^{-2x-3y}$ for $x, y > 0$. Does $X \perp Y$?

$f_{X,Y}(x,y) = (2e^{-2x})(3e^{-3y})$, which factors as $g(x)h(y)$ on the product support $(0,\infty)^2$. So yes, $X \perp Y$, with $X \sim \text{Expo}(2)$ and $Y \sim \text{Expo}(3)$.

## Pitfalls

**1. Marginals don't determine the joint.**  
Already emphasised, but worth repeating: specifying $f_X$ and $f_Y$ separately does not specify $f_{X,Y}$. You need either the joint directly, the factorisation (if independent), or one marginal plus the conditional.

**2. The support matters for independence.**  
Even if $f_{X,Y}(x,y) = g(x)h(y)$ algebraically, if the support is not a product set (i.e. the region where $f > 0$ depends on both $x$ and $y$), then $X$ and $Y$ are **not** independent.

**Example:** $f_{X,Y}(x,y) \propto 1$ on the triangle ${0 < x < y < 1}$. The support is not $[0,1]^2$; it depends on both variables jointly. $X$ and $Y$ are dependent even though the density is "constant."

**3. $\mathbb{E}[Y \mid X]$ is a random variable, not a number.**  
$\mathbb{E}[Y \mid X = x]$ is a number for fixed $x$. $\mathbb{E}[Y \mid X]$ is a function of $X$, hence random. Adam's law says $\mathbb{E}[\mathbb{E}[Y \mid X]] = \mathbb{E}[Y]$ — the outer expectation is over $X$.

## Summary

|Object|Discrete|Continuous|
|---|---|---|
|Joint|$p_{X,Y}(x,y)$|$f_{X,Y}(x,y)$|
|Marginal of $X$|$\sum_y p_{X,Y}(x,y)$|$\int f_{X,Y}(x,y),dy$|
|Conditional $Y \mid X=x$|$p_{X,Y}(x,y)/p_X(x)$|$f_{X,Y}(x,y)/f_X(x)$|
|Independence|$p_{X,Y} = p_X p_Y$|$f_{X,Y} = f_X f_Y$|
|Adam's law|$\mathbb{E}[Y] = \mathbb{E}[\mathbb{E}[Y \mid X]]$|same|
|Eve's law|$\text{Var}(Y) = \mathbb{E}[\text{Var}(Y \mid X)] + \text{Var}(\mathbb{E}[Y \mid X])$|same|
