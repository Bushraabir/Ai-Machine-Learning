
> _Fubini is the theorem that lets you stop worrying about order. The moment you swap $dx,dy$ to $dy,dx$ without thinking, you're invoking it. The point of this note is to know exactly when you're allowed to, and what goes wrong when you're not._

## The Core Question

Given a double integral

$$\iint_{\mathbb{R}^2} f(x,y), dx, dy,$$

when can I evaluate it as an iterated integral, and when does the order of integration not matter?

$$\int_{-\infty}^{\infty}!\int_{-\infty}^{\infty} f(x,y), dy, dx \stackrel{?}{=} \int_{-\infty}^{\infty}!\int_{-\infty}^{\infty} f(x,y), dx, dy.$$

The answer is: **always, if $f \geq 0$** (Tonelli); and **if $f$ is integrable** (Fubini). These are different theorems with different hypotheses, and conflating them is the source of every "illegal swap" in probability.

---

## Tonelli's Theorem

**Theorem (Tonelli).** Let $f : \mathbb{R}^2 \to [0, \infty]$ be measurable. Then:

$$\int!!\int f(x,y), d(x,y) = \int!\left(\int f(x,y), dy\right) dx = \int!\left(\int f(x,y), dx\right) dy.$$

All three expressions are equal (possibly $+\infty$), and the inner integrals are measurable functions of the outer variable.

**Hypotheses:** $f \geq 0$ and measurable. That's it. No finiteness required.

**What it gives:** You can freely interchange the order of integration for nonneg functions. The common value might be $+\infty$, which is allowed.

**Why this is the right theorem for PDFs.** Joint PDFs are nonneg by definition. So every marginalisation step — every $\int_{-\infty}^{\infty} f_{X,Y}(x,y), dy$ — is justified by Tonelli, not Fubini. This is why the [[06.Marginal PDF]] proof works without any integrability assumption beyond what the PDF already satisfies.

---

## Fubini's Theorem

**Theorem (Fubini).** Let $f : \mathbb{R}^2 \to \mathbb{R}$ be measurable. Suppose

$$\int!!\int |f(x,y)|, d(x,y) < \infty,$$

i.e., $f$ is integrable (equivalently, $\int!\int f^+, d(x,y) < \infty$ and $\int!\int f^-, d(x,y) < \infty$ separately). Then:

1. For a.e. $x$, the function $y \mapsto f(x,y)$ is integrable.
2. The function $x \mapsto \int f(x,y), dy$ is integrable.
3. The iterated integrals equal the double integral:

$$\int!!\int f(x,y), d(x,y) = \int!\left(\int f(x,y), dy\right) dx = \int!\left(\int f(x,y), dx\right) dy.$$

**Hypotheses:** $f$ integrable — meaning $\int!\int |f| < \infty$.

**What it gives:** Order of integration is interchangeable for integrable functions (which may change sign).

**The key difference from Tonelli:** Fubini requires absolute integrability. If $f$ takes both positive and negative values and $\int!\int |f| = \infty$, the iterated integrals may exist but give different values — the double integral doesn't exist in the Lebesgue sense.

---

## The Unified Statement

In practice, the two theorems are often stated together as:

> **Fubini-Tonelli.** If either (a) $f \geq 0$, or (b) $\int!\int |f| < \infty$, then all iterated integrals and the double integral coincide.

When in doubt about which applies: if $f \geq 0$, cite Tonelli. If $f$ changes sign, first use Tonelli on $|f|$ to check $\int!\int|f| < \infty$, then apply Fubini.

---

## The Counterexample: Why Fubini Can Fail

This is the example that justifies the hypotheses. Let

$$f(x,y) = \frac{x^2 - y^2}{(x^2+y^2)^2}, \quad (x,y) \neq (0,0).$$

**Iterated integral in one order:**

$$\int_0^1!\left(\int_0^1 \frac{x^2-y^2}{(x^2+y^2)^2}, dy\right) dx.$$

The inner integral $\int_0^1 \frac{x^2-y^2}{(x^2+y^2)^2}, dy = \left[\frac{y}{x^2+y^2}\right]_0^1 = \frac{1}{x^2+1}$.

So the outer integral is $\int_0^1 \frac{1}{x^2+1}, dx = \arctan(1) - \arctan(0) = \frac{\pi}{4}$.

**Iterated integral in the other order:**

By antisymmetry ($f(x,y) = -f(y,x)$), swapping gives $-\pi/4$.

**Conclusion:** $\frac{\pi}{4} \neq -\frac{\pi}{4}$. The two iterated integrals disagree.

**Why did this happen?** Because $\int_0^1!\int_0^1 |f(x,y)|, dx, dy = \infty$ — $f$ is not integrable on $[0,1]^2$. Fubini's hypothesis fails, and the conclusion fails too.

**Moral:** Never swap integration order for a sign-changing function without first verifying absolute integrability.

## How This Appears in Probability

### Marginalisation

$$f_X(x) = \int_{-\infty}^{\infty} f_{X,Y}(x,y), dy.$$

Tonelli applies (PDFs are nonneg). The interchange in the normalisation proof:

$$\int_{-\infty}^{\infty} f_X(x), dx = \int_{-\infty}^{\infty}\int_{-\infty}^{\infty} f_{X,Y}(x,y), dy, dx \stackrel{\text{Tonelli}}{=} \int_{-\infty}^{\infty}\int_{-\infty}^{\infty} f_{X,Y}(x,y), dx, dy = 1.$$

### Expected Value of a Function

$$E[g(X,Y)] = \int!!\int g(x,y), f_{X,Y}(x,y), dx, dy.$$

If $g \geq 0$: Tonelli. If $g$ changes sign: need $E[|g(X,Y)|] < \infty$ before swapping.

This is why $E[X]$ exists requires $E[|X|] < \infty$ — the same logic one dimension down.

### Independence and Product Expectations

If $X \perp Y$ and $g, h$ are measurable:

$$E[g(X)h(Y)] = \int!!\int g(x)h(y), f_X(x), f_Y(y), dx, dy = \int g(x), f_X(x), dx \int h(y), f_Y(y), dy = E[g(X)], E[h(Y)].$$

The factorisation of the double integral into a product requires swapping, which requires Fubini (if $g \cdot h$ changes sign). If $g, h \geq 0$, Tonelli suffices.

### Convolutions

If $X \perp Y$ and $Z = X + Y$:

$$f_Z(z) = \int_{-\infty}^{\infty} f_X(z-y), f_Y(y), dy.$$

This is derived by marginalising the joint of $(Z,Y)$, which involves swapping the order of integration. Tonelli applies (all factors are nonneg PDFs).

### Moment Generating Functions

$$M_X(t) = E[e^{tX}] = \int_{-\infty}^{\infty} e^{tx} f_X(x), dx.$$

When exchanging expectation and differentiation: $\frac{d}{dt} M_X(t) = E[X e^{tX}]$. Justifying the swap of $\frac{d}{dt}$ and $\int$ uses the **dominated convergence theorem** or Fubini (depending on how you set it up). For $t$ in the interior of the MGF's domain, this is fine.

## The Discrete Analogue

For double sums over nonneg terms (i.e., PMFs):

$$\sum_x \sum_y p_{X,Y}(x,y) = \sum_y \sum_x p_{X,Y}(x,y).$$

This is **Tonelli for sums** — also called the "nonneg interchange" or just "obvious by absolute convergence." The same counterexample logic applies: for sums of sign-changing terms, you need absolute convergence ($\sum_x \sum_y |a_{xy}| < \infty$) to interchange freely. The discrete analogue of the Fubini counterexample is a conditionally convergent double series whose iterated sums disagree.

---

## Practical Decision Tree

When I want to swap $\int dx, dy$ to $\int dy, dx$:

```
Is the integrand f ≥ 0 everywhere?
    YES → Tonelli applies. Swap freely. (Citation: Tonelli)
    NO  → Compute ∬|f| dx dy (use Tonelli on |f|, which is nonneg)
              Is ∬|f| < ∞?
                  YES → Fubini applies. Swap freely. (Citation: Fubini)
                  NO  → DO NOT SWAP. The iterated integrals may disagree.
```

In probability: almost everything uses Tonelli (PDFs are nonneg). Fubini enters when computing $E[g(X,Y)]$ for $g$ that may be negative — e.g. covariance computations, characteristic functions (which are complex-valued), or when proving the tower property of conditional expectation rigorously.

---

## A Subtlety: "Almost Everywhere"

Fubini's conclusion says the inner integral exists "for a.e. $x$" — not necessarily for every $x$. The set of $x$ where the inner integral is undefined has Lebesgue measure zero and doesn't affect the outer integral. In practice this is never an issue for the integrands that arise in probability, but it's worth knowing the theorem says a.e., not everywhere.

## Statement in Measure-Theoretic Language (for Later)

The full theorem lives in the language of product measures. If $(X, \mathcal{A}, \mu)$ and $(Y, \mathcal{B}, \nu)$ are $\sigma$-finite measure spaces, then there exists a unique product measure $\mu \otimes \nu$ on $(X \times Y, \mathcal{A} \otimes \mathcal{B})$ such that $(\mu \otimes \nu)(A \times B) = \mu(A)\nu(B)$.

**Tonelli:** For $f \geq 0$ measurable,

$$\int_{X \times Y} f, d(\mu \otimes \nu) = \int_X \int_Y f(x,y), d\nu(y), d\mu(x) = \int_Y \int_X f(x,y), d\mu(x), d\nu(y).$$

**Fubini:** If additionally $\int_{X \times Y} |f|, d(\mu \otimes \nu) < \infty$, the same equalities hold for $f$ (not just $|f|$).

The $\sigma$-finiteness condition on the measure spaces is necessary — the theorem can fail for non-$\sigma$-finite spaces. For probability, $\mu = \nu = $ Lebesgue measure on $\mathbb{R}$, which is $\sigma$-finite. Never a concern in this course.
