

> _The multinomial is the natural generalisation of the binomial — instead of two outcomes per trial, you have $k$ outcomes. Almost everything you know about the binomial lifts here, but the covariance structure is new and worth sitting with. The negative covariance between categories is not a coincidence — it's the same FPC story from hypergeometric, but now it's the constraint that all counts sum to $n$._

---

## The Setup

$n$ independent trials. Each trial results in exactly one of $k$ categories, with probabilities: $$p_1, p_2, \ldots, p_k, \qquad p_i > 0, \quad \sum_{i=1}^k p_i = 1$$

Let $X_i$ = number of trials falling in category $i$. Then $(X_1, X_2, \ldots, X_k)$ follows the **Multinomial$(n, p_1, \ldots, p_k)$** distribution.

Constraints baked in from the start: $$X_1 + X_2 + \cdots + X_k = n \qquad \text{(always, with probability 1)}$$ $$X_i \in {0, 1, \ldots, n} \quad \text{for each } i$$

The binomial is the special case $k = 2$.

---

## The Joint PMF

$$\mathbb{P}(X_1 = n_1, X_2 = n_2, \ldots, X_k = n_k) = \frac{n!}{n_1!, n_2!, \cdots, n_k!}, p_1^{n_1} p_2^{n_2} \cdots p_k^{n_k}$$

for non-negative integers $n_1, \ldots, n_k$ summing to $n$.

**Where this comes from.** The multinomial coefficient $\frac{n!}{n_1! \cdots n_k!}$ counts the number of ways to arrange $n$ trials so that exactly $n_i$ land in category $i$ — it's the number of distinct orderings of a sequence with $n_i$ copies of symbol $i$. Each such ordering has probability $p_1^{n_1} \cdots p_k^{n_k}$ (by independence). Multiply and you're done.

**Sanity check:** Sum over all valid $(n_1, \ldots, n_k)$: $$\sum_{\substack{n_1+\cdots+n_k=n \ n_i \geq 0}} \frac{n!}{n_1!\cdots n_k!} p_1^{n_1}\cdots p_k^{n_k} = (p_1 + \cdots + p_k)^n = 1^n = 1$$

by the **multinomial theorem**. The PMF is indeed a valid probability distribution.

---

## Marginals: Each $X_i$ is Binomial

**Theorem.** $X_i \sim \text{Binomial}(n, p_i)$.

_Proof._ Lump all categories except $i$ into one "failure" category with probability $1 - p_i$. Each trial is now a Bernoulli$(p_i)$. With $n$ independent trials, $X_i \sim \text{Bin}(n, p_i)$. $\square$

Formally: marginalise the joint PMF by summing over all valid $(n_1, \ldots, n_{i-1}, n_{i+1}, \ldots, n_k)$ given $n_i = m$. What remains is $\binom{n}{m} p_i^m (1-p_i)^{n-m}$. But the lumping argument is cleaner.

This immediately gives: $$\mathbb{E}[X_i] = np_i, \qquad \text{Var}(X_i) = np_i(1-p_i)$$

---

## Covariance Between Categories

For $i \neq j$: $$\text{Cov}(X_i, X_j) = -np_ip_j$$

_Proof._ Write each $X_i = \sum_{\ell=1}^n \mathbf{1}[\text{trial } \ell \in \text{category } i]$. Call these indicator r.v.'s $I_{i\ell}$.

$$\text{Cov}(X_i, X_j) = \text{Cov}!\left(\sum_\ell I_{i\ell},, \sum_\ell I_{j\ell}\right) = \sum_{\ell,m} \text{Cov}(I_{i\ell}, I_{jm})$$

**Cross-trial terms** ($\ell \neq m$): Trials are independent, so $I_{i\ell} \perp I_{jm}$, giving $\text{Cov}(I_{i\ell}, I_{jm}) = 0$.

**Same-trial terms** ($\ell = m$): $I_{i\ell}$ and $I_{j\ell}$ are indicators for the _same trial_ landing in categories $i$ and $j$ respectively. Since a trial can't be in both categories simultaneously: $$\mathbb{P}(I_{i\ell} = 1,, I_{j\ell} = 1) = 0$$

So $\mathbb{E}[I_{i\ell} I_{j\ell}] = 0$, and: $$\text{Cov}(I_{i\ell}, I_{j\ell}) = \mathbb{E}[I_{i\ell}I_{j\ell}] - \mathbb{E}[I_{i\ell}]\mathbb{E}[I_{j\ell}] = 0 - p_ip_j = -p_ip_j$$

There are $n$ same-trial terms, so: $$\text{Cov}(X_i, X_j) = n \cdot (-p_ip_j) = -np_ip_j \qquad \square$$

> _The covariance is negative for every pair $i \neq j$. The intuition is exactly the same as in the hypergeometric: the counts are competing. If more trials land in category $i$, there are fewer left to land in category $j$. The constraint $\sum X_i = n$ forces this negative dependence. There is no way to have positive covariance between two categories in a multinomial._

---

## The Covariance Matrix

The full covariance matrix $\Sigma$ of $(X_1, \ldots, X_k)$ has entries:

$$\Sigma_{ij} = \begin{cases} np_i(1-p_i) & i = j \ -np_ip_j & i \neq j \end{cases}$$

In matrix form, letting $\mathbf{p} = (p_1, \ldots, p_k)^\top$: $$\Sigma = n\left(\text{diag}(\mathbf{p}) - \mathbf{p}\mathbf{p}^\top\right)$$

where $\text{diag}(\mathbf{p})$ is the diagonal matrix with $p_i$ on the diagonal.

**This matrix is positive semi-definite but singular.** Singular because $\mathbf{1}^\top \mathbf{X} = n$ always — there's a linear constraint baked into the distribution. Any vector proportional to $\mathbf{1} = (1,1,\ldots,1)^\top$ is in the null space of $\Sigma$: $$\Sigma \mathbf{1} = n(\mathbf{p} - \mathbf{p}\mathbf{p}^\top\mathbf{1}) = n(\mathbf{p} - \mathbf{p}) = \mathbf{0}$$

So $\Sigma$ has rank at most $k-1$. The multinomial distribution is essentially $(k-1)$-dimensional — once you know $n_1, \ldots, n_{k-1}$, the last count $n_k = n - \sum_{i<k} n_i$ is determined.

---

## The Constraint Forces Negative Covariance: A Formal Argument

Here's the clean argument that ties everything together. Since $\sum_{j=1}^k X_j = n$ (constant), $\text{Var}(\sum_j X_j) = 0$. Expand:

$$0 = \text{Var}!\left(\sum_{j=1}^k X_j\right) = \sum_{j=1}^k \text{Var}(X_j) + 2\sum_{i < j} \text{Cov}(X_i, X_j)$$

$$= n\sum_{j=1}^k p_j(1-p_j) + 2\sum_{i<j}\text{Cov}(X_i, X_j)$$

We can verify our covariance formula by checking that it satisfies this:

$$2\sum_{i<j}(-np_ip_j) = -n\sum_{i \neq j} p_ip_j = -n\left[\left(\sum_i p_i\right)^2 - \sum_i p_i^2\right] = -n(1 - \sum_i p_i^2)$$

And:

$$n\sum_j p_j(1-p_j) = n\sum_j p_j - n\sum_j p_j^2 = n - n\sum_j p_j^2$$

Sum:

$$(n - n\sum p_j^2) + (-n + n\sum p_j^2) = 0 \checkmark$$

The constraint $\sum X_i = n$ is _exactly_ what makes the positive diagonal terms cancel with the negative off-diagonal covariances. The distribution is telling you: these $k$ random variables are not free — they live in a $(k-1)$-dimensional simplex.

---

## Lumping Property

If you merge categories $i$ and $j$ into one super-category with probability $p_i + p_j$, the result is again multinomial. Formally:

$$(X_1, \ldots, X_{i-1}, X_i + X_j, X_{i+1}, \ldots, \hat{X}_j, \ldots, X_k) \sim \text{Multinomial}(n;, \ldots, p_i+p_j, \ldots)$$

This follows immediately from the definition — independence of trials plus the fact that probabilities sum correctly. It's why marginalising to $X_i$ gives $\text{Bin}(n, p_i)$: you've lumped all other categories into one.

---

## Conditional Distribution

Given $X_1 = m$, the remaining counts $(X_2, \ldots, X_k)$ have conditional distribution:

$$(X_2, \ldots, X_k) \mid X_1 = m ;\sim; \text{Multinomial}!\left(n-m;; \frac{p_2}{1-p_1}, \ldots, \frac{p_k}{1-p_1}\right)$$

_Proof sketch._ Given that $m$ trials landed in category 1, the remaining $n - m$ trials are independent with each trial landing in category $j \geq 2$ with conditional probability $p_j/(1-p_1)$ (by Bayes and the law of total probability). These renormalised probabilities sum to 1. $\square$

This is the multinomial analogue of the memoryless / conditioning argument for the binomial.

## Connection to Binomial: The $k=2$ Case

With $k=2$, $p_1 = p$, $p_2 = 1-p = q$, $X_1 = X$, $X_2 = n - X$:

|Multinomial formula|Binomial formula|
|---|---|
|$\frac{n!}{n_1! n_2!} p^{n_1} q^{n_2}$|$\binom{n}{k} p^k q^{n-k}$|
|$\mathbb{E}[X_1] = np$|$\mathbb{E}[X] = np$|
|$\text{Var}(X_1) = npq$|$\text{Var}(X) = npq$|
|$\text{Cov}(X_1,X_2) = -np \cdot q$|$\text{Cov}(X, n-X) = -\text{Var}(X) = -npq$|

The covariance check: $\text{Cov}(X, n-X) = \text{Cov}(X,-X) = -\text{Var}(X) = -npq$. And indeed $-np_1p_2 = -npq$. Consistent.

## Summary of Key Facts

|Quantity|Formula|
|---|---|
|PMF|$\frac{n!}{n_1!\cdots n_k!}p_1^{n_1}\cdots p_k^{n_k}$|
|Marginal|$X_i \sim \text{Bin}(n, p_i)$|
|$\mathbb{E}[X_i]$|$np_i$|
|$\text{Var}(X_i)$|$np_i(1-p_i)$|
|$\text{Cov}(X_i, X_j)$, $i\neq j$|$-np_ip_j$|
|Covariance matrix|$n(\text{diag}(\mathbf{p}) - \mathbf{p}\mathbf{p}^\top)$|
|Rank of $\Sigma$|$k - 1$|
|Sum constraint|$\sum X_i = n$ always|

_The thing I keep coming back to: the multinomial covariance matrix has a beautifully simple structure — $n(\text{diag}(\mathbf{p}) - \mathbf{p}\mathbf{p}^\top)$. The outer product $\mathbf{p}\mathbf{p}^\top$ is what kills the rank by one. Every pair of categories is negatively correlated, and the magnitude is determined entirely by $p_i p_j$. Nothing else. That's elegant._
