## [[05.Binomial Distribution]]


$$(x + y)^n = \sum_{k=0}^{n} \binom{n}{k} x^k y^{n-k}$$

The intuition: when you multiply out $n$ copies of $(x + y)$, each term comes from choosing $k$ of the $n$ brackets to contribute an $x$ (and the rest contribute a $y$). That choice — picking $k$ things from $n$ without caring about order — is exactly $\binom{n}{k}$.

Two sanity checks worth knowing:

- Set $x = y = 1$: you get $\sum_{k=0}^{n} \binom{n}{k} = 2^n$, which makes sense — it's the total number of subsets of an $n$-element set.
- Set $x = 1, y = -1$: you get $\sum_{k=0}^{n} (-1)^k \binom{n}{k} = 0$, a nice alternating identity.

## The Sampling Table


|                         | **Ordered**           | **Unordered**      |
| ----------------------- | --------------------- | ------------------ |
| **With replacement**    | $n^k$                 | $\binom{n+k-1}{k}$ |
| **Without replacement** | $n(n-1)\cdots(n-k+1)$ | $\binom{n}{k}$     |


The unordered-with-replacement case ($\binom{n+k-1}{k}$) is the one that always trips people up. See [[04.Bose-Einstein's Non Intuitive Problem]].
