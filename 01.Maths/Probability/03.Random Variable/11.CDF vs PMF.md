## The Setup

A **random variable** $X$ is a function $X : \Omega \to \mathbb{R}$ from the sample space to the reals. Everything about the probabilistic behavior of $X$ is captured by its **distribution** — the assignment of probabilities to subsets of $\mathbb{R}$.

There are two standard ways to represent a distribution for a discrete random variable: the **PMF** and the **CDF**. They contain identical information. But they have very different properties, very different uses, and one of them generalizes to continuous random variables and the other doesn't.

---

## The PMF — Probability Mass Function

For a discrete random variable $X$, the **PMF** is:

$$p_X(x) = P(X = x)$$

**Requirements:**

1. $p_X(x) \geq 0$ for all $x$
2. $\sum_{x} p_X(x) = 1$ (sum over all $x$ in the support)
3. $p_X(x) = 0$ for $x$ not in the support

The PMF gives you the probability of landing exactly on a point. It is a function on the support of $X$ — the set of values $X$ can actually take.

### Example — $X \sim \text{Bin}(3, 1/2)$

$$p_X(k) = \binom{3}{k} \left(\frac{1}{2}\right)^3, \quad k = 0, 1, 2, 3$$

| $k$ | $p_X(k)$ |
| --- | -------- |
| 0   | $1/8$    |
| 1   | $3/8$    |
| 2   | $3/8$    |
| 3   | $1/8$    |

Clean, immediate, intuitive. This is what you write down first when you think about a discrete distribution.

## The CDF — Cumulative Distribution Function

For **any** random variable $X$ (discrete, continuous, or mixed), the **CDF** is:

$$F_X(x) = P(X \leq x), \quad x \in \mathbb{R}$$

This is defined for every real number $x$, not just the support. It asks: _what is the probability that $X$ is no greater than $x$?_

### Example — Same $X \sim \text{Bin}(3, 1/2)$

$$F_X(x) = \begin{cases} 0 & x < 0 \ 1/8 & 0 \leq x < 1 \ 4/8 & 1 \leq x < 2 \ 7/8 & 2 \leq x < 3 \ 1 & x \geq 3 \end{cases}$$

It's a **staircase function** — constant between the support points, jumping at each mass point by exactly $p_X(k)$ at $x = k$.

---

## Properties of the CDF — These Are Theorems, Not Definitions

Every valid CDF satisfies, and is completely characterized by, these four properties:

**1. Non-decreasing:** $$x_1 \leq x_2 \implies F(x_1) \leq F(x_2)$$

_Proof:_ ${X \leq x_1} \subseteq {X \leq x_2}$, so monotonicity of probability gives $F(x_1) \leq F(x_2)$. $\square$

**2. Right-continuous:** $$\lim_{x \downarrow a} F(x) = F(a) \quad \text{for all } a$$

The CDF is continuous from the right. At a jump (a mass point), the value at the jump is the higher value — i.e., $F(a) = P(X \leq a)$ includes the mass at $a$.

_Why right and not left?_ Convention, but a meaningful one: $P(X \leq a)$ includes the endpoint. $P(X < a) = \lim_{x \uparrow a} F(x) = F(a^-)$, the left-hand limit.

**3. Limits at $\pm\infty$:** $$\lim_{x \to -\infty} F(x) = 0, \qquad \lim_{x \to +\infty} F(x) = 1$$

**4. Jumps encode the PMF:** At a mass point $a$: $$P(X = a) = F(a) - F(a^-) = F(a) - \lim_{x \uparrow a} F(x)$$

The size of the jump at $a$ is exactly $p_X(a)$. If there's no jump at $a$, then $P(X = a) = 0$.

> **Theorem (converse):** Any function $F : \mathbb{R} \to [0,1]$ satisfying properties 1–3 is the CDF of some random variable. This is the sense in which CDFs are the fundamental object — you can characterize all valid distributions just by characterizing valid CDFs.

---

## Recovering One from the Other

### PMF from CDF (discrete):

$$p_X(a) = F(a) - F(a^-)$$

Just read off the jump sizes.

### CDF from PMF (discrete):

$$F(x) = \sum_{a \leq x} p_X(a)$$

Sum up all the mass at points $\leq x$.

These are inverses of each other. They carry identical information.

---

## Why the CDF is More Fundamental

The PMF only makes sense for discrete random variables. A continuous random variable (like $X \sim \text{Uniform}(0,1)$) has $P(X = x) = 0$ for every individual $x$ — the PMF is identically zero, completely useless.

The CDF works for everything:

| Type       | PMF                          | CDF                               |
| ---------- | ---------------------------- | --------------------------------- |
| Discrete   | ✓ Works perfectly            | ✓ Staircase                       |
| Continuous | ✗ Identically zero           | ✓ Smooth, differentiable          |
| Mixed      | ✗ Misses the continuous part | ✓ Has both jumps and smooth parts |

For continuous random variables, the analogue of the PMF is the **PDF** (probability density function) $f_X(x)$, defined by:

$$F_X(x) = \int_{-\infty}^x f_X(t), dt$$

and recovered from the CDF by:

$$f_X(x) = F_X'(x)$$

wherever $F$ is differentiable. The CDF is the unifying object. PMF and PDF are its discrete and continuous derivatives, respectively.

> **Key point to internalize:** PMF is to discrete as PDF is to continuous. CDF is the one object that works always. When in doubt, think in terms of the CDF.

---

## Computing Probabilities from the CDF

Once you have the CDF, all interval probabilities come for free:

$$P(a < X \leq b) = F(b) - F(a)$$

$$P(a \leq X \leq b) = F(b) - F(a^-)$$

$$P(a < X < b) = F(b^-) - F(a)$$

$$P(X > a) = 1 - F(a)$$

$$P(X \geq a) = 1 - F(a^-)$$

> Notice: for continuous RVs, $F(a^-) = F(a)$ (no jumps), so $P(X < a) = P(X \leq a)$ — open and closed endpoints don't matter. For discrete RVs, they do. The CDF makes this completely transparent.

## The Staircase vs the Spike — Geometric Intuition

For a discrete RV:

- **PMF:** spikes at the support points, height = probability
- **CDF:** staircase, rises by the spike height at each support point, flat in between

The CDF is the running total of the PMF. At any point $x$, $F(x)$ is the total probability mass to the left of (and including) $x$.

Here's the relationship written plainly:

$$F(x) = \sum_{k \leq x} p(k) \qquad \longleftrightarrow \qquad p(k) = F(k) - F(k-1)$$

for integer-valued RVs. This is literally the discrete analogue of:

$$F(x) = \int_{-\infty}^x f(t), dt \qquad \longleftrightarrow \qquad f(x) = F'(x)$$

for continuous RVs. The CDF is the "integral" (sum) of the PMF/PDF. The PMF/PDF is the "derivative" (difference) of the CDF.

## When to Use Which

**Use the PMF when:**

- You want to compute $P(X = k)$ directly
- You're computing $\mathbb{E}[X]$ or $\text{Var}(X)$ — these are sums weighted by $p(k)$
- You're working with a named discrete distribution (Binomial, Geometric, Poisson...)
- You're reasoning about the shape of the distribution

**Use the CDF when:**

- You need $P(X \leq x)$, $P(X > x)$, or $P(a < X \leq b)$ for general intervals
- You're working with order statistics ($\min$, $\max$ of a sample — their distributions are natural in terms of CDFs)
- You want to find quantiles: the $p$-th quantile is $F^{-1}(p) = \inf{x : F(x) \geq p}$
- You're working with continuous RVs (where you have no choice)
- You want to check if a function is a valid distribution (verify the four CDF properties)

## Order Statistics — Where CDFs Shine

Let $X_1, \ldots, X_n \sim F$ i.i.d. Let $X_{(k)}$ denote the $k$-th smallest (the $k$-th order statistic).

The CDF of the maximum $X_{(n)} = \max(X_1, \ldots, X_n)$:

$$P(X_{(n)} \leq x) = P(X_1 \leq x, \ldots, X_n \leq x) = \prod_{i=1}^n P(X_i \leq x) = F(x)^n$$

Clean. One line. Try doing this with the PMF.

The CDF of the minimum $X_{(1)} = \min(X_1, \ldots, X_n)$:

$$P(X_{(1)} > x) = P(X_1 > x, \ldots, X_n > x) = (1 - F(x))^n$$

$$F_{X_{(1)}}(x) = 1 - (1-F(x))^n$$

These derivations are trivial in CDF language and miserable in any other language. This is one of the most convincing demonstrations that the CDF is the right object for many problems.

---

## A Note on Notation

Be careful: some books write $F(x) = P(X < x)$ (strict inequality, left-continuous). This is an equally valid convention, but it flips the right-continuity property to left-continuity and changes which endpoint is included. Blitzstein uses $F(x) = P(X \leq x)$, which is standard in most modern probability texts and all of Stat 110. Always check which convention a source is using before mixing formulas.

---

_The thing worth remembering: the PMF is what you draw when you want to see a distribution. The CDF is what you use when you want to prove something about it. Every named distribution has both, they carry the same information, but the CDF is the one that works for all random variables — discrete, continuous, and everything in between. Learn to switch between them fluently and a lot of probability becomes much cleaner._
