A **probability distribution** is a function that tells me how probability mass (or density) is spread across the possible values of a random variable.

There are really only two questions that organize everything:
1. **Is the sample space countable or uncountable?** ‚Üí Discrete vs. Continuous
2. **What story generates the randomness?** ‚Üí Which named distribution

# PART I ‚Äî Discrete Distributions

_The probability lives at isolated points. We describe it with a **PMF**: $P(X = k)$._

## 1. Bernoulli$(p)$
**The atom. Everything discrete is built from this.**
**Story:** A single trial. Success with probability $p$, failure with $1-p$.
$$P(X = k) = p^k (1-p)^{1-k}, \qquad k \in {0, 1}.$$

| $E[X]$ | $\text{Var}(X)$ | MGF            |
| ------ | --------------- | -------------- |
| $p$    | $p(1-p)$        | $1 - p + pe^t$ |

> _Every discrete distribution is, at its heart, a way of combining Bernoullis. Keep this in mind._

see more [[06.Bernoulli Distribution]]

---

## 2. Binomial$(n, p)$

**The most important discrete distribution.**

**Story:** $n$ independent Bernoulli$(p)$ trials. $X$ = number of successes.

$$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}, \qquad k = 0, 1, \ldots, n.$$

| $E[X]$ | $\text{Var}(X)$ | MGF              |
| ------ | --------------- | ---------------- |
| $np$   | $np(1-p)$       | $(pe^t + 1-p)^n$ |

**Key insight:** $X = X_1 + \cdots + X_n$ where $X_i \overset{\text{i.i.d.}}{\sim} \text{Bern}(p)$. The mean and variance follow immediately from linearity and additivity of variance (under independence).

**Reproductive property:** $\text{Bin}(n_1, p) + \text{Bin}(n_2, p) = \text{Bin}(n_1 + n_2, p)$ for independent summands.

‚Üí See [[05.Binomial Distribution]] 

---

## 3. Geometric$(p)$

**The waiting distribution for the first success.**

**Story:** Keep flipping a coin with $P(\text{H}) = p$. $X$ = number of flips until the first Head.

There are **two conventions** ‚Äî know which one you're using:

- **Blitzstein convention** (trials until first success, inclusive): $$P(X = k) = (1-p)^{k-1} p, \qquad k = 1, 2, 3, \ldots$$
    
- **Alternative** (failures before first success): $$P(Y = k) = (1-p)^k p, \qquad k = 0, 1, 2, \ldots$$
    

I'll use the first. Then $Y = X - 1$.

| $E[X]$         | $\text{Var}(X)$    | MGF                                             |
| -------------- | ------------------ | ----------------------------------------------- |
| $\dfrac{1}{p}$ | $\dfrac{1-p}{p^2}$ | $\dfrac{pe^t}{1-(1-p)e^t}$, for $t < -\ln(1-p)$ |

**The Memoryless Property ‚Äî Critical:**

$$P(X > m + n \mid X > m) = P(X > n).$$

_Proof:_ $$P(X > m+n \mid X > m) = \frac{P(X > m+n)}{P(X > m)} = \frac{(1-p)^{m+n}}{(1-p)^m} = (1-p)^n = P(X > n). \quad \square$$

> **This is the only memoryless discrete distribution.** Intuitively: if the coin hasn't landed Heads yet, the past failures tell you _nothing_ about when the next success comes. The coin has no memory.


see more: [[03.Geometric Distribution]]

## 4. Negative Binomial$(r, p)$

**The natural extension of Geometric: wait for $r$ successes.**

**Story:** $X$ = number of trials needed to accumulate $r$ successes, where each trial is independent Bern$(p)$.

$$P(X = k) = \binom{k-1}{r-1} p^r (1-p)^{k-r}, \qquad k = r, r+1, r+2, \ldots$$

**Why this PMF:** The $r$-th success occurs on trial $k$. That means: exactly $r-1$ successes in the first $k-1$ trials (there are $\binom{k-1}{r-1}$ ways to arrange them), and then a success on trial $k$ (probability $p$).

| $E[X]$         | $\text{Var}(X)$       |
| -------------- | --------------------- |
| $\dfrac{r}{p}$ | $\dfrac{r(1-p)}{p^2}$ |

**Connection to Geometric:** $\text{NegBin}(r, p)$ is the sum of $r$ independent $\text{Geom}(p)$ random variables. Immediately gives mean $r/p$ and variance $r(1-p)/p^2$ by linearity/additivity.


see more [[04.Negative Binomial Distribution]]

## 5. Hypergeometric$(N, K, n)$

**Sampling without replacement ‚Äî where independence breaks.**

**Story:** An urn has $N$ balls: $K$ are red, $N-K$ are white. Draw $n$ balls _without replacement_. $X$ = number of red balls drawn.

$$P(X = k) = \frac{\binom{K}{k}\binom{N-K}{n-k}}{\binom{N}{n}}, \qquad \max(0, n-(N-K)) \leq k \leq \min(n, K).$$

| $E[X]$                 | $\text{Var}(X)$                                                    |
| ---------------------- | ------------------------------------------------------------------ |
| $n \cdot \dfrac{K}{N}$ | $n \cdot \dfrac{K}{N} \cdot \dfrac{N-K}{N} \cdot \dfrac{N-n}{N-1}$ |

**The finite population correction factor** $(N-n)/(N-1)$ is the key difference from the Binomial. When $N \to \infty$ with $K/N \to p$, the Hypergeometric converges to $\text{Bin}(n, p)$ ‚Äî sampling without replacement from a huge population is approximately sampling with replacement.

> The Binomial assumes independence (sampling _with_ replacement). The Hypergeometric is the honest version when the population is finite and you're depleting it.


see more [[12.Hypergeometric Distribution]]

## 6. Poisson$(\lambda)$

**The distribution of rare events. One of the most important in all of probability.**

**Story:** Events occur in time (or space) at a constant average rate $\lambda$. $X$ = number of events in a unit interval.

$$P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!}, \qquad k = 0, 1, 2, \ldots$$

| $E[X]$    | $\text{Var}(X)$ | MGF                    |
| --------- | --------------- | ---------------------- |
| $\lambda$ | $\lambda$       | $e^{\lambda(e^t - 1)}$ |

The fact that mean = variance is a **signature property** ‚Äî useful for checking if Poisson is a reasonable model.

**Validity:** $\sum_{k=0}^{\infty} \frac{e^{-\lambda}\lambda^k}{k!} = e^{-\lambda} \cdot e^{\lambda} = 1.$ Uses Taylor series for $e^\lambda$.

**Poisson as a Binomial Limit:**

$$\text{Bin}(n, p) \xrightarrow{n \to \infty,, p \to 0,, np \to \lambda} \text{Pois}(\lambda).$$

_Heuristic proof:_ Fix $\lambda = np$, so $p = \lambda/n$.

$$P(X=k) = \binom{n}{k}\left(\frac{\lambda}{n}\right)^k!!\left(1-\frac{\lambda}{n}\right)^{n-k}.$$

As $n \to \infty$: $\binom{n}{k}\frac{1}{n^k} \to \frac{1}{k!}$, $\left(1-\frac{\lambda}{n}\right)^n \to e^{-\lambda}$, $\left(1-\frac{\lambda}{n}\right)^{-k} \to 1$. Hence $P(X=k) \to \frac{e^{-\lambda}\lambda^k}{k!}$.

**Reproductive property:** If $X \sim \text{Pois}(\lambda_1)$ and $Y \sim \text{Pois}(\lambda_2)$ independently, then $X + Y \sim \text{Pois}(\lambda_1 + \lambda_2)$.

see more: [[09.Poisson Distribution]]

# PART II ‚Äî Continuous Distributions

_Probability lives on intervals. We describe it with a **PDF** $f(x)$, where $P(a \leq X \leq b) = \int_a^b f(x),dx$._

**Key facts about PDFs:**

- $f(x) \geq 0$ for all $x$
- $\int_{-\infty}^{\infty} f(x), dx = 1$
- $P(X = x) = 0$ for any single point ‚Äî probability only accumulates over intervals

---

## 7. Uniform$(a, b)$

**Maximum entropy on a bounded interval. The "I have no information" distribution.**

$$f(x) = \frac{1}{b-a}, \qquad x \in [a, b].$$

$$F(x) = \frac{x-a}{b-a}, \qquad x \in [a, b].$$

| $E[X]$           | $\text{Var}(X)$       |
| ---------------- | --------------------- |
| $\dfrac{a+b}{2}$ | $\dfrac{(b-a)^2}{12}$ |

**Key use:** The Universality of the Uniform. If $X$ has CDF $F$, then $F(X) \sim \text{Uniform}(0,1)$. This is why every random number generator produces Uniforms ‚Äî you can simulate _anything_ from it.


see more: [[03.The Uniform]]

---

## 8. Normal$(\mu, \sigma^2)$

**The central distribution of all of probability and statistics.**

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right), \qquad x \in \mathbb{R}.$$

| $E[X]$ | $\text{Var}(X)$ | MGF                          |
| ------ | --------------- | ---------------------------- |
| $\mu$  | $\sigma^2$      | $e^{\mu t + \sigma^2 t^2/2}$ |

**Standard Normal:** $Z \sim N(0,1)$. If $X \sim N(\mu, \sigma^2)$, then $Z = (X-\mu)/\sigma \sim N(0,1)$. The PDF of $Z$ is $\phi(z) = \frac{1}{\sqrt{2\pi}}e^{-z^2/2}$, and its CDF is $\Phi(z)$.

**Why it's everywhere ‚Äî the CLT:** For i.i.d. $X_i$ with mean $\mu$ and variance $\sigma^2$,

$$\frac{X_1 + \cdots + X_n - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0,1).$$

The sum of _any_ i.i.d. random variables, suitably normalized, converges to Normal. This is the deepest theorem in elementary probability.

**Reproductive property:** $N(\mu_1, \sigma_1^2) + N(\mu_2, \sigma_2^2) = N(\mu_1 + \mu_2, \sigma_1^2 + \sigma_2^2)$ for independent summands.

> _Personal note: The Normal is not "natural" because nature is Normal. It's universal because of the CLT. Don't confuse the reason for its prevalence._


see more: [[06.Normal Distribution]]

## 9. Exponential$(\lambda)$

**The continuous analogue of the Geometric. The waiting time distribution.**

**Story:** Events arrive as a Poisson process with rate $\lambda$. $X$ = waiting time until the first event.

$$f(x) = \lambda e^{-\lambda x}, \qquad x > 0.$$ $$F(x) = 1 - e^{-\lambda x}, \qquad x > 0.$$

| $E[X]$               | $\text{Var}(X)$        | MGF                                               |
| -------------------- | ---------------------- | ------------------------------------------------- |
| $\dfrac{1}{\lambda}$ | $\dfrac{1}{\lambda^2}$ | $\dfrac{\lambda}{\lambda - t}$, for $t < \lambda$ |

**Memoryless Property (continuous version):**

$$P(X > s + t \mid X > s) = P(X > t).$$

_Proof:_ $$\frac{P(X > s+t)}{P(X > s)} = \frac{e^{-\lambda(s+t)}}{e^{-\lambda s}} = e^{-\lambda t} = P(X > t). \quad \square$$

> **The Exponential is the only memoryless continuous distribution.** 

see more: [[12.Exponential Distribution]]

## 10. Gamma$(\alpha, \lambda)$

**The Exponential grown up. Waiting time for $\alpha$ events.**

**Story:** Waiting time until the $\alpha$-th event in a Poisson process with rate $\lambda$.

$$f(x) = \frac{\lambda^\alpha x^{\alpha-1} e^{-\lambda x}}{\Gamma(\alpha)}, \qquad x > 0,$$

where $\Gamma(\alpha) = \int_0^\infty t^{\alpha-1} e^{-t},dt$ is the Gamma function. Note $\Gamma(n) = (n-1)!$ for positive integers $n$.

| $E[X]$                    | $\text{Var}(X)$             | MGF                                                               |
| ------------------------- | --------------------------- | ----------------------------------------------------------------- |
| $\dfrac{\alpha}{\lambda}$ | $\dfrac{\alpha}{\lambda^2}$ | $\left(\dfrac{\lambda}{\lambda - t}\right)^\alpha$, $t < \lambda$ |

**Special cases:**

- $\Gamma(1, \lambda) = \text{Expo}(\lambda)$
- $\Gamma(n/2, 1/2) = \chi^2(n)$ (Chi-squared distribution ‚Äî fundamental in statistics)

**Connection to Negative Binomial:** The Gamma is the continuous version. NegBin waits for $r$ discrete successes; Gamma waits for $\alpha$ continuous events.

see more: [[07.Gamma Transformations]]

---

## 11. Beta$(a, b)$

**The distribution of a probability. Lives on $[0,1]$.**

$$f(x) = \frac{x^{a-1}(1-x)^{b-1}}{B(a,b)}, \qquad x \in (0,1),$$

where $B(a,b) = \frac{\Gamma(a)\Gamma(b)}{\Gamma(a+b)}$ is the Beta function.

| $E[X]$           | $\text{Var}(X)$              |
| ---------------- | ---------------------------- |
| $\dfrac{a}{a+b}$ | $\dfrac{ab}{(a+b)^2(a+b+1)}$ |

**Special cases:**

- $\text{Beta}(1,1) = \text{Uniform}(0,1)$
- Larger $a, b$ ‚Üí distribution concentrates near $a/(a+b)$

**Why it matters:** In Bayesian statistics, if your data is Binomial$(n, p)$ and your prior on $p$ is Beta$(a,b)$, the posterior is Beta$(a + \text{successes},, b + \text{failures})$. The Beta is the **conjugate prior** for the Binomial. This is beautiful and I should write a note on it.

**Order statistics connection:** The $k$-th order statistic of $n$ i.i.d. Uniform$(0,1)$ random variables has a Beta$(k, n-k+1)$ distribution.

see more:[[05.Beta Transformations]]

## 12. Cauchy

**The pathological distribution. A warning about heavy tails.**

$$f(x) = \frac{1}{\pi(1+x^2)}, \qquad x \in \mathbb{R}.$$

The mean **does not exist** (the integral $\int_{-\infty}^\infty \frac{|x|}{\pi(1+x^2)}dx$ diverges). Neither does the variance.

**Why it matters:** The ratio of two independent $N(0,1)$ random variables is Cauchy. The Cauchy is a reminder that not every distribution has finite moments ‚Äî and that the CLT has conditions.

> _This is the distribution I show to people who think the Law of Large Numbers works for everything. It doesn't. Sample averages of Cauchy random variables don't converge._


see more: [[09.Cauchy Distribution]]

# PART III ‚Äî Unifying Concepts

## The CDF: One Function to Rule Them All

For any random variable $X$, the **cumulative distribution function** is

$$F(x) = P(X \leq x).$$

Properties:

1. $F$ is non-decreasing
2. $F(x) \to 0$ as $x \to -\infty$; $F(x) \to 1$ as $x \to \infty$
3. $F$ is right-continuous

For discrete $X$: $F$ is a staircase (jumps at the support points). For continuous $X$: $F$ is smooth, $F'(x) = f(x)$ (the PDF).


see more: [[10.Cumulative Distribution Function (CDF)]]

## Moments: Reading the Shape

| Moment              | Formula                 | What it tells me |
| ------------------- | ----------------------- | ---------------- |
| Mean $\mu$          | $E[X]$                  | Center of mass   |
| Variance $\sigma^2$ | $E[(X-\mu)^2]$          | Spread           |
| Skewness            | $E[(X-\mu)^3]/\sigma^3$ | Asymmetry        |
| Kurtosis            | $E[(X-\mu)^4]/\sigma^4$ | Tail weight      |

The **MGF** $M_X(t) = E[e^{tX}]$ encodes all moments: $M_X^{(n)}(0) = E[X^n]$.

see more: [[05.Moment Generating Function (MGF)]]

## The Exponential Family (Big Picture)

Most named distributions belong to the **exponential family**:

$$f(x \mid \theta) = h(x)\exp!\bigl(\eta(\theta)T(x) - A(\theta)\bigr).$$

This includes: Bernoulli, Binomial, Poisson, Exponential, Gamma, Beta, Normal. The Cauchy does **not** belong. The exponential family has beautiful statistical properties ‚Äî sufficient statistics, conjugate priors, etc.

$$\text{Bern}(p) \xrightarrow{\text{sum}} \text{Bin}(n,p) \xrightarrow{n\to\infty,, p\to 0} \text{Pois}(\lambda) \xrightarrow{\lambda \to \infty,, \text{CLT}} N(\lambda, \lambda)$$

$$\text{Geom}(p) \xrightarrow{\text{sum}} \text{NegBin}(r,p) \qquad\qquad \text{Expo}(\lambda) \xrightarrow{\text{sum}} \text{Gamma}(\alpha, \lambda)$$

$$\text{Bin}(n, K/N) \xrightarrow{N\to\infty} \text{Hypergeometric}(N,K,n)$$

$$\text{Beta}(1,1) = \text{Uniform}(0,1)$$
