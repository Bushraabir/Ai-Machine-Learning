The **Cumulative Distribution Function (CDF)** of a random variable \(X\) is a function that gives the **probability that \(X\) takes a value less than or equal to x**:$$
F_X(x) = P(X \le x)
$$
- For **discrete RVs**, the CDF is a **step function**.  
- For **continuous RVs**, the CDF is a **non-decreasing, continuous function**.

>**CDF at a specific value xxx** is the **sum of PMF values for all outcomes ≤ x**, not the sum of all PMF values at once.
 The **CDF builds up gradually**: at each step xxx, you add all the PMF values for outcomes up to that xxx.


## Properties of CDF
1. **Non-decreasing:**  (Increasing)
	$x_1 \le x_2$
	$\implies F_X(x_1) \le F_X(x_2)$
2. **Limits:**  
	$\lim_{x \to -\infty} F_X(x) = 0$ 
	$\lim_{x \to \infty} F_X(x) = 1$
3. **Right-continuous:**  
	$F_X(x) = \lim_{t \to x^+} F_X(t)$

## Relation to PMF / PDF
- **Discrete Random Variable:**  
	$F_X(x) = \sum_{t \le x} p_X(t)$
- **Continuous Random Variable:**  
	$F_X(x) = \int_{-\infty}^{x} f_X(t) dt$

- **Recovering PMF / PDF:**  
  - Discrete: $p_X(x) = F_X(x) - F_X(x^-)$
  - Continuous: $f_X(x) = \frac{d}{dx} F_X(x)$

## Examples

### Example 1: Discrete RV
Toss a fair coin 3 times. Let $X$ = number of heads.  
- PMF:  $p_X(x) = \binom{3}{x} \left(\frac{1}{2}\right)^3, \quad x = 0,1,2,3$
- CDF:  
	$F_X(0) = 1/8$
	$F_X(1) = 1/8 + 3/8 = 1/2$
	$F_X(2) = 1/2 + 3/8 = 7/8$
	$F_X(3) = 1$

### Example 2: Continuous RV
Let $X \sim \text{Uniform}(0,1)$.
- PDF: $f_X(x) = 1$ for $0 \le x \le 1$ 
- CDF:  $F_X(x) =\begin{cases} 0 & x < 0 \\x & 0 \le x \le 1 \\1 & x > 1\end{cases}$

- CDF provides a **complete description of a random variable**.  
- All probabilities can be expressed as differences of CDF values:  $P(a \le X \le b) = F_X(b) - F_X(a)$

- CDFs are **essential for understanding distributions, percentiles, and quantiles**.  
- CDFs unify discrete and continuous cases in one framework.
