
## How to Read This Table

- **Support**: the set on which $p_X$ or $f_X$ is nonzero
- **MGF**: $M_X(t) = E[e^{tX}]$ ‚Äî domain of $t$ matters; listed where it exists in a neighborhood of 0
- $\Gamma(\alpha) = \int_0^\infty x^{\alpha-1} e^{-x},dx$ ‚Äî know this function cold; $\Gamma(n) = (n-1)!$ for $n \in \mathbb{N}$
- $B(\alpha,\beta) = \frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)}$ ‚Äî the Beta function, normalizer for Beta distribution

##  DISCRETE DISTRIBUTIONS

### Bernoulli$(p)$

|||
|---|---|
|**PMF**|$P(X=1) = p,\quad P(X=0) = 1-p$|
|**Support**|${0, 1}$|
|**Mean**|$p$|
|**Variance**|$p(1-p)$|
|**MGF**|$1 - p + pe^t$|

**Story:** One coin flip. Everything else is built on this.

---

### Binomial$(n, p)$

|||
|---|---|
|**PMF**|$\displaystyle\binom{n}{k} p^k (1-p)^{n-k}$|
|**Support**|$k \in {0, 1, \ldots, n}$|
|**Mean**|$np$|
|**Variance**|$np(1-p)$|
|**MGF**|$(1-p+pe^t)^n$|

**Story:** Number of successes in $n$ independent Bernoulli$(p)$ trials.

**Key identity:** $\sum_{k=0}^n \binom{n}{k} p^k(1-p)^{n-k} = 1$ ‚Äî just the binomial theorem applied.

---

### Geometric$(p)$

|||
|---|---|
|**PMF**|$(1-p)^{k-1} p$|
|**Support**|$k \in {1, 2, 3, \ldots}$|
|**Mean**|$1/p$|
|**Variance**|$(1-p)/p^2$|
|**MGF**|$\dfrac{pe^t}{1-(1-p)e^t}$ for $t < -\ln(1-p)$|

**Story:** Number of trials until first success. Memoryless: $P(X > m+n \mid X > m) = P(X > n)$.

**Alternate version** (number of _failures_ before first success): support ${0,1,2,\ldots}$, PMF $(1-p)^k p$, mean $(1-p)/p$. Don't mix these up ‚Äî Blitzstein uses the first version.

---

### Negative Binomial$(r, p)$

|||
|---|---|
|**PMF**|$\displaystyle\binom{n-1}{r-1} p^r (1-p)^{n-r}$|
|**Support**|$n \in {r, r+1, \ldots}$|
|**Mean**|$r/p$|
|**Variance**|$r(1-p)/p^2$|
|**MGF**|$\left(\dfrac{pe^t}{1-(1-p)e^t}\right)^r$|

**Story:** Trials until $r$-th success. It's exactly the sum of $r$ i.i.d. Geometric$(p)$ ‚Äî so the MGF is just the $r$-th power. Elegant.

---

### Poisson$(\lambda)$

|||
|---|---|
|**PMF**|$\displaystyle e^{-\lambda} \frac{\lambda^k}{k!}$|
|**Support**|$k \in {0, 1, 2, \ldots}$|
|**Mean**|$\lambda$|
|**Variance**|$\lambda$|
|**MGF**|$e^{\lambda(e^t - 1)}$|

**Story:** Count of rare events. Limit of Bin$(n, \lambda/n)$ as $n \to \infty$.

**Key property:** Sum of independent Pois$(\lambda_1)$ + Pois$(\lambda_2)$ = Pois$(\lambda_1 + \lambda_2)$. Prove via MGF.

**Mean = Variance** ‚Äî this is the diagnostic. If empirical mean $\neq$ variance, data is over/underdispersed relative to Poisson.

---

### Hypergeometric$(N, K, n)$

|||
|---|---|
|**PMF**|$\displaystyle\frac{\binom{K}{k}\binom{N-K}{n-k}}{\binom{N}{n}}$|
|**Support**|$k \in {\max(0, n+K-N), \ldots, \min(n,K)}$|
|**Mean**|$nK/N$|
|**Variance**|$n\cdot\frac{K}{N}\cdot\frac{N-K}{N}\cdot\frac{N-n}{N-1}$|
|**MGF**|(messy ‚Äî not worth memorizing)|

**Story:** Drawing $n$ items without replacement from population of $N$ with $K$ successes.

**Note:** The variance has the factor $\frac{N-n}{N-1}$ ‚Äî the **finite population correction**. As $N \to \infty$ with $K/N \to p$, this $\to$ Binomial$(n,p)$. With replacement = Binomial.

##  CONTINUOUS DISTRIBUTIONS

### Uniform$(a, b)$

|||
|---|---|
|**PDF**|$\dfrac{1}{b-a}$|
|**Support**|$(a, b)$|
|**CDF**|$\dfrac{x-a}{b-a}$|
|**Mean**|$\dfrac{a+b}{2}$|
|**Variance**|$\dfrac{(b-a)^2}{12}$|
|**MGF**|$\dfrac{e^{tb}-e^{ta}}{t(b-a)}$ for $t \neq 0$|

**Story:** Maximum entropy distribution on a bounded interval. Complete ignorance.

---

### Normal$(\mu, \sigma^2)$

|||
|---|---|
|**PDF**|$\dfrac{1}{\sigma\sqrt{2\pi}}\exp!\left(-\dfrac{(x-\mu)^2}{2\sigma^2}\right)$|
|**Support**|$(-\infty, \infty)$|
|**CDF**|$\Phi!\left(\dfrac{x-\mu}{\sigma}\right)$ ‚Äî no closed form|
|**Mean**|$\mu$|
|**Variance**|$\sigma^2$|
|**MGF**|$\exp!\left(\mu t + \dfrac{\sigma^2 t^2}{2}\right)$|

**Story:** The CLT limit. Characterized by its mean and variance. Symmetric about $\mu$.

**Standard Normal $Z \sim \mathcal{N}(0,1)$:** PDF is $\varphi(z) = \frac{1}{\sqrt{2\pi}}e^{-z^2/2}$. $\Phi(z) = P(Z \leq z)$.

**Key facts:**

- $68$-$95$-$99.7$ rule: $P(|\mu - k\sigma \leq X \leq \mu + k\sigma|)$ for $k=1,2,3$
- If $X \sim \mathcal{N}(\mu,\sigma^2)$ then $aX + b \sim \mathcal{N}(a\mu+b, a^2\sigma^2)$
- Derive the MGF by completing the square in the exponent ‚Äî I've done this derivation and it's clean

### Exponential$(\lambda)$

|||
|---|---|
|**PDF**|$\lambda e^{-\lambda x}$|
|**Support**|$x > 0$|
|**CDF**|$1 - e^{-\lambda x}$|
|**Mean**|$1/\lambda$|
|**Variance**|$1/\lambda^2$|
|**MGF**|$\dfrac{\lambda}{\lambda - t}$ for $t < \lambda$|

**Story:** Waiting time for first event in a Poisson process of rate $\lambda$.

**Memoryless property:** $P(X > s+t \mid X > s) = P(X > t)$. _The only continuous memoryless distribution._

**Proof of memorylessness:** $$P(X > s+t \mid X > s) = \frac{P(X > s+t)}{P(X > s)} = \frac{e^{-\lambda(s+t)}}{e^{-\lambda s}} = e^{-\lambda t} = P(X > t) \quad \checkmark$$

---

### Gamma$(\alpha, \beta)$ ‚Äî _rate parametrization_

> Convention warning: Some books use scale $\theta = 1/\beta$. Blitzstein uses rate $\beta$. I'm using Blitzstein's convention throughout.

|||
|---|---|
|**PDF**|$\dfrac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}$|
|**Support**|$x > 0$|
|**Mean**|$\alpha/\beta$|
|**Variance**|$\alpha/\beta^2$|
|**MGF**|$\left(\dfrac{\beta}{\beta-t}\right)^\alpha$ for $t < \beta$|

**Story:** Sum of $\alpha$ i.i.d. Exp$(\beta)$ random variables (when $\alpha \in \mathbb{N}$). Generalized to $\alpha > 0$ via the Gamma function.

**Special cases:**

- $\alpha = 1$: Exponential$(\beta)$
- $\alpha = k/2$, $\beta = 1/2$: Chi-squared with $k$ d.f.

---

### Beta$(\alpha, \beta)$

|||
|---|---|
|**PDF**|$\dfrac{1}{B(\alpha,\beta)} x^{\alpha-1}(1-x)^{\beta-1}$|
|**Support**|$x \in (0,1)$|
|**Mean**|$\dfrac{\alpha}{\alpha+\beta}$|
|**Variance**|$\dfrac{\alpha\beta}{(\alpha+\beta)^2(\alpha+\beta+1)}$|
|**MGF**|(messy hypergeometric series ‚Äî not useful)|

**Story:** Distribution _over_ $[0,1]$ ‚Äî think of it as a distribution for a probability parameter. Conjugate prior to Binomial in Bayesian inference.

**Special cases:**

- $\alpha = \beta = 1$: Uniform$(0,1)$
- $\alpha = \beta$: Symmetric about $1/2$
- $\alpha < 1, \beta < 1$: U-shaped (bimodal at 0 and 1)

---

### Chi-Squared $\chi^2(k)$

|||
|---|---|
|**PDF**|$\Gamma(k/2, 1/2)$ form: $\dfrac{1}{2^{k/2}\Gamma(k/2)}x^{k/2-1}e^{-x/2}$|
|**Support**|$x > 0$|
|**Mean**|$k$|
|**Variance**|$2k$|
|**MGF**|$(1-2t)^{-k/2}$ for $t < 1/2$|

**Story:** $\chi^2(k) = Z_1^2 + \cdots + Z_k^2$ where $Z_i \overset{iid}{\sim} \mathcal{N}(0,1)$. This is a Gamma$(k/2, 1/2)$.

---

### Student's $t(k)$

|||
|---|---|
|**PDF**|$\dfrac{\Gamma!\left(\frac{k+1}{2}\right)}{\sqrt{k\pi},\Gamma!\left(\frac{k}{2}\right)}\left(1+\dfrac{x^2}{k}\right)^{-(k+1)/2}$|
|**Support**|$(-\infty, \infty)$|
|**Mean**|$0$ (for $k > 1$)|
|**Variance**|$\dfrac{k}{k-2}$ (for $k > 2$)|
|**MGF**|Does not exist|

**Story:** $T = Z / \sqrt{V/k}$ where $Z \sim \mathcal{N}(0,1)$, $V \sim \chi^2(k)$, independent. Heavy tails for small $k$; approaches $\mathcal{N}(0,1)$ as $k \to \infty$.

---

### Cauchy$(x_0, \gamma)$

|||
|---|---|
|**PDF**|$\dfrac{1}{\pi\gamma\left[1+\left(\frac{x-x_0}{\gamma}\right)^2\right]}$|
|**Support**|$(-\infty, \infty)$|
|**Mean**|**Undefined**|
|**Variance**|**Undefined**|
|**MGF**|**Does not exist**|

**Story:** $t(1)$. Also: ratio of two independent standard normals. A pathological distribution ‚Äî symmetric, looks reasonable, but the mean diverges. The sample mean of Cauchy r.v.s has the _same_ distribution regardless of sample size. The CLT simply fails here.

##  Universal Formulas (Both Worlds)

| Concept                       | Formula                                                              |
| ----------------------------- | -------------------------------------------------------------------- |
| **Variance identity**         | $\text{Var}(X) = E[X^2] - (E[X])^2$                                  |
| **Standard deviation**        | $\sigma_X = \sqrt{\text{Var}(X)}$                                    |
| **Covariance**                | $\text{Cov}(X,Y) = E[XY] - E[X]E[Y]$                                 |
| **Correlation**               | $\rho_{XY} = \dfrac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y} \in [-1,1]$  |
| **Linearity of $E$**          | $E[aX + bY] = aE[X] + bE[Y]$ always                                  |
| **Variance of sum (indep)**   | $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y)$ if $X \perp Y$     |
| **Variance of sum (general)** | $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X,Y)$ |
| **MGF moment extraction**     | $E[X^n] = M_X^{(n)}(0)$                                              |
| **MGF of sum (indep)**        | $M_{X+Y}(t) = M_X(t) \cdot M_Y(t)$                                   |
| **LOTUS (discrete)**          | $E[g(X)] = \sum_x g(x) p_X(x)$                                       |
| **LOTUS (continuous)**        | $E[g(X)] = \int g(x) f_X(x), dx$                                     |
| **Adam's Law**                | $E[X] = E[E[X \mid Y]]$                                              |
| **Eve's Law**                 | $\text{Var}(X) = E[\text{Var}(X\mid Y)] + \text{Var}(E[X\mid Y])$    |

##  The "Mean = Variance" Diagnostic

Only the **Poisson** has mean = variance. This is a testable property. In practice, if you see count data and mean $\approx$ variance, reach for Poisson first.
