

**Theorem (Universality of the Uniform).** Let $F$ be a CDF that is continuous and strictly increasing on its support.

1. If $U \sim \text{Uniform}(0,1)$, then $X = F^{-1}(U)$ has CDF $F$.
2. If $X$ has CDF $F$, then $F(X) \sim \text{Uniform}(0,1)$.

That is it. Two lines. The depth is in what it means and what it enables.

---

## The Proof — Both Directions, Carefully

### Direction 2 first: $X \mapsto U$

This direction is cleaner, so I prove it first.

Let $X \sim F$ where $F$ is continuous and strictly increasing. Define $Y = F(X)$. I claim $Y \sim \text{Uniform}(0,1)$.

To show this, I compute the CDF of $Y$. For $y \in [0,1]$:

$$P(Y \leq y) = P(F(X) \leq y)$$

Since $F$ is strictly increasing, $F(X) \leq y$ if and only if $X \leq F^{-1}(y)$. So:

$$P(F(X) \leq y) = P(X \leq F^{-1}(y)) = F(F^{-1}(y)) = y$$

The last step uses the fact that $F^{-1}$ is the true functional inverse when $F$ is strictly increasing. Therefore $P(Y \leq y) = y$ for all $y \in [0,1]$, which is exactly the CDF of $\text{Uniform}(0,1)$. $\square$

### Direction 1: $U \mapsto X$

Let $U \sim \text{Uniform}(0,1)$ and define $X = F^{-1}(U)$. I claim $X \sim F$.

Compute the CDF of $X$. For any $x$:

$$P(X \leq x) = P(F^{-1}(U) \leq x)$$

Since $F$ is strictly increasing, $F^{-1}(U) \leq x$ if and only if $U \leq F(x)$. So:

$$P(F^{-1}(U) \leq x) = P(U \leq F(x)) = F(x)$$

The last step uses $U \sim \text{Uniform}(0,1)$, so $P(U \leq t) = t$ for $t \in [0,1]$, and $F(x) \in [0,1]$ for all $x$. Therefore $X$ has CDF $F$. $\square$

---

## The Logical Structure

The two directions together say:

$$X \sim F \iff F(X) \sim \text{Uniform}(0,1) \iff X = F^{-1}(U), \quad U \sim \text{Uniform}(0,1)$$

This is an equivalence, not just an implication. Every continuous random variable is a deterministic function of a single $\text{Uniform}(0,1)$ random variable, and that function is its own inverse CDF. The Uniform is the universal raw material.

---

## Why the Continuity Assumption is Necessary

The theorem requires $F$ to be continuous and strictly increasing. Both conditions are doing real work.

**Strict monotonicity:** If $F$ is flat on some interval $(a,b)$ — meaning $P(a < X < b) = 0$ — then $F^{-1}$ is not well-defined on the image of that flat region. The inverse CDF is multi-valued, and we have to define $F^{-1}(u) = \inf{x : F(x) \geq u}$ (the generalized inverse). The theorem still holds with this definition, but the proof needs more care.

**Continuity:** The step $P(F(X) \leq y) = P(X \leq F^{-1}(y))$ uses that $F$ is strictly increasing, which requires continuity of $F$ to guarantee $\leq$ on both sides simultaneously. If $F$ has jumps — as in a discrete distribution — then $F(X)$ cannot be Uniform$(0,1)$, because a discrete $X$ maps to a discrete set of values under $F$, and a discrete set has measure zero under the Uniform.

This is why there is no universality theorem for discrete distributions in the same form. You can simulate discrete distributions from a Uniform, but $F(X)$ will not itself be Uniform.

> **Personal note:** The failure of universality for discrete distributions is not a defect — it is informative. It reflects the fundamental difference between the two worlds. In the continuous world, $F$ is a smooth bijection from the support of $X$ to $(0,1)$, so it genuinely transforms $X$ into something uniform. In the discrete world, $F$ is a staircase, and staircases cannot rearrange probability into a flat density.

## The Probability Integral Transform

Direction 2 has its own name: the **Probability Integral Transform** (PIT). It says:

> If $X$ has a continuous distribution with CDF $F$, then $F(X) \sim \text{Uniform}(0,1)$.

This is more than a simulation trick. It is a tool for proving things.

**Application: Testing goodness of fit.** Suppose I have data $x_1, \ldots, x_n$ and I want to test whether they came from a distribution with CDF $F$. If they did, then $F(x_1), \ldots, F(x_n)$ should look like i.i.d. $\text{Uniform}(0,1)$ samples. This is the basis of the Kolmogorov–Smirnov test: compare the empirical CDF of the transformed data against the uniform CDF.

**Application: Generating any distribution from Uniform.** This is Direction 1. In practice, a computer's random number generator produces (pseudo) Uniform$(0,1)$ values. To sample from any continuous distribution with a tractable inverse CDF, just apply $F^{-1}$ to the output. This is called **inverse transform sampling**.

---

## Worked Examples of Inverse Transform Sampling

### Sampling from Exponential$(\lambda)$

The CDF is $F(x) = 1 - e^{-\lambda x}$. Solve $u = 1 - e^{-\lambda x}$ for $x$:

$$1 - u = e^{-\lambda x} \implies -\lambda x = \ln(1-u) \implies x = -\frac{\ln(1-u)}{\lambda}$$

So if $U \sim \text{Uniform}(0,1)$, then $X = -\frac{\ln(1-U)}{\lambda} \sim \text{Exponential}(\lambda)$.

Since $1 - U \sim \text{Uniform}(0,1)$ as well (by symmetry), this simplifies to:

$$X = -\frac{\ln U}{\lambda} \sim \text{Exponential}(\lambda)$$

This is clean. The $\log$ of a Uniform is (up to scaling) an Exponential. I can verify: $P(X > x) = P(-\ln U / \lambda > x) = P(U < e^{-\lambda x}) = e^{-\lambda x}$. Correct.

### Sampling from Cauchy$(0,1)$

The CDF is $F(x) = \frac{1}{2} + \frac{1}{\pi}\arctan(x)$. Solving $u = F(x)$:

$$x = \tan!\left(\pi!\left(u - \frac{1}{2}\right)\right)$$

So $X = \tan(\pi(U - 1/2)) \sim \text{Cauchy}(0,1)$. The tangent of a uniform angle is Cauchy. This is not coincidental — it is exactly the geometric interpretation: if $\theta$ is uniform on $(-\pi/2, \pi/2)$, then $\tan(\theta)$ is Cauchy.

### Sampling from Normal$(\mu, \sigma^2)$

The Normal CDF $\Phi$ has no closed form, so $\Phi^{-1}$ cannot be written explicitly. Inverse transform sampling still works in principle but is slow numerically. In practice, the Box-Muller method is used instead — but that is a different story.

## A Deeper View: What the Theorem Really Says

The universality theorem is really a statement about measure theory, even if I can prove it without that language.

Every probability space $(\Omega, \mathcal{F}, P)$ that supports a $\text{Uniform}(0,1)$ random variable is rich enough to support any continuous random variable whatsoever. The Uniform "contains" all continuous distributions, in the sense that they are all measurable images of it.

This is why in theoretical probability, it is common to work on the canonical probability space $([0,1], \mathcal{B}[0,1], \text{Lebesgue})$ — the unit interval with Lebesgue measure. On this space, the identity map $\omega \mapsto \omega$ is $\text{Uniform}(0,1)$, and by universality, every continuous distribution can be realized as a function of this identity map. The entire theory of continuous random variables lives inside $[0,1]$.

---

## The Generalized Inverse CDF

When $F$ is not strictly increasing — flat regions exist — define:

$$F^{-1}(u) = \inf{x \in \mathbb{R} : F(x) \geq u}$$

This is the **quantile function**. For a strictly increasing continuous $F$, it agrees with the usual inverse. For a discrete $F$, it is still well-defined and gives the smallest $x$ such that $F(x) \geq u$.

**Key fact:** With this definition, Direction 1 holds for _any_ distribution:

> If $U \sim \text{Uniform}(0,1)$, then $F^{-1}(U)$ has CDF $F$.

This is how discrete distributions are simulated from Uniform$(0,1)$: generate $U$, find the smallest $k$ such that $F(k) \geq U$, return $k$. This is exact, not an approximation.

Direction 2 fails for discrete distributions, as argued above — $F(X)$ will not be Uniform when $X$ is discrete. But the simulation direction always works.

---

## Summary of the Core Ideas

The universality theorem has three faces:

**The simulation face.** $X = F^{-1}(U)$ lets you generate any continuous distribution from a Uniform. This is the foundation of Monte Carlo methods.

**The testing face.** $F(X) \sim \text{Uniform}(0,1)$ when $X \sim F$. This is the foundation of goodness-of-fit testing.

**The theoretical face.** All continuous random variables live inside $\text{Uniform}(0,1)$ via their inverse CDFs. The Uniform is not a simple distribution — it is the atom of all continuous probability.
