
## Why Orthogonal Bases Are Worth the Effort

Every basis lets you decompose vectors. But a generic basis forces you to solve a linear system to find the coefficients — you have to invert something. An orthogonal basis makes the coefficients free: each one is just a dot product divided by a squared norm. No system to solve. No matrix to invert.

This is not just computational convenience. It is the reason the normal equations simplify so dramatically when the columns of $A$ are orthogonal, the reason Fourier series work the way they do, and the reason the QR factorization is the right tool for least squares. Everything traces back to this one fact about orthogonal bases.

---

## Definitions

A set of vectors ${q_1, q_2, \ldots, q_n}$ in $\mathbb{R}^m$ is:

- **orthogonal** if $q_i^T q_j = 0$ for all $i \neq j$
- **orthonormal** if additionally $q_i^T q_i = 1$ for all $i$, i.e., $|q_i| = 1$

The compact way to write orthonormality: $q_i^T q_j = \delta_{ij}$, where $\delta_{ij}$ is the Kronecker delta.

**Key fact:** Any orthogonal set of nonzero vectors is automatically linearly independent.

_Proof._ Suppose $\sum_{i=1}^n c_i q_i = 0$. Take the dot product of both sides with $q_j$:

$$q_j^T \left(\sum_{i=1}^n c_i q_i\right) = \sum_{i=1}^n c_i (q_j^T q_i) = c_j |q_j|^2 = 0$$

Since $q_j \neq 0$, we have $|q_j|^2 > 0$, so $c_j = 0$. This holds for every $j$. $\square$

---

## The Orthonormal Matrix $Q$

If the columns of $Q$ are orthonormal, then

$$Q^T Q = I$$

because $(Q^T Q)_{ij} = q_i^T q_j = \delta_{ij}$.

If $Q$ is also square ($m = n$), then $Q^T Q = I$ implies $Q^T = Q^{-1}$, so $Q Q^T = I$ as well. Such $Q$ is called an **orthogonal matrix** (the standard but slightly misleading name — the columns are orthonormal, not merely orthogonal).

If $Q$ is tall ($m > n$), then $Q^T Q = I_n$ but $QQ^T \neq I_m$. In this case $QQ^T$ is the projection matrix onto $C(Q)$.

---

## Projections Become Trivial

Suppose ${q_1, \ldots, q_n}$ is an orthonormal basis for a subspace $S \subseteq \mathbb{R}^m$. The projection of any $b \in \mathbb{R}^m$ onto $S$ is

$$\hat{b} = \sum_{i=1}^n (q_i^T b), q_i$$

Each coefficient $q_i^T b$ is the component of $b$ in the direction of $q_i$. Compare this to the general formula $\hat{b} = A(A^TA)^{-1}A^T b$ — when $A = Q$ has orthonormal columns, $A^T A = I$, so

$$\hat{b} = Q(Q^TQ)^{-1}Q^T b = QQ^T b$$

and in coordinates, $\hat{x} = Q^T b$, i.e., $\hat{x}_i = q_i^T b$. The normal equations reduce to reading off dot products.

This is the payoff. When the basis is orthonormal, projection requires no solving at all.

## The Gram-Schmidt Process

**Problem:** Given linearly independent vectors $a_1, a_2, \ldots, a_n$, produce an orthonormal basis $q_1, q_2, \ldots, q_n$ for the same subspace.

The idea is sequential: at each step, take the next vector $a_k$, subtract off its projections onto all previously constructed orthonormal vectors, and normalize what remains.

### Step 1

$$A_1 = a_1, \qquad q_1 = \frac{A_1}{|A_1|}$$

### Step 2

Subtract the component of $a_2$ along $q_1$:

$$A_2 = a_2 - (q_1^T a_2), q_1$$

Then normalize:

$$q_2 = \frac{A_2}{|A_2|}$$

By construction, $q_1^T A_2 = q_1^T a_2 - (q_1^T a_2)(q_1^T q_1) = 0$. So $A_2 \perp q_1$.

### Step 3

Subtract components along $q_1$ and $q_2$:

$$A_3 = a_3 - (q_1^T a_3), q_1 - (q_2^T a_3), q_2$$

Then normalize:

$$q_3 = \frac{A_3}{|A_3|}$$

### General Step $k$

$$A_k = a_k - \sum_{j=1}^{k-1} (q_j^T a_k), q_j, \qquad q_k = \frac{A_k}{|A_k|}$$

The subtracted piece $\sum_{j=1}^{k-1}(q_j^T a_k)q_j$ is precisely the projection of $a_k$ onto $\text{span}{q_1, \ldots, q_{k-1}}$. What remains, $A_k$, is the component of $a_k$ orthogonal to that subspace.

If at any step $A_k = 0$, the vectors $a_1, \ldots, a_k$ were linearly dependent. Gram-Schmidt detects dependence.

## A Fully Worked Example

Let

$$a_1 = \begin{pmatrix} 1 \ 1 \ 0 \end{pmatrix}, \quad a_2 = \begin{pmatrix} 1 \ 0 \ 1 \end{pmatrix}, \quad a_3 = \begin{pmatrix} 0 \ 1 \ 1 \end{pmatrix}$$

**Step 1.**

$$A_1 = \begin{pmatrix} 1 \ 1 \ 0 \end{pmatrix}, \qquad |A_1| = \sqrt{2}, \qquad q_1 = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \ 1 \ 0 \end{pmatrix}$$

**Step 2.**

$$q_1^T a_2 = \frac{1}{\sqrt{2}}(1 \cdot 1 + 1 \cdot 0 + 0 \cdot 1) = \frac{1}{\sqrt{2}}$$

$$A_2 = a_2 - \frac{1}{\sqrt{2}} \cdot q_1 = \begin{pmatrix} 1 \ 0 \ 1 \end{pmatrix} - \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \ 1 \ 0 \end{pmatrix} = \begin{pmatrix} 1 \ 0 \ 1 \end{pmatrix} - \begin{pmatrix} 1/2 \ 1/2 \ 0 \end{pmatrix} = \begin{pmatrix} 1/2 \ -1/2 \ 1 \end{pmatrix}$$

$$|A_2| = \sqrt{1/4 + 1/4 + 1} = \sqrt{3/2} = \frac{\sqrt{6}}{2}$$

$$q_2 = \frac{2}{\sqrt{6}}\begin{pmatrix} 1/2 \ -1/2 \ 1 \end{pmatrix} = \frac{1}{\sqrt{6}}\begin{pmatrix} 1 \ -1 \ 2 \end{pmatrix}$$

**Check:** $q_1^T q_2 = \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{6}}(1 - 1 + 0) = 0$. $\checkmark$

**Step 3.**

$$q_1^T a_3 = \frac{1}{\sqrt{2}}(0 + 1 + 0) = \frac{1}{\sqrt{2}}$$

$$q_2^T a_3 = \frac{1}{\sqrt{6}}(0 - 1 + 2) = \frac{1}{\sqrt{6}}$$

$$A_3 = a_3 - \frac{1}{\sqrt{2}} q_1 - \frac{1}{\sqrt{6}} q_2$$

$$= \begin{pmatrix} 0 \ 1 \ 1 \end{pmatrix} - \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \ 1 \ 0 \end{pmatrix} - \frac{1}{\sqrt{6}} \cdot \frac{1}{\sqrt{6}}\begin{pmatrix} 1 \ -1 \ 2 \end{pmatrix}$$

$$= \begin{pmatrix} 0 \ 1 \ 1 \end{pmatrix} - \frac{1}{2}\begin{pmatrix} 1 \ 1 \ 0 \end{pmatrix} - \frac{1}{6}\begin{pmatrix} 1 \ -1 \ 2 \end{pmatrix}$$

$$= \begin{pmatrix} 0 - 1/2 - 1/6 \ 1 - 1/2 + 1/6 \ 1 - 0 - 1/3 \end{pmatrix} = \begin{pmatrix} -2/3 \ 2/3 \ 2/3 \end{pmatrix}$$

$$|A_3| = \sqrt{4/9 + 4/9 + 4/9} = \sqrt{4/3} = \frac{2}{\sqrt{3}}$$

$$q_3 = \frac{\sqrt{3}}{2} \cdot \begin{pmatrix} -2/3 \ 2/3 \ 2/3 \end{pmatrix} = \frac{1}{\sqrt{3}}\begin{pmatrix} -1 \ 1 \ 1 \end{pmatrix}$$

**Final orthonormal basis:**

$$q_1 = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \ 1 \ 0 \end{pmatrix}, \qquad q_2 = \frac{1}{\sqrt{6}}\begin{pmatrix} 1 \ -1 \ 2 \end{pmatrix}, \qquad q_3 = \frac{1}{\sqrt{3}}\begin{pmatrix} -1 \ 1 \ 1 \end{pmatrix}$$

These three vectors form an orthonormal basis for $\mathbb{R}^3$.

## Gram-Schmidt Produces the QR Factorization

This is where the algebra crystallizes. Look at what Gram-Schmidt says about $a_1$:

$$a_1 = |A_1|, q_1 = (q_1^T a_1), q_1$$

For $a_2$:

$$a_2 = (q_1^T a_2), q_1 + |A_2|, q_2$$

For $a_3$:

$$a_3 = (q_1^T a_3), q_1 + (q_2^T a_3), q_2 + |A_3|, q_3$$

Writing all three columns together in matrix form: $A = QR$ where

$$Q = \begin{pmatrix} | & | & | \ q_1 & q_2 & q_3 \ | & | & | \end{pmatrix}, \qquad R = \begin{pmatrix} q_1^T a_1 & q_1^T a_2 & q_1^T a_3 \ 0 & q_2^T a_2 & q_2^T a_3 \ 0 & 0 & q_3^T a_3 \end{pmatrix}$$

$R$ is upper triangular with positive diagonal entries $|A_1|, |A_2|, |A_3|$. This is the **QR factorization** of $A$.

The factorization $A = QR$ exists for any matrix with linearly independent columns. It is unique if we require the diagonal of $R$ to be positive.

---

## QR and the Normal Equations

Recall the normal equations for least squares: $A^T A \hat{x} = A^T b$.

Substitute $A = QR$:

$$(QR)^T(QR)\hat{x} = (QR)^T b$$ $$R^T Q^T Q R \hat{x} = R^T Q^T b$$ $$R^T R \hat{x} = R^T Q^T b$$

Since $R$ is invertible (columns of $A$ are independent), $R^T$ is invertible. Multiply both sides by $(R^T)^{-1}$:

$$R \hat{x} = Q^T b$$

This is the payoff. Instead of solving the potentially ill-conditioned $n \times n$ system $A^T A \hat{x} = A^T b$, we solve the triangular system $R\hat{x} = Q^T b$ by back substitution. Triangular systems are numerically stable and cheap.

The solution is $\hat{x} = R^{-1} Q^T b$, and the projection is $\hat{b} = A\hat{x} = QR \cdot R^{-1}Q^T b = QQ^T b$.

---

## Why Not Just Always Use QR?

Computing the QR factorization via Gram-Schmidt costs $O(mn^2)$ operations, which is the same order as forming $A^T A$ and solving it. The difference is numerical stability: forming $A^T A$ squares the condition number of $A$. If $A$ is moderately ill-conditioned (condition number $\kappa$), then $A^T A$ has condition number $\kappa^2$, and solving the normal equations directly can lose half the significant digits. QR avoids this.

For the polynomial fitting problem from the previous note — where $A^T A$ approaches a Hilbert matrix — this is not an academic concern. QR is the right tool.

---

## The Subspace Preservation Property

Gram-Schmidt has a crucial structural property: at each step $k$,

$$\text{span}{q_1, \ldots, q_k} = \text{span}{a_1, \ldots, a_k}$$

The first $k$ orthonormal vectors span exactly the same subspace as the first $k$ original vectors. This is why $R$ is upper triangular and not just any matrix — $a_k$ is expressed using only $q_1, \ldots, q_k$, never any later $q_j$.

---

## Things to Remember (for Myself)

- Orthogonal $\implies$ independent. The proof is one line: dot with $q_j$ and everything else vanishes.
- $Q^T Q = I$ always (when columns are orthonormal). $QQ^T = I$ only when $Q$ is square.
- Gram-Schmidt formula at step $k$: subtract all projections onto previous $q_j$'s, then normalize. One formula, repeated.
- If $A_k = 0$ during Gram-Schmidt, the original vectors were dependent. This is how the process detects dependence.
- $A = QR$ is the output of Gram-Schmidt written in matrix form. $R$ is upper triangular, diagonal entries are the norms $|A_k|$.
- For least squares, $R\hat{x} = Q^Tb$ replaces $A^TA\hat{x} = A^Tb$. Triangular, stable, elegant.
- The condition number of $A^TA$ is $\kappa(A)^2$. This is the concrete reason QR beats normal equations numerically.
