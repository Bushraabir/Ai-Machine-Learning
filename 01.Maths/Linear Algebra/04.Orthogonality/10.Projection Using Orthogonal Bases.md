## The Problem With What We Had

In the last two notes, projecting onto $C(A)$ required solving the normal equations $A^T A \hat{x} = A^T b$. This works, but it has a cost: we form $A^T A$, which squares the condition number of $A$, and then we solve a system whose matrix can be badly conditioned even when $A$ itself is not.

There is a better way, and it comes from asking: what if the columns of $A$ were orthogonal to begin with?

The answer is that $A^T A$ becomes diagonal, the normal equations decouple, and each coefficient is just a dot product divided by a norm squared. No system to solve. The projection formula collapses into something almost trivially clean.

This note is about that situation — and about how to manufacture it when it does not arise naturally.

---

## Projection onto a Single Vector

Start with the simplest case. Project $b \in \mathbb{R}^m$ onto the line spanned by a single nonzero vector $q_1$.

The projection is

$$\hat{b} = \frac{q_1^T b}{q_1^T q_1} q_1$$

The scalar $\hat{x}_1 = \frac{q_1^T b}{q_1^T q_1}$ is the coefficient, and $q_1 \hat{x}_1$ is the closest point on the line to $b$. The error $b - \hat{b}$ is orthogonal to $q_1$ by construction.

If $q_1$ happens to be a **unit vector** — i.e., $|q_1| = 1$ so $q_1^T q_1 = 1$ — then this simplifies to

$$\hat{b} = (q_1^T b), q_1, \qquad \hat{x}_1 = q_1^T b$$

The projection matrix for this single direction is $P_1 = q_1 q_1^T$ (rank 1, $m \times m$).

---

## Projection onto a Subspace with an Orthonormal Basis

Now let $S$ be a $k$-dimensional subspace of $\mathbb{R}^m$, and suppose we have an **orthonormal basis** for $S$:

$$q_1, q_2, \ldots, q_k \in \mathbb{R}^m, \qquad q_i^T q_j = \begin{cases} 1 & i = j \ 0 & i \neq j \end{cases}$$

Form the matrix $Q = \begin{pmatrix} q_1 & q_2 & \cdots & q_k \end{pmatrix}$, which is $m \times k$. The orthonormality conditions say exactly

$$Q^T Q = I_k$$

**Claim:** The projection of $b$ onto $S = C(Q)$ is

$$\hat{b} = QQ^T b$$

and the coefficient vector is $\hat{x} = Q^T b$, meaning

$$\hat{x}_i = q_i^T b \quad \text{for each } i = 1, \ldots, k$$

_Why?_ The normal equations for projecting $b$ onto $C(Q)$ are

$$Q^T Q \hat{x} = Q^T b$$

But $Q^T Q = I_k$, so the normal equations are just

$$I_k \hat{x} = Q^T b \implies \hat{x} = Q^T b$$

The system is already solved. No elimination needed. Each coefficient $\hat{x}_i = q_i^T b$ is computed independently of the others.

The projection matrix is

$$P = Q(Q^T Q)^{-1} Q^T = Q I_k^{-1} Q^T = QQ^T$$

---

## Why This Is So Much Better

With a general basis, the projection required inverting the $k \times k$ matrix $A^T A$. With an orthonormal basis, $A^T A = I$ and no inversion is needed at all.

More precisely: the coefficient of $b$ in the direction $q_i$ is completely decoupled from every other direction. I can compute $\hat{x}_i = q_i^T b$ without knowing anything about $q_j$ for $j \neq i$. This is the defining feature of orthonormal bases — coordinates are independent.

This is also why adding a new basis vector $q_{k+1}$ to an orthonormal set does not change any of the previous coefficients. With a general basis, adding a new column to $A$ changes $A^T A$ and hence changes every coefficient. With an orthonormal basis, old coefficients are untouched.

---

## The Full Projection Formula

Expanding $\hat{b} = QQ^T b$:

$$\hat{b} = \sum_{i=1}^k (q_i^T b), q_i$$

This is completely explicit. The projection onto $S$ is the sum of projections onto each individual direction $q_i$. These projections add because the $q_i$ are orthogonal — they do not interfere with each other.

The projection matrix $P = QQ^T$ is $m \times m$, symmetric, idempotent. Written out:

$$P = \sum_{i=1}^k q_i q_i^T$$

Each $q_i q_i^T$ is the rank-1 projection onto the line spanned by $q_i$. The full projection onto $S$ is their sum. This decomposition is only valid because the $q_i$ are mutually orthogonal — otherwise projecting onto each line separately and adding would double-count the overlapping directions.

## A Concrete Example

Let $S$ be the subspace of $\mathbb{R}^3$ spanned by

$$q_1 = \frac{1}{\sqrt{2}}\begin{pmatrix}1\1\0\end{pmatrix}, \qquad q_2 = \frac{1}{\sqrt{6}}\begin{pmatrix}1\-1\2\end{pmatrix}$$

Check orthonormality:

$$q_1^T q_1 = \frac{1}{2}(1+1+0) = 1 \checkmark$$ $$q_2^T q_2 = \frac{1}{6}(1+1+4) = 1 \checkmark$$ $$q_1^T q_2 = \frac{1}{\sqrt{12}}(1\cdot1 + 1\cdot(-1) + 0\cdot2) = \frac{1}{\sqrt{12}}(0) = 0 \checkmark$$

Project $b = \begin{pmatrix}1\0\0\end{pmatrix}$ onto $S$.

$$\hat{x}_1 = q_1^T b = \frac{1}{\sqrt{2}}(1) = \frac{1}{\sqrt{2}}$$

$$\hat{x}_2 = q_2^T b = \frac{1}{\sqrt{6}}(1) = \frac{1}{\sqrt{6}}$$

$$\hat{b} = \hat{x}_1 q_1 + \hat{x}_2 q_2 = \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{2}}\begin{pmatrix}1\1\0\end{pmatrix} + \frac{1}{\sqrt{6}} \cdot \frac{1}{\sqrt{6}}\begin{pmatrix}1\-1\2\end{pmatrix}$$

$$= \frac{1}{2}\begin{pmatrix}1\1\0\end{pmatrix} + \frac{1}{6}\begin{pmatrix}1\-1\2\end{pmatrix} = \begin{pmatrix}1/2 + 1/6 \ 1/2 - 1/6 \ 1/3\end{pmatrix} = \begin{pmatrix}2/3\1/3\1/3\end{pmatrix}$$

**Verify:** $e = b - \hat{b} = \begin{pmatrix}1/3\-1/3\-1/3\end{pmatrix}$.

Check $e \perp q_1$: $q_1^T e = \frac{1}{\sqrt{2}}(1/3 - 1/3 + 0) = 0$. $\checkmark$

Check $e \perp q_2$: $q_2^T e = \frac{1}{\sqrt{6}}(1/3 + 1/3 - 2/3) = \frac{1}{\sqrt{6}}(0) = 0$. $\checkmark$

The projection matrix is:

$$P = q_1 q_1^T + q_2 q_2^T = \frac{1}{2}\begin{pmatrix}1&1&0\1&1&0\0&0&0\end{pmatrix} + \frac{1}{6}\begin{pmatrix}1&-1&2\-1&1&-2\2&-2&4\end{pmatrix} = \begin{pmatrix}2/3&1/3&1/3\1/3&2/3&-1/3\1/3&-1/3&2/3\end{pmatrix}$$

Verify $Pb = \hat{b}$:

$$\begin{pmatrix}2/3&1/3&1/3\1/3&2/3&-1/3\1/3&-1/3&2/3\end{pmatrix}\begin{pmatrix}1\0\0\end{pmatrix} = \begin{pmatrix}2/3\1/3\1/3\end{pmatrix} \checkmark$$

## The Special Case: $Q$ is Square

If $Q$ is $m \times m$ — i.e., the orthonormal basis spans all of $\mathbb{R}^m$ — then $Q^T Q = I$ and $QQ^T = I$ as well. $Q$ is an **orthogonal matrix** (the standard term, despite the columns being orthonormal). In this case:

$$P = QQ^T = I$$

Projection onto $\mathbb{R}^m$ is the identity — of course. The formula $\hat{x} = Q^T b$ in this case gives the coordinates of $b$ in the orthonormal basis ${q_1, \ldots, q_m}$.

For a square orthogonal $Q$: $Q^{-1} = Q^T$. Inverting an orthogonal matrix is free — just transpose.

---

## Connection to Gram-Schmidt and QR

Where do orthonormal bases come from? If $A$ has independent columns $a_1, \ldots, a_k$, the **Gram-Schmidt process** constructs orthonormal vectors $q_1, \ldots, q_k$ with $C(q_1, \ldots, q_j) = C(a_1, \ldots, a_j)$ for each $j$.

In matrix language, Gram-Schmidt is the **QR factorization** $A = QR$, where:

- $Q$ is $m \times k$ with orthonormal columns
- $R$ is $k \times k$ upper triangular with positive diagonal entries

The normal equations $A^T A \hat{x} = A^T b$ become, via $A = QR$:

$$(QR)^T(QR)\hat{x} = (QR)^T b$$ $$R^T Q^T Q R \hat{x} = R^T Q^T b$$ $$R^T R \hat{x} = R^T Q^T b$$

Since $R$ is invertible (positive diagonal), multiply on the left by $(R^T)^{-1}$:

$$R\hat{x} = Q^T b$$

This is a triangular system — solved by back-substitution in $O(k^2)$ operations, with no formation of $A^T A$ and no squaring of the condition number.

This is why QR is the right way to solve least squares problems numerically. The orthonormal basis implicit in $Q$ is the key.

## Summary of Projection Formulas

|Basis type|Projection coefficient|Projection vector|Projection matrix|
|---|---|---|---|
|Single vector $a$|$\hat{x} = \dfrac{a^T b}{a^T a}$|$\hat{b} = a \hat{x}$|$P = \dfrac{aa^T}{a^T a}$|
|General basis $A$|$\hat{x} = (A^T A)^{-1} A^T b$|$\hat{b} = A\hat{x}$|$P = A(A^T A)^{-1}A^T$|
|Orthonormal basis $Q$|$\hat{x} = Q^T b$|$\hat{b} = QQ^T b$|$P = QQ^T$|

The orthonormal row is simply cleaner in every column.

---

## Things to Remember (for Myself)

- With an orthonormal basis, the normal equations are $I\hat{x} = Q^T b$. They solve themselves.
- Each coefficient $\hat{x}_i = q_i^T b$ depends only on $q_i$ and $b$ — not on the other basis vectors. This independence is the whole point of orthogonality.
- The projection $\hat{b} = \sum (q_i^T b) q_i$ is a sum of rank-1 projections. This sum is valid because the $q_i$ are orthogonal — the components do not overlap.
- $P = QQ^T$ only when $Q$ has orthonormal columns. If $Q$ is square, then $P = I$.
- Gram-Schmidt gives $A = QR$. The normal equations become $R\hat{x} = Q^T b$ — triangular, well-conditioned, no $A^T A$ required.
- Any time I find myself computing $A^T A$ for a projection, I should ask whether Gram-Schmidt first would make the problem cleaner.
