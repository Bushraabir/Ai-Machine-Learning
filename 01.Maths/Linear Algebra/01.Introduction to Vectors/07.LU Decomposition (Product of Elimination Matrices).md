Suppose you need to solve $Ax = b$ for 1000 different right-hand sides $b_1, b_2, \ldots, b_{1000}$ using the same matrix $A$. Running Gaussian elimination fresh each time costs $O(n^3)$ per solve — $1000 \cdot O(n^3)$ total.
>[!important] 
>LU decomposition says: **factor $A$ once** at cost $O(n^3)$, then each new right-hand side costs only $O(n^2)$.

The factorization is: $\boxed{A = LU}$ where $L$ is unit lower triangular (1s on diagonal, multipliers below) and $U$ is upper triangular (the result of elimination). Every new $Ax = b$ becomes two fast triangular solves: $LUx = b \implies \begin{cases} Ly = b & \text{forward substitution — } O(n^2) \ Ux = y & \text{backward substitution — } O(n^2) \end{cases}$
**This is why LAPACK, NumPy, and every serious numerical library uses LU — not matrix inversion — for solving linear systems.**

#### Step 1: Elimination as Matrix Multiplication
Each step of Gaussian elimination — "subtract $\ell$ times row $j$ from row $i$" — is secretly a **left-multiplication by an elimination matrix**.
##### The Elimination Matrix $E_{ij}$

The matrix that eliminates entry $(i, j)$ by subtracting $\ell_{ij}$ times row $j$ from row $i$ is:

$$\boxed{E_{ij} = I - \ell_{ij}, \mathbf{e}_i \mathbf{e}_j^T}$$

- $I$ = identity matrix
- $\mathbf{e}_i$ = $i$-th standard basis column vector (all zeros, 1 in row $i$)
- $\mathbf{e}_j^T$ = $j$-th standard basis row vector (all zeros, 1 in column $j$)
- $\ell_{ij}$ = the multiplier (chosen to zero out entry $(i,j)$)

The outer product $\mathbf{e}_i \mathbf{e}_j^T$ is a matrix of all zeros **except** a single 1 in position $(i, j)$. So $E_{ij}$ is the identity with $-\ell_{ij}$ placed at position $(i, j)$.

### Concrete 4×4 Example

The matrix eliminating entry $(3, 1)$ with multiplier $\ell_{31}$:

$$E_{31} = \begin{pmatrix} 1 & 0 & 0 & 0 \ 0 & 1 & 0 & 0 \ -\ell_{31} & 0 & 1 & 0 \ 0 & 0 & 0 & 1 \end{pmatrix}$$

When you compute $E_{31} A$:

- Rows 1, 2, 4 are **unchanged** (identity rows)
- Row 3 becomes: (Row 3 of $A$) $-\ \ell_{31} \times$ (Row 1 of $A$)

**Result:** entry $(3,1)$ becomes $a_{31} - \ell_{31} \cdot a_{11}$. If we chose $\ell_{31} = a_{31} / a_{11}$, this equals zero. Done.

### The Multiplier Formula

$$\ell_{ij} = \frac{a_{ij}^{(j)}}{a_{jj}^{(j)}}$$

The superscript $(j)$ means "at the stage when column $j$ is being processed" — the entry has already been modified by all previous column eliminations. The denominator $a_{jj}^{(j)}$ is the **pivot**. The numerator is the entry being eliminated.

> ⚠️ **Common mistake:** Multipliers use the _current_ (updated) matrix entries, not the original matrix $A$. After eliminating column 1, the entries in column 2 have changed.

### Inverse of $E_{ij}$

Just flip the sign of the off-diagonal entry:

$$E_{ij}^{-1} = I + \ell_{ij}, \mathbf{e}_i \mathbf{e}_j^T$$

Example:

$$E_{31}^{-1} = \begin{pmatrix} 1 & 0 & 0 & 0 \ 0 & 1 & 0 & 0 \ +\ell_{31} & 0 & 1 & 0 \ 0 & 0 & 0 & 1 \end{pmatrix}$$

**Why so simple?** $E_{31}$ subtracted $\ell_{31}$ times row 1 from row 3. Its inverse _adds_ it back. Undoing one elementary row operation takes exactly one elementary row operation.

**Verify:** $E_{31}^{-1} E_{31} = (I + \ell_{31}\mathbf{e}_3\mathbf{e}_1^T)(I - \ell_{31}\mathbf{e}_3\mathbf{e}_1^T) = I - \ell_{31}^2 \underbrace{(\mathbf{e}_3\mathbf{e}_1^T)(\mathbf{e}_3\mathbf{e}_1^T)}_{= \mathbf{e}_3 (\mathbf{e}_1^T \mathbf{e}_3) \mathbf{e}_1^T = 0 \text{ since } 1 \neq 3} = I$ ✓

## Step 2 — The Full Sequence of Elimination Steps

For a $4 \times 4$ matrix, the complete elimination sequence is:

| Column being eliminated           | Entries zeroed out      | Matrices applied (left-multiplied in this order) |
| --------------------------------- | ----------------------- | ------------------------------------------------ |
| Column 1 (pivot = $a_{11}$)       | $(2,1),; (3,1),; (4,1)$ | $E_{21}$, then $E_{31}$, then $E_{41}$           |
| Column 2 (pivot = $a_{22}^{(2)}$) | $(3,2),; (4,2)$         | $E_{32}$, then $E_{42}$                          |
| Column 3 (pivot = $a_{33}^{(3)}$) | $(4,3)$                 | $E_{43}$                                         |

The full elimination reads (each matrix applied left-to-right, with rightmost acting first on $A$):

$$E_{43}; E_{42}; E_{32}; E_{41}; E_{31}; E_{21}; A = U$$

To recover $A$, multiply both sides on the left by the inverses **in reverse order**:

$$A = E_{21}^{-1}; E_{31}^{-1}; E_{41}^{-1}; E_{32}^{-1}; E_{42}^{-1}; E_{43}^{-1}; U$$

Define:

$$L ;\overset{\text{def}}{=}; E_{21}^{-1}; E_{31}^{-1}; E_{41}^{-1}; E_{32}^{-1}; E_{42}^{-1}; E_{43}^{-1}$$

Then $A = LU$. The key question: **what is this product of 6 matrices?**

---

## Step 3 — The Key Miracle: Multipliers Drop In Cleanly

### The Claim

$$L = \begin{pmatrix} 1 & 0 & 0 & 0 \ \ell_{21} & 1 & 0 & 0 \ \ell_{31} & \ell_{32} & 1 & 0 \ \ell_{41} & \ell_{42} & \ell_{43} & 1 \end{pmatrix}$$

Every multiplier $\ell_{ij}$ sits **exactly in position $(i,j)$** of $L$ — as if you just placed them there by hand. There is no mixing or contamination. The product of 6 complicated matrices gives something this clean.

This is "the miracle of LU" and it's worth understanding deeply.

---

## The Proof — Why Cross Terms Vanish

### Lemma 1: Same-Column Elimination Matrices Compose Cleanly

Take two inverse elimination matrices from **the same column $j$**, operating on different rows $i \neq k$:

$$E_{ij}^{-1}; E_{kj}^{-1} = (I + \ell_{ij}\mathbf{e}_i\mathbf{e}_j^T)(I + \ell_{kj}\mathbf{e}_k\mathbf{e}_j^T)$$

Expand using FOIL:

$$= I + \ell_{ij}\mathbf{e}_i\mathbf{e}_j^T + \ell_{kj}\mathbf{e}_k\mathbf{e}_j^T + \underbrace{\ell_{ij}\ell_{kj}; \mathbf{e}_i; (\mathbf{e}_j^T \mathbf{e}_k); \mathbf{e}_j^T}_{\text{the cross term}}$$

The cross term contains $\mathbf{e}_j^T \mathbf{e}_k$. This is the dot product of two standard basis vectors: it equals 1 if $j = k$, and **0 otherwise**. Since $i, k > j$ and $i \neq k$, we have $j \neq k$, so:

$$\mathbf{e}_j^T \mathbf{e}_k = 0 \implies \text{cross term} = 0$$

Therefore:

$$E_{ij}^{-1}; E_{kj}^{-1} = I + \ell_{ij}\mathbf{e}_i\mathbf{e}_j^T + \ell_{kj}\mathbf{e}_k\mathbf{e}_j^T$$

The two rank-1 corrections **add independently** — $\ell_{ij}$ lands in position $(i,j)$ and $\ell_{kj}$ lands in position $(k,j)$ with no interaction. This extends to any number of same-column elimination matrices.

### Lemma 2: Different-Column Elimination Matrices Also Compose Cleanly

Now take two inverse elimination matrices from **different columns**: $E_{ij}^{-1}$ (column $j$) and $E_{kl}^{-1}$ (column $l$, with $l > j$):

$$(I + \ell_{ij}\mathbf{e}_i\mathbf{e}_j^T)(I + \ell_{kl}\mathbf{e}_k\mathbf{e}_l^T) = I + \ell_{ij}\mathbf{e}_i\mathbf{e}_j^T + \ell_{kl}\mathbf{e}_k\mathbf{e}_l^T + \underbrace{\ell_{ij}\ell_{kl}; \mathbf{e}_i; (\mathbf{e}_j^T \mathbf{e}_k); \mathbf{e}_l^T}_{\text{cross term}}$$

The cross term contains $\mathbf{e}_j^T \mathbf{e}_k$. Since $l > j$ and the matrix $E_{kl}^{-1}$ operates on row $k > l > j$, we have $k \neq j$, so:

$$\mathbf{e}_j^T \mathbf{e}_k = 0 \implies \text{cross term} = 0$$

Again, the multipliers drop independently into their positions.

### Putting It Together

Applying both lemmas across all columns by induction:

$$L = \prod_{\substack{i > j}} E_{ij}^{-1} = I + \sum_{\substack{i > j}} \ell_{ij}; \mathbf{e}_i \mathbf{e}_j^T$$

The sum $\sum_{i > j} \ell_{ij}; \mathbf{e}_i \mathbf{e}_j^T$ is a matrix whose $(i,j)$ entry (for $i > j$) equals $\ell_{ij}$, and all other entries are zero. Adding the identity gives 1s on the diagonal. Therefore:

$$L = \begin{pmatrix} 1 & 0 & 0 & 0 \ \ell_{21} & 1 & 0 & 0 \ \ell_{31} & \ell_{32} & 1 & 0 \ \ell_{41} & \ell_{42} & \ell_{43} & 1 \end{pmatrix} \qquad \blacksquare$$

**Summary of the key condition:** The cross terms vanish because every pair of elimination matrices $(i,j)$ and $(k,l)$ with $j \leq l$ satisfies $k > l \geq j$, so $k \neq j$, giving $\mathbf{e}_j^T \mathbf{e}_k = 0$.

## Worked Example — Full 4×4 LU Decomposition

Let:

$$A = \begin{pmatrix} 2 & 1 & 1 & 0 \ 4 & 3 & 3 & 1 \ 8 & 7 & 9 & 5 \ 6 & 7 & 9 & 8 \end{pmatrix}$$

### Step 1: Eliminate Column 1 (pivot = 2)

Compute multipliers:

$$\ell_{21} = \frac{4}{2} = 2, \qquad \ell_{31} = \frac{8}{2} = 4, \qquad \ell_{41} = \frac{6}{2} = 3$$

Apply $E_{41} E_{31} E_{21}$ to $A$ (subtract $\ell \times$ row 1 from each target row):

$$\begin{pmatrix} 2 & 1 & 1 & 0 \ 4-2(2) & 3-2(1) & 3-2(1) & 1-2(0) \ 8-4(2) & 7-4(1) & 9-4(1) & 5-4(0) \ 6-3(2) & 7-3(1) & 9-3(1) & 8-3(0) \end{pmatrix} = \begin{pmatrix} 2 & 1 & 1 & 0 \ 0 & 1 & 1 & 1 \ 0 & 3 & 5 & 5 \ 0 & 4 & 6 & 8 \end{pmatrix}$$

### Step 2: Eliminate Column 2 (pivot = 1)

Compute multipliers from the **updated** matrix:

$$\ell_{32} = \frac{3}{1} = 3, \qquad \ell_{42} = \frac{4}{1} = 4$$

Apply $E_{42} E_{32}$:

$$\begin{pmatrix} 2 & 1 & 1 & 0 \ 0 & 1 & 1 & 1 \ 0 & 3-3(1) & 5-3(1) & 5-3(1) \ 0 & 4-4(1) & 6-4(1) & 8-4(1) \end{pmatrix} = \begin{pmatrix} 2 & 1 & 1 & 0 \ 0 & 1 & 1 & 1 \ 0 & 0 & 2 & 2 \ 0 & 0 & 2 & 4 \end{pmatrix}$$

### Step 3: Eliminate Column 3 (pivot = 2)

Compute multiplier:

$$\ell_{43} = \frac{2}{2} = 1$$

Apply $E_{43}$:

$$\begin{pmatrix} 2 & 1 & 1 & 0 \ 0 & 1 & 1 & 1 \ 0 & 0 & 2 & 2 \ 0 & 0 & 2-1(2) & 4-1(2) \end{pmatrix} = \begin{pmatrix} 2 & 1 & 1 & 0 \ 0 & 1 & 1 & 1 \ 0 & 0 & 2 & 2 \ 0 & 0 & 0 & 2 \end{pmatrix} = U$$

### Reading Off $L$ and $U$

The **multipliers drop directly into $L$** in their natural positions:

$$L = \begin{pmatrix} 1 & 0 & 0 & 0 \ 2 & 1 & 0 & 0 \ 4 & 3 & 1 & 0 \ 3 & 4 & 1 & 1 \end{pmatrix}, \qquad U = \begin{pmatrix} 2 & 1 & 1 & 0 \ 0 & 1 & 1 & 1 \ 0 & 0 & 2 & 2 \ 0 & 0 & 0 & 2 \end{pmatrix}$$

### Verification

Check that $LU = A$. Let's verify entry $(3,1)$:

$$[LU]_{31} = \text{Row 3 of } L \cdot \text{Col 1 of } U = (4)(2) + (3)(0) + (1)(0) + (0)(0) = 8 \checkmark$$

Check entry $(4, 2)$:

$$[LU]_{42} = (3)(1) + (4)(1) + (1)(0) + (1)(0) = 3 + 4 = 7 \checkmark$$

This matches $A_{42} = 7$. ✓

## What Goes Wrong With Row Swaps

### The Problem

Suppose during column 2 elimination, the pivot $a_{22}^{(2)} = 0$ — we **must** swap rows to continue. Apply the permutation matrix $P_{23}$ that swaps rows 2 and 3.

After swapping, the multiplier $\ell_{31}$ — which was computed relative to the original row order — is now in row 2, not row 3. The subsequent column 2 elimination will modify entry $(2,1)$, **contaminating** the multiplier that was supposed to sit untouched in $L$.

The cross terms that vanished before **now survive**, because the permutation matrix moves entries around and breaks the condition $k \neq j$.

### The Fix: $PA = LU$

Permute $A$ **first** (choosing all row interchanges in advance via **partial pivoting**), then factor:

$$\boxed{PA = LU}$$

where:

- $P$ is the **permutation matrix** recording all row swaps
- $L$ is unit lower triangular with multipliers sitting cleanly in their positions
- $U$ is upper triangular with the pivots on the diagonal

$PA = LU$ exists for any invertible $A$. In practice, LAPACK and NumPy store $P$ implicitly as a permutation vector (pivot indices).

### Existence Without Row Swaps

$A = LU$ (without any permutation) exists if and only if **all leading principal submatrices are invertible** — equivalently, all pivots are nonzero during elimination.

Leading principal submatrices are the upper-left $k \times k$ corners: $A_{1:k, 1:k}$ for $k = 1, \ldots, n$.

## The Order of Multiplication — A Common Confusion

The elimination operations are applied as:

$$\underbrace{E_{43}}_{\text{6th}} \underbrace{E_{42}}_{\text{5th}} \underbrace{E_{32}}_{\text{4th}} \underbrace{E_{41}}_{\text{3rd}} \underbrace{E_{31}}_{\text{2nd}} \underbrace{E_{21}}_{\text{1st}} A = U$$

So $E_{21}$ acts first (rightmost position), then $E_{31}$, and so on.

To recover $A$, invert both sides and rearrange:

$$A = E_{21}^{-1} E_{31}^{-1} E_{41}^{-1} E_{32}^{-1} E_{42}^{-1} E_{43}^{-1} \cdot U = LU$$

Note: the product $E_{21}^{-1} E_{31}^{-1} \cdots$ is **not the same** as reversing the order $(E_{43} \cdots E_{21})^{-1}$. Inverting a product reverses the order:

$$(E_{43} \cdots E_{21})^{-1} = E_{21}^{-1} \cdots E_{43}^{-1}$$

This **is** the same order as what we wrote for $L$! The reversal happens automatically. And thanks to the vanishing cross terms, $L$ is clean regardless of the direction.

## Using $LU$ to Solve $Ax = b$

Once $A = LU$ is computed (cost $O(n^3)$, computed **once**):

### Forward Substitution: Solve $Ly = b$

$L$ is lower triangular, so solve from top to bottom:

$$y_1 = b_1$$ $$y_i = b_i - \sum_{j=1}^{i-1} \ell_{ij}, y_j \quad \text{for } i = 2, \ldots, n$$

### Backward Substitution: Solve $Ux = y$

$U$ is upper triangular, so solve from bottom to top:

$$x_n = y_n / u_{nn}$$ $$x_i = \left(y_i - \sum_{j=i+1}^{n} u_{ij}, x_j\right) / u_{ii} \quad \text{for } i = n-1, \ldots, 1$$

**Cost:** Each solve costs $O(n^2)$. For $k$ right-hand sides: total cost $= O(n^3) + k \cdot O(n^2)$.  
**Compare to:** Computing $A^{-1}$ explicitly costs $O(n^3)$ and then each solve is $A^{-1}b$ at $O(n^2)$ — same asymptotic cost but LU has smaller constants and better numerical stability.

## Common Mistakes and Misconceptions

**Mistake 1: Using original matrix entries for multipliers.**  
After column 1 elimination, the matrix entries change. Multiplier $\ell_{32}$ uses $a_{32}^{(2)}$ (the entry after column 1 is eliminated), not the original $a_{32}$.

**Mistake 2: Thinking $L$ might be contaminated.**  
The vanishing cross terms guarantee that $L$ stores the multipliers exactly as computed, with no modification during subsequent steps.

**Mistake 3: Confusing $E_{21}^{-1} E_{31}^{-1} \cdots$ with $(E_{21} E_{31} \cdots)^{-1}$.**  
These are different. The former is a product of individual inverses in the **same** order; the latter reverses the order. They both happen to give $L$ only because of the structure above.

**Mistake 4: Expecting $A = LU$ to always exist.**  
It fails if any pivot is zero and no row swap is available. Use $PA = LU$ for robustness.

**Mistake 5: Computing $A^{-1}$ explicitly to solve $Ax = b$.**  
Never do this numerically. Use LU. Explicit inversion loses numerical precision and is slower.

## Key Formulas at a Glance

$$E_{ij} = I - \ell_{ij}\mathbf{e}_i\mathbf{e}_j^T \qquad E_{ij}^{-1} = I + \ell_{ij}\mathbf{e}_i\mathbf{e}_j^T$$

$$\ell_{ij} = \frac{a_{ij}^{(j)}}{a_{jj}^{(j)}} \qquad \text{(ratio of entry to pivot, at current elimination stage)}$$

$$A = LU \quad \text{where} \quad L_{ij} = \begin{cases} 1 & i = j \ \ell_{ij} & i > j \ 0 & i < j \end{cases}$$

$$PA = LU \quad \text{(always exists for invertible } A\text{, used when pivoting is needed)}$$
