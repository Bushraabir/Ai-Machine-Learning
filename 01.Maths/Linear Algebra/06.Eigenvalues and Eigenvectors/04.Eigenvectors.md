
## The Point

An eigenvalue $\lambda$ is a number. An eigenvector is the thing that actually tells you what the matrix _does_ in space.

The equation is:

$$A\mathbf{x} = \lambda\mathbf{x}$$

The matrix $A$ acts on $\mathbf{x}$ and returns $\mathbf{x}$ scaled by $\lambda$. The direction of $\mathbf{x}$ is completely preserved. Every other vector gets rotated or sheared into a different direction. Eigenvectors are the exceptional ones -- they live on lines that $A$ maps to themselves.

This is the geometric core. Everything else is computation layered on top of this picture.

---

## How to Find Eigenvectors

Once you have an eigenvalue $\lambda$, finding its eigenvectors is a null space computation. Form the matrix $(A - \lambda I)$ and solve the homogeneous system:

$$(A - \lambda I)\mathbf{x} = \mathbf{0}$$

Row reduce $(A - \lambda I)$. The free variables parametrize the solution. The spanning vectors of the null space are the eigenvectors.

This is not a coincidence of method. The eigenvalue $\lambda$ was chosen precisely to make $(A - \lambda I)$ singular, which guarantees the null space is nontrivial. So a nonzero solution always exists.

> Personal note: at this stage you are just doing elimination. The only difference from a regular null space computation is that the matrix came from subtracting $\lambda I$ from $A$. Keep that straight and the computation is mechanical.

---

## The Eigenspace

The set of all solutions to $(A - \lambda I)\mathbf{x} = \mathbf{0}$, including the zero vector, is a subspace:

$$E_\lambda = \ker(A - \lambda I)$$

This is the **eigenspace** of $\lambda$. Its dimension is the **geometric multiplicity** of $\lambda$.

Every nonzero vector in $E_\lambda$ is an eigenvector for $\lambda$. The eigenspace can have dimension 1, 2, or more -- as high as the algebraic multiplicity of $\lambda$. But it is never dimension 0, because $\lambda$ was chosen to make the null space nontrivial.

---

## Eigenvectors Are Not Unique

If $\mathbf{x}$ is an eigenvector for $\lambda$, so is $c\mathbf{x}$ for any nonzero scalar $c$:

$$A(c\mathbf{x}) = c(A\mathbf{x}) = c(\lambda\mathbf{x}) = \lambda(c\mathbf{x})$$

So we are really computing a _direction_, not a specific vector. We pick a convenient representative -- usually the simplest one with integer or unit entries.

If the eigenspace has dimension $\geq 2$, then any linear combination of basis eigenvectors is again an eigenvector:

$$A(\alpha\mathbf{x}_1 + \beta\mathbf{x}_2) = \alpha\lambda\mathbf{x}_1 + \beta\lambda\mathbf{x}_2 = \lambda(\alpha\mathbf{x}_1 + \beta\mathbf{x}_2)$$

The whole eigenspace is a family of eigenvectors for the same $\lambda$.

## Worked Example: $2 \times 2$

$$A = \begin{pmatrix} 3 & 1 \ 0 & 2 \end{pmatrix}$$

Eigenvalues: $\lambda_1 = 3$, $\lambda_2 = 2$ (upper triangular, read the diagonal).

### Eigenvector for $\lambda_1 = 3$

$$A - 3I = \begin{pmatrix} 3-3 & 1 \ 0 & 2-3 \end{pmatrix} = \begin{pmatrix} 0 & 1 \ 0 & -1 \end{pmatrix}$$

Row reduce: both rows say $x_2 = 0$. The variable $x_1$ is free. Set $x_1 = 1$:

$$\mathbf{x}_1 = \begin{pmatrix} 1 \ 0 \end{pmatrix}$$

Verify: $A\mathbf{x}_1 = \begin{pmatrix} 3 & 1 \ 0 & 2 \end{pmatrix}\begin{pmatrix} 1 \ 0 \end{pmatrix} = \begin{pmatrix} 3 \ 0 \end{pmatrix} = 3\begin{pmatrix} 1 \ 0 \end{pmatrix}$. Correct.

### Eigenvector for $\lambda_2 = 2$

$$A - 2I = \begin{pmatrix} 1 & 1 \ 0 & 0 \end{pmatrix}$$

One equation: $x_1 + x_2 = 0$. The variable $x_2$ is free. Set $x_2 = -1$, then $x_1 = 1$:

$$\mathbf{x}_2 = \begin{pmatrix} 1 \ -1 \end{pmatrix}$$

Verify: $A\mathbf{x}_2 = \begin{pmatrix} 3 & 1 \ 0 & 2 \end{pmatrix}\begin{pmatrix} 1 \ -1 \end{pmatrix} = \begin{pmatrix} 2 \ -2 \end{pmatrix} = 2\begin{pmatrix} 1 \ -1 \end{pmatrix}$. Correct.

## Worked Example: $3 \times 3$

$$A = \begin{pmatrix} 4 & 1 & 0 \ 0 & 4 & 0 \ 0 & 0 & 3 \end{pmatrix}$$

### Eigenvalues

Upper triangular: $\lambda_1 = 4$ (algebraic multiplicity 2), $\lambda_2 = 3$ (multiplicity 1).

### Eigenvector for $\lambda_2 = 3$

$$A - 3I = \begin{pmatrix} 1 & 1 & 0 \ 0 & 1 & 0 \ 0 & 0 & 0 \end{pmatrix}$$

Row 2: $x_2 = 0$. Row 1: $x_1 + x_2 = 0 \Rightarrow x_1 = 0$. Variable $x_3$ is free. Set $x_3 = 1$:

$$\mathbf{x}_2 = \begin{pmatrix} 0 \ 0 \ 1 \end{pmatrix}$$

### Eigenvectors for $\lambda_1 = 4$

$$A - 4I = \begin{pmatrix} 0 & 1 & 0 \ 0 & 0 & 0 \ 0 & 0 & -1 \end{pmatrix}$$

Row 1: $x_2 = 0$. Row 3: $x_3 = 0$. Variable $x_1$ is free. Set $x_1 = 1$:

$$\mathbf{x}_1 = \begin{pmatrix} 1 \ 0 \ 0 \end{pmatrix}$$

The eigenspace $E_4$ has dimension 1, even though $\lambda = 4$ has algebraic multiplicity 2. Geometric multiplicity 1 $<$ algebraic multiplicity 2. The eigenvalue is **defective**. $A$ is not diagonalizable.

This is the critical distinction. The number of linearly independent eigenvectors for $\lambda = 4$ is only 1, not 2. You cannot find a second one. If you try to force it, you get a generalized eigenvector -- that is the Jordan form story.

---

## Worked Example: Repeated Eigenvalue, Full Eigenspace

Compare with:

$$B = \begin{pmatrix} 4 & 0 & 0 \ 0 & 4 & 0 \ 0 & 0 & 3 \end{pmatrix}$$

Same eigenvalues. But now:

$$B - 4I = \begin{pmatrix} 0 & 0 & 0 \ 0 & 0 & 0 \ 0 & 0 & -1 \end{pmatrix}$$

Row 3: $x_3 = 0$. Variables $x_1$ and $x_2$ are both free. The eigenspace is:

$$E_4 = \text{span}\left{ \begin{pmatrix} 1 \ 0 \ 0 \end{pmatrix},, \begin{pmatrix} 0 \ 1 \ 0 \end{pmatrix} \right}$$

Geometric multiplicity 2 $=$ algebraic multiplicity 2. Not defective. $B$ is diagonalizable. The difference from $A$ above is the off-diagonal 1 in position $(1,2)$. That single entry collapses the eigenspace from dimension 2 to dimension 1.

## Linear Independence of Eigenvectors from Distinct Eigenvalues

**Theorem.** Eigenvectors corresponding to distinct eigenvalues are linearly independent.

_Proof._ Let $\lambda_1 \neq \lambda_2$ with eigenvectors $\mathbf{x}_1, \mathbf{x}_2$. Suppose

$$\alpha_1 \mathbf{x}_1 + \alpha_2 \mathbf{x}_2 = \mathbf{0} \tag{1}$$

Apply $A$ to both sides:

$$\alpha_1 \lambda_1 \mathbf{x}_1 + \alpha_2 \lambda_2 \mathbf{x}_2 = \mathbf{0} \tag{2}$$

Subtract $\lambda_2 \times (1)$ from $(2)$:

$$\alpha_1 (\lambda_1 - \lambda_2)\mathbf{x}_1 = \mathbf{0}$$

Since $\lambda_1 \neq \lambda_2$ and $\mathbf{x}_1 \neq \mathbf{0}$, we get $\alpha_1 = 0$. Substituting back into $(1)$ gives $\alpha_2 = 0$.

The argument extends to any finite collection of eigenvectors with distinct eigenvalues by induction. This theorem is the engine behind diagonalization.

## What Can Go Wrong

### The eigenspace is smaller than expected

Algebraic multiplicity 2 does not guarantee geometric multiplicity 2. Always row reduce $(A - \lambda I)$ and count the free variables. Do not assume.

### Eigenvectors from different eigenspaces look parallel

They cannot be. The theorem above guarantees linear independence across distinct eigenvalues. If your computation gives parallel vectors for different $\lambda$, there is an arithmetic error.

### Dividing when you should not

When row reducing $(A - \lambda I)$, the entries involve $\lambda$ as a specific number. Substitute $\lambda$ first, then row reduce numerically. Do not row reduce symbolically and then substitute.

### Confusing the eigenvector with the eigenvalue

$\lambda$ is a scalar. $\mathbf{x}$ is a vector. $A\mathbf{x} = \lambda\mathbf{x}$ means the matrix scales the vector. The scaled vector $\lambda\mathbf{x}$ is not a new object -- it is $\mathbf{x}$ multiplied by a number.

---

## The Full Picture

For an $n \times n$ matrix $A$:

1. Compute $\det(A - \lambda I) = 0$ to find eigenvalues $\lambda_1, \ldots, \lambda_k$ with algebraic multiplicities $m_a(\lambda_i)$.
    
2. For each $\lambda_i$, row reduce $(A - \lambda_i I)$ and find a basis for $\ker(A - \lambda_i I)$. This gives $m_g(\lambda_i) = \dim E_{\lambda_i}$ linearly independent eigenvectors.
    
3. The total number of linearly independent eigenvectors you collect across all eigenvalues is $\sum_i m_g(\lambda_i)$.
    
4. If this sum equals $n$, those eigenvectors form a basis for $\mathbb{R}^n$ and $A$ is diagonalizable. If any $m_g(\lambda_i) < m_a(\lambda_i)$, the sum is less than $n$ and $A$ is not diagonalizable.
    

The eigenvectors are the columns of the matrix $P$ in $A = PDP^{-1}$. The eigenvalues are the diagonal entries of $D$. That is what diagonalization is -- the eigenvectors tell you the right basis, and in that basis $A$ is just scaling.

---

## Summary

$$\boxed{(A - \lambda I)\mathbf{x} = \mathbf{0}, \qquad \mathbf{x} \neq \mathbf{0}}$$

Find $\lambda$ from the characteristic equation. Substitute into $(A - \lambda I)$. Row reduce. Read off the free variables. Write the eigenvectors as explicit vectors.

Always verify: multiply $A\mathbf{x}$ and check you get $\lambda\mathbf{x}$.

Always check: geometric multiplicity versus algebraic multiplicity. This is what determines whether the matrix is diagonalizable.
