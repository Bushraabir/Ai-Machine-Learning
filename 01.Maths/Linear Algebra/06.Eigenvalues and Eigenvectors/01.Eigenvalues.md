## The Central Question

Most of what a matrix does is complicated. It takes a vector and sends it somewhere else, rotates it, shears it, mixes its components. But there is a special class of vectors for which the action of the matrix is almost trivial -- the vector simply gets scaled. It points in the same direction before and after multiplication. These are the eigenvectors, and the scale factors are the eigenvalues.

This is not a small observation. The entire spectral theory of matrices, differential equations, Markov chains, and quantum mechanics rests on this.

---

## Definition

Let $A$ be an $n \times n$ matrix. A scalar $\lambda \in \mathbb{C}$ is called an **eigenvalue** of $A$ if there exists a nonzero vector $\mathbf{x} \in \mathbb{C}^n$ such that

$$A\mathbf{x} = \lambda \mathbf{x}$$

The vector $\mathbf{x}$ is called the **eigenvector** associated to $\lambda$.

The requirement $\mathbf{x} \neq \mathbf{0}$ is not cosmetic. Every scalar $\lambda$ satisfies $A\mathbf{0} = \lambda \mathbf{0}$, so we must exclude the zero vector to get a meaningful definition.

---

## How to Find Eigenvalues: The Characteristic Equation

Rewrite $A\mathbf{x} = \lambda\mathbf{x}$ as:

$$A\mathbf{x} - \lambda\mathbf{x} = \mathbf{0}$$ $$(A - \lambda I)\mathbf{x} = \mathbf{0}$$

For a nonzero solution $\mathbf{x}$ to exist, the matrix $(A - \lambda I)$ must be **singular**, i.e., it must fail to be invertible. The condition for this is:

$$\det(A - \lambda I) = 0$$

This is the **characteristic equation** of $A$. Expanding the determinant gives a degree-$n$ polynomial in $\lambda$, called the **characteristic polynomial**. Its roots are the eigenvalues.

> Personal note: Strang emphasizes this beautifully -- the eigenvalue equation forces us to think about when a matrix fails to be invertible, which is exactly when its null space is nontrivial. The eigenvalue $\lambda$ is special precisely because it makes $(A - \lambda I)$ singular.

---

## Worked Example

Let

$$A = \begin{pmatrix} 3 & 1 \ 0 & 2 \end{pmatrix}$$

### Step 1: Form $(A - \lambda I)$

$$A - \lambda I = \begin{pmatrix} 3 & 1 \ 0 & 2 \end{pmatrix} - \lambda \begin{pmatrix} 1 & 0 \ 0 & 1 \end{pmatrix} = \begin{pmatrix} 3 - \lambda & 1 \ 0 & 2 - \lambda \end{pmatrix}$$

### Step 2: Characteristic Equation

$$\det(A - \lambda I) = (3 - \lambda)(2 - \lambda) - (1)(0) = (3 - \lambda)(2 - \lambda) = 0$$

So $\lambda_1 = 3$ and $\lambda_2 = 2$.

### Step 3: Eigenvectors

**For $\lambda_1 = 3$:**

$$(A - 3I)\mathbf{x} = \begin{pmatrix} 0 & 1 \ 0 & -1 \end{pmatrix}\begin{pmatrix} x_1 \ x_2 \end{pmatrix} = \begin{pmatrix} 0 \ 0 \end{pmatrix}$$

Both rows give $x_2 = 0$, so $x_1$ is free. The eigenvector is $\mathbf{x}_1 = \begin{pmatrix} 1 \ 0 \end{pmatrix}$.

**For $\lambda_2 = 2$:**

$$(A - 2I)\mathbf{x} = \begin{pmatrix} 1 & 1 \ 0 & 0 \end{pmatrix}\begin{pmatrix} x_1 \ x_2 \end{pmatrix} = \begin{pmatrix} 0 \ 0 \end{pmatrix}$$

The first row gives $x_1 + x_2 = 0$, so $x_2$ is free. Set $x_2 = -1$, $x_1 = 1$. The eigenvector is $\mathbf{x}_2 = \begin{pmatrix} 1 \ -1 \end{pmatrix}$.

Notice: $A$ is upper triangular, and the eigenvalues read directly off the diagonal. This is always true for triangular matrices.

## A Larger Example: Full $3 \times 3$

Let

$$A = \begin{pmatrix} 2 & 1 & 0 \ 0 & 2 & 0 \ 0 & 0 & 5 \end{pmatrix}$$

### Characteristic Polynomial

$$A - \lambda I = \begin{pmatrix} 2-\lambda & 1 & 0 \ 0 & 2-\lambda & 0 \ 0 & 0 & 5-\lambda \end{pmatrix}$$

Since this is upper triangular, the determinant is the product of the diagonal entries:

$$\det(A - \lambda I) = (2-\lambda)^2(5-\lambda) = 0$$

The eigenvalues are $\lambda = 2$ (with **algebraic multiplicity** 2) and $\lambda = 5$ (multiplicity 1).

### Eigenvectors for $\lambda = 2$

$$(A - 2I) = \begin{pmatrix} 0 & 1 & 0 \ 0 & 0 & 0 \ 0 & 0 & 3 \end{pmatrix}$$

Row 1 forces $x_2 = 0$. Row 3 forces $x_3 = 0$. $x_1$ is free. So the eigenspace is:

$$E_2 = \text{span}\left{ \begin{pmatrix} 1 \ 0 \ 0 \end{pmatrix} \right}$$

The **geometric multiplicity** of $\lambda = 2$ is 1, while its algebraic multiplicity is 2. This is a defective eigenvalue. $A$ is not diagonalizable.

### Eigenvectors for $\lambda = 5$

$$(A - 5I) = \begin{pmatrix} -3 & 1 & 0 \ 0 & -3 & 0 \ 0 & 0 & 0 \end{pmatrix}$$

Row 2: $-3x_2 = 0 \Rightarrow x_2 = 0$. Row 1: $-3x_1 + x_2 = 0 \Rightarrow x_1 = 0$. $x_3$ is free.

$$E_5 = \text{span}\left{ \begin{pmatrix} 0 \ 0 \ 1 \end{pmatrix} \right}$$

## Key Properties

### Trace and Determinant

For an $n \times n$ matrix $A$ with eigenvalues $\lambda_1, \ldots, \lambda_n$ (counted with multiplicity):

$$\text{tr}(A) = \lambda_1 + \lambda_2 + \cdots + \lambda_n$$ $$\det(A) = \lambda_1 \cdot \lambda_2 \cdots \lambda_n$$

These follow directly from Vieta's formulas applied to the characteristic polynomial. This gives an immediate sanity check on any eigenvalue computation.

For the $2 \times 2$ example above: $\text{tr}(A) = 3 + 2 = 5$ and $\det(A) = 6 = 3 \cdot 2$. Checks out.

### Eigenvalues of Special Matrices

|Matrix type|What to expect|
|---|---|
|Symmetric ($A = A^T$)|All eigenvalues are real|
|Positive definite|All eigenvalues are strictly positive|
|Orthogonal|All eigenvalues have modulus 1|
|Triangular|Eigenvalues are the diagonal entries|
|Singular|At least one eigenvalue is 0|

The last one is worth internalizing: $A$ is singular if and only if 0 is an eigenvalue, because $A\mathbf{x} = \mathbf{0} = 0 \cdot \mathbf{x}$ for some nonzero $\mathbf{x}$.

### Linear Independence of Eigenvectors

**Theorem.** Eigenvectors corresponding to _distinct_ eigenvalues are linearly independent.

_Proof sketch._ Suppose $\lambda_1 \neq \lambda_2$ and $\alpha_1 \mathbf{x}_1 + \alpha_2 \mathbf{x}_2 = \mathbf{0}$. Apply $(A - \lambda_2 I)$ to both sides:

$$\alpha_1 (\lambda_1 - \lambda_2)\mathbf{x}_1 = \mathbf{0}$$

Since $\lambda_1 \neq \lambda_2$ and $\mathbf{x}_1 \neq \mathbf{0}$, we get $\alpha_1 = 0$. Then $\alpha_2 \mathbf{x}_2 = \mathbf{0}$ forces $\alpha_2 = 0$. The general case follows by induction.

This is the theorem that makes diagonalization possible.

## Algebraic vs Geometric Multiplicity

For an eigenvalue $\lambda$:

- **Algebraic multiplicity** $m_a(\lambda)$: the multiplicity of $\lambda$ as a root of $\det(A - \lambda I) = 0$.
- **Geometric multiplicity** $m_g(\lambda)$: the dimension of the eigenspace $\ker(A - \lambda I)$.

Always: $$1 \leq m_g(\lambda) \leq m_a(\lambda)$$

When $m_g(\lambda) < m_a(\lambda)$, the eigenvalue is called **defective**. A matrix with a defective eigenvalue is not diagonalizable. This is where Jordan form enters, but that is a separate note.

---

## The Eigenspace

The set of all eigenvectors for a given $\lambda$, together with the zero vector, forms a subspace:

$$E_\lambda = \ker(A - \lambda I) = {\mathbf{x} : A\mathbf{x} = \lambda \mathbf{x}}$$

This is the **eigenspace** of $\lambda$. Its dimension is the geometric multiplicity. Computing it is just null space computation -- row reduce $(A - \lambda I)$ and find the free variables.

---

## Why This Matters

The reason eigenvalues are so central is that they decouple a system. If $A$ is diagonalizable with $A = PDP^{-1}$, then

$$A^k = PD^kP^{-1}$$

and $D^k$ is trivial to compute -- just raise each diagonal entry to the $k$-th power. This turns problems about matrix powers (Markov chains, recurrences, stability of ODE systems) into scalar problems. The eigenvectors define a basis in which $A$ acts as pure scaling.

Every time I see a system $\mathbf{u}' = A\mathbf{u}$, the solution is $e^{At}\mathbf{u}_0$, and computing $e^{At}$ reduces to knowing the eigenvalues and eigenvectors. That is the payoff.

---

## Things to Not Confuse

- The eigenvalue equation is $A\mathbf{x} = \lambda\mathbf{x}$, not $\mathbf{x}A = \lambda\mathbf{x}$. Left multiplication matters.
- An eigenvector is always nonzero by definition. The eigenspace as a set includes zero, but no eigenvector is zero.
- Different eigenvectors can share the same eigenvalue -- that is exactly what multiplicity $> 1$ means.
- Eigenvalues of $A^T$ are the same as eigenvalues of $A$ (same characteristic polynomial), but the eigenvectors are generally different.

---

## Summary

$$\boxed{\det(A - \lambda I) = 0}$$

This one equation governs everything. Solve it for $\lambda$, then for each $\lambda$ solve $(A - \lambda I)\mathbf{x} = \mathbf{0}$ to get the eigenvectors. Check your answers with trace and determinant. Notice whether algebraic and geometric multiplicities agree. Everything else -- diagonalization, matrix exponentials, PCA, spectral graph theory -- builds on this foundation.
