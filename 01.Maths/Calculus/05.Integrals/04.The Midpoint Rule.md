## 0. Motivation: Why Not Just Use Left or Right Endpoints?

From [[The Definite Integral]], we know:

$$\int_a^b f(x), dx = \lim_{n\to\infty} \sum_{i=1}^n f(x_i^*), \Delta x$$

The choice of sample point $x_i^*$ is free. Left endpoints ($L_n$) and right endpoints ($R_n$) are the obvious first choices — but they are **systematically biased**.

For an increasing function:

- $L_n$ always **underestimates** (samples the low end of each strip)
- $R_n$ always **overestimates** (samples the high end)

The errors do not cancel within each strip; they only cancel when averaged: $T_n = (L_n + R_n)/2$ — the Trapezoid Rule.

But there is a smarter single-point choice: the **midpoint** $\bar{x}_i = \dfrac{x_{i-1} + x_i}{2}$.

The Midpoint Rule is not just "splitting the difference." Its superiority over $L_n$ and $R_n$ has a precise algebraic reason — and it is **second-order accurate**, not first-order. Understanding why is the point of this note.

---

## 1. The Rule: Definition

**Definition (Midpoint Rule — Stewart §7.7).** Let $[a,b]$ be partitioned into $n$ equal subintervals of width $\Delta x = (b-a)/n$. The midpoint of the $i$-th subinterval $[x_{i-1}, x_i]$ is:

$$\bar{x}_i = \frac{x_{i-1} + x_i}{2} = a + \left(i - \frac{1}{2}\right)\Delta x$$

The **Midpoint Rule approximation** is:

$$\boxed{M_n = \sum_{i=1}^n f(\bar{x}_i), \Delta x = \Delta x\left[f(\bar{x}_1) + f(\bar{x}_2) + \cdots + f(\bar{x}_n)\right]}$$

Geometrically: erect a rectangle of height $f(\bar{x}_i)$ over each subinterval. Sum the areas.

---

## 2. Geometric Interpretation

```
f(x)
  |
  |        ___________
  |       |     *     |        ← height = f(x̄ᵢ) at midpoint
  |       |   *   *   |
  |       |  *     *  |
  |       | *       * |
  |_______|___________|______> x
         x_{i-1}  x_i
              ↑
             x̄ᵢ (midpoint)
```

The rectangle's height is set at the **exact center** of the interval. For a curve that is concave up (like $x^2$), the midpoint rectangle slices through the curve — part of the rectangle pokes above the curve, part sits below. These local errors partially cancel.

Contrast with $L_n$:

```
Left endpoint:             Midpoint:
  |                          |
  |**                        |    **
  |  **                      |  **  **
  |    **                    |**      **
  |____**____                |__________
  x₀        x₁              x̄ᵢ
  [systematically low]    [errors partially cancel]
```

## 3. Worked Example from First Principles

**Compute** $\displaystyle\int_1^2 \frac{1}{x}, dx$ using $M_5$.

**Setup:** $[a,b] = [1,2]$, $n = 5$, $\Delta x = 1/5 = 0.2$.

Midpoints: $\bar{x}_i = 1 + (i - 1/2)(0.2)$ for $i = 1, \ldots, 5$.

|$i$|$\bar{x}_i$|$f(\bar{x}_i) = 1/\bar{x}_i$|
|---|---|---|
|1|1.1|0.909091|
|2|1.3|0.769231|
|3|1.5|0.666667|
|4|1.7|0.588235|
|5|1.9|0.526316|
||**Sum**|**3.459540**|

$$M_5 = 0.2 \times 3.459540 = 0.691908$$

True value: $\displaystyle\int_1^2 \frac{1}{x}, dx = \ln 2 \approx 0.693147$.

Error: $|0.693147 - 0.691908| = 0.001239$.

Compare with $n = 5$ Trapezoid Rule on the same problem:

$$T_5 \approx 0.695635, \quad \text{error} = 0.002488$$

The Midpoint Rule error is **about half** the Trapezoid error. This is not a coincidence — it is a theorem (see §5).

## 4. Another Example: $\int_0^1 e^{x^2}, dx$

This integral has no elementary antiderivative. We **must** use numerical methods.

With $n = 4$, $\Delta x = 0.25$, midpoints: $0.125, 0.375, 0.625, 0.875$.

|$\bar{x}_i$|$e^{\bar{x}_i^2}$|
|---|---|
|0.125|$e^{0.015625} \approx 1.01575$|
|0.375|$e^{0.140625} \approx 1.15091$|
|0.625|$e^{0.390625} \approx 1.47784$|
|0.875|$e^{0.765625} \approx 2.15027$|
|**Sum**|**5.79477**|

$$M_4 = 0.25 \times 5.79477 \approx 1.44869$$

True value $\approx 1.46265$. Error $\approx 0.01396$.

> This is the kind of integral that appears constantly in probability theory (related to the normal distribution). Numerical integration is not a fallback — it is the standard tool.

## 5. Error Analysis: Why Midpoint is Better Than Trapezoid

This is the heart of the note. The claim:

$$|E_M| \leq \frac{(b-a)^3}{24n^2} K, \quad |E_T| \leq \frac{(b-a)^3}{12n^2} K, \quad K = \max_{a \leq x \leq b} |f''(x)|$$

So $|E_M| \leq \frac{1}{2}|E_T|$. The Midpoint Rule is **twice as accurate** as the Trapezoid Rule, at the same cost (both use $n$ function evaluations).

### 5.1 Where the Error Formula Comes From

**Proof (one subinterval, then scale).** On a single subinterval $[x_{i-1}, x_i]$ with midpoint $\bar{x} = (x_{i-1}+x_i)/2$ and half-width $h = \Delta x / 2$, expand $f$ in a Taylor series about $\bar{x}$:

$$f(x) = f(\bar{x}) + f'(\bar{x})(x - \bar{x}) + \frac{f''(\bar{x})}{2}(x-\bar{x})^2 + \frac{f'''(\bar{x})}{6}(x-\bar{x})^3 + O((x-\bar{x})^4)$$

Integrate over $[x_{i-1}, x_i] = [\bar{x}-h, \bar{x}+h]$:

$$\int_{\bar{x}-h}^{\bar{x}+h} f(x), dx = 2h, f(\bar{x}) + f'(\bar{x})\underbrace{\int_{-h}^{h} u, du}_{=,0} + \frac{f''(\bar{x})}{2}\underbrace{\int_{-h}^{h} u^2, du}_{=,2h^3/3} + \frac{f'''(\bar{x})}{6}\underbrace{\int_{-h}^{h} u^3, du}_{=,0} + \cdots$$

(The odd-power integrals vanish by symmetry of $[\bar{x}-h, \bar{x}+h]$.)

$$= 2h, f(\bar{x}) + \frac{f''(\bar{x})}{3}h^3 + O(h^5)$$

The Midpoint Rule approximation for this strip is $f(\bar{x}) \cdot 2h$. So the **local error** is:

$$e_i = \int_{\bar{x}-h}^{\bar{x}+h} f, dx - f(\bar{x}) \cdot 2h = \frac{f''(\bar{x})}{3}h^3 + O(h^5)$$

Since $h = \Delta x / 2 = (b-a)/(2n)$:

$$|e_i| \leq \frac{K}{3} \cdot \frac{(b-a)^3}{8n^3}$$

Summing over $n$ subintervals:

$$|E_M| = \left|\sum_{i=1}^n e_i\right| \leq n \cdot \frac{K}{3} \cdot \frac{(b-a)^3}{8n^3} = \frac{K(b-a)^3}{24n^2}$$

$\blacksquare$

### 5.2 Why the First-Order Error Vanishes: The Symmetry Argument

The key insight: expanding about the **midpoint** makes $\int_{-h}^{h} u, du = 0$ by **odd symmetry**. The first-order error term cancels **exactly** — not approximately.

For left/right endpoints, there is no such symmetry. Expanding about $x_{i-1}$ (left endpoint):

$$\int_{x_{i-1}}^{x_i} f, dx = f(x_{i-1})\Delta x + \frac{f'(x_{i-1})}{2}(\Delta x)^2 + O((\Delta x)^3)$$

Local error $\sim C(\Delta x)^2$, total error $\sim C(b-a)\Delta x = O(1/n)$. First order.

For midpoint: local error $\sim C(\Delta x)^3$, total error $\sim Cn \cdot (\Delta x)^3 = O(1/n^2)$. **Second order.**

> The midpoint is special precisely because it is the center of symmetry of the interval. Symmetry kills the linear error term. This is the same reason Gaussian quadrature is so powerful — it chooses nodes to kill even higher-order error terms.

## 6. Comparison of Methods

|Method|Error Order|Formula|Notes|
|---|---|---|---|
|Left $L_n$|$O(1/n)$|$\Delta x\sum f(x_{i-1})$|Biased for monotone $f$|
|Right $R_n$|$O(1/n)$|$\Delta x\sum f(x_i)$|Biased opposite direction|
|Midpoint $M_n$|$O(1/n^2)$|$\Delta x\sum f(\bar{x}_i)$|Best single-point rule|
|Trapezoid $T_n$|$O(1/n^2)$|$\frac{\Delta x}{2}[f(x_0)+2f(x_1)+\cdots+f(x_n)]$|$n+1$ evaluations|
|Simpson $S_n$|$O(1/n^4)$|$\frac{\Delta x}{3}[f(x_0)+4f(x_1)+2f(x_2)+\cdots+f(x_n)]$|Combines $M$ and $T$|

Observe: **Simpson's Rule = $\frac{2}{3}M_n + \frac{1}{3}T_n$**. This is not a coincidence — it is a **Richardson extrapolation**, combining two second-order methods to cancel their leading error terms and achieve fourth order.

### Error Bounds for the Example $\int_1^2 1/x, dx$

$f(x) = 1/x$, $f''(x) = 2/x^3$, max on $[1,2]$: $K = f''(1) = 2$.

$$|E_M| \leq \frac{(2-1)^3}{24(5)^2}(2) = \frac{2}{600} = 0.00\overline{3}$$

Actual error was $0.001239 \leq 0.00\overline{3}$. $\checkmark$

$$|E_T| \leq \frac{(2-1)^3}{12(5)^2}(2) = \frac{2}{300} = 0.00\overline{6}$$

The bound confirms: $|E_M| \leq \frac{1}{2}|E_T|$.

## 7. Choosing $n$: Solving for Required Accuracy

**Problem.** How large must $n$ be to guarantee $|E_M| \leq 10^{-4}$ for $\int_1^2 \frac{1}{x}, dx$?

Use the error bound $|E_M| \leq \frac{K(b-a)^3}{24n^2}$ with $K = 2$, $b-a = 1$:

$$\frac{2}{24n^2} \leq 10^{-4} \implies n^2 \geq \frac{2}{24 \times 10^{-4}} = \frac{2}{0.0024} \approx 833.3$$

$$n \geq \sqrt{833.3} \approx 28.9 \implies \boxed{n = 29}$$

For the Trapezoid Rule with the same tolerance: $n \geq \sqrt{1666.7} \approx 41$. The Midpoint Rule reaches the same accuracy with **29% fewer function evaluations**.

## 8. Visualization: Error Cancellation in Action

```
Concave-up curve (f'' > 0), one subinterval [x_{i-1}, x_i]:

         true curve
            /‾‾‾\
           /     \
  ________/  (+)  \________
  |       \       /        |
  |    (–) \_____/ (–)     |   ← Midpoint rectangle
  |_________________________|
  x_{i-1}      x̄ᵢ       xᵢ

  (+) region: rectangle ABOVE curve (overestimates here)
  (–) region: rectangle BELOW curve (underestimates here)

  For concave-up f: the (–) regions at the tails are EXACTLY
  equal by symmetry → first-order errors cancel.
  Residual error is third-order in h.
```

For a **concave-up** function ($f'' > 0$):

- Midpoint Rule **underestimates** (the rectangle is below the chord)
- Trapezoid Rule **overestimates** (the trapezoid is above the curve)

This means $M_n \leq \int_a^b f \leq T_n$ when $f$ is concave up — giving a **guaranteed enclosure** of the true value.

## 9. Connection to AI / Machine Learning / Deep Learning

### 9.1 Numerical Integration in Probabilistic Models

In Bayesian deep learning and probabilistic ML, we regularly need:

$$\mathbb{E}_{z \sim p(z)}[f(z)] = \int f(z), p(z), dz$$

When $p$ is Gaussian and $f$ is a neural network output, this is intractable analytically. The **Gauss-Hermite quadrature** (a higher-order analog of the Midpoint Rule optimized for Gaussian measure) is used to approximate it. Understanding why the midpoint outperforms endpoints — symmetry cancels error — is the conceptual foundation of quadrature theory.

### 9.2 The Midpoint Method as a Time Integrator

In continuous-time RL and Neural ODEs, we integrate the ODE:

$$\dot{x} = f(x, t), \quad x(0) = x_0$$

The **explicit midpoint method** (a Runge-Kutta 2 scheme):

$$k_1 = f(x_n, t_n)$$ $$k_2 = f!\left(x_n + \frac{h}{2}k_1,; t_n + \frac{h}{2}\right)$$ $$x_{n+1} = x_n + h, k_2$$

This is exactly the Midpoint Rule applied to the integral $x(t_{n+1}) = x_n + \int_{t_n}^{t_{n+1}} f(x(t), t), dt$. Evaluating $f$ at the estimated midpoint of the time step — rather than the left endpoint (Euler's method) — doubles the convergence order from $O(h)$ to $O(h^2)$. Neural ODE solvers (like `torchdiffeq`) use exactly this principle, extended to higher-order Runge-Kutta schemes.

### 9.3 Mini-Batch Gradient Estimation as Midpoint Sampling

When we estimate $\nabla_\theta R(\theta) = \int \nabla_\theta \ell, d\mathcal{D}$ using a mini-batch, the **choice of which data points to include** is analogous to the choice of quadrature node.

- Random sampling (SGD) $\approx$ random Riemann sum (high variance)
- **Stratified sampling**: divide the dataset into strata and sample one from each — this is conceptually the Midpoint Rule (sample the "center" of each stratum)
- **Quasi-random sequences** (Halton, Sobol) place sample points more uniformly, achieving $O(1/n^2)$ or better convergence — exactly the gain from midpoint vs. left endpoint

The variance reduction in **stratified sampling** for gradient estimation is the direct ML analog of the accuracy gain in the Midpoint Rule.

### 9.4 Symmetry in Initialization

The reason the midpoint is optimal on $[x_{i-1}, x_i]$ is the same reason **symmetric weight initialization** is preferred in neural networks. Kaiming and Xavier initializations are designed so that the forward and backward passes have symmetric variance — odd-order moments cancel, just as the $\int u, du = 0$ term vanished in our Taylor expansion. The underlying mathematical principle is identical: **exploit symmetry to cancel leading-order error terms.**

### Summary Table

|Midpoint Rule Concept|ML/DL Analog|
|---|---|
|Midpoint $\bar{x}_i$ = center of interval|Stratified mini-batch center|
|Symmetry kills linear error term|Symmetric initialization kills gradient bias|
|$O(1/n^2)$ vs. $O(1/n)$ for $L_n$, $R_n$|Variance reduction from stratification|
|Explicit midpoint ODE integrator|Runge-Kutta-2 in Neural ODE solvers|

## 10. Quick-Reference Cheatsheet

$$M_n = \Delta x \sum_{i=1}^n f(\bar{x}_i), \quad \bar{x}_i = a + \left(i - \tfrac{1}{2}\right)\Delta x, \quad \Delta x = \frac{b-a}{n}$$

$$|E_M| \leq \frac{K(b-a)^3}{24n^2}, \quad K = \max_{[a,b]}|f''(x)|$$

$$|E_T| \leq \frac{K(b-a)^3}{12n^2} = 2|E_M|_{\max}$$

$$\text{For required tolerance } \varepsilon: \quad n \geq \sqrt{\frac{K(b-a)^3}{24\varepsilon}}$$

$$\text{Key identity: } S_{2n} = \frac{2}{3}M_n + \frac{1}{3}T_n \quad \text{(Simpson from Midpoint + Trapezoid)}$$
