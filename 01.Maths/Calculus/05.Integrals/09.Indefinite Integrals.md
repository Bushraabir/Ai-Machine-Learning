

> _"The indefinite integral is not a number, not a function — it is a family. The $+C$ is not sloppiness or convention. It is the precise record of what differentiation destroyed and integration cannot recover without additional information."_

## 0. From Definite to Indefinite

From [[Fundamental Theorem of Calculus]], we know:

$$\int_a^b f(x), dx = F(b) - F(a), \quad \text{where } F' = f$$

The definite integral is a **number**. It depends on both $a$ and $b$.

Now strip away the bounds entirely. Ask not "what is the accumulated area?" but simply:

> **What function, when differentiated, gives $f(x)$?**

This is the **antidifferentiation problem**, and its answer is the **indefinite integral**:

$$\int f(x), dx = F(x) + C$$

where $F'(x) = f(x)$ and $C \in \mathbb{R}$ is an arbitrary constant.

The indefinite integral is not a single function. It is the **complete family of antiderivatives** of $f$ — one for each value of $C$.

---

## 1. Definition and Notation

**Definition.** If $F'(x) = f(x)$, then $F$ is an **antiderivative** of $f$. The **indefinite integral** of $f$ is:

$$\int f(x), dx = F(x) + C, \qquad C \in \mathbb{R}$$

### Notation Decoded

$$\underbrace{\int}_{\text{integral sign}} \underbrace{f(x)}_{\text{integrand}} \underbrace{dx}_{\text{``with respect to } x\text{''}} = \underbrace{F(x) + C}_{\text{family of antiderivatives}}$$

The $dx$ specifies the **variable of integration** — essential when the integrand involves multiple symbols.

The integral sign $\int$ is an elongated S, from the Latin _summa_ — a reminder that the definite integral is a sum, and the indefinite integral is the tool that evaluates it.

### Why $+C$ is Necessary and Sufficient

**Necessary:** If $F$ is one antiderivative of $f$, then $G = F + C$ is another, since $G' = F' = f$. Without $+C$, we omit infinitely many valid answers.

**Sufficient:** If $F$ and $G$ are both antiderivatives of $f$ on an interval, then $(F - G)' = 0$, so by the **Mean Value Theorem**, $F - G = C$ (constant). Every antiderivative is captured by $F + C$ for some $C$.

**Proof.** Let $H = F - G$. Then $H' = 0$ on $(a,b)$. Pick any $x \in (a,b)$. By MVT applied to $[a,x]$: $H(x) - H(a) = H'(c)(x-a) = 0$. So $H(x) = H(a)$ for all $x$ — $H$ is constant. $\blacksquare$

The $+C$ is not vagueness. It is the exact parameterization of all solutions.

## 2. The Table of Standard Indefinite Integrals

Every differentiation formula, read backwards, gives an integration formula. These are derived directly from the known derivatives.

### Power Functions

$$\int x^n, dx = \frac{x^{n+1}}{n+1} + C, \qquad n \neq -1$$

$$\int x^{-1}, dx = \int \frac{1}{x}, dx = \ln|x| + C$$

$$\int \sqrt{x}, dx = \frac{2}{3}x^{3/2} + C$$

$$\int \frac{1}{\sqrt{x}}, dx = 2\sqrt{x} + C$$

### Exponential and Logarithmic

$$\int e^x, dx = e^x + C$$

$$\int a^x, dx = \frac{a^x}{\ln a} + C, \qquad a > 0,\ a \neq 1$$

$$\int \ln x, dx = x\ln x - x + C$$

### Trigonometric

$$\int \sin x, dx = -\cos x + C$$

$$\int \cos x, dx = \sin x + C$$

$$\int \sec^2 x, dx = \tan x + C$$

$$\int \csc^2 x, dx = -\cot x + C$$

$$\int \sec x \tan x, dx = \sec x + C$$

$$\int \csc x \cot x, dx = -\csc x + C$$

### Inverse Trigonometric

$$\int \frac{1}{\sqrt{1-x^2}}, dx = \arcsin x + C$$

$$\int \frac{1}{1+x^2}, dx = \arctan x + C$$

$$\int \frac{1}{|x|\sqrt{x^2-1}}, dx = \text{arcsec}, x + C$$

### Verification Protocol

Every indefinite integral can be **self-verified** by differentiation:

$$\frac{d}{dx}\left[\int f(x), dx\right] = f(x)$$

Example: verify $\int x^3, dx = \frac{x^4}{4} + C$:

$$\frac{d}{dx}\left(\frac{x^4}{4} + C\right) = x^3 \checkmark$$

This is the gold standard. Always differentiate your answer.

## 3. Properties: Linearity of the Indefinite Integral

The indefinite integral is a **linear operator**:

$$\int [f(x) \pm g(x)], dx = \int f(x), dx \pm \int g(x), dx \tag{Sum Rule}$$

$$\int c\cdot f(x), dx = c\int f(x), dx \tag{Constant Multiple}$$

**Proof of Sum Rule.** Let $F' = f$, $G' = g$. Then $(F+G)' = f+g$. So $F + G$ is an antiderivative of $f + g$:

$$\int [f+g], dx = F + G + C = \int f, dx + \int g, dx \quad \blacksquare$$

_(Technically each $\int$ carries its own $+C$, but their sum is absorbed into a single constant.)_

**What is NOT linear:**

$$\int f(x)\cdot g(x), dx \neq \left(\int f, dx\right)\cdot\left(\int g, dx\right)$$

$$\int \frac{f(x)}{g(x)}, dx \neq \frac{\int f, dx}{\int g, dx}$$

Integration does not distribute over products or quotients. This is why we need techniques like substitution, parts, and partial fractions.

## 4. Worked Examples: Direct Integration

### Example 1: Polynomial

$$\int (3x^4 - 5x^2 + 7x - 2), dx$$

Apply linearity and power rule term by term:

$$= \frac{3x^5}{5} - \frac{5x^3}{3} + \frac{7x^2}{2} - 2x + C$$

**Verify:** differentiate to recover $3x^4 - 5x^2 + 7x - 2$. $\checkmark$

### Example 2: Rewrite Before Integrating

$$\int \frac{x^3 + 2x - 1}{x}, dx = \int\left(x^2 + 2 - \frac{1}{x}\right)dx = \frac{x^3}{3} + 2x - \ln|x| + C$$

**Key habit:** simplify algebraically before reaching for a technique.

### Example 3: Negative and Fractional Exponents

$$\int \left(\sqrt[3]{x^2} + \frac{1}{x^4}\right)dx = \int\left(x^{2/3} + x^{-4}\right)dx = \frac{x^{5/3}}{5/3} + \frac{x^{-3}}{-3} + C = \frac{3}{5}x^{5/3} - \frac{1}{3x^3} + C$$

### Example 4: Trigonometric Rewrite

$$\int \tan^2 x, dx = \int(\sec^2 x - 1), dx = \tan x - x + C$$

The identity $\tan^2 x = \sec^2 x - 1$ converts to a form in the standard table.

### Example 5: Separating a Fraction

$$\int \frac{1 + \cos^2\theta}{\cos^2\theta}, d\theta = \int\left(\sec^2\theta + 1\right)d\theta = \tan\theta + \theta + C$$

### Example 6: Initial Value Problem (Pinning the Constant)

Find $F(x)$ given $F'(x) = 3x^2 - 2x + 5$ and $F(1) = 4$.

**Step 1:** Integrate.

$$F(x) = \int(3x^2 - 2x + 5), dx = x^3 - x^2 + 5x + C$$

**Step 2:** Apply the initial condition.

$$F(1) = 1 - 1 + 5 + C = 5 + C = 4 \implies C = -1$$

$$\boxed{F(x) = x^3 - x^2 + 5x - 1}$$

The initial condition is the **additional information** that pins down a unique antiderivative from the infinite family. This is exactly the "lost constant" discussed in [[Differentiation and Integration as Inverse Processes]].

## 5. The Initial Value Problem (IVP): Antiderivatives with Constraints

The general pattern:

$$\text{Given: } F'(x) = f(x),\ F(a) = F_0 \implies F(x) = F_0 + \int_a^x f(t), dt$$

This is the **fundamental theorem applied as a recovery formula** — the indefinite integral parameterizes all solutions; the initial condition selects the unique one.

### Physics Example: Recovering Position from Acceleration

Given: $a(t) = -9.8$ m/s² (gravity), $v(0) = 20$ m/s (initial velocity upward), $s(0) = 0$ (ground level).

**Step 1:** Integrate acceleration to get velocity.

$$v(t) = \int (-9.8), dt = -9.8t + C_1$$

Apply $v(0) = 20$: $C_1 = 20$.

$$v(t) = -9.8t + 20$$

**Step 2:** Integrate velocity to get position.

$$s(t) = \int(-9.8t + 20), dt = -4.9t^2 + 20t + C_2$$

Apply $s(0) = 0$: $C_2 = 0$.

$$\boxed{s(t) = -4.9t^2 + 20t}$$

**Maximum height:** $v(t) = 0 \Rightarrow t = 20/9.8 \approx 2.04$ s. $s(2.04) \approx 20.4$ m.

Each integration introduces one $C$; each initial condition eliminates one $C$. The system is determined when the number of conditions equals the number of integrations.

## 6. Antiderivatives of Piecewise Functions

If $f$ is defined piecewise, the antiderivative is piecewise — with a separate constant on each piece, **plus continuity constraints** at the junctions.

### Example

$$f(x) = \begin{cases} x & x < 0 \ \cos x & x \geq 0 \end{cases}$$

Integrate on each piece:

$$F(x) = \begin{cases} \frac{x^2}{2} + C_1 & x < 0 \ \sin x + C_2 & x \geq 0 \end{cases}$$

Require continuity at $x = 0$: $\lim_{x\to 0^-} F(x) = \lim_{x\to 0^+} F(x)$:

$$\frac{0}{2} + C_1 = \sin 0 + C_2 \implies C_1 = C_2$$

One free constant remains (as expected — there is still a family of antiderivatives). If we add $F(0) = 3$: $C_2 = 3$, $C_1 = 3$.

## 7. What Integration Cannot Do: Non-Elementary Antiderivatives

Some perfectly continuous functions have **no antiderivative expressible in elementary functions** (polynomials, exponentials, logs, trig, and their compositions). The indefinite integral "exists" in the sense of FTC Part 1, but cannot be written with our usual symbols.

Famous examples:

|Integrand|"Antiderivative"|Name|
|---|---|---|
|$e^{-x^2}$|$\frac{\sqrt{\pi}}{2}\text{erf}(x) + C$|Error function|
|$\sin(x^2)$|$S(x) + C$|Fresnel $S$ (see [[Fresnel Function]])|
|$\cos(x^2)$|$C(x) + C$|Fresnel $C$|
|$\frac{\sin x}{x}$|$\text{Si}(x) + C$|Sine integral|
|$\frac{1}{\ln x}$|$\text{li}(x) + C$|Logarithmic integral|
|$\sqrt{1 - k^2\sin^2 x}$|$E(x,k) + C$|Elliptic integral|

These functions are perfectly well-defined — the Riemann integral of each integrand exists. What fails is the ability to express the antiderivative in closed elementary form. This is a **theorem** (Liouville, 1835; made rigorous by Risch, 1969): there is a decision procedure to determine whether an elementary antiderivative exists, and it says "no" for all of the above.

> **Risch Algorithm:** The algorithmic version of Liouville's theorem. CAS systems (Mathematica, SymPy, Maple) implement it. When your CAS returns an integral unevaluated, it is not confused — it has determined no elementary form exists.

## 8. Visualization: The Family of Antiderivatives

The indefinite integral $\int f(x), dx = F(x) + C$ is a family of curves — **vertical translates** of one another.

```
f(x) = 2x           ∫ 2x dx = x² + C

F(x) = x² + C for various C:

y
  |         C = 3: ___*___
  |              **       **
  |            **           **
  |    C = 0: *_______       *
  |          *         *
  |         *    C=-2:  *__*___
  |        *           *       *
  +---------*-----------*-------> x
           -2           2

All curves have the same shape (same derivative = 2x).
They differ only by vertical translation.
At each x, the slope of every curve = 2x. ← this is f(x).
```

The initial condition $F(a) = F_0$ picks exactly one curve from this family — the one that passes through the point $(a, F_0)$.

## 9. Indefinite vs. Definite: The Precise Relationship

The connection is FTC Part 2:

$$\int_a^b f(x), dx = \left[F(x) + C\right]_a^b = (F(b) + C) - (F(a) + C) = F(b) - F(a)$$

The constant $C$ **cancels** in a definite integral. This is why we can use any antiderivative — the choice of $C$ is irrelevant for definite integrals.

|Feature|Indefinite $\int f, dx$|Definite $\int_a^b f, dx$|
|---|---|---|
|Result|Family of functions $F(x)+C$|A real number|
|Depends on $a$, $b$|No|Yes|
|Contains $+C$|Yes|No (cancels)|
|Role|Antidifferentiation|Area / accumulation|
|Requires FTC to evaluate|No — direct antidifferentiation|Yes (or Riemann sums)|

## 10. The Net Change Theorem (Restatement)

From [[Fundamental Theorem of Calculus]], the net change theorem:

$$F(b) - F(a) = \int_a^b F'(x), dx$$

reads directly as: **the definite integral of the rate of change is the net change in quantity.** The indefinite integral is what allows us to compute the left-hand side: find $F$ such that $F' = f$, then evaluate $F(b) - F(a)$.

The indefinite integral is the **computational workhorse** that makes FTC Part 2 usable.

## 11. Connection to AI / Machine Learning / Deep Learning

### 11.1 Antiderivatives as Potential Functions

In physics-informed ML and conservative dynamics, a vector field $\mathbf{F} = -\nabla V$ is conservative (force = negative gradient of potential). Finding $V$ given $\mathbf{F}$ is the multivariable antidifferentiation problem:

$$V(x) = -\int F(x), dx + C$$

The constant $C$ is a **gauge freedom** — physics depends only on differences in potential, not absolute values. In **Hamiltonian neural networks** (Greydanus et al., 2019), networks are designed to learn a scalar Hamiltonian $H$ such that dynamics follow $\dot{p} = -\partial H/\partial q$, $\dot{q} = \partial H/\partial p$. The antiderivative of the learned energy gradient is never computed explicitly — but the $+C$ gauge freedom is baked in by design.

### 11.2 The $+C$ as a Baseline in Reinforcement Learning

In RL, the value function $V^\pi(s) = \mathbb{E}[\sum_t \gamma^t r_t | s_0 = s]$ is defined up to a constant. Adding a constant baseline $C$ to rewards:

$$r'_t = r_t - C$$

does not change the optimal policy (the gradient of the expected return is unchanged) but reduces variance in policy gradient estimates. This is the RL analog of $+C$: an undetermined constant that can be chosen optimally (typically $C = V^\pi(s_t)$ — the value function baseline itself).

### 11.3 Weight Initialization as Initial Condition

Training a neural network is solving a high-dimensional IVP:

$$\frac{d\theta}{dt} = -\nabla_\theta \mathcal{L}(\theta), \qquad \theta(0) = \theta_{\text{init}}$$

The "antiderivative" is $\theta(t) = \theta(0) - \int_0^t \nabla_\theta \mathcal{L}, d\tau$. The indefinite integral (without initial condition) gives a family of parameter trajectories — one for each starting point $\theta(0)$. The initial condition $\theta(0) = \theta_{\text{init}}$ selects a unique trajectory.

**Why initialization matters:** Different $\theta_{\text{init}}$ correspond to different members of the antiderivative family. The loss landscape has many local minima (different particular solutions to $\dot\theta = -\nabla\mathcal{L}$); initialization chooses which one gradient descent converges to. The "$+C$" freedom in the antiderivative family is exactly the sensitivity to initialization in neural network training.

### 11.4 Symbolic Integration in ML Systems

The **Risch algorithm** (the decision procedure for elementary antiderivatives, §7) is a canonical example of **symbolic AI** — a rule-based system that manipulates mathematical expressions exactly. CAS systems (SymPy, Mathematica) implement it.

Modern ML has a parallel: **neural integration** (Lample & Charton, 2019) — a transformer trained to compute symbolic integrals, outperforming rule-based CAS on many benchmarks. The training data is pairs $(f, \int f, dx)$ generated by differentiating known functions. The network learns the inverse (integration) from the forward (differentiation) by pattern recognition across millions of examples.

This is the ML version of "read the derivative table backwards" — but at scale, with a neural network discovering the patterns rather than a human encoding them.

### 11.5 Score Functions as Antiderivatives of Potentials

In **energy-based models** and **score matching**:

$$p_\theta(x) = \frac{1}{Z(\theta)} e^{-E_\theta(x)}$$

The score is $\nabla_x \log p_\theta(x) = -\nabla_x E_\theta(x)$.

The energy $E_\theta$ is the **antiderivative** (potential) of the negative score field. In score-based diffusion models (DDPM, Score-SDE), we learn the score $s_\theta(x,t) \approx \nabla_x \log p_t(x)$ directly, never computing $E_\theta$ explicitly — because (in high dimensions) the antiderivative may not be a simple scalar function. The $+C$ ambiguity in the potential corresponds to the normalization constant $Z(\theta)$ — which is intractable to compute and is exactly what score matching avoids.

### 11.6 Antiderivatives in Numerical ODE Solvers

Every numerical ODE solver is approximating an antiderivative. Given $\dot{x} = f(x,t)$:

$$x(T) = x(0) + \int_0^T f(x(t), t), dt$$

The **indefinite integral** $\int f, dt$ is what we seek. Euler, Runge-Kutta, and Adams methods are progressively more accurate approximations to this antiderivative. The initial condition $x(0) = x_0$ is what pins the particular solution.

In **Neural ODEs** (Chen et al., 2018), the function $f_\theta$ is a neural network. The ODE solver computes the antiderivative of $f_\theta$ numerically, and backpropagation through this computation (via the adjoint method) is the training algorithm. The whole architecture is: parameterize an integrand ($f_\theta$), integrate it (ODE solver), evaluate loss, differentiate through the integration (adjoint = FTC Part 1), update parameters.

|Indefinite Integral Concept|ML / DL Analog|

## 12. Common Errors and Diagnostics

### Error 1: Forgetting $+C$

$$\int 3x^2, dx = x^3 \quad \leftarrow \text{WRONG}$$ $$\int 3x^2, dx = x^3 + C \quad \leftarrow \text{correct}$$

In a definite integral, $C$ cancels. In an indefinite integral, omitting $C$ means giving one answer when infinitely many are correct.

### Error 2: Integrating Through a Singularity

$$\int_{-1}^{2} \frac{1}{x}, dx \neq [\ln|x|]_{-1}^2 = \ln 2 - \ln 1 = \ln 2$$

$1/x$ has a non-integrable singularity at $x = 0 \in [-1,2]$. The integral **diverges**. Check for singularities in $[a,b]$ before applying FTC Part 2.

### Error 3: $n = -1$ in Power Rule

$$\int x^{-1}, dx \neq \frac{x^0}{0} + C \quad \leftarrow \text{UNDEFINED}$$ $$\int x^{-1}, dx = \ln|x| + C \quad \leftarrow \text{correct}$$

The power rule fails at $n = -1$. This is a separate entry in the table.

### Error 4: Constant Multiple vs. Function Multiple

$$\int 2f(x), dx = 2\int f(x), dx \quad \leftarrow \text{correct (constant multiple)}$$ $$\int x \cdot f(x), dx \neq x\int f(x), dx \quad \leftarrow \text{WRONG (function multiple — use integration by parts)}$$

Only **constants** pull out of the integral.

### Error 5: Misidentifying the Standard Form

$$\int \frac{1}{x^2 + 4}, dx \neq \ln(x^2+4) + C$$

Note $\frac{1}{x^2+4} = \frac{1}{4}\cdot\frac{1}{(x/2)^2+1}$. This requires the $\arctan$ form:

$$= \frac{1}{2}\arctan!\left(\frac{x}{2}\right) + C$$

Always check whether a rational integrand matches an inverse trig form before trying other techniques.

## 13. Quick-Reference Cheatsheet

$$\int f(x), dx = F(x) + C \iff F'(x) = f(x)$$

$$\textbf{Linearity:}\quad \int[\alpha f + \beta g],dx = \alpha\int f,dx + \beta\int g,dx$$

$$\textbf{Power Rule:}\quad \int x^n,dx = \frac{x^{n+1}}{n+1}+C,\quad n\neq -1$$

$$\int x^{-1},dx = \ln|x|+C$$

$$\int e^x,dx = e^x+C \qquad \int a^x,dx = \frac{a^x}{\ln a}+C$$

$$\int \sin x,dx = -\cos x+C \qquad \int \cos x,dx = \sin x+C$$

$$\int \sec^2 x,dx = \tan x+C \qquad \int \csc^2 x,dx = -\cot x+C$$

$$\int \frac{dx}{\sqrt{1-x^2}} = \arcsin x+C \qquad \int \frac{dx}{1+x^2} = \arctan x+C$$

$$\textbf{IVP:}\quad F'=f,\ F(a)=F_0 \implies F(x) = F_0 + \int_a^x f(t),dt$$

$$\textbf{Verify:}\quad \frac{d}{dx}\left[\int f(x),dx\right] = f(x) \quad \text{always check!}$$
