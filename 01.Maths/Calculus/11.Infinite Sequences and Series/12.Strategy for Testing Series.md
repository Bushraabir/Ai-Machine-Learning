## 1. The Landscape — Every Test at a Glance

There is no single algorithm that determines convergence for every series. What exists instead is a **toolkit** of tests, each designed for a particular structural signature. The strategy is:

1. Look at the series carefully.
2. Identify its dominant structure.
3. Match that structure to the right test.
4. Execute rigorously.

The table below is the master reference. Everything that follows is commentary on how to read a series and arrive at the right row.

|Test|Series form|Condition to check|Conclusion|
|---|---|---|---|
|**Divergence Test**|Any $\sum a_n$|$\lim a_n \neq 0$|Diverges|
|**Geometric Series**|$\sum ar^n$|$\|r\| < 1$|Converges to $a/(1-r)$|
|**$p$-Series**|$\sum 1/n^p$|$p > 1$|Converges|
|**Integral Test**|$\sum f(n)$, $f$ positive, decreasing, continuous|$\int_1^\infty f(x),dx$ finite|Same fate as the integral|
|**Direct Comparison**|$0 \leq a_n \leq b_n$|$\sum b_n$ converges|$\sum a_n$ converges|
||$0 \leq b_n \leq a_n$|$\sum b_n$ diverges|$\sum a_n$ diverges|
|**Limit Comparison**|$a_n, b_n > 0$|$\lim a_n/b_n = L \in (0,\infty)$|Same fate|
|**Alternating Series**|$\sum (-1)^n b_n$, $b_n > 0$|$b_n \searrow 0$|Converges|
|**Ratio Test**|Any $\sum a_n$|$\lim \|a_{n+1}/a_n\| = L$|$L<1$: conv; $L>1$: div; $L=1$: inconclusive|
|**Root Test**|Any $\sum a_n$|$\lim \|a_n\|^{1/n} = L$|$L<1$: conv; $L>1$: div; $L=1$: inconclusive|

## 2. The Ratio Test — Statement and Proof

The Ratio Test detects geometric-like decay.

### Theorem

Let $\sum a_n$ be a series with $a_n \neq 0$. Let

$$L = \lim_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right|$$

- If $L < 1$: $\sum a_n$ converges **absolutely**.
- If $L > 1$ (or $L = \infty$): $\sum a_n$ diverges.
- If $L = 1$: the test is **inconclusive**.

### Proof

**Case $L < 1$.** Choose $r$ with $L < r < 1$. By definition of the limit, there exists $N$ such that for all $n \geq N$:

$$\left|\frac{a_{n+1}}{a_n}\right| < r$$

Applying this repeatedly from index $N$:

$$|a_{N+k}| < r^k |a_N| \quad \text{for all } k \geq 0$$

So $\displaystyle\sum_{k=0}^\infty |a_{N+k}| < |a_N| \sum_{k=0}^\infty r^k = \frac{|a_N|}{1-r} < \infty$, since $r < 1$. Absolute convergence of the tail implies absolute convergence of the whole series. $\checkmark$

**Case $L > 1$.** There exists $N$ such that $|a_{n+1}/a_n| > 1$ for all $n \geq N$, so $|a_n|$ is eventually strictly increasing. Thus $a_n \not\to 0$, and the series diverges by the Divergence Test. $\checkmark$

**Why $L = 1$ is inconclusive.** The $p$-series $\sum 1/n^p$ gives $L = \lim (n/(n+1))^p = 1$ for **every** $p$ — yet the series converges for $p > 1$ and diverges for $p \leq 1$. So $L = 1$ carries no information. $\square$

## 3. The Root Test — Statement and Proof

The Root Test is tailor-made for series where the $n$-th term itself is an $n$-th power.

### Theorem

Let $L = \displaystyle\lim_{n\to\infty} |a_n|^{1/n}$.

- $L < 1$: converges absolutely.
- $L > 1$: diverges.
- $L = 1$: inconclusive.

### Proof

**Case $L < 1$.** Choose $r$ with $L < r < 1$. For all $n \geq N$: $|a_n|^{1/n} < r$, i.e., $|a_n| < r^n$. Then $\sum |a_n| < \sum r^n < \infty$ (geometric with ratio $r < 1$). $\checkmark$

**Case $L > 1$.** For all $n \geq N$: $|a_n|^{1/n} > 1$, so $|a_n| > 1$. Terms do not vanish; divergence follows from the Divergence Test. $\checkmark$

**Inconclusive at $L = 1$**: same $p$-series argument as above. $\square$

> **The Cauchy–d'Alembert inequality** relates the two tests: $$\liminf_{n\to\infty} \frac{|a_{n+1}|}{|a_n|} ;\leq; \liminf_{n\to\infty} |a_n|^{1/n} ;\leq; \limsup_{n\to\infty} |a_n|^{1/n} ;\leq; \limsup_{n\to\infty} \frac{|a_{n+1}|}{|a_n|}$$ Consequence: the Root Test is at least as powerful as the Ratio Test. It never fails where the Ratio Test succeeds, but can succeed where the Ratio Test fails (e.g., when the ratio oscillates and has no limit).

## 4. The Master Decision Tree

Work top to bottom. Stop at the first test that yields a conclusion.

```
Given Σ aₙ
│
├─ STEP 1: Does lim aₙ ≠ 0?
│     YES → DIVERGES ✗
│      NO → continue
│
├─ STEP 2: Is it a known special form?
│     Geometric Σ arⁿ → |r|<1: CONV ✓   |r|≥1: DIV ✗
│     p-series Σ 1/nᵖ → p>1:  CONV ✓   p≤1:  DIV ✗
│     Telescoping    → Sum directly
│
├─ STEP 3: Does it alternate signs?
│     YES → Check absolute convergence first (Steps 4–6 on |aₙ|)
│           If |aₙ| converges: ABSOLUTELY CONV ✓ (done)
│           If not: try AST — bₙ ↘ 0 → CONDITIONALLY CONV ✓
│
├─ STEP 4: Contains n!, aⁿ, nⁿ ?
│     YES → Ratio Test or Root Test
│           L < 1: CONV ✓   L > 1: DIV ✗   L = 1: → Step 5
│
├─ STEP 5: Is aₙ rational or algebraic in n?
│     YES → Limit Comparison with 1/nᵖ
│           (p = deg denom − deg numer)
│
└─ STEP 6: Contains ln n, trig, mixed?
      → Integral Test or Direct Comparison
```

## 5. Diagnostic Signals — Reading the Anatomy of $a_n$

Before touching a test, read the structure of $a_n$. Each structural feature is a signal.

**Signal: Factorial or exponential — $n!$, $a^n$, $n^n$** These simplify under the ratio $a_{n+1}/a_n$: $$\frac{(n+1)!}{n!} = n+1, \qquad \frac{a^{n+1}}{a^n} = a, \qquad \frac{(n+1)^{n+1}}{n^n} \to e$$ → **Ratio Test is the natural first move.**

**Signal: Entire term is an $n$-th power — $a_n = f(n)^n$** Then $|a_n|^{1/n} = |f(n)|$, and the Root Test reduces to evaluating $\lim f(n)$. → **Root Test collapses the exponent immediately.**

**Signal: Rational or algebraic in $n$ — polynomials, roots** Identify dominant powers: degree of denominator minus degree of numerator gives $p$. → **Limit Comparison with $\sum 1/n^p$.** Never use the Ratio Test here — it always gives $L = 1$.

**Signal: Alternating signs** Always ask for absolute convergence first. If $\sum |a_n|$ converges, done. Otherwise, check $b_n = |a_n|$ for the two AST conditions. → **Absolute convergence check first, then AST.**

**Signal: Logarithms, iterated logs, $\ln n / n^p$** Logarithms decay slower than any positive power of $n$. The integral test handles these cleanly since $\int (\ln x)^k / x^p, dx$ has a closed form. → **Integral Test.**

## 6. The $L = 1$ Problem

The most common dead end: you compute the Ratio or Root Test ratio, get $L = 1$, and are stuck. This is not a failure — it is a signal to pivot.

**$L = 1$ from Ratio/Root Test always happens for rational $a_n$.** These tests are blind to polynomial decay. The fix:

1. Identify the polynomial-like decay rate $p$ of $|a_n|$. Use Limit Comparison with $\sum 1/n^p$.
2. If $\ln n$ is present, use the Integral Test.
3. If the series alternates, pivot to the Alternating Series Test.
4. If $|a_n|$ is visibly dominated by a known series, use Direct Comparison.

> Never interpret $L = 1$ as "the series might converge." It carries no convergence information whatsoever.

## 7. Worked Examples — Full Diagnostic Process

### Example 1

$$\sum_{n=1}^{\infty} \frac{n^3}{3^n}$$

**Signal:** $3^n$ — exponential. **Test: Ratio.**

$$\frac{a_{n+1}}{a_n} = \frac{(n+1)^3}{3^{n+1}} \cdot \frac{3^n}{n^3} = \frac{1}{3}\left(1 + \frac{1}{n}\right)^3 \to \frac{1}{3} < 1$$

**Converges absolutely.** ✓

---

### Example 2

$$\sum_{n=1}^{\infty} \left(\frac{2n+3}{5n-1}\right)^n$$

**Signal:** Entire term is an $n$-th power. **Test: Root.**

$$|a_n|^{1/n} = \frac{2n+3}{5n-1} \to \frac{2}{5} < 1$$

**Converges absolutely.** ✓

---

### Example 3

$$\sum_{n=1}^{\infty} \frac{3n^2 - 1}{n^4 + 5}$$

**Signal:** Rational. Dominant: $3n^2/n^4 = 3/n^2$. **Test: Limit Comparison** with $b_n = 1/n^2$.

$$L = \lim_{n\to\infty} \frac{(3n^2-1)/(n^4+5)}{1/n^2} = \lim_{n\to\infty} \frac{3n^4 - n^2}{n^4+5} = 3 \in (0,\infty)$$

$\sum 1/n^2$ converges ($p = 2$). **Converges.** ✓

---

### Example 4

$$\sum_{n=1}^{\infty} \frac{(-1)^n \sqrt{n}}{n+1}$$

**Signal:** Alternating. **Step 1 — absolute convergence:** $\sum \sqrt{n}/(n+1) \sim \sum 1/n^{1/2}$ diverges ($p = 1/2$). Not absolute.

**Step 2 — AST:** $b_n = \sqrt{n}/(n+1)$. Limit: $b_n \to 0$. ✓ Monotone: $f(x) = \sqrt{x}/(x+1)$, $f'(x) = (1-x)/(2\sqrt{x}(x+1)^2) < 0$ for $x > 1$. ✓

**Converges conditionally.** ✓

---

### Example 5

$$\sum_{n=2}^{\infty} \frac{1}{n(\ln n)^2}$$

**Signal:** Logarithm present. **Test: Integral.**

$$\int_2^\infty \frac{dx}{x(\ln x)^2} = \left[-\frac{1}{\ln x}\right]_2^\infty = \frac{1}{\ln 2} < \infty$$

**Converges.** ✓

Contrast: $\sum 1/(n \ln n)$ gives $\int_2^\infty dx/(x \ln x) = [\ln \ln x]_2^\infty = \infty$ — **diverges**. The exponent on $\ln n$ is the critical parameter.

---

### Example 6 — Ratio Test Fails: Pivot Required

$$\sum_{n=1}^{\infty} \frac{1}{n^2 + 1}$$

**Ratio Test:** $a_{n+1}/a_n = (n^2+1)/((n+1)^2+1) \to 1$. Inconclusive.

**Pivot:** Rational. Dominant: $1/n^2$. **Limit Comparison** with $1/n^2$:

$$L = \lim \frac{n^2}{n^2+1} = 1 \in (0,\infty)$$

**Converges.** ✓

---

### Example 7 — Nested Factorials

$$\sum_{n=1}^\infty \frac{n! \cdot 2^n}{(2n)!}$$

**Signal:** Factorials on both sides. **Test: Ratio.**

$$\frac{a_{n+1}}{a_n} = \frac{(n+1)!\cdot 2^{n+1}}{(2n+2)!} \cdot \frac{(2n)!}{n!\cdot 2^n} = \frac{2(n+1)}{(2n+2)(2n+1)} = \frac{1}{2n+1} \to 0 < 1$$

**Converges absolutely.** ✓

---

## 8. Boundary Cases — The Inconclusive Zone

Some series sit exactly at the edge where every standard test gives $L = 1$ or a clean comparison is unavailable. These require the **Integral Test** with iterated logarithms:

|Series|Verdict|Key tool|
|---|---|---|
|$\sum 1/n$|Diverges|$p$-series ($p = 1$)|
|$\sum 1/(n \ln n)$|Diverges|Integral Test|
|$\sum 1/(n(\ln n)^2)$|Converges|Integral Test|
|$\sum 1/(n \ln n \cdot \ln\ln n)$|Diverges|Integral Test|
|$\sum 1/(n \ln n \cdot (\ln\ln n)^2)$|Converges|Integral Test|
|$\sum 1/n^{1+\varepsilon}$ (any $\varepsilon > 0$)|Converges|$p$-series|

The pattern: each convergent series can be "weakened" by reducing an exponent above 1 to exactly 1 in one of the logarithmic factors, producing a divergent series. The boundary is not a point — it is a **dense accumulation** of cases.

## 10. Connection to AI, Machine Learning, and Deep Learning

### 10.1 The Strategy as a Meta-Skill for ML Analysis

The discipline of this note — _look at structure first, choose the tool second_ — is exactly the discipline required for theoretical ML. Every convergence proof in learning theory, optimisation, and approximation follows the same logic:

1. Identify the dominant term in the bound.
2. Recognise its class (polynomial, exponential, logarithmic).
3. Apply the appropriate convergence or rate-of-growth argument.

### 10.2 Ratio Test — Convergence of Gradient Descent Iterates

In the analysis of gradient descent on strongly convex functions, the iterate error satisfies:

$$|x_{t+1} - x^_| \leq (1 - \alpha \mu) |x_t - x^_|$$

where $\alpha$ is the learning rate and $\mu$ is the strong convexity constant. This is precisely the Ratio Test condition: the ratio of successive terms is $r = 1 - \alpha\mu < 1$ (for small enough $\alpha$). The total distance travelled $\sum_t |x_{t+1} - x_t|$ is then a geometric series with ratio $r$, which converges — certifying that the algorithm does not take infinitely long steps.

### 10.3 Root Test — PAC-Bayes Bounds and Exponential Families

PAC-Bayes generalisation bounds involve expressions of the form:

$$\text{generalisation gap} \leq \sqrt{\frac{\text{KL}(Q|P) + \ln(1/\delta)}{2n}}$$

When summed over a hierarchy of models or a sequence of hypotheses, the dominant structure is often an exponential in model complexity. The Root Test certifies that such sums converge — ensuring the union bound over the hypothesis class is valid.

### 10.4 Limit Comparison — Complexity and Sample Complexity

In statistical learning theory, the sample complexity of learning a hypothesis class is often $O(d/\varepsilon^2)$ where $d$ is the VC dimension. When comparing two algorithms with sample complexities $n_1(k)$ and $n_2(k)$ as the problem size $k \to \infty$, the Limit Comparison Test applies: if $n_1(k)/n_2(k) \to L \in (0,\infty)$, the two algorithms have the same asymptotic sample complexity class — they are equivalent at scale.

### 10.5 Integral Test — Continuous Relaxations of Discrete Sums

Many quantities in ML (partition functions, normalising constants, expected losses) involve sums $\sum_{n=1}^\infty f(n)$ that are approximated by the integral $\int_1^\infty f(x),dx$. The Euler–Maclaurin formula formalises this: the sum and integral differ by correction terms. The **Integral Test** is the rigorous foundation for this approximation — it guarantees that the sum and integral have the same convergence behaviour, and the error in approximating one by the other is controlled.

### 10.6 The Ratio Test and Radius of Convergence in Transformer Attention

The softmax in Transformer attention involves $e^{x}$, approximated as a power series $\sum x^k/k!$ in linearised attention models (Performer, FNet). The Ratio Test applied to $a_k = x^k/k!$:

$$\frac{|a_{k+1}|}{|a_k|} = \frac{|x|}{k+1} \to 0 \quad \text{for any fixed } x$$

confirms that the Taylor series converges absolutely for all $x \in \mathbb{R}$ — the approximation is valid globally, not just locally. This is the theoretical justification for replacing exact softmax with a truncated series.

## 11. Anti-Patterns — What Not to Do

|Mistake|Consequence|Fix|
|---|---|---|
|Ratio Test on rational $a_n$|Always get $L=1$, no conclusion|Go directly to Limit Comparison|
|Skipping the Divergence Test|Wasted effort on a trivially divergent series|Always do it first|
|Wrong direction in Direct Comparison|Invalid conclusion|Re-read which way the inequality must go|
|AST without checking absolute convergence|Weaker conclusion than available|Check $\sum\|a_n\|$ first|
|Integral Test without checking hypotheses|Invalid application|Verify positive, continuous, decreasing|
|Treating $L=1$ as a conclusion|Incorrect|It is purely inconclusive — switch tools|

## 12. Summary Card

$$\boxed{\textbf{Step 1 (always): } \lim a_n \neq 0 \implies \text{Diverges immediately}}$$

$$\boxed{\textbf{Step 2: } \text{Geometric / } p\text{-series / Telescoping} \implies \text{Use known results}}$$

$$\boxed{\textbf{Step 3: } \text{Alternating} \implies \text{Absolute convergence first, then AST}}$$

$$\boxed{\textbf{Step 4: } n!,, a^n,, n^n \implies \text{Ratio Test or Root Test}}$$

$$\boxed{\textbf{Step 5: } \text{Rational/algebraic} \implies \text{Limit Comparison with } 1/n^p}$$

$$\boxed{\textbf{Step 6: } \ln n,, \text{mixed} \implies \text{Integral Test or Direct Comparison}}$$

$$\boxed{\textbf{If } L = 1 \text{ from Ratio/Root:} \implies \text{pivot — do not stop}}$$
