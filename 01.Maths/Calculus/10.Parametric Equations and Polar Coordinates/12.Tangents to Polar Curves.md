

> _"The derivative in polar coordinates is not a polar object — it is a Cartesian slope computed via a parametric detour. Understanding this detour is understanding the geometry."_

---

## 0. The Central Idea

A polar curve $r = f(\theta)$ is not given as $y = g(x)$. So "the slope of the tangent" is not immediately meaningful in polar language. The strategy:

1. **Reparametrize.** Treat $\theta$ as the parameter, writing $x(\theta) = f(\theta)\cos\theta$ and $y(\theta) = f(\theta)\sin\theta$.
2. **Apply parametric differentiation.** The slope of the tangent is $dy/dx = (dy/d\theta)/(dx/d\theta)$.
3. **Interpret geometrically.** Horizontal tangents, vertical tangents, and the behavior at the pole all have clean geometric meanings.

Everything reduces to parametric curve theory — which means every theorem you proved for parametric curves applies here immediately.

---

## 1. Derivation of the Slope Formula

Let $r = f(\theta)$, where $f$ is differentiable. The Cartesian parametrization is:

$$x(\theta) = f(\theta)\cos\theta, \qquad y(\theta) = f(\theta)\sin\theta$$

Differentiate both with respect to $\theta$ using the product rule:

$$\frac{dx}{d\theta} = f'(\theta)\cos\theta - f(\theta)\sin\theta$$

$$\frac{dy}{d\theta} = f'(\theta)\sin\theta + f(\theta)\cos\theta$$

By the parametric chain rule $\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta}$:

$$\boxed{\frac{dy}{dx} = \frac{f'(\theta)\sin\theta + f(\theta)\cos\theta}{f'(\theta)\cos\theta - f(\theta)\sin\theta}}$$

or equivalently, writing $r$ for $f(\theta)$ and $r'$ for $f'(\theta)$:

$$\frac{dy}{dx} = \frac{r'\sin\theta + r\cos\theta}{r'\cos\theta - r\sin\theta}$$

**This is valid whenever the denominator is nonzero.** Points where both numerator and denominator vanish require separate analysis (they correspond to singular points or the pole).

### 1.1 Verification Against Cartesian

For the horizontal line $\theta = \pi/4$ (i.e., $y = x$), we can write $r = C/(\cos\theta - \sin\theta)^{-1}$... this is awkward. Instead verify on a circle: $r = a$ (constant), so $f'(\theta) = 0$.

$$\frac{dy}{dx} = \frac{0 \cdot \sin\theta + a\cos\theta}{0 \cdot \cos\theta - a\sin\theta} = \frac{a\cos\theta}{-a\sin\theta} = -\cot\theta$$

For the circle $x^2 + y^2 = a^2$, implicit differentiation gives $2x + 2yy' = 0 \Rightarrow y' = -x/y = -(a\cos\theta)/(a\sin\theta) = -\cot\theta$. ✓

---

## 2. Horizontal Tangents

A horizontal tangent occurs where $dy/dx = 0$, i.e., the numerator vanishes while the denominator does not.

$$\frac{dy}{d\theta} = 0 \quad \text{and} \quad \frac{dx}{d\theta} \neq 0$$

$$\Updownarrow$$

$$r'\sin\theta + r\cos\theta = 0 \quad \text{and} \quad r'\cos\theta - r\sin\theta \neq 0$$

**Geometric meaning.** The tangent line is parallel to the $x$-axis (polar axis). The curve has a local extremum in the $y$-direction.

---

## 3. Vertical Tangents

A vertical tangent occurs where $dy/dx$ is undefined with a nonzero numerator:

$$\frac{dx}{d\theta} = 0 \quad \text{and} \quad \frac{dy}{d\theta} \neq 0$$

$$\Updownarrow$$

$$r'\cos\theta - r\sin\theta = 0 \quad \text{and} \quad r'\sin\theta + r\cos\theta \neq 0$$

**Geometric meaning.** The tangent line is parallel to the $y$-axis. The curve has a local extremum in the $x$-direction.

---

## 4. Worked Example: Cardioid $r = 1 + \cos\theta$

This is the central example. We find all horizontal and vertical tangents and determine the tangent at the pole.

$f(\theta) = 1 + \cos\theta$, $\quad f'(\theta) = -\sin\theta$.

$$\frac{dy}{d\theta} = -\sin\theta \cdot \sin\theta + (1 + \cos\theta)\cos\theta = -\sin^2\theta + \cos\theta + \cos^2\theta$$

$$= \cos(2\theta) + \cos\theta$$

Using $\cos(2\theta) = 2\cos^2\theta - 1$:

$$\frac{dy}{d\theta} = 2\cos^2\theta + \cos\theta - 1 = (2\cos\theta - 1)(\cos\theta + 1)$$

$$\frac{dx}{d\theta} = -\sin\theta\cos\theta - (1+\cos\theta)\sin\theta = -\sin\theta(\cos\theta + 1 + \cos\theta) = -\sin\theta(1 + 2\cos\theta)$$

### 4.1 Horizontal Tangents

Set $\frac{dy}{d\theta} = 0$: $(2\cos\theta - 1)(\cos\theta + 1) = 0$.

**Case 1:** $\cos\theta = 1/2 \Rightarrow \theta = \pi/3$ or $\theta = 5\pi/3$.

- $\theta = \pi/3$: $r = 1 + 1/2 = 3/2$. Check $dx/d\theta = -(\sqrt{3}/2)(1 + 1) = -\sqrt{3} \neq 0$. ✓
    - Cartesian: $x = (3/2)\cos(\pi/3) = 3/4$, $y = (3/2)\sin(\pi/3) = 3\sqrt{3}/4$.
- $\theta = 5\pi/3$: $r = 3/2$. By symmetry: $(3/4, -3\sqrt{3}/4)$.

**Case 2:** $\cos\theta = -1 \Rightarrow \theta = \pi$.

- $r = 1 + (-1) = 0$. The curve is at the pole. Check: $dx/d\theta = -\sin(\pi)(1 + 2(-1)) = 0$. Both vanish — this is a singular point at the pole. Handle separately (see §6).

**Horizontal tangents:** at $(3/4, \pm 3\sqrt{3}/4)$.

### 4.2 Vertical Tangents

Set $\frac{dx}{d\theta} = 0$: $-\sin\theta(1 + 2\cos\theta) = 0$.

**Case 1:** $\sin\theta = 0 \Rightarrow \theta = 0$ or $\theta = \pi$.

- $\theta = 0$: $r = 2$. Check $dy/d\theta = (2(1) - 1)(1 + 1) = 1 \cdot 2 = 2 \neq 0$. ✓
    - Cartesian: $(2, 0)$.
- $\theta = \pi$: $r = 0$. Both $dy/d\theta$ and $dx/d\theta$ vanish — singular point again.

**Case 2:** $1 + 2\cos\theta = 0 \Rightarrow \cos\theta = -1/2 \Rightarrow \theta = 2\pi/3$ or $\theta = 4\pi/3$.

- $\theta = 2\pi/3$: $r = 1 - 1/2 = 1/2$. Check $dy/d\theta = (2(-1/2)-1)(-1/2+1) = (-2)(1/2) = -1 \neq 0$. ✓
    - Cartesian: $x = (1/2)(-1/2) = -1/4$, $y = (1/2)(\sqrt{3}/2) = \sqrt{3}/4$.
- $\theta = 4\pi/3$: $r = 1/2$. By symmetry: $(-1/4, -\sqrt{3}/4)$.

**Vertical tangents:** at $(2, 0)$ and $(-1/4, \pm\sqrt{3}/4)$.

```
Cardioid r = 1 + cosθ with tangent points marked:

              (3/4, 3√3/4)  ← horizontal tangent
                   •
                  / \
    VT •─────── /   \ ─────• (2,0) ← vertical tangent
  (-1/4, √3/4)  \   /
                  \ /
                   • ← cusp at pole (θ = π)
              (3/4,-3√3/4) ← horizontal tangent
```

## 5. Worked Example: Rose Curve $r = \cos(2\theta)$

$f(\theta) = \cos(2\theta)$, $\quad f'(\theta) = -2\sin(2\theta)$.

$$\frac{dy}{d\theta} = -2\sin(2\theta)\sin\theta + \cos(2\theta)\cos\theta$$

$$\frac{dx}{d\theta} = -2\sin(2\theta)\cos\theta - \cos(2\theta)\sin\theta$$

**Finding horizontal tangents on the first petal** ($\theta \in [-\pi/4, \pi/4]$):

Use double-angle identities $\sin(2\theta) = 2\sin\theta\cos\theta$, $\cos(2\theta) = 1 - 2\sin^2\theta$:

$$\frac{dy}{d\theta} = -4\sin\theta\cos\theta\sin\theta + (1-2\sin^2\theta)\cos\theta = \cos\theta(-4\sin^2\theta + 1 - 2\sin^2\theta)$$

$$= \cos\theta(1 - 6\sin^2\theta)$$

Setting to zero on $(-\pi/4, \pi/4)$: either $\cos\theta = 0$ (not in range) or $\sin^2\theta = 1/6$, so $\sin\theta = \pm 1/\sqrt{6}$, giving $\theta = \pm\arcsin(1/\sqrt{6})$.

At $\theta = \arcsin(1/\sqrt{6})$: $r = \cos(2\theta) = 1 - 2/6 = 2/3$.

The tip of the petal is at $\theta = 0$: $r = 1$, and $dy/d\theta = \cos(0)(1 - 0) = 1 \neq 0$, $dx/d\theta = 0 - \sin(0) = 0$. So the petal tip has a **vertical tangent**, as expected by symmetry.

## 6. Tangents at the Pole

The pole ($r = 0$) requires special treatment. When $r = 0$ and $f'(\theta) \neq 0$, both $x(\theta)$ and $y(\theta)$ equal zero, and the formula for $dy/dx$ becomes $0/0$.

However, we can evaluate the limit:

$$\lim_{\theta \to \theta_0} \frac{dy/d\theta}{dx/d\theta} = \lim_{\theta \to \theta_0} \frac{f'(\theta)\sin\theta + f(\theta)\cos\theta}{f'(\theta)\cos\theta - f(\theta)\sin\theta}$$

At $\theta = \theta_0$ where $f(\theta_0) = 0$ and $f'(\theta_0) \neq 0$:

$$= \frac{f'(\theta_0)\sin\theta_0 + 0}{f'(\theta_0)\cos\theta_0 - 0} = \frac{\sin\theta_0}{\cos\theta_0} = \tan\theta_0$$

**Theorem.** If $f(\theta_0) = 0$ and $f'(\theta_0) \neq 0$, then the curve $r = f(\theta)$ passes through the pole with tangent direction $\theta = \theta_0$.

**Geometric interpretation.** The tangent line at the pole is simply the ray at angle $\theta_0$. The curve departs the pole along the ray $\theta = \theta_0$ and returns along $\theta = \theta_0 + \pi$ (if it passes through the pole again).

**Corollary.** If $f(\theta_0) = f(\theta_1) = 0$ for two distinct values $\theta_0, \theta_1 \in [0, 2\pi)$, and $f'(\theta_0), f'(\theta_1) \neq 0$, then the curve crosses itself at the pole and the two tangent directions are $\theta_0$ and $\theta_1$.

### 6.1 Example: Rose $r = \sin(2\theta)$

The curve passes through the pole at $\theta = 0, \pi/2, \pi, 3\pi/2$.

$f'(\theta) = 2\cos(2\theta)$.

- $\theta = 0$: $f'(0) = 2 \neq 0$. Tangent direction $\theta = 0$ (positive $x$-axis).
- $\theta = \pi/2$: $f'(\pi/2) = 2\cos(\pi) = -2 \neq 0$. Tangent direction $\theta = \pi/2$ (positive $y$-axis).
- $\theta = \pi$: tangent direction $\theta = \pi$.
- $\theta = 3\pi/2$: tangent direction $\theta = 3\pi/2$.

The four tangent lines at the pole are the coordinate axes — and they are mutually perpendicular, reflecting the 4-fold symmetry of the rose. ✓

### 6.2 Example: Cardioid $r = 1 + \cos\theta$ at the Pole

$f(\pi) = 0$, $f'(\pi) = -\sin(\pi) = 0$. Both vanish! The cardioid is not smooth at the pole — it has a **cusp**. This is the characteristic shape of the cardioid's pointed bottom.

To analyze the cusp, higher-order terms are needed (expand $f(\theta)$ around $\theta = \pi$).

```
Smooth crossing at pole:              Cusp at pole:
  r = sin(2θ)                           r = 1 + cosθ

    \  |  /                                  |
     \ | /                                  /|\
      \|/     ← four tangent lines          / | \
  ────●────                              ───●───   (pole)
      /|\                                   |
     / | \                              tangent undefined
    /  |  \
```

## 7. The Angle $\psi$ Between the Radius Vector and the Tangent

This is an elegant result not always covered in introductory courses, but it appears naturally and is worth knowing.

**Definition.** Let $\psi$ be the angle from the radius vector $\overrightarrow{OP}$ (the ray at angle $\theta$) to the tangent line at $P$, measured counter-clockwise.

**Theorem.**

$$\tan\psi = \frac{r}{dr/d\theta} = \frac{r}{r'}$$

**Proof.** The radius vector makes angle $\theta$ with the polar axis. The tangent makes angle $\phi$ with the polar axis, where $\tan\phi = dy/dx$. So $\psi = \phi - \theta$.

$$\tan\psi = \tan(\phi - \theta) = \frac{\tan\phi - \tan\theta}{1 + \tan\phi\tan\theta}$$

Substituting $\tan\phi = \frac{r'\sin\theta + r\cos\theta}{r'\cos\theta - r\sin\theta}$ and $\tan\theta = \sin\theta/\cos\theta$, after simplification:

$$\tan\psi = \frac{r}{r'} \qquad \square$$

**Geometric consequences:**

- If $r' = 0$ at a point, then $\tan\psi \to \infty$, i.e., $\psi = \pi/2$: the tangent is _perpendicular to the radius_. This occurs at the tip of a rose petal (where $r$ is maximum).
- If $r = 0$ (at the pole), $\tan\psi = 0$, i.e., $\psi = 0$: the tangent _coincides with the radius vector_ — confirming the theorem of §6.
- For the equiangular (logarithmic) spiral $r = ae^{b\theta}$: $r' = abe^{b\theta} = br$, so $\tan\psi = r/(br) = 1/b = \text{const}$. The angle $\psi$ is constant for all points on the spiral — this is the defining property of the logarithmic spiral.

```
The angle ψ:

         tangent line
        /
       /  ← ψ (angle between radius and tangent)
      /ψ
─────●──────────────→ radius vector (angle θ)
     P

For log spiral r = ae^{bθ}: ψ = arctan(1/b) everywhere.
Self-similar: the spiral "looks the same" from every point.
```

## 8. Second Derivative and Concavity

For completeness: the second derivative $d^2y/dx^2$ in polar coordinates. Since $dy/dx$ is itself a function of $\theta$:

$$\frac{d^2y}{dx^2} = \frac{d}{dx}\left(\frac{dy}{dx}\right) = \frac{\frac{d}{d\theta}\left(\frac{dy}{dx}\right)}{\frac{dx}{d\theta}}$$

Let $Q(\theta) = dy/dx$ (computed by the formula in §1). Then:

$$\frac{d^2y}{dx^2} = \frac{Q'(\theta)}{dx/d\theta} = \frac{Q'(\theta)}{r'\cos\theta - r\sin\theta}$$

This is rarely needed in introductory work but appears in curvature calculations.

**Curvature formula in polar.** The signed curvature of $r = f(\theta)$ is:

$$\kappa = \frac{r^2 + 2(r')^2 - rr''}{[r^2 + (r')^2]^{3/2}}$$

This follows from the parametric curvature formula $\kappa = (x'y'' - y'x'')/[(x')^2 + (y')^2]^{3/2}$ applied to the polar parametrization.

## 9. Complete Worked Example: Limaçon $r = 1 + 2\sin\theta$

This limaçon has an inner loop (since $|2| > |1|$). Let's find all tangents and understand the inner loop geometrically.

$f(\theta) = 1 + 2\sin\theta$, $\quad f'(\theta) = 2\cos\theta$.

$$\frac{dy}{d\theta} = 2\cos\theta\sin\theta + (1+2\sin\theta)\cos\theta = \cos\theta(2\sin\theta + 1 + 2\sin\theta) = \cos\theta(1 + 4\sin\theta)$$

$$\frac{dx}{d\theta} = 2\cos\theta\cos\theta - (1+2\sin\theta)\sin\theta = 2\cos^2\theta - \sin\theta - 2\sin^2\theta$$

$$= 2(\cos^2\theta - \sin^2\theta) - \sin\theta = 2\cos(2\theta) - \sin\theta$$

**Horizontal tangents:** $\cos\theta(1 + 4\sin\theta) = 0$.

- $\cos\theta = 0$: $\theta = \pi/2$ or $3\pi/2$.
    - $\theta = \pi/2$: $r = 1 + 2 = 3$. Check $dx/d\theta = 2(-1) - 1 = -3 \neq 0$. ✓ Point: $(0, 3)$.
    - $\theta = 3\pi/2$: $r = 1 - 2 = -1$. $r < 0$: actual point is at distance 1 in direction $3\pi/2 + \pi = \pi/2$, so $(0, 1)$. Check $dx/d\theta = 2(-1) - (-1) = -1 \neq 0$. ✓
- $\sin\theta = -1/4$: $\theta = \pi + \arcsin(1/4)$ or $\theta = -\arcsin(1/4)$ (i.e., $2\pi - \arcsin(1/4)$).
    - At $\theta = -\arcsin(1/4)$: $r = 1 + 2(-1/4) = 1/2$.
    - At $\theta = \pi + \arcsin(1/4)$: $r = 1 + 2(-1/4) = 1/2$.

**Inner loop.** The inner loop forms when $r < 0$, which occurs when $1 + 2\sin\theta < 0$, i.e., $\sin\theta < -1/2$, i.e., $\theta \in (7\pi/6, 11\pi/6)$.

The inner loop passes through the pole at $\theta = 7\pi/6$ and $\theta = 11\pi/6$.

Tangent at pole for $\theta = 7\pi/6$: direction $7\pi/6$ (i.e., $210°$). Tangent at pole for $\theta = 11\pi/6$: direction $11\pi/6$ (i.e., $330°$).

These two tangent lines cross at the pole at angle $|330° - 210°| = 120°$ to each other — the inner loop self-intersects at the pole with a $120°$ crossing angle.

```
Limaçon r = 1 + 2sinθ:

      (0,3) ← horizontal tangent
        •
       / \       outer loop
      /   \
─────●─────● ← (r=0 crossings, pole)
      \   /
       \_/   ← inner loop
       
The inner loop lives where r < 0 (θ ∈ (7π/6, 11π/6))
Self-intersection at pole, crossing angle = 120°
```

## 12. Connection to AI / Machine Learning / Deep Learning

The mathematics of tangent lines in polar coordinates connects to ML in ways that are structural, not cosmetic.

### 12.1 Gradient as a Tangent Object

In machine learning, the gradient $\nabla_\theta \mathcal{L}$ is the tangent direction of steepest ascent of the loss $\mathcal{L}$ as a function of model parameters $\theta$. When the parameter space is naturally polar (e.g., phase angles in complex-valued networks, or directions in weight space), the formula for $dy/dx$ in polar coordinates is literally the expression for how the loss changes as you move tangentially along a polar curve in parameter space.

More precisely: if the loss landscape restricted to a 2D subspace of parameter space is described by a polar curve $r = f(\phi)$ (level set or gradient flow curve), then:

$$\frac{d\mathcal{L}}{d\phi}\bigg|_{\text{tangent}} = \frac{dy/d\phi}{dx/d\phi} \cdot \cos\alpha - \sin\alpha$$

where $\alpha$ encodes the parameterization choice. This is exactly the polar tangent formula in a different notation.

### 12.2 Loss Surface Curvature and $\kappa$

The curvature formula $\kappa = \frac{r^2 + 2(r')^2 - rr''}{[r^2+(r')^2]^{3/2}}$ measures how rapidly the tangent direction rotates. In the loss landscape:

- **High curvature** = sharp minimum = poor generalization (the Hessian has large eigenvalues).
- **Low curvature** = flat minimum = better generalization.

SAM (Sharpness-Aware Minimization) explicitly seeks minima with small $\kappa$ along all directions — i.e., it minimizes the maximum curvature of the loss surface, exactly the quantity $\kappa$ measures in polar parametrizations of 2D cross-sections.

### 12.3 Logarithmic Spiral and Adam Optimizer

The logarithmic spiral $r = ae^{b\theta}$ has constant $\psi = \arctan(1/b)$ — the tangent makes a fixed angle with the radius at every point. This self-similar structure means the spiral "looks the same" at every scale.

The **Adam optimizer** update has a similar self-similar property: by normalizing the gradient by its running second moment, Adam implicitly adapts the step length to be proportional to the "radius" in parameter space from the initialization. The effective trajectory of Adam in a quadratic loss is approximately a logarithmic spiral toward the minimum — the angle $\psi$ is governed by the ratio of the learning rate to the second-moment estimate.

### 12.4 Tangents and Decision Boundaries

A classification decision boundary $f(x) = 0$ in 2D can often be expressed in polar coordinates, especially for radially-structured data. The **tangent to the decision boundary** at a point $P$ determines the local geometry of the margin.

For an SVM with RBF kernel, the decision boundary in polar coordinates near the support vectors behaves like a smooth polar curve. The condition for a **support vector** is that the margin is exactly 1 — geometrically, the tangent to the decision boundary at the support vector is perpendicular to the weight vector $\mathbf{w}$. In polar terms: $\psi = \pi/2$, i.e., $r' = 0$ (a petal tip condition).

### 12.5 Neural Tangent Kernel (NTK)

The **Neural Tangent Kernel** $\Theta(x, x') = \nabla_W f_W(x) \cdot \nabla_W f_W(x')$ measures the inner product of gradients (tangent vectors) in parameter space. In the infinite-width limit, training a neural network is equivalent to kernel regression with kernel $\Theta$.

The tangent vector $\nabla_W f_W(x)$ at input $x$ is precisely the derivative of the output with respect to parameters — the "tangent direction" of the output function as the network moves through parameter space. The polar tangent formula computes the same object in a 2D parametric setting.

### 12.6 Summary Table

|Polar Tangent Concept|ML / DL Analogue|
|---|---|
|$dy/dx$ via parametric chain rule|Gradient via chain rule (backprop)|
|Horizontal tangent: $dy/d\theta = 0$|Gradient zero in output direction|
