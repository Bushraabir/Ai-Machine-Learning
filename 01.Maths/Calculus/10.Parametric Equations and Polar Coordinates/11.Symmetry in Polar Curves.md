## 0. Why Symmetry Deserves Its Own Note

In Cartesian coordinates, symmetry tests are almost trivial: replace $x \to -x$ or $y \to -y$ and check if the equation is unchanged. In polar coordinates, the situation is **genuinely more subtle** вҖ” and more interesting. Because the polar representation of a point is non-unique, the _failure_ of a symmetry test does not mean the symmetry is absent. A curve can possess a symmetry that no algebraic substitution reveals directly.

This is not a quirk to be swept under the rug. It is a precise mathematical phenomenon, and understanding it forces you to think carefully about the difference between a _curve_ (a set of points) and its _equation_ (a representation).

---

## 2. The Three Standard Symmetries

There are three symmetries that appear naturally in polar curve analysis, corresponding to the three fundamental reflections of the plane.

## 3. Symmetry Tests: The Sufficient Conditions

For each symmetry, there is a **sufficient algebraic test**. These are conditions on the equation $r = f(\theta)$ that _guarantee_ the symmetry. Crucially, failing the test does not rule out the symmetry.

---

### 3.1 Symmetry About the Polar Axis (x-axis)

**Geometric meaning.** The curve is unchanged under the reflection $(x, y) \mapsto (x, -y)$.

**In polar terms:** the reflection of $(r, \theta)$ is $(r, -\theta)$. Equivalently (using the non-uniqueness), $(-r, \pi - \theta)$.

**Sufficient Test.**

$$\text{If } f(-\theta) = f(\theta), \text{ then the curve } r = f(\theta) \text{ is symmetric about the polar axis.}$$

**Proof.** If $(r_0, \theta_0)$ lies on the curve, then $r_0 = f(\theta_0)$. We need the reflected point $(r_0, -\theta_0)$ to also lie on the curve. Indeed: $f(-\theta_0) = f(\theta_0) = r_0$. So $(r_0, -\theta_0)$ satisfies $r = f(\theta)$. $\square$

**Secondary test** (using non-uniqueness): If replacing $(r, \theta)$ with $(-r, \pi - \theta)$ leaves the equation unchanged, the curve is also symmetric about the polar axis.

**Example.** $r = 1 + \cos\theta$.

$f(-\theta) = 1 + \cos(-\theta) = 1 + \cos\theta = f(\theta)$. вң“ Symmetric about polar axis.

**Example.** $r = \cos(2\theta)$.

$f(-\theta) = \cos(-2\theta) = \cos(2\theta) = f(\theta)$. вң“ Symmetric about polar axis.

---

### 3.2 Symmetry About the Line $\theta = \pi/2$ (y-axis)

**Geometric meaning.** The curve is unchanged under the reflection $(x, y) \mapsto (-x, y)$.

**In polar terms:** the reflection of $(r, \theta)$ is $(r, \pi - \theta)$. Equivalently, $(-r, -\theta)$.

**Sufficient Test.**

$$\text{If } f(\pi - \theta) = f(\theta), \text{ then the curve is symmetric about } \theta = \pi/2.$$

**Proof.** If $(r_0, \theta_0)$ is on the curve, we need $(r_0, \pi - \theta_0)$ on the curve. $f(\pi - \theta_0) = f(\theta_0) = r_0$. $\square$

**Secondary test:** replacing $(r, \theta)$ with $(-r, -\theta)$ leaves the equation unchanged.

**Example.** $r = 1 + \sin\theta$.

$f(\pi - \theta) = 1 + \sin(\pi - \theta) = 1 + \sin\theta = f(\theta)$. вң“ Symmetric about $\theta = \pi/2$.

### 3.3 Symmetry About the Pole (Origin)

**Geometric meaning.** The curve is unchanged under the map $(x, y) \mapsto (-x, -y)$, i.e., rotation by $\pi$.

**In polar terms:** the reflection of $(r, \theta)$ through the pole is $(-r, \theta)$, which is the same point as $(r, \theta + \pi)$.

**Sufficient Test.**

$$\text{If } f(\theta + \pi) = f(\theta), \text{ then the curve is symmetric about the pole.}$$

Or equivalently: if replacing $r$ with $-r$ leaves the equation unchanged (i.e., $f(-\theta)$ ... actually: if $-r = f(\theta)$ implies $r = f(\theta)$, i.e., the equation is even in $r$).

More precisely:

$$\text{If } f(\theta + \pi) = f(\theta) \text{ for all } \theta, \quad \text{OR} \quad \text{if replacing } (r,\theta) \text{ with } (-r, \theta) \text{ leaves } F(r,\theta)=0 \text{ unchanged.}$$

**Proof.** Under $f(\theta + \pi) = f(\theta)$: if $(r_0, \theta_0)$ is on the curve, then $(r_0, \theta_0 + \pi)$ represents the point diametrically opposite. $f(\theta_0 + \pi) = f(\theta_0) = r_0$, so this opposite point is also on the curve. $\square$

**Example.** $r^2 = \cos(2\theta)$ (lemniscate of Bernoulli).

Replace $r$ with $-r$: $(-r)^2 = r^2 = \cos(2\theta)$. Equation unchanged. вң“ Symmetric about the pole.

**Example.** $r = \cos(3\theta)$ (3-petal rose).

$f(\theta + \pi) = \cos(3\theta + 3\pi) = \cos(3\theta)\cos(3\pi) - \sin(3\theta)\sin(3\pi) = -\cos(3\theta) \neq f(\theta)$ in general.

So the sufficient test _fails_. But the curve still has... wait. Let's check: the 3-petal rose has 3 petals at $120В°$ apart. It is symmetric about the polar axis (check: $f(-\theta) = \cos(-3\theta) = \cos(3\theta)$ вң“), and about $\theta = \pi/3$ and $\theta = 2\pi/3$... but NOT about the pole. A 3-petal rose does not have 180В° rotational symmetry вҖ” rotating by $\pi$ maps petal to gap.

## 4. The Critical Subtlety: Sufficient вү  Necessary

This is the heart of the matter. The three tests above are **sufficient conditions**. A curve can fail all three tests and still possess a symmetry, because the same geometric curve can be represented by multiple equations.

### 4.1 A Canonical Example

Consider $r = \sin(2\theta)$ (a 4-petal rose).

**Test for symmetry about the pole:** $f(\theta + \pi) = \sin(2(\theta+\pi)) = \sin(2\theta + 2\pi) = \sin(2\theta) = f(\theta)$. вң“ Passes.

**Test for symmetry about polar axis:** $f(-\theta) = \sin(-2\theta) = -\sin(2\theta) \neq f(\theta)$ in general. вң— Fails.

Yet clearly $r = \sin(2\theta)$ _is_ symmetric about the polar axis вҖ” the graph is a symmetric 4-petal rose. Why does the test fail?

Because when $\theta$ is replaced by $-\theta$, we get $r = -\sin(2\theta)$. This is _not_ the same equation as $r = \sin(2\theta)$. However, the point $(-\sin(2\theta), \theta)$ in polar coordinates is the _same geometric point_ as $(\sin(2\theta), \theta + \pi)$. And $\sin(2(\theta + \pi)) = \sin(2\theta)$. So the reflected point does lie on the curve вҖ” but you need the alternative representation to see it.

**The correct procedure for $r = \sin(2\theta)$ and polar axis symmetry:**

The reflection of $(r, \theta)$ is $(r, -\theta)$. But equivalently it is $(-r, \pi - \theta)$.

Check the second form: does $r = -\sin(2(\pi - \theta))$... actually let's be systematic.

The reflected point $(r_0, \theta_0)$ through the polar axis is $(r_0, -\theta_0)$. For this to lie on the curve we need $r_0 = \sin(-2\theta_0) = -\sin(2\theta_0)$. But $r_0 = \sin(2\theta_0)$. So we need $\sin(2\theta_0) = -\sin(2\theta_0)$, i.e., $\sin(2\theta_0) = 0$. This is not always true.

But! The _point_ $(r_0, -\theta_0)$ can also be written as $(-r_0, \pi - \theta_0)$. Does $(-r_0, \pi - \theta_0)$ lie on $r = \sin(2\theta)$?

We need $-r_0 = \sin(2(\pi - \theta_0)) = \sin(2\pi - 2\theta_0) = -\sin(2\theta_0) = -r_0$. вң“

Yes! So the reflected point, in its alternative representation, satisfies the equation. The symmetry is real; the naГҜve test just didn't catch it because it only tried one representation.

```
Summary of what can happen:

  Algebraic test passes  вҶ’  Symmetry is definitely present. вң“
  Algebraic test fails   вҶ’  Symmetry may STILL be present.
                            Must check alternative representations.
```

## 5. Complete Decision Procedure

To rigorously determine whether $r = f(\theta)$ is symmetric about the polar axis, you should check **both** forms:

**About polar axis:** symmetric if _either_

1. $f(-\theta) = f(\theta)$ (primary), _or_
2. Replacing $(r, \theta)$ by $(-r, \pi - \theta)$ leaves the equation unchanged: $-r = f(\pi - \theta)$ is equivalent to the original.

**About $\theta = \pi/2$:** symmetric if _either_

1. $f(\pi - \theta) = f(\theta)$ (primary), _or_
2. Replacing $(r, \theta)$ by $(-r, -\theta)$: $-r = f(-\theta)$ is equivalent to the original.

**About the pole:** symmetric if _either_

1. $f(\theta + \pi) = f(\theta)$ (primary), _or_
2. Replacing $r$ by $-r$ leaves the equation unchanged: $-r = f(\theta)$ implies $r = f(\theta)$.

## 6. Important Logical Consequence

**Proposition.** If a curve has any two of the three symmetries, it automatically has the third.

**Proof.** Let $S_1$ = symmetry about polar axis, $S_2$ = symmetry about $\theta = \pi/2$, $S_3$ = symmetry about pole.

$S_3$ = (rotation by $\pi$) = (reflection in $x$-axis) $\circ$ (reflection in $y$-axis) = $S_1 \circ S_2$.

So: $S_1$ and $S_2 \Rightarrow S_3$. Similarly, $S_1$ and $S_3 \Rightarrow S_2$, and $S_2$ and $S_3 \Rightarrow S_1$. $\square$

**Practical use.** If you establish polar axis symmetry and $\theta = \pi/2$ symmetry, you get symmetry about the pole for free.

## 7. Symmetry and Integration: The Payoff

The real value of symmetry in calculus is **halving (or quartering) integrals**.

**Theorem.** If $r = f(\theta)$ is symmetric about the polar axis, then the area enclosed is:

$$A = 2 \cdot \frac{1}{2} \int_0^\pi [f(\theta)]^2, d\theta = \int_0^\pi [f(\theta)]^2, d\theta$$

instead of integrating over $[0, 2\pi]$.

**General principle.** If the curve has an $n$-fold rotational symmetry (e.g., a $2n$-petal rose), integrate over $1/n$-th of the full period and multiply by $n$.

### 7.1 Example: Area of $r = \cos(2\theta)$ (4-petal rose)

The rose $r = \cos(2\theta)$ has 4-fold symmetry (symmetric about both axes and the pole). Compute the area of one petal, then multiply by 4.

One petal exists for $\theta \in [-\pi/4, \pi/4]$ (where $r \geq 0$):

$$A_{\text{petal}} = \frac{1}{2}\int_{-\pi/4}^{\pi/4} \cos^2(2\theta), d\theta = \frac{1}{2}\int_{-\pi/4}^{\pi/4} \frac{1 + \cos(4\theta)}{2}, d\theta$$

$$= \frac{1}{4}\left[\theta + \frac{\sin(4\theta)}{4}\right]_{-\pi/4}^{\pi/4} = \frac{1}{4}\left[\frac{\pi}{4} + 0 - \left(-\frac{\pi}{4} + 0\right)\right] = \frac{1}{4} \cdot \frac{\pi}{2} = \frac{\pi}{8}$$

Total area $= 4 \cdot \frac{\pi}{8} = \frac{\pi}{2}$.

Without symmetry, you would integrate over $[0, 2\pi]$ and carefully subtract the $r < 0$ contributions вҖ” far more error-prone.

### 7.2 Example: Area of a LimaГ§on $r = 2 + \cos\theta$

Symmetric about polar axis ($f(-\theta) = 2 + \cos(-\theta) = 2 + \cos\theta$):

$$A = \int_0^\pi (2 + \cos\theta)^2, d\theta = \int_0^\pi (4 + 4\cos\theta + \cos^2\theta), d\theta$$

$$= \int_0^\pi \left(4 + 4\cos\theta + \frac{1 + \cos(2\theta)}{2}\right)d\theta = \left[4\theta + 4\sin\theta + \frac{\theta}{2} + \frac{\sin(2\theta)}{4}\right]_0^\pi$$

$$= 4\pi + 0 + \frac{\pi}{2} + 0 = \frac{9\pi}{2}$$

## 9. $n$-fold Rotational Symmetry of Rose Curves

This deserves special treatment. The curve $r = \cos(n\theta)$ has a rotational symmetry of order $n$ (if $n$ is odd) or $2n$ (if $n$ is even):

**Odd $n$:** $r = \cos(n\theta)$ has $n$ petals. It has $n$-fold symmetry (invariant under rotation by $2\pi/n$). It passes through the pole $n$ times (once per petal).

**Even $n$:** $r = \cos(n\theta)$ has $2n$ petals. It has $2n$-fold symmetry. The extra petals arise because when $r < 0$, the point plots on the opposite side.

**Why does even $n$ double the petals?**

At $\theta \in [0, 2\pi]$, the function $\cos(n\theta)$ takes both positive and negative values. When $\cos(n\theta) < 0$, we have $r < 0$, which plots the point in the direction $\theta + \pi$. So the "negative" values of $r$ trace petals that the "positive" values didn't. For even $n$, these negative petals fall in the _gaps_ between the positive petals, yielding $2n$ total. For odd $n$, the negative petals overlap with the positive petals (same directions), yielding only $n$ total.

```
r = cos(2Оё): 4 petals          r = cos(3Оё): 3 petals

     вңҝ                               вқ§
  (even n)                        (odd n)
  negative r fills the gaps      negative r overlaps
```

## 11. Connection to AI / Machine Learning / Deep Learning

Symmetry in polar curves is not merely an analytic convenience. It is a special case of the pervasive principle of **equivariance and invariance** that structures modern deep learning.

### 11.1 Symmetry as a Form of Prior Knowledge

When we observe that $r = f(-\theta) = f(\theta)$ (polar axis symmetry), we are encoding that **the function does not depend on the sign of the angle** вҖ” only the magnitude. This is a constraint on the function space. In ML terms:

A model is **invariant** to a transformation $T$ if $f(T(x)) = f(x)$. A model is **equivariant** to $T$ if $f(T(x)) = T'(f(x))$ for some related transformation $T'$.

The polar axis symmetry of a loss function $\mathcal{L}(\theta)$ (say, as a function of a 2D parameter) means the loss has the same value at conjugate angles вҖ” the optimizer can exploit this to reduce search space.

### 11.2 Symmetry and Data Augmentation

If the true data-generating distribution has polar axis symmetry (e.g., a physical system that is symmetric about a fixed axis), then **data augmentation by reflection** is valid and provably optimal. The augmented training set is the orbit of the data under the symmetry group.

This is exactly what is done in image classification: horizontal flips (axis symmetry), 90В° rotations (4-fold symmetry), etc. The theory is: if the distribution has symmetry $G$, a model that is $G$-invariant requires exponentially less data than a generic model (Vapnik-Chervonenkis theory with symmetry constraints).

### 11.3 Symmetric Kernels

A kernel $k(x, x')$ that depends only on $|x - x'|$ is **radially symmetric** вҖ” it is symmetric about every axis through the origin in the difference space. The Gram matrix $K_{ij} = k(x_i, x_j)$ inherits this symmetry: it is symmetric ($K = K^T$), which is why SVMs and Gaussian Processes are PSD (positive semidefinite) by construction.

In polar terms: a kernel $k(r)$ that is a function of $r = |x - x'|$ alone is invariant under all rotations вҖ” the maximal possible symmetry in polar coordinates.

### 11.4 Even/Odd Decomposition and Neural Activation Functions

Any function can be decomposed into its even and odd parts:

$$f(\theta) = \underbrace{\frac{f(\theta) + f(-\theta)}{2}}_{\text{even part}} + \underbrace{\frac{f(\theta) - f(-\theta)}{2}}_{\text{odd part}}$$

The even part has polar axis symmetry; the odd part is anti-symmetric.

This decomposition appears directly in activation function analysis:

- **ReLU**: $\text{ReLU}(x) = \frac{x + |x|}{2}$. The even part is $|x|/2$, the odd part is $x/2$. ReLU is neither even nor odd.
- **Tanh**: an odd function ($\tanh(-x) = -\tanh(x)$). This means networks with tanh activations can represent odd functions exactly with one layer, but need extra width for even functions.
- **GELU** and **Swish**: approximately odd near zero, breaking symmetry away from origin.

The Fourier features $\cos(n\theta)$ (even, polar axis symmetric) and $\sin(n\theta)$ (odd, anti-symmetric) form the complete basis for $L^2([0, 2\pi])$ вҖ” and their even/odd nature is exactly why they are mutually orthogonal.

### 11.5 Equivariant Neural Networks for Molecular Data

In **molecular property prediction** (drug discovery, materials science), molecules are described by 3D atomic coordinates. The energy of a molecule is invariant under rotation and reflection вҖ” it has full SO(3) $\times$ $\mathbb{Z}_2$ symmetry (the symmetry group of a sphere in polar coordinates).

Models like **SchNet**, **DimeNet**, and **SE(3)-Transformers** are architecturally constrained to respect this symmetry. In spherical coordinates $(r, \theta, \phi)$, the angular part decomposes into **spherical harmonics** $Y_\ell^m(\theta, \phi)$ вҖ” the 3D generalization of the polar rose curves $r = \cos(n\theta)$. An equivariant network maps spherical harmonic features to spherical harmonic features, preserving the symmetry exactly.

**Problem 1.** Determine all symmetries of $r = 3 - 3\sin\theta$.

_Polar axis:_ $f(-\theta) = 3 - 3\sin(-\theta) = 3 + 3\sin\theta \neq f(\theta)$. Secondary: $-r = f(\pi - \theta)$? $3 - 3\sin(\pi - \theta) = 3 - 3\sin\theta = f(\theta)$. That's not $-r = f(\pi-\theta)$... Let's just test: is $(r, \pi - \theta)$ on the curve when $(r, \theta)$ is? We need $r = 3 - 3\sin(\pi - \theta) = 3 - 3\sin\theta$ вң“. So symmetric about $\theta = \pi/2$.

_$\theta = \pi/2$:_ $f(\pi - \theta) = 3 - 3\sin(\pi - \theta) = 3 - 3\sin\theta = f(\theta)$. вң“ Symmetric about $\theta = \pi/2$.

_Pole:_ $f(\theta + \pi) = 3 - 3\sin(\theta + \pi) = 3 + 3\sin\theta \neq f(\theta)$. No.

**Answer:** Symmetric about $\theta = \pi/2$ only (it is the upward-opening cardioid).

---

**Problem 2.** $r^2 = 4\sin(2\theta)$. Find all symmetries.

_Polar axis:_ Replace $\theta \to -\theta$: $r^2 = 4\sin(-2\theta) = -4\sin(2\theta)$. Since $r^2 \geq 0$ and $-4\sin(2\theta)$ can be negative, this is not the same. Secondary: replace $(r, \theta) \to (-r, \pi - \theta)$: $(-r)^2 = 4\sin(2(\pi - \theta)) = 4\sin(2\pi - 2\theta) = -4\sin(2\theta)$. Still not the same. вң—

_$\theta = \pi/2$:_ $f(\pi - \theta)$: $r^2 = 4\sin(2(\pi-\theta)) = 4\sin(2\pi - 2\theta) = -4\sin(2\theta)$. вң— Primary fails. Secondary $(-r, -\theta)$: $r^2 = 4\sin(-2\theta) = -4\sin(2\theta)$. вң—

_Pole:_ Replace $r \to -r$: $(-r)^2 = r^2$. Equation unchanged. вң“ Symmetric about pole.

**Answer:** Symmetric about the pole only. (This lemniscate is tilted 45В° relative to the standard one.)

**Problem 3.** Compute the total area enclosed by $r = \sin(3\theta)$ using symmetry.

The 3-petal rose. Check: $f(-\theta) = \sin(-3\theta) = -\sin(3\theta) \neq f(\theta)$. Secondary: $-r = f(\pi - \theta)$: $-r = \sin(3\pi - 3\theta) = \sin(3\pi)\cos(3\theta) - \cos(3\pi)\sin(3\theta) = 0 + \sin(3\theta)$. So $-r = \sin(3\theta)$, i.e., the reflected point $(-r, \pi - \theta)$ satisfies the equation $r = \sin(3\theta)$ вң“ (since $(-r, \pi-\theta)$ means $r' = -r = \sin(3\theta)$ when we rename $r' \to r$... this requires care).

Actually let's just note: $\sin(3\theta) = 0$ at $\theta = 0, \pi/3, 2\pi/3, \pi, \ldots$. Petals lie on $\theta = \pi/6, \pi/2, 5\pi/6$. The curve is symmetric about the $y$-axis (the test via $(-r, -\theta)$: $r = \sin(3\theta)$ and the reflected point $(-r, -\theta)$: does $-r = \sin(-3\theta) = -\sin(3\theta)$, i.e., $r = \sin(3\theta)$ вң“). So symmetric about polar axis via this secondary form.

By symmetry, all 3 petals have equal area. Compute one petal on $[0, \pi/3]$ (where $r \geq 0$):

$$A_{\text{petal}} = \frac{1}{2}\int_0^{\pi/3}\sin^2(3\theta), d\theta = \frac{1}{4}\int_0^{\pi/3}(1 - \cos(6\theta)), d\theta$$

$$= \frac{1}{4}\left[\theta - \frac{\sin(6\theta)}{6}\right]_0^{\pi/3} = \frac{1}{4}\left[\frac{\pi}{3} - 0\right] = \frac{\pi}{12}$$

Total $= 3 \cdot \frac{\pi}{12} = \frac{\pi}{4}$.

## 13. Summary

```
THREE SYMMETRIES IN POLAR COORDINATES:

  1. About polar axis (x-axis):
       Primary test:    f(-Оё) = f(Оё)
       Secondary test:  replacing (r,Оё) with (-r, ПҖ-Оё) preserves equation

  2. About Оё = ПҖ/2 (y-axis):
       Primary test:    f(ПҖ-Оё) = f(Оё)
       Secondary test:  replacing (r,Оё) with (-r, -Оё) preserves equation

  3. About pole (origin):
       Primary test:    f(Оё+ПҖ) = f(Оё)  OR  replacing r with -r preserves equation

  KEY LOGICAL FACTS:
    вҖў Any 2 of the 3 symmetries imply the 3rd
    вҖў Failing primary test вү  no symmetry (non-uniqueness of polar representation)
    вҖў Always try both primary and secondary tests before concluding

  INTEGRATION PAYOFF:
    вҖў 1 symmetry  вҶ’  integrate over half the domain, Г—2
    вҖў 2 symmetries вҶ’  integrate over quarter domain, Г—4
    вҖў n-fold rose  вҶ’  integrate one petal, Г—n
```
