

> _A conic at the origin is a theorem. A conic at $(h,k)$ is the same theorem, translated. The algebra of shifting is just a change of variables — but mastering it cleanly is what separates someone who understands conics from someone who merely memorizes them._

---

## 1. The Core Idea: Translation of Axes

Every conic centered or vertexed at the origin has a clean standard form. When the center (or vertex, for a parabola) moves to $(h, k)$, the entire coordinate system shifts. Define:

$$X = x - h, \qquad Y = y - k$$

Then every standard-form equation in $(X, Y)$ becomes the shifted equation in $(x, y)$ by substitution:

$$X \to (x - h), \qquad Y \to (y - k)$$

This is not a new formula — it is the **same formula** with a rigid translation applied. The geometry is unchanged; only the reference point moves.

```
  y                          y
  |                          |
  |    (at origin)           |         (shifted to (h,k))
  |                          |
  +----------x          (h,k)+----------x
  |                          |
  
  X = x - h,  Y = y - k
  Origin of new system is (h, k) in the old system.
```

## 2. Shifted Parabola

### Standard Forms

|Opens|Vertex at $(h,k)$|Equation|
|---|---|---|
|Up|$(h, k)$|$(x-h)^2 = 4p(y-k)$|
|Down|$(h, k)$|$(x-h)^2 = -4p(y-k)$|
|Right|$(h, k)$|$(y-k)^2 = 4p(x-h)$|
|Left|$(h, k)$|$(y-k)^2 = -4p(x-h)$|

### Key Features (opens up, $(x-h)^2 = 4p(y-k)$)

|Feature|Location|
|---|---|
|Vertex|$(h,\ k)$|
|Focus|$(h,\ k+p)$|
|Directrix|$y = k - p$|
|Axis of symmetry|$x = h$|
|Latus rectum length|$4p$|

### Slope-intercept connection

$$y = a(x-h)^2 + k \implies p = \frac{1}{4|a|}$$

- $a > 0$: opens up; $a < 0$: opens down
- Focus: $\left(h,\ k + \dfrac{1}{4a}\right)$
- Directrix: $y = k - \dfrac{1}{4a}$

### Completing the Square (Parabola)

Given $y = Ax^2 + Bx + C$:

$$y = A!\left(x + \frac{B}{2A}\right)^{!2} + C - \frac{B^2}{4A}$$

$$\implies h = -\frac{B}{2A}, \quad k = C - \frac{B^2}{4A}, \quad p = \frac{1}{4|A|}$$

**Example.** $y = 2x^2 - 8x + 5$

$$= 2(x^2 - 4x) + 5 = 2(x-2)^2 - 8 + 5 = 2(x-2)^2 - 3$$

Vertex $(2, -3)$, opens up, $p = \frac{1}{8}$, focus $\left(2,\ -3 + \frac{1}{8}\right) = \left(2, -\frac{23}{8}\right)$.

## 3. Shifted Ellipse

### Standard Forms

|Major axis|Equation|
|---|---|
|Horizontal|$\dfrac{(x-h)^2}{a^2} + \dfrac{(y-k)^2}{b^2} = 1$, $a > b$|
|Vertical|$\dfrac{(x-h)^2}{b^2} + \dfrac{(y-k)^2}{a^2} = 1$, $a > b$|

### Key Features (horizontal major axis)

|Feature|Location|
|---|---|
|Center|$(h,\ k)$|
|Vertices|$(h \pm a,\ k)$|
|Co-vertices|$(h,\ k \pm b)$|
|Foci|$(h \pm c,\ k)$, $c = \sqrt{a^2-b^2}$|
|Directrices|$x = h \pm \dfrac{a^2}{c}$|

### Completing the Square (Ellipse)

Given $Ax^2 + Cy^2 + Dx + Ey + F = 0$ ($A, C > 0$, same sign):

**Step 1.** Group and factor:

$$A(x^2 + \tfrac{D}{A}x) + C(y^2 + \tfrac{E}{C}y) = -F$$

**Step 2.** Complete each square:

$$A!\left(x + \frac{D}{2A}\right)^{!2} + C!\left(y + \frac{E}{2C}\right)^{!2} = -F + \frac{D^2}{4A} + \frac{E^2}{4C}$$

**Step 3.** Divide both sides by the right-hand side to get the $= 1$ form.

**Example.** $4x^2 + 9y^2 - 16x + 18y - 11 = 0$

$$4(x^2 - 4x) + 9(y^2 + 2y) = 11$$

$$4(x-2)^2 - 16 + 9(y+1)^2 - 9 = 11$$

$$4(x-2)^2 + 9(y+1)^2 = 36$$

$$\frac{(x-2)^2}{9} + \frac{(y+1)^2}{4} = 1$$

Center $(2,-1)$, $a=3$, $b=2$, $c=\sqrt{5}$, horizontal major axis.

## 4. Shifted Hyperbola

### Standard Forms

|Opens|Equation|
|---|---|
|Left/Right|$\dfrac{(x-h)^2}{a^2} - \dfrac{(y-k)^2}{b^2} = 1$|
|Up/Down|$\dfrac{(y-k)^2}{a^2} - \dfrac{(x-h)^2}{b^2} = 1$|

### Key Features (horizontal transverse axis)

|Feature|Location|
|---|---|
|Center|$(h,\ k)$|
|Vertices|$(h \pm a,\ k)$|
|Foci|$(h \pm c,\ k)$, $c = \sqrt{a^2+b^2}$|
|Asymptotes|$y - k = \pm\dfrac{b}{a}(x - h)$|
|Directrices|$x = h \pm \dfrac{a^2}{c}$|

### Completing the Square (Hyperbola)

Given $Ax^2 - Cy^2 + Dx + Ey + F = 0$ ($A, C > 0$, opposite signs):

Same square-completion process as the ellipse, then divide. The sign difference between the two completed-square terms gives the hyperbola form.

**Example.** $9x^2 - 4y^2 - 36x + 8y - 4 = 0$

$$9(x^2 - 4x) - 4(y^2 - 2y) = 4$$

$$9(x-2)^2 - 36 - 4(y-1)^2 + 4 = 4$$

$$9(x-2)^2 - 4(y-1)^2 = 36$$

$$\frac{(x-2)^2}{4} - \frac{(y-1)^2}{9} = 1$$

Center $(2,1)$, $a=2$, $b=3$, $c=\sqrt{13}$, horizontal transverse axis.

Asymptotes: $y - 1 = \pm\dfrac{3}{2}(x-2)$.

## 5. The General Second-Degree Equation

The most general conic (with no rotation, $B = 0$):

$$Ax^2 + Cy^2 + Dx + Ey + F = 0$$

**Classification by signs of $A$ and $C$:**

|Condition|Conic|
|---|---|
|$A = 0$ or $C = 0$ (not both)|Parabola|
|$A = C > 0$|Circle|
|$A, C > 0$, $A \neq C$|Ellipse|
|$A$ and $C$ opposite signs|Hyperbola|

**With rotation ($B \neq 0$):** use discriminant $B^2 - 4AC$:

$$B^2 - 4AC \begin{cases} < 0 & \text{ellipse (or circle)} \ = 0 & \text{parabola} \ > 0 & \text{hyperbola} \end{cases}$$

---

## 6. The Completing-the-Square Algorithm (Universal)

Given any $Ax^2 + Cy^2 + Dx + Ey + F = 0$ (with $B = 0$):

**Step 1.** Move $F$ to the right: $$Ax^2 + Cy^2 + Dx + Ey = -F$$

**Step 2.** Group $x$-terms and $y$-terms, factor out leading coefficients: $$A!\left(x^2 + \frac{D}{A}x\right) + C!\left(y^2 + \frac{E}{C}y\right) = -F$$

_(If $A = 0$ or $C = 0$, skip that group — it's a parabola.)_

**Step 3.** Complete each square. For a group $A(x^2 + px)$, add and subtract $A\cdot\left(\frac{p}{2}\right)^2$: $$A!\left(x + \frac{p}{2}\right)^{!2} - A!\left(\frac{p}{2}\right)^{!2}$$

Move the subtracted constant to the right side.

**Step 4.** Read off $(h, k)$ as the shifts, and divide to normalize.

**Step 5.** Identify: if RHS $> 0$ and same signs → ellipse; opposite signs → hyperbola; one variable missing → parabola.

---

## 7. Degenerate Cases

When completing the square, the right-hand side $R$ determines degeneracy:

**For ellipse form** $\dfrac{(x-h)^2}{a^2} + \dfrac{(y-k)^2}{b^2} = R$:

|$R$|Result|
|---|---|
|$R > 0$|Ellipse (or circle if $a = b$)|
|$R = 0$|Single point $(h, k)$|
|$R < 0$|No real graph (imaginary ellipse)|

**For hyperbola form** $\dfrac{(x-h)^2}{a^2} - \dfrac{(y-k)^2}{b^2} = R$:

|$R$|Result|
|---|---|
|$R \neq 0$|Hyperbola|
|$R = 0$|Two intersecting lines (the asymptotes): $y-k = \pm\frac{b}{a}(x-h)$|

**For parabola form** $(x-h)^2 = 4p(y-k)$:

|Condition|Result|
|---|---|
|$p \neq 0$|Parabola|
|$p = 0$|Horizontal line (degenerate)|

## 8. Worked Examples — Full Pipeline

### Example 1: Complete Classification

$x^2 + 4y^2 + 6x - 8y + 9 = 0$

**Classify:** $A = 1, C = 4$, both positive, $A \neq C$ → **ellipse**.

**Complete the square:**

$(x^2 + 6x) + 4(y^2 - 2y) = -9$

$(x+3)^2 - 9 + 4(y-1)^2 - 4 = -9$

$(x+3)^2 + 4(y-1)^2 = 4$

$$\frac{(x+3)^2}{4} + \frac{(y-1)^2}{1} = 1$$

Center $(-3, 1)$, $a = 2$ (horizontal), $b = 1$, $c = \sqrt{3}$. Foci: $(-3 \pm \sqrt{3},\ 1)$.

---

### Example 2: Degenerate Case

$x^2 - y^2 - 4x + 2y + 3 = 0$

**Classify:** $A = 1, C = -1$, opposite signs → **hyperbola (or degenerate)**.

$(x^2 - 4x) - (y^2 - 2y) = -3$

$(x-2)^2 - 4 - (y-1)^2 + 1 = -3$

$(x-2)^2 - (y-1)^2 = 0$

$R = 0$ → **Two intersecting lines:**

$(x-2) = \pm(y-1) \implies y = x - 1 \quad \text{and} \quad y = -x + 3$

---

### Example 3: Vertical Parabola

$3x^2 - 12x - y + 9 = 0$

**Classify:** $C = 0$ → **parabola** (opens up or down).

$y = 3x^2 - 12x + 9 = 3(x^2 - 4x) + 9 = 3(x-2)^2 - 12 + 9 = 3(x-2)^2 - 3$

$(x-2)^2 = \frac{1}{3}(y+3)$

Vertex $(2, -3)$, opens up, $4p = \frac{1}{3} \Rightarrow p = \frac{1}{12}$.

Focus: $\left(2,\ -3 + \frac{1}{12}\right) = \left(2,\ -\frac{35}{12}\right)$.

## 9. Visualization

### Translation Schematic

```
   Original (at origin)        Shifted (center at (h,k))

       y                              y
       |    *                         |
       |  *   *                       |          *
       | *     *                      |        *   *
       |*       *                     |    (h,k)*     *
       +----------x              -----+--------*-------x
       |                              |
   
   Replace: x → (x-h),  y → (y-k)
   Every point moves RIGHT by h, UP by k
```

### The Three Shifted Conics

```
SHIFTED PARABOLA          SHIFTED ELLIPSE          SHIFTED HYPERBOLA
(x-h)² = 4p(y-k)     (x-h)²/a² + (y-k)²/b²=1   (x-h)²/a²-(y-k)²/b²=1

       *                      ___                    *     *
      * *                    /   \                  * *   * *
      * *                   | (h,k)|               *  |   |  *
       *   ← vertex(h,k)     \___/           ------+--+---+--+------
       |                                           *  |   |  *
       |                     center(h,k)            * *   * *
                                                     *     *
                                                   asymptotes pass
                                                   through (h,k)
```

### Completing the Square — Visual

```
  Ax² + Dx   →   A(x + D/2A)² - D²/4A

  "Half the coefficient, squared, added inside — subtracted outside"
  
  Example:  x² - 6x
          = (x-3)² - 9
          
  Visual: (x-3)² = 0 at x=3 (minimum/vertex at x=3)
```

## 11. Summary: Shifted Standard Forms at a Glance

$$\textbf{Parabola (up):} \quad (x-h)^2 = 4p(y-k)$$ $$\textbf{Parabola (right):} \quad (y-k)^2 = 4p(x-h)$$

$$\textbf{Ellipse (horizontal):} \quad \frac{(x-h)^2}{a^2} + \frac{(y-k)^2}{b^2} = 1, \quad a > b$$

$$\textbf{Ellipse (vertical):} \quad \frac{(x-h)^2}{b^2} + \frac{(y-k)^2}{a^2} = 1, \quad a > b$$

$$\textbf{Circle:} \quad (x-h)^2 + (y-k)^2 = r^2$$

$$\textbf{Hyperbola (horizontal):} \quad \frac{(x-h)^2}{a^2} - \frac{(y-k)^2}{b^2} = 1$$

$$\textbf{Hyperbola (vertical):} \quad \frac{(y-k)^2}{a^2} - \frac{(x-h)^2}{b^2} = 1$$

## 12. Connection to AI, Machine Learning, and Deep Learning

### 12.1 Shifted Gaussians and Kernel Methods

The multivariate Gaussian with mean $\boldsymbol{\mu}$ (the shift vector):

$$p(\mathbf{x}) \propto \exp!\left(-\frac{1}{2}(\mathbf{x} - \boldsymbol{\mu})^\top \Sigma^{-1}(\mathbf{x} - \boldsymbol{\mu})\right)$$

is a **shifted ellipsoid** in exponent space — the center is at $\boldsymbol{\mu}$, exactly analogous to $(h, k)$ in shifted conic form. The RBF kernel:

$$K(\mathbf{x}, \mathbf{y}) = \exp!\left(-\frac{|\mathbf{x} - \mathbf{y}|^2}{2\sigma^2}\right)$$

has level sets that are **shifted spheres** (circles in 2D) centered at $\mathbf{y}$. Anisotropic kernels use shifted ellipsoids. The entire theory of kernel methods is the theory of shifted quadratic forms.

### 12.2 Batch Normalization as a Shift-and-Scale

**Batch normalization** transforms activations $\mathbf{x}$ as:

$$\hat{x}_i = \frac{x_i - \mu_\mathcal{B}}{\sqrt{\sigma_\mathcal{B}^2 + \epsilon}}, \qquad y_i = \gamma\hat{x}_i + \beta$$

The parameter $\beta$ is a **learnable shift** — it moves the distribution of activations to the optimal center, exactly like the $(h,k)$ shift in conics. Without $\beta$, the network is forced to center its activations at zero, a severe geometric constraint. With $\beta$, the loss surface (which has ellipsoidal level sets near a minimum) can be centered at the optimal point.

### 12.3 Feature Centering in PCA and Whitening

The first step of **PCA** is always to center the data:

$$\tilde{\mathbf{x}}_i = \mathbf{x}_i - \bar{\mathbf{x}}$$

This is the ML analog of the $(h, k) \to (0, 0)$ shift: translate the data so its center of mass is at the origin, making the covariance ellipse axis-aligned and amenable to eigendecomposition. Without centering, the principal components are contaminated by the mean — exactly as a non-centered conic equation mixes the $(h,k)$ translation into the $A, B, C$ coefficients.

**Whitening** further rescales the axes so the covariance ellipse becomes a circle — the inverse of introducing $a \neq b$ in the shifted ellipse.

### 12.4 Loss Landscape Shifts and Saddle Points

Near a saddle point $\mathbf{w}_0$ in a neural network loss surface, the second-order approximation is:

$$\mathcal{L}(\mathbf{w}) \approx \mathcal{L}(\mathbf{w}_0) + \frac{1}{2}(\mathbf{w}-\mathbf{w}_0)^\top H(\mathbf{w}-\mathbf{w}_0)$$

where $H$ has both positive and negative eigenvalues. The level sets of this quadratic are **shifted hyperbolas** (in 2D) centered at $\mathbf{w}_0 = (h, k)$. Gradient descent near a saddle point follows the arms of these shifted hyperbolas — moving away from the saddle along the negative-curvature directions (the hyperbola's transverse axis) and slowly along the positive-curvature directions.

Understanding why gradient descent escapes saddle points requires understanding the geometry of shifted hyperbolic level sets.

### 12.5 Anchor-Based Object Detection

In object detection models (YOLO, Faster R-CNN, SSD), each **anchor box** defines a reference rectangle centered at a grid point $(h, k)$ — a shift of the canonical box from the origin. The model predicts **offsets** $(\Delta x, \Delta y, \Delta w, \Delta h)$ relative to the anchor, exactly the shifted-conic philosophy: work in the shifted coordinate system $(x - h, y - k)$, then translate back.

The **IoU (Intersection over Union)** loss surface as a function of these offsets has level curves that are approximately **shifted ellipses** near the optimum (when the predicted box nearly matches the ground truth).

### 12.6 Shifted Quadratics in Reinforcement Learning (Value Functions)

In **linear-quadratic regulators (LQR)** and related RL methods, the optimal value function for a quadratic cost is:

$$V(\mathbf{x}) = (\mathbf{x} - \mathbf{x}^_)^\top P (\mathbf{x} - \mathbf{x}^_)$$

a **shifted paraboloid** centered at the optimal state $\mathbf{x}^_$. This is precisely the $n$-dimensional analog of $(x-h)^2 = 4p(y-k)$. The policy derived from this value function steers toward $\mathbf{x}^_$ — the vertex of the shifted paraboloid — along the optimal trajectory.

**TD learning** and **model-based RL** approximate this shifted quadratic structure; the shift $(h, k) \equiv \mathbf{x}^*$ encodes where the agent is trying to go.
