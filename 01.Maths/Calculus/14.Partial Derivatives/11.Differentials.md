

> _"The derivative tells you the rate. The differential tells you the amount — infinitesimally. One is a number; the other is a linear map. Confusing them is the source of half the errors in applied mathematics."_

---

## §1. Motivation — Why Differentials?

In single-variable calculus, the derivative $f'(a)$ is a number. But the expression

$$dy = f'(x),dx$$

is something different: it is a **linear approximation to the change in $y$** given a small change $dx$ in $x$. This object — a differential — carries more structural information than the derivative alone. It:

- Lives in the **dual space** (acts on tangent vectors, returns scalars)
- Transforms correctly under change of variables (chain rule becomes trivial)
- Generalizes naturally to surfaces, manifolds, and high-dimensional parameter spaces
- Is the building block of integration, differential forms, and the entire apparatus of modern geometry

The goal of this note is to understand differentials rigorously, compute with them fluently, and see exactly how they power machine learning.

## §2. Differentials in One Variable

### 2.1 — The Setup

Let $y = f(x)$, and suppose $f$ is differentiable at $x$. Define:

- $dx$ — an independent variable representing an arbitrary increment in $x$
- $\Delta x = dx$ — the actual change (same thing here)
- $\Delta y = f(x + dx) - f(x)$ — the actual change in $y$

**Definition.** The **differential** of $y$ is:

$$dy = f'(x),dx$$

**Key point.** $dx$ is a free variable — it can be anything. $dy$ is then determined by it. Together, $(dx, dy)$ parameterize the tangent line at $x$.

### 2.2 — The Error

The relationship between $\Delta y$ and $dy$:

$$\Delta y = f'(x),dx + \varepsilon \cdot dx, \quad \text{where } \varepsilon \to 0 \text{ as } dx \to 0$$

More precisely, by Taylor's theorem:

$$\Delta y = dy + \underbrace{\frac{1}{2}f''(x)(dx)^2 + \cdots}_{\text{higher-order terms}}$$

So $dy$ approximates $\Delta y$ with **error $O((dx)^2)$** — quadratically small. This is what makes differentials useful for estimation.

### 2.3 — Geometric Picture

```
  y
  |          * ← actual point (x+dx, f(x+dx))
  |        /·
  |      /  ← Δy (actual rise)
  |    /·· ← dy (tangent line rise)
  |  * ← (x, f(x))
  |________________________ x
       dx
```

$dy$ is the rise along the **tangent line**; $\Delta y$ is the rise along the **curve**. They agree to first order.

## §3. Differentials in Several Variables

### 3.1 — Definition

Let $z = f(x, y)$ be differentiable at $(a, b)$. The **total differential** is:

$$\boxed{dz = f_x(x,y),dx + f_y(x,y),dy = \frac{\partial f}{\partial x},dx + \frac{\partial f}{\partial y},dy}$$

Here $dx$ and $dy$ are **independent** free variables (increments in $x$ and $y$). $dz$ is the resulting increment in $z$ along the **tangent plane**.

**Connection to tangent planes.** This is exactly the tangent plane formula in differential form. If we set $dx = x - a$, $dy = y - b$:

$$z \approx f(a,b) + dz = f(a,b) + f_x(a,b),dx + f_y(a,b),dy$$

The total differential is the tangent plane, written infinitesimally.

### 3.2 — For $n$ Variables

If $w = f(x_1, x_2, \ldots, x_n)$:

$$dw = \frac{\partial f}{\partial x_1},dx_1 + \frac{\partial f}{\partial x_2},dx_2 + \cdots + \frac{\partial f}{\partial x_n},dx_n = \nabla f \cdot d\mathbf{x}$$

This is a **linear functional** on the vector of increments $d\mathbf{x} = (dx_1, \ldots, dx_n)$.

### 3.3 — Formal Properties (Differential Rules)

Differentials obey exactly the same rules as derivatives — because they are derivatives, just written in a different notation.

|Rule|Expression|
|---|---|
|Constant|$d(c) = 0$|
|Sum|$d(f \pm g) = df \pm dg$|
|Product|$d(fg) = f,dg + g,df$|
|Quotient|$d(f/g) = \frac{g,df - f,dg}{g^2}$|
|Chain|$d(f(g)) = f'(g),dg$|
|Power|$d(x^n) = nx^{n-1},dx$|

These rules let you compute differentials **algebraically**, without restating the chain rule every time.

**Example.** $w = x^2 \sin(xy)$. Compute $dw$.

$$dw = d(x^2)\sin(xy) + x^2,d(\sin(xy))$$ $$= 2x,dx \cdot \sin(xy) + x^2 \cos(xy),d(xy)$$ $$= 2x\sin(xy),dx + x^2\cos(xy)(y,dx + x,dy)$$ $$= \left[2x\sin(xy) + x^2 y\cos(xy)\right]dx + x^3\cos(xy),dy$$

Read off: $\frac{\partial w}{\partial x} = 2x\sin(xy) + x^2 y\cos(xy)$, $\frac{\partial w}{\partial y} = x^3\cos(xy)$.

## §4. Error Estimation and Propagation

This is the most immediate application. Given a quantity $Q = f(x_1, \ldots, x_n)$ where each $x_i$ is measured with error $|\Delta x_i| \leq \delta_i$, the **propagated error** is estimated by:

$$|\Delta Q| \approx |dQ| \leq \left|\frac{\partial f}{\partial x_1}\right|\delta_1 + \cdots + \left|\frac{\partial f}{\partial x_n}\right|\delta_n$$

The inequality uses the triangle inequality on the differential.

**Relative error:**

$$\frac{|dQ|}{|Q|} \approx \left|\frac{\partial \ln f}{\partial x_1}\right|\delta_1 + \cdots$$

For products and powers, logarithmic differentiation simplifies the relative error enormously.

### Example — Resistance

Two resistors $R_1 = 100,\Omega \pm 1%$ and $R_2 = 200,\Omega \pm 1%$ in parallel. Find the error in total resistance $R = \frac{R_1 R_2}{R_1 + R_2}$.

Use logarithmic differentials. First,

$$\ln R = \ln R_1 + \ln R_2 - \ln(R_1 + R_2)$$

$$\frac{dR}{R} = \frac{dR_1}{R_1} + \frac{dR_2}{R_2} - \frac{dR_1 + dR_2}{R_1 + R_2}$$

$$= dR_1\left(\frac{1}{R_1} - \frac{1}{R_1+R_2}\right) + dR_2\left(\frac{1}{R_2} - \frac{1}{R_1+R_2}\right)$$

$$= dR_1 \cdot \frac{R_2}{R_1(R_1+R_2)} + dR_2 \cdot \frac{R_1}{R_2(R_1+R_2)}$$

With $R_1 = 100$, $R_2 = 200$, $R_1 + R_2 = 300$:

$$\frac{|dR|}{R} \leq \frac{200}{100 \cdot 300}(1) + \frac{100}{200 \cdot 300}(2) \approx 0.0067 + 0.0033 = 1%$$

So the parallel combination also has roughly $1%$ error.

## §5. The Chain Rule, Written as Differentials

Suppose $z = f(x,y)$, $x = x(t)$, $y = y(t)$. Then:

$$dz = f_x,dx + f_y,dy = f_x \frac{dx}{dt},dt + f_y \frac{dy}{dt},dt = \left(f_x\frac{dx}{dt} + f_y\frac{dy}{dt}\right)dt$$

So $\frac{dz}{dt} = f_x\frac{dx}{dt} + f_y\frac{dy}{dt}$. The chain rule is just **division of the differential by $dt$**.

This is why the notation $dy/dx$ is more than notation — it genuinely behaves like a fraction when differentials are manipulated carefully.

### Implicit Differentiation via Differentials

Given $F(x,y) = 0$, take the differential of both sides:

$$dF = F_x,dx + F_y,dy = 0 \implies \frac{dy}{dx} = -\frac{F_x}{F_y}$$

No quotient rule gymnastics. Just differentiate $F$, set $dF = 0$, solve for $dy/dx$.

**Example.** $x^3 + y^3 = 6xy$. Find $dy/dx$.

$$d(x^3 + y^3) = d(6xy)$$ $$3x^2,dx + 3y^2,dy = 6y,dx + 6x,dy$$ $$(3y^2 - 6x),dy = (6y - 3x^2),dx$$ $$\frac{dy}{dx} = \frac{6y - 3x^2}{3y^2 - 6x} = \frac{2y - x^2}{y^2 - 2x}$$

One line of algebra, no chain rule bookkeeping.

## §6. Differentials as Linear Maps — The Rigorous View

Here is the fully honest definition. Fix a point $\mathbf{a} \in \mathbb{R}^n$.

**Definition.** If $f: \mathbb{R}^n \to \mathbb{R}$ is differentiable at $\mathbf{a}$, the **differential of $f$ at $\mathbf{a}$** is the unique linear map $df_\mathbf{a}: \mathbb{R}^n \to \mathbb{R}$ such that:

$$\lim_{\mathbf{h} \to \mathbf{0}} \frac{f(\mathbf{a} + \mathbf{h}) - f(\mathbf{a}) - df_\mathbf{a}(\mathbf{h})}{|\mathbf{h}|} = 0$$

In coordinates: $df_\mathbf{a}(\mathbf{h}) = \nabla f(\mathbf{a}) \cdot \mathbf{h} = \sum_i \frac{\partial f}{\partial x_i}(\mathbf{a}), h_i$.

**What is $dx_i$?** The differential of the coordinate function $x_i: \mathbb{R}^n \to \mathbb{R}$, $(x_1, \ldots, x_n) \mapsto x_i$. Evaluated on a vector $\mathbf{h}$: $dx_i(\mathbf{h}) = h_i$. So $dx_i$ simply **extracts the $i$-th component**.

Then:

$$df = \sum_i \frac{\partial f}{\partial x_i},dx_i$$

is an equation between linear maps: both sides, when applied to $\mathbf{h}$, give $\nabla f \cdot \mathbf{h}$.

**For vector-valued $f$: the Jacobian.** If $\mathbf{f}: \mathbb{R}^n \to \mathbb{R}^m$, then $d\mathbf{f}_\mathbf{a}$ is the linear map represented by the **Jacobian matrix**:

$$J = \begin{pmatrix} \frac{\partial f_1}{\partial x_1} & \cdots & \frac{\partial f_1}{\partial x_n} \ \vdots & \ddots & \vdots \ \frac{\partial f_m}{\partial x_1} & \cdots & \frac{\partial f_m}{\partial x_n} \end{pmatrix}$$

The differential is $d\mathbf{f}_\mathbf{a}(\mathbf{h}) = J\mathbf{h}$. This is the most general form.

## §7. Second Differentials and Quadratic Approximation

The **second differential** of $f$ at $\mathbf{a}$ is:

$$d^2f = \sum_{i,j} \frac{\partial^2 f}{\partial x_i \partial x_j},dx_i,dx_j$$

This is a **quadratic form** in $dx_i$. It encodes the curvature of $f$, and the associated matrix is the **Hessian** $H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$.

The second-order Taylor expansion:

$$f(\mathbf{a} + \mathbf{h}) = f(\mathbf{a}) + df_\mathbf{a}(\mathbf{h}) + \frac{1}{2}d^2f_\mathbf{a}(\mathbf{h}) + O(|\mathbf{h}|^3)$$

$$= f(\mathbf{a}) + \nabla f \cdot \mathbf{h} + \frac{1}{2}\mathbf{h}^\top H, \mathbf{h} + \cdots$$

The differential $df$ approximates $f$ linearly; $d^2f$ adds the curvature correction.

## §8. Differential Forms (The Bigger Picture)

A **differential $k$-form** on $\mathbb{R}^n$ is an antisymmetric multilinear map on $k$ tangent vectors. The differentials $dx, dy, dz$ are 1-forms. The **exterior product** $dx \wedge dy$ is a 2-form that measures signed area.

The objects you integrate are differential forms:

- $\int_C f,dx + g,dy$ — line integral of a 1-form
- $\iint_S f,dA = \iint f,dx\wedge dy$ — surface integral of a 2-form

**Stokes' theorem** (the grand unification) becomes:

$$\int_{\partial M} \omega = \int_M d\omega$$

where $d$ is the **exterior derivative** — the coordinate-free generalization of $df$. Every theorem of vector calculus (Green's, Divergence, classical Stokes') is a special case.

The total differential $df = f_x,dx + f_y,dy$ is a 1-form. It is **exact** (equals $dg$ for some $g$) if and only if $\frac{\partial f_x}{\partial y} = \frac{\partial f_y}{\partial x}$ — the condition for a conservative vector field. This is why the cross-partial condition appears everywhere.

## §9. Worked Examples

### Example 1 — Differential Computation

Let $f(x,y,z) = x^2 e^{yz}$. Compute $df$.

$$df = 2x e^{yz},dx + x^2 z e^{yz},dy + x^2 y e^{yz},dz$$

### Example 2 — Approximation

Approximate $\sqrt{(3.98)^2 + (3.02)^2 + (5.99)^2}$ using differentials.

Base point: $(4, 3, 6)$. $f(4,3,6) = \sqrt{16+9+36} = \sqrt{61}$.

$$df = \frac{x,dx + y,dy + z,dz}{\sqrt{x^2+y^2+z^2}}$$

With $dx = -0.02$, $dy = 0.02$, $dz = -0.01$:

$$df = \frac{4(-0.02) + 3(0.02) + 6(-0.01)}{\sqrt{61}} = \frac{-0.08 + 0.06 - 0.06}{\sqrt{61}} = \frac{-0.08}{\sqrt{61}}$$

$$f \approx \sqrt{61} - \frac{0.08}{\sqrt{61}} = \frac{61 - 0.08}{\sqrt{61}} = \frac{60.92}{\sqrt{61}} \approx 7.803$$

### Example 3 — Implicit Surface

Find the tangent plane to $x^2 + 2y^2 + 3z^2 = 6$ at $(1, 1, 1)$.

Let $F = x^2 + 2y^2 + 3z^2 - 6$. Then $dF = 2x,dx + 4y,dy + 6z,dz$.

At $(1,1,1)$: $\nabla F = \langle 2, 4, 6 \rangle$. Tangent plane:

$$2(x-1) + 4(y-1) + 6(z-1) = 0 \implies x + 2y + 3z = 6$$

### Example 4 — Change of Variables

Polar coordinates: $x = r\cos\theta$, $y = r\sin\theta$.

$$dx = \cos\theta,dr - r\sin\theta,d\theta$$ $$dy = \sin\theta,dr + r\cos\theta,d\theta$$

The area element: $dA = dx \wedge dy = (\cos\theta,dr - r\sin\theta,d\theta) \wedge (\sin\theta,dr + r\cos\theta,d\theta)$

$$= \cos\theta \cdot r\cos\theta,(dr \wedge d\theta) + (-r\sin\theta)\cdot\sin\theta,(d\theta \wedge dr)$$

$$= r\cos^2\theta,(dr\wedge d\theta) + r\sin^2\theta,(dr \wedge d\theta) = r,dr\wedge d\theta$$

So $dA = r,dr,d\theta$ — the Jacobian factor $r$ emerges automatically from the differential calculus of forms. No determinant computation needed.

## §10. Connection to AI, Machine Learning, and Deep Learning

### 10.1 — Differentials Are What Backpropagation Computes

Let $\mathcal{L} = \mathcal{L}(\boldsymbol{\theta})$ be the loss of a neural network. The total differential is:

$$d\mathcal{L} = \sum_i \frac{\partial \mathcal{L}}{\partial \theta_i},d\theta_i = \nabla_{\boldsymbol{\theta}}\mathcal{L} \cdot d\boldsymbol{\theta}$$

Backpropagation computes $\nabla_{\boldsymbol{\theta}}\mathcal{L}$ — i.e., it computes the **coefficients of the total differential** of the loss with respect to every parameter. The gradient is not a mysterious vector; it is the linear map $d\mathcal{L}$, written in coordinates.

### 10.2 — The Jacobian and Vector Differentials

A neural network layer $\mathbf{f}: \mathbb{R}^n \to \mathbb{R}^m$ computes $\mathbf{y} = \sigma(W\mathbf{x} + \mathbf{b})$. Its differential at $\mathbf{x}$:

$$d\mathbf{y} = \text{diag}(\sigma'(W\mathbf{x}+\mathbf{b})) \cdot W,d\mathbf{x}$$

This is the Jacobian acting on $d\mathbf{x}$. The **forward pass** evaluates $\mathbf{y}$; the **backward pass** evaluates the transpose Jacobian acting on upstream gradients (via the adjoint of the differential):

$$d\mathbf{x} \leftarrow W^\top \text{diag}(\sigma') \cdot d\mathbf{y}$$

The entire backpropagation algorithm is: **compose differentials through the network, then adjoint them back.**

### 10.3 — Automatic Differentiation (Autograd)

Modern frameworks (PyTorch, JAX) implement **automatic differentiation**, which is a computational implementation of the chain rule on differentials. Two modes:

**Forward mode AD**: propagates $d\mathbf{x}$ forward through the computation graph, computing $d\mathbf{y}$ at each node. Efficient when $n \ll m$ (few inputs, many outputs).

**Reverse mode AD** (backprop): propagates $d\mathcal{L}$ backward through the graph via adjoint differentials. Efficient when $m \ll n$ (scalar loss, many parameters). This is why backprop is used in neural nets — $m = 1$ (scalar loss).

Every operation in the computation graph stores both its value and its differential rule. Backprop is just the chain rule $d(f \circ g) = df \circ dg$, applied in reverse.

### 10.4 — Gradient Descent as Differential Integration

Given $d\mathcal{L} = \nabla \mathcal{L} \cdot d\boldsymbol{\theta}$, gradient descent takes:

$$\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \eta,\nabla\mathcal{L}$$

This is a **discretization of the differential equation** $\frac{d\boldsymbol{\theta}}{dt} = -\nabla\mathcal{L}$, i.e., gradient flow. The continuous version is:

$$d\boldsymbol{\theta} = -\nabla\mathcal{L},dt$$

In the continuous limit, $\mathcal{L}$ decreases at rate $|\nabla\mathcal{L}|^2$ per unit time. Adam, RMSProp, and other optimizers scale the differential by the inverse of historical gradient magnitudes — they are adaptive scalings of $d\boldsymbol{\theta}$.

### 10.5 — Second-Order Methods: the Hessian and $d^2\mathcal{L}$

The **Newton step** minimizes the second-order approximation:

$$\mathcal{L}(\boldsymbol{\theta} + d\boldsymbol{\theta}) \approx \mathcal{L}(\boldsymbol{\theta}) + \nabla\mathcal{L}^\top d\boldsymbol{\theta} + \frac{1}{2}d\boldsymbol{\theta}^\top H,d\boldsymbol{\theta}$$

Setting $d(d\mathcal{L})/d(d\boldsymbol{\theta}) = 0$ gives $d\boldsymbol{\theta} = -H^{-1}\nabla\mathcal{L}$. Computing $H^{-1}$ costs $O(n^3)$ — intractable for large networks. Quasi-Newton methods (L-BFGS) approximate $H^{-1}$ from successive gradient differentials $\Delta\nabla\mathcal{L}$ and parameter differentials $\Delta\boldsymbol{\theta}$, using the secant equation:

$$H,\Delta\boldsymbol{\theta} \approx \Delta\nabla\mathcal{L}$$

This is a finite-difference approximation to the second differential $d^2\mathcal{L}$.

### 10.6 — Differentials in GANs and Score Matching

In generative models, **score matching** learns the score function $\nabla_\mathbf{x} \log p(\mathbf{x})$ — the differential of the log-density. Diffusion models (DDPM, Score SDE) define dynamics via stochastic differential equations:

$$d\mathbf{x}_t = \mathbf{f}(\mathbf{x}_t, t),dt + g(t),d\mathbf{W}_t$$

Here $d\mathbf{x}_t$, $dt$, and $d\mathbf{W}_t$ (Wiener differential, i.e., Brownian increment) are stochastic differentials — the Itô calculus extension of the framework we've been building. The score $\nabla_\mathbf{x} \log p_t(\mathbf{x})$ is a differential that the network learns to approximate at each noise level $t$. Reversing the SDE for generation requires the score — i.e., the differential of the log-density.

## §11. Summary — The Differential at a Glance

$$\underbrace{dz}_{\text{linear approx}} = \underbrace{f_x,dx + f_y,dy}_{\text{gradient dotted with increment}}$$

|Situation|What $df$ is|Computed as|
|---|---|---|
|$f: \mathbb{R} \to \mathbb{R}$|A 1-form on $\mathbb{R}$|$f'(x),dx$|
|$f: \mathbb{R}^n \to \mathbb{R}$|A covector / 1-form|$\nabla f \cdot d\mathbf{x}$|
|$\mathbf{f}: \mathbb{R}^n \to \mathbb{R}^m$|A linear map $\mathbb{R}^n \to \mathbb{R}^m$|Jacobian $J,d\mathbf{x}$|
|Neural network $\mathcal{L}$|Gradient vector|Computed by backprop|
|Implicit curve $F = 0$|Constraint on $dx, dy$|$dF = 0$|
|Coordinate change|Jacobian of transformation|$dx\wedge dy = r,dr\wedge d\theta$|

## §12. Theorems to Know Cold

**Theorem 1** _(Differentiability and the Differential)._ $f$ is differentiable at $\mathbf{a}$ iff there exists a linear map $L$ such that $f(\mathbf{a}+\mathbf{h}) = f(\mathbf{a}) + L(\mathbf{h}) + o(|\mathbf{h}|)$. In that case, $L = df_\mathbf{a}$.

**Theorem 2** _(Chain Rule for Differentials)._ If $\mathbf{g}$ is differentiable at $\mathbf{a}$ and $f$ is differentiable at $\mathbf{g}(\mathbf{a})$, then $d(f \circ \mathbf{g})_\mathbf{a} = df_{\mathbf{g}(\mathbf{a})} \circ d\mathbf{g}_\mathbf{a}$.

In matrix form: $J_{f \circ g} = J_f \cdot J_g$.

**Theorem 3** _(Exactness Condition)._ A 1-form $P,dx + Q,dy$ is exact (equals $df$ for some $f$) iff $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$ on a simply connected domain.

**Theorem 4** _(Error Bound)._ If $|f_{x_i}| \leq M_i$ on a region, then: $$|\Delta f - df| \leq \frac{1}{2}\max|f_{x_i x_j}| \cdot (|\Delta\mathbf{x}|)^2$$

**Theorem 5** _(Fundamental Theorem via Differentials)._ $\int_{\mathbf{a}}^{\mathbf{b}} df = f(\mathbf{b}) - f(\mathbf{a})$ for any path — the line integral of an exact form is path-independent.

## §13. Common Errors

> **$dx$ is not infinitesimally small.** In the definition, $dx$ is a free real variable. It only needs to be "small" when you use $df$ as an approximation to $\Delta f$.

> **$df \neq \Delta f$.** They differ by $O((dx)^2)$. Writing $df = \Delta f$ is an approximation, not an equality.

> **Partial derivatives are not the same as differentials.** $\frac{\partial f}{\partial x}$ is a number; $df$ is a linear functional. The partial derivative is the coefficient of $dx$ in $df$, not the same object.

> **Not every expression $P,dx + Q,dy$ is a differential** of some function. Exactness is not free — it requires $P_y = Q_x$.

> **In implicit differentiation**, $dF = 0$ only holds because $F$ is **held constant** on the curve/surface. Make this explicit in your reasoning.
