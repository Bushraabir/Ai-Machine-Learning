## §0. The Optimization Problem

We already know that for a differentiable function $f: \mathbb{R}^n \to \mathbb{R}$:

$$D_{\mathbf{u}} f(\mathbf{a}) = \nabla f(\mathbf{a}) \cdot \mathbf{u}$$

Now ask a sharper question: **over all possible directions $\mathbf{u}$, which one makes $D_{\mathbf{u}} f(\mathbf{a})$ as large as possible?**

This is a **constrained optimization problem**:

$$\maximize_{\mathbf{u} \in \mathbb{R}^n} \quad \nabla f(\mathbf{a}) \cdot \mathbf{u} \qquad \text{subject to} \qquad |\mathbf{u}| = 1$$

The constraint $|\mathbf{u}| = 1$ is essential — without it the problem is unbounded (just scale $\mathbf{u}$ to infinity).

This single optimization problem is the **origin of gradient descent, ascent, and the geometric meaning of the gradient**. Everything in this note flows from solving it rigorously.

---

## §1. Solution via the Cauchy–Schwarz Inequality

### Theorem 1.1 (Maximum Directional Derivative)

Let $f: \mathbb{R}^n \to \mathbb{R}$ be differentiable at $\mathbf{a}$, and suppose $\nabla f(\mathbf{a}) \neq \mathbf{0}$. Then:

$$\max_{|\mathbf{u}| = 1} D_{\mathbf{u}} f(\mathbf{a}) = |\nabla f(\mathbf{a})|$$

achieved **uniquely** at:

$$\mathbf{u}^* = \frac{\nabla f(\mathbf{a})}{|\nabla f(\mathbf{a})|}$$

### Proof

**Step 1: Apply Cauchy–Schwarz.**

The Cauchy–Schwarz inequality states that for any $\mathbf{p}, \mathbf{q} \in \mathbb{R}^n$: $$|\mathbf{p} \cdot \mathbf{q}| \leq |\mathbf{p}| \cdot |\mathbf{q}|$$ with equality if and only if $\mathbf{p}$ and $\mathbf{q}$ are **parallel** (i.e., $\mathbf{q} = \lambda \mathbf{p}$ for some scalar $\lambda \geq 0$).

**Step 2: Apply to our problem.**

Set $\mathbf{p} = \nabla f(\mathbf{a})$ and $\mathbf{q} = \mathbf{u}$ with $|\mathbf{u}| = 1$:

$$D_{\mathbf{u}} f(\mathbf{a}) = \nabla f(\mathbf{a}) \cdot \mathbf{u} \leq |\nabla f(\mathbf{a}) \cdot \mathbf{u}| \leq |\nabla f(\mathbf{a})| \cdot \underbrace{|\mathbf{u}|}_{=1} = |\nabla f(\mathbf{a})|$$

So $D_{\mathbf{u}} f(\mathbf{a}) \leq |\nabla f(\mathbf{a})|$ for all unit vectors $\mathbf{u}$.

**Step 3: Equality is achieved.**

Equality holds in Cauchy–Schwarz when $\mathbf{u} \parallel \nabla f(\mathbf{a})$ and both point in the same direction (positive $\lambda$). The unit vector in this direction is:

$$\mathbf{u}^* = \frac{\nabla f(\mathbf{a})}{|\nabla f(\mathbf{a})|}$$

Verify: $$D_{\mathbf{u}^*} f(\mathbf{a}) = \nabla f(\mathbf{a}) \cdot \frac{\nabla f(\mathbf{a})}{|\nabla f(\mathbf{a})|} = \frac{|\nabla f(\mathbf{a})|^2}{|\nabla f(\mathbf{a})|} = |\nabla f(\mathbf{a})| \checkmark$$

**Step 4: Uniqueness.**

Equality in Cauchy–Schwarz holds for $\mathbf{u}$ parallel to $\nabla f(\mathbf{a})$ with the same orientation. Among unit vectors, this is unique: $\mathbf{u}^* = \nabla f(\mathbf{a})/|\nabla f(\mathbf{a})|$ is the only maximizer. $\blacksquare$

## §2. The Cosine Formulation — Full Picture

Write the dot product using the angle $\theta \in [0, \pi]$ between $\mathbf{u}$ and $\nabla f(\mathbf{a})$:

$$D_{\mathbf{u}} f(\mathbf{a}) = |\nabla f(\mathbf{a})| \cos\theta$$

Since $|\nabla f(\mathbf{a})| \geq 0$ is fixed, maximizing over $\mathbf{u}$ is equivalent to maximizing $\cos\theta$ over all achievable angles.

```
  D_u f
    |
  ‖∇f‖│·····················●  θ = 0  (u parallel to ∇f) — MAXIMUM
    |                      /
    |                    /
    |                  /
  0 │────────────────●──────────  θ = π/2  (u ⊥ ∇f) — zero change
    |              /
    |            /
    |          /
 -‖∇f‖│·····●                    θ = π   (u anti-parallel) — MINIMUM
    |
    └────────────────────────────► θ
         0      π/2       π
```

This gives the **complete classification**:

|$\theta$|$\cos\theta$|$D_{\mathbf{u}} f$|Geometric meaning|
|---|---|---|---|
|$0$|$+1$|$+\|\nabla f\|$|**Maximum** — steepest ascent|
|$(0, \pi/2)$|$(0,1)$|positive|$f$ increases in this direction|
|$\pi/2$|$0$|$0$|**No change** — tangent to level set|
|$(\pi/2, \pi)$|$(-1,0)$|negative|$f$ decreases in this direction|
|$\pi$|$-1$|$-\|\nabla f\|$|**Minimum** — steepest descent|

**Key insight:** The gradient does not just point "uphill" — it encodes the _entire spectrum_ of directional rates of change through the formula $|\nabla f| \cos\theta$.

## §3. The Minimum: Steepest Descent

By symmetry, the **minimum** of $D_{\mathbf{u}} f$ over unit vectors is:

$$\min_{|\mathbf{u}|=1} D_{\mathbf{u}} f(\mathbf{a}) = -|\nabla f(\mathbf{a})|$$

achieved uniquely at:

$$\mathbf{u}_{\min} = -\frac{\nabla f(\mathbf{a})}{|\nabla f(\mathbf{a})|}$$

This is the direction of **steepest descent**. In one line, this is why gradient descent subtracts the gradient.

## §4. Geometric Interpretation: Gradient and Level Sets

### Theorem 4.1

The gradient $\nabla f(\mathbf{a})$ is **perpendicular** to the level set ${f = f(\mathbf{a})}$ at $\mathbf{a}$.

### Proof

Let $\boldsymbol{\gamma}: (-\varepsilon, \varepsilon) \to \mathbb{R}^n$ be any smooth curve with:

- $\boldsymbol{\gamma}(0) = \mathbf{a}$
- $f(\boldsymbol{\gamma}(t)) = f(\mathbf{a})$ for all $t$ (the curve lies on the level set)

Differentiating $f(\boldsymbol{\gamma}(t)) = \text{const}$ with respect to $t$ at $t = 0$, by the chain rule:

$$\nabla f(\mathbf{a}) \cdot \boldsymbol{\gamma}'(0) = 0$$

Since $\boldsymbol{\gamma}'(0)$ is an arbitrary tangent vector to the level set at $\mathbf{a}$, we conclude that $\nabla f(\mathbf{a})$ is perpendicular to every tangent direction of the level set. Therefore $\nabla f(\mathbf{a}) \perp \text{level set}$. $\blacksquare$

```
                      ∇f(a)
         f = 3         ↑
       f = 2        ↗  |  (perpendicular)
     f = 1       ↗     |
               ↗       ●──────────► tangent direction (D_u f = 0)
             ↗         a
Level curves (2D)       └── gradient is orthogonal to level curves
```

### Corollary 4.2 (Normals in Practice)

The equation of the **tangent plane** to the surface $f(x,y,z) = c$ at point $\mathbf{a} = (a_1, a_2, a_3)$ is:

$$\nabla f(\mathbf{a}) \cdot (\mathbf{x} - \mathbf{a}) = 0$$

or explicitly:

$$f_x(\mathbf{a})(x - a_1) + f_y(\mathbf{a})(y - a_2) + f_z(\mathbf{a})(z - a_3) = 0$$

This is the maximization problem's answer repurposed: the gradient is the **normal vector** to the tangent plane.

## §5. Alternative Proof via Lagrange Multipliers

The problem $\max \nabla f \cdot \mathbf{u}$ subject to $|\mathbf{u}|^2 = 1$ is exactly a Lagrange multiplier problem. This approach is more formal and connects to constrained optimization theory.

**Objective:** $\Phi(\mathbf{u}) = \nabla f(\mathbf{a}) \cdot \mathbf{u} = \sum_i g_i u_i$ where $\mathbf{g} = \nabla f(\mathbf{a})$.

**Constraint:** $h(\mathbf{u}) = |\mathbf{u}|^2 - 1 = \sum_i u_i^2 - 1 = 0$.

**KKT condition:** At an extremum, $\nabla_{\mathbf{u}} \Phi = \lambda \nabla_{\mathbf{u}} h$:

$$\frac{\partial \Phi}{\partial u_i} = \lambda \frac{\partial h}{\partial u_i} \implies g_i = \lambda \cdot 2u_i \implies \mathbf{g} = 2\lambda \mathbf{u}$$

So $\mathbf{u} = \mathbf{g}/(2\lambda)$, meaning $\mathbf{u}$ is parallel to $\mathbf{g} = \nabla f(\mathbf{a})$.

Applying the constraint $|\mathbf{u}| = 1$: $\mathbf{u} = \pm \mathbf{g}/|\mathbf{g}|$.

Evaluating the objective:

- $\mathbf{u} = +\mathbf{g}/|\mathbf{g}|$: $\Phi = \mathbf{g} \cdot \mathbf{g}/|\mathbf{g}| = |\mathbf{g}|$ **(maximum)**
- $\mathbf{u} = -\mathbf{g}/|\mathbf{g}|$: $\Phi = -|\mathbf{g}|$ **(minimum)**

This confirms Theorem 1.1 through the lens of constrained optimization. $\blacksquare$

> **Meta-lesson:** Lagrange multipliers and Cauchy–Schwarz give the same answer by different routes. The Cauchy–Schwarz proof is cleaner for this particular problem; Lagrange multipliers generalize to more complex constraints.

## §6. Worked Examples

### Example 6.1 (Finding the Direction of Steepest Ascent)

**Problem:** $f(x,y) = x^2 - 2xy + 3y^2$. At point $P = (2, 1)$:

1. Find the direction of maximum increase.
2. Find the maximum rate of change.
3. Find all directions in which $f$ does not change.

**Solution:**

**Gradient:** $$\nabla f = \langle 2x - 2y,\ -2x + 6y \rangle \implies \nabla f(2,1) = \langle 2, 2 \rangle$$

1. **Direction of max increase:** $\mathbf{u}^* = \langle 2,2 \rangle / |\langle 2,2 \rangle| = \frac{1}{\sqrt{2}} \langle 1,1 \rangle$
    
2. **Maximum rate:** $|\nabla f(2,1)| = \sqrt{4+4} = 2\sqrt{2}$
    
3. **Zero change directions:** Need $\nabla f \cdot \mathbf{u} = 0$, i.e., $\langle 2,2 \rangle \cdot \langle u_1, u_2 \rangle = 0 \implies u_1 + u_2 = 0$. Unit vectors satisfying this: $\mathbf{u} = \frac{1}{\sqrt{2}} \langle 1, -1 \rangle$ and $\mathbf{u} = \frac{1}{\sqrt{2}} \langle -1, 1 \rangle$. These are tangent to the level curve $f = f(2,1) = 4 - 4 + 3 = 3$ at $P$.

### Example 6.2 (3D — MIT 18.02 Exam Style)

**Problem:** The temperature in a region of space is $T(x,y,z) = e^{-x^2 - 2y^2 - z^2}$. A bug is at point $P = (1, 0, 0)$.

1. In what direction should the bug move to cool down most rapidly?
2. What is the rate of cooling in that direction?
3. If the bug moves in direction $\mathbf{v} = \langle 1, 1, 0 \rangle$, is it getting warmer or cooler?

**Solution:**

$$\nabla T = T \cdot \langle -2x, -4y, -2z \rangle$$

At $P = (1,0,0)$: $T(1,0,0) = e^{-1}$, so: $$\nabla T(1,0,0) = e^{-1} \langle -2, 0, 0 \rangle = \langle -2e^{-1}, 0, 0 \rangle$$

1. **Direction of fastest cooling** (minimum of $D_{\mathbf{u}} T$, i.e., most negative directional derivative): $$\mathbf{u}_{\min} = -\frac{\nabla T}{|\nabla T|} = -\frac{\langle -2/e, 0, 0 \rangle}{2/e} = \langle 1, 0, 0 \rangle$$ The bug should move in the $+x$ direction (away from the origin).
    
2. **Rate of cooling:** $-|\nabla T| = -2/e \approx -0.736$ per unit distance.
    
3. **Direction $\mathbf{v} = \langle 1,1,0 \rangle$:** Normalize: $\mathbf{u} = \frac{1}{\sqrt{2}} \langle 1,1,0 \rangle$. $$D_{\mathbf{u}} T(P) = \langle -2/e, 0, 0 \rangle \cdot \frac{1}{\sqrt{2}} \langle 1,1,0 \rangle = \frac{-2}{e\sqrt{2}} = \frac{-\sqrt{2}}{e} < 0$$ The bug is getting **cooler** (negative rate of change of temperature).

### Example 6.3 (Optimization on a Constrained Surface — Connecting Themes)

**Problem:** Find the point on the ellipsoid $x^2/4 + y^2/9 + z^2 = 1$ where the function $f(x,y,z) = x + 2y + 3z$ achieves its maximum.

This is a constrained optimization, but the key insight involves the maximized directional derivative:

At the maximum, the gradient of $f$ must be parallel to the gradient of the constraint $g(x,y,z) = x^2/4 + y^2/9 + z^2 - 1$:

$$\nabla f = \langle 1, 2, 3 \rangle = \lambda \nabla g = \lambda \langle x/2,\ 2y/9,\ 2z \rangle$$

Solving: $x/2 = 1/\lambda \implies x = 2/\lambda$; similarly $y = 9/\lambda$, $z = 3/(2\lambda)$.

Substituting into the constraint: $$\frac{(2/\lambda)^2}{4} + \frac{(9/\lambda)^2}{9} + \frac{(3/2\lambda)^2}{1} = 1$$ $$\frac{1}{\lambda^2} + \frac{9}{\lambda^2} + \frac{9}{4\lambda^2} = 1 \implies \frac{1}{\lambda^2}\left(1 + 9 + \frac{9}{4}\right) = 1 \implies \frac{1}{\lambda^2} \cdot \frac{49}{4} = 1 \implies \lambda = \pm \frac{7}{2}$$

Taking $\lambda = 7/2$ (positive, for the maximum): $x = 4/7$, $y = 18/7$, $z = 3/7$.

The **geometric meaning**: at the maximum, the gradient of $f$ is perpendicular to the ellipsoid (parallel to $\nabla g$), meaning $f$ has no directional derivative along any tangent to the surface — consistent with it being an extremum.

## §7. Why $\nabla f(\mathbf{a}) = \mathbf{0}$ is Special

If $\nabla f(\mathbf{a}) = \mathbf{0}$, then: $$D_{\mathbf{u}} f(\mathbf{a}) = \nabla f(\mathbf{a}) \cdot \mathbf{u} = 0 \cdot \mathbf{u} = 0 \quad \text{for every } \mathbf{u}$$

The function has **zero directional derivative in every direction** — it is (to first order) flat at $\mathbf{a}$. These are precisely the **critical points** of $f$:

- Could be a local minimum, local maximum, or saddle point — the second-order Hessian decides.
- There is no "direction of steepest ascent" — the maximum of $D_{\mathbf{u}} f$ is $0$ in all directions.

This is the multivariable analogue of $f'(a) = 0$ in single-variable calculus.

## §8. The Second Directional Derivative and Curvature

The **second directional derivative** in direction $\mathbf{u}$ is:

$$D_{\mathbf{u}}^2 f(\mathbf{a}) = D_{\mathbf{u}}(D_{\mathbf{u}} f)(\mathbf{a}) = \mathbf{u}^T H_f(\mathbf{a}) \mathbf{u}$$

where $H_f(\mathbf{a})$ is the **Hessian matrix**:

$$(H_f)_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$$

The **Rayleigh quotient** $R(\mathbf{u}) = \mathbf{u}^T H \mathbf{u}$ (subject to $|\mathbf{u}|=1$) is maximized/minimized at the **eigenvectors** of $H$, with extreme values equal to the **eigenvalues**.

This means:

- The direction of **maximum curvature** = eigenvector of $H$ with largest eigenvalue $\lambda_{\max}$
- The direction of **minimum curvature** = eigenvector of $H$ with smallest eigenvalue $\lambda_{\min}$

This directly generalizes Theorem 1.1 to second-order geometry.

```
     Second directional derivative landscape:
     
     Positive definite H (local min):         Indefinite H (saddle):
     
         ↑ large curvature                      ↑ positive curvature
         |                                      |
     ────●──── small curvature             ─────●───── negative curvature
         |                                      |
         ↓                                      ↓
```

## §9. Connection to AI, Machine Learning, and Deep Learning

### 9.1 Gradient Descent IS the Maximization Problem, Inverted

The update rule $\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \eta \nabla \mathcal{L}$ follows from asking:

> "If I can take a step of size $\eta$ in any direction, which direction decreases $\mathcal{L}$ the most?"

The answer: move in the direction of **minimum** directional derivative, which is $-\nabla \mathcal{L}/|\nabla \mathcal{L}|$. Scaling by $\eta|\nabla \mathcal{L}|$ gives the update $-\eta \nabla \mathcal{L}$.

**This is Theorem 1.1 (steepest descent variant) applied to the loss function, one step at a time.**

### 9.2 Adaptive Optimizers Modify the Metric

Adam, RMSProp, and Adagrad do not maximize in the Euclidean direction. They effectively change the inner product to:

$$\langle \mathbf{u}, \mathbf{v} \rangle_{\mathbf{d}} = \sum_i d_i u_i v_i$$

where $\mathbf{d}$ is a per-parameter scaling (diagonal preconditioner). The "steepest descent" direction in this metric is $-\mathbf{d}^{-1} \nabla \mathcal{L}$, which is why Adam-style updates divide by a running estimate of gradient magnitudes.

**Reformulating:** Adaptive methods solve a _modified_ maximization problem:

$$\min_{|\mathbf{u}|_{\mathbf{d}} = 1} D_{\mathbf{u}} \mathcal{L}$$

where $|\mathbf{u}|_{\mathbf{d}}^2 = \mathbf{u}^T D \mathbf{u}$ and $D = \text{diag}(\mathbf{d})$.

### 9.3 Hessian Geometry and Newton's Method

Newton's method replaces the Euclidean metric with the **Hessian metric**:

$$|\mathbf{u}|_H^2 = \mathbf{u}^T H \mathbf{u}$$

The steepest descent direction under this metric solves:

$$\min_{\mathbf{u}^T H \mathbf{u} = 1} \nabla \mathcal{L} \cdot \mathbf{u}$$

The Lagrange multiplier solution gives $\mathbf{u}^* = -H^{-1} \nabla \mathcal{L}$ (up to normalization), which is exactly the Newton step.

**Every optimizer corresponds to a different norm on the direction $\mathbf{u}$:**

|Optimizer|Norm on $\mathbf{u}$|Optimal direction|
|---|---|---|
|Gradient Descent|$\|\mathbf{u}\|_2 = 1$ (Euclidean)|$-\nabla \mathcal{L}/\|\nabla \mathcal{L}\|$|
|Newton's Method|$\|\mathbf{u}\|_H = 1$ (Hessian)|$-H^{-1}\nabla \mathcal{L}$ (normalized)|
|Natural Gradient|$\|\mathbf{u}\|_F = 1$ (Fisher)|$-F^{-1}\nabla \mathcal{L}$ (normalized)|
|Adam/Adagrad|$\|\mathbf{u}\|_D = 1$ (Diagonal)|$-D^{-1}\nabla \mathcal{L}$ (normalized)|

The choice of optimizer is the choice of **which inner product to use when solving the maximization problem** underlying gradient descent.

### 9.4 Loss Landscape Sharpness and Generalization

The **sharpness** of a minimum $\boldsymbol{\theta}^*$ is:

$$\text{sharpness} = \max_{|\mathbf{u}|=1} D_{\mathbf{u}}^2 \mathcal{L}(\boldsymbol{\theta}^_) = \lambda_{\max}(H_{\mathcal{L}}(\boldsymbol{\theta}^_))$$

This is exactly the **maximum of the second directional derivative** (§8). Flat minima (small $\lambda_{\max}$) generalize better; sharp minima (large $\lambda_{\max}$) overfit. SAM minimizes this sharpness by solving:

$$\min_{\boldsymbol{\theta}} \max_{|\boldsymbol{\epsilon}| \leq \rho} \mathcal{L}(\boldsymbol{\theta} + \boldsymbol{\epsilon})$$

The inner $\max$ is approximately $\mathcal{L}(\boldsymbol{\theta} + \rho \nabla \mathcal{L}/|\nabla \mathcal{L}|)$, which is the maximum directional derivative from Theorem 1.1 at the scale $\rho$.

### 9.5 Adversarial Examples: Maximum Directional Derivative of Loss w.r.t. Input

The **Fast Gradient Sign Method (FGSM)** computes:

$$\mathbf{x}_{\text{adv}} = \mathbf{x} + \epsilon \cdot \text{sign}(\nabla_{\mathbf{x}} \mathcal{L})$$

The perturbation $\text{sign}(\nabla_{\mathbf{x}} \mathcal{L})$ maximizes $\mathcal{L}$ under the $L^\infty$ norm constraint $|\boldsymbol{\delta}|_\infty \leq \epsilon$.

Under the $L^\infty$ norm, the unit ball is the hypercube $[-1,1]^n$, and maximizing $\nabla \mathcal{L} \cdot \boldsymbol{\delta}$ over this set gives $\boldsymbol{\delta}^* = \text{sign}(\nabla \mathcal{L})$. This is the **$L^\infty$ analogue of Theorem 1.1** — steepest ascent under the $\ell^\infty$ metric.

**Summary:** The maximization of the directional derivative appears in ML in three distinct roles:

1. **Descent:** minimize it → update parameters to reduce loss
2. **Curvature analysis:** maximize second directional derivative → find sharp/flat minima
3. **Adversarial attacks:** maximize it w.r.t. input → fool the network

## §10. Visualization: The Unit Circle Sweep

In $\mathbb{R}^2$, we can visualize all directional derivatives simultaneously. For $\mathbf{u} = \langle \cos\theta, \sin\theta \rangle$:

$$D_{\mathbf{u}} f(\mathbf{a}) = f_x \cos\theta + f_y \sin\theta = |\nabla f| \cos(\theta - \theta^*)$$

where $\theta^* = \arctan(f_y/f_x)$ is the angle of $\nabla f$.

```
     Polar plot of D_u f vs. θ (for ∇f = ⟨2, 1⟩):

           θ = π/2
              │
        ──────●──────  The "peanut" shape:
       /      │      \  - Maximum at θ* = arctan(1/2) ≈ 26.6°
      │       │       │ - Zero crossings at θ* ± π/2
       \      │      /  - Minimum at θ* + π
        ──────●──────
              │
           θ = -π/2

     Positive lobes: directions where f increases
     Negative lobes: directions where f decreases
     Zero crossings: tangent to level curve at a
```

The shape is always a **figure-8 (lemniscate of directional change)** symmetric about the gradient direction. Its farthest points from the origin are at $\pm|\nabla f|$ — exactly Theorem 1.1.

## §11. Summary Theorems

$$\boxed{\max_{|\mathbf{u}|=1} D_{\mathbf{u}} f(\mathbf{a}) = |\nabla f(\mathbf{a})|, \quad \text{achieved uniquely at } \mathbf{u}^* = \frac{\nabla f(\mathbf{a})}{|\nabla f(\mathbf{a})|}}$$

$$\boxed{\min_{|\mathbf{u}|=1} D_{\mathbf{u}} f(\mathbf{a}) = -|\nabla f(\mathbf{a})|, \quad \text{achieved uniquely at } \mathbf{u}_{\min} = -\frac{\nabla f(\mathbf{a})}{|\nabla f(\mathbf{a})|}}$$

$$\boxed{D_{\mathbf{u}} f(\mathbf{a}) = 0 \iff \mathbf{u} \perp \nabla f(\mathbf{a}) \iff \mathbf{u} \text{ is tangent to the level set at } \mathbf{a}}$$

$$\boxed{\nabla f(\mathbf{a}) = \mathbf{0} \implies D_{\mathbf{u}} f(\mathbf{a}) = 0 \text{ for all } \mathbf{u} \iff \mathbf{a} \text{ is a critical point}}$$

$$\boxed{\max_{|\mathbf{u}|=1} \mathbf{u}^T H \mathbf{u} = \lambda_{\max}(H), \quad \text{achieved at the corresponding eigenvector}}$$
