

> _"Calculus is, at its heart, the art of replacing curved things with flat things — locally. Linear approximation is that replacement made precise."_

---

## §1. The Central Idea

Every differentiable function looks linear when you zoom in far enough. This is not a metaphor — it is a theorem, and it is the reason calculus works at all.

The project of linear approximation is: given a complicated function $f$, find the simplest possible function $L$ (a linear/affine one) such that $L \approx f$ near a chosen point, with the error being negligible compared to the distance to that point.

This single idea:

- Is the foundation of numerical analysis
- Drives every optimization algorithm in machine learning
- Underlies error propagation in physics and engineering
- Generalizes to the Fréchet derivative in functional analysis

The tangent plane note established the geometric picture. Here we go deeper: error bounds, higher-order corrections, the multivariable theory, and what "best linear approximation" truly means.

## §2. Single Variable — Review and Sharpening

### 2.1 Definition

**Definition.** Let $f: \mathbb{R} \to \mathbb{R}$ be differentiable at $x = a$. The **linearization** of $f$ at $a$ is:

$$L(x) = f(a) + f'(a)(x - a)$$

For $x$ near $a$, $f(x) \approx L(x)$, and the approximation is called the **linear approximation** or **tangent line approximation**.

The statement $f'(a)$ exists is _equivalent_ to saying:

$$\lim_{x \to a} \frac{f(x) - f(a)}{x - a} = f'(a) \iff \lim_{x \to a} \frac{f(x) - L(x)}{x - a} = 0$$

That is, the error $f(x) - L(x)$ vanishes **faster than** $|x - a|$. This is little-$o$ notation:

$$f(x) = L(x) + o(x - a) \quad \text{as } x \to a$$

This is the precise meaning of "best linear approximation": $L$ is the unique affine function for which the error is $o(x-a)$.

### 2.2 The Error — Taylor's Theorem

The quality of the approximation is controlled by the second derivative. **Taylor's Theorem with Lagrange remainder:**

$$f(x) = f(a) + f'(a)(x-a) + \frac{f''(c)}{2}(x-a)^2$$

for some $c$ between $a$ and $x$. Therefore, the error satisfies:

$$|f(x) - L(x)| = \left|\frac{f''(c)}{2}\right| (x-a)^2 \leq \frac{M}{2}(x-a)^2$$

where $M = \max_{t \in [a,x]} |f''(t)|$.

**Practical consequence.** If you halve the distance $|x - a|$, the error shrinks by a factor of **4**. Linear approximation is a second-order accurate method.

### 2.3 Quick Examples

**Example A.** Approximate $\sqrt{4.1}$ using $f(x) = \sqrt{x}$, $a = 4$.

$$f'(x) = \frac{1}{2\sqrt{x}} \implies f'(4) = \frac{1}{4}$$ $$L(x) = 2 + \frac{1}{4}(x - 4) \implies L(4.1) = 2 + \frac{0.1}{4} = 2.025$$

Exact: $\sqrt{4.1} \approx 2.02485...$. Error $\approx 0.00015$.

**Error bound check:** $f''(x) = -\frac{1}{4}x^{-3/2}$, so $|f''(c)| \leq \frac{1}{4}(4)^{-3/2} = \frac{1}{32}$ for $c \in [4, 4.1]$.

$$|f(4.1) - L(4.1)| \leq \frac{1}{2} \cdot \frac{1}{32} \cdot (0.1)^2 = \frac{0.01}{64} \approx 0.000156 \checkmark$$

**Example B.** Approximate $e^{0.03}$.

$$f(x) = e^x,\quad a = 0,\quad L(x) = 1 + x$$ $$e^{0.03} \approx 1.03$$

Exact: $1.03045...$. Error $\approx 0.00045$. Consistent with $\frac{M}{2}(0.03)^2 = \frac{e^{0.03}}{2}(0.0009) \approx 0.00046$.

**Example C.** $\sin(\theta) \approx \theta$ for small $\theta$.

$$f(\theta) = \sin\theta,\quad a = 0,\quad L(\theta) = \theta$$

This is perhaps the most used approximation in physics. The error is $O(\theta^3)$ (from the next Taylor term $-\theta^3/6$). Valid when $\theta \ll 1$ radian.

## §3. Multivariable Linear Approximation

### 3.1 Setup

Let $f: \mathbb{R}^n \to \mathbb{R}$ be differentiable at $\mathbf{a} \in \mathbb{R}^n$. Write $\mathbf{a} = (a_1, \ldots, a_n)$ and $\mathbf{x} = (x_1, \ldots, x_n)$.

**Definition.** The **linearization** of $f$ at $\mathbf{a}$ is:

$$L(\mathbf{x}) = f(\mathbf{a}) + \nabla f(\mathbf{a}) \cdot (\mathbf{x} - \mathbf{a}) = f(\mathbf{a}) + \sum_{i=1}^n \frac{\partial f}{\partial x_i}(\mathbf{a}),(x_i - a_i)$$

For $n = 2$: $L(x,y) = f(a,b) + f_x(a,b)(x-a) + f_y(a,b)(y-b)$ — the tangent plane.

**Differentiability condition.** $f$ is differentiable at $\mathbf{a}$ if and only if:

$$\lim_{\mathbf{x} \to \mathbf{a}} \frac{f(\mathbf{x}) - L(\mathbf{x})}{|\mathbf{x} - \mathbf{a}|} = 0$$

Again: error vanishes faster than distance. Linearization is the unique affine function with this property.

### 3.2 The Total Differential

Write $\Delta x_i = x_i - a_i$ and $\Delta f = f(\mathbf{x}) - f(\mathbf{a})$. Then:

$$\Delta f \approx df = \sum_{i=1}^n \frac{\partial f}{\partial x_i}, dx_i = \nabla f \cdot d\mathbf{x}$$

This is the **total differential** — a linear functional on displacement vectors $d\mathbf{x}$.

**Key point.** $df$ is not a number. It is a **1-form**, a linear map $\mathbb{R}^n \to \mathbb{R}$. The gradient $\nabla f$ is the vector representation of this map in the standard basis. This distinction matters in differential geometry and becomes essential in deep learning when working on manifolds.

### 3.3 Error Bound in $\mathbb{R}^n$

If all second-order partial derivatives of $f$ are bounded by $M$ near $\mathbf{a}$, then:

$$|f(\mathbf{x}) - L(\mathbf{x})| \leq \frac{n^2 M}{2} |\mathbf{x} - \mathbf{a}|^2$$

The quadratic scaling in distance holds in all dimensions.

### 3.4 Worked Example — Two Variables

Let $f(x,y) = e^{xy}\cos(y)$. Linearize at $(0, 0)$.

$f(0,0) = e^0 \cos 0 = 1$.

$$f_x = ye^{xy}\cos y \implies f_x(0,0) = 0$$ $$f_y = xe^{xy}\cos y - e^{xy}\sin y \implies f_y(0,0) = 0 - 0 = 0$$

So $L(x,y) = 1$. Interesting: near $(0,0)$, this function is well-approximated by a **constant** — its first-order variation is zero. The tangent plane is horizontal.

Compare: $f(0.1, 0.1) = e^{0.01}\cos(0.1) \approx 1.0101 \times 0.9950 \approx 1.005$. $L = 1$. Error $\approx 0.005 = O(0.01)$. ✓

### 3.5 Worked Example — Error Propagation

A box has measured dimensions $\ell = 50,\text{cm}$, $w = 30,\text{cm}$, $h = 20,\text{cm}$, each with uncertainty $\pm 0.2,\text{cm}$. Estimate the maximum error in volume $V = \ell w h$.

$$dV = \frac{\partial V}{\partial \ell}d\ell + \frac{\partial V}{\partial w}dw + \frac{\partial V}{\partial h}dh = wh,d\ell + \ell h,dw + \ell w,dh$$

Maximum error (taking $|d\ell| = |dw| = |dh| = 0.2$):

$$|dV| \leq |wh||d\ell| + |\ell h||dw| + |\ell w||dh|$$ $$= (30)(20)(0.2) + (50)(20)(0.2) + (50)(30)(0.2)$$ $$= 120 + 200 + 300 = 620,\text{cm}^3$$

Relative error: $\frac{620}{30000} \approx 2.07%$.

Note: the maximum error formula follows the **triangle inequality** applied to the total differential. This is standard in experimental physics.

## §4. Vector-Valued Functions — The Jacobian

Now let $\mathbf{f}: \mathbb{R}^n \to \mathbb{R}^m$, so $\mathbf{f} = (f_1, \ldots, f_m)$ where each $f_i: \mathbb{R}^n \to \mathbb{R}$.

**Definition.** The **Jacobian matrix** of $\mathbf{f}$ at $\mathbf{a}$ is the $m \times n$ matrix:

$$J_{\mathbf{f}}(\mathbf{a}) = D\mathbf{f}(\mathbf{a}) = \begin{pmatrix} \frac{\partial f_1}{\partial x_1} & \cdots & \frac{\partial f_1}{\partial x_n} \ \vdots & \ddots & \vdots \ \frac{\partial f_m}{\partial x_1} & \cdots & \frac{\partial f_m}{\partial x_n} \end{pmatrix}_{\mathbf{x} = \mathbf{a}}$$

The linearization is:

$$\mathbf{f}(\mathbf{x}) \approx \mathbf{f}(\mathbf{a}) + J_{\mathbf{f}}(\mathbf{a}),(\mathbf{x} - \mathbf{a})$$

**Differentiability.** $\mathbf{f}$ is differentiable at $\mathbf{a}$ iff:

$$\lim_{\mathbf{x} \to \mathbf{a}} \frac{|\mathbf{f}(\mathbf{x}) - \mathbf{f}(\mathbf{a}) - J_{\mathbf{f}}(\mathbf{a})(\mathbf{x}-\mathbf{a})|}{|\mathbf{x}-\mathbf{a}|} = 0$$

The Jacobian $J_\mathbf{f}(\mathbf{a})$ is the **best linear map** approximating $\mathbf{f}$ near $\mathbf{a}$. This is the proper, coordinate-free definition of the derivative for maps between Euclidean spaces (the **Fréchet derivative**).

**Chain Rule via Jacobians.** If $\mathbf{g}: \mathbb{R}^p \to \mathbb{R}^n$ and $\mathbf{f}: \mathbb{R}^n \to \mathbb{R}^m$, then:

$$J_{\mathbf{f} \circ \mathbf{g}}(\mathbf{a}) = J_{\mathbf{f}}(\mathbf{g}(\mathbf{a})), J_{\mathbf{g}}(\mathbf{a})$$

This is **matrix multiplication** of the Jacobians. It is the multivariable chain rule, and it is precisely what backpropagation computes.

### 4.1 Example — Jacobian

Let $\mathbf{f}(x,y) = (x^2 y,; \sin(x+y),; e^{xy})$.

$$J_\mathbf{f}(x,y) = \begin{pmatrix} 2xy & x^2 \ \cos(x+y) & \cos(x+y) \ ye^{xy} & xe^{xy} \end{pmatrix}$$

At $(1, 0)$: $J_\mathbf{f}(1,0) = \begin{pmatrix} 0 & 1 \ 1 & 1 \ 0 & 1 \end{pmatrix}$.

Linearization: $\mathbf{f}(x,y) \approx \mathbf{f}(1,0) + J_\mathbf{f}(1,0)\begin{pmatrix} x-1 \ y \end{pmatrix} = \begin{pmatrix} 0 \ 1 \ 1 \end{pmatrix} + \begin{pmatrix} 0 & 1 \ 1 & 1 \ 0 & 1 \end{pmatrix}\begin{pmatrix} x-1 \ y \end{pmatrix}$.

## §5. Linear Approximation vs. Quadratic — When to Use Which

The linearization is the **zeroth-order** approximation in curvature:

|Approximation|Formula|Error Order|
|---|---|---|
|Constant|$f(\mathbf{a})$|$O(\|\Delta\mathbf{x}\|)$|
|Linear (1st order)|$f(\mathbf{a}) + \nabla f \cdot \Delta\mathbf{x}$|$O(\|\Delta\mathbf{x}\|^2)$|
|Quadratic (2nd order)|$+ \frac{1}{2}\Delta\mathbf{x}^T H \Delta\mathbf{x}$|$O(\|\Delta\mathbf{x}\|^3)$|

where $H = \nabla^2 f$ is the **Hessian matrix** of second partial derivatives.

**When linear approximation fails:** near critical points ($\nabla f = 0$), the linear approximation is trivially $f(\mathbf{a})$ (constant). To understand whether it's a min, max, or saddle, you _must_ go to second order. This is the second derivative test generalized to $\mathbb{R}^n$: check the eigenvalues of $H$.

## §6. Geometric Picture

```
        f(x)
         |        ← actual curve / surface
         |   ,·'~
         |  / ← tangent line / plane (linearization)
         | /  · P
         |/___________
         a           x
```

**The zoom picture.** Let $g_\epsilon(t) = \frac{f(a + \epsilon t) - f(a)}{\epsilon}$. As $\epsilon \to 0$, $g_\epsilon \to f'(a) \cdot t$, which is linear. Differentiability says: _when you zoom in on the graph of $f$ around $a$, it looks like a straight line._ In $\mathbb{R}^2$: zoom in on the surface and it looks like a flat plane.

This is why differentiable functions are tractable — they are locally indistinguishable from affine maps.

**Affine vs. linear.** Strictly, $L(\mathbf{x}) = f(\mathbf{a}) + \nabla f(\mathbf{a}) \cdot (\mathbf{x}-\mathbf{a})$ is **affine** (linear + constant), not linear (which would require $L(\mathbf{0}) = 0$). The distinction matters in linear algebra and functional analysis. In common usage, "linear approximation" means "affine approximation."

## §7. Standard Linearizations to Memorize

Near $x = 0$ (i.e., $a = 0$, $\Delta x = x$):

|Function|Linearization|Valid for|
|---|---|---|
|$(1+x)^k$|$1 + kx$|$|
|$e^x$|$1 + x$|$|
|$\ln(1+x)$|$x$|$|
|$\sin x$|$x$|$|
|$\cos x$|$1$|$|
|$\tan x$|$x$|$|
|$\frac{1}{1+x}$|$1 - x$|$|
|$\sqrt{1+x}$|$1 + \frac{x}{2}$|$|

These are the first-order Taylor expansions. Commit them to memory — they appear everywhere.

**Composition trick.** To linearize $f(g(x))$ near $x=0$: first write $g(x) \approx g(0) + g'(0)x$, then feed into $L_f$:

$$f(g(x)) \approx f(g(0)) + f'(g(0)) \cdot g'(0) \cdot x$$

This is just the chain rule, but thinking of it as "linearize inside, then outside" is very efficient.

## §8. Connection to AI, Machine Learning, and Deep Learning

Linear approximation is not a background tool in ML — it **is** the central computational primitive.

### 8.1 — Gradient Descent is Linear Approximation + Greedy Step

Given loss $\mathcal{L}(\boldsymbol{\theta})$, the linear approximation at current parameters $\boldsymbol{\theta}_t$ is:

$$\mathcal{L}(\boldsymbol{\theta}) \approx \mathcal{L}(\boldsymbol{\theta}_t) + \nabla_{\boldsymbol{\theta}}\mathcal{L}(\boldsymbol{\theta}_t)^{\top}(\boldsymbol{\theta} - \boldsymbol{\theta}_t)$$

This is a hyperplane in parameter space. On this plane, $\mathcal{L}$ decreases fastest in the direction $-\nabla \mathcal{L}$. Gradient descent takes a step in that direction:

$$\boldsymbol{\theta}_{t+1} = \boldsymbol{\theta}_t - \eta, \nabla_{\boldsymbol{\theta}}\mathcal{L}(\boldsymbol{\theta}_t)$$

The step size $\eta$ (learning rate) controls how far to trust the linear approximation. Too large: the approximation breaks down and the step overshoots. Too small: slow convergence. **Choosing $\eta$ is equivalent to choosing the radius of trust in the linear model.** This is formalized in **trust region methods**.

### 8.2 — Backpropagation is Jacobian Composition

A neural network $\mathbf{f} = \mathbf{f}_L \circ \cdots \circ \mathbf{f}_1$ is a composition of layer maps. Backpropagation computes $\nabla_{\boldsymbol{\theta}_i} \mathcal{L}$ by the chain rule:

$$\frac{\partial \mathcal{L}}{\partial \boldsymbol{\theta}_i} = \frac{\partial \mathcal{L}}{\partial \mathbf{a}_L} \cdot J_{\mathbf{f}_L} \cdots J_{\mathbf{f}_{i+1}} \cdot \frac{\partial \mathbf{f}_i}{\partial \boldsymbol{\theta}_i}$$

Each $J_{\mathbf{f}_k}$ is the Jacobian of the $k$-th layer — the linear approximation of that layer's transformation. The backward pass propagates a "sensitivity vector" $\boldsymbol{\delta}$ (the gradient with respect to activations) through these Jacobians in reverse. **The entire backward pass is a sequence of matrix-vector products with Jacobians.**

**ReLU note.** The activation $\text{ReLU}(x) = \max(0,x)$ is not differentiable at $x=0$. In practice, the "subgradient" $\mathbf{1}[x > 0]$ is used — a piecewise linear approximation. The network is piecewise linear in its inputs, and the linear regions change depending on which neurons are active.

### 8.3 — Newton's Method and Second-Order Optimization

The linear approximation of $\nabla \mathcal{L}$ (not $\mathcal{L}$ itself) gives Newton's method:

$$\nabla \mathcal{L}(\boldsymbol{\theta}) \approx \nabla \mathcal{L}(\boldsymbol{\theta}_t) + H(\boldsymbol{\theta}_t)(\boldsymbol{\theta} - \boldsymbol{\theta}_t) = 0$$

Solving: $\boldsymbol{\theta}_{t+1} = \boldsymbol{\theta}_t - H(\boldsymbol{\theta}_t)^{-1} \nabla \mathcal{L}(\boldsymbol{\theta}_t)$.

This uses the **Hessian** $H$ to correct the step direction. Newton's method converges quadratically (error squared each step) vs. linearly for gradient descent. The catch: computing $H^{-1}$ for a network with $n$ parameters costs $O(n^3)$ — utterly infeasible for modern networks ($n \sim 10^9$).

Practical alternatives: **L-BFGS** (quasi-Newton, approximates $H^{-1}$ using past gradient differences) and **natural gradient** / **K-FAC** (approximate Fisher information matrix instead of Hessian).

### 8.4 — Linearization of Neural Networks: the NTK Regime

For very wide networks, the output $f(\mathbf{x}; \boldsymbol{\theta})$ barely changes during training — parameters move little. This means the network stays in the regime where its linear approximation in $\boldsymbol{\theta}$ around initialization $\boldsymbol{\theta}_0$ is valid:

$$f(\mathbf{x}; \boldsymbol{\theta}) \approx f(\mathbf{x}; \boldsymbol{\theta}_0) + \nabla_{\boldsymbol{\theta}} f(\mathbf{x}; \boldsymbol{\theta}_0)^{\top}(\boldsymbol{\theta} - \boldsymbol{\theta}_0)$$

In this linear regime, gradient flow (continuous-time gradient descent) has a closed-form solution. The kernel:

$$K(\mathbf{x}, \mathbf{x}') = \nabla_{\boldsymbol{\theta}} f(\mathbf{x}; \boldsymbol{\theta}_0)^{\top} \nabla_{\boldsymbol{\theta}} f(\mathbf{x}'; \boldsymbol{\theta}_0)$$

is the **Neural Tangent Kernel** (NTK). This shows that in the infinite-width limit, training a neural network is equivalent to kernel regression with a fixed kernel. The tangent (linear) approximation of the network function in parameter space is the entire dynamics.

### 8.5 — Sensitivity Analysis and Saliency Maps

Given a trained classifier $f(\mathbf{x})$, the linear approximation at input $\mathbf{x}_0$:

$$f(\mathbf{x}) \approx f(\mathbf{x}_0) + \nabla_{\mathbf{x}} f(\mathbf{x}_0) \cdot (\mathbf{x} - \mathbf{x}_0)$$

tells us which input features, when perturbed, change the output most. The gradient $\nabla_{\mathbf{x}} f$ is the **saliency map** — used in interpretability to visualize which pixels a convolutional network "attends to." This is exactly reading off the linear approximation of $f$ at a given image.

**LIME** (Local Interpretable Model-agnostic Explanations) takes this further: it fits a linear model to the behavior of $f$ in a neighborhood of any input — a literal local linearization — and uses it as an explanation.

### 8.6 — Adversarial Examples

The linear approximation also explains adversarial examples. A small perturbation $\boldsymbol{\epsilon}$ in the direction of $\nabla_{\mathbf{x}} \mathcal{L}$ maximally increases the loss:

$$\mathcal{L}(\mathbf{x} + \boldsymbol{\epsilon}) \approx \mathcal{L}(\mathbf{x}) + \nabla_{\mathbf{x}}\mathcal{L} \cdot \boldsymbol{\epsilon}$$

Taking $\boldsymbol{\epsilon} = \alpha \cdot \text{sign}(\nabla_{\mathbf{x}}\mathcal{L})$ gives the **FGSM** (Fast Gradient Sign Method) adversarial attack (Goodfellow et al., 2014). The adversarial perturbation is explicitly the step in the direction where the linear approximation of the loss increases fastest — it exploits the fact that the network is locally (nearly) linear.

## §9. Formal Structure — The Fréchet Derivative

In full generality: let $X, Y$ be normed vector spaces, $f: X \to Y$, $\mathbf{a} \in X$.

**Definition.** $f$ is **Fréchet differentiable** at $\mathbf{a}$ if there exists a bounded linear operator $A: X \to Y$ such that:

$$\lim_{|\mathbf{h}|_X \to 0} \frac{|f(\mathbf{a} + \mathbf{h}) - f(\mathbf{a}) - A\mathbf{h}|_Y}{|\mathbf{h}|_X} = 0$$

$A$ is called the **Fréchet derivative** $Df(\mathbf{a})$ of $f$ at $\mathbf{a}$.

Special cases:

- $X = \mathbb{R}$, $Y = \mathbb{R}$: $A = f'(a)$ (scalar multiplication) — ordinary derivative
- $X = \mathbb{R}^n$, $Y = \mathbb{R}^m$: $A = J_\mathbf{f}(\mathbf{a})$ (matrix multiplication) — Jacobian
- $X = C([0,1])$, $Y = \mathbb{R}$: $A$ is an integral operator — calculus of variations

The Fréchet derivative is the **unique** best linear approximation in the above sense. The chain rule still holds: $D(f \circ g)(\mathbf{a}) = Df(g(\mathbf{a})) \circ Dg(\mathbf{a})$ (composition of linear operators).

This framework applies to neural networks as maps between function spaces, and to the analysis of infinite-dimensional optimization problems.

## §10. Summary — The Hierarchy of Approximations

$$f(\mathbf{x}) = \underbrace{f(\mathbf{a})}_{\text{0th order}} + \underbrace{\nabla f(\mathbf{a}) \cdot (\mathbf{x}-\mathbf{a})}_{\text{1st order (linear)}} + \underbrace{\frac{1}{2}(\mathbf{x}-\mathbf{a})^T H(\mathbf{a}) (\mathbf{x}-\mathbf{a})}_{\text{2nd order (quadratic)}} + \cdots$$

|Order|Name|Error|Use Case|
|---|---|---|---|
|0|Constant approx.|$O(\|\Delta\mathbf{x}\|)$|Crude estimate|
|1|Linear approx. (this note)|$O(\|\Delta\mathbf{x}\|^2)$|Gradient descent, error propagation, sensitivity|
|2|Quadratic approx.|$O(\|\Delta\mathbf{x}\|^3)$|Newton's method, curvature analysis|
|$k$|Taylor polynomial|$O(\|\Delta\mathbf{x}\|^{k+1})$|Series solutions, numerical methods|

The linear approximation is the sweet spot: cheap to compute (just first derivatives), error decays quadratically, and it captures directional information (which constant approximation misses).

## §11. Things That Will Trip You Up

> **The linear approximation requires differentiability, not just partial derivatives.** Existence of $f_x$ and $f_y$ is not sufficient. You need them to be continuous, or directly verify the $o(|\Delta\mathbf{x}|)$ limit.

> **The total differential $df$ is not the same as $\Delta f$.** The differential is exact only for linear functions. For everything else, $\Delta f = df + O(|\Delta\mathbf{x}|^2)$.

> **Error bounds require bounding the second derivative.** The approximation "$f(x) \approx L(x)$" is qualitative. Quantitative error analysis always needs the Lagrange remainder.

> **Affine $\neq$ linear.** A linear approximation passing through $(a, f(a))$ with $f(a) \neq 0$ is not a linear function — it is affine. This distinction matters in algebra and optimization.

> **The Jacobian is a matrix, not a determinant.** The Jacobian determinant $\det(J)$ is a scalar used for change-of-variables; the Jacobian matrix is the derivative of a vector-valued map. Do not conflate them.

## §12. Theorems to Know Cold

**Theorem 1** _(Linear Approximation, Scalar)._ If $f$ is differentiable at $a$, then for $x$ near $a$: $$f(x) = f(a) + f'(a)(x-a) + o(x-a)$$

**Theorem 2** _(Linear Approximation, Vector)._ If $f: \mathbb{R}^n \to \mathbb{R}$ is differentiable at $\mathbf{a}$, then: $$f(\mathbf{x}) = f(\mathbf{a}) + \nabla f(\mathbf{a}) \cdot (\mathbf{x}-\mathbf{a}) + o(|\mathbf{x}-\mathbf{a}|)$$

**Theorem 3** _(Error Bound via Taylor)._ If $|f''| \leq M$ on $[a, x]$, then $|f(x) - L(x)| \leq \frac{M}{2}(x-a)^2$.

**Theorem 4** _(Jacobian Chain Rule)._ For differentiable $\mathbf{g}: \mathbb{R}^p \to \mathbb{R}^n$ and $\mathbf{f}: \mathbb{R}^n \to \mathbb{R}^m$: $$J_{\mathbf{f} \circ \mathbf{g}}(\mathbf{a}) = J_\mathbf{f}(\mathbf{g}(\mathbf{a})), J_\mathbf{g}(\mathbf{a})$$

**Theorem 5** _(Fréchet Uniqueness)._ The Fréchet derivative, when it exists, is the unique bounded linear operator satisfying the $o(|\mathbf{h}|)$ error condition.
