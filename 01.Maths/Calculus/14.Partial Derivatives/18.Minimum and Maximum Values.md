

> _"Finding extrema is the central problem of optimization. Everything else — Lagrange multipliers, second derivative tests, convexity — is commentary."_

## 0. Preamble — The Problem

Given a function $f : \mathbb{R}^n \to \mathbb{R}$, we want to find where $f$ is largest or smallest. This question has two flavors:

1. **Unconstrained:** optimize $f$ over all of $\mathbb{R}^n$ (or an open domain)
2. **Constrained:** optimize $f$ subject to $g(\mathbf{x}) = c$ (on a curve, surface, etc.)

Both cases reduce to a single geometric idea: _at an extremum, the gradient must vanish — or be parallel to the constraint_.

---

## 1. Absolute vs. Local Extrema

**Definition.** Let $f : D \subseteq \mathbb{R}^n \to \mathbb{R}$.

- $f$ has an **absolute (global) maximum** at $\mathbf{a} \in D$ if $f(\mathbf{a}) \geq f(\mathbf{x})$ for all $\mathbf{x} \in D$.
- $f$ has an **absolute (global) minimum** at $\mathbf{a} \in D$ if $f(\mathbf{a}) \leq f(\mathbf{x})$ for all $\mathbf{x} \in D$.
- $f$ has a **local maximum** at $\mathbf{a}$ if $f(\mathbf{a}) \geq f(\mathbf{x})$ for all $\mathbf{x}$ in some open ball around $\mathbf{a}$.
- $f$ has a **local minimum** at $\mathbf{a}$ if $f(\mathbf{a}) \leq f(\mathbf{x})$ for all $\mathbf{x}$ in some open ball around $\mathbf{a}$.

**Existence — Extreme Value Theorem (multivariable).** If $f$ is continuous on a _closed and bounded_ set $D \subseteq \mathbb{R}^n$, then $f$ attains both its absolute maximum and minimum on $D$.

> ⚠️ Closed _and_ bounded is essential. On an open domain $f(x) = x$ has no maximum on $(0,1)$. On an unbounded domain $f(x,y) = x$ has no maximum on $\mathbb{R}^2$. Both conditions are required.

## 2. Critical Points — Fermat's Theorem in $\mathbb{R}^n$

**Theorem (Fermat).** _If $f$ has a local extremum at an interior point $\mathbf{a}$ of its domain, and $f$ is differentiable at $\mathbf{a}$, then:_ $$\nabla f(\mathbf{a}) = \mathbf{0}$$

**Proof.** Fix any direction $\hat{\mathbf{e}}_i$. Define $g(t) = f(\mathbf{a} + t\hat{\mathbf{e}}_i)$. Then $g$ has a local extremum at $t = 0$, so by the single-variable Fermat theorem: $$g'(0) = \frac{\partial f}{\partial x_i}(\mathbf{a}) = 0$$ Since $i$ was arbitrary, all partial derivatives vanish, so $\nabla f(\mathbf{a}) = \mathbf{0}$. $\square$

**Definition.** A point $\mathbf{a}$ where $\nabla f(\mathbf{a}) = \mathbf{0}$ (or $\nabla f$ does not exist) is called a **critical point** of $f$.

> ⚠️ The converse is _false_. A critical point need not be a local extremum — it could be a **saddle point**. The gradient vanishing is necessary but not sufficient.

**Example of a saddle point.** $f(x,y) = x^2 - y^2$ at the origin.

$\nabla f = \langle 2x, -2y\rangle = \mathbf{0}$ at $(0,0)$. But $f(t, 0) = t^2 > 0$ and $f(0, t) = -t^2 < 0$ for $t \neq 0$, so $(0,0)$ is neither a local max nor a local min — it is a saddle point (a mountain pass).

## 3. The Second Derivative Test — Classifying Critical Points

### 3.1 The Hessian Matrix

At a critical point $\mathbf{a}$, the second-order behavior of $f$ is encoded by the **Hessian matrix**: $$H_f(\mathbf{a}) = \begin{pmatrix} f_{xx} & f_{xy} \ f_{xy} & f_{yy} \end{pmatrix}\Bigg|_{\mathbf{a}}$$

(assuming $f \in C^2$, so $f_{xy} = f_{yx}$ by Clairaut's theorem).

The Hessian is a real symmetric matrix, so by the spectral theorem all its eigenvalues are real. The signs of these eigenvalues determine the local shape of $f$.

### 3.2 The Discriminant (in $\mathbb{R}^2$)

Define the **discriminant**: $$D(\mathbf{a}) = \det H_f(\mathbf{a}) = f_{xx} f_{yy} - (f_{xy})^2$$

**Theorem (Second Derivative Test).** _Let $\mathbf{a}$ be a critical point of $f \in C^2$._

|Condition|Classification|
|---|---|
|$D(\mathbf{a}) > 0$ and $f_{xx}(\mathbf{a}) > 0$|Local **minimum**|
|$D(\mathbf{a}) > 0$ and $f_{xx}(\mathbf{a}) < 0$|Local **maximum**|
|$D(\mathbf{a}) < 0$|**Saddle point**|
|$D(\mathbf{a}) = 0$|**Inconclusive** (higher-order analysis needed)|

**Why this works.** Near $\mathbf{a}$, by Taylor's theorem with the gradient zero: $$f(\mathbf{a} + \mathbf{h}) \approx f(\mathbf{a}) + \frac{1}{2}\mathbf{h}^T H_f(\mathbf{a}), \mathbf{h}$$

The sign of the quadratic form $Q(\mathbf{h}) = \mathbf{h}^T H \mathbf{h}$ determines everything:

- $Q > 0$ for all $\mathbf{h} \neq \mathbf{0}$ ($H$ is **positive definite**, eigenvalues both positive) $\implies$ local min
- $Q < 0$ for all $\mathbf{h} \neq \mathbf{0}$ ($H$ is **negative definite**, eigenvalues both negative) $\implies$ local max
- $Q$ takes both signs ($H$ is **indefinite**, eigenvalues have mixed signs) $\implies$ saddle

The discriminant $D = \lambda_1 \lambda_2$ (product of eigenvalues = determinant) and $f_{xx} = \lambda_1 + \lambda_2 \cdot (\text{rotation terms})$ encode exactly this.

### 3.3 Full Worked Example

Let $f(x,y) = x^3 - 3x + y^2 - 2y$.

**Step 1 — Find critical points.** $$\nabla f = \langle 3x^2 - 3,; 2y - 2\rangle = \mathbf{0}$$ $$3x^2 - 3 = 0 \implies x = \pm 1, \qquad 2y - 2 = 0 \implies y = 1$$

Critical points: $(-1, 1)$ and $(1, 1)$.

**Step 2 — Compute the Hessian.** $$f_{xx} = 6x, \quad f_{xy} = 0, \quad f_{yy} = 2$$ $$H_f = \begin{pmatrix} 6x & 0 \ 0 & 2 \end{pmatrix}$$

**Step 3 — Apply the test.**

At $(-1, 1)$: $D = (6(-1))(2) - 0 = -12 < 0$ → **saddle point**.

At $(1, 1)$: $D = (6)(2) - 0 = 12 > 0$, $f_{xx} = 6 > 0$ → **local minimum**.

$f(1, 1) = 1 - 3 + 1 - 2 = -3$.

## 4. Absolute Extrema on a Closed Bounded Domain

By the Extreme Value Theorem, absolute extrema exist on a closed bounded domain $D$. The strategy is:

> **Closed Domain Method**
> 
> 1. Find all interior critical points (solve $\nabla f = \mathbf{0}$)
> 2. Examine $f$ on the **boundary** of $D$
> 3. Compare all values — the largest is the absolute max, the smallest is the absolute min

The boundary examination in step 2 typically reduces to a single-variable optimization problem on each boundary piece.

**Example.** Find the absolute extrema of $f(x,y) = x^2 - 2xy + 2y$ on the square $D = [0,3] \times [0,2]$.

**Interior critical points:** $$\nabla f = \langle 2x - 2y,; -2x + 2\rangle = \mathbf{0} \implies x = 1,; y = 1$$ $f(1,1) = 1 - 2 + 2 = 1$.

**Boundary analysis (four edges):**

- $y = 0,; x \in [0,3]$: $f = x^2$. Max at $x=3$: $f=9$. Min at $x=0$: $f=0$.
- $y = 2,; x \in [0,3]$: $f = x^2 - 4x + 4 = (x-2)^2$. Max at $x=0$ or $x=3$: $f=4$. Min at $x=2$: $f=0$.
- $x = 0,; y \in [0,2]$: $f = 2y$. Max at $y=2$: $f=4$. Min at $y=0$: $f=0$.
- $x = 3,; y \in [0,2]$: $f = 9 - 6y + 2y = 9 - 4y$. Max at $y=0$: $f=9$. Min at $y=2$: $f=1$.

**Comparing all values:** ${1, 9, 0, 4, 0, 4, 0, 9, 1}$

**Absolute maximum: $9$** at $(3, 0)$. **Absolute minimum: $0$** at $(0,0)$, $(3/2, 0)$... let me refine — at $(0,0)$ and $(2,2)$.

## 5. Constrained Optimization — Lagrange Multipliers

### 5.1 The Geometric Idea

Suppose we want to maximize $f(x,y)$ subject to $g(x,y) = c$.

Geometrically: we are moving along the constraint curve $g = c$ and asking where $f$ is largest. At the constrained maximum, the level curve of $f$ through that point must be _tangent_ to the constraint curve — otherwise we could slide along the constraint and increase $f$ further.

Two curves are tangent $\iff$ they have the same tangent line $\iff$ they have parallel normal vectors.

Their normals are $\nabla f$ and $\nabla g$. So the condition is: $$\nabla f(\mathbf{a}) = \lambda, \nabla g(\mathbf{a})$$ for some scalar $\lambda$ (the **Lagrange multiplier**).

### 5.2 The Formal Theorem

**Theorem (Lagrange).** _Suppose $f, g \in C^1$ and $f$ has a local extremum at $\mathbf{a}$ subject to the constraint $g(\mathbf{a}) = c$. If $\nabla g(\mathbf{a}) \neq \mathbf{0}$, then there exists $\lambda \in \mathbb{R}$ such that:_ $$\nabla f(\mathbf{a}) = \lambda, \nabla g(\mathbf{a})$$

**Proof sketch.** Since $\nabla g(\mathbf{a}) \neq \mathbf{0}$, the constraint surface $g = c$ is a smooth $(n-1)$-manifold near $\mathbf{a}$. Any tangent vector $\mathbf{v}$ to this surface satisfies $\nabla g(\mathbf{a}) \cdot \mathbf{v} = 0$.

Since $\mathbf{a}$ is a constrained extremum of $f$, the directional derivative of $f$ along any direction tangent to the constraint must vanish: $\nabla f(\mathbf{a}) \cdot \mathbf{v} = 0$ for all such $\mathbf{v}$.

Therefore $\nabla f(\mathbf{a})$ is perpendicular to the entire tangent space of the constraint surface — meaning $\nabla f(\mathbf{a})$ is parallel to the normal $\nabla g(\mathbf{a})$, i.e., $\nabla f(\mathbf{a}) = \lambda \nabla g(\mathbf{a})$. $\square$

### 5.3 The Lagrange System

To find constrained extrema of $f$ subject to $g = c$, solve the system: $$\begin{cases} \nabla f = \lambda, \nabla g \ g(\mathbf{x}) = c \end{cases}$$

In two variables $(x,y)$, this is three equations in three unknowns $(x, y, \lambda)$: $$f_x = \lambda g_x, \quad f_y = \lambda g_y, \quad g(x,y) = c$$

### 5.4 Worked Example

**Find the maximum and minimum values of $f(x,y) = x^2 + y^2$ on the ellipse $\frac{x^2}{4} + y^2 = 1$.**

Wait — this is just distance squared from the origin on an ellipse, so we want the closest and farthest points. Let $g(x,y) = \frac{x^2}{4} + y^2$.

Lagrange system: $$f_x = 2x = \lambda \cdot \frac{x}{2}, \quad f_y = 2y = \lambda \cdot 2y, \quad \frac{x^2}{4} + y^2 = 1$$

**Case 1:** $y \neq 0$. From the second equation: $2 = 2\lambda \implies \lambda = 1$. Substituting: $2x = \frac{x}{2} \implies \frac{3x}{2} = 0 \implies x = 0$. Constraint: $y^2 = 1 \implies y = \pm 1$. Points: $(0, \pm 1)$, $f = 1$.

**Case 2:** $y = 0$. Constraint: $\frac{x^2}{4} = 1 \implies x = \pm 2$. From first eq: $2x = \frac{\lambda x}{2} \implies \lambda = 4$. Points: $(\pm 2, 0)$, $f = 4$.

**Conclusion:** Maximum value $4$ at $(\pm 2, 0)$; minimum value $1$ at $(0, \pm 1)$. ✓ (These are the semi-axes of the ellipse.)

### 5.5 Two Constraints

For $f(\mathbf{x})$ subject to $g(\mathbf{x}) = c_1$ and $h(\mathbf{x}) = c_2$: $$\nabla f = \lambda, \nabla g + \mu, \nabla h$$

## 7. The Hessian — General $\mathbb{R}^n$ Version

In $\mathbb{R}^n$, the Hessian at a critical point $\mathbf{a}$ is the $n \times n$ symmetric matrix: $$[H_f(\mathbf{a})]_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}(\mathbf{a})$$

The classification is by definiteness of $H_f(\mathbf{a})$:

|Hessian $H$|All eigenvalues|Conclusion|
|---|---|---|
|Positive definite|$> 0$|Local minimum|
|Negative definite|$< 0$|Local maximum|
|Indefinite|Mixed signs|Saddle point|
|Positive/Negative semidefinite|$\geq 0$ or $\leq 0$|Inconclusive|

The determinant test in $\mathbb{R}^2$ is just the $2 \times 2$ case of checking definiteness via eigenvalue signs ($D = \lambda_1\lambda_2 > 0$ means same sign; $f_{xx} > 0$ means both positive).

## 8. Connection to AI, Machine Learning, and Deep Learning

### 8.1 The Entire Goal of Training is Minimization

Every supervised learning model minimizes a loss function $\mathcal{L}(\boldsymbol{\theta})$ over parameters $\boldsymbol{\theta} \in \mathbb{R}^n$ (where $n$ can be $10^9$ or more). The theory of min/max is the mathematical foundation.

**Critical points of $\mathcal{L}$:** $\nabla_{\boldsymbol{\theta}} \mathcal{L} = \mathbf{0}$. These are where gradient descent stalls. Understanding whether a critical point is a minimum, maximum, or saddle is the central problem of deep learning optimization theory.

### 8.2 The Loss Landscape — What Are We Minimizing Into?

The loss surface $\mathcal{L}(\boldsymbol{\theta})$ of a deep neural network is a hypersurface in billions of dimensions. Its landscape looks nothing like the neat single-variable pictures:

- **Local minima** are rare. In high dimensions, a local minimum requires all eigenvalues of the Hessian to be positive — exponentially unlikely if eigenvalues are random.
- **Saddle points dominate.** Most critical points have Hessians with some positive and some negative eigenvalues. The network gets stuck near saddles, not local minima.
- **Global structure:** Surprisingly, for overparameterized networks (more parameters than data), many local minima have similar loss values — the landscape has many "flat valleys" at the bottom.

**Empirical insight (Dauphin et al., 2014):** The difficulty in training deep networks is _escaping saddle points_, not getting stuck in bad local minima.

### 8.3 Second-Order Methods — Using the Hessian

**Gradient descent** uses only $\nabla\mathcal{L}$ (first-order). **Newton's method** uses $H_\mathcal{L}$ (second-order): $$\boldsymbol{\theta}_{t+1} = \boldsymbol{\theta}_t - \eta, [H_{\mathcal{L}}(\boldsymbol{\theta}_t)]^{-1} \nabla \mathcal{L}(\boldsymbol{\theta}_t)$$

At a local minimum, $H_{\mathcal{L}}$ is positive definite (second derivative test!), so $H^{-1}$ scales the gradient step to take into account the local curvature — taking large steps in flat directions, small steps in sharp directions. This is exactly the second derivative test applied operationally.

**Why not always use Newton's method?** Inverting $H_{\mathcal{L}} \in \mathbb{R}^{n \times n}$ costs $O(n^3)$ — infeasible for $n = 10^9$. This motivates quasi-Newton methods (BFGS, L-BFGS) and natural gradient methods that approximate $H^{-1}$.

### 8.4 Convexity — When Local = Global

**Definition.** $f$ is _convex_ if for all $\mathbf{x}, \mathbf{y}$ and $t \in [0,1]$: $$f(t\mathbf{x} + (1-t)\mathbf{y}) \leq t f(\mathbf{x}) + (1-t)f(\mathbf{y})$$

**Theorem.** If $f$ is convex and $\nabla f(\mathbf{a}) = \mathbf{0}$, then $\mathbf{a}$ is a **global minimum**.

This is the key property. For convex loss functions (logistic regression, SVMs, linear regression), every critical point is a global minimum — the optimization is "safe." For non-convex losses (deep networks), there is no such guarantee.

The Hessian connection: $f$ is convex $\iff$ $H_f$ is positive semidefinite everywhere ($\iff$ all eigenvalues $\geq 0$). This is the global version of the local minimum condition from the second derivative test.

### 8.5 Lagrange Multipliers in ML — Constrained Learning

Lagrange multipliers appear explicitly in:

**Support Vector Machines (SVM).** Maximize the margin subject to classification constraints. The Lagrangian is: $$\mathcal{L}(\mathbf{w}, b, \boldsymbol{\alpha}) = \frac{1}{2}|\mathbf{w}|^2 - \sum_i \alpha_i [y_i(\mathbf{w}\cdot\mathbf{x}_i + b) - 1]$$ The KKT conditions (generalized Lagrange conditions for inequalities) completely characterize the optimal $\mathbf{w}$.

**Regularization as a Lagrange problem.** Minimize $\mathcal{L}(\boldsymbol{\theta})$ subject to $|\boldsymbol{\theta}|^2 \leq C$ is equivalent (by Lagrange) to minimizing $\mathcal{L}(\boldsymbol{\theta}) + \lambda|\boldsymbol{\theta}|^2$ (ridge regression / $L^2$ regularization). The multiplier $\lambda$ is the regularization coefficient.

**Variational autoencoders (VAE).** The ELBO objective is a constrained optimization (maximize likelihood subject to KL divergence constraint), handled via Lagrange-style relaxation.

### 8.6 Summary — Min/Max Theory in the ML Pipeline

|Calculus Concept|ML Manifestation|
|---|---|
|Critical point $\nabla f = \mathbf{0}$|Trained model (gradient descent converges)|
|Second derivative test (Hessian)|Diagnosing minima vs saddle points|
|Saddle point|Gradient descent stall; motivates Adam, momentum|
|Absolute minimum on closed domain|Constrained parameter spaces|
|Lagrange multipliers|SVMs, regularization, constrained decoding|
|Convexity (positive definite Hessian)|Loss functions with guaranteed global min|
|Newton's method|Second-order optimizers (L-BFGS, K-FAC)|

## 9. The Procedure — A Decision Tree

```
Given f on domain D, find extrema
│
├─ Is D closed and bounded?
│   ├─ YES → Extreme Value Theorem guarantees extrema exist
│   │         → Find interior critical points (∇f = 0)
│   │         → Analyze boundary (parameterize, or use Lagrange)
│   │         → Compare all values
│   │
│   └─ NO  → Only local extrema may exist
│             → Find critical points (∇f = 0)
│             → Apply Second Derivative Test (Hessian)
│
└─ Is there a constraint g(x) = c?
    └─ YES → Lagrange: solve ∇f = λ∇g, g = c
             → Evaluate f at solutions
             → Compare values (no 2nd derivative test needed
               if only finitely many solutions and domain compact)
```

## 10. A Subtle Point — When the Test Fails ($D = 0$)

When $D(\mathbf{a}) = 0$, the Hessian is singular (one eigenvalue is zero). The quadratic approximation $\frac{1}{2}\mathbf{h}^T H \mathbf{h}$ is neither positive nor negative definite, and higher-order terms determine the behavior.

**Example.** $f(x,y) = x^4 + y^4$. At $(0,0)$: $\nabla f = \mathbf{0}$, $H = 0$, $D = 0$. Test is inconclusive. But clearly $f \geq 0$ with equality only at origin, so $(0,0)$ is a global minimum.

**Example.** $f(x,y) = x^3$. At $(0,0)$: $\nabla f = \mathbf{0}$, $D = 0$. But $f$ takes both signs near the origin, so $(0,0)$ is a saddle-like point.

When $D = 0$: compute $f$ along various paths through $\mathbf{a}$ directly.
