## 0. Origins and Context

In 1927, mathematician Charles Cobb and economist Paul Douglas fit a simple power-law model to U.S. manufacturing data from 1899–1922. They asked: given observed inputs of capital $K$ and labor $L$, can we predict total output $Q$?

The function they proposed:

$$Q = AK^\alpha L^\beta$$

fit the data remarkably well. More importantly, it had mathematical properties that aligned almost perfectly with economic intuition — constant returns to scale, diminishing marginal products, and tractable closed-form solutions for optimization. It remains, nearly a century later, the workhorse model of production theory in economics and appears throughout machine learning theory as well.

This note treats the Cobb-Douglas function with full mathematical rigor, using it as a sustained application of multivariable calculus.

---

## 1. The Function — Definition and Parameters

### Standard Form

$$Q(K, L) = A K^\alpha L^\beta$$

where:

|Symbol|Name|Domain|Interpretation|
|---|---|---|---|
|$Q$|Output|$\mathbb{R}_{>0}$|Total production (goods, revenue, GDP)|
|$K$|Capital|$\mathbb{R}_{>0}$|Machines, buildings, equipment|
|$L$|Labor|$\mathbb{R}_{>0}$|Worker-hours, headcount|
|$A$|Total Factor Productivity (TFP)|$\mathbb{R}_{>0}$|Technology, efficiency, institutional quality|
|$\alpha$|Capital elasticity|$(0,1)$ typically|Capital's output share|
|$\beta$|Labor elasticity|$(0,1)$ typically|Labor's output share|

The domain of $Q$ is $(K,L) \in \mathbb{R}_{>0}^2$. The function is not defined at $K=0$ or $L=0$ (no output without both inputs — a modeling assumption).

### The Three Regimes

The relationship between $\alpha + \beta$ and $1$ defines the returns-to-scale behavior (see §4):

|Condition|Regime|Economic meaning|
|---|---|---|
|$\alpha + \beta = 1$|Constant Returns to Scale (CRS)|Doubling inputs doubles output|
|$\alpha + \beta > 1$|Increasing Returns to Scale (IRS)|Doubling inputs more than doubles output|
|$\alpha + \beta < 1$|Decreasing Returns to Scale (DRS)|Doubling inputs less than doubles output|

The original Cobb-Douglas paper estimated $\alpha \approx 0.25$, $\beta \approx 0.75$ for U.S. manufacturing — CRS, approximately.

## 2. Partial Derivatives — Marginal Products

The first partial derivatives of $Q$ with respect to each input are the **marginal products** — the additional output from one more unit of an input, holding the other fixed.

### Marginal Product of Capital (MPK)

$$\frac{\partial Q}{\partial K} = A \cdot \alpha K^{\alpha - 1} L^\beta = \alpha \cdot \frac{AK^\alpha L^\beta}{K} = \frac{\alpha Q}{K}$$

### Marginal Product of Labor (MPL)

$$\frac{\partial Q}{\partial L} = A K^\alpha \cdot \beta L^{\beta - 1} = \beta \cdot \frac{AK^\alpha L^\beta}{L} = \frac{\beta Q}{L}$$

### Key Observations

**1. The marginal products are proportional to average products:**

$$\text{MPK} = \alpha \cdot \frac{Q}{K} = \alpha \cdot \text{APK}, \qquad \text{MPL} = \beta \cdot \frac{Q}{L} = \beta \cdot \text{APL}$$

This elegant identity — that marginal products are constant fractions of average products — is a special property of the Cobb-Douglas form.

**2. Diminishing marginal products (for $0 < \alpha, \beta < 1$):**

$$\frac{\partial^2 Q}{\partial K^2} = \alpha(\alpha-1)AK^{\alpha-2}L^\beta < 0 \quad \text{since } \alpha - 1 < 0$$

$$\frac{\partial^2 Q}{\partial L^2} = \beta(\beta-1)AK^\alpha L^{\beta-2} < 0 \quad \text{since } \beta - 1 < 0$$

Each additional unit of capital (or labor) adds less output than the previous unit, when the other input is held fixed. The surface is **concave** along each coordinate direction.

**3. Marginal products are strictly positive:**

$$\frac{\partial Q}{\partial K} > 0, \quad \frac{\partial Q}{\partial L} > 0 \quad \text{for all } K, L > 0$$

More inputs always produce more output — a basic monotonicity assumption.

### Example 2.1 — Numerical

Let $A = 1$, $\alpha = 0.3$, $\beta = 0.7$. At $(K, L) = (100, 200)$:

$$Q = 100^{0.3} \cdot 200^{0.7} = e^{0.3\ln 100 + 0.7\ln 200}$$

$$= e^{0.3(4.605) + 0.7(5.298)} = e^{1.3815 + 3.7086} = e^{5.09} \approx 162.8$$

$$\text{MPK} = \frac{0.3 \times 162.8}{100} \approx 0.488 \quad \text{(units of output per unit of capital)}$$

$$\text{MPL} = \frac{0.7 \times 162.8}{200} \approx 0.570 \quad \text{(units of output per unit of labor)}$$

Interpretation: at this operating point, an additional unit of labor is slightly more productive than an additional unit of capital.

## 3. The Gradient and Economic Sensitivity

The gradient of $Q$ packages both marginal products:

$$\nabla Q(K,L) = \left(\frac{\partial Q}{\partial K},, \frac{\partial Q}{\partial L}\right) = \left(\frac{\alpha Q}{K},, \frac{\beta Q}{L}\right)$$

**Direction of steepest ascent:** $\nabla Q$ points in the direction of maximum output growth from $(K,L)$. A firm seeking to maximally increase production with one unit of additional expenditure should allocate resources in the direction of $\nabla Q$ (weighted by input prices — see §6).

**Total differential:** For small changes $dK$ and $dL$:

$$dQ = \frac{\partial Q}{\partial K}dK + \frac{\partial Q}{\partial L}dL = \frac{\alpha Q}{K}dK + \frac{\beta Q}{L}dL$$

Dividing by $Q$:

$$\frac{dQ}{Q} = \alpha \frac{dK}{K} + \beta \frac{dL}{L}$$

**This is the fundamental identity of the Cobb-Douglas:** the _proportional_ change in output equals $\alpha$ times the proportional change in capital plus $\beta$ times the proportional change in labor. This will be recognized as the elasticity interpretation (§5).

## 4. Returns to Scale — Homogeneity

### Definition

A function $f(K,L)$ is **homogeneous of degree $r$** if for all $\lambda > 0$:

$$f(\lambda K, \lambda L) = \lambda^r f(K, L)$$

### Theorem — Cobb-Douglas is Homogeneous of Degree $\alpha + \beta$

**Proof:**

$$Q(\lambda K, \lambda L) = A(\lambda K)^\alpha (\lambda L)^\beta = A \lambda^\alpha K^\alpha \lambda^\beta L^\beta = \lambda^{\alpha+\beta} A K^\alpha L^\beta = \lambda^{\alpha+\beta} Q(K,L) \quad \blacksquare$$

### Euler's Theorem for Homogeneous Functions

> **Theorem (Euler):** If $f : \mathbb{R}^n \to \mathbb{R}$ is homogeneous of degree $r$ and differentiable, then: $$\sum_{i=1}^n x_i \frac{\partial f}{\partial x_i} = r \cdot f(\mathbf{x})$$

**Proof:** Differentiate $f(\lambda \mathbf{x}) = \lambda^r f(\mathbf{x})$ with respect to $\lambda$ using the chain rule:

$$\sum_{i=1}^n x_i \frac{\partial f}{\partial (\lambda x_i)} = r\lambda^{r-1} f(\mathbf{x})$$

Set $\lambda = 1$:

$$\sum_{i=1}^n x_i \frac{\partial f}{\partial x_i} = r f(\mathbf{x}) \quad \blacksquare$$

### Application to Cobb-Douglas

Applying Euler's theorem to $Q = AK^\alpha L^\beta$:

$$K \frac{\partial Q}{\partial K} + L \frac{\partial Q}{\partial L} = (\alpha + \beta) Q$$

Substituting the marginal products:

$$K \cdot \frac{\alpha Q}{K} + L \cdot \frac{\beta Q}{L} = \alpha Q + \beta Q = (\alpha+\beta)Q \checkmark$$

### Economic Interpretation of Euler's Theorem (CRS Case)

When $\alpha + \beta = 1$:

$$K \cdot \text{MPK} + L \cdot \text{MPL} = Q$$

If each input is paid its marginal product (the competitive equilibrium condition), and if returns to scale are constant, then **total payments to inputs exactly exhaust total output** — no surplus, no deficit. This is the mathematical foundation of the **factor exhaustion theorem** in economics.

## 5. Elasticities — The Logarithmic Derivative

### Definition

The **output elasticity** of input $x_i$ is:

$$\varepsilon_i = \frac{\partial \ln Q}{\partial \ln x_i} = \frac{x_i}{Q}\frac{\partial Q}{\partial x_i}$$

It measures the percentage change in output per 1% change in input $x_i$, all else fixed.

### Theorem — Elasticities of Cobb-Douglas are Constant

$$\varepsilon_K = \frac{K}{Q}\cdot\frac{\alpha Q}{K} = \alpha$$

$$\varepsilon_L = \frac{L}{Q}\cdot\frac{\beta Q}{L} = \beta$$

The parameters $\alpha$ and $\beta$ **are** the elasticities — constant everywhere on the domain. This is the defining property of the Cobb-Douglas function; it is the unique production function with constant output elasticities.

### The Log-Linear Transformation

Taking logarithms:

$$\ln Q = \ln A + \alpha \ln K + \beta \ln L$$

This is **linear** in $\ln A$, $\ln K$, $\ln L$. Setting $q = \ln Q$, $a = \ln A$, $k = \ln K$, $l = \ln L$:

$$q = a + \alpha k + \beta l$$

This is a linear regression model in log-space. This is precisely why Cobb and Douglas could fit the model with simple linear regression on log-transformed data — and why Cobb-Douglas production functions are so tractable in empirical work.

The partial derivatives in log-space:

$$\frac{\partial q}{\partial k} = \alpha, \qquad \frac{\partial q}{\partial l} = \beta$$

Constant. The slopes in log-log space are the elasticities. Plotting $\ln Q$ vs $\ln K$ (with $L$ fixed) gives a straight line of slope $\alpha$.

## 6. Isoquants and the Marginal Rate of Technical Substitution

### Isoquants

An **isoquant** is a level curve of $Q$:

$${(K,L) : AK^\alpha L^\beta = \bar{Q}} \quad \text{for fixed } \bar{Q} > 0$$

Solving for $L$:

$$L = \left(\frac{\bar{Q}}{A}\right)^{1/\beta} K^{-\alpha/\beta}$$

This is a decreasing, convex curve in the $(K,L)$ plane — the familiar downward-sloping, bowed-inward isoquant of economics textbooks.

```
  L (Labor)
  |
  |·
  | ·
  |  ·
  |   ·
  |    ··
  |      ···         Q = Q₃
  |         ····     Q = Q₂
  |             ···· Q = Q₁ (Q₁ < Q₂ < Q₃)
  +------------------→ K (Capital)
  
  Each curve: AK^α L^β = const
  Convex to origin → diminishing MRTS
```

### Marginal Rate of Technical Substitution (MRTS)

The MRTS is the rate at which labor can be substituted for capital (or vice versa) while keeping output constant — the slope of the isoquant (in magnitude).

Using implicit differentiation on $AK^\alpha L^\beta = \bar{Q}$:

Differentiate with respect to $K$ (treating $L = L(K)$):

$$\alpha A K^{\alpha-1}L^\beta + A K^\alpha \beta L^{\beta-1}\frac{dL}{dK} = 0$$

$$\frac{dL}{dK} = -\frac{\alpha A K^{\alpha-1} L^\beta}{\beta A K^\alpha L^{\beta - 1}} = -\frac{\alpha L}{\beta K}$$

Therefore:

$$\text{MRTS}_{KL} = -\frac{dL}{dK} = \frac{\alpha L}{\beta K} = \frac{\text{MPK}}{\text{MPL}}$$

**Interpretation:** The slope of the isoquant equals the ratio of marginal products. To substitute one unit of capital out while staying on the same isoquant, you must add $\text{MPK}/\text{MPL}$ units of labor.

**Diminishing MRTS:** As $K$ increases along an isoquant (and $L$ falls correspondingly):

$$\frac{d(\text{MRTS})}{dK} < 0$$

(Can be verified; requires second-order implicit differentiation.) The isoquant gets flatter — capital becomes progressively less substitutable for labor. This is the mathematical statement of the bowed-in shape of isoquants.

## 7. Firm Optimization — Cost Minimization

A firm wants to produce output $\bar{Q}$ at minimum cost. Input prices: capital costs $r$ per unit, labor costs $w$ per unit. The problem:

$$\min_{K,L > 0} ; rK + wL \quad \text{subject to} \quad AK^\alpha L^\beta = \bar{Q}$$

### Method: Lagrange Multipliers

Form the Lagrangian:

$$\mathcal{L}(K, L, \lambda) = rK + wL - \lambda\left(AK^\alpha L^\beta - \bar{Q}\right)$$

**First-order conditions:**

$$\frac{\partial \mathcal{L}}{\partial K} = r - \lambda \alpha A K^{\alpha-1} L^\beta = 0 \implies r = \lambda \cdot \text{MPK} = \lambda \cdot \frac{\alpha Q}{K}$$

$$\frac{\partial \mathcal{L}}{\partial L} = w - \lambda \beta A K^\alpha L^{\beta-1} = 0 \implies w = \lambda \cdot \text{MPL} = \lambda \cdot \frac{\beta Q}{L}$$

$$\frac{\partial \mathcal{L}}{\partial \lambda} = -(AK^\alpha L^\beta - \bar{Q}) = 0 \implies AK^\alpha L^\beta = \bar{Q}$$

**Divide the first two conditions:**

$$\frac{r}{w} = \frac{\lambda \alpha Q/K}{\lambda \beta Q/L} = \frac{\alpha L}{\beta K} = \text{MRTS}_{KL}$$

The optimality condition: **at the cost-minimizing input mix, the MRTS equals the input price ratio.**

Geometrically: the isoquant is tangent to the isocost line $rK + wL = C$.

### Solving for Optimal Inputs

From $\frac{r}{w} = \frac{\alpha L}{\beta K}$:

$$L = \frac{\beta r}{\alpha w} K$$

Substitute into the constraint $AK^\alpha L^\beta = \bar{Q}$:

$$AK^\alpha \left(\frac{\beta r}{\alpha w}\right)^\beta K^\beta = \bar{Q}$$

$$A \left(\frac{\beta r}{\alpha w}\right)^\beta K^{\alpha+\beta} = \bar{Q}$$

$$K^* = \left[\frac{\bar{Q}}{A}\left(\frac{\alpha w}{\beta r}\right)^\beta\right]^{1/(\alpha+\beta)}$$

$$L^* = \left[\frac{\bar{Q}}{A}\left(\frac{\beta r}{\alpha w}\right)^\alpha\right]^{1/(\alpha+\beta)}$$

### The Cost Function

The minimum cost:

$$C^* = rK^* + wL^* = \bar{Q}^{1/(\alpha+\beta)} \cdot A^{-1/(\alpha+\beta)} \cdot B(r,w,\alpha,\beta)$$

where $B$ is a function of prices and parameters.

**For CRS ($\alpha+\beta=1$):** $C^* = \bar{Q} \cdot c(r,w)$ — cost is linear in output. Doubling output exactly doubles cost. This is the mathematical statement of constant returns: no cost advantage to scale.

**The Lagrange multiplier $\lambda$:** From the first-order conditions:

$$\lambda = \frac{r}{\text{MPK}} = \frac{w}{\text{MPL}} = \frac{\partial C^*}{\partial \bar{Q}}$$

The multiplier equals the **marginal cost** — the rate at which minimum cost increases as the output target rises. The envelope theorem makes this precise.

## 8. The Hessian and Second-Order Analysis

### Computing the Full Hessian

$$\frac{\partial^2 Q}{\partial K^2} = \alpha(\alpha-1)AK^{\alpha-2}L^\beta = \frac{\alpha(\alpha-1)Q}{K^2}$$

$$\frac{\partial^2 Q}{\partial L^2} = \beta(\beta-1)AK^\alpha L^{\beta-2} = \frac{\beta(\beta-1)Q}{L^2}$$

$$\frac{\partial^2 Q}{\partial K \partial L} = \alpha\beta AK^{\alpha-1}L^{\beta-1} = \frac{\alpha\beta Q}{KL}$$

So the Hessian is:

$$H_Q = \frac{Q}{K^2 L^2}\begin{pmatrix} \alpha(\alpha-1)L^2 & \alpha\beta KL \ \alpha\beta KL & \beta(\beta-1)K^2 \end{pmatrix}$$

### Is $Q$ Concave?

Compute $\det(H_Q)$:

$$\det(H_Q) = \frac{Q^2}{K^4L^4}\left[\alpha(\alpha-1)\beta(\beta-1)K^2L^4 - \alpha^2\beta^2K^2L^2 \cdot \frac{L^2}{L^2}\right]$$

Let me compute directly:

$$\det(H_Q) = \frac{Q^2}{K^4 L^4} \cdot K^2 L^2 \left[\alpha(\alpha-1)\beta(\beta-1)L^2 - \alpha^2\beta^2 K^2 \cdot \frac{K^2}{K^2}\right]$$

More cleanly, from the entries:

$$\det(H_Q) = \left(\frac{\alpha(\alpha-1)Q}{K^2}\right)!\left(\frac{\beta(\beta-1)Q}{L^2}\right) - \left(\frac{\alpha\beta Q}{KL}\right)^2$$

$$= \frac{Q^2}{K^2 L^2}\left[\alpha(\alpha-1)\beta(\beta-1) - \alpha^2\beta^2\right]$$

$$= \frac{Q^2 \alpha\beta}{K^2 L^2}\left[(\alpha-1)(\beta-1) - \alpha\beta\right]$$

$$= \frac{Q^2 \alpha\beta}{K^2 L^2}\left[\alpha\beta - \alpha - \beta + 1 - \alpha\beta\right]$$

$$= \frac{Q^2 \alpha\beta}{K^2 L^2}(1 - \alpha - \beta)$$

**Cases:**

- $\alpha + \beta < 1$ (DRS): $\det(H_Q) > 0$ and $Q_{KK} < 0$ → **$H_Q$ negative definite** → $Q$ is **strictly concave**.
    
- $\alpha + \beta = 1$ (CRS): $\det(H_Q) = 0$ → $H_Q$ is negative **semi**definite → $Q$ is concave but not strictly so.
    
- $\alpha + \beta > 1$ (IRS): $\det(H_Q) < 0$ → $H_Q$ is **indefinite** → $Q$ is neither concave nor convex globally.
    

**Economic significance:** Strict concavity of $Q$ (DRS case) implies that the set ${(K,L) : Q(K,L) \geq \bar{Q}}$ is convex — the "upper contour set" is convex. This guarantees that the cost minimization problem has a unique solution and that the isoquants are strictly convex to the origin.

## 9. Multi-Factor Generalization

The Cobb-Douglas generalizes naturally to $n$ inputs:

$$Q(x_1, x_2, \ldots, x_n) = A \prod_{i=1}^n x_i^{\alpha_i}$$

**Partial derivative** (marginal product of input $i$):

$$\frac{\partial Q}{\partial x_i} = \frac{\alpha_i Q}{x_i}$$

**Output elasticity of $x_i$:** $\varepsilon_i = \alpha_i$ (still constant).

**Homogeneity:** $Q(\lambda \mathbf{x}) = \lambda^{\sum \alpha_i} Q(\mathbf{x})$ (homogeneous of degree $\sum \alpha_i$).

**Euler's theorem:**

$$\sum_{i=1}^n x_i \frac{\partial Q}{\partial x_i} = \left(\sum_{i=1}^n \alpha_i\right) Q$$

**Log-linear form:**

$$\ln Q = \ln A + \sum_{i=1}^n \alpha_i \ln x_i$$

This is a multivariate linear regression in log-space — the empirical foundation of all Cobb-Douglas estimation.

## 10. Taylor Expansion Near an Operating Point

Suppose a firm operates at $(K_0, L_0)$ and considers small deviations. Using the multivariable Taylor expansion:

$$Q(K_0 + \Delta K,, L_0 + \Delta L) \approx Q_0 + \frac{\alpha Q_0}{K_0}\Delta K + \frac{\beta Q_0}{L_0}\Delta L$$ $$+ \frac{1}{2}\left[\frac{\alpha(\alpha-1)Q_0}{K_0^2}(\Delta K)^2 + \frac{2\alpha\beta Q_0}{K_0 L_0}\Delta K \Delta L + \frac{\beta(\beta-1)Q_0}{L_0^2}(\Delta L)^2\right] + O(|\mathbf{\Delta}|^3)$$

**Proportional form** (divide by $Q_0$):

$$\frac{\Delta Q}{Q_0} \approx \alpha\frac{\Delta K}{K_0} + \beta\frac{\Delta L}{L_0} + \frac{1}{2}\left[\alpha(\alpha-1)\left(\frac{\Delta K}{K_0}\right)^2 + 2\alpha\beta\frac{\Delta K}{K_0}\frac{\Delta L}{L_0} + \beta(\beta-1)\left(\frac{\Delta L}{L_0}\right)^2\right]$$

The first-order term recovers the elasticity formula from §5. The second-order term contains the **curvature corrections** — negative (for $\alpha,\beta < 1$) since $\alpha - 1 < 0$ and $\beta - 1 < 0$, confirming diminishing returns.

## 11. Connection to AI, Machine Learning, and Deep Learning

### 11.1 Cobb-Douglas as a Scalable Systems Model

In ML infrastructure and systems research, the throughput of distributed training runs is often modeled with Cobb-Douglas-like functions:

$$\text{Throughput} = A \cdot (\text{Compute})^\alpha \cdot (\text{Bandwidth})^\beta \cdot (\text{Memory})^\gamma$$

The partial derivatives give **bottleneck analysis**: which resource has the highest marginal product? Allocate budget toward the input with the highest $\frac{\partial \text{Throughput}}{\partial \text{Resource}} / \text{Resource price}$.

### 11.2 Neural Scaling Laws — A Cobb-Douglas Structure

The empirical **scaling laws** of large language models (Kaplan et al., 2020; Hoffmann et al., 2022) relate loss to compute $C$, data $D$, and parameters $N$:

$$\mathcal{L} \approx A \cdot N^{-\alpha} + B \cdot D^{-\beta} + \text{irreducible loss}$$

This is not exactly Cobb-Douglas, but the analysis of **optimal allocation** — given a fixed compute budget $C = 6ND$ (FLOPs ≈ $6 \times$ parameters $\times$ training tokens), how to split between $N$ and $D$ to minimize $\mathcal{L}$ — is formally identical to the cost minimization problem in §7.

The Chinchilla result (Hoffmann et al.): at fixed compute, the optimal model size and token count satisfy $N^* \propto C^{0.5}$, $D^* \propto C^{0.5}$ — meaning parameters and data should scale equally. This was derived by minimizing the scaling law subject to the compute constraint, using exactly the Lagrange multiplier technique of §7.

### 11.3 The Log-Linear Form as a Neural Network Layer

Recall the log-linear form:

$$\ln Q = \ln A + \alpha \ln K + \beta \ln L$$

This is a **linear layer** applied to log-inputs. In a neural network processing economic or physical data on exponential scales, applying $\log$ before a linear layer exactly learns Cobb-Douglas relationships. This is the theoretical foundation for using log-transforms in regression networks on economic data.

### 11.4 Constant Elasticity of Substitution (CES) — Generalization

Cobb-Douglas is the limiting case of the **CES production function** as substitution elasticity $\sigma \to 1$:

$$Q = A\left[\alpha K^{(\sigma-1)/\sigma} + (1-\alpha)L^{(\sigma-1)/\sigma}\right]^{\sigma/(\sigma-1)}$$

As $\sigma \to 1$: $Q \to AK^\alpha L^{1-\alpha}$ (Cobb-Douglas with $\beta = 1-\alpha$).

In ML, CES-like aggregation functions appear in **mixture-of-experts** models and **attention mechanisms**, where the elasticity parameter $\sigma$ controls how "hard" or "soft" the aggregation is. $\sigma \to \infty$ gives linear (perfect substitution); $\sigma \to 0$ gives Leontief (perfect complements, like an AND gate).

### 11.5 Lagrange Multipliers → Constrained Optimization in ML

The cost minimization problem (§7) is formally identical to:

> Minimize $rK + wL$ subject to $AK^\alpha L^\beta = \bar{Q}$

and the training problem:

> Minimize $\mathcal{L}(\theta)$ subject to $|\theta| \leq C$ (L2 regularization / weight decay as a constraint)

Both are solved by Lagrange multipliers. The KKT conditions in constrained ML optimization are the direct generalization. The Lagrange multiplier $\lambda$ becomes the **regularization strength** — exactly the marginal cost interpretation: $\lambda = \partial \mathcal{L}^*/\partial C$ tells you how much the minimum loss decreases per unit increase in the weight budget.

## 12. Level Curves, Gradient, and Geometric Picture

### Gradient of $Q$

$$\nabla Q = \left(\frac{\alpha Q}{K},, \frac{\beta Q}{L}\right)$$

The gradient is always positive in both components — $Q$ is strictly increasing in each input. The gradient points "uphill" on the production surface, perpendicular to the isoquants.

### Normal to Isoquant

The isoquant $AK^\alpha L^\beta = \bar{Q}$ is a level curve of $Q$. The gradient $\nabla Q$ is perpendicular to this level curve at every point.

At the cost-minimizing point, the isocost line $rK + wL = C^*$ is tangent to the isoquant. Its normal vector is $(r, w)$. Tangency requires:

$$\nabla Q \parallel (r, w) \implies \frac{\alpha Q / K}{\beta Q / L} = \frac{r}{w} \implies \text{MRTS} = \frac{r}{w}$$

This recovers the optimality condition of §7 from a purely geometric argument. The gradient being parallel to the price vector is the multivariable calculus statement of the tangency condition.

```
  L
  |
  |·
  | ·     ← isoquant AK^α L^β = Q̄
  |  ·  
  |   * ← optimal (K*, L*)
  |    \   ← isocost rK + wL = C*
  |     \
  |      \
  +-------\---→ K

  At *: ∇Q ∥ (r, w) — gradient perpendicular to isocost
  MRTS = r/w — slopes match
```

## 13. Summary — The Calculus of Cobb-Douglas

```
Q(K,L) = AK^α L^β
│
├─ FIRST PARTIALS (Marginal Products)
│   ├─ ∂Q/∂K = αQ/K  (MPK — diminishing for α < 1)
│   └─ ∂Q/∂L = βQ/L  (MPL — diminishing for β < 1)
│
├─ GRADIENT
│   └─ ∇Q = (αQ/K, βQ/L) — perpendicular to isoquants
│
├─ HOMOGENEITY
│   ├─ Q(λK, λL) = λ^{α+β} Q(K,L)
│   └─ Euler: K·MPK + L·MPL = (α+β)Q
│
├─ ELASTICITIES
│   ├─ ε_K = α, ε_L = β  (constant — defining property)
│   └─ ln Q = ln A + α ln K + β ln L  (log-linear)
│
├─ ISOQUANTS
│   ├─ Level curves of Q — convex, downward sloping
│   └─ MRTS = αL/βK = MPK/MPL
│
├─ OPTIMIZATION (Cost Minimization)
│   ├─ Lagrange: MRTS = r/w at optimum
│   ├─ K* = [...], L* = [...]  (closed form)
│   └─ λ = marginal cost = ∂C*/∂Q̄
│
├─ HESSIAN (Concavity)
│   ├─ det(H) = Q²αβ(1-α-β)/(K²L²)
│   ├─ α+β < 1 → H negative definite → Q strictly concave
│   └─ α+β = 1 → H negative semidefinite → Q concave
│
└─ IN ML/AI
    ├─ Scaling laws → same optimization structure
    ├─ Log-linear form → linear layer on log-inputs
    ├─ Lagrange multipliers → regularized training
    └─ CES generalization → mixture-of-experts, attention
```
