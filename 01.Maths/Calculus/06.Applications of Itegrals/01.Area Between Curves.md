

> _"The area between two curves is conceptually the simplest application of the definite integral — and yet it is where students first learn to read geometry carefully before writing symbols. The formula is trivial; finding the limits and handling sign changes is where the real work lives."_

## § 0 — From Single Curve to Two Curves

In the definite integral, we measured the signed area between $f(x)$ and the $x$-axis. The $x$-axis is just the function $g(x) = 0$. The natural generalization: what is the area _between_ two arbitrary curves $f$ and $g$?

The idea is unchanged — Riemann rectangles, now with height $f(x_i^_) - g(x_i^_)$ instead of $f(x_i^*)$ alone.

---

## § 1 — The Basic Formula

### Setup

Let $f$ and $g$ be continuous on $[a, b]$ with $f(x) \ge g(x)$ for all $x \in [a, b]$.

The area of the region bounded above by $f$ and below by $g$ on $[a, b]$ is:

$$\boxed{A = \int_a^b \bigl[f(x) - g(x)\bigr],dx}$$

### Derivation from Riemann Sums

Partition $[a,b]$ with sample points $x_i^*$. Each rectangle has:

- Width: $\Delta x_i$
- Height: $f(x_i^_) - g(x_i^_)$ (top curve minus bottom curve)
- Area: $[f(x_i^_) - g(x_i^_)],\Delta x_i \ge 0$

Summing and taking the limit:

$$A = \lim_{|P|\to 0}\sum_{i=1}^n [f(x_i^_) - g(x_i^_)],\Delta x_i = \int_a^b [f(x) - g(x)],dx$$

By linearity of the integral, this equals $\int_a^b f,dx - \int_a^b g,dx$ — the area under $f$ minus the area under $g$.

```
     y
     │        f(x) ← top curve
     │    ╭───────────╮
     │   ╱  shaded     ╲
     │  ╱   region      ╲
     │ ╱  ╭──────────╮   ╲
     │╱   │  g(x)     │    ╲
     │────────────────────────→ x
          a            b

  Height of each rectangle = f(x*) − g(x*)
  Area = ∫ₐᵇ [f(x) − g(x)] dx
```

### Why the Condition $f \ge g$ Matters

Without it, $f(x) - g(x)$ can be negative, and the integral gives a _signed_ area (positive where $f > g$, negative where $g > f$), not the geometric area. When the curves cross, we must split.

---

## § 2 — When Curves Cross: Handling Sign Changes

If $f$ and $g$ intersect within $(a, b)$, the formula must be adapted.

### Procedure

1. Find all intersection points by solving $f(x) = g(x)$ on $[a, b]$. Call them $c_1 < c_2 < \cdots < c_{k}$.
2. On each subinterval between consecutive intersection points, determine which function is on top.
3. Integrate $|f(x) - g(x)|$ on each subinterval (using whichever sign makes the integrand positive):

$$A = \int_a^{c_1}|f - g|,dx + \int_{c_1}^{c_2}|f - g|,dx + \cdots$$

More precisely:

$$\boxed{A = \int_a^b |f(x) - g(x)|,dx}$$

This is always correct, but the absolute value requires splitting at zeros of $f - g$.

```
     y
     │        ╭──── f(x)
     │    ╭───╯  ╲
     │   ╱  A₁    ╲  ╭── g(x)
     │──╱───────────╲╱─────────→ x
     │              │╲    A₂  ╱
     │              │  ╲    ╱
     │              c₁  ╲──╱  c₂
     
  A = A₁ + A₂ = ∫ₐᶜ¹[f−g]dx + ∫ᶜ¹ᶜ²[g−f]dx
  (switch the order in the second integral because g > f there)
```

## § 3 — Integration with Respect to $y$

Sometimes the natural variable is $y$, not $x$. When curves are expressed as $x = f(y)$ and $x = g(y)$, and the region is bounded horizontally, integrate with respect to $y$.

### Formula

Let $f(y) \ge g(y)$ on $[c, d]$ (right curve minus left curve):

$$\boxed{A = \int_c^d \bigl[f(y) - g(y)\bigr],dy}$$

```
   y
   │
 d─┤    ╭──── x = f(y) (right)
   │   ╱│████████
   │  ╱ │████████
   │ ╱  │████████
 c─┤╱   │
   │    │
   └────┴──────────→ x
     g(y)  f(y)

  Width of each strip = f(y*) − g(y*)
  Area = ∫ᶜᵈ [f(y) − g(y)] dy
```

### When to Integrate in $y$

- When curves are given as $x = h(y)$ directly.
- When integrating in $x$ requires splitting into multiple integrals but $y$ gives a single clean integral.
- When the region is easier to describe by horizontal strips than vertical ones.

---

## § 4 — Finding Limits of Integration

The limits come from the **intersection points** of the bounding curves, or from explicitly stated boundaries.

### Solving for Intersections

Set $f(x) = g(x)$ and solve. Always:

1. Find all solutions in the relevant domain.
2. Verify which function is on top between each pair of consecutive solutions (plug in a test point).
3. Sketch before integrating — even a rough sketch prevents sign errors.

> **Personal note.** The single most common error in area problems is not the integration itself — it is setting up the wrong limits or forgetting to check which curve is on top. Sketch first, always.

## § 5 — Worked Examples

### Example 1 — Area Between a Parabola and a Line

Find the area between $f(x) = x^2$ and $g(x) = x + 2$.

**Step 1: Find intersections.** $$x^2 = x + 2 \implies x^2 - x - 2 = 0 \implies (x-2)(x+1) = 0$$ So $x = -1$ and $x = 2$.

**Step 2: Determine which is on top.** Test $x = 0$: $f(0) = 0$, $g(0) = 2$. So $g \ge f$ on $[-1, 2]$.

**Step 3: Integrate.** $$A = \int_{-1}^{2}(x + 2 - x^2),dx = \left[\frac{x^2}{2} + 2x - \frac{x^3}{3}\right]_{-1}^{2}$$

$$= \left(2 + 4 - \frac{8}{3}\right) - \left(\frac{1}{2} - 2 + \frac{1}{3}\right) = \frac{10}{3} - \left(-\frac{7}{6}\right) = \frac{10}{3} + \frac{7}{6} = \frac{27}{6} = \frac{9}{2}$$

```
     y
     │        ╭──── g(x) = x+2
   4─┤       ╱██
     │      ╱████╲
   2─┤     ╱██████╲──── f(x) = x²
     │    ╱████████╲  ╱
     │───╱──────────╲╱────→ x
        -1           2

  Shaded area = 9/2
```

---

### Example 2 — Curves That Cross

Find the area enclosed by $f(x) = \sin x$ and $g(x) = \cos x$ on $[0, \pi]$.

**Step 1: Find intersections on $[0, \pi]$.** $\sin x = \cos x \implies \tan x = 1 \implies x = \pi/4$.

**Step 2: Determine which is on top on each piece.**

- On $[0, \pi/4]$: test $x = 0$: $\sin 0 = 0 < \cos 0 = 1$, so $\cos \ge \sin$.
- On $[\pi/4, \pi]$: test $x = \pi/2$: $\sin(\pi/2) = 1 > \cos(\pi/2) = 0$, so $\sin \ge \cos$.

**Step 3: Integrate.** $$A = \int_0^{\pi/4}(\cos x - \sin x),dx + \int_{\pi/4}^{\pi}(\sin x - \cos x),dx$$

First integral: $$[\sin x + \cos x]_0^{\pi/4} = \left(\frac{\sqrt{2}}{2}+\frac{\sqrt{2}}{2}\right) - (0+1) = \sqrt{2} - 1$$

Second integral: $$[-\cos x - \sin x]_{\pi/4}^{\pi} = (-(-1) - 0) - \left(-\frac{\sqrt{2}}{2}-\frac{\sqrt{2}}{2}\right) = 1 + \sqrt{2}$$

$$A = (\sqrt{2}-1) + (1+\sqrt{2}) = 2\sqrt{2}$$

---

### Example 3 — Integrating in $y$

Find the area of the region bounded by $x = y^2$ and $x = y + 2$.

**In $x$**: the parabola opens rightward — solving for the two $x$-branches is messy. In $y$, this is clean.

**Step 1: Intersections.** $y^2 = y + 2 \implies y^2 - y - 2 = 0 \implies (y-2)(y+1) = 0$, so $y = -1, 2$.

**Step 2: Which is on the right?** Test $y = 0$: $x = y^2 = 0$ vs $x = y+2 = 2$. So $x = y+2$ is the right curve.

**Step 3: Integrate.** $$A = \int_{-1}^{2}\bigl[(y+2) - y^2\bigr],dy = \left[\frac{y^2}{2} + 2y - \frac{y^3}{3}\right]_{-1}^{2}$$

$$= \left(2 + 4 - \frac{8}{3}\right) - \left(\frac{1}{2} - 2 + \frac{1}{3}\right) = \frac{9}{2}$$

The same numerical answer as Example 1 — because the region is the same, just described with a different variable.

---

### Example 4 — Three Bounding Curves

Find the area of the region bounded by $y = x^2$, $y = 2x$, and $y = 0$.

**Step 1: Find all pairwise intersections.**

- $x^2 = 2x \implies x = 0, 2$
- $x^2 = 0 \implies x = 0$
- $2x = 0 \implies x = 0$

The region has vertices at $(0,0)$ and $(2,4)$.

**Step 2: Determine the geometry.** On $[0, 2]$: $2x \ge x^2 \ge 0$. The region is between $y = 2x$ (top) and $y = x^2$ (bottom):

$$A = \int_0^2 (2x - x^2),dx = \left[x^2 - \frac{x^3}{3}\right]_0^2 = 4 - \frac{8}{3} = \frac{4}{3}$$

---

### Example 5 — Area Requiring Both $x$ and $y$ Integration

Find the area between $y = \sqrt{x}$ and $y = x^2$ on $[0, 1]$.

**Intersections:** $\sqrt{x} = x^2 \implies x = 0, 1$.

On $(0,1)$: test $x = 1/4$: $\sqrt{1/4} = 1/2$, $(1/4)^2 = 1/16$. So $\sqrt{x} \ge x^2$.

$$A = \int_0^1\left(\sqrt{x} - x^2\right)dx = \left[\frac{2}{3}x^{3/2} - \frac{x^3}{3}\right]_0^1 = \frac{2}{3} - \frac{1}{3} = \frac{1}{3}$$

## § 7 — Connection to MIT 18.02

In 18.02, the area between curves becomes the foundation for **double integrals** over general regions.

The area of a region $D$ in the plane is:

$$A = \iint_D dA$$

When $D$ is described as ${(x,y) : a \le x \le b,\ g(x) \le y \le f(x)}$:

$$A = \int_a^b \int_{g(x)}^{f(x)} dy,dx = \int_a^b [f(x) - g(x)],dx$$

The inner integral $\int_{g(x)}^{f(x)}dy = f(x) - g(x)$ recovers exactly the area-between-curves formula. So area between curves is the 1D cross-section of a 2D double integral — **the limits of the inner integral are the bounding curves**.

This connection is crucial: in 18.02, setting up double integrals over non-rectangular regions requires exactly the same skill as finding limits for area-between-curves problems.

### Type I and Type II Regions (18.02 Language)

|Type|Description|Integral form|
|---|---|---|
|**Type I**|$a \le x \le b$, $g(x) \le y \le f(x)$|$\int_a^b\int_{g(x)}^{f(x)} h,dy,dx$|
|**Type II**|$c \le y \le d$, $g(y) \le x \le f(y)$|$\int_c^d\int_{g(y)}^{f(y)} h,dx,dy$|

The choice between Type I and Type II is exactly the choice between integrating in $x$ or $y$ in the area problem.

## § 8 — Connection to AI / Machine Learning / Deep Learning

### 8.1 — ROC Curves and AUC

The **Area Under the ROC Curve (AUC)** is the single most used model evaluation metric in binary classification. The ROC curve plots True Positive Rate (TPR) against False Positive Rate (FPR) as the classification threshold varies.

Mathematically, AUC is the area under a parametric curve:

$$\text{AUC} = \int_0^1 \text{TPR}(\text{FPR}),d(\text{FPR})$$

When comparing two classifiers A and B, the **difference in performance** is:

$$\Delta\text{AUC} = \int_0^1 [\text{TPR}_A(t) - \text{TPR}_B(t)],dt$$

This is exactly the area between two curves. A classifier dominates another if this integral is positive — if its ROC curve lies above the other's everywhere. The area between their ROC curves quantifies _by how much_.

```
TPR
 1│         ╭──── Classifier A
  │     ╭───╯████
  │    ╱████████╲
  │   ╱██████████╲─── Classifier B
  │  ╱████████████╲
  │ ╱██████████████╲
  │╱────────────────╲──→ FPR
 0                   1

  Shaded area = ΔAUC = advantage of A over B
```

### 8.2 — Precision-Recall AUC

The **Precision-Recall curve** similarly defines an area metric:

$$\text{AP} = \int_0^1 P(R),dR$$

where $P$ is precision and $R$ is recall. The area between two precision-recall curves compares models — a direct area-between-curves computation over $[0,1]$.

### 8.3 — Wasserstein Distance as Area Between CDFs

For one-dimensional distributions, the **Wasserstein-1 distance** (Earth Mover's Distance) equals the area between cumulative distribution functions:

$$W_1(p, q) = \int_{-\infty}^{\infty}|F_p(x) - F_q(x)|,dx$$

This is the area between two curves $F_p$ and $F_q$. The absolute value means it splits at crossings — exactly the procedure in § 2. This metric is used in **Wasserstein GANs** and **optimal transport** based losses.

```
CDF
 1│              ╭──── F_q(x)
  │         ╭───╯
  │     ╭───╯ ████████
  │ ╭───╯████████╲
  │ │████████████ ╲─── F_p(x)
  │─┼─────────────────────→ x
  
  W₁ = shaded area between the two CDFs
```

### 8.4 — Learning Curves and Sample Efficiency

When plotting validation loss against training steps for two models, the **area between the learning curves** measures cumulative sample efficiency:

$$\Delta = \int_0^T [\mathcal{L}_A(t) - \mathcal{L}_B(t)],dt$$

If model A has lower loss throughout training, $\Delta > 0$ and A is more sample efficient. This integral accounts for the full trajectory, not just the endpoint — a model that trains faster but converges to the same loss may have a substantially smaller $\Delta$.

### 8.5 — Gain Curves in Ranking Models

In information retrieval and recommendation systems, **Cumulative Gain** curves measure ranking quality:

$$\text{DCG}@k = \sum_{i=1}^k \frac{\text{rel}_i}{\log_2(i+1)}$$

The continuous analog is an area under a gain curve. The area _between_ the gain curve of a learned ranker and the random baseline quantifies the model's lift — how much better than chance the ranking is. This is the area-between-curves formula applied to ranked retrieval evaluation.

### 8.6 — Calibration and the Area Between Reliability Curves

A well-calibrated model has predicted probabilities that match empirical frequencies. The **reliability diagram** plots empirical accuracy vs. predicted confidence. The area between this curve and the diagonal $y = x$ (perfect calibration) is the **Expected Calibration Error (ECE)**:

$$\text{ECE} \approx \int_0^1 |f(p) - p|,dp$$

where $f(p)$ is the empirical accuracy at predicted confidence $p$. This is the area between the calibration curve and the line $y = x$ — the splitting at crossings (§ 2) handles over- and under-confidence regions separately.

## § 9 — Common Mistakes

**Mistake 1: Not finding all intersection points.** Solve $f(x) = g(x)$ completely. Quadratics have two roots, trig equations have infinitely many — restrict to the domain and find all of them.

**Mistake 2: Wrong sign (not checking which curve is on top).** $\int_a^b (g - f)$ instead of $\int_a^b (f - g)$ gives the right magnitude but wrong interpretation. Always plug in a test point between the limits.

**Mistake 3: Not splitting when curves cross.** $\int_{-1}^{1}(x^3 - x),dx = 0$ suggests the area is zero — but the curves cross at $x = 0$, and the enclosed regions have equal area on each side. The actual area is $\int_{-1}^{0}(x^3 - x),dx + \int_0^1(x - x^3),dx = 1/2$.

**Mistake 4: Integrating in $x$ when $y$ is far cleaner.** When a region is bounded by $x = y^2 + 1$ and $x = 3 - y^2$, integrating in $x$ requires inverting and splitting. In $y$, it is a single integral. Recognize horizontal strip regions.

**Mistake 5: Using the net signed area formula for a geometric area problem.** $\int_a^b [f(x) - g(x)],dx$ gives signed area. For the geometric area (always positive), use $\int_a^b |f(x) - g(x)|,dx$, splitting at every crossing.

## § 10 — Summary

$$A = \int_a^b [f(x) - g(x)],dx \quad (f \ge g \text{ on } [a,b])$$

$$A = \int_a^b |f(x) - g(x)|,dx \quad (\text{general, split at crossings})$$

$$A = \int_c^d [f(y) - g(y)],dy \quad (\text{integrate in } y: \text{ right} - \text{left})$$

|Concept|Content|
|---|---|
|**Formula**|Top minus bottom (or right minus left in $y$), integrated over the interval|
|**Limits**|Intersection points of the bounding curves|
|**Crossing curves**|Split at each crossing; integrate $\|f-g\|$ on each piece|
|**Choice of variable**|Use whichever gives a single integral over a simply-described domain|
|**18.02 connection**|Area between curves = inner integral of a double integral over a Type I/II region|
|**ML applications**|AUC/ROC comparison, Wasserstein distance, learning curve efficiency, ECE calibration|
