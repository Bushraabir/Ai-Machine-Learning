## 0. Motivation — From Finite to Continuous Averages

You already know how to average $n$ numbers:

$$\bar{y} = \frac{y_1 + y_2 + \cdots + y_n}{n} = \frac{1}{n}\sum_{i=1}^n y_i$$

Now suppose $f$ is a continuous function on $[a, b]$ and you want the "average value" of $f$ over that interval. The function takes **uncountably many** values — you cannot simply sum them and divide by $n$.

**The fix: discretize, then take a limit.**

Partition $[a,b]$ into $n$ equal subintervals of width $\Delta x = \frac{b-a}{n}$. Sample $f$ at points $x_1^_, x_2^_, \ldots, x_n^*$:

$$\bar{f}_n = \frac{1}{n}\sum_{i=1}^n f(x_i^*)$$

Rewrite, multiplying and dividing by $(b-a)$:

$$\bar{f}_n = \frac{1}{b-a}\sum_{i=1}^n f(x_i^*),\Delta x$$

As $n \to \infty$, the sum $\sum f(x_i^*),\Delta x$ converges to $\int_a^b f(x),dx$. So the natural definition is:

$$\boxed{f_{\text{avg}} = \frac{1}{b-a}\int_a^b f(x),dx}$$

---

## 1. The Definition

> **Definition.** Let $f$ be integrable on $[a,b]$. The **average value** (or **mean value**) of $f$ on $[a,b]$ is: $$f_{\text{avg}} = \frac{1}{b-a}\int_a^b f(x),dx$$

**Geometric interpretation.** $f_{\text{avg}}$ is the height of the rectangle with base $[a,b]$ whose area equals $\int_a^b f(x),dx$.

```

|       ╭─╯   ╰─╮

|╱                      ╲ │ └─────────────────┘ │
─┼──────────────────────── a ─────────────────── b ─ x
```

The rectangle (dashed) has exactly the same area as the region under $f$.

## 2. The Mean Value Theorem for Integrals

The average value theorem has a companion: the function must _attain_ its average somewhere.

> **Theorem (MVT for Integrals).** If $f$ is continuous on $[a,b]$, then there exists at least one $c \in [a,b]$ such that: $$f(c) = f_{\text{avg}} = \frac{1}{b-a}\int_a^b f(x),dx$$

Equivalently: $$\int_a^b f(x),dx = f(c)(b-a)$$

**Proof.** Since $f$ is continuous on $[a,b]$, by the **Extreme Value Theorem** it attains its minimum $m = f(x_{\min})$ and maximum $M = f(x_{\max})$. Then for all $x \in [a,b]$:

$$m \leq f(x) \leq M$$

Integrating: $$m(b-a) \leq \int_a^b f(x),dx \leq M(b-a)$$

Dividing by $(b-a) > 0$: $$m \leq \frac{1}{b-a}\int_a^b f(x),dx \leq M$$

So $f_{\text{avg}}$ lies between $m$ and $M$. By the **Intermediate Value Theorem**, since $f$ is continuous and achieves both $m$ and $M$, it must achieve every value in between. In particular, there exists $c \in [a,b]$ with $f(c) = f_{\text{avg}}$. $\square$

> **Remark.** This $c$ need not be unique. The theorem guarantees existence, not uniqueness.

---

## 3. Relationship to the Fundamental Theorem of Calculus

Let $F(x) = \int_a^x f(t),dt$. Then $F'(x) = f(x)$ (FTC Part 1). The MVT for derivatives applied to $F$ on $[a,b]$ says:

$$\frac{F(b) - F(a)}{b - a} = F'(c) = f(c) \quad \text{for some } c \in (a,b)$$

But $F(b) - F(a) = \int_a^b f(x),dx$, so:

$$f(c) = \frac{1}{b-a}\int_a^b f(x),dx = f_{\text{avg}}$$

**The MVT for integrals is the MVT for derivatives applied to the antiderivative.** They are the same theorem wearing different clothes.

## 4. Worked Examples

### Example 1 — Polynomial

**Problem.** Find the average value of $f(x) = x^2$ on $[1, 4]$.

$$f_{\text{avg}} = \frac{1}{4-1}\int_1^4 x^2,dx = \frac{1}{3}\left[\frac{x^3}{3}\right]_1^4 = \frac{1}{3}\cdot\frac{64-1}{3} = \frac{63}{9} = 7$$

**Find $c$.** We need $f(c) = c^2 = 7$, so $c = \sqrt{7} \approx 2.646 \in [1,4]$. ✓

---

### Example 2 — Trigonometric

**Problem.** Find the average value of $f(x) = \sin x$ on $[0, \pi]$.

$$f_{\text{avg}} = \frac{1}{\pi - 0}\int_0^\pi \sin x,dx = \frac{1}{\pi}\left[-\cos x\right]_0^\pi = \frac{1}{\pi}(-\cos\pi + \cos 0) = \frac{1}{\pi}(1+1) = \frac{2}{\pi}$$

> $f_{\text{avg}} = \frac{2}{\pi} \approx 0.637$. The average height of one arch of a sine wave is about 63.7% of its amplitude. This is a number worth remembering.

**Find $c$.** We need $\sin c = \frac{2}{\pi}$, so $c = \arcsin!\left(\frac{2}{\pi}\right) \approx 0.690$ and $c = \pi - \arcsin!\left(\frac{2}{\pi}\right) \approx 2.452$. Two solutions, both in $[0,\pi]$. ✓

---

### Example 3 — Physical Interpretation (Temperature)

**Problem.** The temperature (in °C) in a city over a 12-hour period is $T(t) = 20 + 8\sin!\left(\frac{\pi t}{12}\right)$ for $t \in [0, 12]$. Find the average temperature.

$$T_{\text{avg}} = \frac{1}{12}\int_0^{12}\left(20 + 8\sin\frac{\pi t}{12}\right)dt$$

$$= \frac{1}{12}\left[20t - \frac{96}{\pi}\cos\frac{\pi t}{12}\right]_0^{12}$$

$$= \frac{1}{12}\left[\left(240 - \frac{96}{\pi}\cos\pi\right) - \left(0 - \frac{96}{\pi}\cos 0\right)\right]$$

$$= \frac{1}{12}\left[240 + \frac{96}{\pi} + \frac{96}{\pi}\right] = 20 + \frac{16}{\pi} \approx 25.09 ,^\circ\text{C}$$

> The constant term $20$ contributes its full value to the average (as any constant would). The sinusoidal oscillation contributes $\frac{16}{\pi}$ — a result of integrating $\sin$ over a half-period.

---

### Example 4 — Average Value of a Velocity Function

**Problem.** A particle moves with velocity $v(t) = 3t^2 - 4t + 1$ on $[0, 3]$. Find the average velocity.

$$v_{\text{avg}} = \frac{1}{3}\int_0^3 (3t^2 - 4t + 1),dt = \frac{1}{3}\left[t^3 - 2t^2 + t\right]_0^3$$

$$= \frac{1}{3}(27 - 18 + 3) = \frac{12}{3} = 4$$

> **Note.** Average velocity $= \frac{\text{displacement}}{\text{time}} = \frac{\int_0^3 v(t),dt}{3}$. This is consistent: the integral of velocity is displacement, and dividing by time gives average velocity. The formula for $f_{\text{avg}}$ is exactly the physics formula.

## 5. Average Value in Higher Dimensions

The definition extends naturally. For a function $f(x,y)$ over a region $D$ in $\mathbb{R}^2$:

$$f_{\text{avg}} = \frac{1}{\text{Area}(D)}\iint_D f(x,y),dA$$

For $f(x,y,z)$ over a solid region $E$ in $\mathbb{R}^3$:

$$f_{\text{avg}} = \frac{1}{\text{Vol}(E)}\iiint_E f(x,y,z),dV$$

**Pattern:** average = $\frac{\text{integral of } f}{\text{measure of domain}}$. This is the continuous analogue of $\bar{y} = \frac{\sum y_i}{n}$.

---

## 6. Average Value vs. Average Rate of Change

These are different things. Do not confuse them.

|Concept|Formula|What it measures|
|---|---|---|
|Average value of $f$ on $[a,b]$|$\frac{1}{b-a}\int_a^b f(x),dx$|Typical output of $f$|
|Average rate of change of $f$ on $[a,b]$|$\frac{f(b)-f(a)}{b-a}$|Slope of secant line|
|Average value of $f'$ on $[a,b]$|$\frac{1}{b-a}\int_a^b f'(x),dx = \frac{f(b)-f(a)}{b-a}$|These two are equal!|

> **The average value of $f'$ equals the average rate of change of $f$.** This is just FTC: $$\frac{1}{b-a}\int_a^b f'(x),dx = \frac{f(b)-f(a)}{b-a}$$

## 7. Connection to AI, Machine Learning, and Deep Learning

### 7.1 Expected Value IS Average Value

The expected value of a continuous random variable $X$ with pdf $p(x)$ on $[a,b]$:

$$\mathbb{E}[X] = \int_a^b x, p(x),dx$$

When $p(x) = \frac{1}{b-a}$ (uniform distribution), this becomes:

$$\mathbb{E}[X] = \frac{1}{b-a}\int_a^b x,dx = f_{\text{avg}} \text{ for } f(x) = x$$

More generally, $\mathbb{E}[g(X)] = \int g(x),p(x),dx$ is a **weighted average value** — the same formula with a non-uniform weight $p(x)$. Every expectation in ML is an average value integral.

### 7.2 Loss Functions — Empirical vs. True Average

The **true risk** (population loss) of a model $h$ with loss $\ell$ under data distribution $p(x,y)$:

$$R(h) = \mathbb{E}_{(x,y)\sim p}[\ell(h(x), y)] = \iint \ell(h(x), y),p(x,y),dx,dy$$

This is the average value of the loss function over the data distribution. We cannot compute it — we don't know $p$. So we approximate with the **empirical risk**:

$$\hat{R}(h) = \frac{1}{n}\sum_{i=1}^n \ell(h(x_i), y_i)$$

This is the Riemann sum approximation to the average value integral — exactly the discretization from §0. The gap $R(h) - \hat{R}(h)$ is what **generalization theory** (PAC learning, VC dimension, Rademacher complexity) bounds.

### 7.3 Batch Normalization — Normalizing by Empirical Average

**Batch Normalization** (Ioffe & Szegedy, 2015) normalizes activations in a neural network by:

$$\hat{x}_i = \frac{x_i - \mu_\mathcal{B}}{\sqrt{\sigma_\mathcal{B}^2 + \epsilon}}$$

where $\mu_\mathcal{B} = \frac{1}{m}\sum_{i=1}^m x_i$ is the batch mean — the empirical average value of the activation over the mini-batch. At test time, running averages (approximate integrals) of mean and variance replace batch statistics. The entire operation is the average value formula applied to neural activations.

### 7.4 The Law of Large Numbers — Average Value as a Limit

The **Strong Law of Large Numbers** says: for i.i.d. $X_1, X_2, \ldots$ with $\mathbb{E}[X_i] = \mu$:

$$\frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{\text{a.s.}} \mu = \mathbb{E}[X] = \int x,p(x),dx$$

The sample mean converges to the **average value integral**. This justifies every Monte Carlo method, every stochastic gradient estimate, every empirical loss minimization in deep learning. The LLN is the convergence theorem that says our Riemann-sum approximation to $f_{\text{avg}}$ actually works.

### 7.5 Time-Averaging in Reinforcement Learning

In **reinforcement learning**, the long-run average reward of a policy $\pi$ in a Markov chain is:

$$\bar{r}^\pi = \lim_{T\to\infty} \frac{1}{T}\int_0^T r(s(t), a(t)),dt$$

This is precisely the average value $f_{\text{avg}}$ of the reward function over time — the continuous-time version of the definition. Policy gradient methods differentiate through this average value to update the policy. The MVT for integrals guarantees that the policy must attain this average reward at some time $c$ — useful for theoretical arguments about policy improvement.

### 7.6 Fourier Analysis and Average Value

For a periodic function $f$ with period $2\pi$, the Fourier series expansion is:

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty (a_n \cos nx + b_n \sin nx)$$

The constant term $\frac{a_0}{2}$ is:

$$\frac{a_0}{2} = \frac{1}{2\pi}\int_{-\pi}^\pi f(x),dx = f_{\text{avg}}$$

**The DC component of any Fourier series is the average value of the function.** In signal processing for ML (audio models, time-series, CNNs processing spectrograms), removing the mean ($f_{\text{avg}}$) before analysis is called "DC removal" or "mean subtraction" — a ubiquitous preprocessing step.

## 8. A Rigorous Note — Integrability

The formula $f_{\text{avg}} = \frac{1}{b-a}\int_a^b f(x),dx$ requires $f$ to be **integrable** on $[a,b]$. Sufficient conditions:

- $f$ continuous on $[a,b]$ ✓ (most common case)
- $f$ bounded with finitely many discontinuities ✓
- $f$ monotone on $[a,b]$ ✓

The MVT for Integrals additionally requires **continuity** to invoke the IVT. A merely integrable $f$ has an average value but may not attain it.
