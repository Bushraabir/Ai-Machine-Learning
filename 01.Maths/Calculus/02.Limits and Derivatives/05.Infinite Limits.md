## 0. Motivation: When Functions Blow Up

Consider $$f(x) = \frac{1}{x^2}.$$

As $x \to 0$, the denominator collapses to zero while the numerator stays fixed. The output does not approach any finite number ‚Äî it grows without bound. But it grows _predictably_, _consistently_, from _both_ sides. This is not chaos. It is a precise, describable behavior, and we need language for it.

Infinite limits answer: **what does it mean, rigorously, for a function to "go to infinity"?**

The answer is not "the limit equals $\infty$" as if $\infty$ were a number on the real line. It is a carefully crafted statement about the behavior of $f(x)$ for $x$ near some point $a$.

## 2. Formal Definitions

### 2.1 Limit Equals $+\infty$

> **Definition (Stewart ¬ß2.2; Rudin Ch. 4).** We say $$\lim_{x \to a} f(x) = +\infty$$ if for every $M > 0$, there exists $\delta > 0$ such that $$0 < |x - a| < \delta \implies f(x) > M.$$

**Reading it carefully:**

- The role of $\varepsilon$ in the standard limit is now played by $M$ ‚Äî but $M$ measures largeness, not closeness.
- There is no "$|f(x) - L| < \varepsilon$" because there is no finite $L$ to be close to.
- For _every_ finite ceiling $M$, we can find a neighborhood of $a$ where $f$ exceeds it.
- This is not "the limit is undefined." It is a _specific, precise_ type of behavior.

### 2.2 Limit Equals $-\infty$

> **Definition.** We say $$\lim_{x \to a} f(x) = -\infty$$ if for every $N < 0$ (equivalently, every $M > 0$), there exists $\delta > 0$ such that $$0 < |x - a| < \delta \implies f(x) < N.$$

The function falls below every finite floor.

### 2.3 One-Sided Infinite Limits

We naturally combine the one-sided approach with the infinite behavior:

$$\lim_{x \to a^+} f(x) = +\infty \quad \text{means:} \quad \forall M > 0,\ \exists \delta > 0:\ 0 < x - a < \delta \implies f(x) > M.$$

$$\lim_{x \to a^-} f(x) = -\infty \quad \text{means:} \quad \forall N < 0,\ \exists \delta > 0:\ 0 < a - x < \delta \implies f(x) < N.$$

All four combinations $(\pm\infty,\ a^\pm)$ are possible. Each is its own precise definition.

## 3. Vertical Asymptotes ‚Äî The Geometric Meaning

> **Definition.** The line $x = a$ is a **vertical asymptote** of $f$ if at least one of the following holds: $$\lim_{x \to a^+} f(x) = \pm\infty \quad \text{or} \quad \lim_{x \to a^-} f(x) = \pm\infty.$$

**Key point:** a vertical asymptote is a _consequence_ of an infinite limit ‚Äî not a definition in itself. The limit statement comes first; the asymptote language is shorthand.

## 4. The Extended Real Line $\overline{\mathbb{R}}$

In real analysis, one formally adjoins two symbols to $\mathbb{R}$: $$\overline{\mathbb{R}} = \mathbb{R} \cup {-\infty, +\infty}.$$

Arithmetic rules on $\overline{\mathbb{R}}$ (partial):

|Expression|Value|
|---|---|
|$\infty + \infty$|$+\infty$|
|$-\infty - \infty$|$-\infty$|
|$c \cdot \infty$, $c > 0$|$+\infty$|
|$c \cdot \infty$, $c < 0$|$-\infty$|
|$\infty + (-\infty)$|**undefined** (indeterminate)|
|$0 \cdot \infty$|**undefined** (indeterminate)|
|$\frac{\infty}{\infty}$|**undefined** (indeterminate)|

> **Warning.** The indeterminate forms are not undefined because we are ignorant ‚Äî they are undefined because different functions can approach $0 \cdot \infty$ with any answer. L'H√¥pital's Rule is the tool for resolving them.

## 5. Worked Examples

### Example 5.1 ‚Äî The Prototype: $1/x^2$

$$\lim_{x \to 0} \frac{1}{x^2} = +\infty.$$

**$\varepsilon$-$M$-$\delta$ proof.** Let $M > 0$. We need $\delta > 0$ such that $0 < |x| < \delta \implies \frac{1}{x^2} > M$.

We want $\frac{1}{x^2} > M$, i.e., $x^2 < \frac{1}{M}$, i.e., $|x| < \frac{1}{\sqrt{M}}$. Choose $\delta = \frac{1}{\sqrt{M}}$.

Then $0 < |x| < \delta \implies x^2 < \frac{1}{M} \implies \frac{1}{x^2} > M$. $\blacksquare$

```
    f(x) = 1/x¬≤
      |
      |   |   |
      |   |   |
      |   | | |
      |  || | ||

      +‚Äî‚Äî‚Äî‚Äî‚Äî0‚Äî‚Äî‚Äî‚Äî‚Äî‚Äî‚Äî‚Äî> x

   Symmetric blow-up from both sides ‚Üí limit = +‚àû
```

### Example 5.2 ‚Äî Asymmetric Blow-Up: $1/x$

$$\lim_{x \to 0^+} \frac{1}{x} = +\infty, \qquad \lim_{x \to 0^-} \frac{1}{x} = -\infty.$$

The two one-sided limits are $+\infty$ and $-\infty$ respectively. The two-sided limit **does not exist** ‚Äî not even as an infinite limit ‚Äî because the function diverges in _opposite_ directions.


### Example 5.3 ‚Äî Logarithm at Zero

$$\lim_{x \to 0^+} \ln x = -\infty.$$

**Proof.** Let $N < 0$. We need $\delta > 0$ such that $0 < x < \delta \implies \ln x < N$.

$\ln x < N \iff x < e^N$. Since $N < 0$, we have $e^N \in (0, 1)$. Choose $\delta = e^N > 0$. Then $0 < x < \delta = e^N \implies \ln x < N$. $\blacksquare$

> _This is why log-loss in machine learning requires the predicted probability $\hat{p}$ to be clipped away from zero. If $\hat{p} \to 0^+$, then $-\ln \hat{p} \to +\infty$ and the loss explodes._

---

### Example 5.4 ‚Äî Rational Functions

$$\lim_{x \to 2} \frac{x+1}{(x-2)^2}.$$

Near $x = 2$: numerator $\to 3 > 0$, denominator $(x-2)^2 \to 0^+$. Hence the ratio $\to +\infty$.

$$\lim_{x \to 2} \frac{x+1}{(x-2)^2} = +\infty.$$

Now consider: $$\lim_{x \to 3^+} \frac{x-5}{x-3}.$$

Near $x = 3^+$: numerator $\to -2 < 0$, denominator $\to 0^+$. Hence ratio $\to -\infty$.

$$\lim_{x \to 3^+} \frac{x-5}{x-3} = -\infty.$$

**The sign rule for rational blow-up:**

> If $\lim_{x \to a} p(x) = L \neq 0$ and $\lim_{x \to a} q(x) = 0$, then $\lim_{x \to a} \frac{p(x)}{q(x)} = \pm\infty$. The sign is determined by the signs of $L$ and $q(x)$ near $a$.

|Sign of $L$|Sign of $q(x)$ near $a$|Limit|
|---|---|---|
|$+$|$+$|$+\infty$|
|$+$|$-$|$-\infty$|
|$-$|$+$|$-\infty$|
|$-$|$-$|$+\infty$|

---

### Example 5.5 ‚Äî Oscillation vs. Blow-Up

$$f(x) = \frac{1}{x} \sin!\left(\frac{1}{x}\right), \quad x \neq 0.$$

As $x \to 0$, $|f(x)| \leq \frac{1}{|x|} \to \infty$ in magnitude, but the function oscillates. Neither one-sided limit is $+\infty$ or $-\infty$ in a consistent way ‚Äî the limit does not exist in any sense. **Blow-up requires monotone escape, not mere unboundedness in fluctuation.**

## 6. Indeterminate Forms and L'H√¥pital's Rule

When a limit naively produces an indeterminate form, further analysis is required.

**The seven classical indeterminate forms:**

$$\frac{0}{0}, \quad \frac{\infty}{\infty}, \quad 0 \cdot \infty, \quad \infty - \infty, \quad 0^0, \quad \infty^0, \quad 1^\infty.$$

> **L'H√¥pital's Theorem (Stewart ¬ß4.4).** Suppose $\lim_{x \to a} f(x) = 0$ and $\lim_{x \to a} g(x) = 0$ (or both $\to \pm\infty$), and $g'(x) \neq 0$ near $a$. Then $$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)},$$ provided the right-hand limit exists (possibly $\pm\infty$).

### Key Examples

**$\frac{\infty}{\infty}$ form:** $$\lim_{x \to \infty} \frac{\ln x}{x} = \lim_{x \to \infty} \frac{1/x}{1} = 0.$$

> Logarithm grows slower than any power. Used in complexity analysis.

**$0 \cdot \infty$ form** ‚Äî rewrite as $\frac{0}{0}$ or $\frac{\infty}{\infty}$: $$\lim_{x \to 0^+} x \ln x = \lim_{x \to 0^+} \frac{\ln x}{1/x} = \lim_{x \to 0^+} \frac{1/x}{-1/x^2} = \lim_{x \to 0^+} (-x) = 0.$$

> This is the reason the entropy term $p \ln p \to 0$ as $p \to 0^+$ ‚Äî critical in information theory and cross-entropy loss.

**$\infty - \infty$ form** ‚Äî algebraically combine: $$\lim_{x \to 0} \left(\frac{1}{\sin x} - \frac{1}{x}\right) = \lim_{x \to 0} \frac{x - \sin x}{x \sin x} \overset{\text{L'H}}{=} \lim_{x \to 0} \frac{1 - \cos x}{\sin x + x \cos x} \overset{\text{L'H}}{=} \lim_{x \to 0} \frac{\sin x}{2\cos x - x \sin x} = 0.$$

## 7. Limit Laws Extended to $\pm\infty$

The standard limit laws extend to infinite limits with care. Let $\lim_{x \to a} f(x) = +\infty$ and $\lim_{x \to a} g(x) = L \in \mathbb{R}$.

|Expression|Limit|Condition|
|---|---|---|
|$f(x) + g(x)$|$+\infty$|‚Äî|
|$f(x) \cdot g(x)$|$+\infty$|$L > 0$|
|$f(x) \cdot g(x)$|$-\infty$|$L < 0$|
|$g(x) / f(x)$|$0$|‚Äî|
|$f(x) / g(x)$|$+\infty$|$L > 0$|
|$f(x) + h(x)$, $\lim h = +\infty$|$+\infty$|‚Äî|

The dangerous forms $\infty - \infty$, $0 \cdot \infty$, $\infty/\infty$ are not covered ‚Äî these are the indeterminate cases.

## 8. Rates of Growth ‚Äî A Hierarchy

Infinite limits are about _how fast_ functions blow up. This hierarchy is fundamental:

$$\ln x \ll x^p \ll x^q \ll a^x \ll x! \ll x^x \quad (p < q,\ a > 1, \text{ as } x \to +\infty)$$

More precisely, for $p > 0$, $a > 1$: $$\lim_{x \to \infty} \frac{\ln x}{x^p} = 0, \qquad \lim_{x \to \infty} \frac{x^p}{a^x} = 0, \qquad \lim_{x \to \infty} \frac{a^x}{x!} = 0.$$

```
   Growth rate (schematic, as x ‚Üí ‚àû):

   x^x  ‚Üë fastest
   x!   ‚Üë
   a^x  ‚Üë
   x^q  ‚Üë
   x^p  ‚Üë  (p < q)
   ln x ‚Üë slowest

   Each is "infinitely smaller" than the one above it.
```

> **Why this matters in ML.** Exponential blow-up appears in softmax without numerical tricks. Factorial blow-up appears in combinatorial search. Understanding the growth hierarchy tells you which quantities need to be log-transformed, normalized, or clipped.

## 10. Connection to Continuity and Differentiability

**Infinite limits and continuity:** If $\lim_{x \to a} f(x) = \pm\infty$, then $f$ is **not continuous** at $a$ (continuity requires a finite limit equal to $f(a)$). This is a **type II discontinuity** (infinite discontinuity).

**Classification of discontinuities:**

|Type|Description|Example|
|---|---|---|
|Removable|Limit exists (finite), but $\neq f(a)$|$\frac{\sin x}{x}$ at $0$|
|Jump|Both one-sided limits finite but unequal|$\text{sgn}(x)$ at $0$|
|Infinite|At least one one-sided limit is $\pm\infty$|$\frac{1}{x}$ at $0$|
|Oscillatory|Limit does not exist, neither finite nor infinite|$\sin(1/x)$ at $0$|

**Infinite limits and differentiation:** By the limit definition, if $f'(a) = \pm\infty$, we say $f$ has a **vertical tangent** at $a$. For instance: $$f(x) = x^{1/3}, \qquad f'(x) = \frac{1}{3} x^{-2/3}, \qquad \lim_{x \to 0} f'(x) = +\infty.$$ The tangent line is vertical. $f$ is continuous but not differentiable (in the usual sense) at $0$.

## 11. Connection to AI / Machine Learning / Deep Learning

### 11.1 Numerical Overflow and Underflow

In IEEE 754 floating-point, numbers have finite range. When a computation exceeds `float32` max ($\approx 3.4 \times 10^{38}$), the result is `inf`. This is a _computational_ infinite limit ‚Äî the mathematical function is fine, but the representation fails.

**Softmax instability:** $$\text{softmax}(z)_i = \frac{e^{z_i}}{\sum_j e^{z_j}}.$$ If any $z_i$ is large (say, 1000), $e^{z_i}$ overflows to `inf`. The standard fix: subtract the maximum logit. $$\text{softmax}(z)_i = \frac{e^{z_i - \max_j z_j}}{\sum_j e^{z_j - \max_j z_j}}.$$ This is valid because $\lim_{x \to \infty} \frac{e^x}{e^x} = 1$ ‚Äî the limit of the ratio is preserved. The numerical trick is justified by a limit argument.

### 11.2 Log-Loss and the Entropy Limit

Cross-entropy loss: $$\mathcal{L} = -\sum_i y_i \log \hat{p}_i.$$

If $\hat{p}_i \to 0^+$ and $y_i = 1$: $-\log \hat{p}_i \to +\infty$.

The loss blows up. This is an infinite limit at work. In practice, numerical libraries clip $\hat{p}_i$ to $[\varepsilon, 1-\varepsilon]$ for small $\varepsilon > 0$ to prevent the computation from reaching the vertical asymptote of $-\log x$ at $x = 0$.

The mathematical fact: $\lim_{x \to 0^+} (-\log x) = +\infty$ ‚Äî rigorously proved as in Example 5.3.

### 11.3 Vanishing and Exploding Gradients

In deep networks, gradients are products of Jacobians across layers. Two pathologies arise from infinite limit behavior:

**Exploding gradients:** $|\nabla_\theta \mathcal{L}| \to \infty$ as network depth increases. The product of many matrices with spectral radius $> 1$ grows without bound ‚Äî a discrete-time analogue of $\lim_{n \to \infty} r^n = \infty$ for $r > 1$.

**Vanishing gradients:** $|\nabla_\theta \mathcal{L}| \to 0$ ‚Äî the other extreme. Sigmoid saturates because: $$\lim_{x \to +\infty} \sigma'(x) = \lim_{x \to +\infty} \sigma(x)(1 - \sigma(x)) = 1 \cdot 0 = 0.$$ The derivative approaches zero as input grows. The gradient effectively "dies."

Gradient clipping (for exploding) and residual connections / batch normalization (for vanishing) are engineering responses to these limit behaviors.

### 11.4 Softplus as a Smooth Approximation to ReLU

$$\text{softplus}(x) = \ln(1 + e^x).$$

As $x \to -\infty$: $\text{softplus}(x) \to \ln(1) = 0$. As $x \to +\infty$: $$\text{softplus}(x) = \ln(1 + e^x) \approx \ln(e^x) = x \to +\infty.$$

So softplus approximates ReLU. Its derivative is exactly $\sigma(x)$ (the sigmoid), which is smooth and bounded. The infinite limit behavior at $+\infty$ is _linear_ ‚Äî the same growth rate as ReLU ‚Äî but the transition is smooth.

### 11.5 KL Divergence and Infinite Penalties

The KL divergence: $$D_{KL}(P | Q) = \sum_x P(x) \log \frac{P(x)}{Q(x)}.$$

If $P(x) > 0$ but $Q(x) = 0$ for some $x$: the term $\log \frac{P(x)}{Q(x)} \to +\infty$. Hence $D_{KL}(P | Q) = +\infty$.

This is not a flaw ‚Äî it is a _feature_. It means: if $Q$ assigns zero probability to an event that $P$ says is possible, the distributions are "infinitely different." Variational autoencoders and variational inference must ensure $Q$ has full support to keep $D_{KL}$ finite. This is an infinite limit providing a hard constraint.

### 11.6 Weight Initialization and the Spectral Norm

For a linear layer $y = Wx$, the spectral norm $|W|_2 = \sigma_{\max}(W)$. If weights are poorly initialized, activations can grow without bound through layers: $$\lim_{L \to \infty} |W^L x| = +\infty \quad \text{if } \sigma_{\max}(W) > 1.$$

Xavier/He initialization ensures $\sigma_{\max}(W) \approx 1$, keeping the limit finite. Understanding _why_ initialization matters requires understanding infinite limits.

## 12. Common Pitfalls

> ‚öÝÔ∏è **$\lim f(x) = \infty$ does not mean the limit "does not exist."** It means the limit exists in $\overline{\mathbb{R}}$ and equals $+\infty$. In $\mathbb{R}$, it is true that no finite limit exists ‚Äî but be precise about which sense you mean.

> ‚öÝÔ∏è **$\infty$ is not a number you can plug in.** You cannot write $f(\infty)$ for a function $f : \mathbb{R} \to \mathbb{R}$. The expression $\lim_{x \to \infty}$ is a limit, not a function evaluation.

> ‚öÝÔ∏è **Indeterminate $\neq$ undefined.** $\frac{0}{0}$ is indeterminate: the limit _might_ exist; we just can't determine it from the form alone. $\frac{1}{0}$ (with $1 \neq 0$) is not indeterminate ‚Äî it signals $\pm\infty$ (sign depending on the side of approach).

> ‚öÝÔ∏è **L'H√¥pital applies only to $\frac{0}{0}$ or $\frac{\infty}{\infty}$.** Applying it to $\frac{1}{0}$ or $\frac{3}{\infty}$ (which are not indeterminate) gives wrong answers.

> ‚öÝÔ∏è **Oscillation is not blow-up.** $f(x) = \frac{1}{x}\sin\frac{1}{x}$ is unbounded near $0$ but has no infinite limit. "Unbounded" and "limit equals $\pm\infty$" are different statements.
