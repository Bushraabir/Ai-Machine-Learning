## The Idea: Differentiating the Derivative

If $f'$ is itself differentiable, we can differentiate it. The result is the **second derivative**:

$$f''(x) = \frac{d}{dx}\left[f'(x)\right] = \lim_{h \to 0} \frac{f'(x+h) - f'(x)}{h}$$

And we can keep going. The $n$-th derivative of $f$ is:

$$f^{(n)}(x) = \frac{d}{dx}\left[f^{(n-1)}(x)\right]$$

with the convention $f^{(0)} = f$.

> ­¤ōī **Existence is not free.** $f'$ existing at every point does not automatically mean $f''$ exists. Each step of differentiation is a fresh demand on the function.

## Notation (All Equivalent ŌĆö Know Them All)

For the $n$-th derivative:

|Notation|Read As|Typical Use|
|---|---|---|
|$f^{(n)}(x)$|"$f$ $n$-th"|Pure math, when $n$ is a variable|
|$\dfrac{d^n f}{dx^n}$|"d-$n$-$f$ over d-$x$-$n$"|Leibniz; PDEs, physics|
|$\dfrac{d^n y}{dx^n}$|same|When $y = f(x)$|
|$D^n f$|"D-$n$ of $f$"|Operator notation|
|$f''$, $f'''$|"double prime", "triple prime"|Informal, up to $n=3$|

> ŌÜĀ’ĖÅ **The Leibniz notation $\frac{d^2y}{dx^2}$ is NOT a fraction.** The $2$ in the numerator is an exponent on the operator $d$, and the $2$ in the denominator is an exponent on $x$. This distinction matters when you get to change-of-variables.

## Computing Higher Derivatives: Examples

### Polynomial ŌĆö Terminates

$$f(x) = x^5 - 3x^3 + 2x$$

$$f'(x) = 5x^4 - 9x^2 + 2$$ $$f''(x) = 20x^3 - 18x$$ $$f'''(x) = 60x^2 - 18$$ $$f^{(4)}(x) = 120x$$ $$f^{(5)}(x) = 120$$ $$f^{(6)}(x) = 0 \quad \text{and all further derivatives}$$

**General rule:** $\dfrac{d^n}{dx^n} x^k = \begin{cases} \dfrac{k!}{(k-n)!} x^{k-n} & n \leq k \ 0 & n > k \end{cases}$

### Exponential ŌĆö Never Dies

$$f(x) = e^x \implies f^{(n)}(x) = e^x \quad \text{for all } n \geq 0$$

This is the defining property of $e^x$ ŌĆö it is a **fixed point** of differentiation. More generally:

$$\frac{d^n}{dx^n} e^{ax} = a^n e^{ax}$$

### Trigonometric ŌĆö 4-Periodic

$$f(x) = \sin x$$

$$f'(x) = \cos x, \quad f''(x) = -\sin x, \quad f'''(x) = -\cos x, \quad f^{(4)}(x) = \sin x$$

The cycle repeats with period 4. Compactly:

$$\frac{d^n}{dx^n} \sin x = \sin!\left(x + \frac{n\pi}{2}\right)$$

**Proof:** Check $n = 0, 1, 2, 3$ manually. For general $n$, use induction ŌĆö each differentiation shifts the phase by $\pi/2$.

### Natural Logarithm

$$f(x) = \ln x$$

$$f'(x) = x^{-1}, \quad f''(x) = -x^{-2}, \quad f'''(x) = 2x^{-3}, \quad f^{(4)}(x) = -6x^{-4}$$

**General pattern:**

$$f^{(n)}(x) = \frac{(-1)^{n-1}(n-1)!}{x^n}, \quad n \geq 1$$

_Proof by induction:_ Base case $n=1$: $f'(x) = x^{-1} = \frac{(-1)^0 \cdot 0!}{x^1}$ Ō£ō. Inductive step: differentiate $\frac{(-1)^{n-1}(n-1)!}{x^n} = (-1)^{n-1}(n-1)! \cdot x^{-n}$ to get $(-1)^{n-1}(n-1)! \cdot (-n) x^{-n-1} = \frac{(-1)^n \cdot n!}{x^{n+1}}$. $\blacksquare$

## The Second Derivative: Geometric Meaning

The first derivative measures **slope**. The second derivative measures **how the slope is changing** ŌĆö this is called **concavity**.

### Concavity Defined

- $f'' > 0$ on $(a,b)$: $f$ is **concave up** (holds water, slopes increasing left-to-right)
- $f'' < 0$ on $(a,b)$: $f$ is **concave down** (spills water, slopes decreasing left-to-right)

```
  Concave Up (f'' > 0):       Concave Down (f'' < 0):

         /                         ---___
        /                         /      \
       /                         /        \
      /
   tangent lines lie             tangent lines lie
   BELOW the curve               ABOVE the curve
```

### Inflection Points

A point $c$ is an **inflection point** of $f$ if $f$ is continuous there and $f''$ **changes sign** at $c$.

> ŌÜĀ’ĖÅ **$f''(c) = 0$ is necessary but not sufficient.** Consider $f(x) = x^4$: $f''(0) = 0$ but $f'' \geq 0$ everywhere ŌĆö no sign change, no inflection point.

**Example:** $f(x) = x^3 - 3x$.

$f''(x) = 6x$. Sign changes at $x = 0$: negative for $x < 0$, positive for $x > 0$. So $(0, 0)$ is an inflection point.

## The Second Derivative Test for Local Extrema

**Theorem.** Suppose $f'(c) = 0$ (a critical point).

- If $f''(c) > 0$: local **minimum**
- If $f''(c) < 0$: local **maximum**
- If $f''(c) = 0$: **inconclusive** ŌĆö must use another method (first derivative test, or higher derivatives)

### Why This Works (Intuition)

If $f'(c) = 0$ and $f''(c) > 0$, then $f'$ is increasing at $c$. Since $f'(c) = 0$ and $f'$ is increasing, $f'$ was negative just before $c$ and positive just after ŌĆö $f$ decreased then increased, so $c$ is a minimum.

### Proof (Rigorous Version)

Assume $f'(c) = 0$ and $f''(c) > 0$. By definition of $f''(c)$:

$$f''(c) = \lim_{h \to 0} \frac{f'(c+h) - f'(c)}{h} = \lim_{h \to 0} \frac{f'(c+h)}{h} > 0$$

Since the limit is positive, there exists $\delta > 0$ such that for $0 < |h| < \delta$:

$$\frac{f'(c+h)}{h} > 0$$

This means:

- For $h \in (0, \delta)$: $h > 0$ so $f'(c+h) > 0$
- For $h \in (-\delta, 0)$: $h < 0$ so $f'(c+h) < 0$

Therefore $f' < 0$ just left of $c$ and $f' > 0$ just right of $c$: $c$ is a local minimum. $\blacksquare$

## The $n$-th Derivative Test (General Case)

When $f'(c) = f''(c) = \cdots = f^{(n-1)}(c) = 0$ but $f^{(n)}(c) \neq 0$:

- If $n$ is **odd**: $c$ is an **inflection point** (no extremum)
- If $n$ is **even** and $f^{(n)}(c) > 0$: local **minimum**
- If $n$ is **even** and $f^{(n)}(c) < 0$: local **maximum**

**Example:** $f(x) = x^4$ at $x = 0$.

$f'(0) = f''(0) = f'''(0) = 0$, $f^{(4)}(0) = 24 > 0$. $n = 4$ is even, so local minimum. Ō£ō

**Example:** $f(x) = x^3$ at $x = 0$.

$f'(0) = f''(0) = 0$, $f'''(0) = 6 \neq 0$. $n = 3$ is odd, so inflection point. Ō£ō

## Leibniz Rule for Higher Derivatives of a Product

For the product of two functions, the first derivative gives the product rule. For the $n$-th derivative:

$$\frac{d^n}{dx^n}[f(x)g(x)] = \sum_{k=0}^{n} \binom{n}{k} f^{(k)}(x), g^{(n-k)}(x)$$

This is structurally identical to the **binomial theorem** $(a+b)^n = \sum_{k=0}^n \binom{n}{k} a^k b^{n-k}$.

### Example

Find $\dfrac{d^3}{dx^3}[x^2 e^x]$.

Let $f = x^2$, $g = e^x$. Note $f' = 2x$, $f'' = 2$, $f''' = 0$, and $g^{(k)} = e^x$ for all $k$.

$$\frac{d^3}{dx^3}[x^2 e^x] = \binom{3}{0} x^2 e^x + \binom{3}{1} 2x \cdot e^x + \binom{3}{2} 2 \cdot e^x + \binom{3}{3} 0 \cdot e^x$$

$$= x^2 e^x + 6x e^x + 6e^x = e^x(x^2 + 6x + 6)$$

## Smooth Functions: The Class $C^n$

### Definitions

- $f \in C^0$ (or just $C$): $f$ is **continuous**
- $f \in C^1$: $f$ is continuously differentiable ($f'$ exists and is continuous)
- $f \in C^n$: $f^{(1)}, f^{(2)}, \ldots, f^{(n)}$ all exist and are continuous
- $f \in C^\infty$: $f$ is **smooth** ŌĆö all derivatives of all orders exist
- $f \in C^\omega$: $f$ is **analytic** ŌĆö smooth AND equals its Taylor series locally

### The Strict Hierarchy

$$C^\omega \subsetneq C^\infty \subsetneq \cdots \subsetneq C^n \subsetneq C^{n-1} \subsetneq \cdots \subsetneq C^1 \subsetneq C^0$$

Each inclusion is **strict**. Here's the standard example separating $C^\infty$ from $C^\omega$:

$$f(x) = \begin{cases} e^{-1/x^2} & x \neq 0 \ 0 & x = 0 \end{cases}$$

**Claim:** $f^{(n)}(0) = 0$ for all $n \geq 0$, yet $f \not\equiv 0$.

**Proof sketch:** By induction, one shows $f^{(n)}(x) = P_n(1/x) e^{-1/x^2}$ for some polynomial $P_n$, and $\lim_{x \to 0} x^{-k} e^{-1/x^2} = 0$ for every $k$ (exponential beats any polynomial). So $f^{(n)}(0) = 0$ for all $n$.

The Taylor series of $f$ at $0$ is identically zero ŌĆö but $f$ is not zero. So $f \in C^\infty \setminus C^\omega$.

> ­¤ōī This is the **bump function** prototype ŌĆö the building block of all smooth cutoff functions in differential geometry and PDE theory.

## Taylor's Theorem: Why Higher Derivatives Matter

Higher derivatives are not just abstractions ŌĆö they are the **coefficients of the best polynomial approximation** to $f$.

### Taylor Polynomial of Degree $n$

$$P_n(x) = \sum_{k=0}^{n} \frac{f^{(k)}(a)}{k!}(x-a)^k$$

$$= f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \cdots + \frac{f^{(n)}(a)}{n!}(x-a)^n$$

### Taylor's Theorem with Remainder (Lagrange Form)

**Theorem.** If $f \in C^{n+1}$ on an interval containing $a$ and $x$, then:

$$f(x) = P_n(x) + R_n(x)$$

where the remainder satisfies:

$$R_n(x) = \frac{f^{(n+1)}(c)}{(n+1)!}(x-a)^{n+1}$$

for some $c$ strictly between $a$ and $x$.

**Interpretation:** The error from the degree-$n$ approximation is controlled by the $(n+1)$-th derivative. The smoother the function, the faster the Taylor approximation converges.

### Visualization

```
f(x) = sin(x) near x = 0:

    PŌéü(x) = x                    Ōēł good for |x| < 0.5
    PŌéā(x) = x - x┬│/6             Ōēł good for |x| < 1.5
    PŌéģ(x) = x - x┬│/6 + xŌüĄ/120   Ōēł good for |x| < 2.5

        actual sin(x)
              ___
             /   \
    PŌéü: ---/-----\---  (just a line, diverges fast)
    PŌéā:  _/       \_   (follows better)
    PŌéģ: / matches sin well for much longer \
```

## The Second Derivative in Physics: Acceleration

If $s(t)$ is position as a function of time:

- $v(t) = s'(t)$: **velocity** (rate of change of position)
- $a(t) = s''(t) = v'(t)$: **acceleration** (rate of change of velocity)
- $j(t) = s'''(t)$: **jerk** (rate of change of acceleration ŌĆö relevant in mechanical engineering, roller coaster design)
- $s^{(4)}(t)$: **snap** (or jounce)
- $s^{(5)}(t)$: **crackle**
- $s^{(6)}(t)$: **pop**

Newton's second law $F = ma = ms''(t)$ is a **second-order ODE**. The entire framework of classical mechanics lives in second derivatives.

## Connection to AI, Machine Learning & Deep Learning

Higher derivatives are not just background mathematics in ML ŌĆö they drive some of the most important algorithms.

### 1. Loss Landscape and the Hessian

In training a neural network with parameters $\theta \in \mathbb{R}^d$, the loss function $\mathcal{L}(\theta)$ is a scalar function of millions of variables. The **Hessian matrix** is the matrix of second partial derivatives:

$$H_{ij} = \frac{\partial^2 \mathcal{L}}{\partial \theta_i , \partial \theta_j}$$

The Hessian encodes the **curvature** of the loss landscape:

- **Positive definite** $H$: local minimum (all eigenvalues $> 0$)
- **Negative definite** $H$: local maximum
- **Indefinite** $H$ (mixed eigenvalues): **saddle point**

In high-dimensional spaces (millions of parameters), saddle points are far more common than local minima. This is a second-order phenomenon entirely.

```
Loss landscape (schematic, 2D slice):

          saddle
    \      |      /
     \     |     /
      \____|____/    <-- local min
              
   SGD can escape saddles; Newton's method needs Hessian
```

### 2. Newton's Method and Second-Order Optimization

**Gradient descent** uses only first derivatives: $$\theta \leftarrow \theta - \eta \nabla_\theta \mathcal{L}$$

**Newton's method** uses second derivatives: $$\theta \leftarrow \theta - H^{-1} \nabla_\theta \mathcal{L}$$

This is derived from the second-order Taylor expansion of $\mathcal{L}$: $$\mathcal{L}(\theta + \delta) \approx \mathcal{L}(\theta) + \nabla \mathcal{L}^T \delta + \frac{1}{2} \delta^T H \delta$$

Minimizing over $\delta$: set gradient to zero $\Rightarrow H\delta = -\nabla\mathcal{L} \Rightarrow \delta = -H^{-1}\nabla\mathcal{L}$.

Newton's method converges **quadratically** (number of correct digits doubles each step), versus gradient descent's linear convergence. The catch: computing $H^{-1}$ costs $O(d^3)$ for $d$ parameters ŌĆö completely infeasible when $d = 10^8$.

**K-FAC, L-BFGS, Shampoo** are practical second-order optimizers that approximate $H^{-1}$ efficiently. These are active research areas.

### 3. Activation Function Smoothness

Why does the order of smoothness of activation functions matter?

|Activation|Smoothness|$f''$|
|---|---|---|
|Step function|$C^{-1}$ (not even continuous)|Dirac delta ŌĆö undefined as a function|
|ReLU|$C^0$ (continuous, not $C^1$)|Zero a.e., undefined at $0$|
|Leaky ReLU|$C^0$|Same issue|
|Sigmoid $\sigma$|$C^\infty$|$\sigma''(x) = \sigma(x)(1-\sigma(x))(1-2\sigma(x))$|
|Tanh|$C^\infty$|$\text{tanh}''(x) = -2\tanh(x)\text{sech}^2(x)$|
|GELU|$C^\infty$|Smooth everywhere|
|SiLU / Swish|$C^\infty$|$f(x) = x\sigma(x)$, smooth|

When you compute a **Taylor expansion** of the network output, you need higher derivatives of the activation functions. **Smooth activations** allow the network to represent higher-frequency content more efficiently.

### 4. Meta-Learning and MAML

**Model-Agnostic Meta-Learning (MAML)** (Finn et al., 2017) requires differentiating **through** the optimization process. The outer loop update involves:

$$\frac{\partial}{\partial \theta} \mathcal{L}!\left(\theta - \alpha \nabla_\theta \mathcal{L}(\theta)\right)$$

Expanding this via chain rule produces **second derivatives** $\frac{\partial^2 \mathcal{L}}{\partial \theta^2}$. You're literally computing the gradient of a gradient. This is sometimes called a **meta-gradient** and is a second-order calculation.

**First-order MAML (FOMAML)** drops these second-derivative terms ŌĆö it's faster but less accurate. **Reptile** is another approximation that avoids second derivatives entirely. The choice is a direct engineering tradeoff around the cost of computing $f''$.

### 5. Neural ODEs

**Neural ODEs** (Chen et al., 2018) define dynamics as:

$$\frac{dh}{dt} = f_\theta(h(t), t)$$

Training these requires the **adjoint method**, which involves solving a related ODE backward in time. The sensitivity equations involve derivatives of $f_\theta$ with respect to both state and parameters ŌĆö essentially second-order information. This is the deep learning version of classical sensitivity analysis in ODEs.

### 6. Smooth Loss Landscapes and Generalization

Recent theoretical work (particularly **sharpness-aware minimization, SAM**) argues that minima with small $|H|$ (flat minima, low curvature) **generalize better** than sharp minima (large $|H|$, high curvature).

The intuition: if a minimum is flat (second derivative small), small perturbations to $\theta$ don't change $\mathcal{L}$ much ŌĆö the model is robust. If a minimum is sharp (second derivative large), tiny perturbations to $\theta$ increase loss significantly ŌĆö the model may be overfitted.

SAM explicitly minimizes: $$\max_{|\epsilon| \leq \rho} \mathcal{L}(\theta + \epsilon)$$

which amounts to penalizing large Hessian eigenvalues ŌĆö a second-order regularization strategy.

### 7. Double Descent and the Role of Curvature

The **double descent** phenomenon in overparameterized models (where test loss decreases again after the interpolation threshold) can be partially explained via the spectrum of the Hessian at the interpolating solution. This is an active frontier connecting second-order calculus to statistical learning theory.

## Implicit Differentiation and Higher Derivatives

For implicitly defined curves $F(x, y) = 0$, you can find $y''$ by differentiating $y'$ implicitly.

### Example: $x^2 + y^2 = r^2$ (circle)

**First derivative.** Differentiate implicitly: $$2x + 2y \cdot y' = 0 \implies y' = -\frac{x}{y}$$

**Second derivative.** Differentiate $y' = -x/y$ implicitly: $$y'' = -\frac{y \cdot 1 - x \cdot y'}{y^2} = -\frac{y - x(-x/y)}{y^2} = -\frac{y^2 + x^2}{y^3} = -\frac{r^2}{y^3}$$

This is the formula for the **curvature** of a circle: $\kappa = |y''| / (1 + y'^2)^{3/2}$, which is constant (as expected for a circle).

## Parametric Curves (MIT 18.02 Perspective)

For $x = x(t)$, $y = y(t)$:

$$\frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{\dot{y}}{\dot{x}}$$

For the second derivative ŌĆö **do not differentiate $\dot{y}/\dot{x}$ with respect to $t$ and stop there**:

$$\frac{d^2y}{dx^2} = \frac{d}{dx}!\left[\frac{dy}{dx}\right] = \frac{\dfrac{d}{dt}!\left[\dfrac{\dot{y}}{\dot{x}}\right]}{\dot{x}}$$

**Example:** $x = t^2$, $y = t^3$.

$$\frac{dy}{dx} = \frac{3t^2}{2t} = \frac{3t}{2}$$

$$\frac{d^2y}{dx^2} = \frac{d}{dt}!\left[\frac{3t}{2}\right] \div \dot{x} = \frac{3/2}{2t} = \frac{3}{4t}$$

## Worked Problems

### Problem 1 ŌĆö Find all derivatives

$f(x) = \sin(2x)$.

$f'(x) = 2\cos(2x)$, $f''(x) = -4\sin(2x)$, $f'''(x) = -8\cos(2x)$, $f^{(4)}(x) = 16\sin(2x)$.

Pattern: $f^{(n)}(x) = 2^n \sin(2x + n\pi/2)$.

### Problem 2 ŌĆö Concavity and Inflection

$f(x) = x^4 - 4x^3$.

$f'(x) = 4x^3 - 12x^2$, $f''(x) = 12x^2 - 24x = 12x(x-2)$.

$f'' = 0$ at $x = 0$ and $x = 2$.

Sign of $f''$:

- $x < 0$: $12(-)(-) > 0$ ŌĆö concave up
- $0 < x < 2$: $12(+)(-) < 0$ ŌĆö concave down
- $x > 2$: $12(+)(+) > 0$ ŌĆö concave up

Both $x = 0$ and $x = 2$ are inflection points (sign changes at both).

### Problem 3 ŌĆö Second Derivative Test

$f(x) = x^3 - 6x^2 + 9x + 1$.

$f'(x) = 3x^2 - 12x + 9 = 3(x-1)(x-3)$.

Critical points: $x = 1$, $x = 3$.

$f''(x) = 6x - 12$.

$f''(1) = -6 < 0$ ŌåÆ local maximum at $x = 1$.  
$f''(3) = 6 > 0$ ŌåÆ local minimum at $x = 3$.

### Problem 4 ŌĆö Leibniz Rule

Find $\dfrac{d^4}{dx^4}[x^3 \sin x]$.

Let $f = x^3$ (so $f^{(k)} = 0$ for $k > 3$), $g = \sin x$ (so $g^{(k)} = \sin(x + k\pi/2)$).

$$\frac{d^4}{dx^4}[x^3 \sin x] = \sum_{k=0}^{3} \binom{4}{k} f^{(k)} g^{(4-k)}$$

$$= \binom{4}{0} x^3 \sin(x+2\pi) + \binom{4}{1} 3x^2 \sin(x+\tfrac{3\pi}{2}) + \binom{4}{2} 6x \sin(x+\pi) + \binom{4}{3} 6 \sin(x+\tfrac{\pi}{2})$$

$$= x^3 \sin x - 12x^2 \cos x - 36x \sin x + 24\cos x$$
