

> _"The derivative at a point is a number. The derivative as a function is the machine that produces that number for every point."_ — Personal note, inspired by Stewart §2.2 and MIT 18.02

---

## 0. Preamble — Why This Distinction Matters

When we first define $f'(a)$, we fix a base-point $a$ and ask: _how fast is $f$ changing there?_ The answer is a single real number — the slope of the tangent line at $(a, f(a))$.

But nothing forced us to fix $a$. Let $a$ vary freely across the domain of $f$, collect every such slope, and we get a **new function** $f'$ whose input is a point and whose output is the instantaneous rate of change there. This object — $f'$ as a function in its own right — is the subject of this note.

The conceptual leap is small but profound. Almost everything in analysis, geometry, and machine learning flows from taking this step seriously.

---

## 1. The Formal Definition

### 1.1 Derivative at a Point (Review)

$$f'(a) ;=; \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

provided the limit exists. Geometrically: the slope of the tangent to the graph of $f$ at $x = a$.

### 1.2 Derivative as a Function

**Definition.** Let $f : D \subseteq \mathbb{R} \to \mathbb{R}$. Define the _derived function_ $f' : D' \to \mathbb{R}$ by

$$\boxed{f'(x) ;=; \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}}$$

where $D' = {x \in D : \text{the above limit exists and is finite}}$.

The variable $x$ is now a _free variable_, not a fixed constant. For each $x \in D'$, the expression on the right produces one real number: the slope of $f$ at that particular $x$.

> **Notation zoo.** All of the following mean the same thing: $$f'(x) ;=; \frac{df}{dx} ;=; Df(x) ;=; \dot{f}(x) ;=; y'$$ Leibniz's $\frac{df}{dx}$ is particularly useful in physics and ML because it reminds you _which variable you differentiated with respect to_.

---

## 2. Domain of $f'$ — Where Differentiability Can Fail

The domain $D'$ of $f'$ can be strictly smaller than $D$. The limit defining $f'(x)$ fails to exist at points where the function is:

|Failure mode|Example|What happens geometrically|
|---|---|---|
|**Corner**|$f(x) =|x|
|**Cusp**|$f(x) = x^{2/3}$ at $x = 0$|Tangent line becomes vertical ($\pm\infty$)|
|**Vertical tangent**|$f(x) = x^{1/3}$ at $x = 0$|Slope is $\pm\infty$, limit diverges|
|**Jump discontinuity**|Any step function|Function isn't even continuous there|
|**Oscillation**|$f(x) = \sin(1/x)$ near $x = 0$|Limit does not exist|

**Key theorem (Stewart §2.8, Thm 4):** $$f \text{ differentiable at } a \implies f \text{ continuous at } a$$

The converse is famously false: $|x|$ is continuous everywhere but not differentiable at $0$.

---

## 3. Computing $f'$ from the Definition — Worked Examples

### Example 1: $f(x) = x^2$

$$f'(x) = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h} = \lim_{h \to 0} \frac{2xh + h^2}{h} = \lim_{h \to 0} (2x + h) = 2x$$

So $f' : \mathbb{R} \to \mathbb{R}$, $f'(x) = 2x$. Note: $f'$ is itself a linear function.

### Example 2: $f(x) = \sqrt{x}$, domain $(0, \infty)$

$$f'(x) = \lim_{h \to 0} \frac{\sqrt{x+h} - \sqrt{x}}{h} \cdot \frac{\sqrt{x+h}+\sqrt{x}}{\sqrt{x+h}+\sqrt{x}} = \lim_{h \to 0} \frac{h}{h(\sqrt{x+h}+\sqrt{x})} = \frac{1}{2\sqrt{x}}$$

Domain of $f'$: $(0, \infty)$ — the endpoint $x = 0$ is excluded because $f'(0) = +\infty$.

### Example 3: $f(x) = |x|$ — a non-example

$$\lim_{h \to 0^+} \frac{|h|}{h} = 1, \qquad \lim_{h \to 0^-} \frac{|h|}{h} = -1$$

One-sided limits disagree at $x = 0$, so $0 \notin D'$. For $x \neq 0$:

$$f'(x) = \text{sgn}(x) = \begin{cases} 1 & x > 0 \ -1 & x < 0 \end{cases}$$

## 4. Visualization

### 4.1 The Derivative Function as a "Slope Meter"

```
f(x) = x²              f'(x) = 2x

   ↑                      ↑
   |   *               *  |      /
   |  * *              *  |     /
   | *   *             *  |    /
   |*     *            *  |   /
   +-------→           +--+--/---→
                          0
                          
At x = -2: slope = -4   At x = 0: slope = 0   At x = 2: slope = +4
```

**Reading the derivative graph:**

- Where $f$ is increasing → $f' > 0$
- Where $f$ is decreasing → $f' < 0$
- At a local min/max of $f$ → $f' = 0$ (critical point)
- Where $f$ is concave up → $f'$ is increasing → $f'' > 0$

### 4.2 The "Sliding Tangent" Mental Model

Imagine a ruler tangent to the graph of $f$. As you slide it from left to right, the **slope of the ruler at each position** traces out the graph of $f'$.

For $f(x) = \sin x$:

- At $x = 0$: tangent has slope $+1$ → $f'(0) = 1$ ✓
- At $x = \pi/2$: tangent is horizontal → $f'(\pi/2) = 0$ ✓
- At $x = \pi$: tangent has slope $-1$ → $f'(\pi) = -1$ ✓
- Result: $f'(x) = \cos x$ (which the definition confirms)

```
f(x) = sin(x)         f'(x) = cos(x)

  1 ──╮     ╭──       1 ─╮           ╭─
     │ ╰───╯  │         │ ╲         ╱
  0 ─┼────────┼──     0 ─┼──╲─────╱──
     │        │         │   ╲   ╱
 -1 ─╯        ╰──    -1 ─    ╰─╯
     0   π   2π          0   π   2π
```

## 5. Higher-Order Derivatives

Since $f'$ is itself a function, we may differentiate it again.

$$f''(x) = (f')'(x) = \frac{d^2f}{dx^2} = \lim_{h \to 0}\frac{f'(x+h) - f'(x)}{h}$$

Continuing inductively:

$$f^{(n)}(x) = \frac{d^n f}{dx^n}$$

**Physical meaning:**

- $f(t)$: position
- $f'(t)$: velocity
- $f''(t)$: acceleration
- $f'''(t)$: jerk (rate of change of acceleration)

**Smoothness classes.** We say $f \in C^n(D)$ if $f$ is $n$-times continuously differentiable on $D$. The hierarchy is:

$$C^\infty \subset \cdots \subset C^2 \subset C^1 \subset C^0$$

These distinctions matter enormously in analysis — and in ML, where assumptions about smoothness of loss surfaces are crucial.

## 6. Connection to AI, Machine Learning, and Deep Learning

This is where the derivative-as-a-function becomes operationally indispensable. Here are the key conceptual links:

### 6.1 Gradient Descent — The Engine of All of Modern ML

A neural network has a **loss function** $\mathcal{L}(\theta)$ where $\theta \in \mathbb{R}^n$ is a vector of millions of parameters (weights and biases). Training the network means minimizing $\mathcal{L}$.

The update rule for **gradient descent** is:

$$\theta_{t+1} = \theta_t - \eta \nabla_\theta \mathcal{L}(\theta_t)$$

where $\eta > 0$ is the learning rate and $\nabla_\theta \mathcal{L}$ is the **gradient** — the multivariable generalization of $f'$.

> **The derivative-as-a-function is exactly what makes this possible.** We don't just need the gradient at one fixed $\theta$. We need it as a _function of $\theta$_ so that we can evaluate it at whichever $\theta_t$ we're currently at during training. We're sliding along the loss surface, re-computing the slope at every step.

### 6.2 Backpropagation — Chain Rule Applied at Scale

**Backpropagation** (Rumelhart et al., 1986) is an efficient algorithm for computing $\nabla_\theta \mathcal{L}$. Its mathematical core is the **chain rule**:

If $y = f(u)$ and $u = g(x)$, then: $$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$$

In a deep network with $L$ layers: $$\mathcal{L} = \ell_L \circ \ell_{L-1} \circ \cdots \circ \ell_1(\mathbf{x})$$

Backprop computes $\frac{\partial \mathcal{L}}{\partial \theta_k}$ for every layer $k$ by walking backward through the chain rule, reusing intermediate derivatives. This only makes sense if each $\ell_i$ is **differentiable as a function** — i.e., it has a well-defined derivative function we can evaluate at any point.

> **This is why activation functions like ReLU are analyzed so carefully.** $\text{ReLU}(x) = \max(0, x)$ has $\text{ReLU}'(x) = \mathbf{1}_{x > 0}$ — it is not differentiable at $x = 0$, but this is handled by convention (subgradient). Sigmoid $\sigma(x) = \frac{1}{1+e^{-x}}$ satisfies $\sigma'(x) = \sigma(x)(1 - \sigma(x))$ — a clean derivative function that's easy to compute and enables smooth gradients.

### 6.3 Taylor Expansion and Second-Order Methods

The Taylor expansion of $\mathcal{L}$ near a point $\theta_0$ is: $$\mathcal{L}(\theta_0 + \Delta\theta) \approx \mathcal{L}(\theta_0) + \nabla\mathcal{L}(\theta_0)^T \Delta\theta + \frac{1}{2}\Delta\theta^T H \Delta\theta + \cdots$$

where $H = \nabla^2 \mathcal{L}$ is the **Hessian** (matrix of second partial derivatives). Second-order optimizers (Newton's method, L-BFGS, Shampoo) use $H$ to take more intelligent steps. The Hessian is only available because the derivative is itself differentiable.

### 6.4 Automatic Differentiation (Autograd)

PyTorch, JAX, TensorFlow all implement **automatic differentiation**: they symbolically track the derivative function through every operation so that `loss.backward()` computes exact gradients (up to floating point) rather than finite differences. This is computationally possible because:

1. Every elementary function ($+$, $\times$, $\sin$, exp, ...) has a known derivative function.
2. The chain rule lets you compose them.
3. The derivative is a _function_, so it can itself be differentiated for higher-order methods.

```python
# PyTorch: x.grad is f'(x), computed via the chain rule
import torch
x = torch.tensor(2.0, requires_grad=True)
y = x**2 + 3*x          # f(x) = x² + 3x
y.backward()             # computes f'(x) = 2x + 3 at x = 2
print(x.grad)            # → tensor(7.)  ✓  (since 2(2)+3 = 7)
```

### 6.5 Differentiability as an Architectural Design Constraint

The entire paradigm of differentiable programming (or "software 2.0") requires every component of a pipeline to have a derivative _function_. This drives architectural choices:

|Constraint|Solution|Why|
|---|---|---|
|Argmax is not differentiable|Softmax (smooth approximation)|Enables gradients to flow through|
|Sorting is not differentiable|NeuralSort, differentiable ranking|Same|
|Discrete latent variables|Gumbel-Softmax trick|Same|
|Hard attention (argmax)|Soft attention|Allows gradient-based training|

## 7. Theorems About Differentiable Functions

### 7.1 Differentiability Implies Continuity

**Theorem.** If $f$ is differentiable at $a$, then $f$ is continuous at $a$.

_Proof._ We have $f(a+h) - f(a) = \frac{f(a+h)-f(a)}{h} \cdot h$. Taking $h \to 0$: the first factor converges to $f'(a)$ and the second to $0$, so the product converges to $0$. Hence $\lim_{h\to 0} f(a+h) = f(a)$. $\blacksquare$

### 7.2 Rolle's Theorem and the Mean Value Theorem

**Mean Value Theorem.** If $f$ is continuous on $[a,b]$ and differentiable on $(a,b)$, then $\exists, c \in (a,b)$ such that:

$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

The MVT says: the derivative function $f'$ must take the value of the average slope at some interior point. This theorem is the backbone of most convergence proofs in optimization theory — including guarantees on gradient descent.

## 8. Summary and Structural Map

```
Fix a point a                      Let a vary
─────────────────────────────────────────────────────────────────────
f'(a) ∈ ℝ         ──────────────→     f' : D' → ℝ  (a new function)
(a number)                             (assigns a slope to each x)

                                             ↓
                              Differentiate f' again → f''
                                             ↓
                              f(n): higher-order derivatives
                                             ↓
                              Smoothness classes: C0 ⊂ C1 ⊂ C2 ⊂ C∞
                                             ↓
                              Gradient of loss in ML (∇L: ℝⁿ → ℝⁿ)
                                             ↓
                              Backpropagation via chain rule
                                             ↓
                              Autograd systems (PyTorch, JAX)
```
