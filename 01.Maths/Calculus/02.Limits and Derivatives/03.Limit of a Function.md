
> _"The limit is not about what happens at the point. It is about what the function is trying to become. This distinction вҖ” between the value and the approach вҖ” is the philosophical heart of analysis."_

---

## В§1. The Motivating Crisis

Consider the function:

$$f(x) = \frac{x^2 - 1}{x - 1}$$

At $x = 1$: the numerator is $0$, the denominator is $0$. The function is **undefined** at $x = 1$.

And yet вҖ” something is happening near $x = 1$. Factor:

$$f(x) = \frac{(x-1)(x+1)}{x-1} = x + 1 \quad \text{for } x \neq 1$$

The function _wants_ to equal $2$ at $x = 1$, even though it doesn't exist there.

This "wanting" is the limit. Calculus is built on making this precise.

|$x$|$f(x) = \frac{x^2-1}{x-1}$|
|---|---|
|$0.9$|$1.9$|
|$0.99$|$1.99$|
|$0.999$|$1.999$|
|$1$|**undefined**|
|$1.001$|$2.001$|
|$1.01$|$2.01$|
|$1.1$|$2.1$|

The values approach $2$ from both sides. We write:

$$\lim_{x \to 1} \frac{x^2-1}{x-1} = 2$$

## В§2. Informal Definition

**Informal Definition.** We write $\displaystyle\lim_{x \to a} f(x) = L$ and say _"the limit of $f(x)$ as $x$ approaches $a$ equals $L$"_ if we can make $f(x)$ as close to $L$ as we like by taking $x$ sufficiently close to $a$ (but $x \neq a$).

Three things to note immediately:

1. **$x \neq a$ is mandatory.** The limit is about approach, not arrival. The value $f(a)$ is irrelevant вҖ” $f$ need not even be defined at $a$.
2. **Both sides.** $x$ must be able to approach $a$ from the left _and_ from the right, and both approaches must yield $L$.
3. **$L$ must be a finite real number** (for now вҖ” we will extend to $\pm\infty$).

---

## В§3. The Formal $\varepsilon$-$\delta$ Definition

The informal definition uses phrases like "as close as we like" and "sufficiently close." These are meaningless without precision. The $\varepsilon$-$\delta$ definition eliminates all ambiguity.

**Definition (Cauchy-Weierstrass).** We say $\displaystyle\lim_{x \to a} f(x) = L$ if:

$$\forall \varepsilon > 0,\ \exists \delta > 0\ \text{ such that }\ 0 < |x - a| < \delta \implies |f(x) - L| < \varepsilon$$

### Parsing the Definition

- $\varepsilon$ (epsilon) measures the **tolerated error in output**: how close $f(x)$ must be to $L$.
- $\delta$ (delta) measures the **required precision in input**: how close $x$ must be to $a$.
- The challenge is: _given any_ $\varepsilon > 0$ (no matter how tiny), _find_ a $\delta > 0$ that works.
- $0 < |x - a|$ encodes $x \neq a$.


### $\varepsilon$-$\delta$ Proof: Example

**Claim:** $\displaystyle\lim_{x \to 3} (2x - 1) = 5$.

**Proof.** Let $\varepsilon > 0$ be given. We need $|f(x) - 5| < \varepsilon$ whenever $0 < |x - 3| < \delta$.

Compute: $$|f(x) - 5| = |(2x-1) - 5| = |2x - 6| = 2|x - 3|$$

So $|f(x) - 5| < \varepsilon \iff 2|x-3| < \varepsilon \iff |x-3| < \dfrac{\varepsilon}{2}$.

Choose $\delta = \dfrac{\varepsilon}{2}$. Then:

$$0 < |x-3| < \delta = \frac{\varepsilon}{2} \implies |f(x) - 5| = 2|x-3| < 2 \cdot \frac{\varepsilon}{2} = \varepsilon \quad \blacksquare$$

> _The key technique: work backwards from $|f(x) - L| < \varepsilon$ to find the required $\delta$. The algebra determines $\delta$; then you write the proof forwards._

## В§4. One-Sided Limits

Sometimes $f$ approaches different values from the left and from the right. We need finer tools.

**Definition.** The **left-hand limit** is:

$$\lim_{x \to a^-} f(x) = L \quad \text{(x approaches a from below)}$$

**Definition.** The **right-hand limit** is:

$$\lim_{x \to a^+} f(x) = L \quad \text{(x approaches a from above)}$$

**Theorem.**

$$\lim_{x \to a} f(x) = L \iff \lim_{x \to a^-} f(x) = L \quad \text{and} \quad \lim_{x \to a^+} f(x) = L$$

_The two-sided limit exists if and only if both one-sided limits exist and are equal._

### Example: The Heaviside Function

$$H(x) = \begin{cases} 0 & x < 0 \ 1 & x \geq 0 \end{cases}$$

$$\lim_{x \to 0^-} H(x) = 0, \qquad \lim_{x \to 0^+} H(x) = 1$$

Since $0 \neq 1$, the two-sided limit $\displaystyle\lim_{x \to 0} H(x)$ **does not exist**.

```
H(x):

  1 |          *вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”
    |          |
  0 |вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”  |
    |__________|____________ x
              0

  Left limit = 0, Right limit = 1 вҶ’ no two-sided limit.
```

### Example: The Absolute Value

$$f(x) = \frac{|x|}{x}$$

$$\lim_{x \to 0^-} \frac{|x|}{x} = \frac{-x}{x} = -1, \qquad \lim_{x \to 0^+} \frac{|x|}{x} = \frac{x}{x} = 1$$

Limit does not exist at $0$.

## В§5. When Limits Fail to Exist

There are three canonical failure modes:

### Case 1: Jump Discontinuity

Left and right limits both exist but are unequal. (Heaviside function above.)

### Case 2: Infinite Oscillation

$$f(x) = \sin!\left(\frac{1}{x}\right) \quad \text{as } x \to 0$$

```
sin(1/x) near x = 0:

  1 |  ~~~~~~~~~~~~~~~~~~~~
    |  oscillations become
  0 |  infinitely rapid
    |  as x вҶ’ 0
 -1 |  ~~~~~~~~~~~~~~~~~~~~
    |________________________ x
    -0.1  0  0.1
```

The function oscillates infinitely between $-1$ and $1$. Neither one-sided limit exists. No limit.

### Case 3: Divergence to Infinity

$$\lim_{x \to 0} \frac{1}{x^2} = +\infty$$

This is a _convention_ вҖ” we say the limit is $+\infty$ to describe the behavior, but $+\infty$ is not a real number. The limit _does not exist_ in $\mathbb{R}$; we are describing _how_ it fails.

## В§6. Limit Laws

Once we know the limit of simple functions, we can build complex limits algebraically.

**Theorem (Limit Laws).** If $\displaystyle\lim_{x \to a} f(x) = L$ and $\displaystyle\lim_{x \to a} g(x) = M$, then:

|Law|Statement|
|---|---|
|Sum|$\lim [f + g] = L + M$|
|Difference|$\lim [f - g] = L - M$|
|Constant multiple|$\lim [cf] = cL$|
|Product|$\lim [f \cdot g] = L \cdot M$|
|Quotient|$\lim [f/g] = L/M$, provided $M \neq 0$|
|Power|$\lim [f^n] = L^n$|
|Root|$\lim \sqrt[n]{f} = \sqrt[n]{L}$, provided $L > 0$ for even $n$|

**Corollary (Direct Substitution Property).** If $f$ is a polynomial or rational function and $a$ is in the domain of $f$, then:

$$\lim_{x \to a} f(x) = f(a)$$

This is the content of _continuity_ вҖ” for "nice" functions, the limit and the value agree.

## В§7. The Squeeze Theorem

**Theorem (Squeeze / Sandwich Theorem).** If:

$$g(x) \leq f(x) \leq h(x)$$

near $a$ (but not necessarily at $a$), and:

$$\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L$$

then:

$$\lim_{x \to a} f(x) = L$$

### The Fundamental Trigonometric Limit

**Claim:** $\displaystyle\lim_{x \to 0} \frac{\sin x}{x} = 1$.

**Proof sketch (geometric squeeze).**

For $0 < x < \pi/2$, compare areas:

$$\text{Area}(\triangle OAB) \leq \text{Area(sector } OAB) \leq \text{Area}(\triangle OAT)$$

$$\frac{1}{2}\sin x \leq \frac{x}{2} \leq \frac{1}{2}\tan x$$

Divide by $\frac{1}{2}\sin x > 0$:

$$1 \leq \frac{x}{\sin x} \leq \frac{1}{\cos x}$$

Take reciprocals (reversing inequalities):

$$\cos x \leq \frac{\sin x}{x} \leq 1$$

As $x \to 0$: $\cos x \to 1$ and $1 \to 1$. By squeeze:

$$\lim_{x \to 0} \frac{\sin x}{x} = 1 \qquad \blacksquare$$

(Proof: multiply numerator and denominator by $(1 + \cos x)$ and use $\sin^2 x + \cos^2 x = 1$.)

## В§8. Limits at Infinity and Horizontal Asymptotes

**Definition.** $\displaystyle\lim_{x \to +\infty} f(x) = L$ means: for every $\varepsilon > 0$, there exists $N$ such that $x > N \implies |f(x) - L| < \varepsilon$.

The line $y = L$ is a **horizontal asymptote**.

### Examples

$$\lim_{x \to \infty} \frac{1}{x} = 0, \qquad \lim_{x \to \infty} \frac{3x^2 + 2x}{x^2 - 1} = 3, \qquad \lim_{x \to \infty} e^{-x} = 0$$

**Technique for rational functions:** divide numerator and denominator by the highest power of $x$.

$$\lim_{x \to \infty} \frac{3x^2 + 2x}{x^2 - 1} = \lim_{x \to \infty} \frac{3 + 2/x}{1 - 1/x^2} = \frac{3 + 0}{1 - 0} = 3$$

## В§9. Infinite Limits and Vertical Asymptotes

**Definition.** $\displaystyle\lim_{x \to a} f(x) = +\infty$ means: for every $M > 0$, there exists $\delta > 0$ such that $0 < |x - a| < \delta \implies f(x) > M$.

The line $x = a$ is a **vertical asymptote**.

### Example

$$\lim_{x \to 0^+} \ln x = -\infty, \qquad \lim_{x \to 0} \frac{1}{x^2} = +\infty, \qquad \lim_{x \to 1^+} \frac{1}{x-1} = +\infty$$

## В§11. Connection to Continuity

**Definition.** $f$ is **continuous at $a$** if all three conditions hold:

1. $f(a)$ is defined.
2. $\displaystyle\lim_{x \to a} f(x)$ exists.
3. $\displaystyle\lim_{x \to a} f(x) = f(a)$.

Failure of any one condition creates a **discontinuity**:

|Failure mode|Type|Example|
|---|---|---|
|$f(a)$ undefined, limit exists|Removable|$\frac{x^2-1}{x-1}$ at $x=1$|
|One-sided limits unequal|Jump|$H(x)$ at $x=0$|
|Limit is $\pm\infty$|Infinite|$1/x^2$ at $x=0$|
|Limit doesn't exist (oscillation)|Essential|$\sin(1/x)$ at $x=0$|

**Removable discontinuity** is the most important: the limit exists, but $f(a)$ is either missing or wrong. We can _repair_ it by redefining $f(a) = L$.

> _The derivative $f'(a) = \lim_{h\to 0} \frac{f(a+h)-f(a)}{h}$ is a limit of a difference quotient. Every result in differential calculus вҖ” every differentiation rule, every theorem вҖ” is a theorem about limits in disguise._

## В§12. Connection to AI, Machine Learning, and Deep Learning

> _The limit is the mathematical foundation of every continuous operation in ML. Understanding limits is understanding why neural networks can approximate, generalize, and converge._

### 12.1 The Universal Approximation Theorem

The Universal Approximation Theorem states (informally): a feedforward neural network with one hidden layer and enough neurons can approximate any continuous function $f : \mathbb{R}^n \to \mathbb{R}$ to arbitrary precision.

The word **arbitrary precision** is a limit statement:

$$\forall \varepsilon > 0,\ \exists \text{ a network } N_\varepsilon \text{ such that } \sup_x |N_\varepsilon(x) - f(x)| < \varepsilon$$

This is $\varepsilon$-$\delta$ logic applied to function spaces. Neural networks are approximating the _limit_ of a sequence of functions. The depth of modern networks pushes this from width-approximation toward depth-approximation, but the $\varepsilon$-precision language is identical.

### 12.2 Gradient Descent Convergence

For a convex loss $\mathcal{L}$, gradient descent with step size $\eta$ produces iterates $\mathbf{w}^{(t)}$ satisfying:

$$\lim_{t \to \infty} \mathcal{L}(\mathbf{w}^{(t)}) = \mathcal{L}^*$$

This is a **limit at infinity** вҖ” the loss approaches its minimum as iteration count grows. The convergence rate determines _how fast_ this limit is approached. For strongly convex $\mathcal{L}$:

$$\mathcal{L}(\mathbf{w}^{(t)}) - \mathcal{L}^* \leq \left(1 - \eta\mu\right)^t (\mathcal{L}(\mathbf{w}^{(0)}) - \mathcal{L}^*)$$

The right side $\to 0$ as $t \to \infty$ (geometric decay). The limit is zero; convergence is guaranteed.

### 12.3 Softmax and the Limit as Temperature вҶ’ 0

The **softmax** function:

$$\text{softmax}(\mathbf{z}/T)_i = \frac{e^{z_i/T}}{\sum_j e^{z_j/T}}$$

with temperature $T > 0$. Consider:

$$\lim_{T \to 0^+} \text{softmax}(\mathbf{z}/T) = \mathbf{e}_{\arg\max_i z_i}$$

As $T \to 0^+$, softmax approaches the **argmax** function вҖ” a one-hot vector. This is a limit of a continuous function approaching a discontinuous one. It explains why during inference (low temperature) the model becomes more "decisive," while during training (higher temperature) probability mass is spread more smoothly.

### 12.4 Vanishing Gradients and Limits

In deep networks, the gradient of the loss with respect to early-layer weights involves products of Jacobians. For sigmoid activations $\sigma(x) = \frac{1}{1+e^{-x}}$:

$$\sigma'(x) = \sigma(x)(1-\sigma(x)) \leq \frac{1}{4}$$

In a network with $L$ layers, the gradient magnitude is bounded by:

$$|\nabla_{\mathbf{w}_1} \mathcal{L}| \leq \left(\frac{1}{4}\right)^L \cdot C$$

As $L \to \infty$:

$$\lim_{L \to \infty} \left(\frac{1}{4}\right)^L = 0$$

This is the **vanishing gradient problem** вҖ” a limit at infinity converging to zero, making early layers untrainable. The solution (BatchNorm, ResNets, ReLU) is fundamentally about preventing this limit from being reached.

### 12.5 L'HГҙpital's Rule in Loss Analysis

The $\frac{0}{0}$ indeterminate form appears in ML theory. For example, when analyzing the cross-entropy loss:

$$\mathcal{L} = -\sum_i y_i \log \hat{p}_i$$

as $\hat{p}_i \to 0$ for a true class ($y_i = 1$):

$$\lim_{\hat{p} \to 0^+} \hat{p} \log \hat{p} = \lim_{\hat{p} \to 0^+} \frac{\log \hat{p}}{1/\hat{p}} \stackrel{\text{L'H}}{=} \lim_{\hat{p} \to 0^+} \frac{1/\hat{p}}{-1/\hat{p}^2} = \lim_{\hat{p} \to 0^+} (-\hat{p}) = 0$$

This ensures the loss remains finite (and the KL divergence is well-defined) even at boundary probabilities.

### 12.6 Activation Functions as Limit Constructions

|Activation|Limit property|
|---|---|
|ReLU: $\max(0,x)$|$\lim_{x\to 0^-} = 0$, $\lim_{x\to 0^+} = 0$; continuous but not differentiable|
|Sigmoid: $\frac{1}{1+e^{-x}}$|$\lim_{x\to -\infty} = 0$, $\lim_{x\to +\infty} = 1$; horizontal asymptotes|
|Tanh: $\frac{e^x - e^{-x}}{e^x + e^{-x}}$|$\lim_{x\to \pm\infty} = \pm 1$; bounded output|
|Softplus: $\ln(1+e^x)$|$\lim_{x\to -\infty} \approx e^x \to 0$; smooth ReLU approximation|

Every activation function is designed with its limiting behavior in mind вҖ” what does it do to very large or very small inputs? That is a question about limits at infinity.

## В§13. L'HГҙpital's Rule

For completeness вҖ” the most powerful tool for $\frac{0}{0}$ and $\frac{\infty}{\infty}$ indeterminate forms.

**Theorem (L'HГҙpital).** If $\displaystyle\lim_{x \to a} f(x) = 0$ and $\displaystyle\lim_{x \to a} g(x) = 0$ (or both $\pm\infty$), and $g'(x) \neq 0$ near $a$, then:

$$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$$

provided the right side exists.

### Example

$$\lim_{x \to 0} \frac{e^x - 1}{x} \stackrel{\text{L'H}}{=} \lim_{x \to 0} \frac{e^x}{1} = 1$$

This is the derivative of $e^x$ at $x = 0$ вҖ” L'HГҙpital confirms the definition.

> _Warning: L'HГҙpital requires the $\frac{0}{0}$ or $\frac{\infty}{\infty}$ form. Applying it to $\lim_{x\to 0} \frac{\sin x}{x + 1} = \frac{0}{1}$ (not indeterminate!) would give the wrong answer._

## В§14. Precise Proof Technique Summary

When writing $\varepsilon$-$\delta$ proofs:

1. **Scratch work (backwards):** Start from $|f(x) - L| < \varepsilon$. Simplify. Find what $|x - a| <\ ?$ must be.
2. **Bound complications:** If $|f(x) - L|$ involves $|x - a|$ multiplied by something that depends on $x$, bound that factor by assuming $|x-a| < 1$ (say), then choose $\delta = \min(1, \varepsilon/\text{bound})$.
3. **Write the proof forwards:** State $\delta$, assume $0 < |x-a| < \delta$, chain inequalities to conclude $|f(x) - L| < \varepsilon$.

### Example with Bounding

**Claim:** $\displaystyle\lim_{x \to 2} x^2 = 4$.

**Scratch:** $|x^2 - 4| = |x-2||x+2|$. Need to bound $|x+2|$.

Assume $|x-2| < 1$, so $1 < x < 3$, so $|x+2| < 5$.

Thus $|x^2-4| < 5|x-2|$. Need $5|x-2| < \varepsilon$, i.e., $|x-2| < \varepsilon/5$.

**Proof.** Let $\varepsilon > 0$. Set $\delta = \min!\left(1, \dfrac{\varepsilon}{5}\right)$.

If $0 < |x-2| < \delta$, then $|x-2| < 1 \Rightarrow |x+2| < 5$, and:

$$|x^2 - 4| = |x-2||x+2| < \frac{\varepsilon}{5} \cdot 5 = \varepsilon \quad \blacksquare$$
