

> _"The homogeneous equation describes the soul of the system. The nonhomogeneous equation describes what happens when the world pushes back."_

## § 0 — The Setup and the Question

We have spent considerable effort understanding the homogeneous equation $L[y] = 0$. Now we turn on an external force. The **nonhomogeneous second order linear ODE** is:

$$\boxed{y'' + p(x)y' + q(x)y = g(x)}$$

where $g(x) \not\equiv 0$ is the **forcing function** (also called the **nonhomogeneous term**, the **input**, or the **driving term**).

**The fundamental question:** Given that we understand $L[y] = 0$ completely, how do we find all solutions to $L[y] = g$?

The answer has a breathtaking algebraic structure — one that appears in linear algebra, functional analysis, and throughout applied mathematics. Understanding it deeply means you will never need to memorize a method; you will _derive_ every method from first principles.

---

## § 1 — The Structure Theorem

This is the most important theorem of the subject. Everything else is a consequence.

### 1.1 Statement

> **Theorem (Structure of Solutions).** Let $p, q, g$ be continuous on an open interval $I$. Let ${y_1, y_2}$ be a fundamental set of solutions of the homogeneous equation $L[y] = 0$, and let $y_p$ be **any particular solution** of $L[y] = g$. Then every solution of $L[y] = g$ on $I$ has the form:
> 
> $$y = \underbrace{c_1 y_1 + c_2 y_2}_{y_h \text{ — homogeneous general solution}} + \underbrace{y_p}_{\text{particular solution}}$$
> 
> for some constants $c_1, c_2 \in \mathbb{R}$.

### 1.2 Proof

**Part 1 — $y_h + y_p$ solves $L[y] = g$:**

$$L[y_h + y_p] = L[y_h] + L[y_p] = 0 + g = g \qquad \checkmark$$

**Part 2 — Every solution has this form:**

Let $y$ be any solution of $L[y] = g$. Consider $z = y - y_p$. Then: $$L[z] = L[y] - L[y_p] = g - g = 0$$

So $z$ solves the homogeneous equation. Since ${y_1, y_2}$ is a fundamental set, $z = c_1 y_1 + c_2 y_2$ for some constants. Therefore: $$y = y_p + z = y_p + c_1 y_1 + c_2 y_2 \qquad \blacksquare$$

### 1.3 The Linear Algebra Interpretation

In linear algebra, if $A\mathbf{x} = \mathbf{b}$ has a solution $\mathbf{x}_p$, then the general solution is:

$$\mathbf{x} = \mathbf{x}_p + \mathbf{x}_{\text{null}}$$

where $\mathbf{x}_{\text{null}} \in \ker(A)$. This is **exactly** the structure theorem above:

- $A \leftrightarrow L$ (the linear operator)
- $\mathbf{x}_p \leftrightarrow y_p$ (particular solution)
- $\ker(A) \leftrightarrow \text{span}{y_1, y_2}$ (null space, 2-dimensional)
- $\mathbf{b} \leftrightarrow g$ (the forcing)

The solution set of $L[y] = g$ is a **coset** of the solution space of $L[y] = 0$ — an affine subspace, not a vector space (it does not contain $0$ unless $y_p = 0$).

> **The operational consequence:** We need only find _one_ particular solution. Any one will do — the constants $c_1, c_2$ absorb the difference between different choices. This is the license to be clever about how we find $y_p$.

## § 2 — Method I: Undetermined Coefficients

### 2.1 The Idea

Valid for **constant coefficient** equations $ay'' + by' + cy = g(x)$ when $g(x)$ belongs to a special class of functions closed under differentiation: polynomials, exponentials, sines, cosines, and their products.

**The principle:** $L[y_p]$ must equal $g(x)$. If $g$ is a sum of exponentials times polynomials times trig, then $y_p$ should live in the same "family" — because differentiating such functions stays within the family. We guess the form, substitute, and solve for the unknown coefficients.

### 2.2 The Trial Function Table

For constant coefficient $L = aD^2 + bD + c$ and the given $g(x)$, the trial $y_p$ is:

|$g(x)$|Trial $y_p$|
|---|---|
|$P_n(x) = a_n x^n + \cdots + a_0$|$x^s(A_n x^n + \cdots + A_1 x + A_0)$|
|$e^{\alpha x}$|$x^s A e^{\alpha x}$|
|$\cos(\beta x)$ or $\sin(\beta x)$|$x^s(A\cos\beta x + B\sin\beta x)$|
|$e^{\alpha x}\cos(\beta x)$ or $e^{\alpha x}\sin(\beta x)$|$x^s e^{\alpha x}(A\cos\beta x + B\sin\beta x)$|
|$P_n(x)e^{\alpha x}$|$x^s(A_n x^n + \cdots + A_0)e^{\alpha x}$|
|Sum of above|Sum of corresponding trials|

**The multiplicity rule for $s$:**

- $s = 0$ if no term in the trial is a solution of $L[y] = 0$.
- $s = 1$ if some term in the trial **is** a solution of $L[y] = 0$ (simple resonance).
- $s = 2$ if some term in the trial is a solution corresponding to a **double root** (double resonance).

> **Why does the multiplicity rule exist?** If the trial $y_p$ satisfies the homogeneous equation, then $L[y_p] = 0 \neq g(x)$ — the substitution gives $0 = g$ and no coefficient can be determined. Multiplying by $x^s$ (with $s$ chosen minimally) moves the trial out of the kernel of $L$, making it possible to match $g$.

### 2.3 Worked Examples

#### Example 1 — Polynomial Forcing

**Problem:** $y'' - 2y' - 3y = 3x^2 + 4x$.

Homogeneous: $r^2 - 2r - 3 = (r-3)(r+1) = 0$. Roots $r = 3, -1$.

$g(x) = 3x^2 + 4x$ is a polynomial of degree 2. Trial: $y_p = Ax^2 + Bx + C$ (no resonance, $s = 0$).

$$y_p' = 2Ax + B, \quad y_p'' = 2A$$

Substituting: $$2A - 2(2Ax + B) - 3(Ax^2 + Bx + C) = 3x^2 + 4x$$ $$-3Ax^2 + (-4A - 3B)x + (2A - 2B - 3C) = 3x^2 + 4x + 0$$

Matching coefficients:

- $x^2$: $-3A = 3 \implies A = -1$
- $x^1$: $-4A - 3B = 4 \implies 4 - 3B = 4 \implies B = 0$
- $x^0$: $2A - 2B - 3C = 0 \implies -2 - 3C = 0 \implies C = -\frac{2}{3}$

$$y_p = -x^2 - \frac{2}{3}, \quad y = c_1 e^{3x} + c_2 e^{-x} - x^2 - \frac{2}{3}$$

#### Example 2 — Exponential Forcing, No Resonance

**Problem:** $y'' - 5y' + 6y = 2e^{4x}$.

Characteristic roots: $r = 2, 3$. Since $e^{4x}$ is not a homogeneous solution ($4 \neq 2, 3$), try $y_p = Ae^{4x}$.

$$16Ae^{4x} - 20Ae^{4x} + 6Ae^{4x} = 2Ae^{4x} = 2e^{4x} \implies A = 1$$

$$y = c_1 e^{2x} + c_2 e^{3x} + e^{4x}$$

#### Example 3 — Resonance ($s = 1$)

**Problem:** $y'' - 5y' + 6y = 2e^{2x}$.

Roots $r = 2, 3$. Now $e^{2x}$ **is** a homogeneous solution (simple root $r = 2$). Multiply by $x$: try $y_p = Axe^{2x}$.

$$y_p' = A(1 + 2x)e^{2x}, \quad y_p'' = A(4 + 4x)e^{2x}$$

$$A(4 + 4x)e^{2x} - 5A(1 + 2x)e^{2x} + 6Axe^{2x} = A(4 + 4x - 5 - 10x + 6x)e^{2x} = -Ae^{2x}$$

Setting $-A = 2$: $A = -2$.

$$y = c_1 e^{2x} + c_2 e^{3x} - 2xe^{2x}$$

#### Example 4 — Double Resonance ($s = 2$)

**Problem:** $y'' - 4y' + 4y = e^{2x}$.

Characteristic: $(r-2)^2 = 0$. Double root $r = 2$. Both $e^{2x}$ and $xe^{2x}$ are homogeneous solutions. Try $y_p = Ax^2 e^{2x}$.

$$y_p' = A(2x + 2x^2)e^{2x}, \quad y_p'' = A(2 + 8x + 4x^2)e^{2x}$$

$$A(2 + 8x + 4x^2)e^{2x} - 4A(2x + 2x^2)e^{2x} + 4Ax^2 e^{2x} = 2Ae^{2x} = e^{2x}$$

$$A = \frac{1}{2}, \quad y = (c_1 + c_2 x)e^{2x} + \frac{x^2}{2}e^{2x}$$

#### Example 5 — Trigonometric Forcing

**Problem:** $y'' + 9y = 2\sin(3x)$.

Characteristic: $r^2 + 9 = 0$, roots $\pm 3i$. The homogeneous solution contains $\sin 3x$ and $\cos 3x$ — resonance! Try $y_p = x(A\cos 3x + B\sin 3x)$.

$$y_p' = (A\cos 3x + B\sin 3x) + x(-3A\sin 3x + 3B\cos 3x)$$ $$y_p'' = (-3A\sin 3x + 3B\cos 3x) + (-3A\sin 3x + 3B\cos 3x) + x(-9A\cos 3x - 9B\sin 3x)$$ $$= 6B\cos 3x - 6A\sin 3x - 9x(A\cos 3x + B\sin 3x)$$

$$y_p'' + 9y_p = 6B\cos 3x - 6A\sin 3x = 2\sin 3x$$

Matching: $6B = 0 \Rightarrow B = 0$ and $-6A = 2 \Rightarrow A = -\frac{1}{3}$.

$$y = c_1\cos 3x + c_2\sin 3x - \frac{x}{3}\cos 3x$$

> **Physical meaning of resonance:** The forcing $\sin(3x)$ is at exactly the natural frequency of the oscillator ($\omega_0 = 3$). The system has no way to "absorb" the energy — each cycle, the driving force is perfectly in phase with the response, pumping in more energy. The amplitude grows without bound as $\frac{x}{3}\cos 3x$. In a physical system with no damping, this is catastrophic. Resonance is why bridges need careful engineering and why MRI machines have such tight frequency tolerances.

#### Example 6 — Sum of Forcing Terms

**Problem:** $y'' + 4y = x + e^x + \sin 2x$.

Roots: $\pm 2i$. Homogeneous: $c_1\cos 2x + c_2\sin 2x$.

Decompose $g = g_1 + g_2 + g_3$ and find $y_{p1}$, $y_{p2}$, $y_{p3}$ separately.

**$g_1 = x$:** Trial $Ax + B$ (no resonance). $4(Ax + B) = x \Rightarrow A = 1/4$, $B = 0$. So $y_{p1} = x/4$.

**$g_2 = e^x$:** Trial $Ae^x$. $(1 + 4)Ae^x = e^x \Rightarrow A = 1/5$. So $y_{p2} = e^x/5$.

**$g_3 = \sin 2x$:** $\sin 2x$ is a homogeneous solution (resonance, $s = 1$). Trial $x(A\cos 2x + B\sin 2x)$.

$$y_{p3}'' + 4y_{p3} = -4B\cos 2x + 4A\sin 2x\cdot(-1) = \sin 2x$$

Wait — let me redo: working through as in Example 5 with $\omega = 2$:

$$y_{p3}'' + 4y_{p3} = 4A\cos 2x + 4B\sin 2x \cdot (\ldots)$$

Differentiating $y_{p3} = x(A\cos 2x + B\sin 2x)$: $$y_{p3}'' = -4x(A\cos 2x + B\sin 2x) + (-4A\sin 2x + 4B\cos 2x)$$ $$y_{p3}'' + 4y_{p3} = 4B\cos 2x - 4A\sin 2x = \sin 2x$$

So $4B = 0$, $-4A = 1 \Rightarrow A = -1/4$. Thus $y_{p3} = -\frac{x}{4}\cos 2x$.

$$y = c_1\cos 2x + c_2\sin 2x + \frac{x}{4} + \frac{e^x}{5} - \frac{x}{4}\cos 2x$$

## § 3 — Method II: Variation of Parameters

### 3.1 Motivation

Undetermined coefficients is fast but has two crippling restrictions:

1. The equation must have **constant coefficients**.
2. $g(x)$ must belong to the "nice" family (polynomials $\times$ exponentials $\times$ trig).

Variation of parameters works for **any** $g(x)$ and **any** equation where we can find a fundamental set — including variable-coefficient equations.

The price: we must compute two integrals, which may not have closed forms.

### 3.2 Derivation from First Principles

Given the fundamental set ${y_1, y_2}$ of $L[y] = 0$, **promote the constants to functions**:

$$y_p = u_1(x)y_1(x) + u_2(x)y_2(x)$$

We have two unknowns $u_1, u_2$ and one equation $L[y_p] = g$. We impose a **constraint** to simplify — specifically, to eliminate second derivatives of $u_1, u_2$:

$$\textbf{Constraint: } u_1'y_1 + u_2'y_2 = 0 \tag{C}$$

Under this constraint, differentiate $y_p$:

$$y_p' = u_1'y_1 + u_1 y_1' + u_2'y_2 + u_2 y_2' = u_1 y_1' + u_2 y_2' \quad \text{(using C)}$$

$$y_p'' = u_1'y_1' + u_1 y_1'' + u_2'y_2' + u_2 y_2''$$

Substitute into $y_p'' + p y_p' + q y_p = g$:

$$u_1'y_1' + u_1 y_1'' + u_2'y_2' + u_2 y_2'' + p(u_1 y_1' + u_2 y_2') + q(u_1 y_1 + u_2 y_2) = g$$

$$u_1'y_1' + u_2'y_2' + u_1(y_1'' + py_1' + qy_1) + u_2(y_2'' + py_2' + qy_2) = g$$

Since $y_1, y_2$ solve $L[y] = 0$, the bracketed terms vanish:

$$\textbf{ODE condition: } u_1'y_1' + u_2'y_2' = g \tag{ODE}$$

### 3.3 The System and Its Solution

Equations (C) and (ODE) form a $2 \times 2$ linear system for $u_1'$ and $u_2'$:

$$\begin{pmatrix} y_1 & y_2 \ y_1' & y_2' \end{pmatrix} \begin{pmatrix} u_1' \ u_2' \end{pmatrix} = \begin{pmatrix} 0 \ g \end{pmatrix}$$

The coefficient matrix has determinant $W(x) = y_1 y_2' - y_1' y_2 \neq 0$. By Cramer's rule:

$$u_1' = -\frac{y_2 g}{W}, \qquad u_2' = \frac{y_1 g}{W}$$

Integrating:

$$u_1(x) = -\int \frac{y_2(x), g(x)}{W(x)},dx, \qquad u_2(x) = \int \frac{y_1(x), g(x)}{W(x)},dx$$

**The particular solution:**

$$\boxed{y_p(x) = -y_1(x)\int \frac{y_2(x), g(x)}{W(x)},dx + y_2(x)\int \frac{y_1(x), g(x)}{W(x)},dx}$$

### 3.4 The Formula as a Green's Function

Combining the integrals:

$$y_p(x) = \int_{x_0}^x \frac{y_1(t)y_2(x) - y_1(x)y_2(t)}{W(t)}, g(t),dt = \int_{x_0}^x G(x,t), g(t),dt$$

where $G(x,t) = \frac{y_1(t)y_2(x) - y_1(x)y_2(t)}{W(t)}$ is the **Green's function** of the operator $L$ (with initial conditions). The particular solution is the **convolution** of the Green's function with the forcing — every forcing value $g(t)$ is weighted by how much influence it has at position $x$, given by $G(x,t)$.

### 3.5 Worked Examples

#### Example 7 — Variation of Parameters with Trig Denominator

**Problem:** $y'' + y = \sec x$, $x \in (-\pi/2, \pi/2)$.

Undetermined coefficients fails: $\sec x$ is not in the "nice" family.

Homogeneous: $r^2 + 1 = 0$, $y_1 = \cos x$, $y_2 = \sin x$.

Wronskian: $W = \cos x \cos x - (-\sin x)\sin x = \cos^2 x + \sin^2 x = 1$.

$$u_1' = -\frac{\sin x \cdot \sec x}{1} = -\frac{\sin x}{\cos x} = -\tan x$$

$$u_2' = \frac{\cos x \cdot \sec x}{1} = 1$$

Integrating:

$$u_1 = \int (-\tan x),dx = \ln|\cos x|, \qquad u_2 = x$$

$$y_p = \cos x \cdot \ln|\cos x| + x\sin x$$

$$y = c_1\cos x + c_2\sin x + \cos x\ln|\cos x| + x\sin x$$

#### Example 8 — Variable Coefficient Equation

**Problem:** $x^2 y'' - 2xy' + 2y = x^3\ln x$, $x > 0$.

This is an **Euler–Cauchy** equation. The homogeneous equation $x^2 y'' - 2xy' + 2y = 0$ has solutions found via $y = x^m$:

$m(m-1) - 2m + 2 = m^2 - 3m + 2 = (m-1)(m-2) = 0$, so $y_1 = x$, $y_2 = x^2$.

Wronskian: $W = x \cdot 2x - 1 \cdot x^2 = x^2$.

Divide the ODE by $x^2$ to get standard form: $y'' - \frac{2}{x}y' + \frac{2}{x^2}y = x\ln x$.

So $g(x) = x\ln x$.

$$u_1' = -\frac{x^2 \cdot x\ln x}{x^2} = -x\ln x$$

$$u_2' = \frac{x \cdot x\ln x}{x^2} = \ln x$$

Integrating by parts:

$$u_1 = -\int x\ln x,dx = -\frac{x^2}{2}\ln x + \frac{x^2}{4}$$

$$u_2 = \int \ln x,dx = x\ln x - x$$

$$y_p = x\left(-\frac{x^2}{2}\ln x + \frac{x^2}{4}\right) + x^2(x\ln x - x) = -\frac{x^3}{2}\ln x + \frac{x^3}{4} + x^3\ln x - x^3$$

$$y_p = \frac{x^3}{2}\ln x - \frac{3x^3}{4}$$

Since $-\frac{3}{4}x^3$ looks like it could be absorbed (but $x^3$ is not a homogeneous solution — check: $y_1 = x$, $y_2 = x^2$, so $x^3$ is genuinely particular):

$$y = c_1 x + c_2 x^2 + \frac{x^3}{2}\ln x - \frac{3x^3}{4}$$

#### Example 9 — When Integrals Don't Close

**Problem:** $y'' + y = \frac{1}{1 + x^2}$.

Homogeneous: $y_1 = \cos x$, $y_2 = \sin x$, $W = 1$.

$$u_1' = -\frac{\sin x}{1 + x^2}, \qquad u_2' = \frac{\cos x}{1 + x^2}$$

These integrals have no closed form in elementary functions. But the **existence** of the solution is guaranteed; we write:

$$y_p = -\cos x \int_0^x \frac{\sin t}{1 + t^2},dt + \sin x \int_0^x \frac{\cos t}{1 + t^2},dt$$

This is a perfectly valid, well-defined function — it just cannot be written in closed form. Numerically, it is computed as easily as any integral. This is why variation of parameters is the theoretically complete method: it always produces the answer, even when the answer is not "nice."

## § 4 — The Operator Method and Exponential Shift

### 4.1 The Annihilator Method

For constant coefficient equations, there is an elegant operator-theoretic method that systematizes undetermined coefficients.

The **differential operator** $D = d/dx$ acts on functions. The ODE $L[y] = ay'' + by' + cy = g$ is written as:

$$L(D)y = (aD^2 + bD + c)y = g$$

For the trial to work, we need $L(D)[y_p] = g$. If we can find a differential operator $A(D)$ — the **annihilator** — such that $A(D)[g] = 0$, then applying $A(D)$ to both sides:

$$A(D)L(D)[y] = 0$$

This is now a **homogeneous** equation with characteristic polynomial $A(r)L(r) = 0$. Its general solution contains $y_h$ (from $L(r) = 0$) plus extra terms from $A(r) = 0$. These extra terms form the trial $y_p$ — we substitute back into the original equation to find the coefficients.

**Annihilators for common functions:**

|$g(x)$|Annihilator $A(D)$|
|---|---|
|$x^n$|$D^{n+1}$|
|$e^{\alpha x}$|$D - \alpha$|
|$e^{\alpha x}x^n$|$(D - \alpha)^{n+1}$|
|$\cos\beta x$ or $\sin\beta x$|$D^2 + \beta^2$|
|$e^{\alpha x}\cos\beta x$ or $e^{\alpha x}\sin\beta x$|$(D - \alpha)^2 + \beta^2$|

**Example:** $y'' + 4y = x\cos 2x$.

Annihilator of $x\cos 2x$: $(D^2 + 4)^2$ (since $e^{0 \cdot x}\cos 2x$ has annihilator $(D^2 + 4)$, and the extra $x$ requires squaring).

Apply $(D^2 + 4)^2$ to both sides: $(D^2 + 4)^3 y = 0$.

Characteristic roots of $(r^2 + 4)^3 = 0$: $r = \pm 2i$ (each with multiplicity 3).

General solution of $(D^2 + 4)^3 y = 0$: $$y = (c_1 + c_2 x + c_3 x^2)\cos 2x + (c_4 + c_5 x + c_6 x^2)\sin 2x$$

The homogeneous part of the original (from $(D^2 + 4)y = 0$): $c_1\cos 2x + c_4\sin 2x$. The particular trial is the remaining part: $(c_2 x + c_3 x^2)\cos 2x + (c_5 x + c_6 x^2)\sin 2x$. Substitute into the original ODE to find the four free coefficients.

### 4.2 The Exponential Shift Theorem

> **Theorem.** For any polynomial $p(D)$ with constant coefficients: $$p(D)[e^{\alpha x} f(x)] = e^{\alpha x} p(D + \alpha)[f(x)]$$

**Proof.** By direct computation using the product rule and the fact that $D[e^{\alpha x}] = \alpha e^{\alpha x}$: $$D[e^{\alpha x} f] = e^{\alpha x}(\alpha f + f') = e^{\alpha x}(D + \alpha)[f]$$

Iterating: $D^n[e^{\alpha x}f] = e^{\alpha x}(D + \alpha)^n[f]$. Linearity extends to any polynomial. $\blacksquare$

**Application:** To find $y_p$ for $p(D)[y] = e^{\alpha x}g(x)$, write $y_p = e^{\alpha x}v(x)$:

$$e^{\alpha x}p(D + \alpha)[v] = e^{\alpha x}g(x) \implies p(D + \alpha)[v] = g(x)$$

This replaces the original operator with a shifted operator — often simpler.

**Example:** $(D - 2)^2 y = e^{2x}\sin x$.

Try $y_p = e^{2x}v$. Then $(D - 2)^2[e^{2x}v] = e^{2x}D^2[v] = e^{2x}\sin x$.

So $v'' = \sin x \Rightarrow v = -\sin x$. Thus $y_p = -e^{2x}\sin x$.

_Check:_ $(D-2)^2[-e^{2x}\sin x] = e^{2x}D^2[-\sin x] = e^{2x}\sin x$. ✓

## § 5 — Superposition for Multiple Forcing Terms

A direct consequence of linearity:

> **Principle of Superposition.** If $y_{p_k}$ is a particular solution of $L[y] = g_k(x)$ for $k = 1, 2, \ldots, n$, then $y_p = \sum_{k=1}^n c_k y_{p_k}$ is a particular solution of $L[y] = \sum_{k=1}^n c_k g_k(x)$.

**Proof:** $L\left[\sum c_k y_{p_k}\right] = \sum c_k L[y_{p_k}] = \sum c_k g_k$. $\blacksquare$

This is why, in Example 6, we found three separate $y_p$'s and added them. It is also why Fourier series is so powerful for PDEs: we can solve for each Fourier component separately, then superpose.

## § 6 — Choosing the Right Method

|Condition|Method to Use|Why|
|---|---|---|
|Constant coefficients, $g$ is polynomial $\times$ exp $\times$ trig|Undetermined coefficients|Fast, systematic, no integration needed|
|Constant coefficients, $g$ is complex (e.g., $\sec x$, $\ln x$, $1/(1+x^2)$)|Variation of parameters|Handles any $g$|
|Variable coefficients (non-Euler) with known ${y_1, y_2}$|Variation of parameters|Only method that applies|
|Euler–Cauchy equation|Reduce to constant coefficients via $x = e^t$, then either method||
|Need explicit operator algebra, complex $g$ decomposes nicely|Annihilator method|More systematic than guessing|
|Many superposed forcing terms|Superposition + undetermined coefficients|One trial per term|

**Common mistake to avoid:** Choosing undetermined coefficients when $g$ has a resonant term, forgetting to multiply by $x^s$. Always check whether each component of the trial solves the homogeneous equation.

## § 7 — The Full Solution Pipeline

For any nonhomogeneous linear second order ODE:

```
Step 1: Write in standard form y'' + p(x)y' + q(x)y = g(x)

Step 2: Solve L[y] = 0
        → Find characteristic roots (constant coeff)
        → Or use reduction of order, series, etc.
        → Get fundamental set {y₁, y₂}

Step 3: Find one particular solution yₚ
        → Undetermined coefficients (if applicable)
        → Variation of parameters (always works)

Step 4: Write y = c₁y₁ + c₂y₂ + yₚ

Step 5: If IVP/BVP given, apply conditions to find c₁, c₂
```

## § 8 — The RLC Circuit: The Canonical Physical Model

The equation:

$$L\ddot{q} + R\dot{q} + \frac{1}{C}q = \mathcal{E}(t)$$

where $q(t)$ is charge, $\mathcal{E}(t)$ is the EMF source, models an **RLC circuit**.

Rewriting: $\ddot{q} + \frac{R}{L}\dot{q} + \frac{1}{LC}q = \frac{\mathcal{E}(t)}{L}$.

This is $L[q] = g(t)$ with $p = R/L$, $q_{\text{coeff}} = 1/LC$, $g = \mathcal{E}/L$.

**Forced oscillations with $\mathcal{E}(t) = E_0\cos\omega t$:**

The particular solution represents the **steady-state response**: $$q_p = \frac{E_0/L}{\sqrt{(\omega_0^2 - \omega^2)^2 + (R\omega/L)^2}}\cos(\omega t - \delta)$$

where $\omega_0 = 1/\sqrt{LC}$ and $\delta = \arctan\left(\frac{R\omega/L}{\omega_0^2 - \omega^2}\right)$.

The homogeneous solution $q_h$ (transient) decays to zero if $R > 0$. The particular solution $q_p$ (steady state) persists. In the long run, only $y_p$ matters — the particular solution is the physically observable behavior.

```
  q(t)
   |     Transient (yₕ)        Total response (y = yₕ + yₚ)
   |   /\/\/\/\/\___            /\/\/\/\___________
   |  /               ‾‾‾---   /                  ~~~~~~~~~~
   |_/_________________________________> t
                              Steady state (yₚ) persists
```

## § 9 — Connection to AI, Machine Learning, and Deep Learning

### 9.1 The Homogeneous/Particular Split in Representation Learning

The structure theorem says every solution is $y = y_h + y_p$:

- $y_h$: lives in the **null space** of $L$ — determined by initial conditions, independent of the forcing.
- $y_p$: the **particular response** to the forcing — depends entirely on $g$, not on initial conditions.

In deep learning, this decomposition appears in the study of **neural network representations**:

- The **null space** of the linear map at each layer corresponds to directions in weight space that do not affect the output. Training ignores these directions — they are the "homogeneous" part.
- The **particular solution** corresponds to the minimum-norm solution found by gradient descent when the loss is underdetermined. The **Moore–Penrose pseudoinverse** gives this minimum-norm solution — the direct linear algebra analogue of variation of parameters.

### 9.2 Undetermined Coefficients and Basis Functions in ML

The idea of undetermined coefficients — guessing a function form and solving for coefficients — is the philosophy behind:

**Basis function regression:** Model $f(x) = \sum_{k=1}^n w_k \phi_k(x)$ where $\phi_k$ are fixed basis functions (polynomials, RBFs, Fourier modes). Fitting $w_k$ to data is undetermined coefficients for the statistical operator $\mathcal{L}[f] = f - \text{data}$.

**Kernel methods (SVMs, Gaussian Processes):** The kernel is a Green's function. The GP prediction $\mu(x) = \int K(x, x') \alpha(x'),dx'$ is variation of parameters — the forcing $g = \alpha(x')$ (the support vectors or posterior weights) is convolved with the Green's function $K(x, x')$ of the kernel operator.

**Neural tangent kernel (NTK):** In the infinite-width limit, training a neural network is governed by the NTK, and the solution to the training dynamics is:

$$\mathbf{f}(t) = \mathbf{f}(\infty) + e^{-\Theta t}(\mathbf{f}(0) - \mathbf{f}(\infty))$$

This is **exactly** the structure theorem: $\mathbf{f}(\infty)$ is the particular solution (the target), and $e^{-\Theta t}(\mathbf{f}(0) - \mathbf{f}(\infty))$ is the homogeneous part (decaying transient from initial conditions). Training is convergence of the transient to zero.

### 9.3 Forcing and Fine-Tuning: Transfer Learning as Nonhomogeneous ODE

Pretrain a model on a large dataset (find $y_h$ — the homogeneous solution, the "soul" of the model). Fine-tune on a small task-specific dataset (add $y_p$ — the particular solution, the task-specific response to the "forcing" of the new data).

The total model $= y_h + y_p$ (pretrained knowledge + task adaptation). The structure theorem says: $y_p$ is not unique — any particular solution works, and the constants in $y_h$ absorb the difference. In practice, different fine-tuning methods (LoRA, adapters, full fine-tuning) correspond to different choices of $y_p$. LoRA, in particular, restricts $y_p$ to be **low-rank** — a structured ansatz, analogous to undetermined coefficients (guess a restricted form, solve for coefficients).

### 9.4 Resonance and Adversarial Examples

**Resonance** in the ODE occurs when the forcing frequency matches a natural frequency of the homogeneous equation — the response grows without bound.

The analogue in neural networks is **adversarial examples**: inputs that are precisely tuned to match the "natural frequencies" of the network's decision boundary, producing catastrophically wrong outputs with imperceptible input changes. Just as resonance is not a bug in the ODE (it is mathematically correct behavior) but is catastrophic physically, adversarial sensitivity is a mathematically correct consequence of the network's geometry, but practically disastrous.

More formally: the Hessian of the loss $\nabla^2 \mathcal{L}$ has eigenvectors — directions of sharp curvature. Perturbing inputs in these directions (the "resonant modes") causes the largest change in the output. Adversarial attacks are, in this sense, forcing at the resonant frequency of the network operator.

### 9.5 Variation of Parameters and Attention

Variation of parameters computes:

$$y_p(x) = \int G(x, t) g(t),dt$$

The Green's function $G(x,t)$ weights the forcing at $t$ by its influence at $x$. **Self-attention** computes:

$$\text{Output}_i = \sum_j A_{ij} V_j$$

where $A_{ij} = \text{softmax}(Q_i \cdot K_j^T / \sqrt{d})$ weights the values $V_j$ by the relevance of position $j$ to position $i$. This is a **discrete, learned variation of parameters**: $A_{ij}$ is a data-dependent Green's function, $V_j$ is the "forcing" at position $j$, and the output is the "particular solution" at position $i$.

The key difference: in classical VoP, $G(x,t)$ is determined by the differential operator. In attention, $A_{ij}$ is **learned from data** — the network discovers its own Green's function.

### 9.6 Superposition and Ensemble Methods

The principle of superposition — $L[y] = \sum g_k \Rightarrow y_p = \sum y_{p_k}$ — is the theoretical basis of **ensemble methods** in machine learning:

- **Gradient boosting:** Sequentially solve for $y_{p_k}$ where each $g_k$ is the residual from the previous model. The total model $\sum y_{p_k}$ is the superposition of particular solutions to residual forcing terms.
- **Mixture of Experts (MoE):** Each expert handles a "component" of the forcing function (the input distribution). The gating network superimposes expert outputs — a learned superposition of particular solutions.
- **Random forests:** Each tree is a particular solution to a noisy version of the forcing (bootstrap sample). The ensemble average superimposes, reducing variance while maintaining consistency.

### 9.7 Steady State, Transients, and Training Dynamics

The ODE decomposition $y = y_h + y_p$ with $y_h$ decaying (transient) and $y_p$ persisting (steady state) is mirrored in neural network training:

- **Early training (transient phase):** The network is far from the optimum. The homogeneous component (the response to initial conditions — random initialization) dominates. The loss decreases rapidly.
- **Late training (steady state):** The transient has decayed. The network is near $y_p$ — the particular solution corresponding to the data. The loss plateaus. Further training does not help.

**Learning rate warmup** delays the forcing ramp-up, letting the optimizer "find its feet" — analogous to turning on the forcing $g(x)$ gradually to avoid resonant transients. **Cosine annealing** mirrors the smooth approach to steady state of a damped forced oscillator.

## § 10 — Common Mistakes and Pitfalls

|Mistake|What goes wrong|Fix|
|---|---|---|
|Not checking for resonance in undetermined coefficients|Substituting a homogeneous solution as trial gives $0 = g$; coefficients are undetermined|Check each component of the trial against every homogeneous solution; multiply by $x^s$|
|Forgetting the constraint $u_1'y_1 + u_2'y_2 = 0$ in VoP derivation|Without the constraint, the system is underdetermined and the algebra doesn't simplify|The constraint is a choice — it is the price for having only first derivatives of $u_1, u_2$ in the equations|
|Using the non-standard form in VoP|VoP formula assumes the leading coefficient is 1|Always divide by $P(x)$ first; $g(x)$ in the formula is $g/P$, not the original right-hand side|
|Adding particular solutions that differ by a homogeneous solution|This gives a "wrong" looking $y_p$ but is actually fine|Any particular solution works; the constants $c_1, c_2$ absorb the difference|
|Applying undetermined coefficients to variable-coefficient equations|The method fails — differentiating $p(x)y$ introduces new functions that cannot be matched|Use variation of parameters or reduce to a solvable form|
|Thinking the particular solution must satisfy any boundary/initial condition|$y_p$ is just one solution; $c_1, c_2$ are determined by the conditions|Apply BCs/ICs to the full solution $y_h + y_p$, not to $y_p$ alone|

## § 11 — Summary

### The Master Formula

$$\boxed{y = \underbrace{c_1 y_1 + c_2 y_2}_{y_h} + \underbrace{y_p}_{\text{particular}}}$$

where ${y_1, y_2}$ is the fundamental set of $L[y] = 0$ and $y_p$ is any solution of $L[y] = g$.

### Finding $y_p$: Decision Tree

```
Is the equation constant-coefficient?
│
├─ YES: Is g(x) a polynomial × exponential × trig product?
│       │
│       ├─ YES: Undetermined Coefficients
│       │         → Identify trial form from table
│       │         → Check for resonance → multiply by xˢ
│       │         → Substitute and match coefficients
│       │
│       └─ NO:  Variation of Parameters
│
└─ NO:  Do you have the fundamental set {y₁, y₂}?
        │
        ├─ YES: Variation of Parameters
        │         u₁' = -y₂g/W,  u₂' = y₁g/W
        │         yₚ = u₁y₁ + u₂y₂
        │
        └─ NO:  Find {y₁, y₂} first (series, reduction of order, etc.)
```

### The Wronskian Reminder

$$W(y_1, y_2)(x) = y_1 y_2' - y_1' y_2 \neq 0 \quad \text{(Abel: either always 0 or never 0)}$$
