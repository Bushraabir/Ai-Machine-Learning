## 0. What Does "Symmetry" Mean, Precisely?

Informally, a symmetry is a transformation that **leaves something unchanged**. A square looks the same after a 90° rotation. The sine curve looks the same after a shift of $2\pi$.

Formally, a **symmetry of a function** $f$ is a transformation $T$ of the domain such that: $$f(T(x)) = f(x) \quad \text{for all } x \text{ in the domain}$$

Different choices of $T$ give different symmetries. The two classical ones in single-variable calculus are:

|Transformation|Symmetry type|Condition|
|---|---|---|
|$T(x) = -x$ (reflect across $y$-axis)|**Even**|$f(-x) = f(x)$|
|$T(x) = -x$ + sign flip|**Odd**|$f(-x) = -f(x)$|
|$T(x) = x + p$ (shift by $p$)|**Periodic**|$f(x+p) = f(x)$|

Each of these is a _theorem-generator_: once you know a function has a symmetry, you get facts for free.

## 1. Even Functions — Symmetry About the $y$-Axis

**Definition.** $f$ is **even** if its domain is symmetric about $0$ and: $$f(-x) = f(x) \quad \text{for all } x \text{ in the domain}$$

**Geometric meaning.** The graph is symmetric about the $y$-axis: the left half is the mirror image of the right half.

**Canonical examples:** $$f(x) = x^2, \quad f(x) = x^4, \quad f(x) = \cos x, \quad f(x) = |x|, \quad f(x) = e^{-x^2}$$

**The Gaussian $e^{-x^2}$:**

```
         *
        ***
       *****
      *******
  **************
──────────────────→
      0
```

Even: the bell curve is perfectly symmetric. This is not a coincidence — it encodes the fact that the normal distribution treats $+\delta$ and $-\delta$ deviations identically.

### 1.1 Integral Consequence

If $f$ is even and integrable on $[-a, a]$: $$\int_{-a}^{a} f(x), dx = 2\int_0^a f(x), dx$$

**Proof.** Split the integral and substitute $x \mapsto -x$ on the left piece: $$\int_{-a}^{0} f(x), dx \overset{x = -u}{=} \int_{a}^{0} f(-u)(-du) = \int_0^a f(-u), du = \int_0^a f(u), du$$ Adding to $\int_0^a f(x), dx$ gives the result. $\square$

**Example.** $\int_{-2}^{2} (x^4 + \cos x), dx = 2\int_0^2 (x^4 + \cos x), dx = 2\left[\frac{x^5}{5} + \sin x\right]_0^2 = 2\left(\frac{32}{5} + \sin 2\right)$

Half the work — literally.

## 2. Odd Functions — Symmetry About the Origin

**Definition.** $f$ is **odd** if its domain is symmetric about $0$ and: $$f(-x) = -f(x) \quad \text{for all } x \text{ in the domain}$$

**Geometric meaning.** The graph has **180° rotational symmetry** about the origin: rotating the graph by $\pi$ about $(0,0)$ maps it to itself.

**Canonical examples:** $$f(x) = x, \quad f(x) = x^3, \quad f(x) = \sin x, \quad f(x) = \tan x, \quad f(x) = \sinh x$$

**Special value.** If $f$ is odd and $0$ is in its domain, then: $$f(0) = f(-0) = -f(0) \implies 2f(0) = 0 \implies f(0) = 0$$ _Every odd function (with $0$ in its domain) passes through the origin._ This is a theorem, not a coincidence.

### 2.1 Integral Consequence

If $f$ is odd and integrable on $[-a, a]$: $$\int_{-a}^{a} f(x), dx = 0$$

**Proof.** By the same substitution as above, $\int_{-a}^{0} f(x), dx = -\int_0^a f(u), du$. These cancel. $\square$

**Example.** $\int_{-\pi}^{\pi} \sin(x)\cos^2(x), dx = 0$ — because $\sin(x)\cos^2(x)$ is odd (odd × even = odd). No computation needed.

## 3. The Decomposition Theorem

**Theorem.** Every function $f$ defined on a domain symmetric about $0$ can be written _uniquely_ as the sum of an even function and an odd function: $$f(x) = \underbrace{\frac{f(x) + f(-x)}{2}}_{f_e(x),\ \text{even}} + \underbrace{\frac{f(x) - f(-x)}{2}}_{f_o(x),\ \text{odd}}$$

**Proof.**

_Existence._ Define $f_e$ and $f_o$ as above. Then: $$f_e(-x) = \frac{f(-x) + f(x)}{2} = f_e(x) \checkmark$$ $$f_o(-x) = \frac{f(-x) - f(x)}{2} = -f_o(x) \checkmark$$ $$f_e(x) + f_o(x) = \frac{f(x) + f(-x)}{2} + \frac{f(x) - f(-x)}{2} = f(x) \checkmark$$

_Uniqueness._ Suppose $f = g + h$ where $g$ even, $h$ odd. Then $f(-x) = g(x) - h(x)$. Adding and subtracting with $f(x) = g(x) + h(x)$ gives $g(x) = \frac{f(x)+f(-x)}{2}$ and $h(x) = \frac{f(x)-f(-x)}{2}$ uniquely. $\square$

**Example.** Decompose $f(x) = e^x$: $$f_e(x) = \frac{e^x + e^{-x}}{2} = \cosh x \quad \text{(hyperbolic cosine, even)}$$ $$f_o(x) = \frac{e^x - e^{-x}}{2} = \sinh x \quad \text{(hyperbolic sine, odd)}$$

So $e^x = \cosh x + \sinh x$ — a fundamental identity in hyperbolic functions.

### 3.1 Arithmetic of Symmetry

|Operation|Result|
|---|---|
|even $+$ even|even|
|odd $+$ odd|odd|
|even $+$ odd|neither (in general)|
|even $\times$ even|even|
|odd $\times$ odd|**even**|
|even $\times$ odd|odd|
|$f \circ g$ where $g$ even|even|
|derivative of even|odd|
|derivative of odd|even|

> The last two rows are worth proving. If $f(-x) = f(x)$, differentiate both sides: $-f'(-x) = f'(x)$, i.e., $f'(-x) = -f'(x)$. So $f'$ is odd. Symmetry is preserved — and transformed — under differentiation.

## 4. Periodic Functions — Translational Symmetry

**Definition.** $f$ is **periodic with period $p > 0$** if: $$f(x + p) = f(x) \quad \text{for all } x$$

The **fundamental period** (or _primitive period_) is the smallest such $p$.

**Examples:**

|Function|Period|
|---|---|
|$\sin x$, $\cos x$|$2\pi$|
|$\tan x$, $\cot x$|$\pi$|
|$\sin(kx)$|$2\pi / k$|
|$e^{ix}$ (complex)|$2\pi$|
|Sawtooth wave|$T$ (by definition)|

**Consequence.** If $f$ has period $p$, then $\int_a^{a+p} f(x), dx$ is the _same_ for every $a$ — integrating over any full period gives the same result.

**Proof.** Let $I(a) = \int_a^{a+p} f(x), dx$. Then $I'(a) = f(a+p) - f(a) = 0$, so $I$ is constant. $\square$

## 5. Symmetry in the Plane — Even and Odd Together

Consider the relation $x^2 + y^2 = 1$. This has _three_ symmetries:

- Reflection across $y$-axis: $(x,y) \mapsto (-x, y)$
- Reflection across $x$-axis: $(x,y) \mapsto (x, -y)$
- Rotation by $\pi$: $(x,y) \mapsto (-x, -y)$

This is not a function, but each symmetry tells us: if $(a,b)$ satisfies the equation, so do $(-a,b)$, $(a,-b)$, $(-a,-b)$.

For a **function** $y = f(x)$:

- Symmetry about $y$-axis $\Leftrightarrow$ $f$ is even
- Symmetry about $x$-axis $\Rightarrow$ _not_ a function (unless $f \equiv 0$) — because $(x, y)$ and $(x, -y)$ would both be on the graph
- Symmetry about origin $\Leftrightarrow$ $f$ is odd

## 6. Worked Examples

**Example 1.** Determine if $f(x) = \dfrac{x^2}{x^4 + 1}$ is even, odd, or neither.

$$f(-x) = \frac{(-x)^2}{(-x)^4 + 1} = \frac{x^2}{x^4 + 1} = f(x)$$

**Even.** (Numerator is $x^2$, even; denominator is $x^4 + 1$, even; even/even = even.)

---

**Example 2.** Determine if $g(x) = x^3 \sin x$ is even, odd, or neither.

$$g(-x) = (-x)^3 \sin(-x) = (-x^3)(-\sin x) = x^3 \sin x = g(x)$$

**Even.** (Odd × odd = even — consistent with our table.)

---

**Example 3.** Evaluate $\displaystyle\int_{-1}^{1} \left(x^5 + 3x^3 - x\right) e^{x^2}, dx$.

Let $h(x) = (x^5 + 3x^3 - x)e^{x^2}$.

- $x^5 + 3x^3 - x$ is **odd** (sum of odd powers).
- $e^{x^2}$ is **even** (since $(-x)^2 = x^2$).
- odd × even = **odd**.

Therefore $\displaystyle\int_{-1}^{1} h(x), dx = 0$.

No antiderivative needed.

---

**Example 4.** Decompose $f(x) = x^2 + x + 1$ into even and odd parts.

$$f_e(x) = \frac{(x^2+x+1) + (x^2-x+1)}{2} = \frac{2x^2+2}{2} = x^2 + 1$$

$$f_o(x) = \frac{(x^2+x+1) - (x^2-x+1)}{2} = \frac{2x}{2} = x$$

Check: $f_e(x) + f_o(x) = (x^2+1) + x = x^2 + x + 1 = f(x)$. $\checkmark$

## 7. Symmetry in Several Variables (MIT 18.02)

For $f: \mathbb{R}^2 \to \mathbb{R}$, symmetry generalizes:

- **Even in $x$**: $f(-x, y) = f(x, y)$ — graph symmetric about $yz$-plane
- **Even in $y$**: $f(x, -y) = f(x, y)$ — graph symmetric about $xz$-plane
- **Radially symmetric**: $f(x,y)$ depends only on $r = \sqrt{x^2+y^2}$, i.e., $f(x,y) = g(r)$

**Example.** $f(x,y) = e^{-(x^2+y^2)} = e^{-r^2}$ is radially symmetric. Its graph is a symmetric bell surface. The **Gaussian integral**: $$\int_{-\infty}^{\infty}\int_{-\infty}^{\infty} e^{-(x^2+y^2)}, dx, dy = \pi$$

is computed by switching to polar coordinates — a trick made _possible_ by the radial symmetry. From this, we extract $\int_{-\infty}^{\infty} e^{-x^2}, dx = \sqrt{\pi}$.

**Double integral consequence of symmetry.** If the domain $D$ is symmetric about the $y$-axis and $f$ is odd in $x$: $$\iint_D f(x,y), dA = 0$$

Same logic, same power.

## 8. Connections to AI / Machine Learning

### 8.1 Symmetric Activation Functions

|Activation|Even/Odd|Consequence|
|---|---|---|
|$\tanh(x)$|Odd|$\tanh(0) = 0$; gradients symmetric; zero-centered outputs|
|$\text{sigmoid}(x)$|Neither|Outputs in $(0,1)$; not zero-centered → gradient issues|
|$\text{ReLU}(x)$|Neither|Asymmetric; dying neuron problem for $x < 0$|
|$\sin(x)$ (SIREN networks)|Odd|Used in implicit neural representations|

The **zero-centered** property of odd activations (like $\tanh$) is practically important: if activations are not zero-centered, gradients in earlier layers tend to be all-positive or all-negative, causing zig-zag gradient updates. This is called the **zero-mean activation problem** and was a key reason $\tanh$ was preferred over sigmoid in early deep learning.

### 8.2 Symmetry in the Loss Function

The **Mean Squared Error** loss: $$\mathcal{L}(\theta) = \frac{1}{N}\sum_{i=1}^N (y_i - \hat{y}_i)^2$$

is **even** as a function of the residual $r_i = y_i - \hat{y}_i$: it treats overestimation and underestimation identically. This is a _modeling choice_ — you might not want that.

The **Huber loss** and **MAE** are also even in the residual. An **asymmetric loss** (neither even nor odd) would be used when, e.g., false negatives are more costly than false positives in a medical diagnostic setting.

### 8.3 The Even-Odd Decomposition and Fourier Analysis

The decomposition $f = f_e + f_o$ is the _seed_ of **Fourier analysis**:

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \underbrace{a_n \cos(nx)}_{\text{even terms}} + \underbrace{b_n \sin(nx)}_{\text{odd terms}}$$

The even part of $f$ determines the $a_n$ (cosine) coefficients; the odd part determines the $b_n$ (sine) coefficients. In **signal processing** and **sequence modeling** (Transformers), Fourier features and positional encodings exploit this decomposition:

- **RoPE (Rotary Position Embedding)** in modern LLMs encodes position using rotation matrices built from $\cos$ and $\sin$ — i.e., from even and odd functions.
- **Fourier Features** in kernel methods: random Fourier features approximate shift-invariant (translationally symmetric) kernels.

### 8.4 Equivariance and Invariance — The Modern Generalization

In modern deep learning, symmetry is formalized as:

- **Invariance**: $f(T(x)) = f(x)$ — the output doesn't change under transformation $T$. (Even functions are $\mathbb{Z}_2$-invariant.)
- **Equivariance**: $f(T(x)) = T'(f(x))$ — the output transforms _consistently_ with the input.

**Convolutional Neural Networks (CNNs)** are equivariant to _translation_: if you shift the image, the feature map shifts correspondingly. This is translational symmetry built into the architecture.

**Graph Neural Networks (GNNs)** are equivariant to _permutation_ of nodes.

**$\text{SE}(3)$-equivariant networks** (used in protein structure prediction, e.g., AlphaFold) are equivariant to 3D rotation and translation — they respect the physical symmetry of molecules.

> The mathematical foundation: the functions $f: X \to Y$ that respect the action of a group $G$ form a rich structure. **Geometric deep learning** (Bronstein et al., 2021) unifies CNNs, GNNs, Transformers, and more under this framework. Every major architectural innovation in deep learning can be understood as _encoding the right symmetry_ into the model.

## 9. Summary

```
SYMMETRY OF f
│
├── REFLECTIVE SYMMETRY
│   ├── About y-axis:  f(-x) = f(x)   → EVEN function
│   └── About origin:  f(-x) = -f(x)  → ODD function
│
├── TRANSLATIONAL SYMMETRY
│   └── f(x+p) = f(x)                 → PERIODIC, period p
│
├── RADIAL SYMMETRY (ℝ²)
│   └── f(x,y) = g(√(x²+y²))         → depends only on r
│
└── GENERAL (Group Theory)
    ├── f(T(x)) = f(x)                → T-INVARIANT
    └── f(T(x)) = T'(f(x))           → T-EQUIVARIANT
                                         (CNN, GNN, AlphaFold)
```

**Key theorems to remember:**

1. $\int_{-a}^{a} f_{\text{odd}}, dx = 0$
2. $\int_{-a}^{a} f_{\text{even}}, dx = 2\int_0^a f_{\text{even}}, dx$
3. Every $f$ on a symmetric domain decomposes uniquely as $f_e + f_o$
4. Derivative of even = odd; derivative of odd = even
5. $f(0) = 0$ for every odd function with $0$ in its domain
