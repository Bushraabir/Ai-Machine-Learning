> _"A piecewise function is not a collection of functions. It is one function that happens to be described by different rules on different parts of its domain. The key question is always: what happens at the boundaries?"_

---

## 0. Motivation — Why Piecewise?

Sometimes a single algebraic formula cannot (or should not) capture the full behavior of a rule. Nature, economics, and computation are full of processes that _change their character_ at certain thresholds:

- A tax bracket system: tax rate changes at income thresholds.
- A spring: restoring force is linear until the spring is overstretched, then the model breaks.
- A neuron: fires only when input exceeds a threshold.
- Absolute value: reflects negative inputs, leaves positive ones alone.

Piecewise functions are the honest mathematical language for these situations.

---

## 1. Formal Definition

**Definition.** A **piecewise defined function** on a domain $D \subseteq \mathbb{R}$ is a function $f: D \to \mathbb{R}$ specified by:

$$f(x) = \begin{cases} f_1(x) & \text{if } x \in D_1 \ f_2(x) & \text{if } x \in D_2 \ \vdots & \vdots \ f_n(x) & \text{if } x \in D_n \end{cases}$$

where $D_1, D_2, \ldots, D_n$ are _disjoint_ subsets of $D$ whose union is $D$:

$$D = D_1 \cup D_2 \cup \cdots \cup D_n, \qquad D_i \cap D_j = \emptyset \text{ for } i \neq j$$

Each $f_k$ is called a **branch** of $f$, and the points where the domain partitions — where we transition from one branch to the next — are called **breakpoints** (or **transition points**).

> **Crucial point.** $f$ is _one_ function. It has one domain, one graph, one value at each input. The piecewise notation is about _how we describe it_, not about its unity as a mathematical object.

---

## 2. Canonical Examples

### 2.1 Absolute Value

$$|x| = \begin{cases} x & \text{if } x \geq 0 \ -x & \text{if } x < 0 \end{cases}$$

- **Domain:** $\mathbb{R}$. **Range:** $[0, \infty)$.
- **Breakpoint:** $x = 0$.
- At the breakpoint: both branches agree — $f(0) = 0$ from either side.
- **Graph:** V-shape with vertex at the origin.

**Key property:** $$|x| = \sqrt{x^2} \qquad \text{(alternative closed form)}$$

This is a non-trivial identity. It shows that $|x|$ has an algebraic representation — but the piecewise form is often more useful for analysis.

---

### 2.2 The Sign Function

$$\text{sgn}(x) = \begin{cases} 1 & \text{if } x > 0 \ 0 & \text{if } x = 0 \ -1 & \text{if } x < 0 \end{cases}$$

- **Domain:** $\mathbb{R}$. **Range:** ${-1, 0, 1}$.
- **Breakpoint:** $x = 0$.
- Note: $|x| = x \cdot \text{sgn}(x)$ for $x \neq 0$.

---

### 2.3 The Heaviside Step Function

$$H(x) = \begin{cases} 0 & \text{if } x < 0 \ 1 & \text{if } x \geq 0 \end{cases}$$

- **Domain:** $\mathbb{R}$. **Range:** ${0, 1}$.
- Named after Oliver Heaviside; fundamental in signal processing and differential equations.
- Models "switching on" at $x = 0$.
- Any piecewise constant function can be written as a linear combination of shifted Heaviside functions.

> **Observation.** $\text{ReLU}(x) = x \cdot H(x)$. The Heaviside function is the "derivative" of ReLU in the distributional sense: $H'(x) = \delta(x)$, the Dirac delta. This connection — between step functions, ramp functions, and impulses — is the language of distribution theory.

---

### 2.4 ReLU (Rectified Linear Unit)

$$\text{ReLU}(x) = \max(x, 0) = \begin{cases} x & \text{if } x > 0 \ 0 & \text{if } x \leq 0 \end{cases}$$

- **Domain:** $\mathbb{R}$. **Range:** $[0, \infty)$.
- **Breakpoint:** $x = 0$.
- Continuous everywhere. Not differentiable at $x = 0$.

```
f(x) = ReLU(x)

    |          /
    |         /
    |        /
    |       /
    |      /
    |     /
    |    /
    |___/____________
   -3 -2 -1  0  1  2  3
```

> **Why not sigmoid?** Sigmoid $\sigma(x) = \frac{1}{1+e^{-x}}$ is smooth everywhere but $\sigma'(x) \to 0$ as $|x| \to \infty$. In deep networks, multiplying many near-zero derivatives during backprop yields vanishing gradients. ReLU's derivative is exactly $1$ for $x > 0$ — gradients flow unchanged. This is why virtually every modern deep network uses ReLU or a variant in its hidden layers.

---

### 2.5 A Generic "Gluing" Example

$$f(x) = \begin{cases} x^2 + 1 & \text{if } x \leq 0 \ 2x + 1 & \text{if } x > 0 \end{cases}$$

- **Breakpoint:** $x = 0$.
- Left branch value at $0$: $f(0^-) = 0^2 + 1 = 1$.
- Right branch limit at $0$: $\lim_{x \to 0^+} (2x+1) = 1$.
- Actual value: $f(0) = 1$ (left branch applies).

Both the value and the limit agree: $f$ is **continuous** at $x = 0$.

Now compare with:

$$g(x) = \begin{cases} x + 3 & \text{if } x < 1 \ x^2 & \text{if } x \geq 1 \end{cases}$$

At $x = 1$: left limit $= 1 + 3 = 4$, right branch $= 1^2 = 1$. These disagree — **jump discontinuity** at $x = 1$.

## 3. Continuity at a Breakpoint

This is the _central analytical question_ for piecewise functions. Let $c$ be a breakpoint.

**Definition (Continuity at $c$).** $f$ is continuous at $c$ if and only if:

$$\lim_{x \to c^-} f(x) = \lim_{x \to c^+} f(x) = f(c)$$

All three must be equal: the left-hand limit, the right-hand limit, and the actual value.

**Procedure:** Given a piecewise function with breakpoint $c$:

1. Compute $L^- = \lim_{x \to c^-} f(x)$ using the branch defined left of $c$.
2. Compute $L^+ = \lim_{x \to c^+} f(x)$ using the branch defined right of $c$.
3. Determine $f(c)$ from whichever branch includes $c$ in its domain.
4. $f$ is continuous at $c$ iff $L^- = L^+ = f(c)$.

**Example.** For $|x|$ at $c = 0$:

- $L^- = \lim_{x \to 0^-} (-x) = 0$.
- $L^+ = \lim_{x \to 0^+} x = 0$.
- $f(0) = 0$.
- All equal $\Rightarrow$ continuous at $0$. ✓

**Example.** For $H(x)$ (Heaviside) at $c = 0$:

- $L^- = \lim_{x \to 0^-} 0 = 0$.
- $L^+ = \lim_{x \to 0^+} 1 = 1$.
- $L^- \neq L^+ \Rightarrow$ **jump discontinuity** at $0$.

---

## 4. Differentiability at a Breakpoint

Even if $f$ is continuous at the breakpoint $c$, it may fail to be differentiable there.

**Definition.** $f$ is differentiable at $c$ if and only if the following limit exists:

$$f'(c) = \lim_{h \to 0} \frac{f(c+h) - f(c)}{h}$$

For a piecewise function, this requires checking the **one-sided derivatives**:

$$f'_-(c) = \lim_{h \to 0^-} \frac{f(c+h) - f(c)}{h}, \qquad f'_+(c) = \lim_{h \to 0^+} \frac{f(c+h) - f(c)}{h}$$

$f$ is differentiable at $c$ iff $f'_-(c) = f'_+(c)$ (and the common value is $f'(c)$).

**Theorem.** Differentiability $\Rightarrow$ Continuity. The converse fails.

**Example: $|x|$ at $c = 0$.**

$$f'_-(0) = \lim_{h \to 0^-} \frac{|h| - 0}{h} = \lim_{h \to 0^-} \frac{-h}{h} = -1$$

$$f'_+(0) = \lim_{h \to 0^+} \frac{|h| - 0}{h} = \lim_{h \to 0^+} \frac{h}{h} = 1$$

$f'_-(0) = -1 \neq 1 = f'_+(0)$. Therefore $|x|$ is **not differentiable** at $0$, despite being continuous there.

The graph has a **corner** (kink) at the origin — the left tangent slope is $-1$, the right tangent slope is $+1$.

**Example: ReLU at $c = 0$.**

$$\text{ReLU}'_-(0) = \lim_{h \to 0^-} \frac{0 - 0}{h} = 0$$

$$\text{ReLU}'_+(0) = \lim_{h \to 0^+} \frac{h - 0}{h} = 1$$

Again $0 \neq 1$: **not differentiable** at $0$. Same kink, different slopes.

> **Deep Learning Consequence.** Since ReLU is not differentiable at $0$, the gradient $\frac{\partial}{\partial x}\text{ReLU}(x)$ is undefined there in the classical sense. In practice, frameworks like PyTorch and TensorFlow assign the **subgradient** $0$ (or sometimes $0.5$) at $x = 0$. This is justified rigorously by **convex analysis**: for a convex function, the subdifferential $\partial f(x)$ at a point of non-differentiability is an interval, and we pick any element. For ReLU, $\partial \text{ReLU}(0) = [0, 1]$.

## 5. Constructing Piecewise Functions: The Matching Conditions

A common exam and problem-set task: given a piecewise function with a free parameter, find the parameter value that makes $f$ continuous (or differentiable) at the breakpoint.

**Example.** Find $a$ and $b$ so that:

$$f(x) = \begin{cases} ax + b & \text{if } x < 1 \ x^2 + 1 & \text{if } 1 \leq x < 3 \ 2bx - a & \text{if } x \geq 3 \end{cases}$$

is continuous on $\mathbb{R}$.

**At $x = 1$:** $$L^- = a(1) + b = a + b$$ $$L^+ = (1)^2 + 1 = 2, \quad f(1) = 2$$ Continuity: $a + b = 2$. — (I)

**At $x = 3$:** $$L^- = (3)^2 + 1 = 10$$ $$L^+ = f(3) = 2b(3) - a = 6b - a$$ Continuity: $6b - a = 10$. — (II)

Solving (I) and (II): from (I), $a = 2 - b$. Substituting into (II): $$6b - (2 - b) = 10 \implies 7b - 2 = 10 \implies b = \frac{12}{7}, \quad a = 2 - \frac{12}{7} = \frac{2}{7}$$

## 6. Integration of Piecewise Functions

Integration respects the partition of the domain.

**Theorem.** If $f$ is piecewise continuous on $[a, b]$ with breakpoints $a = c_0 < c_1 < \cdots < c_n = b$, then:

$$\int_a^b f(x), dx = \sum_{k=0}^{n-1} \int_{c_k}^{c_{k+1}} f_k(x), dx$$

provided each integral on the right exists.

**Example.** Compute $\int_{-2}^{3} |x|, dx$.

Split at the breakpoint $x = 0$: $$\int_{-2}^{3} |x|, dx = \int_{-2}^{0} (-x), dx + \int_{0}^{3} x, dx$$ $$= \left[-\frac{x^2}{2}\right]_{-2}^{0} + \left[\frac{x^2}{2}\right]_{0}^{3} = \left(0 - (-2)\right) + \left(\frac{9}{2} - 0\right) = 2 + \frac{9}{2} = \frac{13}{2}$$

## 7. A Menagerie of Important Piecewise Functions

### 7.1 The Floor and Ceiling Functions

**Floor:** $\lfloor x \rfloor$ = greatest integer $\leq x$.

$$\lfloor x \rfloor = n \quad \text{if } n \leq x < n+1, \quad n \in \mathbb{Z}$$

**Ceiling:** $\lceil x \rceil$ = least integer $\geq x$.

- Both are piecewise constant, with jump discontinuities at every integer.
- Used in: hashing, bucket indexing, stochastic rounding in quantized neural networks.

### 7.2 Leaky ReLU

$$\text{LReLU}_\alpha(x) = \begin{cases} x & \text{if } x > 0 \ \alpha x & \text{if } x \leq 0 \end{cases}, \quad 0 < \alpha \ll 1$$

Typically $\alpha = 0.01$. The motivation: ReLU neurons can "die" — once a neuron's input is consistently negative, its gradient is exactly $0$ and weights never update. Leaky ReLU gives a small nonzero gradient for negative inputs, preventing dead neurons.

At $x = 0$: continuous (both branches give $0$), not differentiable (slopes $1$ vs $\alpha$).

### 7.3 ELU (Exponential Linear Unit)

$$\text{ELU}_\alpha(x) = \begin{cases} x & \text{if } x > 0 \ \alpha(e^x - 1) & \text{if } x \leq 0 \end{cases}, \quad \alpha > 0$$

At $x = 0$: both branches give $0$ (continuous). Left derivative: $\alpha e^0 = \alpha$. Right derivative: $1$. Unless $\alpha = 1$, not differentiable at $0$.

For $\alpha = 1$, ELU is continuous _and_ differentiable at $0$ (derivative $= 1$ from both sides). It is, however, not twice-differentiable there.

### 7.4 Huber Loss (ML Loss Function)

$$L_\delta(r) = \begin{cases} \dfrac{1}{2} r^2 & \text{if } |r| \leq \delta \ \delta \left(|r| - \dfrac{\delta}{2}\right) & \text{if } |r| > \delta \end{cases}$$

where $r = y - \hat{y}$ is the residual.

- Quadratic near zero (like MSE) — sensitive to small errors.
- Linear for large residuals (like MAE) — robust to outliers.
- Continuous at $r = \pm\delta$. Differentiable at $r = \pm\delta$ (left and right derivatives match). Not twice-differentiable there.

> The Huber loss is used in regression when the training data contains outliers. MSE ($r^2$) penalizes outliers quadratically and can dominate the loss; Huber dampens this to linear growth.

## 8. Smooth Piecewise Functions via Blending

Can we make a piecewise function that is _infinitely differentiable_ (smooth, $C^\infty$)?

Yes — using **smooth cutoff functions** (bump functions). But this requires functions like:

$$\phi(x) = \begin{cases} e^{-1/x} & \text{if } x > 0 \ 0 & \text{if } x \leq 0 \end{cases}$$

**Claim:** $\phi$ is $C^\infty$ at $x = 0$, and $\phi^{(n)}(0) = 0$ for all $n \geq 0$.

_Proof sketch._ By induction, for $x > 0$, $\phi^{(n)}(x) = p_n(1/x) e^{-1/x}$ for some polynomial $p_n$. As $x \to 0^+$, $e^{-1/x} \to 0$ faster than any polynomial grows: $\lim_{t \to \infty} t^k e^{-t} = 0$ for all $k$. Hence $\phi^{(n)}(0^+) = 0 = \phi^{(n)}(0)$.

This function is smooth but **not analytic** at $0$ — its Taylor series is identically zero, yet the function is not. This example is the prototype for the fact that $C^\infty \supsetneq C^\omega$ (smooth does not imply analytic) — a cornerstone of differential geometry and PDE theory.

## 9. Piecewise Linear Functions and Neural Networks

**Definition.** A function $f: \mathbb{R} \to \mathbb{R}$ is **piecewise linear** if there exist $a = c_0 < c_1 < \cdots < c_n = b$ such that $f$ is linear on each $[c_{k-1}, c_k]$.

Any piecewise linear function on $\mathbb{R}$ can be written as:

$$f(x) = \alpha x + \beta + \sum_{k=1}^{n} \gamma_k , \text{ReLU}(x - c_k)$$

for appropriate constants $\alpha, \beta, \gamma_1, \ldots, \gamma_n$. This is a fundamental theorem, and it explains why **a neural network with ReLU activations computes a piecewise linear function**.

**Theorem (Universal Approximation, informal).** A feedforward network with a single hidden layer and sufficiently many ReLU neurons can approximate any continuous function on a compact domain to arbitrary precision.

The proof strategy: approximate the target function by a piecewise linear function (possible since continuous functions on compact sets are uniformly approximable by piecewise linears), then represent that piecewise linear function as a sum of ReLU terms.

> **This is why depth matters.** A shallow network needs exponentially many neurons to represent certain functions that a deep network can represent with polynomially many. Deep networks compose piecewise linear functions — each layer introduces new breakpoints _exponentially_ in the depth. This is the mathematical core of the "depth efficiency" argument in deep learning theory.

```
FUNCTION       GRAPH SHAPE              DIFFERENTIABLE AT KINK?
──────────────────────────────────────────────────────────────────
|x|            V-shape at origin        No  (slopes: -1 vs +1)
ReLU(x)        Hockey stick at origin   No  (slopes: 0 vs 1)
LReLU(x)       Kinked line at origin    No  (slopes: α vs 1)
ELU(x),α=1     Smooth join at origin    Yes (slopes: 1 vs 1) ← C¹
Huber(r)       Quadratic + linear       Yes at ±δ              ← C¹
H(x)           Step at origin           Not even continuous!
floor(x)       Staircase                Not continuous at ℤ
```

**Key insight from the table:** The engineering choice of activation function is a choice about the _regularity_ of the function class you're learning over. ReLU → $C^0$ but not $C^1$. Smooth activations (GELU, Swish) → $C^\infty$. The choice affects optimization landscape smoothness, gradient behavior, and generalization.

## 11. GELU — The Modern Activation

For completeness: the **GELU** (Gaussian Error Linear Unit), used in transformers (GPT, BERT):

$$\text{GELU}(x) = x \cdot \Phi(x) = x \cdot \frac{1}{2}\left[1 + \text{erf}!\left(\frac{x}{\sqrt{2}}\right)\right]$$

where $\Phi$ is the CDF of the standard normal. This is _not_ piecewise-defined — it is smooth ($C^\infty$) everywhere. It approximates ReLU but with a smooth, probabilistic interpretation: it weights the input by the probability that a standard Gaussian exceeds it. The smooth transition avoids the kink at $0$ entirely.

|---|---|
|Is $f$ continuous at breakpoint $c$?|Check: $L^- = L^+ = f(c)$|
|Is $f$ differentiable at breakpoint $c$?|Check: $f'_-(c) = f'_+(c)$|
|Integrate $f$ over interval containing breakpoints|Split integral, sum pieces|
|Find parameter for continuity|Set up equation $L^- = L^+$, solve|
|Represent piecewise linear $f$ algebraically|Write as $\alpha x + \beta + \sum \gamma_k \text{ReLU}(x - c_k)$|
