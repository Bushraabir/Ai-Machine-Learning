## 0. Motivation — Why Should We Care?

Before the machinery, a word of honesty: logarithms are not a trick. They are the _inverse_ of the most natural function in analysis — the exponential — and they appear wherever growth, decay, entropy, or information lives.

In 1614, John Napier introduced logarithms to turn **multiplication into addition** (a computational miracle before computers). Today they appear in:

- **Information theory**: Shannon entropy $H = -\sum p_i \log p_i$
- **Signal processing**: decibels, Fourier analysis
- **Physics**: thermodynamic entropy $S = k_B \ln \Omega$
- **Machine learning / deep learning**: cross-entropy loss, log-likelihood, softmax, log-sum-exp trick — _we will return to all of these_

## 1. Definition — The Right One

There are two equivalent entry points. I prefer the **integral definition** because it requires no prior knowledge of $e$ and makes differentiability immediate.

### 1.1 The Natural Logarithm via an Integral

$$\boxed{\ln x ;:=; \int_1^x \frac{1}{t},dt, \qquad x > 0}$$

**Why is this a good definition?**

We _want_ a function $L$ satisfying $L'(x) = 1/x$ with $L(1) = 0$. The First Fundamental Theorem of Calculus hands us exactly that. Every property of $\ln$ — its domain, sign, monotonicity, the logarithm laws — follows from this integral and from properties of Riemann integrals.

> **Geometric picture.** $\ln x$ is the _signed area_ under the hyperbola $y = 1/t$ from $t=1$ to $t=x$.
> 
> - $x > 1$: area is positive, so $\ln x > 0$.
> - $x = 1$: zero-width strip, so $\ln 1 = 0$.
> - $0 < x < 1$: we integrate _backwards_, so $\ln x < 0$.

```
y = 1/t

│ ██████
│ ████████
│ ███████████
│_________________________
0    1        x        t

Shaded area = ln x  (for x > 1)
```

### 1.2 The Number $e$ — Emerging from the Definition

Define $e$ as the unique number satisfying $\ln e = 1$, i.e.,

$$\int_1^e \frac{1}{t},dt = 1.$$

It can be shown this number exists (by IVT: $\ln$ is continuous, $\ln 1 = 0$, $\ln$ is unbounded above) and is unique (strict monotonicity). Numerically, $e \approx 2.71828\ldots$

## 2. Fundamental Properties

All of the following follow from the integral definition. **I will prove each.**

### 2.1 Domain and Range

$\ln : (0, \infty) \to \mathbb{R}$ is a **bijection**.

- **Domain** $(0,\infty)$: $1/t$ is only integrable on intervals avoiding $t=0$.
- **Range** $\mathbb{R}$:
    - $\ln x \to +\infty$ as $x \to \infty$ (compare with $\int_1^x \frac{1}{t},dt \geq \int_1^x \frac{1}{x},dt = \frac{x-1}{x}$... or use the harmonic series argument).
    - $\ln x \to -\infty$ as $x \to 0^+$ (substitute $u = 1/t$).

### 2.2 Logarithm Laws

**Theorem.** For $a, b > 0$ and $r \in \mathbb{Q}$:

$$\ln(ab) = \ln a + \ln b \tag{L1}$$ $$\ln!\left(\tfrac{a}{b}\right) = \ln a - \ln b \tag{L2}$$ $$\ln(a^r) = r\ln a \tag{L3}$$

**Proof of (L1).** Fix $a > 0$ and let $f(x) = \ln(ax)$. Then $$f'(x) = \frac{a}{ax} = \frac{1}{x} = (\ln x)'.$$ So $f(x) = \ln x + C$ for some constant. Setting $x=1$: $f(1) = \ln a = \ln 1 + C = C$. Therefore $\ln(ax) = \ln x + \ln a$. Setting $x = b$ gives (L1). $\blacksquare$

(L2) follows from (L1): $\ln(a/b) + \ln b = \ln!\left(\frac{a}{b} \cdot b\right) = \ln a$.

(L3) for integer $r$ follows by induction from (L1); the rational and real cases require a bit more care.

### 2.3 Derivative and Monotonicity

$$\frac{d}{dx}\ln x = \frac{1}{x} > 0 \quad\text{for all } x > 0.$$

So $\ln$ is **strictly increasing**. Also, $\frac{d^2}{dx^2}\ln x = -1/x^2 < 0$, so $\ln$ is **strictly concave** on $(0,\infty)$.

> **Remark (concavity matters in ML).** The strict concavity of $\ln$ is exactly why Jensen's inequality gives $\ln(\mathbb{E}[X]) \geq \mathbb{E}[\ln X]$ — a fact that underlies the **EM algorithm** and variational inference.

### 2.4 The Chain Rule with $\ln$

$$\frac{d}{dx}\ln|u(x)| = \frac{u'(x)}{u(x)}$$

This is the workhorse for integration: $\int \frac{u'}{u},dx = \ln|u| + C$.

**Example.** $\int \frac{2x}{x^2+1},dx = \ln(x^2+1) + C$. _(Here $u = x^2+1 > 0$, so no absolute value needed.)_

## 3. General Logarithms

### 3.1 Change of Base

For any base $b > 0$, $b \neq 1$, define:

$$\log_b x := \frac{\ln x}{\ln b}$$

This is not a separate function — it is a **rescaled natural logarithm**. The base $b=10$ gives $\log_{10}$ (common log), and $b=e$ gives $\ln$ itself.

**Derivative:** $$\frac{d}{dx}\log_b x = \frac{1}{x \ln b}$$

> In practice: unless forced otherwise, always work in base $e$. The extra factor of $\ln b$ in denominators is noise.

### 3.2 The Exponential Function — The Inverse

Since $\ln:(0,\infty)\to\mathbb{R}$ is a bijection, its inverse $\exp = \ln^{-1}: \mathbb{R} \to (0,\infty)$ exists. We denote $\exp(x) = e^x$. From inverse function differentiation:

$$\frac{d}{dx}e^x = e^x.$$

The mutual inverse relationship: $$\ln(e^x) = x \quad \forall x \in \mathbb{R}, \qquad e^{\ln x} = x \quad \forall x > 0.$$

## 4. Asymptotic Behavior — Two Important Limits

These are indispensable and constantly appear in analysis and ML.

**Theorem.** $$\lim_{x \to \infty} \frac{\ln x}{x^p} = 0 \quad \text{for any } p > 0 \tag{A1}$$ $$\lim_{x \to 0^+} x^p \ln x = 0 \quad \text{for any } p > 0 \tag{A2}$$

**Proof of (A1).** For $x > 1$: $$\ln x = \int_1^x \frac{dt}{t} \leq \int_1^x t^{p/2 - 1},dt = \frac{2}{p}\left(x^{p/2}-1\right) \leq \frac{2}{p} x^{p/2}.$$ So $0 \leq \frac{\ln x}{x^p} \leq \frac{2}{p} x^{-p/2} \to 0$. $\blacksquare$

**Verbal summary:** _Logarithms grow more slowly than any positive power of $x$. Any power of $x$ damps out $|\ln x|$ at the origin._

```
Growth rate hierarchy (slowest to fastest):
ln(ln x) ≪ ln x ≪ x^ε ≪ x ≪ x^2 ≪ e^x ≪ e^(e^x)
for any fixed ε > 0.
```

## 5. Integration Involving $\ln$

### 5.1 The Fundamental Antiderivative

$$\int \frac{1}{x},dx = \ln|x| + C$$

The absolute value handles $x < 0$: for $x < 0$, let $u = -x$, then $du = -dx$ and $\int \frac{1}{x}dx = \int \frac{-du}{u} = -\ln u = \ln|x|$.

### 5.2 Integration by Parts: $\int \ln x, dx$

$$\int \ln x, dx = x\ln x - x + C$$

**Proof.** Set $u = \ln x$, $dv = dx$. Then $du = \frac{1}{x}dx$, $v = x$. $$\int \ln x, dx = x\ln x - \int x \cdot \frac{1}{x}, dx = x\ln x - x + C. \qquad \blacksquare$$

### 5.3 Logarithmic Integration Pattern

$$\int \frac{f'(x)}{f(x)},dx = \ln|f(x)| + C$$

**Example.** $\displaystyle\int \tan x, dx = \int \frac{\sin x}{\cos x}, dx = -\ln|\cos x| + C = \ln|\sec x| + C$.

## 6. Logarithmic Differentiation

A technique for differentiating products, quotients, or functions with variable exponents.

**Strategy:** Take $\ln$ of both sides, differentiate implicitly, solve for $y'$.

**Example 1.** Let $y = x^x$ for $x > 0$. $$\ln y = x \ln x \implies \frac{y'}{y} = \ln x + 1 \implies y' = x^x(\ln x + 1).$$

**Example 2.** Let $y = \dfrac{x^2 \sqrt{x+1}}{(2x-1)^3}$.

$$\ln y = 2\ln x + \tfrac{1}{2}\ln(x+1) - 3\ln(2x-1)$$ $$\frac{y'}{y} = \frac{2}{x} + \frac{1}{2(x+1)} - \frac{6}{2x-1}$$

Multiply through by $y$ to get $y'$. _Much cleaner than the quotient rule._

## 7. Taylor Series and Approximations

Near $x = 0$:

$$\ln(1+x) = x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots = \sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n} x^n, \qquad |x| \leq 1,; x \neq -1$$

**Proof sketch.** We know $\frac{d}{dx}\ln(1+x) = \frac{1}{1+x} = \sum_{n=0}^\infty (-1)^n x^n$ for $|x| < 1$. Integrating term by term and using $\ln(1) = 0$ gives the series.

**First-order approximation:** For small $x$, $$\ln(1+x) \approx x.$$ This is _everywhere_ in ML. The log-likelihood gradient, the natural gradient, the derivation of KL divergence — all lean on this.

## 8. Connections to AI, Machine Learning, and Deep Learning

This is where the abstract machinery becomes viscerally useful. I want to show _why_ logarithms, specifically, appear — not just that they do.

### 8.1 Cross-Entropy Loss

In a $K$-class classification problem, let $\mathbf{p} = (p_1, \ldots, p_K)$ be the true distribution (a one-hot vector in practice) and $\hat{\mathbf{p}} = (\hat{p}_1,\ldots,\hat{p}_K)$ be the model's predicted probabilities (output of a softmax). The **cross-entropy loss** is:

$$\mathcal{L} = -\sum_{k=1}^K p_k \ln \hat{p}_k$$

For a one-hot label where $p_c = 1$ for correct class $c$:

$$\mathcal{L} = -\ln \hat{p}_c$$

**Why $\ln$?** Because we are doing **maximum likelihood estimation**. The likelihood of data ${x_i, y_i}_{i=1}^N$ under the model is: $$\mathcal{L}_{\text{MLE}} = \prod_{i=1}^N \hat{p}_{y_i}(x_i).$$ Maximizing this is equivalent to minimizing $-\ln \mathcal{L}_{\text{MLE}} = -\sum_i \ln \hat{p}_{y_i}(x_i)$. The product became a **sum** because of property (L1). This is why gradient descent on the cross-entropy loss _is_ maximum likelihood.

> **Key calculus here.** $\frac{\partial}{\partial z_c}(-\ln \hat{p}_c) = \hat{p}_c - 1$ (after passing through the softmax — a beautiful calculation involving $\frac{d}{dx}\ln$ and the chain rule). This is why the gradient of cross-entropy + softmax is so clean.

### 8.2 The Log-Sum-Exp Trick — Numerical Stability

The **softmax** is: $$\hat{p}_k = \frac{e^{z_k}}{\sum_{j} e^{z_j}}$$

For large logits $z_k$ (e.g., $z_k = 1000$), $e^{z_k}$ overflows to `inf` on a computer. The fix: use the identity

$$\ln!\sum_j e^{z_j} = c + \ln!\sum_j e^{z_j - c}$$

(follows from (L1): $\ln(e^c \cdot \sum e^{z_j - c}) = c + \ln\sum e^{z_j-c}$). Choose $c = \max_j z_j$. Now the largest term inside the sum is $e^0 = 1$, and everything is numerically safe.

> This is _applied logarithm arithmetic_ — property (L1) saves numerical computation.

### 8.3 KL Divergence

The **Kullback–Leibler divergence** from distribution $q$ to $p$:

$$D_{\mathrm{KL}}(p | q) = \sum_k p_k \ln \frac{p_k}{q_k} = \sum_k p_k(\ln p_k - \ln q_k)$$

Properties:

- $D_{\mathrm{KL}} \geq 0$ always (**Gibbs' inequality**, proved using concavity of $\ln$ and Jensen's inequality: $-\ln$ is convex, so $-\mathbb{E}_p[\ln(q/p)] \geq -\ln\mathbb{E}_p[q/p] = 0$).
- $D_{\mathrm{KL}} = 0$ iff $p = q$.
- **Not symmetric** — it is not a metric, but it measures "information lost when using $q$ to approximate $p$."

In **variational autoencoders (VAEs)**, the loss is ELBO = reconstruction loss + $D_{\mathrm{KL}}(q_\phi(z|x) | p(z))$. The KL term regularizes the latent space toward the prior. Its gradient is computable exactly when both distributions are Gaussian, using the formula for Gaussian KL — which requires $\ln$ of the determinants of covariance matrices.

### 8.4 Entropy and Information

**Shannon entropy:** $H(p) = -\sum_k p_k \ln p_k$

This measures the average uncertainty (or information content) in a distribution. The $\ln$ makes it **additive** for independent events (from (L1)): $H(p \otimes q) = H(p) + H(q)$.

In **decision tree learning** (CART, ID3), splitting criteria use information gain $= H(\text{parent}) - \sum H(\text{children})$. The logarithm is not cosmetic — it is the _unique_ (up to scalar) function making information additive. (Shannon's theorem.)

### 8.5 Log-Likelihood and Gradient Descent

Training a neural network by gradient descent on negative log-likelihood:

$$\theta^* = \arg\min_\theta \left[-\sum_{i=1}^N \ln p_\theta(y_i | x_i)\right]$$

The **gradient** is:

$$\nabla_\theta \left[-\ln p_\theta(y|x)\right] = -\nabla_\theta \ln p_\theta(y|x) = -\frac{\nabla_\theta p_\theta(y|x)}{p_\theta(y|x)}$$

(by the chain rule: $\frac{d}{dx}\ln f(x) = \frac{f'(x)}{f(x)}$)

This **score function** $\nabla_\theta \ln p_\theta$ appears in policy gradient methods (REINFORCE), the Fisher information matrix, and natural gradient descent. Every one of these rests on the single formula $(\ln f)' = f'/f$.

### 8.6 Logarithmic Learning Rate Schedules

Practitioners use schedules like: $$\eta_t = \eta_0 / \ln(t+2)$$

This decays _very_ slowly (slower than $1/t$, faster than constant) — precisely because $\ln t$ grows slowly (recall (A1)). The asymptotic hierarchy of §4 is not academic — it tells you _which schedule converges, and how fast_.

## 9. Worked Examples

### Example 9.1 — Differentiate $f(x) = \ln(\sin^2 x + 1)$

$$f'(x) = \frac{2\sin x \cos x}{\sin^2 x + 1} = \frac{\sin 2x}{\sin^2 x + 1}$$

_(Chain rule: outer derivative $1/u$, inner derivative $2\sin x\cos x = \sin 2x$.)_

### Example 9.2 — Evaluate $\int_0^1 \frac{x}{x^2+1},dx$

Let $u = x^2+1$, $du = 2x,dx$: $$\int_0^1 \frac{x}{x^2+1}dx = \frac{1}{2}\int_1^2 \frac{du}{u} = \frac{1}{2}\left[\ln u\right]_1^2 = \frac{\ln 2}{2}.$$

### Example 9.3 — Logarithmic Differentiation for $y = \frac{(x^2+1)^3 e^{-x}}{\sqrt{x+1}}$

$$\ln y = 3\ln(x^2+1) - x - \frac{1}{2}\ln(x+1)$$ $$\frac{y'}{y} = \frac{6x}{x^2+1} - 1 - \frac{1}{2(x+1)}$$ $$y' = \frac{(x^2+1)^3 e^{-x}}{\sqrt{x+1}} \cdot \left[\frac{6x}{x^2+1} - 1 - \frac{1}{2(x+1)}\right]$$

### Example 9.4 — A Limit Using L'Hôpital

$$\lim_{x \to 0^+} x \ln x$$

Form $0 \cdot (-\infty)$. Rewrite as $\frac{\ln x}{1/x}$, form $\frac{-\infty}{\infty}$. Apply L'Hôpital:

$$\lim_{x \to 0^+} \frac{\ln x}{1/x} = \lim_{x \to 0^+} \frac{1/x}{-1/x^2} = \lim_{x \to 0^+} (-x) = 0.$$

_(Consistent with (A2): $p=1$ there.)_

### Example 9.5 — Cross-Entropy Gradient (ML Application)

Suppose $\hat{p} = \sigma(z) = 1/(1+e^{-z})$ (sigmoid, binary classification). Loss: $\mathcal{L} = -y\ln\hat{p} - (1-y)\ln(1-\hat{p})$. Compute $\partial \mathcal{L}/\partial z$:

$$\frac{\partial \mathcal{L}}{\partial z} = -\frac{y}{\hat{p}}\hat{p}' + \frac{1-y}{1-\hat{p}}\hat{p}'$$

where $\hat{p}' = \hat{p}(1-\hat{p})$ (standard sigmoid derivative). Substituting:

$$= -y(1-\hat{p}) + (1-y)\hat{p} = \hat{p} - y.$$

The loss gradient is simply **prediction minus label** — a beautiful fact enabled entirely by $\frac{d}{dx}\ln x = 1/x$ eating the denominator.

## 12. Personal Marginalia

> _Why is the natural log "natural"?_ Because $\frac{d}{dx}\ln_b x = \frac{1}{x \ln b}$, and only $b = e$ gives the clean coefficient $1$. Any other base clutters every derivative and integral with a constant factor. Nature (and mathematicians) despise clutter.

> _The logarithm is the original data scientist's transform._ Raw data spanning orders of magnitude — incomes, earthquake energies, neural firing rates, word frequencies (Zipf's law) — becomes Gaussian-ish after a log transform. Why? Because many natural phenomena are log-normally distributed: their _logarithm_ is normal. The moment you take a log, you're doing exponential-family statistics.

> _Jensen's inequality + concavity of $\ln$ = half of modern probabilistic machine learning._ The ELBO in variational inference, the EM algorithm, contrastive learning bounds — all are applications of $\ln(\mathbb{E}[X]) \geq \mathbb{E}[\ln X]$ combined with the definition of KL divergence. One concavity fact, infinite consequences.
