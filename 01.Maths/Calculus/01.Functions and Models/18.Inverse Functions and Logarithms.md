## 0. Motivation

The exponential $e^x$ encodes growth. Its inverse — the logarithm — decodes it: _given the result of growth, how long did it run?_ In machine learning, log-likelihoods, cross-entropy, softmax, and the entire information-theoretic foundation of learning are built on logarithms. Inverse functions more broadly appear whenever we ask: _can we recover the input from the output?_ This note answers that question rigorously.

---

## 1. One-to-One Functions

### 1.1 Definition

A function $f$ is **one-to-one** (injective) if distinct inputs produce distinct outputs:

$$ \boxed{x_1 \neq x_2 \implies f(x_1) \neq f(x_2)} $$

Equivalently (contrapositive):

$$ f(x_1) = f(x_2) \implies x_1 = x_2 $$

### 1.2 The Horizontal Line Test

$f$ is one-to-one if and only if every horizontal line $y = c$ intersects the graph of $f$ **at most once**.

```
  One-to-one ✓              NOT one-to-one ✗
  
  y │    /                  y │    ∩
    │   /                     │   / \
 ───┼──/──── y=c           ───┼──/───\── y=c (hits twice!)
    │ /                       │ /     \
    │/                        │/       \
    └────── x                 └────────── x
    
    f(x) = x³                 f(x) = x²
```

### 1.3 Strictly Monotone ⟹ One-to-One

**Theorem.** If $f$ is strictly increasing or strictly decreasing on an interval $I$, then $f$ is one-to-one on $I$.

**Proof.** Suppose $f$ is strictly increasing and $x_1 \neq x_2$. WLOG $x_1 < x_2$. Then $f(x_1) < f(x_2)$, so $f(x_1) \neq f(x_2)$. $\square$

> This is why restricting domain restores injectivity. $f(x) = x^2$ is not injective on $\mathbb{R}$, but is strictly increasing on $[0,\infty)$, hence injective there.

## 2. The Inverse Function

### 2.1 Definition

Let $f: A \to B$ be one-to-one and onto (bijective onto its range). The **inverse function** $f^{-1}: B \to A$ is defined by:

$$ \boxed{f^{-1}(y) = x \iff f(x) = y} $$

The domain of $f^{-1}$ is the range of $f$, and the range of $f^{-1}$ is the domain of $f$.

> **Notation warning.** $f^{-1}(x) \neq \dfrac{1}{f(x)}$. The superscript $-1$ denotes the inverse function, **not** the reciprocal. The reciprocal is written $[f(x)]^{-1}$.

### 2.2 The Cancellation Identities

For all $x$ in the appropriate domain:

$$ f^{-1}(f(x)) = x \qquad \text{(for all } x \in D_f\text{)} $$ $$ f(f^{-1}(x)) = x \qquad \text{(for all } x \in D_{f^{-1}}\text{)} $$

These say: applying $f$ then $f^{-1}$ (or vice versa) returns to the original input.

### 2.3 How to Find $f^{-1}$ Explicitly

**Algorithm:**

1. Write $y = f(x)$.
2. Solve for $x$ in terms of $y$.
3. Swap $x \leftrightarrow y$ (by convention, we want $y = f^{-1}(x)$).
4. State the domain of $f^{-1}$ (= range of $f$).

**Example.** $f(x) = 2x + 3$.

$$ y = 2x + 3 \implies x = \frac{y-3}{2} \implies f^{-1}(x) = \frac{x-3}{2} $$

**Example.** $f(x) = x^3 + 1$.

$$ y = x^3 + 1 \implies x = (y-1)^{1/3} \implies f^{-1}(x) = (x-1)^{1/3} $$

**Example (domain restriction).** $f(x) = x^2$ on $[0, \infty)$.

$$ y = x^2, ; x \geq 0 \implies x = \sqrt{y} \implies f^{-1}(x) = \sqrt{x}, \quad x \geq 0 $$

Without the restriction, the inverse would not be a function (two $x$ values map to the same $y$).

### 2.4 Graph of $f^{-1}$

The graph of $f^{-1}$ is the **reflection** of the graph of $f$ across the line $y = x$.

```
  y │         /  f(x)
    │     ╱  /
    │   ╱  ·/·
    │ ╱  ·/ · (reflected points)
    │╱  /  ╱  f⁻¹(x)
  ──┼──/──╱──────── x
    │ / ╱
    │/╱
    
    Points (a,b) on f  ↔  points (b,a) on f⁻¹
```

**Proof.** If $(a, b)$ is on the graph of $f$, then $f(a) = b$, so $f^{-1}(b) = a$, meaning $(b, a)$ is on the graph of $f^{-1}$. Swapping coordinates $(a,b) \mapsto (b,a)$ is exactly reflection across $y = x$. $\square$

### 2.5 Derivative of the Inverse (Preview to Chapter 3)

If $f$ is differentiable and one-to-one, then:

$$ \boxed{(f^{-1})'(a) = \frac{1}{f'(f^{-1}(a))}} $$

**Intuition.** Differentiate the cancellation identity $f(f^{-1}(x)) = x$ using the chain rule:

$$ f'(f^{-1}(x)) \cdot (f^{-1})'(x) = 1 \implies (f^{-1})'(x) = \frac{1}{f'(f^{-1}(x))} $$

Geometrically: if $f$ has slope $m$ at a point, $f^{-1}$ has slope $1/m$ at the reflected point. Steep $f$ gives shallow $f^{-1}$, and vice versa.

## 3. The Exponential Function

### 3.1 Definition and Properties

For $a > 0$, $a \neq 1$, the exponential function $f(x) = a^x$ satisfies:

$$ a^{x+y} = a^x \cdot a^y, \qquad (a^x)^y = a^{xy}, \qquad a^0 = 1 $$

It is strictly monotone (increasing if $a>1$, decreasing if $0 < a < 1$), hence injective on all of $\mathbb{R}$.

The **natural exponential** $f(x) = e^x$, where $e = 2.71828\ldots$, is the unique exponential whose derivative equals itself:

$$ \frac{d}{dx} e^x = e^x $$

```
  y │        e^x
    │       /
    │      /
    │     /
    │   _/
    │__/
  ──┼──────────── x
    │
    
  Domain: ℝ,  Range: (0, ∞)
  Horizontal asymptote: y = 0 as x → -∞
```

## 4. The Logarithmic Function

### 4.1 Definition

Since $f(x) = a^x$ is one-to-one, it has an inverse. The **logarithm base $a$** is:

$$ \boxed{\log_a x = y \iff a^y = x} $$

Domain: $(0, \infty)$. Range: $\mathbb{R}$. This is the answer to: _"To what power must I raise $a$ to get $x$?"_

### 4.2 The Natural Logarithm

The inverse of $e^x$ is the **natural logarithm**:

$$ \ln x = y \iff e^y = x $$

This is the most important logarithm in analysis. All other logarithms are scalar multiples of it:

$$ \log_a x = \frac{\ln x}{\ln a} $$

### 4.3 Graph

```
  y │        ln(x)
    │       /
    │     _/
    │   _/
    │__/
  ──┼──│──────────── x
    │  1  (passes through (1,0))
    │
    
  Domain: (0,∞),  Range: ℝ
  Vertical asymptote: x = 0
  
  Reflection of e^x across y=x:
  
  y │     /  e^x
    │   ·/·
    │ ·/  ·    ←── y=x
    │/  ·  ·
    │     · ·  ln(x)
  ──┼──────────────── x
```

### 4.4 Laws of Logarithms

Let $x, y > 0$ and $r \in \mathbb{R}$:

$$ \ln(xy) = \ln x + \ln y $$ $$ \ln!\left(\frac{x}{y}\right) = \ln x - \ln y $$ $$ \ln(x^r) = r \ln x $$

**Proof of the first law.** Let $a = \ln x$ and $b = \ln y$, so $x = e^a$ and $y = e^b$. Then:

$$ \ln(xy) = \ln(e^a \cdot e^b) = \ln(e^{a+b}) = a + b = \ln x + \ln y \quad \square $$

The other two follow similarly.

### 4.5 Cancellation Identities for $\ln$ and $e^x$

$$ e^{\ln x} = x \qquad (x > 0) $$ $$ \ln(e^x) = x \qquad (x \in \mathbb{R}) $$

These are just the general cancellation identities for $f$ and $f^{-1}$ specialized to $e^x$ and $\ln$.

### 4.6 The Derivative

$$ \frac{d}{dx} \ln x = \frac{1}{x}, \qquad x > 0 $$

**Proof via inverse function derivative.** We know $e^{\ln x} = x$. Differentiate both sides:

$$ e^{\ln x} \cdot \frac{d}{dx}\ln x = 1 \implies x \cdot \frac{d}{dx}\ln x = 1 \implies \frac{d}{dx}\ln x = \frac{1}{x} \quad \square $$

More generally: $\dfrac{d}{dx} \ln|x| = \dfrac{1}{x}$ for $x \neq 0$.

### 4.7 Natural Log of Common Values

|$x$|$\ln x$|Reason|
|---|---|---|
|$1$|$0$|$e^0 = 1$|
|$e$|$1$|$e^1 = e$|
|$e^2$|$2$|$e^2 = e^2$|
|$1/e$|$-1$|$e^{-1} = 1/e$|
|$\sqrt{e}$|$1/2$|$e^{1/2} = \sqrt{e}$|

## 5. Inverse Trigonometric Functions

### 5.1 Arcsine

$\sin x$ is not injective on $\mathbb{R}$. Restricting to $[-\pi/2, \pi/2]$ makes it strictly increasing, hence invertible.

$$ \arcsin(x) = \sin^{-1}(x) = y \iff \sin(y) = x \text{ and } y \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right] $$

Domain: $[-1, 1]$. Range: $[-\pi/2, \pi/2]$.

```
  y │  π/2 ─────────────·
    │              ·
    │          ·
  ──┼─────·────────────── x
    │  ·         -1   1
    │·
  -π/2
  
  Cancellation: sin(arcsin x) = x  for x ∈ [-1,1]
               arcsin(sin x) = x  for x ∈ [-π/2, π/2]
```

**Derivative:**

$$ \frac{d}{dx}\arcsin(x) = \frac{1}{\sqrt{1-x^2}}, \quad |x| < 1 $$

**Proof.** Let $y = \arcsin x$, so $\sin y = x$. Differentiate: $\cos y \cdot y' = 1$, so $y' = \dfrac{1}{\cos y} = \dfrac{1}{\sqrt{1 - \sin^2 y}} = \dfrac{1}{\sqrt{1-x^2}}$. $\square$

### 5.2 Arctangent

$\tan x$ restricted to $(-\pi/2, \pi/2)$:

$$ \arctan(x) = y \iff \tan(y) = x \text{ and } y \in \left(-\frac{\pi}{2}, \frac{\pi}{2}\right) $$

Domain: $\mathbb{R}$. Range: $(-\pi/2, \pi/2)$.

$$ \lim_{x \to \infty} \arctan x = \frac{\pi}{2}, \qquad \lim_{x \to -\infty} \arctan x = -\frac{\pi}{2} $$

```
  y │  π/2 ─ ─ ─ ─ ─ ─ ─ ─ (asymptote)
    │          ___──────
    │    ___───
  ──┼───────────────────── x
    │     ───___
    │ ─────
  -π/2 ─ ─ ─ ─ ─ ─ ─ ─ ─ (asymptote)
```

**Derivative:**

$$ \frac{d}{dx}\arctan(x) = \frac{1}{1+x^2} $$

**Proof.** Let $y = \arctan x$, so $\tan y = x$. Differentiate: $\sec^2 y \cdot y' = 1$, so $y' = \cos^2 y = \dfrac{1}{1+\tan^2 y} = \dfrac{1}{1+x^2}$. $\square$

### 5.3 Summary of Inverse Trig Derivatives

|Function|Domain|Derivative|
|---|---|---|
|$\arcsin x$|$(-1,1)$|$\dfrac{1}{\sqrt{1-x^2}}$|
|$\arccos x$|$(-1,1)$|$-\dfrac{1}{\sqrt{1-x^2}}$|
|$\arctan x$|$\mathbb{R}$|$\dfrac{1}{1+x^2}$|

Note: $\arcsin x + \arccos x = \pi/2$ (complementary), so their derivatives sum to zero.

## 6. Solving Exponential and Logarithmic Equations

### 6.1 Strategy

- To isolate an exponential: take $\ln$ of both sides.
- To isolate a logarithm: exponentiate both sides.

**Example.** Solve $3e^{2x} = 15$:

$$ e^{2x} = 5 \implies 2x = \ln 5 \implies x = \frac{\ln 5}{2} $$

**Example.** Solve $\ln(x+1) + \ln(x-1) = 1$:

$$ \ln((x+1)(x-1)) = 1 \implies x^2 - 1 = e \implies x = \sqrt{e+1} $$

(Reject $-\sqrt{e+1}$ since $x > 1$ required for domain.)

## 7. Connection to AI / Machine Learning / Deep Learning

### 7.1 The Sigmoid as an Inverse of Logit

The **sigmoid function** used in logistic regression and neural network output layers is:

$$ \sigma(x) = \frac{1}{1+e^{-x}} $$

Its inverse is the **logit** (log-odds):

$$ \sigma^{-1}(p) = \ln!\left(\frac{p}{1-p}\right) $$

Sigmoid maps $\mathbb{R} \to (0,1)$ — converting any real score into a probability. Its inverse maps probabilities back to log-odds. The fact that $\sigma$ and logit are mutual inverses is why logistic regression and binary cross-entropy form a natural pair.

```
  Sigmoid:                         Logit (inverse):
  
  y  1 ─ ─ ─ ─────────────        y  │         /
     │      ___/                      │        /
     │   __/                          │       /
  0.5│  /                          ───┼──────/────── x
     │ /                              │    /       0.5  1
     │/                               │   /
  ───┼──────────── x               (−∞)│  /
     
  Domain: ℝ → (0,1)               Domain: (0,1) → ℝ
```

### 7.2 Log-Likelihood and Cross-Entropy

Maximum Likelihood Estimation (MLE) — the theoretical basis for training — maximizes:

$$ \mathcal{L}(\theta) = \prod_{i=1}^{n} p_\theta(x_i) $$

Because this product is numerically unstable and hard to differentiate, we take the **log**:

$$ \ell(\theta) = \ln \mathcal{L}(\theta) = \sum_{i=1}^{n} \ln p_\theta(x_i) $$

The logarithm converts a product into a sum — this is the third law of logarithms at work. Maximizing $\ell(\theta)$ is equivalent to maximizing $\mathcal{L}(\theta)$ because $\ln$ is strictly increasing (a consequence of it being a one-to-one function). This is the entire theoretical justification.

**Cross-entropy loss** (minimized in classification):

$$ H(y, \hat{y}) = -\sum_{c} y_c \ln \hat{y}_c $$

The $\ln$ here comes directly from log-likelihood. Training a classifier with cross-entropy _is_ MLE.

### 7.3 Softmax Is a Multi-Class Generalization of Sigmoid

The **softmax** function maps a vector $\mathbf{z} \in \mathbb{R}^K$ to a probability distribution:

$$ \text{softmax}(\mathbf{z})_k = \frac{e^{z_k}}{\sum_{j=1}^K e^{z_j}} $$

Its inverse is the **log-softmax**:

$$ \ln \text{softmax}(\mathbf{z})_k = z_k - \ln!\left(\sum_j e^{z_j}\right) $$

Log-softmax is more numerically stable to compute directly than softmax followed by $\ln$. This is why `torch.nn.LogSoftmax` exists as a separate primitive in PyTorch.

### 7.4 Entropy and Information Theory

The **entropy** of a distribution $p$ is:

$$ H(p) = -\sum_x p(x) \ln p(x) = \mathbb{E}[-\ln p(X)] $$

The $\ln$ here quantifies _surprise_: low-probability events have high $-\ln p$ (lots of information). High-probability events carry little information. The logarithm is the unique function that converts multiplicative independence into additive information:

$$ \text{if } p(A \cap B) = p(A)p(B), \text{ then } -\ln p(A \cap B) = -\ln p(A) - \ln p(B) $$

This additive property — the logarithm law — is why information is measured in log-units (bits or nats).

### 7.5 Numerical Stability: The Log-Sum-Exp Trick

Softmax involves $\sum_j e^{z_j}$. For large $z_j$, this overflows. The trick:

$$ \ln!\left(\sum_j e^{z_j}\right) = c + \ln!\left(\sum_j e^{z_j - c}\right), \quad c = \max_j z_j $$

This works because $\ln(e^c \cdot X) = c + \ln X$. The cancellation identity for $\ln$ and $e^x$ makes this manipulation exact. Every modern deep learning framework uses this.

### 7.6 Learning Rate Schedules and Log-Scale

Hyperparameter search over learning rates is done on a **log scale**. Searching $\eta \in [10^{-5}, 10^{-1}]$ uniformly in log-space means sampling $\log_{10}\eta \in [-5, -1]$ uniformly. This reflects that the learning rate's effect on training is multiplicative, not additive — the natural domain for a logarithm.

### 7.7 Invertibility and Autoencoders

In an **autoencoder**, the encoder $f: \mathbb{R}^n \to \mathbb{R}^k$ (with $k \ll n$) compresses input to a latent code, and the decoder $g: \mathbb{R}^k \to \mathbb{R}^n$ reconstructs. The goal is for $g \approx f^{-1}$, i.e., the decoder approximates the inverse of the encoder. The theory of whether a map can have a left-inverse is precisely the theory of injective (one-to-one) functions. A non-injective encoder cannot have a true inverse — information is irreversibly lost.

### 7.8 Normalizing Flows — Exact Invertibility

**Normalizing flows** are generative models that require the transformation $f: \mathbb{R}^n \to \mathbb{R}^n$ to be exactly invertible. The change-of-variables formula for densities involves the Jacobian determinant:

$$ p_X(x) = p_Z(f^{-1}(x)) \cdot \left|\det J_{f^{-1}}(x)\right| $$

This requires $f$ to be a bijection. The entire architecture of a normalizing flow (RealNVP, Glow, etc.) is engineered so that $f^{-1}$ can be computed efficiently. Invertibility is not a convenience — it is the structural requirement.

## 8. Summary

```
  Core Chain:
  
  a^x  ──────────────────►  Strictly monotone
  (a>1, strictly increasing)       │
                                   ▼
                           One-to-one (injective)
                                   │
                                   ▼
                           Inverse exists: log_a(x)
                                   │
                                   ▼
                    Cancellations: a^(log_a x) = x
                                   log_a(a^x) = x
```

|Concept|Key Property|ML Relevance|
|---|---|---|
|One-to-one|$f(x_1)=f(x_2)\Rightarrow x_1=x_2$|Invertible encoders, normalizing flows|
|Inverse function|$f^{-1}(f(x))=x$|Decoder $\approx$ encoder$^{-1}$; logit is sigmoid$^{-1}$|
|Natural log|$\ln(e^x)=x$; $\frac{d}{dx}\ln x = 1/x$|Log-likelihood, cross-entropy, info theory|
|Log laws|Product $\to$ sum|MLE $\to$ sum of log-likelihoods|
|$\arctan$|Maps $\mathbb{R}\to(-\pi/2,\pi/2)$|Bounded activation, gradient stability|
|Log-sum-exp|Algebraic identity|Numerical stability in softmax|
