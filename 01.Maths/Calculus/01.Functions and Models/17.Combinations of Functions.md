## 0. Motivation — Why Combine Functions?

In practice, almost no interesting quantity is computed by a single primitive rule. A neural network applies a linear map, then a nonlinearity, then another linear map — that is already three functions composed. Understanding precisely _how_ functions interact is therefore not bookkeeping; it is the substrate on which all of modern machine learning is built.

---

## 1. Algebraic Combinations

Let $f$ and $g$ be functions with domains $D_f, D_g \subseteq \mathbb{R}$.

### 1.1 Pointwise Operations

For every $x \in D_f \cap D_g$, define:

$$ (f + g)(x) = f(x) + g(x) $$ $$ (f - g)(x) = f(x) - g(x) $$ $$ (fg)(x) = f(x),g(x) $$ $$ \left(\frac{f}{g}\right)(x) = \frac{f(x)}{g(x)}, \quad g(x) \neq 0 $$

> **Domain rule.** The natural domain of any algebraic combination is $D_f \cap D_g$ (with the additional constraint $g(x)\neq 0$ for division). This intersection can be strictly smaller than either factor — always check.

### 1.2 Worked Example

Let $f(x) = \sqrt{x}$ and $g(x) = \sqrt{4 - x^2}$.

$$ D_f = [0, \infty), \qquad D_g = [-2, 2] $$

$$ D_f \cap D_g = [0, 2] $$

So $(f + g)(x) = \sqrt{x} + \sqrt{4-x^2}$ is only defined on $[0,2]$.

```
 f(x) = √x          g(x) = √(4−x²)        (f+g)(x)
 
 |        /          |  ╭───╮              |  ╭──
 |      /            | /     \             | /    \
 |    /              |/       \            |/      
 |  /                0    2   -2           0   2
 0                                              
```

_Notice how the combined domain is the intersection._

## 2. Composition of Functions

### 2.1 Definition

$$ \boxed{(f \circ g)(x) = f\bigl(g(x)\bigr)} $$

The **output** of $g$ is fed as **input** to $f$. Formally:

$$ \text{dom}(f \circ g) = {x \in D_g : g(x) \in D_f} $$

> **Order matters.** $f \circ g \neq g \circ f$ in general. This is the non-commutativity of composition — crucial in deep learning where layer order is everything.

### 2.2 Schematic

```
          g              f
   x ──────────► g(x) ──────────► f(g(x))

   Input      Intermediate      Final output
```

### 2.3 Three-Function Composition

For three functions $f$, $g$, $h$:

$$ (f \circ g \circ h)(x) = f(g(h(x))) $$

This is how a 3-layer network works: input $\to$ hidden$_1$ $\to$ hidden$_2$ $\to$ output.

### 2.4 Decomposition — The Reverse Problem

Given a complicated function, identify "outer" and "inner":

|Composite $F(x)$|Outer $f$|Inner $g$|
|---|---|---|
|$\sin(x^2)$|$\sin(u)$|$x^2$|
|$e^{-x^2}$|$e^u$|$-x^2$|
|$\sqrt{\ln x}$|$\sqrt{u}$|$\ln x$|
|$\ln(\sin x)$|$\ln u$|$\sin x$|
|$(3x^2+1)^{10}$|$u^{10}$|$3x^2+1$|

> **Why bother?** Because the Chain Rule says $\frac{d}{dx}f(g(x)) = f'(g(x))\cdot g'(x)$. Decomposition is the prerequisite to differentiation.

### 2.5 Domain of Composition — Careful Example

Let $f(u) = \sqrt{u}$ and $g(x) = x - 1$.

$$ (f \circ g)(x) = \sqrt{x-1}, \qquad \text{dom} = {x : x - 1 \geq 0} = [1,\infty) $$

Now the other way: $g(f(x)) = \sqrt{x} - 1$, which requires $x \geq 0$, so $\text{dom}(g \circ f) = [0,\infty)$.

Completely different domains. Order is not cosmetic.

## 3. Visualization of Composition

Think of the number line mapping:

```
  f ∘ g pipeline:
  
  ℝ ────g────► ℝ ────f────► ℝ
  x     ──►   g(x)   ──►   f(g(x))
  
  Example: f(u)=u², g(x)=cos(x)
  
  x=0  ──► cos(0)=1 ──► 1²=1
  x=π/2──► cos(π/2)=0 ──► 0²=0
  x=π  ──► cos(π)=-1 ──► (-1)²=1
```

Notice that $f(g(x)) = \cos^2(x)$ collapses the negative outputs of cosine because squaring makes them positive.

## 4. Scalar Multiples and Vertical/Horizontal Transformations

These are special algebraic combinations worth naming explicitly.

Let $c > 0$. Starting from $y = f(x)$:

|Combination|Effect|
|---|---|
|$y = f(x) + c$|Shift up by $c$|
|$y = f(x) - c$|Shift down by $c$|
|$y = f(x + c)$|Shift left by $c$|
|$y = f(x - c)$|Shift right by $c$|
|$y = c,f(x)$|Vertical stretch/shrink|
|$y = f(cx)$|Horizontal shrink/stretch|
|$y = -f(x)$|Reflect over $x$-axis|
|$y = f(-x)$|Reflect over $y$-axis|

> **Note.** $f(cx)$ for $c>1$ _compresses_ horizontally — input argument reaches the same value faster. In signal processing, this is time compression; in ML, this relates to feature scaling.

## 5. Even and Odd Symmetry Under Combinations

A useful structural result:

|$f$|$g$|$f + g$|$fg$|
|---|---|---|---|
|even|even|even|even|
|odd|odd|odd|even|
|even|odd|neither in general|odd|

> **Proof sketch (product, odd × odd):** $(fg)(-x) = f(-x)g(-x) = (-f(x))(-g(x)) = f(x)g(x) = (fg)(x)$. So the product is **even**. $\square$

This is used in Fourier analysis and in understanding symmetry of activation functions in neural networks.

## 6. Connection to AI / Machine Learning / Deep Learning

This section is why everything above actually matters beyond the exam.

### 6.1 A Neural Network _Is_ a Composition of Functions

A feedforward neural network with $L$ layers computes:

$$ \hat{y} = (f_L \circ f_{L-1} \circ \cdots \circ f_1)(x) $$

where each layer function is:

$$ f_\ell(z) = \sigma(W_\ell z + b_\ell) $$

Here $W_\ell z + b_\ell$ is an **algebraic combination** (affine map), and $\sigma$ is a fixed nonlinearity. Composition stacks these layers.

```
  Input x
     │
     ▼
  ┌──────────────────┐
  │  f₁(x) = σ(W₁x+b₁) │  ← affine + nonlinearity
  └──────────────────┘
     │
     ▼
  ┌──────────────────┐
  │  f₂(z) = σ(W₂z+b₂) │
  └──────────────────┘
     │
     ▼
  ┌──────────────────┐
  │  f₃(z) = W₃z+b₃    │  ← output (no activation on last layer for regression)
  └──────────────────┘
     │
     ▼
  Output ŷ
```

### 6.2 The Chain Rule = Backpropagation

Because the network is a composition, differentiating it requires the **chain rule** applied repeatedly:

$$ \frac{\partial \mathcal{L}}{\partial W_\ell} = \frac{\partial \mathcal{L}}{\partial f_L} \cdot \frac{\partial f_L}{\partial f_{L-1}} \cdots \frac{\partial f_{\ell+1}}{\partial f_\ell} \cdot \frac{\partial f_\ell}{\partial W_\ell} $$

This is backpropagation. It is the chain rule for composed functions, applied to a computational graph. Understanding $f \circ g$ and its derivative is the _entire_ theoretical basis for training neural networks.

### 6.3 Activation Functions Are the Nonlinear Piece

Without a nonlinear activation, the composition of linear maps is still linear:

$$ W_2(W_1 x + b_1) + b_2 = (W_2 W_1)x + (W_2 b_1 + b_2) $$

This is just one affine map — no depth is gained. The activation function $\sigma$ breaks linearity. Common choices:

|Name|Formula|Domain|Notes|
|---|---|---|---|
|ReLU|$\max(0, x)$|$\mathbb{R}$|Not differentiable at 0; dominant in practice|
|Sigmoid|$\dfrac{1}{1+e^{-x}}$|$\mathbb{R}$|Output in $(0,1)$; used in binary classification|
|Tanh|$\tanh(x)$|$\mathbb{R}$|Output in $(-1,1)$; odd function|
|GELU|$x \cdot \Phi(x)$|$\mathbb{R}$|Used in Transformers/BERT|

All of these are combinations of elementary functions (exponentials, polynomials, products).

### 6.4 Feature Engineering as Algebraic Combination

Before deep learning, practitioners manually built new features by algebraically combining raw inputs:

$$ x_{\text{new}} = \frac{x_1}{x_2}, \quad x_{\text{new}} = \sqrt{x_1^2 + x_2^2}, \quad x_{\text{new}} = \ln(1 + x_1) $$

Deep networks _learn_ these combinations. The network is discovering useful algebraic and composed transformations of the input automatically — which is why representation learning works.

### 6.5 Loss Functions Are Algebraic Combinations of Outputs and Labels

The cross-entropy loss:

$$ \mathcal{L} = -\sum_{i} y_i \log \hat{y}_i $$

is a product and sum applied to $\log \circ \hat{f}$ — a composed and then algebraically combined function.

## 7. Multivariable Setting (MIT 18.02 Preview)

In higher dimensions, functions map $\mathbb{R}^n \to \mathbb{R}^m$. Algebraic combinations extend naturally. Composition becomes:

$$ (f \circ g)(\mathbf{x}) = f(g(\mathbf{x})), \quad g: \mathbb{R}^n \to \mathbb{R}^k, \quad f: \mathbb{R}^k \to \mathbb{R}^m $$

The chain rule for composed multivariate functions involves the **Jacobian matrix**:

$$ D(f \circ g)(\mathbf{x}) = Df(g(\mathbf{x})) \cdot Dg(\mathbf{x}) $$

where $\cdot$ denotes matrix multiplication. This is the foundation of automatic differentiation (autograd) in PyTorch and JAX.

## 9. Exercises

**Exercise 1.** Let $f(x) = \dfrac{1}{\sqrt{x-1}}$ and $g(x) = \ln(x)$. Find the domain of $f \circ g$.

> _Hint: You need $g(x) \in D_f$, which means $g(x) > 1$._

**Exercise 2.** Show by direct computation that $(f \circ g)(x) \neq (g \circ f)(x)$ for $f(x) = e^x$ and $g(x) = x^2$.

**Exercise 3.** Decompose $F(x) = \sin^3(\ln x)$ into three functions $f, g, h$ such that $F = f \circ g \circ h$.

**Exercise 4.** If $f$ is even and $g$ is odd, is $f \circ g$ even, odd, or neither? Prove your answer.

> _Answer: $(f \circ g)(-x) = f(g(-x)) = f(-g(x)) = f(g(x))$ since $f$ is even. So $f \circ g$ is **even**._

**Exercise 5 (ML flavour).** Consider a 2-layer network: $$ \hat{y} = W_2 \cdot \text{ReLU}(W_1 x + b_1) + b_2 $$ Write this as an explicit composition $f \circ g$ and identify the domain of $f$ and $g$. What happens to the domain if we replace ReLU with sigmoid?
