
## § 0 — Motivation: Why Should You Care?

Before we define anything, a confession: the name "hyperbolic" is not an accident, and these are not just "the sinh/cosh things you memorize for an exam." They appear naturally in:

- The shape of a **hanging cable** (catenary): $y = a\cosh(x/a)$
- **Special relativity**: Lorentz boosts are hyperbolic rotations
- Solutions to the **heat equation** and **wave equation**
- The **backpropagation** algorithm in deep learning ($\tanh$ is one of the most classical activation functions)
- **Geodesics** on the hyperbolic plane (differential geometry)

They also make many integrals tractable that would otherwise require painful trigonometric substitutions.

---

## § 1 — Definitions: Exponential Heart

Everything comes from two combinations of $e^x$:

$$\boxed{\sinh x = \frac{e^x - e^{-x}}{2}, \qquad \cosh x = \frac{e^x + e^{-x}}{2}}$$

Think of $e^x = \cosh x + \sinh x$ as splitting the exponential into its **even part** ($\cosh$) and **odd part** ($\sinh$).

The remaining four follow exactly as in trigonometry:

$$\tanh x = \frac{\sinh x}{\cosh x} = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$

$$\operatorname{csch} x = \frac{1}{\sinh x}, \quad \operatorname{sech} x = \frac{1}{\cosh x}, \quad \coth x = \frac{1}{\tanh x}$$

> **Personal note:** I always re-derive $\tanh$ from scratch instead of memorizing it. If you know the two boxed formulas above, you know everything.

## § 2 — The Hyperbola Connection (Why "Hyperbolic"?)

Recall that $(\cos\theta, \sin\theta)$ parametrizes the **unit circle** $x^2 + y^2 = 1$.

**Claim:** $(\cosh t, \sinh t)$ parametrizes the **right branch** of the unit hyperbola $x^2 - y^2 = 1$.

**Proof:** $$\cosh^2 t - \sinh^2 t = \left(\frac{e^t+e^{-t}}{2}\right)^2 - \left(\frac{e^t-e^{-t}}{2}\right)^2$$

$$= \frac{(e^t+e^{-t})^2 - (e^t-e^{-t})^2}{4} = \frac{4e^t e^{-t}}{4} = \frac{4}{4} = 1 \qquad \square$$

This is the **fundamental identity**:

$$\boxed{\cosh^2 x - \sinh^2 x = 1}$$

The parameter $t$ is **twice the area** of the hyperbolic sector (analogous to how $\theta/2$ is the area of a circular sector). This geometric fact is why $t$ is called the **hyperbolic angle**.

```
Circular:                        Hyperbolic:
   ┌──────────────┐                 ┌──────────────┐
   │     *        │                 │      *       │
   │  *     *     │                 │   *          │
   │ *       *    │                 │  *           │
   │  *     *     │                 │   *          │
   │     *        │                 │      *       │
   └──────────────┘                 └──────────────┘
  (cos θ, sin θ) on circle         (cosh t, sinh t) on hyperbola
  x² + y² = 1                      x² - y² = 1
```

## § 3 — Identities: The Rosetta Stone

Every trigonometric identity has a hyperbolic analogue, with **one sign twist** (Osborn's Rule: replace every product of two sines with its negative).

|Circular|Hyperbolic|
|---|---|
|$\cos^2 x + \sin^2 x = 1$|$\cosh^2 x - \sinh^2 x = 1$|
|$1 + \tan^2 x = \sec^2 x$|$1 - \tanh^2 x = \operatorname{sech}^2 x$|
|$\sin(x+y) = \sin x \cos y + \cos x \sin y$|$\sinh(x+y) = \sinh x \cosh y + \cosh x \sinh y$|
|$\cos(x+y) = \cos x \cos y - \sin x \sin y$|$\cosh(x+y) = \cosh x \cosh y + \sinh x \sinh y$|
|$\cos 2x = 2\cos^2 x - 1$|$\cosh 2x = 2\cosh^2 x - 1$|
|$\sin 2x = 2\sin x \cos x$|$\sinh 2x = 2\sinh x \cosh x$|

**Proof of** $1 - \tanh^2 x = \operatorname{sech}^2 x$:

Divide $\cosh^2 x - \sinh^2 x = 1$ through by $\cosh^2 x$:

$$1 - \frac{\sinh^2 x}{\cosh^2 x} = \frac{1}{\cosh^2 x} \implies 1 - \tanh^2 x = \operatorname{sech}^2 x \qquad \square$$

## § 4 — Graphs and Key Properties

### $\sinh x$: Odd, unbounded

$$\sinh(-x) = \frac{e^{-x}-e^{x}}{2} = -\sinh x$$

- Domain: $\mathbb{R}$, Range: $\mathbb{R}$
- $\sinh(0) = 0$
- Strictly increasing, one-to-one
- Looks like $x$ near origin, then grows like $e^x/2$

```
     ↑ sinh x
  4  │                    *
  3  │                *
  2  │            *
  1  │        *
  0  │────────────────────→ x
 -1  │    *
 -2  │*
     -2  -1   0   1   2
```

### $\cosh x$: Even, minimum at 1

$$\cosh(-x) = \cosh x, \quad \cosh(0) = 1$$

- Domain: $\mathbb{R}$, Range: $[1, \infty)$
- Convex, resembles a parabola near origin but grows exponentially
- This is the **catenary** (shape of a hanging chain)

### $\tanh x$: The S-curve

$$\tanh(-x) = -\tanh x, \quad \tanh(0) = 0$$

$$\lim_{x\to+\infty}\tanh x = 1, \quad \lim_{x\to-\infty}\tanh x = -1$$

- Domain: $\mathbb{R}$, Range: $(-1, 1)$
- Strictly increasing, one-to-one
- **Squashes all of $\mathbb{R}$ into $(-1,1)$** — this is why it's invaluable in neural networks

```
     ↑ tanh x
  1  │─────────────────────────── (asymptote)
     │               ****
     │          ****
     │       ***
  0  │────────────────────────→ x
     │   ***
     │****
 -1  │─────────────────────────── (asymptote)
     -3  -2  -1   0   1   2   3
```

## § 5 — Derivatives

**Theorem.** The derivatives of the hyperbolic functions are:

$$\frac{d}{dx}\sinh x = \cosh x, \qquad \frac{d}{dx}\cosh x = \sinh x$$

**Proof** (both at once):

$$\frac{d}{dx}\sinh x = \frac{d}{dx}\left(\frac{e^x - e^{-x}}{2}\right) = \frac{e^x + e^{-x}}{2} = \cosh x$$

$$\frac{d}{dx}\cosh x = \frac{d}{dx}\left(\frac{e^x + e^{-x}}{2}\right) = \frac{e^x - e^{-x}}{2} = \sinh x \qquad \square$$

> **Contrast with trig:** $\frac{d}{dx}\cos x = -\sin x$, but $\frac{d}{dx}\cosh x = +\sinh x$. The minus sign is missing — this is the same sign twist from Osborn's Rule.

The full table:

|$f(x)$|$f'(x)$|
|---|---|
|$\sinh x$|$\cosh x$|
|$\cosh x$|$\sinh x$|
|$\tanh x$|$\operatorname{sech}^2 x$|
|$\operatorname{csch} x$|$-\operatorname{csch} x \coth x$|
|$\operatorname{sech} x$|$-\operatorname{sech} x \tanh x$|
|$\coth x$|$-\operatorname{csch}^2 x$|

**Proof for $\tanh x$** (quotient rule):

$$\frac{d}{dx}\tanh x = \frac{\cosh x \cdot \cosh x - \sinh x \cdot \sinh x}{\cosh^2 x} = \frac{\cosh^2 x - \sinh^2 x}{\cosh^2 x} = \frac{1}{\cosh^2 x} = \operatorname{sech}^2 x \qquad \square$$

## § 6 — Inverse Hyperbolic Functions

Since $\sinh$ and $\tanh$ are strictly monotone on $\mathbb{R}$, and $\cosh$ is monotone on $[0,\infty)$, the inverses exist.

**Crucial fact:** The inverses can be expressed in closed form via logarithms:

$$\sinh^{-1} x = \ln!\left(x + \sqrt{x^2+1}\right), \quad x \in \mathbb{R}$$

$$\cosh^{-1} x = \ln!\left(x + \sqrt{x^2-1}\right), \quad x \geq 1$$

$$\tanh^{-1} x = \frac{1}{2}\ln!\left(\frac{1+x}{1-x}\right), \quad |x| < 1$$

**Proof of the $\sinh^{-1}$ formula:** Let $y = \sinh^{-1} x$, so $\sinh y = x$:

$$\frac{e^y - e^{-y}}{2} = x \implies e^y - e^{-y} = 2x \implies e^{2y} - 2xe^y - 1 = 0$$

This is a quadratic in $e^y$. By the quadratic formula:

$$e^y = \frac{2x \pm \sqrt{4x^2 + 4}}{2} = x \pm \sqrt{x^2+1}$$

Since $e^y > 0$ and $\sqrt{x^2+1} > |x|$, we must take the $+$ sign:

$$y = \ln!\left(x + \sqrt{x^2+1}\right) \qquad \square$$

### Derivatives of Inverses

$$\frac{d}{dx}\sinh^{-1} x = \frac{1}{\sqrt{x^2+1}}, \quad \frac{d}{dx}\cosh^{-1} x = \frac{1}{\sqrt{x^2-1}}, \quad \frac{d}{dx}\tanh^{-1} x = \frac{1}{1-x^2}$$

**Proof of the first** (implicit differentiation): If $\sinh y = x$, then $\cosh y \cdot \frac{dy}{dx} = 1$, so

$$\frac{dy}{dx} = \frac{1}{\cosh y} = \frac{1}{\sqrt{1 + \sinh^2 y}} = \frac{1}{\sqrt{1+x^2}} \qquad \square$$

## § 7 — Integration Formulas

The derivatives immediately give antiderivatives:

$$\int \cosh x , dx = \sinh x + C, \quad \int \sinh x , dx = \cosh x + C$$

$$\int \operatorname{sech}^2 x , dx = \tanh x + C$$

The real power comes from recognizing when a substitution reduces to these:

$$\int \frac{dx}{\sqrt{x^2+a^2}} = \sinh^{-1}!\left(\frac{x}{a}\right) + C = \ln!\left(x + \sqrt{x^2+a^2}\right) + C$$

$$\int \frac{dx}{\sqrt{x^2-a^2}} = \cosh^{-1}!\left(\frac{x}{a}\right) + C \quad (x > a > 0)$$

$$\int \frac{dx}{a^2-x^2} = \frac{1}{a}\tanh^{-1}!\left(\frac{x}{a}\right) + C \quad (|x|<a)$$

### Example: Hyperbolic Substitution

**Evaluate** $\displaystyle\int \sqrt{x^2+1}, dx$.

_Let_ $x = \sinh t$, so $dx = \cosh t , dt$ and $\sqrt{x^2+1} = \sqrt{\sinh^2 t + 1} = \cosh t$.

$$\int \cosh t \cdot \cosh t , dt = \int \cosh^2 t , dt = \int \frac{1 + \cosh 2t}{2}, dt = \frac{t}{2} + \frac{\sinh 2t}{4} + C$$

Back-substituting ($t = \sinh^{-1} x$, $\sinh 2t = 2\sinh t \cosh t = 2x\sqrt{x^2+1}$):

$$= \frac{1}{2}\sinh^{-1} x + \frac{x\sqrt{x^2+1}}{2} + C$$

Compare the trigonometric route ($x = \tan\theta$) — it's messier. Hyperbolic substitution is cleaner for $\sqrt{x^2 + a^2}$ integrands.

## § 8 — Taylor Series and Asymptotic Behavior

From the Taylor series of $e^x$:

$$\cosh x = \sum_{n=0}^{\infty} \frac{x^{2n}}{(2n)!} = 1 + \frac{x^2}{2!} + \frac{x^4}{4!} + \cdots$$

$$\sinh x = \sum_{n=0}^{\infty} \frac{x^{2n+1}}{(2n+1)!} = x + \frac{x^3}{3!} + \frac{x^5}{5!} + \cdots$$

$$\tanh x = x - \frac{x^3}{3} + \frac{2x^5}{15} - \cdots \quad (|x| < \pi/2)$$

These are exactly the circular Taylor series but **without the alternating signs** (another instance of the Osborn sign flip).

**Asymptotics:** For large $x$:

$$\sinh x \approx \cosh x \approx \frac{e^x}{2}, \qquad \tanh x \approx 1 - 2e^{-2x}$$

## § 9 — Connection to AI, Machine Learning, Deep Learning

This is where the abstract becomes immediately applicable.

### 9.1 — $\tanh$ as an Activation Function

In a neural network, a neuron computes $\sigma(\mathbf{w}^\top \mathbf{x} + b)$ where $\sigma$ is the **activation function**. The $\tanh$ function was one of the first standard choices:

$$\tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}} \in (-1, 1)$$

**Why $\tanh$?**

1. **Zero-centered output.** The output range $(-1,1)$ means the mean activation is near 0. This matters because if activations are all positive (as with the sigmoid $\sigma = 1/(1+e^{-x}) \in (0,1)$), gradients in backpropagation always have the same sign, causing inefficient zig-zag updates.
    
2. **Saturating nonlinearity.** It introduces the nonlinearity without which a deep network collapses to a linear map ($\because$ composition of linear maps is linear).
    
3. **Smooth and differentiable.** Gradient-based optimization (SGD, Adam) requires $\frac{d}{dz}\tanh z = \operatorname{sech}^2 z = 1 - \tanh^2 z$. Notice: if you've already computed $\tanh(z)$, the derivative costs almost nothing to compute — a single multiplication and subtraction. This is computationally elegant.
    

**The vanishing gradient problem:** $\operatorname{sech}^2 z \leq 1$ and $\operatorname{sech}^2(z) \to 0$ as $|z| \to \infty$. In deep networks, gradients are products of many such terms. When neurons saturate (large $|z|$), $\operatorname{sech}^2(z) \approx 0$, and the gradient signal vanishes through the chain rule. This is one reason ReLU ($\max(0,x)$) became popular — it doesn't saturate for $x>0$.

```
Backpropagation chain rule:

∂L/∂w₁ = (∂L/∂a_n) · (∏ sech²(zᵢ)) · (∂a₁/∂w₁)
                          i=2..n

If n is large and neurons saturate → product → 0 (vanishing gradient)
```

### 9.2 — $\sinh^{-1}$ and Data Preprocessing

In some ML pipelines, the transformation $x \mapsto \sinh^{-1}(x) = \ln(x + \sqrt{x^2+1})$ is used as a **variance-stabilizing transform** for heavy-tailed data. Unlike $\log(x)$ (defined only for $x>0$), $\sinh^{-1}$ is defined on all of $\mathbb{R}$ and behaves like $\ln(2x)$ for large $x$ and like $x$ near 0. It is used in financial data preprocessing where returns can be negative.

### 9.3 — $\cosh$ in the Loss Function Landscape

The **log-cosh loss** is a smooth approximation to the mean absolute error:

$$L(y, \hat{y}) = \sum_i \ln\cosh(y_i - \hat{y}_i)$$

Since $\ln\cosh(x) \approx \frac{x^2}{2}$ for small $x$ (MSE behavior) and $\approx |x| - \ln 2$ for large $x$ (MAE behavior), it inherits robustness to outliers from MAE while remaining twice-differentiable everywhere. This matters for optimization since MSE is sensitive to outliers and Huber loss has a non-smooth kink at the transition point.

### 9.4 — $\tanh$ in Recurrent Neural Networks (LSTMs)

In **Long Short-Term Memory** (LSTM) networks (Hochreiter & Schmidhuber, 1997), both $\tanh$ and the sigmoid $\sigma$ appear in the core update equations:

$$\tilde{C}_t = \tanh(W_c \cdot [h_{t-1}, x_t] + b_c) \quad \text{(candidate cell state)}$$ $$h_t = o_t \odot \tanh(C_t) \quad \text{(output/hidden state)}$$

The role of $\tanh$ here: **squash the cell state** into $(-1, 1)$ before it becomes the hidden state. It keeps magnitudes bounded over long sequences, partially mitigating the exploding gradient problem.

### 9.5 — Summary Table for AI

|Function|Where it appears|Why|
|---|---|---|
|$\tanh$|Hidden layer activations, LSTM gates|Zero-centered, smooth nonlinearity|
|$\operatorname{sech}^2 = \tanh'$|Backpropagation through $\tanh$|Derivative needed for gradient flow|
|$\ln\cosh$|Loss function (log-cosh loss)|Smooth robust regression|
|$\sinh^{-1}$|Data preprocessing|Variance-stabilizing transform|
|$\cosh, \sinh$|Hyperbolic geometry, manifold learning|Hyperbolic embeddings (Poincaré disk)|

> **Deep cut:** The **Poincaré embedding** model (Nickel & Kiela, 2017) embeds hierarchical data (like WordNet) in **hyperbolic space** rather than Euclidean space, because hyperbolic space can represent tree-like structures with exponentially growing neighborhoods more efficiently. The metric on the Poincaré disk is defined using $\cosh$ of a certain distance.

## § 10 — The Catenary: A Classical Application

**Problem.** A flexible inextensible cable of uniform linear density $\rho$ hangs under gravity between two supports. Find the shape.

Let the lowest point be the origin. Let $T_0$ be the tension there (horizontal). At arc length $s$ from the lowest point, the tension $T$ acts tangentially, and $\rho g s$ is the weight of the hanging segment. Force balance:

$$T\cos\theta = T_0, \qquad T\sin\theta = \rho g s$$

Dividing: $\tan\theta = \frac{\rho g s}{T_0}$. Since $\tan\theta = y'$ and $s = \int_0^x \sqrt{1+(y')^2}, dx'$, differentiating both sides in $x$:

$$y'' = \frac{\rho g}{T_0}\sqrt{1+(y')^2}$$

Let $a = T_0/(\rho g)$ and $p = y'$. Then $p' = \frac{1}{a}\sqrt{1+p^2}$, which separates:

$$\int \frac{dp}{\sqrt{1+p^2}} = \int \frac{dx}{a} \implies \sinh^{-1}(p) = \frac{x}{a} + C$$

With $p(0) = y'(0) = 0 \Rightarrow C = 0$. So $y' = \sinh(x/a)$. Integrating:

$$\boxed{y = a\cosh!\left(\frac{x}{a}\right) + C}$$

The catenary. The constant $C$ is fixed by the height at the lowest point.
