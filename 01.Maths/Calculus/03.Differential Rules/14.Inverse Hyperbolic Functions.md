## ¬ß 0 ‚Äî Why Bother With Inverses?

The honest answer: **integrals**.

The forms $\dfrac{1}{\sqrt{x^2+1}}$, $\dfrac{1}{\sqrt{x^2-1}}$, $\dfrac{1}{1-x^2}$ appear constantly in calculus, physics, and machine learning. Without inverse hyperbolic functions, you are forced into trigonometric substitution ‚Äî doable, but slower and more error-prone.

With inverse hyperbolic functions, these integrals become one-line antiderivatives.

A second reason: inverse hyperbolic functions are **not transcendental mysteries** ‚Äî they collapse to logarithms. Every one of them is expressible in closed form. This makes them simultaneously computable, differentiable, and elegant.

## ¬ß 2 ‚Äî Logarithmic Closed Forms

This is the core theorem. Every inverse hyperbolic function reduces to a logarithm.

### Theorem 2.1

$$\boxed{\sinh^{-1} x = \ln!\left(x + \sqrt{x^2 + 1}\right), \quad x \in \mathbb{R}}$$

**Proof.** Let $y = \sinh^{-1} x$, so $\sinh y = x$:

$$\frac{e^y - e^{-y}}{2} = x \implies e^y - e^{-y} = 2x$$

Multiply through by $e^y$:

$$e^{2y} - 2x e^y - 1 = 0$$

This is a quadratic in $e^y$. Quadratic formula:

$$e^y = \frac{2x \pm \sqrt{4x^2 + 4}}{2} = x \pm \sqrt{x^2 + 1}$$

Since $e^y > 0$ for all $y$, and $\sqrt{x^2+1} > |x|$, the expression $x - \sqrt{x^2+1} < 0$ always. So we must take $+$:

$$e^y = x + \sqrt{x^2+1} \implies y = \ln!\left(x + \sqrt{x^2 + 1}\right) \qquad \square$$

---

### Theorem 2.2

$$\boxed{\cosh^{-1} x = \ln!\left(x + \sqrt{x^2 - 1}\right), \quad x \geq 1}$$

**Proof.** Let $y = \cosh^{-1} x \geq 0$, so $\cosh y = x$:

$$\frac{e^y + e^{-y}}{2} = x \implies e^{2y} - 2xe^y + 1 = 0$$

Quadratic formula:

$$e^y = \frac{2x \pm \sqrt{4x^2 - 4}}{2} = x \pm \sqrt{x^2 - 1}$$

Both roots are positive when $x \geq 1$, but they correspond to $y \geq 0$ and $y \leq 0$ respectively. Since we restrict $\cosh^{-1}$ to the branch $y \geq 0$, we need $e^y \geq 1$, which requires $e^y = x + \sqrt{x^2-1}$:

$$y = \ln!\left(x + \sqrt{x^2-1}\right) \qquad \square$$

> **Sanity check:** At $x = 1$: $\cosh^{-1}(1) = \ln(1 + 0) = 0$. ‚úì

---

### Theorem 2.3

$$\boxed{\tanh^{-1} x = \frac{1}{2}\ln!\left(\frac{1+x}{1-x}\right), \quad |x| < 1}$$

**Proof.** Let $y = \tanh^{-1} x$, so $\tanh y = x$:

$$\frac{e^y - e^{-y}}{e^y + e^{-y}} = x$$

Cross-multiply: $e^y - e^{-y} = x(e^y + e^{-y})$. Rearrange:

$$e^y(1 - x) = e^{-y}(1 + x) \implies e^{2y} = \frac{1+x}{1-x}$$

Take logarithm:

$$2y = \ln!\left(\frac{1+x}{1-x}\right) \implies y = \frac{1}{2}\ln!\left(\frac{1+x}{1-x}\right) \qquad \square$$

> **Observe:** The argument $\frac{1+x}{1-x}$ must be positive, which requires $|x| < 1$ ‚Äî consistent with $\tanh: \mathbb{R} \to (-1,1)$.

---

### The Remaining Three

By analogous quadratic arguments (left as exercises):

$$\coth^{-1} x = \frac{1}{2}\ln!\left(\frac{x+1}{x-1}\right), \quad |x| > 1$$

$$\operatorname{sech}^{-1} x = \ln!\left(\frac{1 + \sqrt{1-x^2}}{x}\right), \quad 0 < x \leq 1$$

$$\operatorname{csch}^{-1} x = \ln!\left(\frac{1}{x} + \frac{\sqrt{1+x^2}}{|x|}\right), \quad x \neq 0$$

## ¬ß 4 ‚Äî Derivatives: The Algebraic Computation

These are among the most useful derivatives in calculus.

### Theorem 4.1 (Derivatives of Inverse Hyperbolics)

$$\frac{d}{dx}\sinh^{-1} x = \frac{1}{\sqrt{x^2+1}}$$

$$\frac{d}{dx}\cosh^{-1} x = \frac{1}{\sqrt{x^2-1}}, \quad x > 1$$

$$\frac{d}{dx}\tanh^{-1} x = \frac{1}{1-x^2}, \quad |x| < 1$$

$$\frac{d}{dx}\operatorname{sech}^{-1} x = \frac{-1}{x\sqrt{1-x^2}}, \quad 0 < x < 1$$

$$\frac{d}{dx}\operatorname{csch}^{-1} x = \frac{-1}{|x|\sqrt{1+x^2}}, \quad x \neq 0$$

$$\frac{d}{dx}\coth^{-1} x = \frac{1}{1-x^2}, \quad |x| > 1$$

> **Striking observation:** $\tanh^{-1}$ and $\coth^{-1}$ have the same formal derivative $\frac{1}{1-x^2}$, but on disjoint domains ($|x|<1$ vs $|x|>1$). Together they provide the complete antiderivative of $\frac{1}{1-x^2}$ on $\mathbb{R}\setminus{\pm 1}$.

---

### Full Proof: Three Methods for $(\sinh^{-1} x)'$

**Method 1: Implicit Differentiation** _(cleanest)_

Let $y = \sinh^{-1} x$, so $\sinh y = x$. Differentiate both sides with respect to $x$:

$$\cosh y \cdot \frac{dy}{dx} = 1 \implies \frac{dy}{dx} = \frac{1}{\cosh y}$$

Now use the fundamental identity $\cosh^2 y - \sinh^2 y = 1$:

$$\cosh y = \sqrt{1 + \sinh^2 y} = \sqrt{1 + x^2}$$

(positive square root since $\cosh y > 0$ always). Therefore:

$$\frac{d}{dx}\sinh^{-1} x = \frac{1}{\sqrt{1+x^2}} \qquad \square$$

**Method 2: Differentiate the Logarithm Directly** _(confirms the formula)_

$$\frac{d}{dx}\ln!\left(x + \sqrt{x^2+1}\right) = \frac{1}{x + \sqrt{x^2+1}} \cdot \frac{d}{dx}!\left(x + \sqrt{x^2+1}\right)$$

The inner derivative:

$$\frac{d}{dx}!\left(x + \sqrt{x^2+1}\right) = 1 + \frac{x}{\sqrt{x^2+1}} = \frac{\sqrt{x^2+1} + x}{\sqrt{x^2+1}}$$

So:

$$\frac{d}{dx}\sinh^{-1} x = \frac{1}{x+\sqrt{x^2+1}} \cdot \frac{x + \sqrt{x^2+1}}{\sqrt{x^2+1}} = \frac{1}{\sqrt{x^2+1}} \qquad \square$$

**Method 3: Inverse Function Theorem** _(most general)_

If $f$ is differentiable and bijective, $(f^{-1})'(x) = \dfrac{1}{f'(f^{-1}(x))}$.

Here $f = \sinh$, $f' = \cosh$, $f^{-1} = \sinh^{-1}$:

$$(\sinh^{-1})'(x) = \frac{1}{\cosh(\sinh^{-1} x)} = \frac{1}{\sqrt{1+x^2}} \qquad \square$$

---

### Proof for $(\cosh^{-1} x)'$

Let $y = \cosh^{-1} x \geq 0$, so $\cosh y = x$. Implicit differentiation:

$$\sinh y \cdot \frac{dy}{dx} = 1 \implies \frac{dy}{dx} = \frac{1}{\sinh y}$$

From the identity: $\sinh y = \sqrt{\cosh^2 y - 1} = \sqrt{x^2-1}$ (positive root since $y \geq 0 \Rightarrow \sinh y \geq 0$):

$$\frac{d}{dx}\cosh^{-1} x = \frac{1}{\sqrt{x^2-1}}, \quad x > 1 \qquad \square$$

---

### Proof for $(\tanh^{-1} x)'$

Let $y = \tanh^{-1} x$, so $\tanh y = x$. Implicit differentiation:

$$\operatorname{sech}^2 y \cdot \frac{dy}{dx} = 1 \implies \frac{dy}{dx} = \frac{1}{\operatorname{sech}^2 y} = \cosh^2 y$$

Use $1 - \tanh^2 y = \operatorname{sech}^2 y$:

$$\operatorname{sech}^2 y = 1 - x^2 \implies \frac{dy}{dx} = \frac{1}{1-x^2} \qquad \square$$

## ¬ß 5 ‚Äî Integration: The Payoff

The derivatives immediately give antiderivatives. These are the standard integral forms that appear throughout calculus.

### Standard Forms

$$\int \frac{dx}{\sqrt{x^2+a^2}} = \sinh^{-1}!\left(\frac{x}{a}\right) + C = \ln!\left(x + \sqrt{x^2+a^2}\right) + C, \quad a > 0$$

$$\int \frac{dx}{\sqrt{x^2-a^2}} = \cosh^{-1}!\left(\frac{x}{a}\right) + C = \ln!\left(x + \sqrt{x^2-a^2}\right) + C, \quad x > a > 0$$

$$\int \frac{dx}{a^2-x^2} = \frac{1}{a}\tanh^{-1}!\left(\frac{x}{a}\right) + C, \quad |x| < a$$

$$\int \frac{dx}{\sqrt{a^2-x^2}} = \sin^{-1}!\left(\frac{x}{a}\right) + C \quad \text{(circular, for contrast)}$$

> **The pattern:** $\sqrt{x^2 + a^2}$ ‚Üí use $\sinh^{-1}$. $\sqrt{x^2 - a^2}$ ‚Üí use $\cosh^{-1}$. $a^2 - x^2$ in denominator ‚Üí use $\tanh^{-1}$.

**Proof of the first form.** Let $u = x/a$:

$$\int \frac{dx}{\sqrt{x^2+a^2}} = \int \frac{a, du}{\sqrt{a^2 u^2 + a^2}} = \int \frac{a, du}{a\sqrt{u^2+1}} = \int \frac{du}{\sqrt{u^2+1}} = \sinh^{-1}(u) + C \qquad \square$$

---

### Worked Examples

**Example 5.1.** Evaluate $\displaystyle\int_0^2 \frac{dx}{\sqrt{x^2+4}}$.

$$= \left[\sinh^{-1}!\left(\frac{x}{2}\right)\right]_0^2 = \sinh^{-1}(1) - \sinh^{-1}(0) = \ln(1+\sqrt{2}) - 0 = \ln(1+\sqrt{2})$$

Numerically: $\approx 0.8814$.

---

**Example 5.2.** Evaluate $\displaystyle\int \frac{dx}{4-x^2}$.

Here $a = 2$, $|x| < 2$:

$$= \frac{1}{2}\tanh^{-1}!\left(\frac{x}{2}\right) + C = \frac{1}{4}\ln!\left(\frac{2+x}{2-x}\right) + C$$

_Double-check by partial fractions:_

$$\frac{1}{4-x^2} = \frac{1}{(2-x)(2+x)} = \frac{1}{4}\left(\frac{1}{2-x} + \frac{1}{2+x}\right)$$

Integrating: $\frac{1}{4}\left(-\ln|2-x| + \ln|2+x|\right) + C = \frac{1}{4}\ln\frac{2+x}{2-x} + C$. ‚úì

---

**Example 5.3.** Evaluate $\displaystyle\int \frac{x, dx}{\sqrt{x^4+1}}$.

Let $u = x^2$, $du = 2x, dx$:

$$= \frac{1}{2}\int \frac{du}{\sqrt{u^2+1}} = \frac{1}{2}\sinh^{-1}(x^2) + C = \frac{1}{2}\ln!\left(x^2 + \sqrt{x^4+1}\right) + C$$

---

**Example 5.4.** (Completing the square) Evaluate $\displaystyle\int \frac{dx}{\sqrt{x^2 - 4x + 5}}$.

Complete the square: $x^2 - 4x + 5 = (x-2)^2 + 1$. Let $u = x - 2$:

$$= \int \frac{du}{\sqrt{u^2+1}} = \sinh^{-1}(u) + C = \sinh^{-1}(x-2) + C = \ln!\left((x-2) + \sqrt{x^2-4x+5}\right) + C$$

## ¬ß 6 ‚Äî Identities and Relations

### Reflection Identities

$$\sinh^{-1}(-x) = -\sinh^{-1}(x) \quad \text{(odd)}$$

$$\tanh^{-1}(-x) = -\tanh^{-1}(x) \quad \text{(odd)}$$

$$\cosh^{-1}(x) \geq 0 \quad \text{(not odd ‚Äî domain is } [1,\infty)\text{)}$$

### Reciprocal Relations

$$\operatorname{csch}^{-1} x = \sinh^{-1}!\left(\frac{1}{x}\right), \quad x \neq 0$$

$$\operatorname{sech}^{-1} x = \cosh^{-1}!\left(\frac{1}{x}\right), \quad 0 < x \leq 1$$

$$\coth^{-1} x = \tanh^{-1}!\left(\frac{1}{x}\right), \quad |x| > 1$$

**Proof of the first:** Let $y = \operatorname{csch}^{-1} x$, so $\operatorname{csch} y = x$, i.e., $\sinh y = 1/x$. This means $y = \sinh^{-1}(1/x)$. $\square$

### Linking $\tanh^{-1}$ to $\coth^{-1}$

Note $\tanh^{-1}$ and $\coth^{-1}$ share the derivative $\frac{1}{1-x^2}$ on complementary domains. Together:

$$\int \frac{dx}{1-x^2} = \begin{cases} \tanh^{-1} x + C & |x| < 1 \ \coth^{-1} x + C & |x| > 1 \end{cases} = \frac{1}{2}\ln\left|\frac{1+x}{1-x}\right| + C$$

The last form is valid on all of $\mathbb{R} \setminus {\pm 1}$ and unifies both cases.

## ¬ß 7 ‚Äî Taylor Series of Inverse Hyperbolic Functions

For $|x| < 1$:

$$\sinh^{-1} x = \sum_{n=0}^{\infty} \frac{(-1)^n (2n)!}{4^n (n!)^2 (2n+1)} x^{2n+1} = x - \frac{x^3}{6} + \frac{3x^5}{40} - \frac{15x^7}{336} + \cdots$$

This follows from the binomial series applied to $(1+x^2)^{-1/2}$, then integrated term by term:

$$\frac{1}{\sqrt{1+x^2}} = \sum_{n=0}^{\infty} \binom{-1/2}{n} x^{2n} = 1 - \frac{x^2}{2} + \frac{3x^4}{8} - \cdots$$

Integrating:

$$\sinh^{-1} x = \int_0^x \frac{dt}{\sqrt{1+t^2}} = x - \frac{x^3}{6} + \frac{3x^5}{40} - \cdots$$

For $|x| < 1$:

$$\tanh^{-1} x = \sum_{n=0}^{\infty} \frac{x^{2n+1}}{2n+1} = x + \frac{x^3}{3} + \frac{x^5}{5} + \cdots$$

**Proof** (from the logarithm form):

$$\tanh^{-1} x = \frac{1}{2}\ln\frac{1+x}{1-x} = \frac{1}{2}\left[\ln(1+x) - \ln(1-x)\right]$$

Using $\ln(1+u) = u - u^2/2 + u^3/3 - \cdots$:

$$= \frac{1}{2}\left[\left(x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots\right) - \left(-x - \frac{x^2}{2} - \frac{x^3}{3} - \cdots\right)\right] = x + \frac{x^3}{3} + \frac{x^5}{5} + \cdots \qquad \square$$

> **Personal note:** This is the **Leibniz-Gregory series** for $\pi/4$: setting $x = 1$ (formally, as the series converges at the boundary) gives $\tanh^{-1}(1)$... but $\tanh^{-1}(1) = +\infty$. The analogous identity for $\arctan$ gives $\pi/4 = 1 - 1/3 + 1/5 - \cdots$ ‚Äî note the alternating signs, reflecting the circular vs. hyperbolic difference.

## ¬ß 8 ‚Äî Comparison: Circular vs. Hyperbolic Inverses

|Property|$\arcsin x$|$\sinh^{-1} x$|
|---|---|---|
|Domain|$[-1, 1]$|$\mathbb{R}$|
|Range|$[-\pi/2, \pi/2]$|$\mathbb{R}$|
|Derivative|$\frac{1}{\sqrt{1-x^2}}$|$\frac{1}{\sqrt{1+x^2}}$|
|Closed form|Not a logarithm|$\ln(x+\sqrt{x^2+1})$|

|Property|$\arctan x$|$\tanh^{-1} x$|
|---|---|---|
|Domain|$\mathbb{R}$|$(-1, 1)$|

## ¬ß 9 ‚Äî Connection to AI, Machine Learning, Deep Learning

### 9.1 ‚Äî Inverse $\tanh$ in Policy Optimization (Reinforcement Learning)

In reinforcement learning, particularly **Soft Actor-Critic (SAC)** and similar algorithms, the agent's policy must output actions in a bounded range, say $[-1, 1]$. A common trick:

1. Sample $z \sim \mathcal{N}(0, I)$ (Gaussian, unbounded)
2. Apply $a = \tanh(z)$ to squash to $(-1, 1)$

But training requires computing the **log-probability** of $a$ under the original distribution. By the change-of-variables formula:

$$\log p(a) = \log p(z) - \sum_i \log!\left|\frac{d(\tanh z_i)}{dz_i}\right| = \log p(z) - \sum_i \log(1 - \tanh^2 z_i)$$

During inference, given observed action $a$, we need $z = \tanh^{-1}(a)$ ‚Äî the inverse! This is numerically computed as:

$$z = \tanh^{-1}(a) = \frac{1}{2}\ln\frac{1+a}{1-a}$$

Implementation note: near $|a| = 1$, this is numerically unstable. In practice:

```python
# PyTorch / JAX safe implementation
def atanh_safe(x, eps=1e-6):
    x_clipped = x.clamp(-1 + eps, 1 - eps)
    return 0.5 * (torch.log1p(x_clipped) - torch.log1p(-x_clipped))
```

### 9.2 ‚Äî $\sinh^{-1}$ as a Variance-Stabilizing Transform

In financial machine learning and regression with heavy-tailed targets, the transform:

$$\tilde{y} = \sinh^{-1}(y) = \ln!\left(y + \sqrt{y^2+1}\right)$$

is preferred over $\log(y)$ because:

- **Defined for all reals** (returns can be negative)
- **Behaves like $\ln(2y)$** for large $y$ ‚Äî compresses scale
- **Behaves like $y$** near zero ‚Äî preserves small values

This is used in Kaggle competitions involving financial targets, where the evaluation metric is often computed on transformed values.

### 9.3 ‚Äî $\tanh^{-1}$ and Fisher's Z-Transformation

In statistics (which underpins all of ML evaluation), **Fisher's Z-transformation** of a correlation coefficient $r \in (-1, 1)$ is:

$$Z = \tanh^{-1}(r) = \frac{1}{2}\ln\frac{1+r}{1-r}$$

This stabilizes the variance of the sample correlation, making it approximately normally distributed. Used in:

- Hypothesis testing for correlations
- Confidence intervals for model performance metrics
- Comparing feature correlations across datasets in exploratory data analysis

### 9.4 ‚Äî Gradient Flow Through $\sinh^{-1}$ Activations

Recently (2020s), **$\sinh^{-1}$ (arcsinh)** has been proposed as an activation function in neural networks. Its derivative is:

$$\frac{d}{dx}\sinh^{-1}(x) = \frac{1}{\sqrt{x^2+1}}$$

Key properties as an activation:

- **Not bounded** (unlike $\tanh$) ‚Äî avoids saturation and vanishing gradients
- **Sublinear growth** ‚Äî gradient $\frac{1}{\sqrt{x^2+1}} \to 0$ slowly as $|x| \to \infty$ ‚Äî does not explode like ReLU for large inputs
- **Smooth and odd** ‚Äî zero-centered, differentiable everywhere (unlike ReLU)

Compared to popular activations:

| Activation | Range | Gradient at large $|x|$ | Smooth? | |---|---|---|---| | ReLU | $[0, \infty)$ | 0 or 1 | No (kink at 0) | | $\tanh$ | $(-1,1)$ | $\to 0$ (saturates) | Yes | | GELU | $\approx \mathbb{R}$ | $\to 1$ | Yes | | $\sinh^{-1}$ (arcsinh) | $\mathbb{R}$ | $\to 0$ slowly | Yes |

### 9.5 ‚Äî Hyperbolic Geometry in Representation Learning

The **Poincar√© disk model** of hyperbolic geometry uses the metric:

$$ds^2 = \frac{4(dx^2 + dy^2)}{(1-|x|^2)^2}$$

Distances in this space involve $\cosh^{-1}$:

$$d(u, v) = \cosh^{-1}!\left(1 + 2\frac{|u-v|^2}{(1-|u|^2)(1-|v|^2)}\right)$$

Why does ML care? Hyperbolic space has **exponentially growing volume** ‚Äî perfect for embedding trees and hierarchies. The number of nodes at depth $d$ in a tree grows as $b^d$; hyperbolic space can accommodate this exponential branching with low distortion.

**Poincar√© embeddings** (Nickel & Kiela, NeurIPS 2017) use $\cosh^{-1}$ in the loss function to embed WordNet noun hierarchy with far fewer dimensions than Euclidean embeddings require.

```
Euclidean space:  volume grows as r^n  (polynomial)
Hyperbolic space: volume grows as e^r  (exponential)
Trees:            nodes grow as b^d    (exponential)
‚Üí Hyperbolic space is the natural home for hierarchical data
```

## ¬ß 10 ‚Äî A Subtlety: Two Antiderivative Conventions

Consider $\displaystyle\int \frac{dx}{1-x^2}$.

**Via partial fractions:**

$$\frac{1}{1-x^2} = \frac{1}{(1-x)(1+x)} = \frac{1}{2}\left(\frac{1}{1+x} + \frac{1}{1-x}\right)$$

$$\int \frac{dx}{1-x^2} = \frac{1}{2}\ln\frac{1+x}{1-x} + C = \frac{1}{2}\ln!\left|\frac{1+x}{1-x}\right| + C$$

**Via inverse hyperbolic:** On $|x| < 1$: $= \tanh^{-1}(x) + C$. On $|x| > 1$: $= \coth^{-1}(x) + C$.

Both are correct ‚Äî they differ by a constant on each connected component of the domain. The absolute value version is the most general single formula.

> **Personal note:** This subtlety trips up many students. The issue is that $\ln$ requires a positive argument, but $\frac{1+x}{1-x}$ is negative when $|x| > 1$. The absolute value fixes this while keeping the formula unified. Always specify the domain when writing antiderivatives of $\frac{1}{1-x^2}$.

$$(\sinh^{-1} x)' = \frac{1}{\sqrt{x^2+1}}, \quad (\cosh^{-1} x)' = \frac{1}{\sqrt{x^2-1}}, \quad (\tanh^{-1} x)' = \frac{1}{1-x^2}$$

### Integral Forms

$$\int \frac{dx}{\sqrt{x^2+a^2}} = \sinh^{-1}!\tfrac{x}{a}+C, \quad \int\frac{dx}{\sqrt{x^2-a^2}} = \cosh^{-1}!\tfrac{x}{a}+C, \quad \int\frac{dx}{a^2-x^2}=\tfrac{1}{a}\tanh^{-1}!\tfrac{x}{a}+C$$
