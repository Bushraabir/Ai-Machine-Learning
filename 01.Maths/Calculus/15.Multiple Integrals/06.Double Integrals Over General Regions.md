## §0 — Personal Preamble

When we first define $\iint_R f,dA$ over a **rectangle**, life is pleasant: both limits are constants and Fubini's theorem is almost a triviality. The interesting mathematics begins the moment the region $D$ stops being a box. This note develops the full rigorous theory, catalogues the practical techniques, supplies worked examples, and — because it cannot be avoided — explains why this machinery is exactly what gradient descent and the study of probability distributions secretly rely on.

All references are to **Stewart, _Calculus_ (8e)** (§§15.2–15.3) and **MIT 18.02** lecture notes.

---

## §1 — Motivation: Why Leave the Rectangle?

In applications the domain of integration is almost never a rectangle.

- A probability density $p(x,y)$ is integrated over the region ${(x,y): x^2 + y^2 \le 1}$.
- A neural network's weight space carries a prior supported on a simplex.
- A fluid-dynamics calculation integrates pressure over the cross-section of a pipe.

We need a definition of $\iint_D f,dA$ that handles **arbitrary bounded regions** $D \subset \mathbb{R}^2$.

---

## §2 — The Rigorous Definition (Extension by Zero)

**Setup.** Let $D \subset \mathbb{R}^2$ be a bounded region and $f : D \to \mathbb{R}$ bounded. Enclose $D$ in a rectangle $R = [a,b]\times[c,d]$.

**Define the extended function:**

$$ F(x,y) = \begin{cases} f(x,y) & \text{if } (x,y) \in D \ 0 & \text{if } (x,y) \in R \setminus D \end{cases} $$

**Definition (Stewart §15.2).** If $F$ is integrable over $R$, we set

$$ \boxed{\iint_D f,dA ;:=; \iint_R F,dA} $$

**Remark.** The value is independent of which rectangle $R$ we choose (since $F = 0$ outside $D$). The non-trivial part is proving $F$ is integrable — this requires the boundary $\partial D$ to have **measure zero** (e.g., $\partial D$ is a piecewise-smooth curve), which covers every region we encounter in practice.

---

## §3 — Type I and Type II Regions

The payoff of the abstract definition is the following concrete evaluation theorem. We isolate two archetypal shapes.

### 3.1 Type I Region

$$ D = {(x,y) : a \le x \le b,; g_1(x) \le y \le g_2(x)} $$

```
y
│      g₂(x)
│    ╭──────╮
│   /        \
│  /  D       \
│  \          /
│   ╰──────╯
│      g₁(x)
│
└───────────────── x
    a           b
```

**Theorem (Fubini, Type I).** If $f$ is continuous on $D$:

$$ \iint_D f(x,y),dA = \int_a^b \int_{g_1(x)}^{g_2(x)} f(x,y),dy,dx $$

**Proof sketch.** The extension-by-zero $F$ equals $f$ when $g_1(x) \le y \le g_2(x)$ and $0$ elsewhere. By Fubini on the rectangle:

$$ \iint_R F,dA = \int_a^b \left[\int_c^d F(x,y),dy\right]dx = \int_a^b \left[\int_{g_1(x)}^{g_2(x)} f(x,y),dy\right]dx \qquad \square $$

### 3.2 Type II Region

$$ D = {(x,y) : c \le y \le d,; h_1(y) \le x \le h_2(y)} $$

```
y
d ─────────────────╮
│                   │
│      D            │
│                   │
c ─────────────────╯
│  h₁(y)      h₂(y)
└──────────────────── x
```

**Theorem (Fubini, Type II).** If $f$ is continuous on $D$:

$$ \iint_D f(x,y),dA = \int_c^d \int_{h_1(y)}^{h_2(y)} f(x,y),dx,dy $$

## §4 — Properties (The Algebra of Double Integrals)

Let $D, D_1, D_2$ be regions and $f, g$ integrable.

|Property|Statement|
|---|---|
|Linearity|$\iint_D (\alpha f + \beta g),dA = \alpha\iint_D f,dA + \beta\iint_D g,dA$|
|Additivity|If $D = D_1 \cup D_2$, $D_1 \cap D_2$ has area 0: $\iint_D f,dA = \iint_{D_1} f,dA + \iint_{D_2} f,dA$|
|Monotonicity|$f \le g$ on $D \Rightarrow \iint_D f,dA \le \iint_D g,dA$|
|Area formula|$\iint_D 1,dA = \text{Area}(D)$|
|Estimation|$m\cdot\text{Area}(D) \le \iint_D f,dA \le M\cdot\text{Area}(D)$ when $m \le f \le M$|

## §5 — Worked Examples

### Example 1 — Triangle (Type I)

**Compute** $\displaystyle\iint_D x,dA$ where $D$ is the triangle with vertices $(0,0)$, $(1,0)$, $(1,1)$.

**Setting up the region.** The boundary lines are $x=1$, $y=0$, and $y=x$. As a **Type I** region:

$$ D = {(x,y) : 0 \le x \le 1,; 0 \le y \le x} $$

```
y
1 ─        ╱│
│         ╱ │
│        ╱  │
│       ╱ D │
│      ╱    │
0 ────╱─────┤── x
     0      1
```

**Evaluation.**

$$ \iint_D x,dA = \int_0^1 \int_0^x x,dy,dx = \int_0^1 x\cdot[y]_0^x,dx = \int_0^1 x^2,dx = \frac{1}{3} $$

**Check via Type II.** Rewrite as $D = {0 \le y \le 1,; y \le x \le 1}$:

$$ \int_0^1 \int_y^1 x,dx,dy = \int_0^1 \frac{1-y^2}{2},dy = \frac{1}{2}\left[y - \frac{y^3}{3}\right]_0^1 = \frac{1}{3} \checkmark $$

---

### Example 2 — Reversing the Order of Integration

**Compute** $\displaystyle\int_0^1 \int_x^1 e^{y^2},dy,dx$.

The inner integral $\int_x^1 e^{y^2},dy$ has **no closed form** in this order. Reverse order.

**The region.** The original order (Type I) describes:

$$ D = {(x,y) : 0 \le x \le 1,; x \le y \le 1} $$

As **Type II**:

$$ D = {(x,y) : 0 \le y \le 1,; 0 \le x \le y} $$

```
y
1 ─╮
│╲ │
│ ╲│ D
│  ╲
│   ╲
0 ───╲─── x
0     1
```

**Evaluation.**

$$ \int_0^1 \int_0^y e^{y^2},dx,dy = \int_0^1 y,e^{y^2},dy = \frac{1}{2}\left[e^{y^2}\right]_0^1 = \frac{e-1}{2} $$

> **Key lesson.** Switching the order of integration is not a trick — it is a rigorous operation guaranteed by Fubini's theorem. The limits change precisely because the _same_ region $D$ is described in two different orders.

---

### Example 3 — Parabolic Region

**Compute** $\displaystyle\iint_D (x+y),dA$ where $D = {(x,y) : 0 \le x \le 1,; x^2 \le y \le \sqrt{x}}$.

Note $y = x^2$ and $y = \sqrt{x}$ intersect at $(0,0)$ and $(1,1)$, and $x^2 \le \sqrt{x}$ on $[0,1]$.

```
y
1 ─────────────────╮  ← y = √x
│          ╭───╯
│       ╭──╯   D
│    ╭──╯
│ ╭──╯  ← y = x²
0 ─────────────────── x
  0                 1
```

$$ \iint_D (x+y),dA = \int_0^1\int_{x^2}^{\sqrt{x}}(x+y),dy,dx $$

**Inner integral:**

$$ \int_{x^2}^{\sqrt{x}}(x+y),dy = \left[xy + \frac{y^2}{2}\right]_{x^2}^{\sqrt{x}} = \left(x^{3/2} + \frac{x}{2}\right) - \left(x^3 + \frac{x^4}{2}\right) $$

**Outer integral:**

$$ \int_0^1\left(x^{3/2} + \frac{x}{2} - x^3 - \frac{x^4}{2}\right)dx = \frac{2}{5} + \frac{1}{4} - \frac{1}{4} - \frac{1}{10} = \frac{3}{10} $$

---

### Example 4 — Non-elementary Region via Additivity

**Compute the area** of the region enclosed between $y = \sin x$ and $y = \cos x$ on $[0, \pi]$.

The curves cross at $x = \pi/4$. Split $D = D_1 \cup D_2$:

$$ \text{Area} = \int_0^{\pi/4}(\cos x - \sin x),dx + \int_{\pi/4}^{\pi}(\sin x - \cos x),dx $$

$$ = [\sin x + \cos x]_0^{\pi/4} + [-\cos x - \sin x]_{\pi/4}^\pi = (\sqrt{2}-1) + (1+\sqrt{2}) = 2\sqrt{2} $$

## §6 — The Non-Obvious Subtlety: When Fubini Fails

Fubini's theorem requires **absolute integrability**: $\iint_D |f|,dA < \infty$.

**Counterexample (Sierpiński-type).** Define

$$ f(x,y) = \frac{x^2 - y^2}{(x^2+y^2)^2}, \quad D = [0,1]^2 $$

One can verify:

$$ \int_0^1\left(\int_0^1 f,dy\right)dx = \frac{\pi}{4}, \qquad \int_0^1\left(\int_0^1 f,dx\right)dy = -\frac{\pi}{4} $$

The iterated integrals are **unequal** because $\iint_D |f|,dA = \infty$. This is the canonical warning that order cannot be swapped carelessly.

## §7 — Connection to Probability Theory

Let $(X, Y)$ be a jointly continuous random vector with density $p(x,y) \ge 0$.

The **marginal density** of $X$ is:

$$ p_X(x) = \int_{-\infty}^\infty p(x,y),dy $$

This is precisely the inner integral of a double integral over the strip ${X = x}$. Computing $\mathbb{E}[g(X,Y)]$ is:

$$ \mathbb{E}[g(X,Y)] = \iint_{\mathbb{R}^2} g(x,y),p(x,y),dA $$

Fubini's theorem is invoked _every time_ we compute a marginal or compute an expectation by iterated integration.

## §8 — Connection to AI / Machine Learning / Deep Learning

This is not decorative. Double integrals over general regions are structurally embedded in modern ML.

### 8.1 Continuous Normalizing Flows (CNFs)

A **normalizing flow** maps a simple distribution $p_Z$ to a complex distribution $p_X$ via an invertible map $\phi: \mathbb{R}^d \to \mathbb{R}^d$. The probability of $X \in D$ is:

$$ \Pr(X \in D) = \iint_D p_X(x,y),dA = \iint_{\phi^{-1}(D)} p_Z(z_1,z_2),|\det J_\phi|,dA $$

The change of variables — a multivariable generalization of substitution in double integrals — is the mathematical core of this technique. The region $\phi^{-1}(D)$ is typically not a rectangle: it is a **general region** determined by the geometry of the flow.

### 8.2 Gaussian Processes and Kernel Machines

The covariance operator in a Gaussian Process is:

$$ (Kf)(x) = \int_D k(x, x') f(x'),dx' $$

For 2D spatial data (images, spatial predictions), this becomes:

$$ (Kf)(x_1, x_2) = \iint_D k!\left((x_1,x_2),(x_1',x_2')\right) f(x_1',x_2'),dx_1',dx_2' $$

The domain $D$ is the support of the data — often a non-rectangular region like a geographic boundary or an image mask.

### 8.3 Variational Inference and the ELBO

The **Evidence Lower Bound** in variational autoencoders is:

$$ \mathcal{L}(q) = \mathbb{E}_{q(z)}!\left[\log p(x \mid z)\right] - \mathrm{KL}(q(z) | p(z)) $$

Both terms involve integrals of the form $\iint q(z) \log[\cdot],dz$. In practice, the variational family $q$ is a Gaussian, and the reparameterisation trick converts this into a double integral over a rectangle in the $(\mu, \sigma)$ parameter space — but the original problem lives over a general latent region.

### 8.4 Monte Carlo Integration: Numerical Double Integrals

When the region $D$ is complex (e.g., the feasible region of a constrained optimisation), double integrals are estimated by:

$$ \iint_D f,dA \approx \frac{\text{Area}(D)}{N}\sum_{i=1}^N f(x_i, y_i), \quad (x_i, y_i) \sim \text{Uniform}(D) $$

Sampling uniformly from a _general_ region $D$ — not a rectangle — is a non-trivial computational geometry problem, and the correctness of the estimator rests on the theory of this section.

### 8.5 Convolutional Neural Networks (CNFs) — Continuous Limit

A 2D convolution is:

$$ (f * g)(x, y) = \iint_{\mathbb{R}^2} f(x - x', y - y'),g(x', y'),dA $$

In the continuous limit of a CNN, the "receptive field" of a neuron is a bounded, possibly non-rectangular region $D$, and the activation is $\iint_D f(x-x', y-y'),g(x',y'),dA$. Understanding how varying $D$ changes the integral is precisely the theory developed here.

## §10 — Theoretical Depth: Measure-Theoretic Footnotes

_(For the reader who wants the full story.)_

**Why does $\partial D$ having measure zero matter?** The Riemann integral on $R$ exists iff the set of discontinuities of $F$ has **Lebesgue measure zero** (Lebesgue's theorem on Riemann integrability). The discontinuities of $F$ are precisely the points of $\partial D$ where $f \ne 0$, so if $\partial D$ has measure zero, $F$ is Riemann integrable.

**Lebesgue's criterion.** $F : R \to \mathbb{R}$ is Riemann integrable $\iff$ $F$ is bounded and the set of its discontinuities has Lebesgue measure zero.

This is why **any region bounded by finitely many smooth curves** is legal: a smooth curve in $\mathbb{R}^2$ has 2D Lebesgue measure zero.

**Fubini–Tonelli.** In Lebesgue theory, for $f \ge 0$ measurable (Tonelli) or $\int\int |f|,dA < \infty$ (Fubini):

$$ \int_D f,d(m_1 \times m_2) = \int_a^b \left[\int_{g_1(x)}^{g_2(x)} f(x,y),dm_1(y)\right]dm_1(x) $$

The Riemann version we use is a special case.

## §11 — Common Mistakes and Pathologies

|Mistake|What Goes Wrong|Remedy|
|---|---|---|
|Swapping limits without re-drawing|Describes a different region or the empty set|Always re-derive limits from a fresh sketch|
|Forgetting to check $g_1 \le g_2$|Negative area contribution|Verify inequality holds on the entire $x$-interval|
|Splitting at the wrong $x$-value|Region mis-described|Find intersections of boundary curves first|
|Applying Fubini when $\int\int|f|= \infty$|
|Treating a non-Type-I region as Type I|Variable limits depend on $y$, not $x$|Check: are the top/bottom bounds functions of $x$ only?|

## §12 — Summary Theorem Sheet

**Theorem 1 (Fubini, Type I).** If $f$ is continuous on $D = {a \le x \le b,; g_1(x) \le y \le g_2(x)}$: $$\iint_D f,dA = \int_a^b\int_{g_1(x)}^{g_2(x)} f(x,y),dy,dx$$

**Theorem 2 (Fubini, Type II).** If $f$ is continuous on $D = {c \le y \le d,; h_1(y) \le x \le h_2(y)}$: $$\iint_D f,dA = \int_c^d\int_{h_1(y)}^{h_2(y)} f(x,y),dx,dy$$

**Theorem 3 (Additivity).** If $D = D_1 \cup D_2$ and $D_1 \cap D_2$ has area zero: $$\iint_D f,dA = \iint_{D_1} f,dA + \iint_{D_2} f,dA$$

**Theorem 4 (Area).** $\text{Area}(D) = \iint_D 1,dA$
