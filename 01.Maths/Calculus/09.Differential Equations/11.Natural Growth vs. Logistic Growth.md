## 1. The Two Models at a Glance

Both models govern a population $P(t)$ (or any growing quantity). They differ in one fundamental assumption: whether the **per-capita growth rate** is constant or declines as $P$ grows.

||**Natural Growth**|**Logistic Growth**|
|---|---|---|
|**ODE**|$\dfrac{dP}{dt} = kP$|$\dfrac{dP}{dt} = rP!\left(1 - \dfrac{P}{K}\right)$|
|**Per-capita rate**|$\dfrac{1}{P}\dfrac{dP}{dt} = k$ (constant)|$\dfrac{1}{P}\dfrac{dP}{dt} = r!\left(1 - \dfrac{P}{K}\right)$ (decreasing)|
|**Solution**|$P(t) = P_0 e^{kt}$|$P(t) = \dfrac{K}{1 + \left(\frac{K-P_0}{P_0}\right)e^{-rt}}$|
|**Long-run behavior**|$P \to \infty$ (if $k>0$)|$P \to K$|
|**Type of ODE**|Separable, linear|Separable, nonlinear|
|**Parameters**|$k$ (growth rate), $P_0$|$r$ (intrinsic rate), $K$ (carrying capacity), $P_0$|

## 2. Natural Growth — The Malthusian Model

### 2.1 The ODE and Its Meaning

$$\frac{dP}{dt} = kP, \quad P(0) = P_0 \tag{1}$$

**Verbal reading:** the rate of change of population is proportional to the current population. Every individual reproduces at the same rate $k$, regardless of how crowded the environment is.

$k > 0$: growth. $k < 0$: decay (same equation models radioactive decay, drug clearance, etc.).

### 2.2 Derivation of the Solution

Separate variables ($P \neq 0$):

$$\frac{dP}{P} = k,dt \implies \ln|P| = kt + C_1 \implies P(t) = Ae^{kt}.$$

Apply $P(0) = P_0$: $A = P_0$.

$$\boxed{P(t) = P_0 e^{kt}}$$

The singular solution $P \equiv 0$ is the **extinction equilibrium** — unstable for $k > 0$, stable for $k < 0$.

### 2.3 Key Properties

**Doubling time** (for $k > 0$): Solve $P_0 e^{kt_d} = 2P_0$:

$$t_d = \frac{\ln 2}{k} \approx \frac{0.693}{k}$$

**Half-life** (for $k < 0$): $t_{1/2} = \dfrac{\ln 2}{|k|}$.

**Scale invariance:** $\dfrac{P(t+s)}{P(t)} = e^{ks}$ depends only on the elapsed time $s$, not on $t$. The model has no memory of when it started — it looks the same at every moment.

**Phase line:**

```
k > 0:
         dP/dt > 0       dP/dt > 0
────────────[P=0]────────────────────→  P
          unstable      (unbounded growth)

k < 0:
         dP/dt < 0       dP/dt < 0
←────────────[P=0]────────────────────  P
                        stable
```

### 2.4 When Is It Valid?

Natural growth is a **local approximation** — valid when $P \ll K$ (far below the carrying capacity). It is exact for:

- Radioactive decay
- Continuous compounding of interest
- Early-phase epidemic spread (before herd effects)
- First few generations of a bacterial colony

## 3. Logistic Growth — The Verhulst Model

### 3.1 The ODE and Its Meaning

$$\frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right), \quad P(0) = P_0 \tag{2}$$

**Verbal reading:** growth rate is $rP$ (Malthusian) multiplied by a **saturation factor** $(1 - P/K)$ that decreases linearly from $1$ (when $P = 0$, no competition) to $0$ (when $P = K$, fully saturated).

- $r > 0$: intrinsic growth rate (what the rate would be with no crowding)
- $K > 0$: **carrying capacity** — the population the environment can sustain
- The factor $(1 - P/K)$ encodes **intraspecific competition**: each individual slightly reduces everyone else's growth rate

### 3.2 Derivation of the Solution

Separate variables:

$$\frac{dP}{P(1 - P/K)} = r,dt$$

**Partial fractions** (multiply numerator and denominator by $K$):

$$\frac{K}{P(K - P)} = \frac{1}{P} + \frac{1}{K - P}$$

So:

$$\int \left(\frac{1}{P} + \frac{1}{K-P}\right)dP = \int r,dt$$

$$\ln|P| - \ln|K - P| = rt + C_1$$

$$\ln\left|\frac{P}{K-P}\right| = rt + C_1 \implies \frac{P}{K-P} = Ae^{rt}$$

Solve for $P$:

$$P = Ae^{rt}(K - P) \implies P(1 + Ae^{rt}) = KAe^{rt} \implies P = \frac{KAe^{rt}}{1 + Ae^{rt}} = \frac{K}{1 + A^{-1}e^{-rt}}$$

Apply $P(0) = P_0$: $A = \dfrac{P_0}{K - P_0}$, so $A^{-1} = \dfrac{K - P_0}{P_0}$.

$$\boxed{P(t) = \frac{K}{1 + \left(\dfrac{K - P_0}{P_0}\right)e^{-rt}}}$$

### 3.3 Key Properties

**Equilibria:** Set $dP/dt = 0$:

- $P^* = 0$ — **unstable** (for $r > 0$): any perturbation above 0 grows
- $P^* = K$ — **stable**: all solutions with $P_0 > 0$ converge to $K$

**Inflection point:** The growth rate $dP/dt = rP(1 - P/K)$ is a downward parabola in $P$, maximized at:

$$P = \frac{K}{2}, \quad \text{where} \quad \frac{dP}{dt}\bigg|_{\max} = \frac{rK}{4}$$

The inflection point of $P(t)$ occurs at $t^* = \dfrac{1}{r}\ln!\left(\dfrac{K-P_0}{P_0}\right)$ (when $P_0 < K/2$).

**S-shaped (sigmoid) curve:** $P(t)$ starts concave up (accelerating), inflects at $P = K/2$, then concave down (decelerating) toward $K$.

**Phase line:**

```
dP/dt
  │         ▲
  │       /   \
  │      /     \
  │     /       \
  │    /         \
  │   /           \
──┼──/─────────────\────→ P
  0               K

  ←────[P=0]──────────[P=K]────→  P
      unstable         stable
         ○ ─────────────── ●
```

### 3.4 The Linearization Near Equilibria

__Near $P^_ = K$:_* Let $P = K - u$ where $u \ll K$ (small perturbation below $K$):

$$\frac{du}{dt} = -\frac{dP}{dt} = -r(K-u)\frac{u}{K} \approx -ru \quad \text{(linearized)}$$

So $u(t) \approx u_0 e^{-rt}$ — the approach to $K$ is **exponential with rate $r$**. This is a key asymptotic result: the logistic model converges to carrying capacity at the same exponential rate $r$ it would have grown at if unconstrained.

## 4. Head-to-Head Comparison

### 4.1 The Per-Capita Growth Rate

This is the most illuminating comparison. Define $g(P) = \dfrac{1}{P}\dfrac{dP}{dt}$:

|Model|$g(P)$|Behavior|
|---|---|---|
|Natural growth|$k$|Horizontal line — constant|
|Logistic|$r(1 - P/K)$|Decreasing line — hits 0 at $P = K$|

```
g(P) = (1/P)(dP/dt)

k ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   ← Natural growth (constant)
  \
   \
    \
     \
      \
       \─────────────────────────── → P
       K                            ← Logistic (linear decline)
```

The logistic model's $g(P)$ is the **most natural first correction** to the Malthusian model: if resources are finite, the per-capita rate must decrease as population increases. The linear decrease in $g$ is the simplest possible such correction.

### 4.2 Solution Curves on the Same Axes

```
P(t)
K ─ ─ ─ ─ ─ ─ ─ ─ ─────────────────────────  ← logistic asymptote
                  /
               __/
            __/      ← logistic (S-curve)
         __/
K/2 ──__/────────────────────────────────────  ← inflection point
      /
     /
    /  ↗ ↗ ↗ ← natural growth (exponential, P₀ small)
   /↗↗↗
──/──────────────────────────────────────────→ t
P₀       t*

(Both curves coincide near t=0 when P₀ ≪ K)
```

**For small $t$ and $P_0 \ll K$:** $1 - P/K \approx 1$, so logistic $\approx$ natural growth. The two models are indistinguishable early on — this is why Malthus's model appeared correct before populations hit resource limits.

### 4.3 Structural Comparison Table

|Property|Natural Growth|Logistic|
|---|---|---|
|ODE order|1st|1st|
|Linearity|**Linear**|**Nonlinear**|
|Equilibria|One: $P=0$|Two: $P=0$ (unstable), $P=K$ (stable)|
|Solution curve shape|Pure exponential|Sigmoid (S-curve)|
|Long-run behavior|$\infty$|$K$|
|Inflection point|None|$P = K/2$|
|Parameters|2: $k, P_0$|3: $r, K, P_0$|
|Solvable in closed form|Yes (trivially)|Yes (partial fractions)|
|Biologically realistic|Only for $P \ll K$|For all $P \in (0, K)$|
|Self-correction|None|Yes — growth slows as $P \to K$|

## 5. Phase Portrait Analysis

The phase portrait plots $\dot{P} = dP/dt$ against $P$.

**Natural growth:** $\dot{P} = kP$ — a line through the origin. No equilibrium except $P = 0$.

**Logistic:** $\dot{P} = rP(1 - P/K)$ — a downward-opening parabola with roots at $P = 0$ and $P = K$.

```
dP/dt
  │
rK/4─ ─ ─ ─ ─ ─ ─●─ ─ ─ ─ ─           ← maximum growth rate
  │              /   \
  │             /     \
  │            /       \
  │           /         \
──┼──────────/───────────\──────────→ P
  │         /             \
  0        /               K
           ↑               ↑
       unstable          stable
       equilibrium      equilibrium
```

The area under the parabola (from $0$ to $K$) represents total growth over the population's history — a finite quantity, unlike the exponential case.

## 6. The Sigmoid Connection and the Logistic Function

Setting $K = 1$, $r = 1$, $P_0 = 1/2$ (so $A^{-1} = 1$):

$$P(t) = \frac{1}{1 + e^{-t}} =: \sigma(t)$$

This is the **standard logistic (sigmoid) function**. It satisfies:

$$\sigma'(t) = \sigma(t)(1 - \sigma(t))$$

which is exactly the logistic ODE with $K = 1$, $r = 1$. The sigmoid is not just an activation function chosen for computational convenience — it is the **exact closed-form solution of the logistic growth ODE**, and inherits all of the qualitative properties above.

## 7. Connection to AI, Machine Learning & Deep Learning

---

### 7.1 Sigmoid Activation — The Logistic Solution in Every Neuron

As shown above, every forward pass through a sigmoid neuron evaluates $\sigma(x) = (1 + e^{-x})^{-1}$, which is the logistic solution with $K = 1$. The properties that make logistic growth biologically sensible are exactly the properties that make sigmoid activations useful:

- **Bounded output** ($0 < \sigma < 1$): prevents unbounded activation, analogous to $P$ bounded by $K$.
- **Soft saturation near 0 and 1**: gradient nearly zero far from the inflection — the activation is "decided."
- **Linear near the inflection** ($\sigma \approx 1/2$, i.e., $x \approx 0$): maximum sensitivity, analogous to maximum growth rate at $P = K/2$.

The **vanishing gradient problem** in deep networks is the biological carrying-capacity effect in disguise: when $P \approx K$ (neuron saturated), $dP/dt \approx 0$ — the gradient through a saturated sigmoid neuron is nearly zero, slowing or stopping learning.

---

### 7.2 Logistic Regression — Modeling Class Probability

In binary classification, logistic regression models:

$$P(Y=1 \mid \mathbf{x}) = \sigma(\mathbf{w}^\top \mathbf{x} + b) = \frac{1}{1 + e^{-(\mathbf{w}^\top \mathbf{x} + b)}}$$

The logistic curve is chosen precisely because:

- It maps $\mathbb{R} \to (0,1)$ (valid probability)
- It is the **maximum entropy** model for binary outcomes with linear constraints on the log-odds
- Its derivative $\sigma'(\cdot) = \sigma(\cdot)(1 - \sigma(\cdot))$ is clean and makes the cross-entropy gradient tractable

The logistic regression model is the logistic ODE evaluated at $t = \mathbf{w}^\top \mathbf{x} + b$.

---

### 7.3 Neural Scaling Laws — Logistic vs. Power Law Growth

Empirical **scaling laws** (Kaplan et al., 2020) show that test loss $\mathcal{L}$ of large language models decreases as a power law in compute $C$:

$$\mathcal{L}(C) \sim C^{-\alpha}$$

But for a fixed task, performance (e.g., accuracy $A$) often follows a **logistic curve** in log-compute:

$$A(\log C) = \frac{A_{\max}}{1 + Be^{-r\log C}}$$

The carrying capacity $A_{\max}$ is the task ceiling — analogous to $K$. The model improves rapidly at first (exponential phase), then saturates as it approaches the ceiling. The inflection point $A = A_{\max}/2$ corresponds to the "half-compute" point — the sweet spot of maximum marginal gain per FLOP.

---

### 7.4 Training Dynamics — Loss Curves as Logistic Decay

The **training loss** of a neural network often follows a curve qualitatively similar to the logistic solution reflected: starting flat, then dropping steeply, then flattening again as it approaches the minimum. Near a local minimum, the linearized gradient flow gives:

$$\frac{d\mathcal{L}}{dt} = -\lambda \mathcal{L} \implies \mathcal{L}(t) = \mathcal{L}_{\min} + (\mathcal{L}_0 - \mathcal{L}_{\min})e^{-\lambda t}$$

which is the **natural growth/decay** equation. So early training (far from minimum) is logistic-like; late training (near minimum) is Malthusian (exponential) in the loss gap. The crossover between regimes is the inflection point of the loss curve.

---

### 7.5 Population-Based Optimization — Evolutionary Algorithms

In **genetic algorithms** and **evolutionary strategies**, the frequency of a fit genotype in a population follows the **logistic equation** exactly (Fisher's fundamental theorem of natural selection, 1930). The carrying capacity $K = 1$ (the genotype eventually fixes at 100% frequency), and $r$ is determined by the fitness differential.

Evolutionary optimization algorithms explicitly simulate logistic dynamics among candidate solutions — and the logistic model's inflection point corresponds to the phase transition from **exploration** (many diverse solutions) to **exploitation** (converging to the best).

---

### 7.6 Epidemic Models and Federated Learning

The **SIR epidemic model** contains a logistic subsystem:

$$\frac{dI}{dt} = \beta SI - \gamma I = I(\beta S - \gamma)$$

Early in the epidemic ($S \approx S_0 \approx N$), this is natural growth with rate $\beta S_0 - \gamma$. As $S$ depletes, growth slows and reverses — qualitatively logistic. The **basic reproduction number** $R_0 = \beta N / \gamma$ is the logistic $K/P_0$ analogue: $R_0 > 1$ means the "infection population" grows, $R_0 < 1$ means decay.

In **federated learning**, privacy-noise injection and client dropout create epidemic-like dynamics: information (gradient signal) "infects" aggregation rounds logistically, with $K$ set by the privacy budget.

## 8. Modified Growth Models — Beyond the Two Classics

Once we understand natural and logistic growth, we can classify modifications:

### 8.1 Gompertz Model

$$\frac{dP}{dt} = rP\ln!\left(\frac{K}{P}\right)$$

Per-capita rate: $g(P) = r\ln(K/P)$ — logarithmic decline instead of linear. Used for tumor growth (cells divide faster than they die when small; slowdown is faster than logistic near $K$).

Solution: $P(t) = K\exp!\left(-e^{-rt + c}\right)$ where $c = \ln\ln(K/P_0)$.

### 8.2 Allee Effect — Minimum Viable Population

$$\frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right)\left(\frac{P}{A} - 1\right), \quad 0 < A < K$$

Three equilibria: $0$ (stable), $A$ (unstable — the **Allee threshold**), $K$ (stable). If $P < A$, the population collapses to extinction. Used in conservation biology and — in ML — models the **loss barrier** phenomenon where optimization gets stuck near a saddle before finding the basin of attraction.

### 8.3 Harvesting Models

$$\frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right) - H$$

With constant harvesting $H$. If $H < rK/4$ (below the maximum growth rate), two equilibria exist; if $H > rK/4$, population collapses. The **maximum sustainable yield** is $H^* = rK/4$, achieved at $P = K/2$ — the inflection point again.

## 10. Summary — The Core Insight

|Question|Natural Growth|Logistic|
|---|---|---|
|What drives growth?|$P$ itself|$P$ and available "room" $(1 - P/K)$|
|What stops growth?|Nothing (in the model)|Carrying capacity $K$|
|What is the long run?|Infinity|$K$|
|When is it valid?|$P \ll K$, short time|All $P \in [0, K]$, all time|
|What does it miss?|Everything about limits|Allee effects, stochastic fluctuations|

> **The deepest insight:** the logistic model is the natural growth model **plus a feedback term** $-rP^2/K$ that represents competition. This feedback is what makes the model self-correcting and bounded. In ML, this is precisely the role of **regularization**: adding a term that penalizes large weights (analogous to large $P$), preventing the unbounded growth that unconstrained optimization would produce.
