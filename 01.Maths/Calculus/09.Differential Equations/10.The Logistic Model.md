## 1. Motivation вЂ” From Malthus to Verhulst

### 1.1 The Malthusian Model (Too Simple)

The simplest population model assumes growth proportional to current population:

$$\frac{dP}{dt} = kP, \quad k > 0$$

Solution: $P(t) = P_0 e^{kt}$ вЂ” unbounded exponential growth. This is biologically absurd for large $t$: no environment has infinite resources.

### 1.2 The Logistic Correction (Pierre FranГ§ois Verhulst, 1838)

Verhulst's insight: the **per-capita growth rate** $\frac{1}{P}\frac{dP}{dt}$ should not be constant вЂ” it should **decrease linearly** as $P$ approaches the **carrying capacity** $K$ of the environment:

$$\frac{1}{P}\frac{dP}{dt} = r\left(1 - \frac{P}{K}\right)$$

This gives the **logistic equation:**

$$\boxed{\frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right)} \tag{1}$$

**Parameters:**

- $r > 0$: intrinsic growth rate (units: $1/\text{time}$)
- $K > 0$: carrying capacity (units: population)
- $P_0 = P(0)$: initial population

**Qualitative behavior before solving:**

- If $P \ll K$: $1 - P/K \approx 1$, so $dP/dt \approx rP$ вЂ” exponential growth.
- If $P = K$: $dP/dt = 0$ вЂ” equilibrium.
- If $P > K$: $dP/dt < 0$ вЂ” population declines back toward $K$.

## 2. Solving the Logistic ODE вЂ” Full Derivation

Equation (1) is **separable**: $g(t) = r$, $h(P) = P(1 - P/K)$.

### Step 1 вЂ” Separate

$$\frac{dP}{P(1 - P/K)} = r, dt \tag{2}$$

### Step 2 вЂ” Partial Fractions

Decompose the left side. Write $1 - P/K = (K-P)/K$:

$$\frac{1}{P \cdot (K-P)/K} = \frac{K}{P(K-P)}$$

Partial fractions:

$$\frac{K}{P(K-P)} = \frac{A}{P} + \frac{B}{K-P}$$

$$K = A(K-P) + BP$$

- Set $P = 0$: $K = AK \implies A = 1$.
- Set $P = K$: $K = BK \implies B = 1$.

So:

$$\frac{K}{P(K-P)} = \frac{1}{P} + \frac{1}{K-P}$$

### Step 3 вЂ” Integrate Both Sides

$$\int \left(\frac{1}{P} + \frac{1}{K-P}\right) dP = \int r, dt$$

$$\ln|P| - \ln|K - P| = rt + C_1$$

$$\ln\left|\frac{P}{K-P}\right| = rt + C_1$$

$$\frac{P}{K-P} = Ae^{rt}, \quad A = e^{C_1} > 0 \tag{3}$$

### Step 4 вЂ” Solve for $P$

From (3): $P = Ae^{rt}(K - P)$:

$$P = KAe^{rt} - PAe^{rt}$$

$$P(1 + Ae^{rt}) = KAe^{rt}$$

$$P(t) = \frac{KAe^{rt}}{1 + Ae^{rt}} = \frac{K}{1 + A^{-1}e^{-rt}}$$

### Step 5 вЂ” Apply Initial Condition

At $t = 0$: $P(0) = P_0$:

$$P_0 = \frac{K}{1 + A^{-1}} \implies 1 + A^{-1} = \frac{K}{P_0} \implies A^{-1} = \frac{K - P_0}{P_0}$$

Let $B = A^{-1} = \frac{K - P_0}{P_0}$. The **general solution** is:

$$\boxed{P(t) = \frac{K}{1 + \left(\dfrac{K - P_0}{P_0}\right)e^{-rt}}} \tag{4}$$

### Step 6 вЂ” Singular Solutions

Setting $h(P) = 0$:

- $P \equiv 0$: trivially satisfies (1) (extinct population). Recovered from (4) as $P_0 = 0$.
- $P \equiv K$: also satisfies (1). Recovered from (4) as $P_0 = K$ (giving $B = 0$).

Both are **unstable/stable equilibria** respectively (see В§4).

### 3.2 The Inflection Point

Where is growth fastest? Differentiate (1):

$$\frac{d^2P}{dt^2} = r\frac{dP}{dt}\left(1 - \frac{P}{K}\right) + rP\left(-\frac{1}{K}\frac{dP}{dt}\right) = r\frac{dP}{dt}\left(1 - \frac{2P}{K}\right)$$

Setting $d^2P/dt^2 = 0$ (and $dP/dt \neq 0$):

$$1 - \frac{2P}{K} = 0 \implies \boxed{P^* = \frac{K}{2}}$$

The **inflection point** вЂ” where the population grows fastest вЂ” occurs exactly at **half the carrying capacity**. This is a structural feature independent of $r$ and $P_0$.

The time $t^_$ at which $P(t^_) = K/2$:

$$\frac{K}{2} = \frac{K}{1 + Be^{-rt^_}} \implies 1 + Be^{-rt^_} = 2 \implies e^{-rt^*} = \frac{1}{B}$$

$$t^* = \frac{1}{r}\ln B = \frac{1}{r}\ln\frac{K - P_0}{P_0}$$

### 3.3 Limiting Behavior

$$\lim_{t \to \infty} P(t) = K \qquad \text{(approaches carrying capacity)}$$ $$\lim_{t \to -\infty} P(t) = 0 \qquad \text{(approached zero in the past)}$$

## 4. Phase Line Analysis вЂ” Qualitative Theory

The logistic ODE $\dot{P} = rP(1 - P/K)$ is **autonomous**: the right side depends only on $P$, not $t$ explicitly.

### Equilibria

Set $\dot{P} = 0$: $$rP\left(1 - \frac{P}{K}\right) = 0 \implies P^* = 0 \quad \text{or} \quad P^* = K$$

### Stability via $\frac{d}{dP}[rP(1-P/K)]$

Let $f(P) = rP(1 - P/K)$. Then $f'(P) = r(1 - 2P/K)$.

- At $P^* = 0$: $f'(0) = r > 0$ в†’ **unstable equilibrium** (small perturbations grow).
- At $P^* = K$: $f'(K) = r(1 - 2) = -r < 0$ в†’ **stable equilibrium** (small perturbations decay).

## 5. The Logistic Function as a Special Case

Setting $K = 1$, $r = 1$, $P_0 = 1/2$ (so $B = 1$):

$$\sigma(t) = \frac{1}{1 + e^{-t}}$$

This is the **standard logistic / sigmoid function**. Its derivative:

$$\sigma'(t) = \sigma(t)(1 - \sigma(t))$$

This is **exactly the logistic ODE** evaluated at $K = 1$: $\dot{P} = P(1-P)$.

The sigmoid is not just a convenient S-shape вЂ” it is the **explicit solution to the logistic ODE**, evaluated at $K = 1$.

Its key analytic properties:

|Property|Value|
|---|---|
|$\sigma(0)$|$1/2$ (inflection point)|
|$\lim_{t\to\infty}\sigma(t)$|$1$|
|$\lim_{t\to-\infty}\sigma(t)$|$0$|
|$\sigma'(t)$|$\sigma(t)(1-\sigma(t)) \leq 1/4$|
|Maximum of $\sigma'$|$1/4$, attained at $t = 0$|
|$\sigma(-t)$|$1 - \sigma(t)$ (antisymmetry about $(0, 1/2)$)|

## 7. Variations and Extensions

### 7.1 Logistic with Harvesting

$$\frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right) - H$$

where $H > 0$ is a constant harvesting rate. The equilibria shift; for $H > rK/4$ (the maximum of $rP(1-P/K)$) there are **no equilibria** and the population collapses to zero in finite time. The threshold $H^* = rK/4$ is the **maximum sustainable yield**.

### 7.2 Gompertz Model

$$\frac{dP}{dt} = rP\ln\frac{K}{P}$$

Used in **tumor growth modeling** and sometimes ML (Gompertz activation). The per-capita rate $r\ln(K/P)$ decays logarithmically rather than linearly. More pessimistic than logistic for large $P$.

### 7.3 Allee Effect

$$\frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right)\left(\frac{P}{A} - 1\right), \quad 0 < A < K$$

Introduces a **minimum viable population** $A$: populations below $A$ decline to extinction. Three equilibria: $0$ (stable), $A$ (unstable), $K$ (stable).

## 8. Connection to AI, Machine Learning & Deep Learning

The logistic model is not background material for ML вЂ” it **is** ML, in several precise senses.

---

### 8.1 Logistic Regression вЂ” The Static Logistic Model

In binary classification, given features $\mathbf{x} \in \mathbb{R}^d$, logistic regression models:

$$P(y = 1 \mid \mathbf{x}) = \sigma(\mathbf{w}^\top \mathbf{x} + b) = \frac{1}{1 + e^{-(\mathbf{w}^\top \mathbf{x} + b)}}$$

The sigmoid here is the logistic solution at a fixed "time" $t = \mathbf{w}^\top\mathbf{x} + b$. The model is literally evaluating the logistic ODE solution on a linear projection of the input.

**Training** minimizes the **cross-entropy loss**:

$$\mathcal{L} = -\frac{1}{n}\sum_{i=1}^n \left[y_i \ln \hat{p}_i + (1-y_i)\ln(1-\hat{p}_i)\right]$$

The gradient:

$$\frac{\partial \mathcal{L}}{\partial \mathbf{w}} = \frac{1}{n}\sum_{i=1}^n (\hat{p}_i - y_i)\mathbf{x}_i$$

uses $\sigma'(z) = \sigma(z)(1-\sigma(z))$ вЂ” the logistic ODE identity вЂ” to simplify the chain rule derivative cleanly.

---

### 8.2 Sigmoid Activation вЂ” Every Neuron Solves a Logistic ODE

A sigmoid neuron computes $\sigma(z)$ where $z = \mathbf{w}^\top\mathbf{x} + b$. The value $\sigma(z)$ is the solution to $\dot{P} = P(1-P)$ evaluated at $t = z$.

The **vanishing gradient problem** in deep networks has an ODE interpretation: for large $|z|$, $\sigma'(z) = \sigma(z)(1-\sigma(z)) \to 0$ вЂ” the logistic curve flattens at both ends. Gradients flowing through many sigmoid layers multiply these near-zero derivatives, causing exponential decay of gradient signal (vanishing) or, in certain architectures, exponential growth (exploding).

This is why ReLU (which does not saturate) largely replaced sigmoid in hidden layers вЂ” it does not suffer from the saturation inherent in the logistic ODE's asymptotic approach to $0$ and $1$.

---

### 8.3 Softmax вЂ” The Multiclass Logistic Model

Softmax generalizes the sigmoid to $C$ classes:

$$\text{softmax}(\mathbf{z})_c = \frac{e^{z_c}}{\sum_{j=1}^C e^{z_j}}$$

For $C = 2$: $\text{softmax}(\mathbf{z})_1 = \frac{e^{z_1}}{e^{z_1} + e^{z_2}} = \frac{1}{1 + e^{-(z_1 - z_2)}} = \sigma(z_1 - z_2)$.

Softmax is the multivariate generalization of the logistic solution вЂ” the carrying capacity $K$ becomes a normalization constraint forcing probabilities to sum to 1.

---

### 8.4 Neural ODEs and the Logistic Vector Field

A **Neural ODE** (Chen et al., 2018) defines the hidden state dynamics as:

$$\frac{d\mathbf{h}}{dt} = f_\theta(\mathbf{h}(t), t)$$

The logistic ODE is the simplest nontrivial choice: $f_\theta(h) = rh(1-h)$ (component-wise, with $K=1$). The **residual network** (ResNet) layer:

$$\mathbf{h}_{t+1} = \mathbf{h}_t + f(\mathbf{h}_t)$$

is the **Euler discretization** of a Neural ODE with step size $\Delta t = 1$. As the number of layers $\to \infty$ and step size $\to 0$, the ResNet converges to a Neural ODE. The logistic model is the prototype for understanding what these continuous-depth networks are computing.

---

### 8.5 Population Dynamics of SGD вЂ” Loss Landscape as Carrying Capacity

There is a striking analogy between the logistic model and **stochastic gradient descent (SGD) convergence**:

|Logistic model|SGD convergence|
|---|---|
|Population $P(t)$|Model performance (e.g. accuracy)|
|Carrying capacity $K$|Optimal achievable accuracy|
|Growth rate $r$|Learning rate $\eta$|
|$1 - P/K$ (saturation)|Diminishing returns near optimum|
|Inflection at $K/2$|Fastest learning at intermediate performance|
|$t \to \infty: P \to K$|Training convergence|

Under certain loss landscape assumptions, the training accuracy $a(t)$ satisfies approximately:

$$\frac{da}{dt} \approx \eta \cdot a \cdot (1 - a)$$

вЂ” a logistic ODE with $K = 1$. The S-shaped learning curve (fast initial progress, slowing near perfect accuracy) is the logistic curve. This model motivates **learning rate warmup** (matching the low-growth phase near $P_0$) and **learning rate decay** (near saturation at $K$).

---

### 8.6 Diffusion Models вЂ” Logistic Noise Schedules

In **DDPM** (denoising diffusion probabilistic models), the noise schedule $\bar\alpha_t$ (fraction of signal remaining at timestep $t$) is often chosen as a **cosine schedule** which closely approximates the logistic curve run in reverse:

$$\bar\alpha_t = \frac{\sigma(s)}{\sigma(0)} \cdot \frac{\sigma(1 - t/T + s)}{\sigma(s)}, \quad s = 0.008$$

The logistic shape ensures the signal-to-noise ratio degrades smoothly вЂ” neither too fast (poor sample quality) nor too slow (poor training) вЂ” mirroring the inflection point structure of the logistic ODE.

---

### 8.7 Grokking вЂ” Delayed Generalization as Logistic Dynamics

**Grokking** (Power et al., 2022) refers to the phenomenon where a neural network first memorizes training data (loss on train $\to 0$), then вЂ” after a long delay вЂ” suddenly generalizes (test loss $\to 0$).

The generalization dynamics follow a near-perfect logistic curve: a long near-zero plateau, then rapid S-shaped improvement, then saturation at high accuracy. The "carrying capacity" is set by the model's capacity and the task's algorithmic structure. The logistic ODE is the minimal model that captures this behavior.

## 9. The Logistic Map вЂ” Discrete Chaos

The **discrete-time** logistic model is:

$$P_{n+1} = rP_n\left(1 - P_n\right), \quad r \in (0, 4]$$

(Euler discretization of $\dot{P} = rP(1-P)$ with step size 1, $K=1$.)

Its dynamics are radically different from the continuous ODE:

|$r$ range|Behavior|
|---|---|
|$0 < r < 1$|$P_n \to 0$|
|$1 < r < 3$|$P_n \to$ fixed point $1 - 1/r$|
|$3 < r < 3.57$|Period doubling (2, 4, 8, 16, ...)|
|$r > 3.57$|**Chaos**|
|$r = 4$|Full chaos on $[0,1]$|

The logistic map is the canonical example of **deterministic chaos** вЂ” a simple, deterministic, one-dimensional rule generating unpredictable trajectories. It demonstrates how discretization (Euler with too large a step) can destroy the stability guaranteed by the continuous ODE.

> In ML, this is a warning: discretizing a Neural ODE with too large a time step can destabilize training, turning smooth convergent dynamics into chaotic parameter oscillations.

## 11. Common Pitfalls

1. **Confusing $r$ and $K$.** $r$ sets the speed of growth; $K$ sets the ceiling. They are independent parameters with different units ($r$: time$^{-1}$; $K$: population).
2. **Forgetting the $P > K$ case.** If $P_0 > K$, the solution still exists and decreases monotonically to $K$. The formula (4) handles this: $B < 0$, but the solution remains valid and positive.
3. **Assuming the inflection is at $t = 0$.** The inflection is at $P = K/2$, which occurs at $t^* = \frac{1}{r}\ln\frac{K-P_0}{P_0}$. Only when $P_0 = K/2$ is $t^* = 0$.
4. **Linearizing too aggressively.** Near $P = 0$, $\dot{P} \approx rP$ (exponential). Near $P = K$, letting $u = K - P$ gives $\dot{u} = -ru + O(u^2)$ (exponential decay). The global S-shape is not captured by either local linearization alone.
5. **Applying the logistic to non-bounded phenomena.** The model assumes a fixed carrying capacity $K$. If $K$ itself changes over time (e.g., technology shifts the food supply), you need the **non-autonomous logistic** $\dot{P} = r(t)P(1 - P/K(t))$, which is generally not separable.
