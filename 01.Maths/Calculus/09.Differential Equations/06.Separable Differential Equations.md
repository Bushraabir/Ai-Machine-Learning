## 1. What Is a Separable Equation?

A first-order ODE of the form

$$\frac{dy}{dx} = f(x, y)$$

is called **separable** if $f$ factors as a product of a function of $x$ alone and a function of $y$ alone:

$$\frac{dy}{dx} = g(x) \cdot h(y)$$

The key insight: we can _algebraically separate_ all $y$-dependence to one side and all $x$-dependence to the other, then integrate both sides independently.

> **Ref:** Stewart §9.3 — "Separable Equations"; MIT 18.02 Lecture Notes on ODEs.

---

## 2. The Rigorous Setup

### 2.1 Formal Statement

Let $g : I \to \mathbb{R}$ and $h : J \to \mathbb{R}$ be continuous on open intervals $I, J \subseteq \mathbb{R}$, with $h(y) \neq 0$ on $J$. Consider

$$\frac{dy}{dx} = g(x), h(y). \tag{1}$$

**Claim:** Any solution $y = \phi(x)$ satisfying $\phi(x_0) = y_0$ (for some $(x_0, y_0) \in I \times J$) is implicitly defined by

$$H(y) = G(x) + C, \tag{2}$$

where $H(y) = \displaystyle\int \frac{dy}{h(y)}$ and $G(x) = \displaystyle\int g(x), dx$.

---

### 2.2 The Method (Leibniz Notation — Rigorous Justification)

We **formally** write:

$$\frac{1}{h(y)}, dy = g(x), dx$$

This notation is shorthand for the following rigorous argument:

**Proof.**  
Suppose $y = \phi(x)$ is a differentiable solution of (1) on some interval. Define

$$F(x) = H(\phi(x)) - G(x),$$

where $H' = \frac{1}{h}$ and $G' = g$. By the chain rule:

$$F'(x) = H'(\phi(x)) \cdot \phi'(x) - G'(x) = \frac{\phi'(x)}{h(\phi(x))} - g(x).$$

Since $\phi$ solves (1), $\phi'(x) = g(x), h(\phi(x))$, so:

$$F'(x) = \frac{g(x), h(\phi(x))}{h(\phi(x))} - g(x) = g(x) - g(x) = 0.$$

Therefore $F(x) = C$ for some constant $C$, i.e., $H(\phi(x)) = G(x) + C$. $\blacksquare$

**Existence & uniqueness** of the solution is then guaranteed by the **Picard–Lindelöf theorem** (provided $g$ and $h$ satisfy a Lipschitz condition), which tells us this implicit equation does define a unique local solution through any point $(x_0, y_0)$.

---

## 3. The Method — Step by Step

Given $\dfrac{dy}{dx} = g(x), h(y)$:

1. **Separate:** Write $\dfrac{dy}{h(y)} = g(x), dx$ (valid when $h(y) \neq 0$).
2. **Integrate both sides:** $\displaystyle\int \frac{dy}{h(y)} = \int g(x), dx + C$.
3. **Solve for $y$** (explicitly if possible, implicitly if not).
4. **Check singular solutions** where $h(y) = 0$ — these are constant solutions $y = y_0$ with $h(y_0) = 0$, and they may or may not be captured by the general solution at specific values of $C$.

## 4. Examples

### Example 1 — Exponential Growth/Decay

$$\frac{dy}{dx} = ky, \quad k \in \mathbb{R}$$

**Separate** (assuming $y \neq 0$):

$$\frac{dy}{y} = k, dx$$

**Integrate:**

$$\ln|y| = kx + C_1 \implies |y| = e^{C_1} e^{kx} \implies y = A e^{kx}, \quad A \in \mathbb{R} \setminus {0}.$$

**Check $y = 0$:** Substituting back, $0 = k \cdot 0$ ✓. So $y = 0$ is also a solution, recovered by $A = 0$.

**General solution:**

$$\boxed{y = Ae^{kx}, \quad A \in \mathbb{R}}$$

> This is the backbone of **radioactive decay** ($k < 0$), **population growth** ($k > 0$), and — critically — **gradient descent convergence analysis** in ML.

---

### Example 2 — Logistic Equation

$$\frac{dy}{dx} = ry\left(1 - \frac{y}{K}\right), \quad r, K > 0$$

This is separable: $g(x) = r$, $h(y) = y(1 - y/K)$.

**Separate:**

$$\frac{dy}{y(1 - y/K)} = r, dx$$

**Partial fractions** on the left ($\frac{K}{y(K-y)} = \frac{1}{y} + \frac{1}{K-y}$):

$$\int \left(\frac{1}{y} + \frac{1}{K - y}\right) dy = \int r, dx$$

$$\ln|y| - \ln|K - y| = rx + C_1$$

$$\ln\left|\frac{y}{K - y}\right| = rx + C_1 \implies \frac{y}{K - y} = Ae^{rx}$$

Solving for $y$:

$$\boxed{y(x) = \frac{K}{1 + Be^{-rx}}, \quad B = \frac{K - y_0}{y_0}}$$

where $y(0) = y_0$.

> The **sigmoid / logistic function** $\sigma(x) = \frac{1}{1 + e^{-x}}$ is the special case $K = 1, r = 1, B = 1$. It is literally the solution to a separable ODE, and it is the activation function used in logistic regression and as a neuron model in neural networks.

---

### Example 3 — Separable but Implicit Solution

$$\frac{dy}{dx} = \frac{x^2}{1 - y^2}$$

**Separate:**

$$(1 - y^2), dy = x^2, dx$$

**Integrate:**

$$y - \frac{y^3}{3} = \frac{x^3}{3} + C$$

This **cannot** be solved explicitly for $y$ in general — the solution curve is defined implicitly. This is perfectly fine; the implicit relation still defines $y$ as a function of $x$ near any point where the implicit function theorem applies (i.e., where $\partial/\partial y,(y - y^3/3 - x^3/3 - C) = 1 - y^2 \neq 0$).

---

### Example 4 — Singular Solutions

$$\frac{dy}{dx} = y^{1/3}$$

**Separate** ($y \neq 0$):

$$y^{-1/3}, dy = dx \implies \frac{3}{2} y^{2/3} = x + C \implies y = \left(\frac{2(x+C)}{3}\right)^{3/2}$$

**Singular solution:** $y \equiv 0$ satisfies the ODE (since $0^{1/3} = 0$), but it is **not** recovered from the general solution for any finite $C$.

> Note: $h(y) = y^{1/3}$ is not Lipschitz at $y = 0$, so Picard–Lindelöf does not guarantee uniqueness there — consistent with having two solutions through $(x_0, 0)$.

### Slope Field Intuition

For a separable equation $\frac{dy}{dx} = g(x)h(y)$, the slope at $(x, y)$ factors. This means:

- Along any horizontal line $y = c$: the slope is $g(x) \cdot h(c)$, scaling only with $x$.
- Along any vertical line $x = a$: the slope is $g(a) \cdot h(y)$, scaling only with $y$.

The level curves of $H(y) - G(x) = C$ are the **solution curves**.

## 6. Connection to AI, Machine Learning & Deep Learning

This is where separable equations stop being "just calculus" and become **the mathematical substrate of modern AI.**

---

### 6.1 The Sigmoid is the Solution to a Separable ODE

As shown in Example 2, the logistic equation

$$\frac{d\sigma}{dx} = r\sigma(1 - \sigma)$$

has the solution $\sigma(x) = \frac{1}{1 + e^{-rx}}$.

In neural networks, $\sigma$ is used as an **activation function** — every forward pass through a sigmoid neuron is evaluating the closed-form solution to a separable ODE.

---

### 6.2 Exponential Learning Rate Decay

In training neural networks, a common scheduler decays the learning rate $\eta(t)$ as:

$$\frac{d\eta}{dt} = -k\eta \implies \eta(t) = \eta_0 e^{-kt}$$

This is the exponential decay ODE (Example 1) — a separable equation. The choice of $k$ directly controls **convergence speed vs. stability** of gradient descent.

---

### 6.3 Neural ODEs — Continuous-Depth Networks

> _"What if a neural network were an ODE?"_  
> — Chen et al., NeurIPS 2018

**Neural ODEs** replace the discrete layer update

$$\mathbf{h}_{t+1} = \mathbf{h}_t + f(\mathbf{h}_t, \theta)$$

with a continuous-time ODE:

$$\frac{d\mathbf{h}}{dt} = f(\mathbf{h}(t), t, \theta)$$

where $\mathbf{h}(t) \in \mathbb{R}^n$ is the hidden state and $f$ is a neural network parameterized by $\theta$.

When $f(\mathbf{h}, t) = g(t) \cdot \tilde{f}(\mathbf{h})$, this ODE is **separable**, and the solution structure we've studied applies directly. Training such a model involves solving an ODE at each forward pass (via Runge-Kutta solvers) and computing gradients via the **adjoint method** — a continuous analogue of backpropagation rooted in the theory of ODEs.

---

### 6.4 Diffusion Models (Score-Based Generative Models)

Modern image generators (Stable Diffusion, DALL-E) are built on **stochastic differential equations (SDEs)** of the form

$$d\mathbf{x} = f(\mathbf{x}, t), dt + g(t), d\mathbf{W}$$

The **probability flow ODE** (the deterministic skeleton of this SDE) is:

$$\frac{d\mathbf{x}}{dt} = f(\mathbf{x}, t) - \frac{1}{2} g(t)^2 \nabla_\mathbf{x} \log p_t(\mathbf{x})$$

When the score function $\nabla_\mathbf{x} \log p_t(\mathbf{x})$ depends only on $t$ (not $\mathbf{x}$), this collapses to a **separable** ODE. Understanding separability is the first step toward understanding why these models can run both forward (adding noise) and backward (denoising = generating) in time.

---

### 6.5 Gradient Flow and Loss Dynamics

In continuous-time gradient descent (gradient flow), the parameters $\theta(t)$ evolve as:

$$\frac{d\theta}{dt} = -\nabla_\theta \mathcal{L}(\theta)$$

For certain simple loss landscapes (e.g., quadratic loss $\mathcal{L}(\theta) = \frac{1}{2}\lambda \theta^2$), this is separable:

$$\frac{d\theta}{dt} = -\lambda\theta \implies \theta(t) = \theta_0 e^{-\lambda t}$$

The **convergence rate** to the minimum is controlled by the eigenvalues $\lambda$ of the Hessian of $\mathcal{L}$ — a fact directly read off from the separable ODE solution.

## 8. Common Pitfalls

1. **Dividing by $h(y) = 0$:** Always check for constant solutions (singular solutions) separately.
2. **Losing the $\pm$ in $\ln|y|$:** The absolute value matters — it's why $A$ can be any real number.
3. **Forgetting $+C$ on one side:** The integration constant appears once; don't write it on both sides.
4. **Assuming an explicit solution always exists:** Many separable equations yield only implicit solutions (Example 3). That's not failure — it's the nature of the problem.
5. **Checking the domain of validity:** The solution $y(x)$ may only be valid on a subinterval of $\mathbb{R}$, determined by where the implicit function theorem applies.
