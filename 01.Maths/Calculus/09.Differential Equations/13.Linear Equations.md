## 0. Preliminary Remarks

This note is organized the way I actually think about linear equations — not as a drill on slope-intercept form, but as the foundational skeleton of analysis, geometry, and learning theory. Every deep idea in applied mathematics (optimization, neural networks, least squares) eventually reduces to asking: _when does a linear system have a solution, and how do we find it?_

We begin with the simplest case (one variable, one equation) and build upward with full rigor.

---

## 1. Definition: Linear Equation in One Variable

> **Definition 1.1.** A _linear equation in one variable_ $x$ is any equation of the form $$ax = b, \quad a, b \in \mathbb{R},\ a \neq 0.$$

The qualifier $a \neq 0$ is not cosmetic — it is the exact condition that guarantees existence and **uniqueness** of a solution.

**Proof of Existence and Uniqueness.** Suppose $ax = b$ with $a \neq 0$. Since $a \in \mathbb{R}^{\times}$ (the multiplicative group of nonzero reals), its multiplicative inverse $a^{-1}$ exists. Multiply both sides on the left by $a^{-1}$: $$a^{-1}(ax) = a^{-1}b \implies (a^{-1}a)x = a^{-1}b \implies 1 \cdot x = \frac{b}{a} \implies x = \frac{b}{a}.$$ _Existence:_ $x_0 = b/a$ is a solution.  
_Uniqueness:_ Suppose $x_1$ is another solution, so $ax_1 = b = ax_0$. Multiply by $a^{-1}$: $x_1 = x_0$. $\blacksquare$

> **Remark.** If $a = 0$: the equation becomes $0 = b$. If $b \neq 0$, there is _no_ solution. If $b = 0$, _every_ $x \in \mathbb{R}$ is a solution. The trichotomy {no solution, unique solution, infinitely many solutions} already appears here and persists into systems of any dimension.

## 2. Linear Equations in Two Variables

> **Definition 2.1.** A _linear equation in two variables_ $x, y$ is any equation of the form $$ax + by = c, \quad (a, b) \neq (0, 0),\ a, b, c \in \mathbb{R}.$$

The solution set is a **line** in $\mathbb{R}^2$.

**Why is it a line?** The equation defines a level set of the linear functional $L(x,y) = ax + by$. Level sets of linear maps on $\mathbb{R}^2$ are affine subspaces of dimension 1 — i.e., lines. (We'll return to this language in §5.)

### 2.1 Standard Forms

|Form|Equation|When to use|
|---|---|---|
|**Standard**|$ax + by = c$|Systems, dot-product geometry|
|**Slope-intercept**|$y = mx + k$|Reading off slope/intercept|
|**Point-slope**|$y - y_0 = m(x - x_0)$|Building from a point + slope|
|**Parametric**|$(x,y) = (x_0, y_0) + t(d_1, d_2)$|Vectors, direction, ML feature paths|

**Conversion Example.** Given $3x - 2y = 6$:

- Slope-intercept: $y = \tfrac{3}{2}x - 3$, so slope $m = 3/2$, $y$-intercept $= -3$.
- Parametric: take $t=x$, then $(x, y) = t(1, 3/2) + (0, -3)$.

### 2.2 Geometric Intuition

```
        y
        |          / slope = 3/2
    1   |         /
        |        /  3x - 2y = 6
   -3 --+-------/---------- x
        |      / (0,-3)
        |     /
        |    * (2,0)  ← x-intercept
```

The line $3x - 2y = 6$ crosses:

- $x$-axis at $(2, 0)$: set $y=0 \Rightarrow x = 2$
- $y$-axis at $(0, -3)$: set $x=0 \Rightarrow y = -3$

### 2.3 The Normal Vector Perspective

The equation $ax + by = c$ can be written as the **dot product**: $$\mathbf{n} \cdot \mathbf{x} = c, \quad \mathbf{n} = \begin{pmatrix} a \ b \end{pmatrix},\ \mathbf{x} = \begin{pmatrix} x \ y \end{pmatrix}.$$

The vector $\mathbf{n}$ is _perpendicular_ to the line. This is not a coincidence — it is the geometric definition of a hyperplane.

> **Lemma 2.2.** $\mathbf{n}$ is orthogonal to every direction vector of the line $ax + by = c$.

**Proof.** A direction vector $\mathbf{d} = (d_1, d_2)$ satisfies $\mathbf{n} \cdot (\mathbf{x}_0 + t\mathbf{d}) = c$ for all $t$. Expanding: $\mathbf{n} \cdot \mathbf{x}_0 + t(\mathbf{n} \cdot \mathbf{d}) = c$. Since $\mathbf{n} \cdot \mathbf{x}_0 = c$, we get $t(\mathbf{n} \cdot \mathbf{d}) = 0$ for all $t$, hence $\mathbf{n} \cdot \mathbf{d} = 0$. $\blacksquare$

## 3. Systems of Linear Equations

> **Definition 3.1.** A _system of $m$ linear equations in $n$ unknowns_ is a collection $$\sum_{j=1}^{n} a_{ij} x_j = b_i, \quad i = 1, \ldots, m.$$ In matrix form: $A\mathbf{x} = \mathbf{b}$, where $A \in \mathbb{R}^{m \times n}$, $\mathbf{x} \in \mathbb{R}^n$, $\mathbf{b} \in \mathbb{R}^m$.

### 3.1 The Fundamental Trichotomy (Theorem)

> **Theorem 3.2** _(Existence and Uniqueness for Linear Systems)._ For the system $A\mathbf{x} = \mathbf{b}$, exactly one of the following holds:
> 
> 1. **No solution** — the system is _inconsistent_.
> 2. **Exactly one solution** — the system is _consistent_ and the solution is _unique_.
> 3. **Infinitely many solutions** — the system is _consistent_ with a free-parameter family.

**Proof sketch.** By Gaussian elimination, $A$ row-reduces to a row echelon form $R$. The system $R\mathbf{x} = \mathbf{c}$ (transformed RHS) is inconsistent iff a row of the form $[0\ 0\ \cdots\ 0\ |\ c]$ with $c \neq 0$ appears (case 1). Otherwise: if every column of $R$ has a pivot, the solution is unique (case 2); if there are free variables (columns without pivots), infinitely many solutions exist (case 3). $\blacksquare$

### 3.2 $2 \times 2$ Case — Geometric Picture

Two lines in $\mathbb{R}^2$:

- **Intersect at one point** → unique solution
- **Parallel (distinct)** → no solution
- **Identical** → infinitely many solutions

```
Case 1: Unique          Case 2: No Solution      Case 3: Infinite
  y  /  /                  y  //                   y  
    X ← solution             ||  ← parallel           same line
   / /                      ||
```

### 3.3 Worked Example

Solve: $$\begin{cases} 2x + y = 5 \ x - y = 1 \end{cases}$$

**Elimination method:**  
Add the equations: $3x = 6 \Rightarrow x = 2$.  
Substitute: $2 - y = 1 \Rightarrow y = 1$.  
Solution: $(x, y) = (2, 1)$.

**Matrix form:** $$\begin{pmatrix} 2 & 1 \ 1 & -1 \end{pmatrix} \begin{pmatrix} x \ y \end{pmatrix} = \begin{pmatrix} 5 \ 1 \end{pmatrix}$$

$\det(A) = 2(-1) - 1(1) = -3 \neq 0$, so the unique solution exists. By Cramer's rule: $$x = \frac{\det\begin{pmatrix}5&1\1&-1\end{pmatrix}}{-3} = \frac{-6}{-3} = 2, \quad y = \frac{\det\begin{pmatrix}2&5\1&1\end{pmatrix}}{-3} = \frac{-3}{-3} = 1.$$

## 4. Linear Functions vs. Affine Functions

This distinction matters enormously in machine learning.

> **Definition 4.1.** A function $f: \mathbb{R}^n \to \mathbb{R}$ is _linear_ if:
> 
> 1. $f(\mathbf{u} + \mathbf{v}) = f(\mathbf{u}) + f(\mathbf{v})$ for all $\mathbf{u}, \mathbf{v}$
> 2. $f(c\mathbf{u}) = c f(\mathbf{u})$ for all $c \in \mathbb{R}$

> **Definition 4.2.** A function $f: \mathbb{R}^n \to \mathbb{R}$ is _affine_ if $f(\mathbf{x}) = \mathbf{w}^T \mathbf{x} + b$ for some $\mathbf{w} \in \mathbb{R}^n$, $b \in \mathbb{R}$.

> **Remark.** Every linear function is affine (with $b=0$), but not vice versa. The equation $y = 3x + 5$ defines an **affine** function, **not** a linear one in the strict algebraic sense, since $f(0) = 5 \neq 0$. In casual usage (and in Stewart), "linear equation" allows a nonzero constant term. In linear algebra, we must be precise.

## 5. Hyperplanes: Generalization to $\mathbb{R}^n$

> **Definition 5.1.** A _hyperplane_ in $\mathbb{R}^n$ is the solution set of a single linear equation: $$H = {\mathbf{x} \in \mathbb{R}^n : \mathbf{w}^T \mathbf{x} = b}.$$

Dimension of $H$ as an affine subspace: $n - 1$.

|$n$|Hyperplane|
|---|---|
|1|Point|
|2|Line|
|3|Plane|
|$n$|$(n-1)$-dimensional flat|

The vector $\mathbf{w}$ is the **normal** to $H$. The scalar $b$ shifts $H$ away from the origin.

## 6. Connection to AI / Machine Learning / Deep Learning

> This is where the theory stops being "abstract" and starts being the engine underneath every model you will ever train.

### 6.1 Linear Regression

**Setup.** Given data ${(\mathbf{x}_i, y_i)}_{i=1}^N$ with $\mathbf{x}_i \in \mathbb{R}^n$, fit: $$\hat{y} = \mathbf{w}^T \mathbf{x} + b.$$

The optimal parameters solve the **normal equations** — a linear system: $$A^T A \mathbf{w} = A^T \mathbf{y}, \quad A = \begin{pmatrix} \mathbf{x}_1^T \ \vdots \ \mathbf{x}_N^T \end{pmatrix}.$$

This is _literally_ solving a linear equation. The entire field of least squares is built on the existence/uniqueness theorem for $A\mathbf{x} = \mathbf{b}$.

**Geometric insight:** $\hat{\mathbf{y}} = A\mathbf{w}$ is the **orthogonal projection** of $\mathbf{y}$ onto the column space of $A$. The residual $\mathbf{y} - \hat{\mathbf{y}}$ is orthogonal to every column of $A$ — this is exactly the normal equation $A^T(\mathbf{y} - A\mathbf{w}) = \mathbf{0}$.

### 6.2 Perceptron and Support Vector Machines

A **perceptron** classifies input $\mathbf{x}$ as: $$\hat{y} = \text{sign}(\mathbf{w}^T \mathbf{x} + b).$$

The decision boundary $\mathbf{w}^T \mathbf{x} + b = 0$ is exactly a **hyperplane** (Definition 5.1). The weights $\mathbf{w}$ are the normal vector; $b$ is the bias that shifts it.

An **SVM** finds the hyperplane that maximizes the margin between two classes — a quadratic program whose solution is determined by solving a linear system at its dual.

### 6.3 Neural Networks: Linear Layers

Every **fully connected (dense) layer** computes: $$\mathbf{z} = W\mathbf{x} + \mathbf{b}, \quad W \in \mathbb{R}^{m \times n},\ \mathbf{x} \in \mathbb{R}^n,\ \mathbf{b} \in \mathbb{R}^m.$$

This is a **system of $m$ linear equations** applied simultaneously. Without **nonlinear activations** (ReLU, sigmoid, etc.), a composition of linear layers collapses to a single linear map: $$W_L \cdots W_2 W_1 \mathbf{x} = W_{\text{eff}} \mathbf{x}.$$

A 100-layer fully linear network has identical expressive power to a single linear layer. **Depth only helps because of nonlinearity.** Nonlinearity is the entire point of deep learning, and linear algebra is the scaffolding it runs on.

### 6.4 Gradient Descent and Linear Systems

Training a neural network minimizes a loss $\mathcal{L}(\mathbf{w})$. At convergence, $\nabla_\mathbf{w} \mathcal{L} = \mathbf{0}$ — a (generally nonlinear) system of equations. For a **quadratic loss** (as in linear regression), this reduces to: $$\nabla \mathcal{L} = 2A^T(A\mathbf{w} - \mathbf{y}) = \mathbf{0} \iff A^T A \mathbf{w} = A^T \mathbf{y}.$$

A linear system again.

**Gradient descent** iterates $\mathbf{w}_{k+1} = \mathbf{w}_k - \eta \nabla \mathcal{L}(\mathbf{w}_k)$. Its convergence rate is governed by the **condition number** $\kappa(A^T A) = \lambda_{\max}/\lambda_{\min}$ — a quantity from the spectral theory of linear operators.


## 7. Worked Examples

### Example 7.1 — Unique Solution

$$3x - 2y = 4,\quad x + y = 3.$$ From the second: $x = 3 - y$. Substitute: $3(3-y) - 2y = 4 \Rightarrow 9 - 5y = 4 \Rightarrow y = 1,\ x = 2$.

### Example 7.2 — No Solution

$$2x + 4y = 6,\quad x + 2y = 4.$$ Divide first by 2: $x + 2y = 3$. But second says $x + 2y = 4$. Contradiction. $\emptyset$.

### Example 7.3 — Infinitely Many Solutions

$$2x + 4y = 6,\quad x + 2y = 3.$$ Both reduce to $x + 2y = 3$. Free variable $y = t \in \mathbb{R}$: solution set ${(3 - 2t,\ t) : t \in \mathbb{R}}$.

### Example 7.4 — ML Context

Suppose two data points: $(x_1, y_1) = (1, 2)$ and $(x_2, y_2) = (3, 4)$. Fit $y = wx + b$: $$\begin{pmatrix}1 & 1 \ 3 & 1\end{pmatrix}\begin{pmatrix}w \ b\end{pmatrix} = \begin{pmatrix}2 \ 4\end{pmatrix}.$$ $\det = 1\cdot1 - 3\cdot1 = -2 \neq 0$. Unique solution: $w = 1,\ b = 1$. Model: $\hat{y} = x + 1$.

## 8. Key Theorems — Summary

|Theorem|Statement|
|---|---|
|**Existence & Uniqueness** (1 var)|$ax = b,\ a\neq 0 \Rightarrow$ unique solution $x = b/a$|
|**Fundamental Trichotomy**|Any linear system has 0, 1, or $\infty$ solutions|
|**Normal Vector Theorem**|$\mathbf{n} = (a,b)$ is $\perp$ to the line $ax + by = c$|
|**Linear Layer Collapse**|Composition of linear maps is linear; nonlinearity is essential for depth|
|**Normal Equations**|Least squares $\Leftrightarrow$ solving $A^TA\mathbf{w} = A^T\mathbf{y}$|

## 9. Things to Prove Yourself

1. Show that the solution set of $A\mathbf{x} = \mathbf{b}$ (when consistent) is a coset of the null space $\ker(A)$.
2. Verify that $\mathbf{n} = (a,b)$ is perpendicular to the line $ax + by = c$ using vectors. (Lemma 2.2 done in full detail.)
3. Prove that a composition of two affine maps $f(\mathbf{x}) = A\mathbf{x}+\mathbf{a}$ and $g(\mathbf{x}) = B\mathbf{x}+\mathbf{b}$ is again affine. What is its linear part?
4. Derive the normal equations $A^TA\mathbf{w} = A^T\mathbf{y}$ by minimizing $|A\mathbf{w} - \mathbf{y}|^2$ via calculus.
5. Show that a network of $L$ linear layers (no activations) is equivalent to a single linear transformation.
