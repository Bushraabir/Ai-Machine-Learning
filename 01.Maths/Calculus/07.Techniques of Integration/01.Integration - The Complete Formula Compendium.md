## §0 · The Fundamental Theorem of Calculus

Before any formula, the engine:

**FTC Part I.** If $f$ is continuous on $[a,b]$ and $F(x) = \int_a^x f(t),dt$, then $F'(x) = f(x)$.

**FTC Part II.** If $F$ is any antiderivative of $f$ on $[a,b]$, then

$$\int_a^b f(x),dx = F(b) - F(a)$$

> _Why it matters:_ every single formula below is just "find $F$ such that $F' = f$." The FTC is what makes the list below _useful_.

## §3 · Trigonometric Forms

### 3a · The Six Basic Trig Integrals

|#|Integrand|Antiderivative|
|---|---|---|
|12|$\sin x$|$-\cos x + C$|
|13|$\cos x$|$\sin x + C$|
|14|$\tan x$|$-\ln\|\cos x\| + C = \ln\|\sec x\|+C$|
|15|$\cot x$|$\ln\|\sin x\| + C$|
|16|$\sec x$|$\ln\|\sec x + \tan x\| + C$ ⚑|
|17|$\csc x$|$-\ln\|\csc x + \cot x\| + C = \ln\|\csc x - \cot x\| + C$ ⚑|

**Proof of #14:** $$\int \tan x,dx = \int \frac{\sin x}{\cos x},dx$$ Let $u = \cos x$, $du = -\sin x,dx$: $$= -\int \frac{du}{u} = -\ln|u| + C = -\ln|\cos x| + C$$

**Proof of #16 ⚑ (multiply by the conjugate trick):** $$\int \sec x,dx = \int \sec x \cdot \frac{\sec x + \tan x}{\sec x + \tan x},dx = \int \frac{\sec^2 x + \sec x \tan x}{\sec x + \tan x},dx$$ Let $u = \sec x + \tan x$, $du = (\sec x \tan x + \sec^2 x),dx$: $$= \int \frac{du}{u} = \ln|u| + C = \ln|\sec x + \tan x| + C$$

### 3c · Reduction via Pythagorean Identities

For $\int \sin^2 x,dx$ and $\int \cos^2 x,dx$, use **half-angle identities**:

$$\sin^2 x = \frac{1 - \cos 2x}{2}, \qquad \cos^2 x = \frac{1 + \cos 2x}{2}$$

$$\int \sin^2 x,dx = \frac{x}{2} - \frac{\sin 2x}{4} + C$$

$$\int \cos^2 x,dx = \frac{x}{2} + \frac{\sin 2x}{4} + C$$

$$\int \tan^2 x,dx = \int(\sec^2 x - 1),dx = \tan x - x + C$$

**AI/ML Connection 🤖:** Fourier analysis — the mathematical foundation of **CNNs and signal processing in audio/speech models** — decomposes functions into $\sin$ and $\cos$ components. Computing Fourier coefficients requires: $$a_n = \frac{2}{T}\int_0^T f(t)\cos(2\pi nt/T),dt, \quad b_n = \frac{2}{T}\int_0^T f(t)\sin(2\pi nt/T),dt$$ These integrals are computed analytically via the trig formulas above, and numerically via FFT. The **positional encodings** in Transformer architectures (BERT, GPT) are literally $\sin(pos/10000^{2i/d})$ and $\cos(pos/10000^{2i/d})$.

## §6 · Integration by Substitution ($u$-sub)

**Theorem.** If $g$ is differentiable and $f$ is continuous: $$\int f(g(x)),g'(x),dx = \int f(u),du \quad \text{where } u = g(x)$$

This is the chain rule _reversed_.

**Recognition pattern:** look for a function and its derivative together.

**Examples:**

$$\int 2x,e^{x^2},dx \xrightarrow{u=x^2} \int e^u,du = e^{x^2} + C$$

$$\int \frac{x}{\sqrt{1-x^2}},dx \xrightarrow{u=1-x^2} -\frac{1}{2}\int u^{-1/2},du = -\sqrt{1-x^2} + C$$

$$\int_0^{\pi/2} \sin^3 x\cos x,dx \xrightarrow{u=\sin x} \int_0^1 u^3,du = \frac{1}{4}$$

**AI/ML Connection 🤖:** The **change of variables formula** in probability — $p_Y(y) = p_X(g^{-1}(y))\cdot|{(g^{-1})'(y)}|$ — is the probabilistic cousin of $u$-substitution. This underpins **normalizing flows** in generative modelling (RealNVP, Glow), where we learn invertible transformations and must track the Jacobian determinant — the multivariable version of $g'(x)$.

## §7 · Integration by Parts

**Theorem (reverse product rule):** $$\int u,dv = uv - \int v,du$$

**LIATE mnemonic** for choosing $u$:

> **L**ogarithm · **I**nverse trig · **A**lgebraic · **T**rigonometric · **E**xponential

**Canonical examples:**

$$\int x e^x,dx = xe^x - e^x + C = e^x(x-1)+C$$

$$\int x^2 e^x,dx = e^x(x^2 - 2x + 2) + C \quad \text{(apply IBP twice)}$$

$$\int x\ln x,dx = \frac{x^2}{2}\ln x - \frac{x^2}{4} + C$$

$$\int e^x\sin x,dx = \frac{e^x(\sin x - \cos x)}{2} + C \quad \text{(cyclic IBP)} $$

**Proof of cyclic IBP for $\int e^x \sin x,dx$ ⚑:** Let $I = \int e^x\sin x,dx$. IBP with $u=\sin x$, $dv=e^x dx$: $$I = e^x \sin x - \int e^x \cos x,dx$$ IBP again with $u=\cos x$, $dv=e^x dx$: $$I = e^x\sin x - \left[e^x\cos x + \int e^x\sin x,dx\right] = e^x\sin x - e^x\cos x - I$$ $$2I = e^x(\sin x - \cos x), \quad I = \frac{e^x(\sin x - \cos x)}{2} + C$$

**Reduction formulae (very useful):**

$$\int \sin^n x,dx = -\frac{\sin^{n-1}x\cos x}{n} + \frac{n-1}{n}\int \sin^{n-2}x,dx$$

$$\int \cos^n x,dx = \frac{\cos^{n-1}x\sin x}{n} + \frac{n-1}{n}\int \cos^{n-2}x,dx$$

$$\int x^n e^x,dx = x^n e^x - n\int x^{n-1}e^x,dx$$

**AI/ML Connection 🤖:** IBP is the engine behind **backpropagation** in the continuous limit. The **adjoint method** used in **Neural ODEs** (Chen et al., 2018) and **physics-informed neural networks (PINNs)** derives the gradient of a loss through an ODE solution using the integration-by-parts identity on the relevant operator. Also, computing **Fisher information** $\mathcal{I}(\theta) = -\mathbb{E}\left[\frac{\partial^2}{\partial\theta^2}\ln p(x;\theta)\right]$ requires IBP to relate second-order terms to first-order ones.

## §9 · Partial Fractions

**Setup:** For a rational function $P(x)/Q(x)$ with $\deg P < \deg Q$, factor $Q$ and decompose:

**Case 1 — Distinct linear factors:** $$\frac{1}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}$$

**Case 2 — Repeated linear factors:** $$\frac{P(x)}{(x-a)^n} = \frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \cdots + \frac{A_n}{(x-a)^n}$$

**Case 3 — Irreducible quadratic factors:** $$\frac{P(x)}{x^2+bx+c} = \frac{Ax+B}{x^2+bx+c}$$

**Worked example (complete):** $$\int \frac{2x+1}{x^2-x-2},dx = \int \frac{2x+1}{(x-2)(x+1)},dx$$ Partial fractions: $\frac{2x+1}{(x-2)(x+1)} = \frac{A}{x-2}+\frac{B}{x+1}$

Multiply through: $2x+1 = A(x+1)+B(x-2)$

Set $x=2$: $5 = 3A \Rightarrow A = 5/3$. Set $x=-1$: $-1 = -3B \Rightarrow B = 1/3$.

$$\int \frac{5/3}{x-2},dx + \int \frac{1/3}{x+1},dx = \frac{5}{3}\ln|x-2| + \frac{1}{3}\ln|x+1| + C$$

**AI/ML Connection 🤖:** Partial fractions appear in the **$z$-transform** and **Laplace transform**, used extensively in **signal processing for audio/speech models** and in analysing the stability of **recurrent neural networks (RNNs)**. The transfer function of an RNN or an LSTM filter bank is a rational function in $z$ or $s$, and partial fraction decomposition reveals the poles — i.e., the time constants of memory decay.

## §10 · Improper Integrals

**Type I (infinite limits):** $$\int_a^\infty f(x),dx = \lim_{t\to\infty}\int_a^t f(x),dx$$

**Type II (unbounded integrand):** $$\int_a^b f(x),dx = \lim_{t\to a^+}\int_t^b f(x),dx \quad \text{if } f \text{ unbounded near } a$$

### Key convergence tests:

**$p$-integral:** $$\int_1^\infty \frac{dx}{x^p} = \begin{cases} \frac{1}{p-1} & p > 1 \ \text{diverges} & p \leq 1 \end{cases}$$

$$\int_0^1 \frac{dx}{x^p} = \begin{cases} \frac{1}{1-p} & p < 1 \ \text{diverges} & p \geq 1 \end{cases}$$

**Gaussian integral ⚑ (MIT 18.02 — polar coordinates):** $$\int_{-\infty}^\infty e^{-x^2},dx = \sqrt{\pi}$$

_Proof sketch:_ Let $I = \int_{-\infty}^\infty e^{-x^2},dx$. Then: $$I^2 = \int_{-\infty}^\infty\int_{-\infty}^\infty e^{-(x^2+y^2)},dx,dy$$ Convert to polar $x=r\cos\theta$, $y=r\sin\theta$, $dx,dy = r,dr,d\theta$: $$I^2 = \int_0^{2\pi}\int_0^\infty e^{-r^2} r,dr,d\theta = 2\pi \cdot \frac{1}{2} = \pi$$ Therefore $I = \sqrt{\pi}$.

**Normal distribution normalisation:** $$\int_{-\infty}^\infty \frac{1}{\sigma\sqrt{2\pi}}e^{-(x-\mu)^2/(2\sigma^2)},dx = 1$$

This follows from the Gaussian integral above with substitution $t = (x-\mu)/(\sigma\sqrt{2})$.

**AI/ML Connection 🤖:** Improper integrals are _everywhere_ in ML:

- **Expected loss:** $\mathbb{E}[\mathcal{L}(\theta)] = \int \mathcal{L}(x,\theta) p(x),dx$ over all data $x \in \mathbb{R}^d$
- **Entropy:** $H(p) = -\int p(x)\ln p(x),dx$
- **ELBO in VAEs:** $\mathcal{L}(\phi,\theta) = \mathbb{E}_{q_\phi(z|x)}[\ln p_\theta(x|z)] - D_{KL}(q_\phi(z|x)|p(z))$ — both terms are improper integrals over $\mathbb{R}^d$
- **Score matching:** $\int |\nabla_x \ln p_\theta(x) - \nabla_x \ln p_{data}(x)|^2 p_{data}(x),dx$ — this is what **diffusion models** (DDPM, Score-based generative models) minimise

## §12 · Wallis's Formulas & Beta/Gamma Function Connections

**Wallis integrals:** $$W_n = \int_0^{\pi/2}\sin^n x,dx = \int_0^{\pi/2}\cos^n x,dx$$

$$W_{2n} = \frac{(2n-1)!!}{(2n)!!}\cdot\frac{\pi}{2}, \quad W_{2n+1} = \frac{(2n)!!}{(2n+1)!!}$$

where $n!! = n(n-2)(n-4)\cdots$

**Gamma function:** $$\Gamma(n) = \int_0^\infty x^{n-1}e^{-x},dx, \quad \Gamma(n+1) = n,\Gamma(n), \quad \Gamma(n) = (n-1)! \text{ for } n\in\mathbb{N}$$

$$\Gamma(1/2) = \sqrt{\pi}$$

**Beta function:** $$B(m,n) = \int_0^1 x^{m-1}(1-x)^{n-1},dx = \frac{\Gamma(m)\Gamma(n)}{\Gamma(m+n)}$$

**AI/ML Connection 🤖:** The **Beta and Gamma functions** are the theoretical foundation of **Bayesian conjugate priors**. The Beta distribution $\text{Beta}(\alpha,\beta) \propto x^{\alpha-1}(1-x)^{\beta-1}$ is the conjugate prior for the Bernoulli/Binomial — understanding why its normalisation constant is $B(\alpha,\beta)$ requires this integral. Similarly, the **Dirichlet distribution** (used in **LDA topic models** and as priors in **Bayesian deep learning**) generalises the Beta, with normalisation constant involving $\Gamma$ functions.

## §13 · Multivariable Integrals (MIT 18.02)

### Double Integrals

**Rectangular region:** $$\iint_R f(x,y),dA = \int_a^b\int_c^d f(x,y),dy,dx = \int_c^d\int_a^b f(x,y),dx,dy$$

(Fubini's theorem — interchange valid when $f$ continuous on closed rectangle.)

**General region (Type I):** $$\iint_D f,dA = \int_a^b\int_{g_1(x)}^{g_2(x)} f(x,y),dy,dx$$

**Polar coordinates:** $x = r\cos\theta$, $y = r\sin\theta$, $dA = r,dr,d\theta$ $$\iint_R f(x,y),dA = \int_\alpha^\beta\int_{r_1(\theta)}^{r_2(\theta)} f(r\cos\theta,r\sin\theta),r,dr,d\theta$$

### Triple Integrals

**Cylindrical:** $(r,\theta,z)$, $dV = r,dr,d\theta,dz$ $$\iiint f,dV = \int\int\int f(r\cos\theta,r\sin\theta,z),r,dr,d\theta,dz$$

**Spherical:** $(\rho,\theta,\phi)$, $dV = \rho^2\sin\phi,d\rho,d\theta,d\phi$ $$\iiint f,dV = \int\int\int f(\rho\sin\phi\cos\theta,\rho\sin\phi\sin\theta,\rho\cos\phi),\rho^2\sin\phi,d\rho,d\theta,d\phi$$

**Change of Variables (general):** For $T:(u,v)\mapsto(x,y)$: $$\iint_R f(x,y),dA = \iint_S f(x(u,v),y(u,v))\left|\frac{\partial(x,y)}{\partial(u,v)}\right|du,dv$$

where $\left|\frac{\partial(x,y)}{\partial(u,v)}\right| = \left|\det J\right|$ is the **Jacobian determinant**.

**AI/ML Connection 🤖 — Central to Deep Learning:**

- **Multivariable integration underpins all of probability in $\mathbb{R}^n$.** Every joint distribution $p(x_1,\ldots,x_n)$ over a dataset, every marginalisation $p(x_1) = \int \cdots \int p(x_1,\ldots,x_n),dx_2\cdots dx_n$ is a multiple integral.
- The **Jacobian** $\left|\det J\right|$ is exactly the term tracked in **normalizing flows** and **diffusion models**: when you transform data $x$ through a neural network $f_\theta$, the log-likelihood involves $\ln p(f^{-1}(z)) + \ln|\det J_{f^{-1}}|$.
- **Monte Carlo integration:** For high-dimensional integrals (like $\int_{\mathbb{R}^{1000}} p(x),dx$), deterministic quadrature is infeasible — hence **Monte Carlo estimation** $\mathbb{E}_p[f] \approx \frac{1}{N}\sum f(x_i)$, which is how all deep learning expectations are computed.
- **Attention mechanism (Transformers):** $\text{Attention}(Q,K,V) = \text{softmax}(QK^T/\sqrt{d_k})V$ can be interpreted as a discretisation of an integral operator $\int K(x,y) v(y),dy$ — the **kernel integral** of $v$ weighted by attention scores.

## §14 · Line Integrals (MIT 18.02)

**Scalar line integral:** $$\int_C f(x,y),ds = \int_a^b f(\mathbf{r}(t)),|\mathbf{r}'(t)|,dt$$

**Vector line integral (work integral):** $$\int_C \mathbf{F}\cdot d\mathbf{r} = \int_a^b \mathbf{F}(\mathbf{r}(t))\cdot\mathbf{r}'(t),dt = \int_C P,dx + Q,dy$$

**Green's Theorem:** $$\oint_C P,dx + Q,dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)dA$$

**AI/ML Connection 🤖:** The **gradient flow** interpretation of neural network training uses line integrals. The total loss change along a training trajectory $\theta(t)$ from $\theta_0$ to $\theta^*$ is $\int_0^T \nabla_\theta\mathcal{L}\cdot\dot{\theta},dt$ — a line integral along the optimisation path. This is why **path-dependent** phenomena like gradient vanishing/exploding in RNNs are properly understood using these tools. **Stokes' theorem** (the 3D generalisation of Green's) appears in theoretical analyses of **diffusion processes** underlying score-based generative models.

## §15 · Surface Integrals (MIT 18.02)

**Surface area element:** $$d\mathbf{S} = \mathbf{n},dS, \quad dS = \left|\frac{\partial\mathbf{r}}{\partial u}\times\frac{\partial\mathbf{r}}{\partial v}\right|du,dv$$

**Flux integral:** $$\iint_S \mathbf{F}\cdot d\mathbf{S} = \iint_D \mathbf{F}(\mathbf{r}(u,v))\cdot\left(\frac{\partial\mathbf{r}}{\partial u}\times\frac{\partial\mathbf{r}}{\partial v}\right)du,dv$$

**Divergence Theorem (Gauss):** $$\oiint_S \mathbf{F}\cdot d\mathbf{S} = \iiint_V (\nabla\cdot\mathbf{F}),dV$$

**Stokes' Theorem:** $$\oint_{\partial S} \mathbf{F}\cdot d\mathbf{r} = \iint_S (\nabla\times\mathbf{F})\cdot d\mathbf{S}$$

**AI/ML Connection 🤖:** The **Divergence Theorem** is the continuous analogue of **conservation laws** in physics-informed neural networks. **Neural PDE solvers** (e.g., Physics-Informed Neural Networks by Raissi et al.) solve PDEs whose weak forms involve surface and volume integrals. In **geometric deep learning** on 3D meshes or point clouds, integrals over surfaces are discretised and approximated using graph neural networks.

### Strategy Selection Guide

```
Given ∫f(x)dx, ask in order:

1. Is it a basic form? → Use §1–§5 directly.

2. Can u-sub work? → Look for g(x) and g'(x) together.
   (Especially: e^{u}, ln, trig, rational with recognisable numerator)

3. Does it involve √(a²−x²), √(x²+a²), √(x²−a²)?
   → Trig substitution (§8)

4. Is it a product: polynomial × (exp/trig/ln)?
   → Integration by parts, LIATE (§7)

5. Is it a rational function P/Q?
   → Partial fractions (§9)
   → If quadratic denominator doesn't factor: complete the square (§11)

6. Does it involve sin^n or cos^n?
   → Even power: half-angle identity
   → Odd power: factor one, u-sub the rest

7. None of the above → combination of techniques, or look it up.
```

## §17 · Quick Reference: Definite Integral Properties

|Property|Formula|
|---|---|
|Linearity|$\int_a^b [af+bg],dx = a\int_a^b f + b\int_a^b g$|
|Reversal|$\int_a^b f,dx = -\int_b^a f,dx$|
|Additivity|$\int_a^b f,dx = \int_a^c f,dx + \int_c^b f,dx$|
|Symmetry (even)|$\int_{-a}^a f,dx = 2\int_0^a f,dx$ if $f$ even|
|Symmetry (odd)|$\int_{-a}^a f,dx = 0$ if $f$ odd|
|Comparison|$f\leq g$ on $[a,b] \Rightarrow \int_a^b f \leq \int_a^b g$|
|Mean Value Theorem|$\exists c \in [a,b]:\ f(c) = \frac{1}{b-a}\int_a^b f(x),dx$|

**AI/ML Connection 🤖 — The odd symmetry property** is why $\int_{-\infty}^\infty x \cdot p(x),dx = 0$ when $p$ is a symmetric distribution and why **batch normalisation** in neural networks often centres the distribution (removing the odd part). The **Mean Value Theorem for integrals** is the theoretical backbone of **Monte Carlo integration** — the sample mean approximates the integral.
