## §1 · The Core Idea — Why Trig Substitution Works

We want to integrate expressions involving $\sqrt{a^2 - x^2}$, $\sqrt{a^2 + x^2}$, or $\sqrt{x^2 - a^2}$.

These square roots resist every technique seen so far — they are not derivatives of simple functions, and $u$-substitution leaves the radical intact.

**The key insight:** the Pythagorean theorem eliminates square roots of quadratics. Specifically:

$$1 - \sin^2\theta = \cos^2\theta \implies \sqrt{a^2 - a^2\sin^2\theta} = a\cos\theta$$

$$1 + \tan^2\theta = \sec^2\theta \implies \sqrt{a^2 + a^2\tan^2\theta} = a\sec\theta$$

$$\sec^2\theta - 1 = \tan^2\theta \implies \sqrt{a^2\sec^2\theta - a^2} = a\tan\theta$$

We introduce $x = a\cdot(\text{trig function of }\theta)$ to force the radicand into a perfect square.

> _This is the substitution $u = g(\theta)$ running in reverse: we substitute $x$ as a function of $\theta$, integrate in $\theta$, then back-substitute to express the answer in $x$. The triangle is the back-substitution machine._

## §2 · The Three Cases — Substitution Table

|Radical form|Substitution|Range|Identity used|Simplified radical|
|---|---|---|---|---|
|$\sqrt{a^2 - x^2}$|$x = a\sin\theta$|$-\dfrac{\pi}{2} \leq \theta \leq \dfrac{\pi}{2}$|$1-\sin^2\theta = \cos^2\theta$|$a\cos\theta$|
|$\sqrt{a^2 + x^2}$|$x = a\tan\theta$|$-\dfrac{\pi}{2} < \theta < \dfrac{\pi}{2}$|$1+\tan^2\theta = \sec^2\theta$|$a\sec\theta$|
|$\sqrt{x^2 - a^2}$|$x = a\sec\theta$|$0 \leq \theta < \dfrac{\pi}{2}$ or $\pi \leq \theta < \dfrac{3\pi}{2}$|$\sec^2\theta-1 = \tan^2\theta$|$a\tan\theta$|

**Why these specific ranges?**

For $x = a\sin\theta$: we need $\cos\theta \geq 0$ so that $\sqrt{a^2-x^2} = a\cos\theta$ (not $-a\cos\theta$). This holds on $[-\pi/2, \pi/2]$.

For $x = a\tan\theta$: we need $\sec\theta > 0$, which holds on $(-\pi/2, \pi/2)$.

For $x = a\sec\theta$: $\tan\theta \geq 0$ on $[0, \pi/2)$; if $x < -a$, use $[\pi, 3\pi/2)$.

> _The range restriction is not bureaucracy — it ensures the substitution is a bijection (invertible) on that domain, which is required for $u$-substitution to be valid. Ignoring the range leads to subtle sign errors in back-substitution._

## §3 · The Reference Triangles

Draw these triangles _before_ doing any algebra. They encode all back-substitution automatically.

### Case 1: $x = a\sin\theta$

```
        hypotenuse = a
       /|
      / |
     /  |  opposite = x
    /   |
   / θ  |
  /_____|
adjacent = √(a²-x²)
```

$$\sin\theta = \frac{x}{a}, \quad \cos\theta = \frac{\sqrt{a^2-x^2}}{a}, \quad \tan\theta = \frac{x}{\sqrt{a^2-x^2}}$$

### Case 2: $x = a\tan\theta$

```
      hypotenuse = √(a²+x²)
       /|
      / |
     /  |  opposite = x
    /   |
   / θ  |
  /_____|
adjacent = a
```

$$\sin\theta = \frac{x}{\sqrt{a^2+x^2}}, \quad \cos\theta = \frac{a}{\sqrt{a^2+x^2}}, \quad \tan\theta = \frac{x}{a}$$

### Case 3: $x = a\sec\theta$

```
      hypotenuse = x
       /|
      / |
     /  |  opposite = √(x²-a²)
    /   |
   / θ  |
  /_____|
adjacent = a
```

$$\sin\theta = \frac{\sqrt{x^2-a^2}}{x}, \quad \cos\theta = \frac{a}{x}, \quad \tan\theta = \frac{\sqrt{x^2-a^2}}{a}$$

> **Rule:** Label the triangle with the two sides you _know_ from the substitution. The third side follows from the Pythagorean theorem. All trig ratios are then read directly from the triangle — no algebra required.

## §4 · Case 1 — $\sqrt{a^2 - x^2}$: Substitution $x = a\sin\theta$

### Worked Example 1: $\displaystyle\int \frac{dx}{\sqrt{4-x^2}}$ (recovering arcsin)

Let $x = 2\sin\theta$, $dx = 2\cos\theta,d\theta$, $\sqrt{4-x^2} = 2\cos\theta$.

$$\int\frac{2\cos\theta,d\theta}{2\cos\theta} = \int d\theta = \theta + C$$

Back-substitute from triangle: $\sin\theta = x/2$, so $\theta = \arcsin(x/2)$.

$$= \arcsin\frac{x}{2} + C$$

**This recovers the arcsin formula.** Trig sub is the _proof_ that formula is correct.

---

### Worked Example 2: $\displaystyle\int\sqrt{9-x^2},dx$

Let $x = 3\sin\theta$, $dx = 3\cos\theta,d\theta$, $\sqrt{9-x^2} = 3\cos\theta$.

$$= \int 3\cos\theta \cdot 3\cos\theta,d\theta = 9\int\cos^2\theta,d\theta = 9\int\frac{1+\cos 2\theta}{2},d\theta$$

$$= \frac{9}{2}\left(\theta + \frac{\sin 2\theta}{2}\right) + C = \frac{9}{2}\theta + \frac{9\sin 2\theta}{4} + C$$

Now $\sin 2\theta = 2\sin\theta\cos\theta$. From triangle: $\sin\theta = x/3$, $\cos\theta = \sqrt{9-x^2}/3$.

$$\sin 2\theta = 2\cdot\frac{x}{3}\cdot\frac{\sqrt{9-x^2}}{3} = \frac{2x\sqrt{9-x^2}}{9}$$

$$= \frac{9}{2}\arcsin\frac{x}{3} + \frac{9}{4}\cdot\frac{2x\sqrt{9-x^2}}{9} + C = \boxed{\frac{9}{2}\arcsin\frac{x}{3} + \frac{x\sqrt{9-x^2}}{2} + C}$$

**Geometric interpretation:** $\int_{-3}^3\sqrt{9-x^2},dx = 9\pi/2$ is the area of a semicircle of radius 3. ✓ (since $\pi r^2/2 = 9\pi/2$.)

---

### Worked Example 3: $\displaystyle\int\frac{x^2}{\sqrt{9-x^2}},dx$

Let $x = 3\sin\theta$, $dx = 3\cos\theta,d\theta$, $\sqrt{9-x^2} = 3\cos\theta$, $x^2 = 9\sin^2\theta$.

$$= \int\frac{9\sin^2\theta}{3\cos\theta}\cdot 3\cos\theta,d\theta = 9\int\sin^2\theta,d\theta = \frac{9}{2}\int(1-\cos 2\theta),d\theta$$

$$= \frac{9}{2}\left(\theta - \frac{\sin 2\theta}{2}\right) + C = \frac{9}{2}\theta - \frac{9\sin 2\theta}{4} + C$$

From triangle: $\theta = \arcsin(x/3)$, $\sin 2\theta = 2x\sqrt{9-x^2}/9$.

$$= \boxed{\frac{9}{2}\arcsin\frac{x}{3} - \frac{x\sqrt{9-x^2}}{2} + C}$$

---

### Worked Example 4 (definite): $\displaystyle\int_0^{3/2}\frac{x^2}{\sqrt{9-x^2}},dx$

**Method 1 — back-substitute, then apply limits.** Use the antiderivative from Example 3: $$\left[\frac{9}{2}\arcsin\frac{x}{3} - \frac{x\sqrt{9-x^2}}{2}\right]_0^{3/2}$$ $$= \frac{9}{2}\arcsin\frac{1}{2} - \frac{(3/2)\sqrt{9-9/4}}{2} - 0 = \frac{9}{2}\cdot\frac{\pi}{6} - \frac{(3/2)(3\sqrt{3}/2)}{2} = \frac{3\pi}{4} - \frac{9\sqrt{3}}{8}$$

**Method 2 — change limits to $\theta$.** When $x=0$: $\sin\theta=0$, so $\theta=0$. When $x=3/2$: $\sin\theta=1/2$, so $\theta=\pi/6$.

$$9\int_0^{\pi/6}\sin^2\theta,d\theta = \frac{9}{2}\left[\theta-\frac{\sin 2\theta}{2}\right]_0^{\pi/6} = \frac{9}{2}\left(\frac{\pi}{6}-\frac{\sin(\pi/3)}{2}\right) = \frac{9}{2}\left(\frac{\pi}{6}-\frac{\sqrt{3}}{4}\right) = \frac{3\pi}{4}-\frac{9\sqrt{3}}{8}$$

Both methods agree. ✓

## §5 · Case 2 — $\sqrt{a^2 + x^2}$: Substitution $x = a\tan\theta$

### Worked Example 5: $\displaystyle\int\frac{dx}{\sqrt{x^2+4}}$

Let $x = 2\tan\theta$, $dx = 2\sec^2\theta,d\theta$, $\sqrt{x^2+4} = 2\sec\theta$.

$$= \int\frac{2\sec^2\theta,d\theta}{2\sec\theta} = \int\sec\theta,d\theta = \ln|\sec\theta+\tan\theta|+C$$

From triangle: $\tan\theta = x/2$, $\sec\theta = \sqrt{x^2+4}/2$.

$$= \ln\left|\frac{\sqrt{x^2+4}}{2}+\frac{x}{2}\right|+C = \ln\left|x+\sqrt{x^2+4}\right|+C$$

(absorbed the $\ln(1/2)$ into $C$.)

$$= \boxed{\ln\left(x+\sqrt{x^2+4}\right)+C}$$

This equals $\sinh^{-1}(x/2)+C$ — connecting to the inverse hyperbolic forms.

---

### Worked Example 6: $\displaystyle\int\frac{dx}{(x^2+4)^{3/2}}$

Let $x = 2\tan\theta$, $dx = 2\sec^2\theta,d\theta$, $(x^2+4)^{3/2} = 8\sec^3\theta$.

$$= \int\frac{2\sec^2\theta,d\theta}{8\sec^3\theta} = \frac{1}{4}\int\frac{d\theta}{\sec\theta} = \frac{1}{4}\int\cos\theta,d\theta = \frac{\sin\theta}{4}+C$$

From triangle: $\sin\theta = x/\sqrt{x^2+4}$.

$$= \boxed{\frac{x}{4\sqrt{x^2+4}}+C}$$

**Verification:** $\frac{d}{dx}\left[\frac{x}{4\sqrt{x^2+4}}\right] = \frac{1}{4}\cdot\frac{\sqrt{x^2+4}-x\cdot\frac{x}{\sqrt{x^2+4}}}{x^2+4} = \frac{1}{4}\cdot\frac{4}{(x^2+4)^{3/2}} = \frac{1}{(x^2+4)^{3/2}}$ ✓

---

### Worked Example 7: $\displaystyle\int x^3\sqrt{x^2+9},dx$

**Note:** Before jumping to trig sub, check if $u$-sub works. Let $u = x^2+9$, $du = 2x,dx$, $x^2 = u-9$:

$$= \int(u-9)\sqrt{u}\cdot\frac{du}{2} = \frac{1}{2}\int(u^{3/2}-9u^{1/2}),du = \frac{1}{2}\left(\frac{2u^{5/2}}{5}-\frac{18u^{3/2}}{3}\right)+C$$

$$= \frac{(x^2+9)^{5/2}}{5}-3(x^2+9)^{3/2}+C$$

> **Lesson:** Trig sub is not always the best tool even when a radical is present. When $x^{\text{odd}}\cdot\sqrt{x^2+a^2}$ appears, try $u = x^2+a^2$ first — it is often cleaner.

---

### Worked Example 8: $\displaystyle\int\sqrt{x^2+4},dx$

Let $x=2\tan\theta$, $dx=2\sec^2\theta,d\theta$, $\sqrt{x^2+4}=2\sec\theta$.

$$= \int 2\sec\theta\cdot 2\sec^2\theta,d\theta = 4\int\sec^3\theta,d\theta$$

Using the result from the Trigonometric Integrals note:

$$= 4\cdot\frac{\sec\theta\tan\theta+\ln|\sec\theta+\tan\theta|}{2}+C = 2\sec\theta\tan\theta+2\ln|\sec\theta+\tan\theta|+C$$

From triangle: $\sec\theta = \sqrt{x^2+4}/2$, $\tan\theta = x/2$.

$$= 2\cdot\frac{\sqrt{x^2+4}}{2}\cdot\frac{x}{2}+2\ln\left|\frac{\sqrt{x^2+4}}{2}+\frac{x}{2}\right|+C$$

$$= \boxed{\frac{x\sqrt{x^2+4}}{2}+2\ln\left(x+\sqrt{x^2+4}\right)+C}$$

(absorbed $\ln(1/2)$ into $C$.)

## §6 · Case 3 — $\sqrt{x^2 - a^2}$: Substitution $x = a\sec\theta$

This case requires the most care because $|x| \geq a$, and the domain splits into $x \geq a$ and $x \leq -a$.

### Worked Example 9: $\displaystyle\int\frac{\sqrt{x^2-9}}{x},dx$

Let $x = 3\sec\theta$, $dx = 3\sec\theta\tan\theta,d\theta$, $\sqrt{x^2-9} = 3\tan\theta$.

$$= \int\frac{3\tan\theta}{3\sec\theta}\cdot 3\sec\theta\tan\theta,d\theta = 3\int\tan^2\theta,d\theta = 3\int(\sec^2\theta-1),d\theta$$

$$= 3(\tan\theta - \theta)+C$$

From triangle: $\tan\theta = \sqrt{x^2-9}/3$, $\sec\theta = x/3$ so $\theta = \text{arcsec}(x/3)$.

$$= \boxed{\sqrt{x^2-9} - 3,\text{arcsec}\frac{x}{3}+C}$$

---

### Worked Example 10: $\displaystyle\int\frac{dx}{x^2\sqrt{x^2-25}}$

Let $x = 5\sec\theta$, $dx = 5\sec\theta\tan\theta,d\theta$, $\sqrt{x^2-25}=5\tan\theta$, $x^2=25\sec^2\theta$.

$$= \int\frac{5\sec\theta\tan\theta,d\theta}{25\sec^2\theta\cdot 5\tan\theta} = \frac{1}{25}\int\frac{d\theta}{\sec\theta} = \frac{1}{25}\int\cos\theta,d\theta = \frac{\sin\theta}{25}+C$$

From triangle: $\sin\theta = \sqrt{x^2-25}/x$.

$$= \boxed{\frac{\sqrt{x^2-25}}{25x}+C}$$

## §7 · Completing the Square — Extending Trig Sub to General Quadratics

When the expression under the radical is $ax^2+bx+c$ (with $a\neq 1$ or $b\neq 0$), complete the square first to reduce to one of the three standard forms.

**Algorithm:** $$ax^2+bx+c = a\left[\left(x+\frac{b}{2a}\right)^2 + \frac{c}{a}-\frac{b^2}{4a^2}\right]$$

Let $u = x + b/(2a)$, which shifts the variable. Then the form under the radical becomes $A(u^2\pm k^2)$ or $A(k^2-u^2)$, and the appropriate substitution applies.

---

### Worked Example 11: $\displaystyle\int\frac{dx}{\sqrt{x^2-6x+13}}$

Complete the square: $x^2-6x+13 = (x-3)^2+4$.

Let $u = x-3$, $du = dx$:

$$= \int\frac{du}{\sqrt{u^2+4}}$$

This is Example 5 with $a=2$! From that result:

$$= \ln\left|u+\sqrt{u^2+4}\right|+C = \boxed{\ln\left|(x-3)+\sqrt{x^2-6x+13}\right|+C}$$

---

### Worked Example 12: $\displaystyle\int\frac{2x+1}{\sqrt{-x^2+4x+5}},dx$

Complete the square: $-x^2+4x+5 = -(x^2-4x)+5 = -[(x-2)^2-4]+5 = 9-(x-2)^2$.

Let $u = x-2$, $x = u+2$, $dx = du$, $2x+1 = 2(u+2)+1 = 2u+5$:

$$= \int\frac{2u+5}{\sqrt{9-u^2}},du = \int\frac{2u}{\sqrt{9-u^2}},du + 5\int\frac{du}{\sqrt{9-u^2}}$$

First integral: let $w = 9-u^2$, $dw = -2u,du$: $$\int\frac{2u}{\sqrt{9-u^2}},du = -\int w^{-1/2},dw = -2\sqrt{w} = -2\sqrt{9-u^2}$$

Second integral: standard form #22: $$5\int\frac{du}{\sqrt{9-u^2}} = 5\arcsin\frac{u}{3}$$

$$= -2\sqrt{9-(x-2)^2}+5\arcsin\frac{x-2}{3}+C = \boxed{-2\sqrt{-x^2+4x+5}+5\arcsin\frac{x-2}{3}+C}$$

---

### Worked Example 13 (the hardest type): $\displaystyle\int\frac{dx}{(5+4x-x^2)^{3/2}}$

Complete the square: $5+4x-x^2 = -(x^2-4x-5) = -[(x-2)^2-9] = 9-(x-2)^2$.

So $(5+4x-x^2)^{3/2} = (9-(x-2)^2)^{3/2}$.

Let $u=x-2$:

$$= \int\frac{du}{(9-u^2)^{3/2}}$$

Now $u = 3\sin\theta$, $du = 3\cos\theta,d\theta$, $(9-u^2)^{3/2} = 27\cos^3\theta$:

$$= \int\frac{3\cos\theta,d\theta}{27\cos^3\theta} = \frac{1}{9}\int\sec^2\theta,d\theta = \frac{\tan\theta}{9}+C$$

From triangle: $\sin\theta = u/3$, so $\cos\theta = \sqrt{9-u^2}/3$, $\tan\theta = u/\sqrt{9-u^2}$.

$$= \frac{u}{9\sqrt{9-u^2}}+C = \boxed{\frac{x-2}{9\sqrt{5+4x-x^2}}+C}$$

## §9 · Strategy — When to Use Trig Sub vs. Other Methods

```
Expression involves √(a²−x²), √(a²+x²), or √(x²−a²)?
│
├─ Is there an odd power of x outside the radical?
│   Example: x³√(x²+4), x√(9−x²)
│   → Try u = (expression under radical) FIRST
│      If u-sub closes → done. If not → trig sub.
│
├─ Is it purely the radical alone, or radical with even powers?
│   Example: ∫√(a²−x²)dx, ∫x²/√(a²−x²)dx
│   → Trig sub directly
│
├─ Is the quadratic not in standard form?
│   Example: √(x²−6x+13), √(5+4x−x²)
│   → Complete the square first → standard form → trig sub
│
└─ Is the integrand rational in √(a²±x²) after substitution?
    Example: ∫dx/(x²√(x²−25))
    → Trig sub → result is trig integral (usually just ∫cosθ or ∫sinθ)
```

**The five-step procedure:**

1. **Identify** the radical form and the value of $a$.
2. **Make the substitution** from the table in §2.
3. **Draw the triangle** and label all three sides.
4. **Convert entirely** to $\theta$: radical, $dx$, any remaining $x$ terms.
5. **Integrate in $\theta$**, then **back-substitute** using the triangle.

## §10 · Connection to Inverse Hyperbolic Functions

The results of trig sub are often rewritable using inverse hyperbolic functions (which arise naturally from the $x = a\tan\theta$ and $x = a\sec\theta$ substitutions):

$$\int\frac{dx}{\sqrt{x^2+a^2}} = \ln\left(x+\sqrt{x^2+a^2}\right)+C = \sinh^{-1}\frac{x}{a}+C$$

$$\int\frac{dx}{\sqrt{x^2-a^2}} = \ln\left|x+\sqrt{x^2-a^2}\right|+C = \cosh^{-1}\frac{x}{a}+C, \quad x>a$$

$$\int\frac{dx}{a^2-x^2} = \frac{1}{a}\tanh^{-1}\frac{x}{a}+C = \frac{1}{2a}\ln\left|\frac{a+x}{a-x}\right|+C, \quad |x|<a$$

The equivalence $\ln(x+\sqrt{x^2+a^2}) = \sinh^{-1}(x/a)$ follows from the definition $\sinh^{-1}(t) = \ln(t+\sqrt{t^2+1})$. Both forms appear in tables; know that they are equal.

## §11 · Hard Examples — Comprehensive Integration

### Worked Example 14: $\displaystyle\int\frac{\sqrt{4x^2-9}}{x^2},dx$, $x > 3/2$

Factor: $\sqrt{4x^2-9} = \sqrt{4(x^2-9/4)} = 2\sqrt{x^2-(3/2)^2}$.

Let $a = 3/2$, $x = \frac{3}{2}\sec\theta$, $dx = \frac{3}{2}\sec\theta\tan\theta,d\theta$, $\sqrt{x^2-9/4} = \frac{3}{2}\tan\theta$.

$$= \int\frac{2\cdot\frac{3}{2}\tan\theta}{\frac{9}{4}\sec^2\theta}\cdot\frac{3}{2}\sec\theta\tan\theta,d\theta = \int\frac{3\tan\theta\cdot\frac{3}{2}\sec\theta\tan\theta}{\frac{9}{4}\sec^2\theta},d\theta = \int\frac{\frac{9}{2}\sec\theta\tan^2\theta}{\frac{9}{4}\sec^2\theta},d\theta$$

$$= 2\int\frac{\tan^2\theta}{\sec\theta},d\theta = 2\int\sin\theta\tan\theta,d\theta = 2\int\frac{\sin^2\theta}{\cos\theta},d\theta$$

$$= 2\int\frac{1-\cos^2\theta}{\cos\theta},d\theta = 2\int(\sec\theta-\cos\theta),d\theta = 2\ln|\sec\theta+\tan\theta|-2\sin\theta+C$$

From triangle ($x = \frac{3}{2}\sec\theta$): $\sec\theta = \frac{2x}{3}$, $\tan\theta = \frac{\sqrt{4x^2-9}}{3}$, $\sin\theta = \frac{\sqrt{4x^2-9}}{2x}$.

$$= \boxed{2\ln\left|\frac{2x}{3}+\frac{\sqrt{4x^2-9}}{3}\right|-\frac{\sqrt{4x^2-9}}{x}+C}$$

### Worked Example 15: $\displaystyle\int_0^2\frac{x^2}{\sqrt{4-x^2}^3},dx$

Let $x = 2\sin\theta$. Limits: $x=0\Rightarrow\theta=0$; $x=2\Rightarrow\theta=\pi/2$.

$(4-x^2)^{3/2} = 8\cos^3\theta$, $x^2 = 4\sin^2\theta$, $dx = 2\cos\theta,d\theta$.

$$= \int_0^{\pi/2}\frac{4\sin^2\theta\cdot 2\cos\theta,d\theta}{8\cos^3\theta} = \int_0^{\pi/2}\frac{\sin^2\theta}{\cos^2\theta},d\theta = \int_0^{\pi/2}\tan^2\theta,d\theta$$

$$= \int_0^{\pi/2}(\sec^2\theta-1),d\theta = [\tan\theta-\theta]_0^{\pi/2}$$

But $\tan(\pi/2) = \infty$! The integral **diverges** — this is an improper integral. Verify: near $x=2$, $(4-x^2)^{3/2}\to 0$ faster than $x^2\to 4$, so the integrand blows up. ✓

## §12 · AI / Machine Learning Connections

### 12.1 The Gaussian Integral — Foundation of All Probabilistic ML

The most important integral in machine learning is:

$$\int_{-\infty}^\infty e^{-x^2},dx = \sqrt{\pi}$$

Its **multivariable proof** (MIT 18.02) uses polar coordinate substitution — which is trig sub in disguise. The key step:

$$\left(\int_{-\infty}^\infty e^{-x^2},dx\right)^2 = \int_0^{2\pi}\int_0^\infty e^{-r^2},r,dr,d\theta$$

The substitution $x = r\cos\theta$, $y = r\sin\theta$ is trig substitution in 2D. Every normal distribution, every Gaussian process, every VAE, every diffusion model is normalised by this integral.

The generalisation to $d$ dimensions:

$$\int_{\mathbb{R}^d} e^{-|x|^2/(2\sigma^2)},dx = (2\pi\sigma^2)^{d/2}$$

is proved by $d$ sequential trig substitutions (or one spherical coordinate substitution — the $d$-dimensional analogue).

---

### 12.2 Normalisation Constants in Kernel Methods

**Gaussian RBF kernel:** $K(x,x') = e^{-|x-x'|^2/(2\sigma^2)}$

The fact that this kernel is positive-definite (required for a valid SVM kernel) is proved via Bochner's theorem: $K$ is the Fourier transform of $p(\omega) = (2\pi\sigma^2)^{-d/2}e^{-\sigma^2|\omega|^2/2}$, which requires the integral

$$\hat{K}(\omega) = \int_{\mathbb{R}^d} K(x)e^{-i\omega^Tx},dx$$

This is a $d$-dimensional trig integral, evaluated by completing the square in the exponent and using the Gaussian integral result — whose proof is the polar trig substitution above.

---

### 12.3 Computing the Partition Function in Energy-Based Models

In statistical mechanics and **energy-based models (EBMs)**, the normalisation constant (partition function) is:

$$Z = \int_{\mathbb{R}^d} e^{-E(x)/T},dx$$

For a quadratic energy $E(x) = \frac{1}{2}x^T A x$ (multivariate Gaussian):

$$Z = \int e^{-\frac{1}{2}x^TAx},dx = \frac{(2\pi)^{d/2}}{\sqrt{\det A}}$$

This is proved by:

1. Diagonalise $A = Q\Lambda Q^T$ (change of variables $u = Q^Tx$, Jacobian = 1)
2. Each dimension separates: $\prod_{i=1}^d \int e^{-\lambda_i u_i^2/2},du_i = \prod_i\sqrt{2\pi/\lambda_i}$
3. Each factor uses the Gaussian integral, proved via trig substitution in polar form

The result $Z = (2\pi)^{d/2}/\sqrt{\det A}$ appears in every **Gaussian process**, every **Bayesian linear regression**, and every **Laplace approximation** in deep learning.

### 12.4 Arc Length and Geodesics in Latent Space

In **variational autoencoders** and **Riemannian manifold learning**, one computes arc lengths along curves in a learned latent space. The arc length of a parametric curve $\gamma(t) = (x(t), y(t))$ is:

$$L = \int_a^b\sqrt{x'(t)^2 + y'(t)^2},dt$$

For the unit circle $x = \cos t$, $y = \sin t$: $L = \int_0^{2\pi} 1,dt = 2\pi$ — trivial. But for an **ellipse** $x = a\cos t$, $y = b\sin t$:

$$L = \int_0^{2\pi}\sqrt{a^2\sin^2 t + b^2\cos^2 t},dt$$

This is an **elliptic integral** — not expressible in closed form — but it is reduced to standard form via trig substitution. **Geodesic distance** in learned latent manifolds (used in disentangled representation learning) requires exactly this arc-length computation.

### 12.5 Attention Score Normalisation — The $\sqrt{d_k}$ Factor

The scaled dot-product attention in Transformers:

$$\text{Attention}(Q,K,V) = \text{softmax}!\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

The $\sqrt{d_k}$ in the denominator is explained as follows. If $q_i, k_j \sim \mathcal{N}(0,1)$ independently, then $q_i \cdot k_j = \sum_{l=1}^{d_k} q_{il}k_{jl}$ has variance $d_k$.

The standard deviation of $q\cdot k$ is $\sqrt{d_k}$, computed via:

$$\text{Var}\left[\sum_{l=1}^{d_k} q_l k_l\right] = \sum_{l=1}^{d_k}\text{Var}[q_l k_l] = d_k$$

where $\text{Var}[q_l k_l] = \mathbb{E}[q_l^2 k_l^2] = \mathbb{E}[q_l^2]\mathbb{E}[k_l^2] = 1$ (computed using the Gaussian integral $\int x^2 e^{-x^2/2}/\sqrt{2\pi},dx = 1$, which again uses the trig-based Gaussian integral at its core).

### 12.6 Normalising Flows — The Jacobian Determinant

In **normalising flows** (RealNVP, Glow, Neural Spline Flows), we learn an invertible map $f:\mathbb{R}^d\to\mathbb{R}^d$ and the log-likelihood is:

$$\log p_X(x) = \log p_Z(f(x)) + \log\left|\det\frac{\partial f}{\partial x}\right|$$

For **radial flows** $f(z) = z + \beta h(\alpha,r)(z-z_0)$ where $r = |z-z_0|$, computing $\det\partial f/\partial z$ requires evaluating:

$$\int_S h(\alpha, r),dS$$

over a sphere — a surface integral evaluated via spherical trig substitution. The residual stream geometry of **transformer circuits** is analysed using exactly these integrals.

## §13 · Common Mistakes

**Mistake 1 — Wrong case.** Seeing $\sqrt{x^2+9}$ and substituting $x = 3\sin\theta$ (Case 1) instead of $x = 3\tan\theta$ (Case 2). The sine substitution gives $\sqrt{9-9\sin^2\theta+9} = \sqrt{18-9\sin^2\theta}$, which is not simpler. Always match the form to the table.

**Mistake 2 — Forgetting $dx$.** Writing $x = a\sin\theta$ but forgetting to replace $dx = a\cos\theta,d\theta$. This is the most common error — the $d\theta$ substitution _must_ be applied to $dx$ as well.

**Mistake 3 — Sign errors from the range.** $\sqrt{a^2-a^2\sin^2\theta} = a|\cos\theta|$. On $[-\pi/2,\pi/2]$, $\cos\theta \geq 0$, so $|\cos\theta| = \cos\theta$. If you accidentally use $\theta$ outside this range, you get a sign flip.

**Mistake 4 — Not completing the square first.** Attempting trig sub on $\sqrt{x^2-4x+8}$ directly. The correct first step is completing the square: $x^2-4x+8=(x-2)^2+4$, then substituting $u=x-2$.

**Mistake 5 — Incorrect back-substitution.** Writing $\sin\theta = x/a$ then guessing $\cos\theta = \sqrt{1-x^2/a^2}$ from memory, with an error in the sign or the structure. **Draw the triangle** every time — it prevents all back-substitution errors.

**Mistake 6 — Using trig sub when $u$-sub suffices.** $\int x\sqrt{a^2-x^2},dx$: let $u=a^2-x^2$, $du=-2x,dx$. This gives $-\frac{1}{2}\int u^{1/2},du = -\frac{u^{3/2}}{3}+C = -\frac{(a^2-x^2)^{3/2}}{3}+C$. No trig needed.
