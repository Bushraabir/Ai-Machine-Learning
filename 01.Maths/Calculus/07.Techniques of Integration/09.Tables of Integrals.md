## 0. How to Use This Table

Every entry here is a **theorem**, not a fact to memorise blindly. Each formula can be derived from first principles using the techniques in [[Integration Strategy]] and [[Rationalising Substitutions]]. The derivation sketches are included so the table stays alive.

**Conventions throughout:**

- $a, b, n$ are real constants with $a \ne 0$ unless stated.
- $n \ne -1$ in power rules unless stated.
- $+, C$ (arbitrary constant) is omitted from every entry — always append it to indefinite integrals.
- All logarithms are natural: $\ln = \log_e$.
- $\sqrt{x^2 \pm a^2}$ and $\sqrt{a^2 - x^2}$ implicitly assume the expression under the root is non-negative.

## 1. Basic Forms

### 1.1 Power Rule

$$\int x^n,dx = \frac{x^{n+1}}{n+1}, \qquad n \ne -1.$$

$$\int x^{-1},dx = \int \frac{dx}{x} = \ln|x|.$$

**Why:** $\frac{d}{dx}\frac{x^{n+1}}{n+1} = x^n$. The case $n=-1$ is separate because $\frac{x^0}{0}$ is undefined.

**Generalisation:** $$\int (ax+b)^n,dx = \frac{(ax+b)^{n+1}}{a(n+1)}, \qquad n \ne -1.$$

$$\int \frac{dx}{ax+b} = \frac{1}{a}\ln|ax+b|.$$

---

### 1.2 Exponential

$$\int e^x,dx = e^x.$$

$$\int e^{ax},dx = \frac{e^{ax}}{a}.$$

$$\int a^x,dx = \frac{a^x}{\ln a}, \qquad a > 0,\ a \ne 1.$$

**Why:** $\frac{d}{dx}e^{ax} = ae^{ax}$, so divide by $a$. For $a^x$: write $a^x = e^{x\ln a}$ and apply the previous formula.

---

### 1.3 Logarithm

$$\int \ln x,dx = x\ln x - x.$$

**Derivation (IBP):** $u = \ln x$, $dv = dx$: $$x\ln x - \int x\cdot\frac{1}{x},dx = x\ln x - x.$$

$$\int \log_a x,dx = \frac{x\ln x - x}{\ln a}.$$

$$\int x^n \ln x,dx = \frac{x^{n+1}}{n+1}\ln x - \frac{x^{n+1}}{(n+1)^2}, \qquad n \ne -1.$$

---

## 2. Trigonometric Integrals

### 2.1 Basic Trig

$$\int \sin x,dx = -\cos x.$$

$$\int \cos x,dx = \sin x.$$

$$\int \tan x,dx = -\ln|\cos x| = \ln|\sec x|.$$

$$\int \cot x,dx = \ln|\sin x|.$$

$$\int \sec x,dx = \ln|\sec x + \tan x|.$$

$$\int \csc x,dx = -\ln|\csc x + \cot x| = \ln|\csc x - \cot x|.$$

**Derivation of $\int \sec x,dx$:** Multiply and divide by $(\sec x + \tan x)$: $$\int \frac{\sec x(\sec x + \tan x)}{\sec x + \tan x},dx = \int \frac{\sec^2 x + \sec x\tan x}{\sec x + \tan x},dx.$$ The numerator is exactly the derivative of the denominator. So the integral is $\ln|\sec x + \tan x|$.

---

### 2.2 Squares

$$\int \sin^2 x,dx = \frac{x}{2} - \frac{\sin 2x}{4}.$$

$$\int \cos^2 x,dx = \frac{x}{2} + \frac{\sin 2x}{4}.$$

$$\int \tan^2 x,dx = \tan x - x.$$

$$\int \sec^2 x,dx = \tan x.$$

$$\int \csc^2 x,dx = -\cot x.$$

$$\int \sec x\tan x,dx = \sec x.$$

$$\int \csc x\cot x,dx = -\csc x.$$

**Why the squares:** Use $\sin^2 x = \frac{1-\cos 2x}{2}$, $\cos^2 x = \frac{1+\cos 2x}{2}$, and $\tan^2 x = \sec^2 x - 1$.

---

### 2.3 Higher Powers — Reduction Formulas

Rather than memorising each power separately, derive the next from the previous. These are the **reduction formulas**:

$$\int \sin^n x,dx = -\frac{\sin^{n-1}x\cos x}{n} + \frac{n-1}{n}\int \sin^{n-2}x,dx.$$

$$\int \cos^n x,dx = \frac{\cos^{n-1}x\sin x}{n} + \frac{n-1}{n}\int \cos^{n-2}x,dx.$$

$$\int \tan^n x,dx = \frac{\tan^{n-1}x}{n-1} - \int \tan^{n-2}x,dx.$$

$$\int \sec^n x,dx = \frac{\sec^{n-2}x\tan x}{n-1} + \frac{n-2}{n-1}\int \sec^{n-2}x,dx.$$

**Derivation of the sine reduction formula (IBP):**

Let $u = \sin^{n-1}x$, $dv = \sin x,dx$. Then $du = (n-1)\sin^{n-2}x\cos x,dx$, $v = -\cos x$: $$\int \sin^n x,dx = -\sin^{n-1}x\cos x + (n-1)\int \sin^{n-2}x\cos^2 x,dx.$$

Replace $\cos^2 x = 1 - \sin^2 x$ and solve algebraically for $\int \sin^n x,dx$.

#### Example: $\int \sin^4 x,dx$

Apply the reduction formula twice: $$\int\sin^4 x = -\frac{\sin^3 x\cos x}{4} + \frac{3}{4}\int\sin^2 x,dx = -\frac{\sin^3 x\cos x}{4} + \frac{3}{4}\left(\frac{x}{2}-\frac{\sin 2x}{4}\right).$$ $$= -\frac{\sin^3 x\cos x}{4} + \frac{3x}{8} - \frac{3\sin 2x}{16}.$$

---

### 2.4 Products of Sines and Cosines

**Product-to-sum identities** (essential for these integrals):

$$\sin mx\cos nx = \frac{1}{2}[\sin(m+n)x + \sin(m-n)x].$$

$$\sin mx\sin nx = \frac{1}{2}[\cos(m-n)x - \cos(m+n)x].$$

$$\cos mx\cos nx = \frac{1}{2}[\cos(m-n)x + \cos(m+n)x].$$

These are derived from the angle-addition formulas. They convert products into sums, which integrate term by term.

$$\int \sin mx\cos nx,dx = -\frac{\cos(m+n)x}{2(m+n)} - \frac{\cos(m-n)x}{2(m-n)}, \quad m \ne \pm n.$$

## 3. Inverse Trigonometric Forms

These appear constantly when $x^2$ or $a^2 - x^2$ is in the denominator.

$$\int \frac{dx}{\sqrt{a^2 - x^2}} = \arcsin\frac{x}{a}, \qquad |x| < a.$$

$$\int \frac{dx}{a^2 + x^2} = \frac{1}{a}\arctan\frac{x}{a}.$$

$$\int \frac{dx}{x\sqrt{x^2 - a^2}} = \frac{1}{a}\operatorname{arcsec}\frac{|x|}{a}.$$

$$\int \frac{dx}{\sqrt{x^2 + a^2}} = \ln!\left|x + \sqrt{x^2+a^2}\right| = \sinh^{-1}!\frac{x}{a}.$$

$$\int \frac{dx}{\sqrt{x^2 - a^2}} = \ln!\left|x + \sqrt{x^2-a^2}\right| = \cosh^{-1}!\frac{x}{a}.$$

**Derivation of $\int \frac{dx}{a^2+x^2}$:** Factor out $a^2$: $$\frac{1}{a^2}\int \frac{dx}{1+(x/a)^2}.$$ Set $u = x/a$, $du = dx/a$: $$\frac{1}{a}\int\frac{du}{1+u^2} = \frac{1}{a}\arctan u = \frac{1}{a}\arctan\frac{x}{a}.$$

---

### 3.1 Completing the Square

When the denominator is $ax^2 + bx + c$ (irreducible), complete the square to reduce to the standard forms above.

$$ax^2 + bx + c = a!\left[\left(x + \frac{b}{2a}\right)^2 + \frac{4ac-b^2}{4a^2}\right].$$

Let $u = x + \frac{b}{2a}$, $k^2 = \frac{4ac-b^2}{4a^2}$:

$$\int \frac{dx}{ax^2+bx+c} = \frac{1}{a}\int \frac{du}{u^2+k^2} = \frac{1}{ak}\arctan\frac{u}{k}.$$

---

## 4. Integrals Involving $\sqrt{a^2 - x^2}$

$$\int \sqrt{a^2 - x^2},dx = \frac{x\sqrt{a^2-x^2}}{2} + \frac{a^2}{2}\arcsin\frac{x}{a}.$$

$$\int \frac{x^2,dx}{\sqrt{a^2-x^2}} = -\frac{x\sqrt{a^2-x^2}}{2} + \frac{a^2}{2}\arcsin\frac{x}{a}.$$

$$\int \frac{dx}{x\sqrt{a^2-x^2}} = -\frac{1}{a}\ln\left|\frac{a+\sqrt{a^2-x^2}}{x}\right|.$$

$$\int \frac{\sqrt{a^2-x^2}}{x},dx = \sqrt{a^2-x^2} - a\ln\left|\frac{a+\sqrt{a^2-x^2}}{x}\right|.$$

**Derivation of $\int\sqrt{a^2-x^2},dx$:** Set $x = a\sin\theta$: $$\int a^2\cos^2\theta,d\theta = \frac{a^2}{2}\int(1+\cos 2\theta),d\theta = \frac{a^2}{2}\left(\theta + \frac{\sin 2\theta}{2}\right).$$ Back-substitute: $\theta = \arcsin(x/a)$, $\sin 2\theta = 2\sin\theta\cos\theta = \frac{2x\sqrt{a^2-x^2}}{a^2}$.

---

## 5. Integrals Involving $\sqrt{x^2 + a^2}$

$$\int \sqrt{x^2+a^2},dx = \frac{x\sqrt{x^2+a^2}}{2} + \frac{a^2}{2}\ln!\left|x+\sqrt{x^2+a^2}\right|.$$

$$\int \frac{dx}{\sqrt{x^2+a^2}} = \ln!\left|x + \sqrt{x^2+a^2}\right|.$$

$$\int \frac{x^2,dx}{\sqrt{x^2+a^2}} = \frac{x\sqrt{x^2+a^2}}{2} - \frac{a^2}{2}\ln!\left|x+\sqrt{x^2+a^2}\right|.$$

$$\int \frac{\sqrt{x^2+a^2}}{x},dx = \sqrt{x^2+a^2} - a\ln\left|\frac{a+\sqrt{x^2+a^2}}{x}\right|.$$

$$\int \frac{dx}{x^2\sqrt{x^2+a^2}} = -\frac{\sqrt{x^2+a^2}}{a^2 x}.$$

---

## 6. Integrals Involving $\sqrt{x^2 - a^2}$

$$\int \sqrt{x^2-a^2},dx = \frac{x\sqrt{x^2-a^2}}{2} - \frac{a^2}{2}\ln!\left|x+\sqrt{x^2-a^2}\right|.$$

$$\int \frac{dx}{\sqrt{x^2-a^2}} = \ln!\left|x+\sqrt{x^2-a^2}\right|.$$

$$\int \frac{dx}{x^2\sqrt{x^2-a^2}} = \frac{\sqrt{x^2-a^2}}{a^2 x}.$$

$$\int \frac{\sqrt{x^2-a^2}}{x},dx = \sqrt{x^2-a^2} - a\operatorname{arcsec}\frac{|x|}{a}.$$

> **Mnemonic for signs:** $\sqrt{a^2-x^2}$ entries contain $\arcsin$; $\sqrt{x^2+a^2}$ and $\sqrt{x^2-a^2}$ entries contain $\ln|x + \sqrt{\cdot}|$. The sign in front of the $\ln$ term flips between the two.

---

## 7. Rational Functions

### 7.1 Linear Denominator

$$\int \frac{dx}{(x+a)^n} = \frac{(x+a)^{1-n}}{1-n}, \quad n \ne 1.$$

$$\int \frac{x,dx}{ax+b} = \frac{x}{a} - \frac{b}{a^2}\ln|ax+b|.$$

$$\int \frac{x,dx}{(ax+b)^2} = \frac{b}{a^2(ax+b)} + \frac{1}{a^2}\ln|ax+b|.$$

---

### 7.2 Quadratic Denominator

$$\int \frac{dx}{x^2-a^2} = \frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right|. \tag{partial fractions}$$

$$\int \frac{dx}{(x^2+a^2)^2} = \frac{x}{2a^2(x^2+a^2)} + \frac{1}{2a^3}\arctan\frac{x}{a}.$$

**Derivation of $\int \frac{dx}{x^2-a^2}$:** Partial fractions: $\frac{1}{x^2-a^2} = \frac{1}{2a}!\left(\frac{1}{x-a} - \frac{1}{x+a}\right)$.

---

### 7.3 Reduction Formula for Repeated Quadratics

$$\int \frac{dx}{(x^2+a^2)^n} = \frac{x}{2a^2(n-1)(x^2+a^2)^{n-1}} + \frac{2n-3}{2a^2(n-1)}\int\frac{dx}{(x^2+a^2)^{n-1}}.$$

---

## 8. Hyperbolic Functions

Hyperbolic functions appear in the integrals of $\sqrt{x^2 \pm a^2}$ and in physics/ML (they parametrise hyperboloids and appear in certain activation functions).

$$\int \sinh x,dx = \cosh x.$$

$$\int \cosh x,dx = \sinh x.$$

$$\int \tanh x,dx = \ln\cosh x.$$

$$\int \operatorname{sech}^2 x,dx = \tanh x.$$

$$\int \operatorname{csch}^2 x,dx = -\coth x.$$

$$\int \operatorname{sech} x\tanh x,dx = -\operatorname{sech} x.$$

### Inverse Hyperbolic (= logarithmic) Forms

$$\int \frac{dx}{\sqrt{x^2+1}} = \sinh^{-1} x = \ln(x + \sqrt{x^2+1}).$$

$$\int \frac{dx}{\sqrt{x^2-1}} = \cosh^{-1} x = \ln(x+\sqrt{x^2-1}), \quad x > 1.$$

$$\int \frac{dx}{1-x^2} = \tanh^{-1} x = \frac{1}{2}\ln\frac{1+x}{1-x}, \quad |x| < 1.$$

> **Note:** These are the same as the log forms in §3 and §5–6, just written differently. When a CAS returns $\sinh^{-1}$ and you expected $\ln$, they are identical.

---

## 9. Products: Exponential × Trig (IBP Results)

$$\int e^{ax}\sin bx,dx = \frac{e^{ax}(a\sin bx - b\cos bx)}{a^2+b^2}.$$

$$\int e^{ax}\cos bx,dx = \frac{e^{ax}(a\cos bx + b\sin bx)}{a^2+b^2}.$$

**Derivation:** Apply IBP twice (circular trick, §4.3 in [[Integration Strategy]]) and solve. The denominator $a^2+b^2$ always appears — it is the squared magnitude of the complex number $a + bi$.

**Alternative (complex exponential trick):** $$\int e^{ax}\cos bx,dx = \operatorname{Re}!\int e^{(a+ib)x},dx = \operatorname{Re}\frac{e^{(a+ib)x}}{a+ib} = \operatorname{Re}\frac{(a-ib)e^{ax}(\cos bx + i\sin bx)}{a^2+b^2}.$$

This is the "physicist's method" — it works over $\mathbb{C}$ and recovers both formulas simultaneously by taking real and imaginary parts.

---

## 10. Products: Polynomial × Elementary

These are tabular IBP results (see [[Integration Strategy]] §4.4). The pattern: $$\int p(x)\cdot f(x),dx = p(x)F(x) - p'(x)F_2(x) + p''(x)F_3(x) - \cdots$$ where $F_k$ is the $k$-th antiderivative of $f$.

$$\int x e^{ax},dx = \frac{e^{ax}(ax-1)}{a^2}.$$

$$\int x^2 e^{ax},dx = \frac{e^{ax}(a^2x^2 - 2ax + 2)}{a^3}.$$

$$\int x\sin ax,dx = \frac{\sin ax}{a^2} - \frac{x\cos ax}{a}.$$

$$\int x\cos ax,dx = \frac{\cos ax}{a^2} + \frac{x\sin ax}{a}.$$

$$\int x^2\sin ax,dx = \frac{2x\sin ax}{a^2} - \frac{(a^2x^2-2)\cos ax}{a^3}.$$

$$\int x^2\cos ax,dx = \frac{2x\cos ax}{a^2} + \frac{(a^2x^2-2)\sin ax}{a^3}.$$

## 11. Important Definite Integrals

These are used so frequently in analysis, probability, and ML that they deserve their own section.

### 11.1 Gaussian Integrals

$$\int_{-\infty}^{\infty} e^{-x^2},dx = \sqrt{\pi}. \tag{fundamental}$$

$$\int_0^{\infty} e^{-x^2},dx = \frac{\sqrt{\pi}}{2}.$$

$$\int_{-\infty}^{\infty} e^{-ax^2},dx = \sqrt{\frac{\pi}{a}}, \qquad a > 0.$$

$$\int_{-\infty}^{\infty} e^{-ax^2+bx},dx = \sqrt{\frac{\pi}{a}},e^{b^2/(4a)}, \qquad a > 0.$$

$$\int_{-\infty}^{\infty} x^2 e^{-ax^2},dx = \frac{\sqrt{\pi}}{2a^{3/2}}.$$

$$\int_0^{\infty} x^n e^{-ax},dx = \frac{n!}{a^{n+1}} = \frac{\Gamma(n+1)}{a^{n+1}}.$$

**Derivation of the fundamental Gaussian (polar trick):**

Let $I = \int_{-\infty}^\infty e^{-x^2}dx$. Then: $$I^2 = \left(\int_{-\infty}^\infty e^{-x^2}dx\right)!\left(\int_{-\infty}^\infty e^{-y^2}dy\right) = \int_{-\infty}^\infty\int_{-\infty}^\infty e^{-(x^2+y^2)},dx,dy.$$

Switch to polar: $x^2+y^2 = r^2$, $dx,dy = r,dr,d\theta$: $$I^2 = \int_0^{2\pi}\int_0^\infty e^{-r^2}r,dr,d\theta = 2\pi\cdot\frac{1}{2} = \pi.$$

Therefore $I = \sqrt{\pi}$.

> This is the most important integration technique in probability theory, and the foundation of all Gaussian distribution computations in ML.

---

### 11.2 Beta and Gamma Functions

The **Gamma function** is the continuous extension of the factorial: $$\Gamma(n) = \int_0^\infty t^{n-1}e^{-t},dt, \qquad n > 0.$$

Key values and properties:

$$\Gamma(n+1) = n,\Gamma(n), \qquad \Gamma(1) = 1, \qquad \Gamma(n) = (n-1)!\text{ for }n \in \mathbb{Z}^+.$$

$$\Gamma!\left(\tfrac{1}{2}\right) = \sqrt{\pi}.$$

The **Beta function**: $$B(m,n) = \int_0^1 t^{m-1}(1-t)^{n-1},dt = \frac{\Gamma(m),\Gamma(n)}{\Gamma(m+n)}.$$

This gives a clean formula for integrals of the form $\int_0^{\pi/2}\sin^m\theta\cos^n\theta,d\theta$: $$\int_0^{\pi/2}\sin^m\theta\cos^n\theta,d\theta = \frac{1}{2}B!\left(\frac{m+1}{2},\frac{n+1}{2}\right) = \frac{\Gamma!\left(\frac{m+1}{2}\right)\Gamma!\left(\frac{n+1}{2}\right)}{2,\Gamma!\left(\frac{m+n+2}{2}\right)}.$$

#### Wallis' Formula (special case)

$$\int_0^{\pi/2}\sin^n x,dx = \int_0^{\pi/2}\cos^n x,dx = \begin{cases} \dfrac{(n-1)!!}{n!!}\cdot\dfrac{\pi}{2} & n \text{ even} \[6pt] \dfrac{(n-1)!!}{n!!} & n \text{ odd} \end{cases}$$

where $n!! = n(n-2)(n-4)\cdots$ is the double factorial.

---

### 11.3 Dirichlet / Sinc Integral

$$\int_0^\infty \frac{\sin x}{x},dx = \frac{\pi}{2}.$$

This has no elementary antiderivative but the definite value is $\pi/2$. Derived via Feynman's trick or contour integration. It appears in signal processing (the sinc function $\operatorname{sinc}(x) = \frac{\sin\pi x}{\pi x}$ is the ideal low-pass filter).

## 12. Visualisation — The Anatomy of a Table Entry

Every entry in the table is a compression of the following pipeline:

```
Raw integrand f(x)
        │
        ▼
   Identify structure
   (radical? trig? rational? product?)
        │
        ▼
   Apply substitution / method
   (sub, IBP, trig sub, partial fractions)
        │
        ▼
   Reach a basic form
   (power, exp, log, arctan, arcsin, ...)
        │
        ▼
   Integrate the basic form
        │
        ▼
   Back-substitute
        │
        ▼
   Compact expression = TABLE ENTRY
```

The table is the output of this pipeline, run once by someone else, packaged for reuse. **When you use a table entry, mentally trace back through the pipeline** — this is what keeps the knowledge alive.

## 13. Connection to AI / Machine Learning

### 13.1 The Gaussian Table Entries Power All of Bayesian ML

The integrals in §11.1 are not background mathematics — they are used directly in every training loop that involves a Gaussian prior or likelihood.

**Marginalisation:** Given $p(x \mid \mu, \sigma^2) = \mathcal{N}(x; \mu, \sigma^2)$ and a Gaussian prior $p(\mu) = \mathcal{N}(\mu; m, v)$: $$p(x) = \int p(x|\mu),p(\mu),d\mu.$$ This integral is another Gaussian, computed by completing the square in the exponent — precisely the formula $\int e^{-ax^2+bx}dx = \sqrt{\pi/a},e^{b^2/4a}$ from §11.1.

**Reparametrisation trick (VAEs):** The ELBO in a Variational Autoencoder requires: $$\mathbb{E}_{z \sim \mathcal{N}(\mu,\sigma^2)}[f(z)] \approx \frac{1}{S}\sum_{s=1}^S f(\mu + \sigma\epsilon_s), \quad \epsilon_s \sim \mathcal{N}(0,1).$$ The change of variable $z = \mu + \sigma\epsilon$ is exactly the substitution that transforms $\int e^{-az^2+bz}dz$ into $\int e^{-\epsilon^2}d\epsilon$.

---

### 13.2 Gamma Function → Softmax → Attention

The **softmax** function used in neural network classifiers and Transformer attention is: $$\text{softmax}(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}.$$

Its continuous analogue is the **Gibbs measure**, and the partition function $Z = \int e^{f(x)}dx$ is a generalised Gaussian integral. Computing or approximating $Z$ (the normalising constant) is one of the central problems of probabilistic ML. The Gamma and Beta functions provide closed forms when $f$ has polynomial structure.

---

### 13.3 Beta Distribution → Bayesian A/B Testing

The Beta function $B(m,n)$ is the normalising constant of the **Beta distribution**: $$p(\theta \mid \alpha, \beta) = \frac{\theta^{\alpha-1}(1-\theta)^{\beta-1}}{B(\alpha,\beta)}, \qquad \theta \in [0,1].$$

This is the conjugate prior for the Bernoulli likelihood. Every Bayesian A/B test, Thompson sampling algorithm, and bandit policy in recommendation systems uses this distribution. The integral table entry $B(m,n) = \Gamma(m)\Gamma(n)/\Gamma(m+n)$ is what makes the posterior update analytic.

---

### 13.4 Reduction Formulas → Recursive Neural Computations

The reduction formulas in §2.3 have the structure: $$I_n = g(x,n) + \lambda_n I_{n-2}.$$

This is a **recurrence relation** — the same structural pattern as:

- RNN hidden state update: $h_t = f(h_{t-1}, x_t)$
- LSTM cell state: $c_t = f_{\text{forget}}\odot c_{t-1} + f_{\text{input}}\odot \tilde{c}_t$
- Dynamic programming in RL: $V(s) = r(s) + \gamma \max_{a} V(s')$

Reduction formulas are the calculus instantiation of the principle that complex computations decompose recursively into simpler ones — the same principle that makes deep learning architectures powerful.

---

### 13.5 Sinc Function → Fourier Analysis → CNNs

The sinc function $\operatorname{sinc}(f) = \frac{\sin(\pi f)}{\pi f}$ is the Fourier transform of the rectangular window. This connects to:

- **Convolutional Neural Networks:** convolution in the spatial domain = multiplication in the frequency domain (Convolution Theorem). Understanding CNNs rigorously requires Fourier analysis, which rests on these integral tables.
- **Sampling theory:** the Nyquist theorem says a signal bandlimited to $W$ Hz is perfectly reconstructed from samples at $2W$ Hz. The proof uses $\int_{-\infty}^\infty \operatorname{sinc}(x),dx = 1$ (the sinc integral from §11.3).
