## 0. The Fundamental Problem

Given $f(x)$, find $F(x)$ such that $F'(x) = f(x)$. Unlike differentiation — which is algorithmic — antidifferentiation has no guaranteed algorithm. What we have instead is a **decision tree of pattern recognition**, sharpened by practice.

The goal of this note is to internalise that decision tree so deeply it becomes instinct.

---

## 1. The Master Decision Tree

Before touching the integral, **look at the integrand**. Ask these questions in order.

```
∫ f(x) dx
    │
    ├─ 1. Is it a BASIC FORM? ──────────────────────────────► Integrate directly
    │       (power, exp, trig, 1/x, inverse trig)
    │
    ├─ 2. Can I SIMPLIFY first? ────────────────────────────► Algebra, then retry
    │       (expand, factor, trig identity, long division)
    │
    ├─ 3. Is there an OBVIOUS SUBSTITUTION? ────────────────► u-substitution
    │       (composite function, f'(x) lurking as a factor)
    │
    ├─ 4. Is it a PRODUCT of two "unrelated" families? ─────► Integration by Parts
    │       (poly × exp, poly × trig, poly × log, etc.)
    │
    ├─ 5. Does it involve √(a²−x²), √(a²+x²), √(x²−a²)? ──► Trig Substitution
    │
    ├─ 6. Is it RATIONAL (ratio of polynomials)? ───────────► Partial Fractions
    │       (degree check first: if deg num ≥ deg den, divide)
    │
    ├─ 7. Does it involve √[n]{ax+b} or mixed roots? ───────► Rationalising Sub
    │       (see [[Rationalising Substitutions]])
    │
    ├─ 8. Is it rational in sin x, cos x? ──────────────────► Weierstrass t=tan(x/2)
    │
    └─ 9. None of the above? ───────────────────────────────► Try IBP again,
              Try a non-obvious substitution,                    look for a trick,
              consult a table, or accept                         or accept it has
              no elementary antiderivative.                      no closed form.
```

> **Principle:** Never start integrating until you have a _plan_. Thirty seconds of looking saves ten minutes of wrong turns.

## 2. Step 0 — Simplify Before Anything Else

This step is so often skipped it deserves its own section.

### 2.1 Algebraic Simplification

$$\int \frac{x^2 + 3x + 2}{x+1},dx \quad\longrightarrow\quad \text{factor: } \frac{(x+1)(x+2)}{x+1} = x+2 \quad\longrightarrow\quad \int (x+2),dx.$$

### 2.2 Trigonometric Identities

Know these cold. They are the most frequent simplifiers:

|Identity|Use case|
|---|---|
|$\sin^2 x = \frac{1-\cos 2x}{2}$|$\int \sin^2 x,dx$|
|$\cos^2 x = \frac{1+\cos 2x}{2}$|$\int \cos^2 x,dx$|
|$\sin^2 x + \cos^2 x = 1$|Reduce odd powers|
|$1 + \tan^2 x = \sec^2 x$|$\int \tan^n x,dx$|
|$\sin 2x = 2\sin x\cos x$|Products $\sin x\cos x$|
|$\cos 2x = \cos^2 x - \sin^2 x$|Mixed even powers|

### 2.3 Long Division

If $\deg(\text{numerator}) \ge \deg(\text{denominator})$: **always divide first.**

$$\int \frac{x^3}{x^2+1},dx \quad\longrightarrow\quad \frac{x^3}{x^2+1} = x - \frac{x}{x^2+1} \quad\longrightarrow\quad \frac{x^2}{2} - \frac{1}{2}\ln(x^2+1) + C.$$

## 3. $u$-Substitution — The Chain Rule Reversed

**Signature:** the integrand is $f(g(x))\cdot g'(x)$ for some $g$.

**Rule:** Set $u = g(x)$, $du = g'(x),dx$. Then $$\int f(g(x)),g'(x),dx = \int f(u),du.$$

### 3.1 Spotting the Substitution

Ask: _"Is there a function and its derivative both present?"_

|Integrand|Substitution|
|---|---|
|$\int 2x e^{x^2}dx$|$u = x^2$, $du = 2x,dx$|
|$\int \cos x \sqrt{\sin x},dx$|$u = \sin x$, $du = \cos x,dx$|
|$\int \frac{x}{x^2+1},dx$|$u = x^2+1$, $du = 2x,dx$|
|$\int \tan x,dx$|$u = \cos x$, $du = -\sin x,dx$|
|$\int \frac{(\ln x)^2}{x},dx$|$u = \ln x$, $du = dx/x$|

### 3.2 When $g'(x)$ is "almost" there

Sometimes a constant factor is missing:

$$\int x e^{x^2}dx: \quad \text{set } u = x^2,\ du = 2x,dx \implies dx = \frac{du}{2x} \implies \int \frac{e^u}{2},du = \frac{e^{x^2}}{2} + C.$$

### 3.3 Example — Non-obvious $u$

Compute $\displaystyle I = \int \frac{e^{\sqrt{x}}}{\sqrt{x}},dx$.

Set $u = \sqrt{x}$, $du = \frac{1}{2\sqrt{x}}dx$, so $\frac{dx}{\sqrt{x}} = 2,du$: $$I = \int e^u \cdot 2,du = 2e^{\sqrt{x}} + C.$$

## 4. Integration by Parts (IBP) — The Product Rule Reversed

**Formula:** $$\int u,dv = uv - \int v,du.$$

**Derivation:** $(uv)' = u'v + uv'$, so $uv' = (uv)' - u'v$. Integrate both sides.

### 4.1 The LIATE Rule for Choosing $u$

Choose $u$ to be the factor **highest** in this list; let $dv$ be the rest:

$$\underbrace{\mathbf{L}}_{\ln, \log} \succ \underbrace{\mathbf{I}}_{\arcsin, \arctan, \ldots} \succ \underbrace{\mathbf{A}}_{\text{algebraic}} \succ \underbrace{\mathbf{T}}_{\sin, \cos, \ldots} \succ \underbrace{\mathbf{E}}_{e^x, a^x}$$

> LIATE is a heuristic, not a theorem. It fails occasionally (see §4.4), but it is right ~90% of the time.

### 4.2 Standard Examples

**Type L·A:** $\displaystyle\int x\ln x,dx$

$u = \ln x$, $dv = x,dx$, $du = \frac{dx}{x}$, $v = \frac{x^2}{2}$: $$= \frac{x^2}{2}\ln x - \int \frac{x^2}{2}\cdot\frac{1}{x},dx = \frac{x^2}{2}\ln x - \frac{x^2}{4} + C.$$

**Type A·E:** $\displaystyle\int x^2 e^x,dx$

Apply IBP twice (the polynomial degree drops each time): $$u = x^2,\ dv = e^x dx \implies x^2 e^x - 2\int x e^x,dx.$$ Apply again: $\int xe^x,dx = xe^x - e^x + C$. Total: $$\int x^2 e^x,dx = e^x(x^2 - 2x + 2) + C.$$

**Type I alone:** $\displaystyle\int \arctan x,dx$

$u = \arctan x$, $dv = dx$, $du = \frac{dx}{1+x^2}$, $v = x$: $$= x\arctan x - \int \frac{x}{1+x^2},dx = x\arctan x - \frac{1}{2}\ln(1+x^2) + C.$$

### 4.3 The Circular IBP Trick

**When IBP leads back to the original integral.** This is not failure — it is the solution.

Compute $\displaystyle I = \int e^x \sin x,dx$.

IBP once ($u = \sin x$, $dv = e^x dx$): $$I = e^x\sin x - \int e^x\cos x,dx.$$

IBP again on $\int e^x\cos x,dx$ ($u = \cos x$, $dv = e^x dx$): $$I = e^x\sin x - \left[e^x\cos x + \int e^x\sin x,dx\right] = e^x\sin x - e^x\cos x - I.$$

Solve for $I$: $$\boxed{I = \frac{e^x(\sin x - \cos x)}{2} + C.}$$

### 4.4 Tabular (Rapid) IBP

When one factor differentiates to zero in finitely many steps and the other integrates indefinitely (e.g., poly × exp), use the **tabular method**:

For $\int x^3 e^x,dx$, build the table with alternating signs:

|Sign|Differentiate $x^3$|Integrate $e^x$|
|---|---|---|
|$+$|$x^3$|$e^x$|
|$-$|$3x^2$|$e^x$|
|$+$|$6x$|$e^x$|
|$-$|$6$|$e^x$|
|$+$|$0$|$e^x$|

Answer: $e^x(x^3 - 3x^2 + 6x - 6) + C$.

## 5. Trigonometric Integrals

### 5.1 Powers of Sine and Cosine: $\int \sin^m x\cos^n x,dx$

**Case 1: $m$ odd.** Save one $\sin x$, convert the rest via $\sin^2 x = 1 - \cos^2 x$. Sub $u = \cos x$.

**Case 2: $n$ odd.** Save one $\cos x$, convert the rest via $\cos^2 x = 1 - \sin^2 x$. Sub $u = \sin x$.

**Case 3: Both even.** Use half-angle identities to reduce degrees.

#### Example: $\int \sin^3 x\cos^2 x,dx$

$m=3$ is odd. Write $\sin^3 x = \sin x(1-\cos^2 x)$. Let $u = \cos x$, $du = -\sin x,dx$: $$\int (1-u^2)u^2(-du) = -\int(u^2 - u^4),du = -\frac{u^3}{3} + \frac{u^5}{5} + C = -\frac{\cos^3 x}{3} + \frac{\cos^5 x}{5} + C.$$

### 5.2 Powers of Tangent and Secant: $\int \tan^m x\sec^n x,dx$

**Case 1: $n$ even.** Save $\sec^2 x$, convert rest via $\sec^2 x = 1+\tan^2 x$. Sub $u = \tan x$.

**Case 2: $m$ odd.** Save $\sec x\tan x$, convert rest via $\tan^2 x = \sec^2 x - 1$. Sub $u = \sec x$.

## 6. Trigonometric Substitution

**When:** The integrand contains $\sqrt{a^2 - x^2}$, $\sqrt{a^2 + x^2}$, or $\sqrt{x^2 - a^2}$.

|Form|Substitution|Range|Useful identity|
|---|---|---|---|
|$\sqrt{a^2 - x^2}$|$x = a\sin\theta$|$\theta \in [-\pi/2, \pi/2]$|$a^2 - a^2\sin^2\theta = a^2\cos^2\theta$|
|$\sqrt{a^2 + x^2}$|$x = a\tan\theta$|$\theta \in (-\pi/2, \pi/2)$|$a^2 + a^2\tan^2\theta = a^2\sec^2\theta$|
|$\sqrt{x^2 - a^2}$|$x = a\sec\theta$|$\theta \in [0, \pi/2)$|$a^2\sec^2\theta - a^2 = a^2\tan^2\theta$|

**Always draw a right triangle** to back-substitute. The triangle encodes the relationship between $x$, $\theta$, and the radical without needing to invert trig functions by memory.

#### Example: $\int \frac{dx}{x^2\sqrt{x^2+4}}$

Set $x = 2\tan\theta$, $dx = 2\sec^2\theta,d\theta$, $\sqrt{x^2+4} = 2\sec\theta$:

$$\int \frac{2\sec^2\theta,d\theta}{4\tan^2\theta\cdot 2\sec\theta} = \int \frac{\sec\theta}{4\tan^2\theta},d\theta = \frac{1}{4}\int \frac{\cos\theta}{\sin^2\theta},d\theta = -\frac{1}{4\sin\theta} + C.$$

From the triangle: $\sin\theta = \frac{x}{\sqrt{x^2+4}}$, so: $$\boxed{I = -\frac{\sqrt{x^2+4}}{4x} + C.}$$

## 7. Partial Fractions

**When:** Integrand is $\frac{P(x)}{Q(x)}$ with $\deg P < \deg Q$ (divide first otherwise).

### 7.1 Decomposition Rules

Factor $Q(x)$ over $\mathbb{R}$:

|Factor in $Q(x)$|Partial fraction terms|
|---|---|
|$(x-a)$|$\dfrac{A}{x-a}$|
|$(x-a)^k$|$\dfrac{A_1}{x-a} + \dfrac{A_2}{(x-a)^2} + \cdots + \dfrac{A_k}{(x-a)^k}$|
|$(x^2+bx+c)$ irreducible|$\dfrac{Ax+B}{x^2+bx+c}$|
|$(x^2+bx+c)^k$ irreducible|$\dfrac{A_1 x + B_1}{x^2+bx+c} + \cdots + \dfrac{A_k x + B_k}{(x^2+bx+c)^k}$|

### 7.2 Finding Coefficients

- **Cover-up (Heaviside):** Multiply both sides by $(x-a)$ and set $x = a$. Works for linear factors.
- **Compare coefficients:** Expand and match powers of $x$ on both sides.
- **Strategic substitution:** Plug in convenient values of $x$.

#### Example

$\displaystyle\int \frac{2x+3}{(x-1)(x+2)},dx$. Decompose: $$\frac{2x+3}{(x-1)(x+2)} = \frac{A}{x-1} + \frac{B}{x+2}.$$

$x=1$: $A = \frac{5}{3}$. $\quad x=-2$: $B = \frac{1}{3}$.

$$I = \frac{5}{3}\ln|x-1| + \frac{1}{3}\ln|x+2| + C.$$

## 9. Special Techniques Worth Knowing

### 9.1 Adding and Subtracting in the Numerator

$$\int \frac{x}{x^2 - 2x + 5},dx = \int \frac{\frac{1}{2}(2x-2)}{x^2-2x+5},dx + \int \frac{1}{x^2-2x+5},dx$$

The first integral is $\frac{1}{2}\ln|x^2-2x+5|$ (log form). The second, after completing the square $(x-1)^2 + 4$, is $\frac{1}{2}\arctan\frac{x-1}{2} + C$.

### 9.2 Differentiating Under the Integral Sign (Feynman's Trick)

Introduce a parameter $\alpha$ and differentiate with respect to it. Useful for integrals like:

$$I(\alpha) = \int_0^1 \frac{x^\alpha - 1}{\ln x},dx \quad\Rightarrow\quad I'(\alpha) = \int_0^1 x^\alpha,dx = \frac{1}{\alpha+1} \quad\Rightarrow\quad I(\alpha) = \ln(\alpha+1).$$

### 9.3 Symmetry of Definite Integrals

- **Odd function over symmetric interval:** $\displaystyle\int_{-a}^a f(x),dx = 0$ if $f$ is odd.
- **Even function:** $\displaystyle\int_{-a}^a f(x),dx = 2\int_0^a f(x),dx$ if $f$ is even.
- **King's Rule:** $\displaystyle\int_a^b f(x),dx = \int_a^b f(a+b-x),dx$. Often simplifies symmetric integrands dramatically.

#### Example (King's Rule)

$\displaystyle I = \int_0^{\pi} \frac{x\sin x}{1+\cos^2 x},dx$. Apply King's Rule ($x \to \pi - x$): $$I = \int_0^{\pi} \frac{(\pi-x)\sin x}{1+\cos^2 x},dx.$$ Add: $2I = \pi\int_0^\pi \frac{\sin x}{1+\cos^2 x},dx$. Let $u = \cos x$: $$2I = \pi\int_{-1}^1 \frac{du}{1+u^2} = \pi\left[\arctan u\right]_{-1}^1 = \pi\cdot\frac{\pi}{2} \implies I = \frac{\pi^2}{4}.$$

## 10. Non-Elementary Integrals

Some functions have **no antiderivative expressible in elementary functions** (polynomials, exponentials, logs, trig, their inverses, and compositions/combinations thereof). This is a theorem, not a failure.

Famous examples:

|Integral|Name|
|---|---|
|$\displaystyle\int e^{-x^2}dx$|Gaussian (related to $\operatorname{erf}(x)$)|
|$\displaystyle\int \frac{\sin x}{x},dx$|Sine integral $\operatorname{Si}(x)$|
|$\displaystyle\int \frac{e^x}{x},dx$|Exponential integral $\operatorname{Ei}(x)$|
|$\displaystyle\int \frac{1}{\ln x},dx$|Logarithmic integral $\operatorname{li}(x)$|
|$\displaystyle\int \sqrt{1-k^2\sin^2 x},dx$|Elliptic integral|

> The Gaussian integral $\int_{-\infty}^\infty e^{-x^2}dx = \sqrt{\pi}$ is evaluated using the **polar coordinate trick** in $\mathbb{R}^2$ — a 2D version of change-of-variables. This exact integral is the heartbeat of every Gaussian distribution in statistics and ML.

## 11. Connection to AI / Machine Learning

### 11.1 The Gaussian Integral is Everywhere

Every Gaussian distribution $\mathcal{N}(\mu, \sigma^2)$ normalises because: $$\int_{-\infty}^\infty e^{-x^2/2},dx = \sqrt{2\pi}.$$ This fact — and the techniques needed to evaluate related integrals (completing the square, substitution) — underpins:

- **Bayesian inference:** posterior distributions, marginalisation
- **Variational Autoencoders (VAEs):** the ELBO involves $\mathbb{E}_{q(z)}[\cdot]$
- **Kalman filters:** optimal state estimation with Gaussian noise
- **Gaussian Processes:** kernel machines for regression

### 11.2 Integration = Expectation

In machine learning, integrals appear primarily as **expectations**: $$\mathbb{E}_{x \sim p}[f(x)] = \int f(x),p(x),dx.$$

When these integrals are intractable (which is almost always), the options are:

- **Monte Carlo:** approximate $\int f,dp \approx \frac{1}{N}\sum_i f(x_i)$ where $x_i \sim p$.
- **Quadrature:** Gauss–Legendre, Gauss–Hermite (optimally choose nodes and weights for polynomial exactness).
- **Variational methods:** replace the hard distribution $p$ with a tractable family $q$, minimise $\operatorname{KL}(q|p)$.

All three approaches _implicitly use_ the integration theory developed here — quadrature uses the substitutions and exact integral formulas to calibrate, variational methods use Gaussian integral formulas to compute the ELBO analytically.

### 11.3 Backpropagation as the Chain Rule

The chain rule for derivatives is exactly what $u$-substitution reverses. In deep networks: $$\frac{\partial \mathcal{L}}{\partial w} = \frac{\partial \mathcal{L}}{\partial z}\cdot\frac{\partial z}{\partial w} \quad\longleftrightarrow\quad \int f(g(x))g'(x),dx = \int f(u),du.$$

The Jacobian that appears in normalising flows is the multivariate version of the $g'(x)$ factor in substitution — this is not a metaphor, it is the same theorem applied in $\mathbb{R}^n$.

### 11.4 Partial Fractions and System Identification

In control theory and signal processing (which underlie sequence models and recurrent networks), the **transfer function** of a linear system is a rational function $H(s) = \frac{P(s)}{Q(s)}$. Its **inverse Laplace transform** — recovering the impulse response — requires partial fractions decomposition. RNNs and state-space models are the discrete-time analogues of these continuous linear systems.

### 11.5 Numerical Integration in Training

When an exact integral is unavailable, ML relies on numerical methods:

|Numerical method|Idea|Error order|
|---|---|---|
|Riemann sum|Constant approximation per subinterval|$O(h)$|
|Trapezoid rule|Linear approximation per subinterval|$O(h^2)$|
|Simpson's rule|Quadratic approximation|$O(h^4)$|
|Gauss–Legendre|Optimal node placement for polynomials|Exponential (for smooth $f$)|

Training a model on continuous data (audio, physics simulations) often requires integrating differential equations — done via **numerical ODE solvers** (Runge–Kutta), which are high-order quadrature schemes in disguise. Neural ODEs explicitly parametrise $\frac{dz}{dt} = f_\theta(z, t)$ and integrate it forward.
