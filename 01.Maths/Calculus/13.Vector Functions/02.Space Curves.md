

> _"A curve is not a set of points — it is a map. The same set of points can be traversed slowly, quickly, backwards, or repeatedly. The parametrization carries information the image does not."_ — personal note, Stewart §13.1–13.4, MIT 18.02

---

## 0. Motivation

In the previous note ([[Vector Functions]]) we defined $\mathbf{r} : D \to \mathbb{R}^3$ and noted that the tip of $\mathbf{r}(t)$ sweeps a **space curve**. Now we take that curve seriously as a geometric object in its own right.

The central questions are:

1. **How long is the curve?** → Arc length
2. **How fast is it turning?** → Curvature $\kappa$
3. **In which direction is it turning?** → Principal normal $\mathbf{N}$
4. **Is it twisting out of a plane?** → Torsion $\tau$ and binormal $\mathbf{B}$
5. **What is the best local coordinate frame?** → Frenet–Serret frame ${\mathbf{T}, \mathbf{N}, \mathbf{B}}$

These are the tools of **differential geometry of curves**, and they reappear — sometimes verbatim, sometimes in spirit — in the geometry of loss landscapes, attention mechanisms, and manifold learning in deep learning.

---

## 1. Space Curves — Formal Definition

> **Definition 1.1 (Parametric Curve).** A _parametric curve_ in $\mathbb{R}^3$ is a continuous map $$\mathbf{r} : [a, b] \to \mathbb{R}^3, \qquad \mathbf{r}(t) = \langle f(t), g(t), h(t) \rangle$$

> **Definition 1.2 (Space Curve / Image).** The _space curve_ $C$ is the image: $$C = {\mathbf{r}(t) : t \in [a,b]} \subset \mathbb{R}^3$$

> **Definition 1.3 (Smooth Curve).** $\mathbf{r}$ is _smooth_ on $(a,b)$ if $\mathbf{r}'(t)$ exists and $\mathbf{r}'(t) \neq \mathbf{0}$ for all $t \in (a,b)$.

**Why $\mathbf{r}'(t) \neq \mathbf{0}$ matters.** At a point where $\mathbf{r}'(t_0) = \mathbf{0}$, the curve has no well-defined tangent direction — these are _singular points_ or _cusps_. Requiring $\mathbf{r}' \neq \mathbf{0}$ ensures the curve is locally "non-degenerate," which is necessary for curvature and torsion to be defined.

**Example 1.4 (a cusp).** $\mathbf{r}(t) = \langle t^2, t^3 \rangle$ has $\mathbf{r}'(0) = \langle 0, 0 \rangle$. At $t=0$ the curve has a cusp — it comes to a point and reverses direction. This is not a smooth point.

```
        y
        |      /
        |     / ← curve approaching cusp from right
        |    *  ← cusp at origin (r'(0) = 0)
        |     \
        |      \ ← curve leaving cusp downward
        +----------x
```

## 2. Reparametrization

**Definition 2.1.** A _reparametrization_ of $\mathbf{r}(t)$ is a composition $\mathbf{r}(\phi(s))$ where $\phi : [c,d] \to [a,b]$ is a smooth bijection with $\phi'(s) \neq 0$.

- If $\phi' > 0$: _orientation-preserving_ — same traversal direction.
- If $\phi' < 0$: _orientation-reversing_ — traversed backwards.

**Key insight.** The geometry of the curve (lengths, angles, curvature) must be independent of parametrization. Quantities that depend on the parametrization (e.g., speed $|\mathbf{r}'(t)|$) are _extrinsic to the curve_ but _intrinsic to the motion_.

**The canonical reparametrization** is by arc length $s$, because it gives the curve a "built-in ruler" independent of any particular traversal speed.

## 3. Arc Length (Revisited Rigorously)

### 3.1 Definition via Riemann Sums

Partition $[a,b]$ into $n$ subintervals: $a = t_0 < t_1 < \cdots < t_n = b$.

The _polygonal approximation_ has total length: $$L_n = \sum_{i=1}^n |\mathbf{r}(t_i) - \mathbf{r}(t_{i-1})|$$

**Definition 3.1 (Arc Length).** The arc length of $C$ is: $$L = \lim_{n \to \infty} L_n = \int_a^b |\mathbf{r}'(t)|,dt$$

provided this limit exists (which it does when $\mathbf{r}'$ is continuous).

_Proof of the integral formula._ By the Mean Value Theorem applied component-wise: $$|\mathbf{r}(t_i) - \mathbf{r}(t_{i-1})| = |\mathbf{r}'(t_i^_)|,\Delta t_i + O((\Delta t_i)^2)$$ for some $t_i^_ \in (t_{i-1}, t_i)$. Summing and taking the limit gives the Riemann integral. $\blacksquare$

### 3.2 The Arc Length Function

$$\boxed{s(t) = \int_a^t |\mathbf{r}'(u)|,du}$$

By the Fundamental Theorem of Calculus: $$\frac{ds}{dt} = |\mathbf{r}'(t)| \geq 0$$

This says: **speed = rate of arc length accumulation**. If $|\mathbf{r}'(t)| = 1$ for all $t$ (unit-speed parametrization), then $s(t) = t - a$.

### 3.3 Reparametrization by Arc Length

Since $\frac{ds}{dt} = |\mathbf{r}'(t)| > 0$ (smooth curve), $s(t)$ is strictly increasing, hence invertible. Write $t = t(s)$. Then:

$$\tilde{\mathbf{r}}(s) = \mathbf{r}(t(s))$$

has speed $|\tilde{\mathbf{r}}'(s)| = |\mathbf{r}'(t)| \cdot |t'(s)| = |\mathbf{r}'(t)| \cdot \frac{1}{|\mathbf{r}'(t)|} = 1$.

**$\tilde{\mathbf{r}}(s)$ is a unit-speed curve.** This is the most natural parametrization — the parameter $s$ is literally distance along the curve.

**Example 3.4 (helix reparametrized).** $\mathbf{r}(t) = \langle \cos t, \sin t, t \rangle$, $|\mathbf{r}'(t)| = \sqrt{2}$, so $s = \sqrt{2},t$, i.e., $t = s/\sqrt{2}$:

$$\tilde{\mathbf{r}}(s) = \left\langle \cos\frac{s}{\sqrt{2}},\ \sin\frac{s}{\sqrt{2}},\ \frac{s}{\sqrt{2}} \right\rangle$$

Now $s$ measures true distance along the helix.

## 4. The Frenet–Serret Frame

This is the heart of the geometry. At each smooth point of a curve we build a canonical orthonormal basis adapted to the curve's local shape.

### 4.1 Unit Tangent Vector $\mathbf{T}$

$$\boxed{\mathbf{T}(t) = \frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|}}$$

- Points in the direction of motion.
- $|\mathbf{T}| = 1$ always.
- By Lemma 6.3 from [[Vector Functions]]: $\mathbf{T} \cdot \mathbf{T}' = 0$.

### 4.2 Curvature $\kappa$

**Idea.** Curvature measures how quickly the tangent direction is turning per unit arc length. It is a property of the _curve_, not the parametrization.

> **Definition 4.1 (Curvature).** $$\kappa = \left|\frac{d\mathbf{T}}{ds}\right|$$

Since $d\mathbf{T}/ds = \mathbf{T}'(t)/|\mathbf{r}'(t)|$ (chain rule), we have:

$$\boxed{\kappa(t) = \frac{|\mathbf{T}'(t)|}{|\mathbf{r}'(t)|}}$$

**Computational formula (no need to find $\mathbf{T}$ explicitly):**

$$\boxed{\kappa = \frac{|\mathbf{r}'(t) \times \mathbf{r}''(t)|}{|\mathbf{r}'(t)|^3}}$$

_Proof._ Let $v = |\mathbf{r}'|$. Then $\mathbf{r}' = v\mathbf{T}$. Differentiate: $$\mathbf{r}'' = v'\mathbf{T} + v\mathbf{T}'$$ Take cross product with $\mathbf{r}' = v\mathbf{T}$: $$\mathbf{r}' \times \mathbf{r}'' = v\mathbf{T} \times (v'\mathbf{T} + v\mathbf{T}') = v^2(\mathbf{T} \times \mathbf{T}')$$ (since $\mathbf{T} \times \mathbf{T} = \mathbf{0}$). Then $|\mathbf{r}' \times \mathbf{r}''| = v^2|\mathbf{T} \times \mathbf{T}'| = v^2|\mathbf{T}'|$ (since $\mathbf{T} \perp \mathbf{T}'$ and $|\mathbf{T}|=1$, so $|\mathbf{T} \times \mathbf{T}'| = |\mathbf{T}'|$). Thus: $$\kappa = \frac{|\mathbf{T}'|}{v} = \frac{|\mathbf{r}' \times \mathbf{r}''|}{v^3} \qquad \blacksquare$$

**Curvature of a plane curve $y = f(x)$** (Stewart §13.3): $$\kappa = \frac{|f''(x)|}{(1 + [f'(x)]^2)^{3/2}}$$

**Special cases:**

- Straight line: $\kappa = 0$ (tangent never turns).
- Circle of radius $R$: $\kappa = 1/R$ (constant; smaller circles curve more).
- **Radius of curvature:** $\rho = 1/\kappa$ — the radius of the _osculating circle_ (the circle that best fits the curve at that point).

```
Large κ (tight curve):         Small κ (gentle curve):

      *                              *
    * * *                         *     *
   *  |  *                       *       *
      |                         *         *
   osculating circle is small   osculating circle is large
   κ = 1/R, small R → large κ
```

### 4.3 Principal Normal Vector $\mathbf{N}$

Since $\mathbf{T} \perp \mathbf{T}'$, the vector $\mathbf{T}'(t)$ is perpendicular to $\mathbf{T}(t)$. Normalize it:

$$\boxed{\mathbf{N}(t) = \frac{\mathbf{T}'(t)}{|\mathbf{T}'(t)|}}$$

- $\mathbf{N}$ points toward the _center of curvature_ (the center of the osculating circle).
- $\mathbf{T} \perp \mathbf{N}$ always.
- For a curve turning left in the $xy$-plane, $\mathbf{N}$ points left.

**Intrinsic relation:** $$\frac{d\mathbf{T}}{ds} = \kappa,\mathbf{N}$$

This is the first Frenet–Serret formula.

### 4.4 Binormal Vector $\mathbf{B}$

$$\boxed{\mathbf{B}(t) = \mathbf{T}(t) \times \mathbf{N}(t)}$$

- $|\mathbf{B}| = 1$ (since $\mathbf{T} \perp \mathbf{N}$, $|\mathbf{T}|=|\mathbf{N}|=1$).
- $\mathbf{B} \perp \mathbf{T}$, $\mathbf{B} \perp \mathbf{N}$.
- ${\mathbf{T}, \mathbf{N}, \mathbf{B}}$ is a **right-handed orthonormal frame** at every point.

**Geometric meaning.** The _osculating plane_ at a point is spanned by $\mathbf{T}$ and $\mathbf{N}$. $\mathbf{B}$ is the normal to this plane — it points "out of the plane of bending."

```
                  B (out of page)
                  ↑
                  |
         N ←------*------→ T
         (center  |  (tangent,
          of      | direction
        curvature)|  of motion)
                  |
              osculating plane = T-N plane
```

### 4.5 Torsion $\tau$

Curvature measures bending; **torsion measures twisting**.

> **Definition 4.2 (Torsion).** $$\tau = -\frac{d\mathbf{B}}{ds} \cdot \mathbf{N}$$

**Computational formula:** $$\boxed{\tau = \frac{(\mathbf{r}' \times \mathbf{r}'') \cdot \mathbf{r}'''}{|\mathbf{r}' \times \mathbf{r}''|^2}}$$

**Geometric meaning:**

- $\tau = 0$: the curve lies in a plane (a _plane curve_).
- $\tau \neq 0$: the curve spirals out of every plane — it truly lives in 3D.
- The sign of $\tau$ indicates the direction of twisting (right-handed or left-handed helix).

## 5. The Frenet–Serret Formulas

This is the fundamental theorem of space curve theory.

> **Theorem 5.1 (Frenet–Serret).** For a smooth unit-speed curve with $\kappa > 0$: $$\boxed{\frac{d\mathbf{T}}{ds} = \kappa,\mathbf{N}}$$ $$\boxed{\frac{d\mathbf{N}}{ds} = -\kappa,\mathbf{T} + \tau,\mathbf{B}}$$ $$\boxed{\frac{d\mathbf{B}}{ds} = -\tau,\mathbf{N}}$$

In matrix form: $$\frac{d}{ds}\begin{pmatrix}\mathbf{T}\\mathbf{N}\\mathbf{B}\end{pmatrix} = \begin{pmatrix}0 & \kappa & 0 \ -\kappa & 0 & \tau \ 0 & -\tau & 0\end{pmatrix}\begin{pmatrix}\mathbf{T}\\mathbf{N}\\mathbf{B}\end{pmatrix}$$

The matrix is **skew-symmetric** — this encodes that ${\mathbf{T},\mathbf{N},\mathbf{B}}$ is an orthonormal frame rotating as we move along the curve. The evolution of the frame is governed by the _Darboux vector_ $\boldsymbol{\omega} = \tau\mathbf{T} + \kappa\mathbf{B}$, so $\mathbf{T}' = \boldsymbol{\omega} \times \mathbf{T}$, $\mathbf{N}' = \boldsymbol{\omega} \times \mathbf{N}$, $\mathbf{B}' = \boldsymbol{\omega} \times \mathbf{B}$.

_Proof of $d\mathbf{T}/ds = \kappa\mathbf{N}$._ By definition of $\kappa$ and $\mathbf{N}$: $$\frac{d\mathbf{T}}{ds} = \left|\frac{d\mathbf{T}}{ds}\right| \cdot \frac{d\mathbf{T}/ds}{|d\mathbf{T}/ds|} = \kappa,\mathbf{N} \qquad \blacksquare$$

_Proof of $d\mathbf{B}/ds = -\tau\mathbf{N}$._ Since $\mathbf{B} = \mathbf{T} \times \mathbf{N}$: $$\frac{d\mathbf{B}}{ds} = \frac{d\mathbf{T}}{ds} \times \mathbf{N} + \mathbf{T} \times \frac{d\mathbf{N}}{ds} = \kappa\mathbf{N} \times \mathbf{N} + \mathbf{T} \times \frac{d\mathbf{N}}{ds} = \mathbf{T} \times \frac{d\mathbf{N}}{ds}$$ Since $|\mathbf{B}|=1$, we have $d\mathbf{B}/ds \perp \mathbf{B}$. Also $d\mathbf{B}/ds \perp \mathbf{T}$ (shown by differentiating $\mathbf{T}\cdot\mathbf{B}=0$). Hence $d\mathbf{B}/ds \parallel \mathbf{N}$, so $d\mathbf{B}/ds = -\tau\mathbf{N}$ for some scalar $\tau$. $\blacksquare$

**The Fundamental Theorem of Curve Theory.** Given smooth functions $\kappa(s) > 0$ and $\tau(s)$, there exists a unique (up to rigid motion) smooth curve in $\mathbb{R}^3$ with those curvature and torsion. $\kappa$ and $\tau$ are the _complete invariants_ of the curve.

## 6. Worked Examples

### 6.1 The Circular Helix — Full Analysis

$$\mathbf{r}(t) = \langle a\cos t,\ a\sin t,\ bt \rangle, \quad a > 0, b \neq 0$$

**Derivatives:** $$\mathbf{r}'(t) = \langle -a\sin t,\ a\cos t,\ b \rangle, \qquad |\mathbf{r}'| = \sqrt{a^2 + b^2} =: c$$ $$\mathbf{r}''(t) = \langle -a\cos t,\ -a\sin t,\ 0 \rangle$$ $$\mathbf{r}'''(t) = \langle a\sin t,\ -a\cos t,\ 0 \rangle$$

**Unit tangent:** $$\mathbf{T}(t) = \frac{1}{c}\langle -a\sin t, a\cos t, b \rangle$$

**Curvature:** $$\mathbf{r}' \times \mathbf{r}'' = \langle ab\sin t,\ -ab\cos t,\ a^2 \rangle$$ $$|\mathbf{r}' \times \mathbf{r}''| = \sqrt{a^2b^2 + a^4} = a\sqrt{b^2 + a^2} = ac$$ $$\boxed{\kappa = \frac{ac}{c^3} = \frac{a}{c^2} = \frac{a}{a^2 + b^2}}$$

Constant! The helix has **constant curvature**.

**Torsion:** $$(\mathbf{r}' \times \mathbf{r}'') \cdot \mathbf{r}''' = \langle ab\sin t, -ab\cos t, a^2 \rangle \cdot \langle a\sin t, -a\cos t, 0 \rangle$$ $$= a^2b\sin^2 t + a^2b\cos^2 t = a^2b$$ $$\boxed{\tau = \frac{a^2b}{(ac)^2} = \frac{b}{a^2+b^2}}$$

Constant! The helix has **constant torsion**.

**Interpretation:**

- If $b > 0$: right-handed helix ($\tau > 0$), rises counterclockwise.
- If $b < 0$: left-handed helix ($\tau < 0$).
- If $b = 0$: reduces to a circle, $\kappa = 1/a$, $\tau = 0$.
- If $a = 0$: reduces to a straight line, $\kappa = 0$.

**The helix is characterized (up to rigid motion) by having constant $\kappa$ and $\tau$.**

### 6.2 A Plane Curve

$$\mathbf{r}(t) = \langle t, t^2, 0 \rangle$$

This lies in the $z=0$ plane, so $\tau = 0$ (verify with the formula as an exercise).

$$\mathbf{r}' = \langle 1, 2t, 0\rangle, \quad \mathbf{r}'' = \langle 0, 2, 0\rangle$$ $$\mathbf{r}' \times \mathbf{r}'' = \langle 0, 0, 2\rangle, \quad |\mathbf{r}' \times \mathbf{r}''| = 2$$ $$\kappa(t) = \frac{2}{(1+4t^2)^{3/2}}$$

At $t=0$: $\kappa = 2$ (maximum curvature, tightest turn at the vertex of the parabola). As $t \to \pm\infty$: $\kappa \to 0$ (the parabola straightens out).

```
     y
     |
     *     *        ← curvature decreasing away from vertex
     |  *     *
     | *       *
     |*_________*___ x
   vertex (κ = 2)
```

## 7. Normal and Osculating Planes

**Definition 7.1 (Osculating Plane).** The _osculating plane_ at $\mathbf{r}(t_0)$ is the plane spanned by $\mathbf{T}(t_0)$ and $\mathbf{N}(t_0)$, passing through $\mathbf{r}(t_0)$. Normal to this plane: $\mathbf{B}(t_0)$.

Equation: $\mathbf{B}(t_0) \cdot (\mathbf{x} - \mathbf{r}(t_0)) = 0$.

**Definition 7.2 (Normal Plane).** The _normal plane_ at $\mathbf{r}(t_0)$ is perpendicular to $\mathbf{T}(t_0)$, containing $\mathbf{N}$ and $\mathbf{B}$.

Equation: $\mathbf{T}(t_0) \cdot (\mathbf{x} - \mathbf{r}(t_0)) = 0$.

**Definition 7.3 (Rectifying Plane).** Spanned by $\mathbf{T}$ and $\mathbf{B}$; contains the tangent line and the binormal.

```
            B
            ↑
            |  /T
   N ←------*------
            |osculating plane = span(T,N)
            | normal plane = span(N,B)
```

## 8. Motion in Space — Physics Interpretation

If $\mathbf{r}(t)$ is the position of a particle:

$$\text{velocity: } \mathbf{v}(t) = \mathbf{r}'(t) = v,\mathbf{T}(t), \quad v = |\mathbf{v}| = |\mathbf{r}'|$$

$$\text{acceleration: } \mathbf{a}(t) = \mathbf{r}''(t) = v'\mathbf{T} + v\mathbf{T}' = v'\mathbf{T} + v\frac{d\mathbf{T}}{ds}\frac{ds}{dt} = v'\mathbf{T} + \kappa v^2,\mathbf{N}$$

$$\boxed{\mathbf{a} = \underbrace{v'}_{\text{tangential}}\mathbf{T} + \underbrace{\kappa v^2}_{\text{centripetal}}\mathbf{N}}$$

The acceleration decomposes into:

- **Tangential component** $a_T = v' = \frac{d^2s}{dt^2}$: changes the speed.
- **Normal component** $a_N = \kappa v^2$: changes the direction (centripetal — always toward the center of curvature).

**Component formulas (no need to find $\mathbf{T}$ and $\mathbf{N}$):** $$a_T = \frac{\mathbf{r}' \cdot \mathbf{r}''}{|\mathbf{r}'|}, \qquad a_N = \frac{|\mathbf{r}' \times \mathbf{r}''|}{|\mathbf{r}'|}$$

_Verify: $a_N = \kappa v^2 = \frac{|\mathbf{r}'\times\mathbf{r}''|}{|\mathbf{r}'|^3} \cdot |\mathbf{r}'|^2 = \frac{|\mathbf{r}'\times\mathbf{r}''|}{|\mathbf{r}'|}$. ✓_

**Example 8.1.** For the helix $\mathbf{r}(t) = \langle \cos t, \sin t, t \rangle$: $$v = |\mathbf{r}'| = \sqrt{2}, \quad v' = 0 \quad (\text{constant speed})$$ $$\mathbf{a}(t) = \langle -\cos t, -\sin t, 0\rangle$$ $$a_T = 0, \quad a_N = \kappa v^2 = \frac{1}{2} \cdot 2 = 1$$

The particle moves at constant speed on the helix — all acceleration is centripetal.

## 10. Connection to AI / Machine Learning / Deep Learning

### 10.1 Loss Landscape as a Surface; Optimizer Trajectory as a Space Curve

The training trajectory $\boldsymbol{\theta}(t) \in \mathbb{R}^p$ is a space curve in parameter space (very high-dimensional, but the geometry applies). The **curvature** $\kappa$ of this trajectory tells us:

- High $\kappa$: the optimizer is turning sharply — it has found a direction of high curvature in the loss landscape, indicating a _saddle point_ or _sharp minimum_.
- Low $\kappa$: the optimizer is moving in a nearly straight line — the gradient is consistent, loss is decreasing steadily.

The **Hessian** $H = \nabla^2 \mathcal{L}$ encodes the curvature of the loss surface, and second-order optimizers (Newton's method, natural gradient) use this curvature to correct the trajectory — exactly as the Frenet frame uses $\kappa$ and $\tau$ to describe the curve's geometry.

### 10.2 Frenet–Serret in Sequence Models

In a sequence model (RNN, Transformer), the hidden state $\mathbf{h}(t)$ traces a curve in $\mathbb{R}^d$. The Frenet frame ${\mathbf{T}, \mathbf{N}, \mathbf{B}}$ along this trajectory gives:

- $\mathbf{T}(t)$: the direction of change in the hidden state — what the model is "attending to" at step $t$.
- $\kappa(t)$: how sharply the model is changing direction in state space — high curvature at semantic boundaries (e.g., transitions between topics in text).
- $\tau(t)$: how much the model's plane of attention is rotating — high torsion may indicate complex, multi-dimensional context shifts.

This perspective is exploited in _geometric deep learning_ (Bronstein et al., 2021) and in interpretability research that studies the geometry of activation trajectories.

### 10.3 Arc Length as Complexity / Path Length Regularization

In ML, the _path length_ of the optimizer trajectory: $$L = \int_0^T |\boldsymbol{\theta}'(t)|,dt$$ measures the total distance traveled in parameter space. Flat minima (low sharpness) correspond to regions where the trajectory length is short before convergence. **Sharpness-aware minimization (SAM)** explicitly penalizes trajectories that travel into high-curvature regions of the loss landscape — it is, in spirit, a curvature-based regularizer.

### 10.4 The Osculating Circle and Attention Radius

The _osculating circle_ at a point has radius $\rho = 1/\kappa$ and is the best local circular approximation to the curve. In attention mechanisms, the **attention radius** (how far back in the sequence the model attends) is analogous: at points of high curvature (sharp transitions), the model must attend locally; at low curvature (slow drift), the model can attend over longer ranges.

### 10.5 Curvature of Decision Boundaries

For a binary classifier with decision boundary ${f(\mathbf{x}) = 0}$, the curvature $\kappa$ of this boundary (as a curve in $\mathbb{R}^2$ or a surface in $\mathbb{R}^n$) controls:

- **Generalization**: highly curved boundaries tend to overfit.
- **Adversarial robustness**: inputs near high-curvature regions of the boundary are vulnerable to small perturbations that cross it.

Regularization methods like weight decay implicitly reduce the curvature of decision boundaries — a direct differential-geometric interpretation.

### 10.6 Geodesics on Data Manifolds

Real data (images, audio, text) lies on a low-dimensional _manifold_ embedded in high-dimensional space. The _geodesic_ between two points on this manifold is the shortest path along the manifold — the analogue of a straight line in curved space. Computing geodesics uses exactly the machinery of arc length and curvature developed here:

$$\text{geodesic length} = \int_a^b \sqrt{g_{ij},\dot{x}^i \dot{x}^j},dt$$

where $g_{ij}$ is the Riemannian metric on the manifold. This is the foundation of _Riemannian manifold optimization_, used in training on curved parameter spaces (e.g., hyperbolic embeddings for hierarchical data).

|Quantity|Formula|Meaning|
|---|---|---|
|Arc length|$L = \int_a^b \|\mathbf{r}'\|,dt$|Total distance along curve|
|Arc length fn|$s(t) = \int_a^t \|\mathbf{r}'\|,du$|Distance from start to $t$|
|Unit tangent|$\mathbf{T} = \mathbf{r}'/\|\mathbf{r}'\|$|Direction of motion|
|Curvature|$\kappa = \|\mathbf{T}'\|/\|\mathbf{r}'\|$|Rate of turning (per arc length)|
|Curvature (alt)|$\kappa = \|\mathbf{r}' \times \mathbf{r}''\|/\|\mathbf{r}'\|^3$|Computational formula|
|Principal normal|$\mathbf{N} = \mathbf{T}'/\|\mathbf{T}'\|$|Points toward center of curvature|
|Binormal|$\mathbf{B} = \mathbf{T} \times \mathbf{N}$|Normal to osculating plane|
|Torsion|$\tau = (\mathbf{r}'\times\mathbf{r}'')\cdot\mathbf{r}'''/\|\mathbf{r}'\times\mathbf{r}''\|^2$|Rate of twisting|

|Osculating circle|radius $\rho = 1/\kappa$|Best local circle approximation|

**Frenet–Serret at a glance:** $$\mathbf{T}' = \kappa v,\mathbf{N}, \qquad \mathbf{N}' = v(-\kappa\mathbf{T} + \tau\mathbf{B}), \qquad \mathbf{B}' = -\tau v,\mathbf{N}$$ (where derivatives are with respect to $t$, so chain rule gives factors of $v = |\mathbf{r}'|$).
