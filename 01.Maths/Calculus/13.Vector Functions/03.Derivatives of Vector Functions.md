

> _"The derivative of a vector function is itself a vector function — direction and magnitude both change, and calculus tracks both simultaneously."_

---

## 0. Motivation & Setup

In single-variable calculus we differentiated $f : \mathbb{R} \to \mathbb{R}$. Now we step up: let the **output** itself live in $\mathbb{R}^n$. A **vector-valued function** is a map

$$\mathbf{r} : I \subseteq \mathbb{R} ;\longrightarrow; \mathbb{R}^n, \qquad t \mapsto \mathbf{r}(t)$$

In $\mathbb{R}^3$ this is written component-wise as

$$\mathbf{r}(t) = \langle, f(t),; g(t),; h(t),\rangle = f(t),\mathbf{i} + g(t),\mathbf{j} + h(t),\mathbf{k}$$

Each component $f, g, h$ is an ordinary real-valued function. This is the key observation: **vector calculus decomposes into scalar calculus, component by component.**

> **Geometric picture.** As $t$ varies, the tip of $\mathbf{r}(t)$ traces a _space curve_ $C$. The derivative will give the tangent vector to this curve — a fundamental object in geometry and physics.

---

## 1. The Derivative — Definition

### 1.1 Limit Definition

$$\boxed{\mathbf{r}'(t) = \lim_{h \to 0} \frac{\mathbf{r}(t+h) - \mathbf{r}(t)}{h}}$$

**Unpacking this:** $\mathbf{r}(t+h) - \mathbf{r}(t)$ is a **vector** (a displacement along the curve). We scale it by the scalar $1/h$. In the limit, this secant vector rotates and shrinks into the **tangent vector** at $\mathbf{r}(t)$.

> **Note (Stewart §13.2).** The limit is taken in $\mathbb{R}^n$: a vector limit converges iff every component converges.

### 1.2 The Component-Wise Theorem

**Theorem.** If $\mathbf{r}(t) = \langle f(t), g(t), h(t)\rangle$ and $f, g, h$ are differentiable, then

$$\mathbf{r}'(t) = \langle f'(t),; g'(t),; h'(t)\rangle$$

**Proof.**

$$\mathbf{r}'(t) = \lim_{h\to 0}\frac{\langle f(t+h), g(t+h), k(t+h)\rangle - \langle f(t), g(t), h(t)\rangle}{h}$$

$$= \lim_{h\to 0}\left\langle \frac{f(t+h)-f(t)}{h},;\frac{g(t+h)-g(t)}{h},;\frac{h(t+h)-h(t)}{h}\right\rangle$$

Since limit distributes over finite vector components (as each component limit exists by hypothesis):

$$= \langle f'(t),; g'(t),; h'(t)\rangle \qquad \square$$

> This theorem is the workhorse. All computation reduces to differentiating three scalar functions.

## 2. Geometric Interpretation

Let $P = \mathbf{r}(t_0)$ be a point on the curve $C$.

- $\mathbf{r}(t_0 + h) - \mathbf{r}(t_0)$ is a **secant vector** from $P$ toward a nearby point.
- As $h \to 0$, this secant rotates into the **tangent direction**.
- $\mathbf{r}'(t_0)$, if nonzero, is the **tangent vector** to $C$ at $P$.

### Unit Tangent Vector

$$\mathbf{T}(t) = \frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|}$$

This strips out the parametrization speed and retains only direction — crucial for Frenet–Serret theory.

### Visualization

```
           r(t+h)
          /
         / ← secant vector r(t+h) - r(t)
        /
  r(t) •————————→  r'(t)  ← tangent (limit as h→0)
        \
         \___  curve C
```

As $h \to 0$: the chord $\overrightarrow{r(t),r(t+h)}$ becomes tangent to the curve at $r(t)$.

## 3. Differentiation Rules

Let $\mathbf{u}(t), \mathbf{v}(t)$ be differentiable vector functions, $f(t)$ a differentiable scalar, $\mathbf{c}$ a constant vector.

|Rule|Formula|
|---|---|
|Constant|$\dfrac{d}{dt}\mathbf{c} = \mathbf{0}$|
|Linearity|$(\mathbf{u} + \mathbf{v})' = \mathbf{u}' + \mathbf{v}'$|
|Scalar multiple|$(f,\mathbf{u})' = f',\mathbf{u} + f,\mathbf{u}'$|
|Dot product|$(\mathbf{u}\cdot\mathbf{v})' = \mathbf{u}'\cdot\mathbf{v} + \mathbf{u}\cdot\mathbf{v}'$|
|Cross product|$(\mathbf{u}\times\mathbf{v})' = \mathbf{u}'\times\mathbf{v} + \mathbf{u}\times\mathbf{v}'$ _(order matters!)_|
|Chain rule|$\dfrac{d}{dt}\mathbf{u}(f(t)) = f'(t),\mathbf{u}'(f(t))$|

**Proof of the dot product rule** (representative):

$$(\mathbf{u}\cdot\mathbf{v})' = (u_1 v_1 + u_2 v_2 + u_3 v_3)' = u_1'v_1 + u_1 v_1' + \cdots = \mathbf{u}'\cdot\mathbf{v} + \mathbf{u}\cdot\mathbf{v}' \qquad \square$$

### Key Consequence — Constant Length Vectors

**Proposition.** $|\mathbf{r}(t)| = c$ (constant) $\iff$ $\mathbf{r}(t) \perp \mathbf{r}'(t)$ for all $t$.

**Proof.** $$|\mathbf{r}|^2 = c^2 \implies \frac{d}{dt}(\mathbf{r}\cdot\mathbf{r}) = 0 \implies 2,\mathbf{r}\cdot\mathbf{r}' = 0 \implies \mathbf{r}\perp\mathbf{r}' \qquad \square$$

> **Physical reading:** If a particle moves on a sphere of fixed radius, its velocity is always perpendicular to its position vector. Uniform circular motion is the classical case.

## 4. Worked Examples

### Example 1 — Helix

$$\mathbf{r}(t) = \langle \cos t,; \sin t,; t \rangle$$

**Derivative:** $$\mathbf{r}'(t) = \langle -\sin t,; \cos t,; 1 \rangle$$

**Speed** (magnitude of velocity): $$|\mathbf{r}'(t)| = \sqrt{\sin^2 t + \cos^2 t + 1} = \sqrt{2}$$

The helix has **constant speed** — it winds around the $z$-axis at a uniform rate while rising at rate 1.

**Unit tangent:** $$\mathbf{T}(t) = \frac{1}{\sqrt{2}}\langle -\sin t,; \cos t,; 1\rangle$$

```
z
│     ↗ T(t)  ← tangent vector
│    /
│   •←—— r(t) traces the helix
│  /
│ /
└────────── x,y plane
```

---

### Example 2 — Verifying Perpendicularity

Let $\mathbf{r}(t) = \langle \cos t, \sin t, 0\rangle$ (unit circle in $xy$-plane).

$$|\mathbf{r}(t)| = 1 \quad \text{(constant)}$$ $$\mathbf{r}'(t) = \langle -\sin t, \cos t, 0\rangle$$ $$\mathbf{r}(t)\cdot\mathbf{r}'(t) = -\cos t \sin t + \sin t \cos t = 0 \checkmark$$

---

### Example 3 — Cross Product Rule

Let $\mathbf{u}(t) = \langle t, t^2, 0\rangle$, $\mathbf{v}(t) = \langle 1, 0, t\rangle$.

$$\mathbf{u}'(t) = \langle 1, 2t, 0\rangle, \qquad \mathbf{v}'(t) = \langle 0, 0, 1\rangle$$

$$(\mathbf{u}\times\mathbf{v})' = \mathbf{u}'\times\mathbf{v} + \mathbf{u}\times\mathbf{v}'$$

$$= \langle 1,2t,0\rangle \times \langle 1,0,t\rangle + \langle t,t^2,0\rangle\times\langle 0,0,1\rangle$$

Computing each cross product via the determinant formula:

$$\langle 1,2t,0\rangle\times\langle 1,0,t\rangle = \langle 2t^2, -t, -2t\rangle$$

$$\langle t,t^2,0\rangle\times\langle 0,0,1\rangle = \langle t^2, -t, 0\rangle$$

$$\therefore \quad (\mathbf{u}\times\mathbf{v})' = \langle 2t^2+t^2,; -t-t,; -2t\rangle = \langle 3t^2, -2t, -2t\rangle$$

_(Always verify by computing $\mathbf{u}\times\mathbf{v}$ directly and differentiating — a useful exercise.)_

## 5. Higher Derivatives

Just as in scalar calculus, we iterate:

$$\mathbf{r}''(t) = \langle f''(t), g''(t), h''(t)\rangle$$

**Physics interpretation:**

- $\mathbf{r}(t)$ = position
- $\mathbf{r}'(t) = \mathbf{v}(t)$ = velocity
- $\mathbf{r}''(t) = \mathbf{a}(t)$ = acceleration

Newton's second law: $\mathbf{F} = m,\mathbf{r}''(t)$.

---

## 6. Arc Length & Parametrization

The **arc length** from $t = a$ to $t = b$:

$$L = \int_a^b |\mathbf{r}'(t)|,dt = \int_a^b\sqrt{[f'(t)]^2 + [g'(t)]^2 + [h'(t)]^2},dt$$

The **arc length function**:

$$s(t) = \int_a^t |\mathbf{r}'(u)|,du \implies \frac{ds}{dt} = |\mathbf{r}'(t)|$$

A curve is **parametrized by arc length** when $|\mathbf{r}'(s)| = 1$ for all $s$ — the "natural" parametrization, since one unit of parameter = one unit of distance traveled.

## 7. Connection to AI / Machine Learning / Deep Learning

> This is not a tangential remark. The mathematics here is _foundational infrastructure_ for modern ML.

### 7.1 Gradient as a Vector Function

In ML, a model has parameters $\boldsymbol{\theta} \in \mathbb{R}^n$. The loss $\mathcal{L}(\boldsymbol{\theta})$ is a scalar. Training is the process of following

$$\boldsymbol{\theta}(t+1) = \boldsymbol{\theta}(t) - \eta,\nabla_{\boldsymbol{\theta}}\mathcal{L}$$

where $\nabla_{\boldsymbol{\theta}}\mathcal{L} \in \mathbb{R}^n$ is a **vector** of partial derivatives — a vector-valued function of $\boldsymbol{\theta}$.

Gradient descent traces a **trajectory** $\boldsymbol{\theta}(t)$ in parameter space. Its tangent vector at each step is $-\nabla\mathcal{L}$. The entire theory of §1–§3 applies here.

### 7.2 Neural ODE — The Continuous Limit

A deep residual network (ResNet) computes:

$$\mathbf{h}_{t+1} = \mathbf{h}_t + F(\mathbf{h}_t, \boldsymbol{\theta}_t)$$

Taking the continuous limit (Chen et al., NeurIPS 2018 — _Neural ODEs_):

$$\frac{d\mathbf{h}}{dt} = F(\mathbf{h}(t), \boldsymbol{\theta}(t))$$

This is a **vector ODE** — $\mathbf{h}(t) \in \mathbb{R}^d$ is a vector-valued function, and its derivative is exactly $\mathbf{r}'(t)$ from our theory. The hidden state evolves as a _space curve_ in representation space.

### 7.3 Recurrent Networks & Sequence Models

In an RNN, the hidden state $\mathbf{h}_t$ is updated step by step. The "derivative" of $\mathbf{h}$ with respect to time encodes how the representation changes — instabilities in this (vanishing/exploding gradients) are precisely failures of the sequence $\mathbf{h}'(t)$ to behave well. LSTM and Transformer architectures are engineering solutions to control this derivative.

### 7.4 Backpropagation as Chain Rule

In deep learning, the chain rule for vector functions:

$$\frac{d}{dt}\mathbf{u}(f(t)) = f'(t),\mathbf{u}'(f(t))$$

generalizes to the **Jacobian chain rule** in $\mathbb{R}^n$. Backpropagation is exactly this rule applied layer by layer. Every backward pass through a neural network is a computation of $\mathbf{r}'(t)$ for a composition of vector functions.

### 7.5 Physics-Informed Neural Networks (PINNs)

PINNs train networks to satisfy physical laws of the form $\mathbf{r}''(t) = \mathbf{F}(\mathbf{r}, \mathbf{r}', t)$. The loss function penalizes violations of the ODE, and the network must learn a trajectory whose **vector derivative** satisfies the physics.

> **Summary table:**

|Classical Concept|ML / DL Analogue|
|---|---|
|Space curve $\mathbf{r}(t)$|Parameter trajectory $\boldsymbol{\theta}(t)$ during training|
|Tangent vector $\mathbf{r}'(t)$|Negative gradient $-\nabla\mathcal{L}$ (GD direction)|
|Chain rule for vector functions|Backpropagation|
|Vector ODE $\mathbf{r}'' = \mathbf{F}$|Neural ODE / ResNet limit|
|Arc length|Optimization path length; relates to convergence analysis|
|Unit tangent $\mathbf{T}(t)$|Normalized gradient (in gradient normalization methods)|

## 8. Summary of Key Facts

$$\mathbf{r}'(t) = \langle f'(t), g'(t), h'(t)\rangle \qquad \text{[component-wise differentiation]}$$

$$|\mathbf{r}| = c \iff \mathbf{r}\cdot\mathbf{r}' = 0 \qquad \text{[constant length $\Rightarrow$ orthogonality]}$$

$$L = \int_a^b|\mathbf{r}'(t)|,dt \qquad \text{[arc length]}$$

$$\mathbf{T}(t) = \frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|} \qquad \text{[unit tangent]}$$
