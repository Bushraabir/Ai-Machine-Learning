

> _"A vector function is simply a machine: feed it a number, and it hands back an arrow."_ — personal note, inspired by Stewart §13.1 & MIT 18.02

## 0. Motivation — Why Not Just Use Scalars?

In single-variable calculus we tracked **one number** evolving over time. But the real world moves in space. A particle doesn't just have a speed — it has a **position**, a **velocity**, and an **acceleration**, all of which are arrows pointing somewhere in $\mathbb{R}^3$.

Vector functions are the language for this. And, as we will see later, they are the hidden backbone of neural networks, where every forward pass is literally a composition of vector-valued maps.

---

## 1. Definition

> **Definition 1.1 (Vector Function / Vector-Valued Function).** A _vector function_ (or _vector-valued function_) is a map $$\mathbf{r} : D \subseteq \mathbb{R} \longrightarrow \mathbb{R}^n$$ where $D$ is the _domain_ (a subset of real numbers) and $n \geq 1$.

For $n = 3$ — the most common case in Stewart and MIT 18.02 — we write:

$$\boxed{\mathbf{r}(t) = \langle f(t),, g(t),, h(t) \rangle = f(t),\mathbf{i} + g(t),\mathbf{j} + h(t),\mathbf{k}}$$

where $f, g, h : D \to \mathbb{R}$ are the **component functions**.

**Remark.** The two notations $\langle f, g, h \rangle$ and $f\mathbf{i} + g\mathbf{j} + h\mathbf{k}$ are identical. I prefer angle-bracket notation — it is cleaner and avoids confusion between the unit vector $\mathbf{k}$ and a generic index.

---

## 2. Domain

The natural domain of $\mathbf{r}(t) = \langle f(t), g(t), h(t) \rangle$ is:

$$D = \mathrm{dom}(f) \cap \mathrm{dom}(g) \cap \mathrm{dom}(h)$$

**Example 2.1.** Find the domain of $\mathbf{r}(t) = \langle \sqrt{t-1},\ \ln(3-t),\ t^2 \rangle$.

- $f(t) = \sqrt{t-1}$ requires $t \geq 1$.
- $g(t) = \ln(3-t)$ requires $t < 3$.
- $h(t) = t^2$ is defined everywhere.

$$D = [1, 3)$$

_This is exactly how you restrict inputs in a machine learning pipeline: each feature/component imposes its own constraint, and the valid input set is the intersection._

---

## 3. The Space Curve

As $t$ varies over $D$, the tip of $\mathbf{r}(t)$ traces out a **space curve** $C$ in $\mathbb{R}^3$.

```
          z
          |
          |       • r(t₂)
          |      /
          |   • r(t₁)
          |  /
          | /   C (the curve)
          |/_________________ y
         /
        x
```

The curve $C$ is the **image** (range) of $\mathbf{r}$, not the function itself. The function carries the **parametrization** — the speed and direction of traversal. Two different vector functions can trace the same curve at different speeds. This distinction matters enormously in physics and in optimization (gradient descent traces a curve in parameter space — the parametrization is what the learning rate controls).

## 4. Concrete Examples

### 4.1 The Helix

$$\mathbf{r}(t) = \langle \cos t,\ \sin t,\ t \rangle, \quad t \in [0, 4\pi]$$

- $x(t) = \cos t,\ y(t) = \sin t$ traces a circle: $x^2 + y^2 = 1$.
- $z(t) = t$ climbs linearly.
- Together: a **circular helix** — like a coil spring or a DNA strand.

**Visualization (top view vs side view):**

```
Top view (xy-plane):       Side view (xz-plane):

       y                         z
       |   * *                   |        *
       |  *   *              4π  |       *
       | *     *                 |      *
       |*       *            2π  |     *
       +----------x              |    *
        (circle)              0  |___*_______ x
```

### 4.2 A Line in $\mathbb{R}^3$

$$\mathbf{r}(t) = \langle 1,2,3 \rangle + t\langle 4,-1,2 \rangle = \langle 1+4t,\ 2-t,\ 3+2t \rangle$$

This is the line through $(1,2,3)$ in direction $(4,-1,2)$. Every line in $\mathbb{R}^3$ can be written this way.

**Connection to ML:** In linear regression, the prediction $\hat{y} = \mathbf{w}^\top \mathbf{x} + b$ defines a hyperplane. The weight vector $\mathbf{w}$ is literally the direction vector of a linear map on input space. The parametric line is the 1D analogue.

### 4.3 Lissajous Curve (2D)

$$\mathbf{r}(t) = \langle \sin(2t),\ \sin(3t) \rangle, \quad t \in [0, 2\pi]$$

The component frequencies being incommensurate ($2:3$) produces a curve that almost-but-not-quite closes. This is used in signal processing and appears in the analysis of recurrent networks (the hidden state traces a curve in state space whose topology encodes memory).

## 5. Limits and Continuity

**Definition 5.1 (Limit).** Let $\mathbf{r}(t) = \langle f(t), g(t), h(t) \rangle$. Then:

$$\lim_{t \to a} \mathbf{r}(t) = \left\langle \lim_{t \to a} f(t),\ \lim_{t \to a} g(t),\ \lim_{t \to a} h(t) \right\rangle$$

provided each component limit exists.

**Definition 5.2 (Continuity).** $\mathbf{r}$ is continuous at $t = a$ if:

$$\lim_{t \to a} \mathbf{r}(t) = \mathbf{r}(a)$$

This reduces, _component by component_, to continuity of $f$, $g$, and $h$ at $a$.

**Theorem 5.3.** $\mathbf{r}$ is continuous at $a$ $\iff$ $f$, $g$, and $h$ are each continuous at $a$.

_Proof._ Follows immediately from the component-wise definition of the limit and the equivalence of the Euclidean norm with the $\ell^\infty$ norm in finite dimensions. $\blacksquare$

**Remark (AI context).** Activation functions in neural networks are required to be continuous (and usually differentiable) precisely so that the output vector $\mathbf{r}(t)$ — interpreting $t$ as a parameter controlling the input — varies smoothly. ReLU is continuous but not differentiable at 0, which is why subdifferentials are needed in backprop.

## 6. The Derivative

### 6.1 Definition from First Principles

$$\mathbf{r}'(t) = \frac{d\mathbf{r}}{dt} = \lim_{\Delta t \to 0} \frac{\mathbf{r}(t + \Delta t) - \mathbf{r}(t)}{\Delta t}$$

Geometrically: $\mathbf{r}(t+\Delta t) - \mathbf{r}(t)$ is a **secant vector**. As $\Delta t \to 0$, this secant vector (after scaling) approaches the **tangent vector** to the curve at $\mathbf{r}(t)$.

```
                   r(t + Δt)
                  *
                 /↑  ← secant vector
                / |
    r(t) →    *  |
              |   (as Δt → 0, secant → tangent)
              ↓ tangent r'(t)
```

### 6.2 Computation Rule

**Theorem 6.1.** If $\mathbf{r}(t) = \langle f(t), g(t), h(t) \rangle$ and $f, g, h$ are differentiable, then:

$$\mathbf{r}'(t) = \langle f'(t),\ g'(t),\ h'(t) \rangle$$

_Proof._ By the component-wise definition of the limit: $$\lim_{\Delta t \to 0} \frac{\mathbf{r}(t+\Delta t) - \mathbf{r}(t)}{\Delta t} = \lim_{\Delta t \to 0} \left\langle \frac{f(t+\Delta t)-f(t)}{\Delta t},\ \frac{g(t+\Delta t)-g(t)}{\Delta t},\ \frac{h(t+\Delta t)-h(t)}{\Delta t} \right\rangle$$ $$= \langle f'(t),\ g'(t),\ h'(t) \rangle \qquad \blacksquare$$

**Example 6.2.** Differentiate $\mathbf{r}(t) = \langle t^3,\ e^{2t},\ \sin t \rangle$:

$$\mathbf{r}'(t) = \langle 3t^2,\ 2e^{2t},\ \cos t \rangle$$

### 6.3 Differentiation Rules

Let $\mathbf{u}(t)$ and $\mathbf{v}(t)$ be differentiable vector functions, $f(t)$ a scalar function.

|Rule|Formula|
|---|---|
|Sum|$\frac{d}{dt}[\mathbf{u} + \mathbf{v}] = \mathbf{u}' + \mathbf{v}'$|
|Scalar multiple|$\frac{d}{dt}[f,\mathbf{u}] = f'\mathbf{u} + f\mathbf{u}'$|
|Dot product|$\frac{d}{dt}[\mathbf{u} \cdot \mathbf{v}] = \mathbf{u}' \cdot \mathbf{v} + \mathbf{u} \cdot \mathbf{v}'$|
|Cross product|$\frac{d}{dt}[\mathbf{u} \times \mathbf{v}] = \mathbf{u}' \times \mathbf{v} + \mathbf{u} \times \mathbf{v}'$|
|Chain rule|$\frac{d}{dt}[\mathbf{u}(f(t))] = f'(t),\mathbf{u}'(f(t))$|

**Warning (cross product).** Order matters: $\mathbf{u}' \times \mathbf{v} + \mathbf{u} \times \mathbf{v}'$, not the reverse. The cross product is anticommutative.

**Key lemma — constant length implies orthogonal derivative:**

**Lemma 6.3.** If $|\mathbf{r}(t)| = c$ (constant) for all $t$, then $\mathbf{r}(t) \cdot \mathbf{r}'(t) = 0$.

_Proof._ $\mathbf{r}(t) \cdot \mathbf{r}(t) = c^2$. Differentiate both sides: $$\frac{d}{dt}[\mathbf{r} \cdot \mathbf{r}] = 2\mathbf{r}(t) \cdot \mathbf{r}'(t) = 0$$ Hence $\mathbf{r}(t) \perp \mathbf{r}'(t)$. $\blacksquare$

This lemma is the reason the velocity of uniform circular motion is always perpendicular to the position vector.

## 7. The Unit Tangent Vector

**Definition 7.1.** The _unit tangent vector_ at $t$ is:

$$\mathbf{T}(t) = \frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|}$$

defined wherever $\mathbf{r}'(t) \neq \mathbf{0}$.

Note: $|\mathbf{T}(t)| = 1$ by construction, so by Lemma 6.3, $\mathbf{T} \cdot \mathbf{T}' = 0$ — the derivative of the unit tangent is perpendicular to it. This is the entry point to curvature and the Frenet–Serret frame (Stewart §13.3).

---

## 8. Integration

**Definition 8.1.** The _definite integral_ of $\mathbf{r}$ over $[a,b]$ is:

$$\int_a^b \mathbf{r}(t),dt = \left\langle \int_a^b f(t),dt,\ \int_a^b g(t),dt,\ \int_a^b h(t),dt \right\rangle$$

**Example 8.1.** Compute $\displaystyle\int_0^{\pi} \langle \cos t,\ \sin t,\ t \rangle,dt$:

$$= \left\langle [\sin t]_0^{\pi},\ [-\cos t]_0^{\pi},\ \left[\frac{t^2}{2}\right]_0^{\pi} \right\rangle = \left\langle 0,\ 2,\ \frac{\pi^2}{2} \right\rangle$$

**Application — recovering position from velocity.** If $\mathbf{v}(t) = \mathbf{r}'(t)$ is known:

$$\mathbf{r}(t) = \int \mathbf{v}(t),dt + \mathbf{C}$$

where $\mathbf{C}$ is a _constant vector_ of integration (three free constants, one per component).

---

## 9. Arc Length

**Theorem 9.1.** The length of the curve traced by $\mathbf{r}(t)$ for $t \in [a,b]$ is:

$$L = \int_a^b |\mathbf{r}'(t)|,dt = \int_a^b \sqrt{[f'(t)]^2 + [g'(t)]^2 + [h'(t)]^2},dt$$

_Intuition._ Partition $[a,b]$ into $n$ subintervals. On each tiny interval of length $\Delta t$, the curve is approximately a straight segment of length $\approx |\mathbf{r}'(t)|,\Delta t$. Sum and take $n \to \infty$.

**Example 9.2 (arc length of a helix).** $\mathbf{r}(t) = \langle \cos t, \sin t, t \rangle$ on $[0, 2\pi]$:

$$\mathbf{r}'(t) = \langle -\sin t, \cos t, 1 \rangle \implies |\mathbf{r}'(t)| = \sqrt{\sin^2 t + \cos^2 t + 1} = \sqrt{2}$$

$$L = \int_0^{2\pi} \sqrt{2},dt = 2\sqrt{2},\pi$$

The helix is longer than the circle ($2\pi$) by a factor of $\sqrt{2}$ — the extra path cost of climbing vertically.

### Arc Length Parameter

Define the _arc length function_:

$$s(t) = \int_a^t |\mathbf{r}'(u)|,du$$

Then $\frac{ds}{dt} = |\mathbf{r}'(t)|$. Reparametrizing by $s$ gives a "unit-speed" curve — the most natural parametrization geometrically (independent of how fast the particle moves).

## 10. Connection to AI / Machine Learning / Deep Learning

This is where vector functions stop being pure mathematics and become engineering infrastructure.

### 10.1 The Forward Pass as a Composition of Vector Functions

A feedforward neural network with $L$ layers is precisely a composition of vector-valued functions:

$$\mathbf{r}^{(0)} = \mathbf{x} \in \mathbb{R}^{n_0}$$ $$\mathbf{r}^{(\ell)}(t) = \sigma!\left(W^{(\ell)},\mathbf{r}^{(\ell-1)} + \mathbf{b}^{(\ell)}\right) \in \mathbb{R}^{n_\ell}$$

Each layer is a vector function: it takes a vector in $\mathbb{R}^{n_{\ell-1}}$ and outputs one in $\mathbb{R}^{n_\ell}$. The activation $\sigma$ is applied component-wise.

The loss $\mathcal{L}$ is a scalar function of the final output — a scalar function of a vector function. **This is multivariable calculus, exactly.**

### 10.2 Gradients Are Derivatives of Scalar-Output Vector Functions

If $\mathcal{L} : \mathbb{R}^n \to \mathbb{R}$, the gradient $\nabla \mathcal{L}$ is the vector of partial derivatives. Backpropagation is the chain rule for compositions of vector functions (the Jacobian chain rule):

$$\frac{\partial \mathcal{L}}{\partial \mathbf{r}^{(\ell)}} = \left(\frac{\partial \mathbf{r}^{(\ell+1)}}{\partial \mathbf{r}^{(\ell)}}\right)^{!\top} \frac{\partial \mathcal{L}}{\partial \mathbf{r}^{(\ell+1)}}$$

The matrix $\dfrac{\partial \mathbf{r}^{(\ell+1)}}{\partial \mathbf{r}^{(\ell)}}$ is the **Jacobian** — the matrix of all first-order partial derivatives. This is the full generalization of the scalar derivative $f'(t)$.

### 10.3 Gradient Descent Traces a Curve in Parameter Space

Let $\boldsymbol{\theta}(k)$ be the parameter vector at training step $k$. Gradient descent defines:

$$\boldsymbol{\theta}(k+1) = \boldsymbol{\theta}(k) - \eta,\nabla_{!\boldsymbol{\theta}}\mathcal{L}$$

In the continuous limit ($\eta \to 0$), this is the **gradient flow ODE**:

$$\dot{\boldsymbol{\theta}}(t) = -\nabla_{!\boldsymbol{\theta}}\mathcal{L}(\boldsymbol{\theta}(t))$$

which is exactly a vector function $\boldsymbol{\theta} : \mathbb{R} \to \mathbb{R}^p$ (where $p$ is the number of parameters). Convergence analysis of optimizers studies the geometry of this curve in parameter space — the tangent vector, arc length, curvature, and whether the curve reaches a minimum.

### 10.4 Recurrent Networks — Trajectories in State Space

An RNN with hidden state $\mathbf{h}(t) \in \mathbb{R}^d$ evolves as:

$$\mathbf{h}(t+1) = \sigma!\left(W,\mathbf{h}(t) + U,\mathbf{x}(t) + \mathbf{b}\right)$$

This is literally a **discrete dynamical system** — the hidden state traces a trajectory in $\mathbb{R}^d$, a vector function of time $t$ (the sequence index). The vanishing/exploding gradient problem is a statement about the arc length and curvature properties of this trajectory: if the Jacobian $\partial \mathbf{h}(t)/\partial \mathbf{h}(0)$ has eigenvalues $< 1$, the trajectory contracts, gradients vanish; if $> 1$, it diverges.

### 10.5 Embeddings Are Points on Curves

Word embeddings, image embeddings, and graph embeddings map discrete objects into $\mathbb{R}^n$. Interpolation between two embeddings $\mathbf{e}_1$ and $\mathbf{e}_2$:

$$\mathbf{r}(t) = (1-t),\mathbf{e}_1 + t,\mathbf{e}_2, \quad t \in [0,1]$$

is a vector function — the line segment in embedding space. In latent diffusion models, the denoising process defines a continuous curve in the embedding space from Gaussian noise to the data manifold.
