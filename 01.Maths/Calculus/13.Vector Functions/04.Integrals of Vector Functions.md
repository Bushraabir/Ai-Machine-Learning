

> _"Integration of a vector function is antidifferentiation done simultaneously in each direction — the result is a family of curves, each one a possible trajectory through space."_

---

## 0. Motivation

We know how to differentiate $\mathbf{r}(t)$. Integration is the reverse operation. Two questions drive this section:

1. **Definite integral:** Given a velocity $\mathbf{v}(t)$, how much has the particle _displaced_ over $[a,b]$?
2. **Indefinite integral (antiderivative):** Given $\mathbf{r}'(t)$, recover $\mathbf{r}(t)$ up to a constant.

Both reduce to scalar calculus, component by component — the same philosophy as differentiation.

---

## 1. The Definite Integral — Definition

### 1.1 Riemann Sum Construction

Partition $[a,b]$ into $n$ subintervals of width $\Delta t = (b-a)/n$. Pick sample points $t_i^*$. Form the **Riemann sum**:

$$\sum_{i=1}^{n} \mathbf{r}(t_i^*),\Delta t ;\in; \mathbb{R}^3$$

This is a sum of _vectors_ scaled by a scalar — the result is a vector. Taking the limit:

$$\int_a^b \mathbf{r}(t),dt = \lim_{n\to\infty}\sum_{i=1}^{n}\mathbf{r}(t_i^*),\Delta t$$

### 1.2 The Component-Wise Theorem

**Theorem.** If $\mathbf{r}(t) = \langle f(t), g(t), h(t)\rangle$ and $f,g,h$ are integrable on $[a,b]$, then

$$\boxed{\int_a^b \mathbf{r}(t),dt = \left\langle \int_a^b f(t),dt,;\int_a^b g(t),dt,;\int_a^b h(t),dt \right\rangle}$$

**Proof.**

Fix any unit vector $\mathbf{e}_1 = \langle 1,0,0\rangle$. Then

$$\mathbf{e}_1 \cdot \int_a^b \mathbf{r}(t),dt = \int_a^b \mathbf{e}_1\cdot\mathbf{r}(t),dt = \int_a^b f(t),dt$$

where the interchange of the dot product with the limit is justified because the dot product is a continuous linear functional on $\mathbb{R}^3$. Applying the same argument to $\mathbf{e}_2, \mathbf{e}_3$ extracts $g$ and $h$. Since the three components are determined, the vector integral equals $\langle\int f, \int g, \int h\rangle$. $\square$

> **Remark.** This proof makes the argument rigorous: we are not "defining" the integral component-wise by hand — we prove it _follows_ from the limit definition together with linearity of the dot product.

## 2. The Indefinite Integral (Antiderivative)

$$\int \mathbf{r}(t),dt = \langle F(t), G(t), H(t)\rangle + \mathbf{C}$$

where $F' = f$, $G' = g$, $H' = h$, and $\mathbf{C} = \langle c_1, c_2, c_3\rangle$ is a **constant vector** (three arbitrary constants, one per component).

> **Critical point:** The constant of integration is a _vector_, not a scalar. Each component contributes its own free constant. Initial conditions pin down all three simultaneously.

### The Fundamental Theorem of Calculus — Vector Version

If $\mathbf{R}(t)$ is an antiderivative of $\mathbf{r}(t)$ on $[a,b]$, then

$$\int_a^b \mathbf{r}(t),dt = \mathbf{R}(b) - \mathbf{R}(a)$$

**Proof.** Apply the scalar FTC to each component. $\square$

## 3. Properties

Let $\mathbf{u}(t), \mathbf{v}(t)$ be integrable vector functions, $c \in \mathbb{R}$ a scalar constant.

|Property|Formula|

## 4. Worked Examples

### Example 1 — Basic Definite Integral

$$\mathbf{r}(t) = \langle 2t,; e^t,; \cos t\rangle, \qquad \int_0^{\pi}\mathbf{r}(t),dt$$

Component by component:

$$\int_0^{\pi} 2t,dt = \left[t^2\right]_0^{\pi} = \pi^2$$

$$\int_0^{\pi} e^t,dt = \left[e^t\right]_0^{\pi} = e^{\pi} - 1$$

$$\int_0^{\pi}\cos t,dt = \left[\sin t\right]_0^{\pi} = 0$$

$$\therefore\quad \int_0^{\pi}\mathbf{r}(t),dt = \langle \pi^2,; e^{\pi}-1,; 0\rangle$$

---

### Example 2 — Recovering Position from Velocity

**Given:** $\mathbf{v}(t) = \langle 3t^2,; 2t,; 1\rangle$ and initial condition $\mathbf{r}(0) = \langle 1, -1, 2\rangle$.

**Find:** $\mathbf{r}(t)$.

**Step 1.** Integrate:

$$\mathbf{r}(t) = \int\mathbf{v}(t),dt = \langle t^3 + c_1,; t^2 + c_2,; t + c_3\rangle$$

**Step 2.** Apply initial condition $\mathbf{r}(0) = \langle 1,-1,2\rangle$:

$$\langle c_1, c_2, c_3\rangle = \langle 1,-1,2\rangle$$

**Result:**

$$\boxed{\mathbf{r}(t) = \langle t^3+1,; t^2-1,; t+2\rangle}$$

> **Geometric reading:** The initial condition is a single _point_ in $\mathbb{R}^3$, but it pins down all three constants at once. Without it, the antiderivative is a _family_ of parallel space curves, shifted by $\mathbf{C}$.

---

### Example 3 — Recovering Position from Acceleration (Two Integrations)

**Given:** $\mathbf{a}(t) = \langle 0, -g, 0\rangle$ (projectile, $g = 9.8$ m/s²), with

$$\mathbf{v}(0) = \langle v_0\cos\theta,; v_0\sin\theta,; 0\rangle, \qquad \mathbf{r}(0) = \mathbf{0}$$

**First integration** (acceleration → velocity):

$$\mathbf{v}(t) = \int\mathbf{a}(t),dt = \langle C_1,; -gt + C_2,; C_3\rangle$$

Apply $\mathbf{v}(0)$: $C_1 = v_0\cos\theta$, $C_2 = v_0\sin\theta$, $C_3 = 0$.

$$\mathbf{v}(t) = \langle v_0\cos\theta,; v_0\sin\theta - gt,; 0\rangle$$

**Second integration** (velocity → position):

$$\mathbf{r}(t) = \int\mathbf{v}(t),dt = \langle v_0 t\cos\theta + D_1,; v_0 t\sin\theta - \tfrac{1}{2}gt^2 + D_2,; D_3\rangle$$

Apply $\mathbf{r}(0) = \mathbf{0}$: all $D_i = 0$.

$$\boxed{\mathbf{r}(t) = \langle v_0 t\cos\theta,; v_0 t\sin\theta - \tfrac{1}{2}gt^2,; 0\rangle}$$

The classical projectile equations — derived rigorously as two vector integrations.

```
y
│           •  ← peak
│        ·     ·
│      ·          ·
│    ·               ·
│  · ← r(t) trajectory  ·
│·________________________·_____ x
  θ↗  (2D slice, z=0)
```

### Example 4 — Integral Involving Cross Products

Sometimes we need $\int_a^b [\mathbf{u}(t)\times\mathbf{v}(t)],dt$. Compute the cross product first (getting a vector function), then integrate component-wise.

**Example:** $\mathbf{u}(t) = \langle 1,t,0\rangle$, $\mathbf{v}(t) = \langle 0,1,t\rangle$.

$$\mathbf{u}\times\mathbf{v} = \begin{vmatrix}\mathbf{i} & \mathbf{j} & \mathbf{k}\ 1 & t & 0\ 0 & 1 & t\end{vmatrix} = \langle t^2 - 0,; 0 - t,; 1-0\rangle = \langle t^2,; -t,; 1\rangle$$

$$\int_0^1 (\mathbf{u}\times\mathbf{v}),dt = \left\langle \frac{t^3}{3}, -\frac{t^2}{2}, t\right\rangle_0^1 = \left\langle \frac{1}{3}, -\frac{1}{2}, 1\right\rangle$$

## 5. Arc Length Revisited — Via Integration

From the derivatives note, arc length is

$$L = \int_a^b |\mathbf{r}'(t)|,dt$$

This is a _scalar_ integral of the magnitude of a vector function — not a vector integral itself. But it arises directly from the theory: $|\mathbf{r}'(t)|$ is the speed, and integrating speed gives distance.

### Arc Length Function

$$s(t) = \int_a^t |\mathbf{r}'(u)|,du \implies \frac{ds}{dt} = |\mathbf{r}'(t)|$$

To **reparametrize by arc length**: solve $s = s(t)$ for $t = t(s)$, then form $\tilde{\mathbf{r}}(s) = \mathbf{r}(t(s))$. One then verifies $|\tilde{\mathbf{r}}'(s)| = 1$.

## 6. The Displacement vs. Distance Distinction

This is a subtle point worth dwelling on.

|Quantity|Formula|Type|
|---|---|---|
|**Displacement**|$\displaystyle\int_a^b \mathbf{v}(t),dt = \mathbf{r}(b) - \mathbf{r}(a)$|Vector|
|**Distance traveled**|$\displaystyle\int_a^b|\mathbf{v}(t)|

A particle that returns to its start has _zero displacement_ but nonzero _distance traveled_ — both computed by integrals, but of different types.

> **Analogy with scalar calculus:** $\int_a^b f,dt$ vs $\int_a^b|f|,dt$ — net signed area vs total area. The vector version is geometrically richer.

## 7. Connection to AI / Machine Learning / Deep Learning

### 7.1 Gradient Flow as a Vector Integral

In continuous gradient descent, the parameter trajectory satisfies

$$\frac{d\boldsymbol{\theta}}{dt} = -\nabla_{\boldsymbol{\theta}}\mathcal{L}(\boldsymbol{\theta}(t))$$

**Integrating** both sides from $0$ to $T$:

$$\boldsymbol{\theta}(T) = \boldsymbol{\theta}(0) + \int_0^T \left(-\nabla\mathcal{L}(\boldsymbol{\theta}(t))\right)dt$$

This is exactly the vector antiderivative problem: given the "velocity" $-\nabla\mathcal{L}$, recover the trajectory $\boldsymbol{\theta}(t)$. The initial condition $\boldsymbol{\theta}(0)$ (random initialization) plays the role of the constant vector $\mathbf{C}$.

The _displacement_ after training is $\boldsymbol{\theta}(T) - \boldsymbol{\theta}(0)$ — a vector in parameter space measuring how far initialization moved during learning.

### 7.2 Neural ODEs — Forward Pass as Integration

A Neural ODE (Chen et al., 2018) defines the forward pass as:

$$\mathbf{h}(T) = \mathbf{h}(0) + \int_0^T F(\mathbf{h}(t), \boldsymbol{\theta}, t),dt$$

where $F$ is a neural network. The **output** of the model is literally a vector integral. This is computed using an ODE solver (e.g., Runge–Kutta) at training time — numerical integration of a vector function is the entire forward pass.

```
h(0) ──→ [ODE Solver: ∫F dt] ──→ h(T) ──→ Output
         ↑
    F = neural network
    evaluates r'(t) at each step
```

The backward pass (adjoint method) also involves integrating a vector ODE — _backwards in time_.

### 7.3 Physics-Informed Neural Networks (PINNs)

A PINN solving Newton's law $\mathbf{r}''(t) = \mathbf{F}/m$ enforces:

$$\mathbf{r}(t) = \mathbf{r}(0) + \mathbf{v}(0),t + \int_0^t\int_0^{s}\frac{\mathbf{F}(\mathbf{r}(u))}{m},du,ds$$

Two vector integrations, mirroring Example 3 exactly. The network must implicitly learn these integrals from data, and the loss function penalizes violations.

### 7.4 Displacement vs. Distance in Optimization

The distinction from §6 has a direct ML interpretation:

- **Displacement** $\int_0^T(-\nabla\mathcal{L}),dt = \boldsymbol{\theta}(T) - \boldsymbol{\theta}(0)$: the net change in parameters. Can be small even after long training if the optimizer oscillates.
- **Path length** $\int_0^T|\nabla\mathcal{L}(\boldsymbol{\theta}(t))|,dt$: total distance traveled in parameter space. A diagnostic for optimizer efficiency — large path length with small displacement indicates oscillation or poor conditioning.

> This is why **adaptive optimizers** (Adam, RMSProp) work well: they effectively normalize the gradient vector at each step, keeping $|\mathbf{v}(t)|$ bounded and ensuring displacement ≈ path length.

### 7.5 Attention and Integral Transforms

The **attention mechanism** in Transformers can be written in continuous form as

$$\text{Attn}(\mathbf{q}) = \int K({\mathbf{q}}, \mathbf{k}),\mathbf{v}(\mathbf{k}),d\mathbf{k}$$

where $K$ is a scalar kernel and $\mathbf{v}$ is a vector-valued function of keys. This is a **weighted vector integral** — a continuous generalization of the discrete softmax attention sum. The "value" output is a vector obtained by integrating a vector function against a scalar weight.

> **Summary table:**

|Classical Concept|ML / DL Analogue|
|---|---|
|$\int\mathbf{r}'(t),dt = \mathbf{r}(t)+\mathbf{C}$|Integrating gradient flow to recover $\boldsymbol{\theta}(t)$|
|Initial condition pins $\mathbf{C}$|Random initialization $\boldsymbol{\theta}(0)$|
|Two integrations: $\mathbf{a}\to\mathbf{v}\to\mathbf{r}$|Neural ODE forward pass (nested integration)|
|Displacement $\neq$ distance|Net weight change vs total optimization path|
|$\int_a^b\mathbf{r}(t),dt$ as Riemann sum limit|Numerical ODE solvers (Euler, RK4) in Neural ODEs|
|$\int K,\mathbf{v},d\mathbf{k}$|Continuous attention mechanism in Transformers|

## 8. Summary of Key Formulas

$$\int_a^b\mathbf{r}(t),dt = \left\langle\int_a^b f,;\int_a^b g,;\int_a^b h\right\rangle \qquad \text{[component-wise]}$$

$$\int\mathbf{r}(t),dt = \mathbf{R}(t) + \mathbf{C}, \quad \mathbf{C}\in\mathbb{R}^3 \qquad \text{[constant vector]}$$

$$\int_a^b\mathbf{r}(t),dt = \mathbf{R}(b) - \mathbf{R}(a) \qquad \text{[FTC — vector version]}$$

$$\left|\int_a^b\mathbf{r}(t),dt\right| \leq \int_a^b|\mathbf{r}(t)|,dt \qquad \text{[norm inequality]}$$

$$L = \int_a^b|\mathbf{r}'(t)|,dt \qquad \text{[arc length]}$$
