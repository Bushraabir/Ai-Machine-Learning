

> _"Every differentiation formula, when read backwards, is an antidifferentiation formula."_ — Stewart, _Calculus_ (§4.9)

---

## 1. The Central Question

Differentiation asks: _given $F$, find $F'$._

Antidifferentiation inverts the question: _given $f$, find $F$ such that $F' = f$._

This is the founding problem of integral calculus, and it turns out to be **far harder** than differentiation — there is no single algorithm analogous to the chain rule or product rule. We are, in a very real sense, doing archaeology: reconstructing a function from its rate of change.

---

## 2. Definition

> **Definition.** A function $F$ is called an **antiderivative** of $f$ on an interval $I$ if $$F'(x) = f(x) \quad \text{for all } x \in I.$$

### Non-uniqueness — and its resolution

Suppose $F$ is _one_ antiderivative of $f$. Is it the only one? No. Consider $G(x) = F(x) + 7$. Then $G'(x) = F'(x) = f(x)$. The constant vanishes under differentiation.

The key theorem (which rests on the Mean Value Theorem) tells us that constants are the _only_ freedom:

> **Theorem.** If $F'(x) = G'(x)$ for all $x \in I$, then $G(x) = F(x) + C$ for some constant $C \in \mathbb{R}$.

_Proof sketch._ Let $H = G - F$. Then $H'(x) = 0$ on $I$. By the MVT, for any $a, b \in I$, $$H(b) - H(a) = H'(c)(b-a) = 0,$$ so $H$ is constant. $\blacksquare$

This gives us the canonical notation:

$$\int f(x), dx = F(x) + C$$

where $C \in \mathbb{R}$ is the **constant of integration**. The integral sign $\int$ is an elongated "S" (for _summa_), but here it acts as the **antidifferentiation operator** — the connection to sums will be made precise by the Fundamental Theorem.

---

## 3. Geometric Interpretation

The constant $C$ parameterises a **family of curves**. Each value of $C$ gives a distinct antiderivative, and these curves are vertical translates of one another — they never intersect, and at every $x$-value they all carry the **same slope** $f(x)$.

```
        F(x) + 2  ──────────────────
        F(x) + 1  ──────────────────    ← all have slope f(x) at each x
        F(x)      ──────────────────
        F(x) − 1  ──────────────────
```

**Example.** Let $f(x) = 2x$. Then $F(x) = x^2 + C$ is the family of upward-opening parabolas. Each parabola has slope $2x$ at every point — they are vertical shifts of $y = x^2$.

Initial conditions collapse this family to a single curve. If additionally $F(0) = 3$, then $0 + C = 3 \Rightarrow C = 3$, so $F(x) = x^2 + 3$.

## 4. The Standard Table of Antiderivatives

These are read directly from differentiation identities run in reverse.

|$f(x)$|$\int f(x),dx$|Source identity|
|---|---|---|
|$x^n$, $n \neq -1$|$\dfrac{x^{n+1}}{n+1} + C$|Power rule|
|$x^{-1} = \frac{1}{x}$|$\ln\|x\| + C$|$\frac{d}{dx}\ln\|x\| = \frac{1}{x}$|
|$e^x$|$e^x + C$|$\frac{d}{dx}e^x = e^x$|
|$a^x$|$\dfrac{a^x}{\ln a} + C$|$\frac{d}{dx}a^x = a^x \ln a$|
|$\sin x$|$-\cos x + C$|$\frac{d}{dx}(-\cos x) = \sin x$|
|$\cos x$|$\sin x + C$||
|$\sec^2 x$|$\tan x + C$||
|$\csc^2 x$|$-\cot x + C$||
|$\sec x \tan x$|$\sec x + C$||
|$\dfrac{1}{\sqrt{1-x^2}}$|$\arcsin x + C$||
|$\dfrac{1}{1+x^2}$|$\arctan x + C$||

> **Notational remark.** We write $\ln|x|$ (not $\ln x$) because $\frac{d}{dx}\ln(-x) = \frac{1}{x}$ for $x < 0$ as well. The absolute value ensures validity on both sides of the origin.

---

## 5. Linearity of the Antiderivative

The antiderivative operator is **linear**:

$$\int \bigl[\alpha f(x) + \beta g(x)\bigr],dx = \alpha \int f(x),dx + \beta \int g(x),dx$$

This follows immediately from the linearity of the derivative. It does **not**, however, extend to products or quotients — there is no product rule or quotient rule for antiderivatives (those become integration by parts and partial fractions, respectively).

## 6. Worked Examples

### 6.1 Power Functions

**Find** $\displaystyle\int \left(3x^4 - 5\sqrt{x} + \frac{2}{x^3}\right)dx$.

Rewrite: $3x^4 - 5x^{1/2} + 2x^{-3}$.

$$= \frac{3x^5}{5} - 5\cdot\frac{x^{3/2}}{3/2} + 2\cdot\frac{x^{-2}}{-2} + C = \frac{3x^5}{5} - \frac{10x^{3/2}}{3} - \frac{1}{x^2} + C$$

**Verification** (always do this!): differentiate and confirm you recover the integrand. ✓

---

### 6.2 An Initial Value Problem

**Find** $f$ given $f'(x) = \sin x - e^x$ and $f(0) = 2$.

$$f(x) = \int (\sin x - e^x),dx = -\cos x - e^x + C$$

Apply initial condition: $f(0) = -1 - 1 + C = 2 \Rightarrow C = 4$.

$$\boxed{f(x) = -\cos x - e^x + 4}$$

---

### 6.3 Rectilinear Motion

This is the setting where antiderivatives appear most naturally in physics.

Given acceleration $a(t) = -9.8$ m/s² (constant gravitational field), initial velocity $v(0) = 20$ m/s, initial position $s(0) = 0$:

$$v(t) = \int a(t),dt = -9.8t + C_1 \xrightarrow{v(0)=20} v(t) = -9.8t + 20$$

$$s(t) = \int v(t),dt = -4.9t^2 + 20t + C_2 \xrightarrow{s(0)=0} s(t) = -4.9t^2 + 20t$$

The object reaches maximum height when $v(t) = 0$: $t = 20/9.8 \approx 2.04$ s, at height $s(2.04) \approx 20.4$ m.

---

### 6.4 A Subtlety — Piecewise Domains

Consider $\int \frac{1}{x},dx$ when the domain is **not** a single interval. If $f$ is defined on $(-\infty, 0) \cup (0, \infty)$, the most general antiderivative is:

$$F(x) = \begin{cases} \ln(-x) + C_1 & x < 0 \ \ln x + C_2 & x > 0 \end{cases}$$

with _independent_ constants on each component. Writing $\ln|x| + C$ implicitly assumes the same constant on both sides — technically a special case. This subtlety matters in complex analysis and when computing integrals over disconnected domains. See MIT 18.02 notes on antiderivatives of rational functions.

## 7. The Fundamental Theorem of Calculus — The Bridge

The antiderivative does not stand alone. The Fundamental Theorem (Stewart §5.3) **unifies** it with the definite integral:

> **FTC Part 2.** If $F$ is any antiderivative of $f$ on $[a,b]$, then $$\int_a^b f(x),dx = F(b) - F(a).$$

This is why antiderivatives matter: computing a definite integral (an area, a net displacement, a total quantity) reduces to evaluating an antiderivative at two points. Without antiderivatives, we would be forced to compute Riemann sums by hand for every integral.

## 8. Connection to AI, Machine Learning, and Deep Learning

This is not merely formal mathematics — antiderivatives are **quietly present throughout the machinery of modern AI**.

### 8.1 Gradient Descent and the Reverse of Differentiation

Every neural network is trained by computing gradients — derivatives of a loss function with respect to millions of parameters. The optimizer then takes a step in the negative gradient direction.

Antiderivatives enter when we ask: _what function has this gradient?_ In other words, given $\nabla_\theta \mathcal{L}(\theta)$, can we recover $\mathcal{L}(\theta)$? This is the antiderivative (or more precisely, the **antigradient**) problem. It is trivially solvable only when the gradient field is conservative — i.e., when $\nabla \times (\nabla \mathcal{L}) = 0$. In practice, we do not need to recover $\mathcal{L}$ explicitly, but the existence and structure of antiderivatives underpins the mathematical well-posedness of the optimisation.

### 8.2 Activation Functions — Antiderivatives of Each Other

Several standard activation functions are antiderivatives of each other or of simpler functions:

|Function|Derivative|Relevance|
|---|---|---|
|Sigmoid $\sigma(x) = \frac{1}{1+e^{-x}}$|$\sigma(x)(1-\sigma(x))$|$\int \sigma'(x),dx = \sigma(x)$ — backprop uses this|
|Softplus $\ln(1 + e^x)$|$\sigma(x)$|Softplus is the **antiderivative** of the sigmoid!|
|ReLU $\max(0,x)$|$\mathbf{1}_{x>0}$ (Heaviside)|ReLU is the antiderivative of the Heaviside step function|

The fact that Softplus is the antiderivative of Sigmoid is not a coincidence — it reflects a design principle: smooth activations should have smooth, integrable derivatives for stable gradient flow during training.

### 8.3 Normalising Flows and Change of Variables

**Normalising flows** are a class of generative models (used in diffusion model theory, variational inference) that learn an invertible map $T: \mathbb{R}^d \to \mathbb{R}^d$ transforming a simple distribution (e.g. Gaussian) into a complex data distribution. The density transformation requires the **Jacobian determinant**:

$$p_X(x) = p_Z(T^{-1}(x)) \cdot \left|\det \frac{\partial T^{-1}}{\partial x}\right|$$

In 1D, this reduces exactly to the change-of-variables formula for integrals:

$$\int_a^b f(x),dx = \int_{T^{-1}(a)}^{T^{-1}(b)} f(T(u)), T'(u), du$$

which is the **substitution rule** for antiderivatives. Learning the map $T$ requires computing and differentiating through these Jacobians — antiderivative structure determines whether the density is tractable.

### 8.4 Physics-Informed Neural Networks (PINNs)

In PINNs, a neural network $u_\theta(x, t)$ is trained to satisfy a differential equation (e.g., the heat equation $u_t = \kappa u_{xx}$). The loss function penalises the **residual** of the PDE. Solving such a PDE analytically would require finding antiderivatives; the network learns to approximate these antiderivatives without closed form. In this sense, PINNs are **implicit antiderivative approximators**.

### 8.5 Probability and the CDF

Every probability density function $p(x)$ integrates to 1. Its cumulative distribution function is:

$$F(x) = \int_{-\infty}^x p(t),dt$$

This $F$ is precisely the antiderivative of $p$ (up to the lower limit). In variational autoencoders, diffusion models, and score-based generative models, one constantly passes between densities and their CDFs — between a function and its antiderivative.

### 8.6 Backpropagation as Reverse-Mode Differentiation

Backpropagation computes $\frac{\partial \mathcal{L}}{\partial \theta}$ via the chain rule. The **adjoint method** (used in neural ODEs and optimal control) goes further: it computes sensitivities by solving an **adjoint ODE backward in time**, which is analytically the problem of finding the antiderivative of the adjoint dynamics. Neural ODEs (Chen et al., 2018) parameterise the hidden state dynamics as $\frac{dh}{dt} = f_\theta(h, t)$ — and computing the hidden state itself requires integrating (antidifferentiating) this dynamics forward in time.

---

## 9. Summary

$$\boxed{\int f(x),dx = F(x) + C \iff F'(x) = f(x)}$$

The antiderivative:

- **Is not unique** — unique up to an additive constant $C$.
- **Is computed** by reversing differentiation identities.
- **Is specified uniquely** by an initial condition (IVP).
- **Is connected to definite integrals** through the FTC.
- **Appears throughout AI** — in activation function design, normalising flows, PINNs, CDFs, and Neural ODEs.

The constant $C$ is not mere notational clutter. It represents genuine non-uniqueness in the inverse problem of differentiation — and in physical models, it encodes the **initial conditions** of the system.
