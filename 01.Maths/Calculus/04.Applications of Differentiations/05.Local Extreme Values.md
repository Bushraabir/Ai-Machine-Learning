

> _"An extremum is not merely where the derivative vanishes — it is where the function hesitates, reverses its testimony, and commits to a new direction. Understanding why requires the full weight of the Mean Value Theorem."_

---

## §1. The Central Question

Given a function $f$, we want to know: **where does $f$ achieve its highest or lowest values locally?**

This is not just a geometric curiosity. The same question — phrased in higher dimensions — is exactly what gradient descent answers during neural network training. Every parameter update in backpropagation is navigating the local geometry of a loss surface. The theory begins here, with a single real variable.

---

## §2. Definitions — Absolute vs. Local

### 2.1 Absolute (Global) Extrema

> **Definition.** A function $f$ has an **absolute maximum** at $c$ if $f(c) \geq f(x)$ for all $x$ in the domain of $f$. Similarly, $f$ has an **absolute minimum** at $c$ if $f(c) \leq f(x)$ for all $x$ in the domain.

These are the global champions — no point anywhere does better.

### 2.2 Local (Relative) Extrema

> **Definition.** A function $f$ has a **local maximum** at $c$ if there exists an open interval $(a, b)$ containing $c$ such that $f(c) \geq f(x)$ for all $x \in (a, b)$.

> **Definition.** A function $f$ has a **local minimum** at $c$ if there exists an open interval $(a, b)$ containing $c$ such that $f(c) \leq f(x)$ for all $x \in (a, b)$.

**Key point:** Local extrema are _neighborhood champions_ — $f(c)$ beats all nearby competitors, but not necessarily the whole field.

```
Absolute vs. Local:

          absolute max
               *
              / \
 local max   /   \
     *       /     \
    / \     /       \
   /   \___/         \
  /    local min      \______ → x
 /
absolute min is at x → -∞ (if it exists)

Every absolute extremum is also a local extremum.
The converse is FALSE.
```

### 2.3 Why the Distinction Matters

In optimization, we often _want_ the absolute minimum of the loss function, but gradient-based methods can only _guarantee_ finding a local minimum. The relationship between local and global minima — and whether they coincide — is a deep question in both calculus and machine learning.

---

## §3. Fermat's Theorem — The Necessary Condition

### 3.1 Statement

> **Theorem (Fermat).** If $f$ has a local maximum or local minimum at $c$, and if $f'(c)$ exists, then $f'(c) = 0$.

### 3.2 Full Proof

**Proof (local maximum case).**

Suppose $f$ has a local maximum at $c$. Then $\exists, \delta > 0$ such that: $$f(c) \geq f(x) \quad \text{for all } x \in (c - \delta,, c + \delta)$$

Consider the difference quotient approaching from the right. For $0 < h < \delta$: $$f(c + h) \leq f(c) \implies f(c + h) - f(c) \leq 0 \implies \frac{f(c+h) - f(c)}{h} \leq 0$$

Taking the limit as $h \to 0^+$: $$f'(c) = \lim_{h \to 0^+} \frac{f(c+h) - f(c)}{h} \leq 0 \quad \cdots (1)$$

Now approach from the left. For $-\delta < h < 0$: $$f(c + h) \leq f(c) \implies f(c + h) - f(c) \leq 0$$

But now $h < 0$, so dividing flips the inequality: $$\frac{f(c+h) - f(c)}{h} \geq 0$$

Taking the limit as $h \to 0^-$: $$f'(c) = \lim_{h \to 0^-} \frac{f(c+h) - f(c)}{h} \geq 0 \quad \cdots (2)$$

Since $f'(c)$ exists (by hypothesis), both one-sided limits equal $f'(c)$. From $(1)$ and $(2)$: $$f'(c) \leq 0 \quad \text{and} \quad f'(c) \geq 0$$

Therefore $f'(c) = 0$. $\square$

_(The local minimum proof is identical with all inequalities reversed.)_

### 3.3 The Critical Converse Failure

**Fermat's theorem is NOT reversible.** $f'(c) = 0$ does **not** imply a local extremum at $c$.

**Canonical counterexample:** $f(x) = x^3$

$$f'(x) = 3x^2 \implies f'(0) = 0$$

But $f$ is strictly increasing everywhere — $x = 0$ is an **inflection point**, not an extremum.

```
f(x) = x³         f'(x) = 3x²

     |  /              |
     | /               |   ___
     |/                |  /
-----+-----        ----+------
    /|                 |
   / |               (always ≥ 0)
  /  |
  
f'(0) = 0 but no extremum.    f' never changes sign through 0.
```

**Second canonical counterexample:** $f(x) = x^{1/3}$

$$f'(x) = \frac{1}{3}x^{-2/3} \quad \text{— undefined at } x = 0$$

Yet $f$ has a local (and absolute) minimum at $x = 0$. Extrema can occur where $f'$ fails to exist.

This motivates the following:

---

## §4. Critical Numbers

> **Definition.** A number $c$ in the domain of $f$ is a **critical number** (or _critical point_) of $f$ if either:
> 
> 1. $f'(c) = 0$, **or**
> 2. $f'(c)$ does not exist.

**Fermat's Theorem restated:** If $f$ has a local extremum at $c$, then $c$ is a critical number.

**Critical numbers are necessary but not sufficient for local extrema.**

The task is then to classify each critical number: is it a local max, local min, or neither? Two tools exist for this: the First Derivative Test and the Second Derivative Test.

---

## §5. The Extreme Value Theorem (Existence)

Before classifying extrema, a theorem guarantees they exist on closed intervals:

> **Theorem (Extreme Value Theorem).** If $f$ is continuous on a closed interval $[a, b]$, then $f$ attains both an absolute maximum and an absolute minimum on $[a, b]$.

**The closed interval method** for finding absolute extrema:

1. Find all critical numbers $c$ in $(a, b)$.
2. Evaluate $f$ at each critical number and at both endpoints $a$, $b$.
3. The largest value is the absolute maximum; the smallest is the absolute minimum.

**Why endpoints matter:** Local extrema detected by $f' = 0$ live in the interior. But absolute extrema can occur at the boundary of the domain — and the derivative says nothing there.

```
Closed interval [a, b]:

        interior max             endpoint max
              *                         *
             / \                       /
            /   \     *               /
           /     \___/ \             /
          /             \___________/
         a                          b

Interior critical pts + endpoints = candidates for absolute extrema.
```

## §6. The First Derivative Test

### 6.1 Statement

> **First Derivative Test.** Suppose $c$ is a critical number of a continuous function $f$.
> 
> - If $f'$ changes from **positive to negative** at $c$: $f$ has a **local maximum** at $c$.
> - If $f'$ changes from **negative to positive** at $c$: $f$ has a **local minimum** at $c$.
> - If $f'$ does **not change sign** at $c$: $f$ has **no local extremum** at $c$.

### 6.2 Why This Works — Proof Sketch

**Local maximum case.** Suppose $f'$ changes $+ \to -$ at $c$. Then:

- On $(c - \delta, c)$: $f'(x) > 0 \implies f$ is increasing toward $c$.
- On $(c, c + \delta)$: $f'(x) < 0 \implies f$ is decreasing away from $c$.

So $f$ rises to $c$ then falls — this is exactly a local maximum. $\square$

_(Formally this uses the Increasing/Decreasing Test, which in turn uses the Mean Value Theorem — see the previous note. So the MVT is, once again, the load-bearing structure.)_

### 6.3 How to Apply It — The Sign Chart Method

**Algorithm:**

1. Find all critical numbers of $f$.
2. Plot them on a number line, creating open intervals between them.
3. Choose a **test value** in each interval.
4. Evaluate $f'$ at the test value — record only the **sign**.
5. Read off the behavior from the sign changes.

**Example.** $f(x) = x^4 - 8x^2 + 3$

$$f'(x) = 4x^3 - 16x = 4x(x^2 - 4) = 4x(x-2)(x+2)$$

Critical numbers: $x = -2, ; 0, ; 2$.

Sign chart for $f'(x) = 4x(x-2)(x+2)$:

|Interval|Test pt|Sign of $4x$|Sign of $(x-2)$|Sign of $(x+2)$|$f'$|$f$|
|---|---|---|---|---|---|---|
|$(-\infty, -2)$|$-3$|$-$|$-$|$-$|$-$|$\searrow$|
|$(-2, 0)$|$-1$|$-$|$-$|$+$|$+$|$\nearrow$|
|$(0, 2)$|$1$|$+$|$-$|$+$|$-$|$\searrow$|
|$(2, \infty)$|$3$|$+$|$+$|$+$|$+$|$\nearrow$|

Sign transitions:

- $x = -2$: $f'$ goes $- \to +$ → **local minimum**, $f(-2) = 16 - 32 + 3 = -13$
- $x = 0$: $f'$ goes $+ \to -$ → **local maximum**, $f(0) = 3$
- $x = 2$: $f'$ goes $- \to +$ → **local minimum**, $f(2) = 16 - 32 + 3 = -13$

```
f(x) = x⁴ − 8x² + 3:

  f' < 0      f' > 0     f' < 0      f' > 0
       \       /  \       /
        \     /    \     /
    ↓    \   / ↑    \   /  ↑
          \ /        \ /
           *          *
        (-2,−13)   (2,−13)    ← local min (equal by symmetry)
         
              (0,3)
              *                ← local max
             / \
```

**Note on symmetry:** $f(-x) = f(x)$ (even function), so the graph is symmetric about the $y$-axis, which explains why the two minima have equal value. Always exploit symmetry — it halves the work and is a consistency check.

### 6.4 The First Derivative Test Never Fails

Unlike the Second Derivative Test (§7), the First Derivative Test:

- Works when $f''$ does not exist.
- Works when $f''(c) = 0$ (the inconclusive case for the second test).
- Applies to non-differentiable critical numbers (where $f'$ DNE).

It is the gold standard. The second derivative test is a shortcut that sometimes breaks down.

---

## §7. The Second Derivative Test

### 7.1 Statement

> **Second Derivative Test.** Suppose $f''$ is continuous near $c$, and $f'(c) = 0$.
> 
> - If $f''(c) > 0$: $f$ has a **local minimum** at $c$.
> - If $f''(c) < 0$: $f$ has a **local maximum** at $c$.
> - If $f''(c) = 0$: **inconclusive** — the test says nothing. Use the first derivative test.

### 7.2 Full Proof

**Local minimum case: $f'(c) = 0$, $f''(c) > 0$.**

By definition of the derivative: $$f''(c) = \lim_{x \to c} \frac{f'(x) - f'(c)}{x - c} = \lim_{x \to c} \frac{f'(x)}{x - c} > 0$$

(using $f'(c) = 0$). Since this limit is positive, there exists $\delta > 0$ such that for all $x \in (c - \delta, c + \delta) \setminus {c}$:

$$\frac{f'(x)}{x - c} > 0$$

This means $f'(x)$ and $(x - c)$ have the **same sign**:

- $x \in (c - \delta, c)$: $x - c < 0 \implies f'(x) < 0$
- $x \in (c, c + \delta)$: $x - c > 0 \implies f'(x) > 0$

So $f'$ changes from negative to positive at $c$. By the First Derivative Test: **local minimum**. $\square$

_The local maximum case is identical with $f''(c) < 0$ and all signs reversed._

### 7.3 Geometric Interpretation

The second derivative test is really the First Derivative Test applied one level up:

- $f''(c) > 0$ means $f'$ is **increasing** through $0$ at $c$.
- An increasing function that passes through $0$ goes from negative to positive.
- Negative to positive in $f'$ means local minimum in $f$.

```
f''(c) > 0 → f' increasing:        f''(c) < 0 → f' decreasing:

  f'                                  f'
   |     /                             |\ 
   |    /                              | \
   |   /                              0|  \
---0--/--------→ x                 ---+---\---→ x
   | /     c                         c     \
   |/                                       \

f' goes −→+ at c: LOCAL MIN        f' goes +→− at c: LOCAL MAX
```

### 7.4 The Inconclusive Case — All Three Behaviors Possible

When $f'(c) = 0$ and $f''(c) = 0$, anything can happen:

|Function|$f'(0)$|$f''(0)$|Behavior at $0$|
|---|---|---|---|
|$f(x) = x^4$|$0$|$0$|Local (and absolute) min|
|$f(x) = -x^4$|$0$|$0$|Local (and absolute) max|
|$f(x) = x^3$|$0$|$0$|Inflection — neither|

```
x⁴ (local min):     -x⁴ (local max):     x³ (neither):

      ___                                    /
     /   \              \_____/            /
    /     \                               /
---*-------              ---*---         ---*---
   0                        0               0
   
f''(0)=0 in all three cases. Test fails to distinguish them.
```

**Rule:** When $f''(c) = 0$, abandon the second derivative test and fall back to the first derivative test. Always.

### 7.5 Comparison: First vs. Second Derivative Test

|Feature|First Derivative Test|Second Derivative Test|
|---|---|---|
|Requires|$f$ continuous at $c$, $f'$ defined nearby|$f''$ continuous near $c$|
|Works when $f''(c) = 0$|✓ Yes|✗ No (inconclusive)|
|Works when $f'$ DNE at $c$|✓ Yes|✗ No|
|Computational cost|Sign chart required|Single evaluation $f''(c)$|
|Can it fail?|No|Yes (when $f''(c) = 0$)|

**Practical note:** Use the second derivative test when $f''$ is easy to compute and you already have $f''$ from concavity analysis. Fall back to the first derivative test when the second test is inconclusive or when $f''$ is tedious.

## §8. A Complete Worked Example

**Find and classify all local extrema of $f(x) = x^2 e^{-x}$.**

**Step 1: Domain.** All of $\mathbb{R}$.

**Step 2: Find $f'$.**

$$f'(x) = 2x e^{-x} + x^2(-e^{-x}) = e^{-x}(2x - x^2) = x(2-x)e^{-x}$$

Note: $e^{-x} > 0$ always, so the sign of $f'$ is determined entirely by $x(2-x)$.

**Step 3: Critical numbers.**

$f'(x) = 0 \iff x(2-x) = 0 \iff x = 0 \text{ or } x = 2$

$f'$ exists everywhere (product of smooth functions).

**Step 4: Sign chart.**

|Interval|Sign of $x$|Sign of $(2-x)$|Sign of $f'$|$f$|
|---|---|---|---|---|
|$(-\infty, 0)$|$-$|$+$|$-$|$\searrow$|
|$(0, 2)$|$+$|$+$|$+$|$\nearrow$|
|$(2, \infty)$|$+$|$-$|$-$|$\searrow$|

**Step 5: Apply First Derivative Test.**

- $x = 0$: $f'$ goes $- \to +$ → **local minimum**, $f(0) = 0 \cdot e^0 = 0$.
- $x = 2$: $f'$ goes $+ \to -$ → **local maximum**, $f(2) = 4e^{-2} \approx 0.541$.

**Verification via Second Derivative Test:**

$$f''(x) = \frac{d}{dx}\left[x(2-x)e^{-x}\right] = e^{-x}(2 - 4x + x^2)$$

- $f''(0) = e^0(2) = 2 > 0$ → **local minimum** ✓
- $f''(2) = e^{-2}(2 - 8 + 4) = -2e^{-2} < 0$ → **local maximum** ✓

**Step 6: Asymptotic behavior.**

As $x \to +\infty$: $e^{-x} \to 0$ faster than any polynomial grows, so $f(x) \to 0$. As $x \to -\infty$: $e^{-x} \to +\infty$, so $f(x) \to +\infty$.

```
f(x) = x²e^{-x}:

      ↑ f
      |
   ↗  |
  /   |             local max (2, 4e⁻²)
 /    |               *
/     |              / \
      |             /   \______________→ 0
      |            /
------*-----------*-----------------------→ x
      0           2
   local min     local max

f(0)=0 (local min), f(2)≈0.541 (local max), f→0 as x→+∞
```

## §9. Extrema at Non-Differentiable Points

Fermat's theorem only addresses the case where $f'(c)$ exists. But extrema can live precisely where $f'$ breaks down. Three common scenarios:

### 9.1 Corner Points

$f(x) = |x|$ has a global (and local) minimum at $x = 0$, where $f'(0)$ does not exist (left derivative $= -1$, right derivative $= +1$).

The first derivative test still works: $f'$ changes from $-1$ (negative) to $+1$ (positive) — this is a sign change from negative to positive, so it is a local minimum.

### 9.2 Cusp Points

$f(x) = x^{2/3}$ has a global minimum at $x = 0$. Here $f'(x) = \frac{2}{3}x^{-1/3} \to \pm\infty$ as $x \to 0^\pm$ — the derivative blows up rather than being undefined. Still a critical number, still a local minimum.

### 9.3 Endpoints (on closed intervals)

On $[a, b]$, the endpoints $a$ and $b$ are automatically critical in the sense that the Extreme Value Theorem mandates checking them. The derivative is one-sided at endpoints; Fermat's theorem does not apply.

```
Corner:         Cusp:           Endpoint:
  /\              |               ___
 /  \            / \             /   |
/    \          /   \           /    |
      (0)      (0)              a    b

f'(0) DNE     f'(0) DNE       f'(a) is one-sided
All three are critical numbers and can be extrema.
```

## §10. Connection to AI, Machine Learning, and Deep Learning

### 10.1 The Loss Landscape as a High-Dimensional Cousin

In training a neural network with parameters $\boldsymbol{\theta} \in \mathbb{R}^p$ (where $p$ can be billions), we minimize the loss function $\mathcal{L}(\boldsymbol{\theta})$. The local extreme value theory of one variable is the seed from which the entire theory grows:

|One-variable calculus|Neural network training|
|---|---|
|Local minimum at $c$|Converged parameter vector $\boldsymbol{\theta}^*$|
|$f'(c) = 0$|$\nabla_{\boldsymbol{\theta}} \mathcal{L} = \mathbf{0}$ (zero gradient)|
|Critical number|Stationary point of $\mathcal{L}$|
|Local max / min / saddle|Determined by eigenvalues of Hessian $H = \nabla^2 \mathcal{L}$|
|$f''(c) > 0$ → local min|All eigenvalues of $H$ positive → local min|
|$f''(c) < 0$ → local max|All eigenvalues of $H$ negative → local max|
|$f''(c) = 0$ → inconclusive|Zero eigenvalue → degenerate direction|
|Mixed signs in $f''$ → impossible in $\mathbb{R}^1$|Mixed eigenvalues → **saddle point** (common in $\mathbb{R}^p$!)|

The Second Derivative Test generalizes to the **multivariable second derivative test** using the Hessian matrix, which is precisely what MIT 18.02 develops.

### 10.2 Why Local Minima Are Acceptable in Deep Learning

A profound insight from modern ML theory: **not all local minima are created equal**, but in overparameterized neural networks, most local minima are acceptable.

The intuition from one-variable calculus: if a function has many local minima but they all have nearly the same function value, then finding _any_ local minimum is as good as finding the global one.

In neural networks:

- The loss landscape in high dimensions has exponentially many local minima.
- Empirically, these local minima tend to have **similar test performance**.
- The real enemies are **saddle points** (which do not exist in $\mathbb{R}^1$) — gradient descent can stall near them because the gradient is small even though the point is not a minimum.

### 10.3 Saddle Points — The High-Dimensional Surprise

In $\mathbb{R}^1$, a critical point with $f'(c) = 0$ is either a local max, local min, or an inflection point (where $f'$ doesn't change sign). There is no "saddle point" in one variable in the usual sense.

In $\mathbb{R}^p$, a **saddle point** is a stationary point where the Hessian has both positive and negative eigenvalues — it is a minimum in some directions and a maximum in others.

```
Saddle point of f(x,y) = x² − y²:

      ↑ y
       \   ↑ concave down in y direction
        \  |
    \    \ |   /
     -----*-----→ x
    /     |    \
          | ↑ concave up in x direction

∂f/∂x = 0, ∂f/∂y = 0 at origin.
Hessian = diag(2, −2): one positive, one negative eigenvalue.
→ Saddle point. Neither a max nor a min.
```

**Dauphin et al. (2014)** showed that in high-dimensional loss surfaces, saddle points are far more common than local minima: the probability that a random critical point is a local minimum (all Hessian eigenvalues positive) decreases exponentially with dimension. In practice, gradient descent spends most of its time escaping saddle points, not climbing out of local minima.

This is why **SGD with noise** (stochastic gradient descent) is better than pure gradient descent — the noise provides random perturbations that help escape the flat regions near saddle points.

### 10.4 The First Derivative Test as Gradient Descent

The First Derivative Test says: follow the sign of $f'$ to find a local minimum. If $f' > 0$, decrease $x$. If $f' < 0$, increase $x$. Keep moving until $f' = 0$.

This is exactly **gradient descent**: $$\theta_{n+1} = \theta_n - \eta \cdot f'(\theta_n)$$

The learning rate $\eta$ controls how large each step is. The algorithm traces the sign chart we drew in §6 — it follows $f'$, crossing the $x$-axis (critical point) to find the minimum.

```
Gradient Descent as First Derivative Test:

f'(x) > 0: move left (decrease θ)    f'(x) < 0: move right (increase θ)

         f'(x) > 0     f'(x) < 0
         ←←←←←←←←     →→→→→→→→→
              *         *
        local max    local min ←  gradient descent converges here

The algorithm IS the first derivative test, animated.
```

### 10.5 The Second Derivative Test as Newton's Method

The Second Derivative Test is: at a critical point, the sign of $f''$ determines the type of extremum. Operationally, this corresponds to **Newton's method** for optimization:

$$\theta_{n+1} = \theta_n - \frac{f'(\theta_n)}{f''(\theta_n)}$$

Newton's method uses the _curvature_ ($f''$) to scale the step size. Near a local minimum where $f''(c) > 0$, the step $\frac{f'(\theta)}{f''(\theta)}$ points in the right direction and has the right magnitude. This is exactly using the Second Derivative Test to navigate toward the minimum.

In high dimensions, Newton's method becomes: $$\boldsymbol{\theta}_{n+1} = \boldsymbol{\theta}_n - H^{-1} \nabla \mathcal{L}(\boldsymbol{\theta}_n)$$

where $H$ is the Hessian. This is the high-dimensional Second Derivative Test applied as an algorithm.

**Why don't we use Newton's method in deep learning?** Because computing and inverting $H \in \mathbb{R}^{p \times p}$ costs $O(p^3)$ operations — with $p = 10^9$ parameters, this is completely infeasible. Adaptive optimizers (Adam, RMSProp) approximate the diagonal of $H^{-1}$ cheaply using running averages of squared gradients.

### 10.6 Critical Points in Loss Curves During Training

When practitioners plot the training loss over time:

```
Training loss:

 ℒ
 |
 |\ 
 | \        ← rapidly decreasing (f' < 0, steep slope)
 |  \
 |   \___   ← plateau (near a saddle point? f' ≈ 0)
 |       \
 |        \___________   ← convergence (approaching local min)
 |
 +----------------------→ epochs

Each "plateau" corresponds to a region near a critical point.
Escaping plateaus is the hard problem.
```

The rapid decrease corresponds to being far from any critical point (large $|f'|$). Plateaus correspond to being near a saddle point or a "flat" region where $f'$ is small but not zero. Final convergence is approaching a local minimum where $f' \approx 0$ and $f'' > 0$.

## §11. The Full Logical Architecture

The theorems of this section form a strict logical hierarchy:

```
MEAN VALUE THEOREM
        |
        ↓
INCREASING/DECREASING TEST
(f' > 0 ⟹ f increasing)
        |
        ↓
FIRST DERIVATIVE TEST
(sign change in f' ⟹ local extremum)
        |
        ↓
SECOND DERIVATIVE TEST
(sign of f''(c) when f'(c)=0 ⟹ type of extremum)

Each lower result depends on the one above it.
MVT is the root of the entire tree.
```

The Second Derivative Test is not an independent theorem — it is a corollary of the First Derivative Test applied to $f'$ (using the fact that $f'' > 0$ implies $f'$ is increasing, which combined with $f'(c) = 0$ gives the sign change the First Derivative Test needs).

|$f$ continuous on $[a,b]$|Absolute extrema exist (EVT)|

## §13. Why the Hypotheses Matter — Edge Cases

**Continuity is essential in the First Derivative Test.** If $f$ is discontinuous at $c$, the sign change of $f'$ does not guarantee an extremum. Consider:

$$f(x) = \begin{cases} x & x < 0 \ x + 2 & x \geq 0 \end{cases}$$

$f'(x) = 1$ everywhere it exists — no sign change, no critical number. But this is a jump discontinuity, not an extremum example. More pathologically, one can construct functions where $f'$ changes sign but $f$ jumps over the "extremum."

**Fermat's theorem requires $f'(c)$ to exist.** When $f'(c)$ DNE, Fermat says nothing — but $c$ is still a critical number and must still be checked for extrema (by the definition of critical number, not by Fermat).

**The Second Derivative Test requires $f''$ to be continuous near $c$.** If $f''$ is discontinuous, the proof that $f'$ changes sign breaks down.

These hypotheses are not bureaucratic fine print — they are load-bearing walls. Every calculus proof is only as strong as the hypotheses it rests on.
