

> _"The derivative does not merely describe the function — it is the function's autobiography. Every rise, fall, bend, and inflection is written in $f'$ and $f''$."_

---

## §1. The Central Philosophy

A function $f$ is a static object — it just _is_. Its derivative $f'$ is dynamic — it tells you how $f$ is _moving_ at each point. The second derivative $f''$ tells you how _that movement_ is itself changing.

This hierarchy:

$$f \xrightarrow{\text{differentiate}} f' \xrightarrow{\text{differentiate}} f''$$

corresponds to:

$$\text{height} \xrightarrow{} \text{slope} \xrightarrow{} \text{curvature}$$

Everything about the shape of a graph — its rises and falls, its peaks and valleys, its bends and straightenings — is encoded in these two derived functions.

## §2. The First Derivative — Increasing and Decreasing

### 2.1 The Theorem

> **Theorem (Increasing/Decreasing Test).** Let $f$ be differentiable on an open interval $I$.
> 
> - If $f'(x) > 0$ for all $x \in I$, then $f$ is **strictly increasing** on $I$.
> - If $f'(x) < 0$ for all $x \in I$, then $f$ is **strictly decreasing** on $I$.
> - If $f'(x) = 0$ for all $x \in I$, then $f$ is **constant** on $I$.

**Proof of the increasing case.**

Take any $x_1 < x_2$ in $I$. By the **Mean Value Theorem**, $\exists, c \in (x_1, x_2)$: $$f(x_2) - f(x_1) = f'(c)(x_2 - x_1)$$

Since $f'(c) > 0$ and $x_2 - x_1 > 0$, we get $f(x_2) - f(x_1) > 0$, i.e., $f(x_2) > f(x_1)$. $\square$

_(The MVT is doing all the real work here — this is why MVT is called the "workhorse of differential calculus.")_

---

### 2.2 Finding Intervals — The Method

To determine where $f$ increases/decreases:

1. Find all **critical numbers**: points where $f'(x) = 0$ or $f'(x)$ does not exist.
2. These critical numbers **partition** the real line into open intervals.
3. Pick a **test value** in each interval; evaluate $f'$.
4. Sign of $f'$ → increasing or decreasing on that interval.

**Example.** $f(x) = x^3 - 3x^2 - 9x + 4$

$f'(x) = 3x^2 - 6x - 9 = 3(x^2 - 2x - 3) = 3(x-3)(x+1)$

Critical numbers: $x = -1, ; x = 3$

|Interval|Test value|$f'$|Behavior|
|---|---|---|---|
|$(-\infty, -1)$|$x = -2$|$3(-5)(-1) = 15 > 0$|Increasing ↑|
|$(-1, 3)$|$x = 0$|$3(-3)(1) = -9 < 0$|Decreasing ↓|
|$(3, \infty)$|$x = 4$|$3(1)(5) = 15 > 0$|Increasing ↑|

```
        ↑           ↓           ↑
  ------+---\-------+-------/---+------
       -1    \             /    3
              \           /
               \_________/
```

## §3. Critical Points and Local Extrema

### 3.1 Definitions

> **Definition.** A number $c$ in the domain of $f$ is a **critical number** if either $f'(c) = 0$ or $f'(c)$ does not exist.

> **Fermat's Theorem.** If $f$ has a local maximum or minimum at $c$, then $c$ is a critical number of $f$.

**Important converse failure:** A critical number need not yield a local extremum. The canonical example: $f(x) = x^3$ has $f'(0) = 0$ but $x = 0$ is neither a maximum nor a minimum — it's an **inflection point**.

```
f(x) = x³

        |  /
        | /
        |/
--------+--------
       /|
      / |
     /  |

f'(0) = 0, but no extremum at x = 0.
```

---

### 3.2 The First Derivative Test

> **First Derivative Test.** Suppose $c$ is a critical number of a continuous function $f$.
> 
> - If $f'$ changes **positive → negative** at $c$: **local maximum** at $c$.
> - If $f'$ changes **negative → positive** at $c$: **local minimum** at $c$.
> - If $f'$ does **not change sign** at $c$: **no local extremum** (inflection).

**Visual mnemonic:**

```
LOCAL MAX:         LOCAL MIN:         NO EXTREMUM:
    *                                     
   / \               ___              ___/
  /   \             /   \            /
 /     \           /     \         __
         \/                \/
f'>0  f'<0         f'<0  f'>0      f'>0  f'>0
```

**Continuing the example** $f(x) = x^3 - 3x^2 - 9x + 4$:

- At $x = -1$: $f'$ goes $+ \to -$ → **local max**, $f(-1) = 1 - 3 + 9 + 4 = 9$
- At $x = 3$: $f'$ goes $- \to +$ → **local min**, $f(3) = 27 - 27 - 27 + 4 = -23$

## §4. The Second Derivative — Concavity

### 4.1 Concavity Defined

> **Definition.** A differentiable function $f$ is:
> 
> - **Concave upward** on $I$ if $f'$ is _increasing_ on $I$.
> - **Concave downward** on $I$ if $f'$ is _decreasing_ on $I$.

Equivalently (when $f''$ exists):

> **Concavity Test.**
> 
> - $f''(x) > 0$ on $I$ $\Rightarrow$ $f$ is concave **up** on $I$ (holds water ∪)
> - $f''(x) < 0$ on $I$ $\Rightarrow$ $f$ is concave **down** on $I$ (spills water ∩)

**Geometric intuition.** Concave up means the tangent lines lie _below_ the curve. Concave down means they lie _above_. This is not a cosmetic property — it determines how well linear approximations (tangent lines) over- or under-estimate the function.

```
CONCAVE UP (∪)          CONCAVE DOWN (∩)
   f'' > 0                  f'' < 0

    curve                  curve
   /      \               /      \
  /        \             /        \
 / tangents \           / tangents \
     below                 above

tangent UNDERESTIMATES   tangent OVERESTIMATES
```

---

### 4.2 Inflection Points

> **Definition.** A point $(c, f(c))$ is an **inflection point** of $f$ if $f$ is continuous at $c$ and the concavity of $f$ changes at $c$.

**Necessary condition:** If $f''(c)$ exists and $(c, f(c))$ is an inflection point, then $f''(c) = 0$.

**Warning:** $f''(c) = 0$ does NOT guarantee an inflection point. Again, $f(x) = x^4$: $f''(0) = 0$ but concavity does not change at $0$ (it's concave up everywhere).

**Example.** $f(x) = x^3 - 3x^2 - 9x + 4$

$f''(x) = 6x - 6 = 6(x-1)$

$f''(x) = 0$ at $x = 1$.

- $x < 1$: $f''(x) < 0$ → concave down
- $x > 1$: $f''(x) > 0$ → concave up

Concavity changes at $x = 1$ → **inflection point** at $(1, f(1)) = (1, -8)$.

```
Full shape of f(x) = x³ - 3x² - 9x + 4:

      local max (−1, 9)
         *
        / \
       /   \         ← concave down
      /     \
     /  IP   \
    /  (1,−8) \
   /            *    ← inflection
                 \
                  \  ← concave up
                   *
            local min (3, −23)
```

## §5. The Second Derivative Test for Local Extrema

> **Second Derivative Test.** Suppose $f''$ is continuous near $c$ and $f'(c) = 0$.
> 
> - If $f''(c) > 0$: **local minimum** at $c$.
> - If $f''(c) < 0$: **local maximum** at $c$.
> - If $f''(c) = 0$: **inconclusive** — use the first derivative test.

**Why does this work?**

If $f'(c) = 0$ and $f''(c) > 0$, then $f'$ is increasing through $0$ at $c$. So $f'$ goes from negative (before $c$) to positive (after $c$) — exactly the first derivative test's condition for a local minimum.

**Proof sketch.**

$f''(c) > 0$ means, by definition of derivative: $$f''(c) = \lim_{x \to c} \frac{f'(x) - f'(c)}{x - c} = \lim_{x \to c} \frac{f'(x)}{x-c} > 0$$

For $x$ near $c$: $f'(x)$ and $(x-c)$ have the **same sign**. So:

- $x < c \Rightarrow f'(x) < 0$
- $x > c \Rightarrow f'(x) > 0$

This is exactly a local minimum by the first derivative test. $\square$

**The inconclusive case — three behaviors are possible:**

|Function|$f'(0)$|$f''(0)$|Behavior at 0|
|---|---|---|---|
|$f(x) = x^4$|0|0|Local min|
|$f(x) = -x^4$|0|0|Local max|
|$f(x) = x^3$|0|0|Inflection|

When the second derivative test fails, go back to the first derivative test. It never fails (as long as $f'$ is defined nearby).

## §6. Asymptotes — Behavior at the Boundary

Shape analysis is incomplete without understanding what happens as $x \to \pm\infty$ or as $x$ approaches a discontinuity.

### 6.1 Horizontal Asymptotes

$$\lim_{x \to \infty} f(x) = L \quad \text{or} \quad \lim_{x \to -\infty} f(x) = L$$

implies $y = L$ is a horizontal asymptote.

### 6.2 Vertical Asymptotes

$$\lim_{x \to a^+} f(x) = \pm\infty \quad \text{or} \quad \lim_{x \to a^-} f(x) = \pm\infty$$

implies $x = a$ is a vertical asymptote. These arise at zeros of denominators, logarithm arguments going to zero, etc.

### 6.3 Slant (Oblique) Asymptotes

When $\deg(\text{numerator}) = \deg(\text{denominator}) + 1$, polynomial long division yields: $$f(x) = (mx + b) + \frac{r(x)}{q(x)}$$

where $\frac{r(x)}{q(x)} \to 0$ as $x \to \infty$. The line $y = mx + b$ is the slant asymptote.

## §7. A Systematic Curve-Sketching Algorithm

This is the full synthesis — a procedure that transforms $f$ into a picture using everything above.

**Algorithm (Stewart §4.5):**

1. **Domain** — Where is $f$ defined?
2. **Intercepts** — $y$-intercept at $f(0)$; $x$-intercepts where $f(x) = 0$.
3. **Symmetry** — Even ($f(-x)=f(x)$)? Odd ($f(-x)=-f(x)$)? Periodic?
4. **Asymptotes** — Horizontal, vertical, slant.
5. **Intervals of increase/decrease** — Find $f'$, critical numbers, sign chart.
6. **Local extrema** — Apply first or second derivative test.
7. **Concavity and inflection points** — Find $f''$, solve $f''=0$, sign chart.
8. **Sketch** — Plot key points, connect with appropriate shape.

**Full worked example.** $f(x) = \dfrac{x^2}{x^2 - 1}$

**Domain:** $x \neq \pm 1$, so $(-\infty,-1) \cup (-1,1) \cup (1,\infty)$.

**Symmetry:** $f(-x) = \frac{x^2}{x^2-1} = f(x)$ → **even function**, symmetric about $y$-axis.

**Intercepts:** $f(0) = 0$ → $y$-intercept $(0,0)$; $f(x) = 0$ gives $x = 0$ → $x$-intercept $(0,0)$.

**Asymptotes:**

- $x \to \pm 1$: denominator $\to 0$, numerator $\to 1$ → vertical asymptotes $x = \pm 1$
- $x \to \pm\infty$: $\frac{x^2}{x^2-1} = \frac{1}{1-1/x^2} \to 1$ → horizontal asymptote $y = 1$

**First derivative:** $$f'(x) = \frac{2x(x^2-1) - x^2(2x)}{(x^2-1)^2} = \frac{2x^3 - 2x - 2x^3}{(x^2-1)^2} = \frac{-2x}{(x^2-1)^2}$$

Critical number: $x = 0$ (denominator never $0$ in domain).

Sign chart for $f'(x) = \frac{-2x}{(x^2-1)^2}$: denominator always positive; sign determined by $-2x$.

|Interval|Sign of $-2x$|$f'$|Behavior|
|---|---|---|---|
|$(-\infty,-1)$|$+$|$+$|Increasing|
|$(-1,0)$|$+$|$+$|Increasing|
|$(0,1)$|$-$|$-$|Decreasing|
|$(1,\infty)$|$-$|$-$|Decreasing|

At $x=0$: $f'$ goes $+ \to -$ → **local max**, $f(0) = 0$.

**Second derivative** (after computation): $$f''(x) = \frac{6x^2+2}{(x^2-1)^3}$$

Numerator $6x^2+2 > 0$ always. Sign of $f''$ determined by $(x^2-1)^3$:

- $|x| > 1$: $(x^2-1)^3 > 0$ → $f'' > 0$ → **concave up**
- $|x| < 1$: $(x^2-1)^3 < 0$ → $f'' < 0$ → **concave down**

$f'' \neq 0$ in domain → **no inflection points**.

```
Final sketch:

y
|           |           |
|   \       |     /     |
... *-------+-------* ...   y = 1 (asymptote)
|    \      |(0,0) /    |
|     \     *     /     |
|      \   / \   /      |
|       \_/   \_/       |
|      local max at 0   |
        x=-1       x=1   (vertical asymptotes)
```

## §8. Connection to AI, Machine Learning, and Deep Learning

### 8.1 Loss Landscape Topology

In training a neural network, the loss function $\mathcal{L}: \mathbb{R}^p \to \mathbb{R}$ (over millions of parameters) has a **shape** — a high-dimensional analog of the curves we've been analyzing. Every concept from this note has a direct counterpart:

|Calculus concept|ML/DL counterpart|
|---|---|
|Increasing/decreasing|Loss going up/down during training|
|Critical point ($f'=0$)|Stationary point — where gradient descent stops|
|Local minimum|A solution the optimizer converged to|
|Saddle point|Critical point that is a min in some directions, max in others — a major obstacle in deep learning|
|Concave up region ($f''>0$)|"Sharp" minimum — bad for generalization|
|Concave down region ($f''<0$)|Descent away from a local max|
|Inflection point|Transition in curvature of the loss curve|

### 8.2 Saddle Points — The Deep Learning Problem

In $\mathbb{R}^1$, a critical point is either a max, min, or inflection point. In $\mathbb{R}^p$ (high-dimensional parameter space), there is a fourth type: **saddle points** — points where $\nabla \mathcal{L} = \mathbf{0}$ but the Hessian $H = \nabla^2 \mathcal{L}$ has **both positive and negative eigenvalues**.

```
Saddle point in ℝ²:
f(x,y) = x² - y²

        ↑ y
        |
  \     |     /     ← concave down in y
   \    |    /
----\---+---/----→ x
   / \  |  / \
  /   \ | /   \    ← concave up in x
        |
```

At the origin: $\frac{\partial f}{\partial x} = 0$, $\frac{\partial f}{\partial y} = 0$, but it's neither a max nor a min.

**Why this matters:** Early deep learning theory (Dauphin et al., 2014) suggested that saddle points — not local minima — are the primary obstacle for gradient-based optimizers. In high dimensions, most critical points are saddle points (the probability that _all_ eigenvalues of the Hessian are positive is exponentially small in the dimension).

SGD and Adam navigate these saddle points through noise and momentum — but the underlying geometry is exactly the second derivative / concavity analysis from §4.

### 8.3 Sharp vs. Flat Minima and Generalization

The **concavity at a minimum** predicts generalization:

- **Sharp minimum** ($f''$ very large): The loss surface curves steeply. A small perturbation of $\theta$ causes a large increase in training loss — and even larger increases in test loss (since train and test distributions differ slightly). **Poor generalization.**
    
- **Flat minimum** ($f''$ small): The loss surface is nearly flat. Small perturbations don't hurt much. **Better generalization.**
    

```
Sharp minimum:          Flat minimum:
     |                  _____________
     *                 /             \
    /|\               /               \
   / | \             /                 \
  /  |  \           /                   \

  Bad for test        Good for test
```

This is the theoretical foundation behind:

- **Sharpness-Aware Minimization (SAM)** — explicitly seeks flat minima by perturbing weights and minimizing the worst-case loss in a neighborhood
- **Weight averaging methods** (SWA, EWA) — average iterates to land in flatter regions
- **Batch size effects** — large batch sizes tend to find sharper minima (Keskar et al., 2017)

All of this is concavity / second derivative analysis, lifted to high dimensions.

### 8.4 Learning Rate Schedules and Curvature

The optimal learning rate at a point is inversely proportional to the curvature ($f''$) there:

$$\eta_{\text{optimal}} \propto \frac{1}{f''(c)}$$

This is **Newton's method**: $\theta \leftarrow \theta - \frac{f'(\theta)}{f''(\theta)}$.

In high dimensions, this becomes $\theta \leftarrow \theta - H^{-1}\nabla \mathcal{L}(\theta)$.

**Adaptive optimizers** (Adam, RMSProp, Adagrad) approximate this by maintaining running estimates of the second moment of the gradient — a cheap surrogate for curvature. The reason they work better than vanilla SGD is exactly that they adapt the step size to local curvature:

- High curvature region ($f''$ large) → small step
- Low curvature region ($f''$ small) → large step

This is the geometric insight from §4 operationalized in code.

### 8.5 Activation Functions and Concavity

The choice of activation function in a neural network is, in part, a choice about concavity:

|Activation|Concavity|Consequence|
|---|---|---|
|Sigmoid $\sigma(x)$|Concave up for $x<0$, down for $x>0$|Vanishing gradients; inflection at $x=0$|
|ReLU $\max(0,x)$|Linear (no concavity)|No vanishing gradient; not differentiable at 0|
|GELU, Swish|Smooth, with inflection|Empirically superior; behaves like ReLU but differentiable|
|Softplus $\log(1+e^x)$|Always concave up|Smooth ReLU approximation|

The **inflection point of the sigmoid** at $x=0$ is the reason sigmoid saturates and causes vanishing gradients in deep networks — the concavity flips, and $\sigma''$ changes sign there. ReLU eliminates this by having no curvature at all (above zero). The entire history of activation function design is an applied study in concavity.

## §9. The Full Picture — Derivative Dictionary

```
FUNCTION f(x)
│
├─── f'(x) > 0  →  f is INCREASING   (graph rises left to right)
├─── f'(x) < 0  →  f is DECREASING   (graph falls left to right)
├─── f'(x) = 0  →  CRITICAL POINT    (horizontal tangent)
│         │
│         ├── f' changes + → −  →  LOCAL MAX
│         ├── f' changes − → +  →  LOCAL MIN
│         └── f' doesn't change sign  →  INFLECTION (if f''=0 too)
│
└─── f''(x) > 0  →  CONCAVE UP    ∪  (tangents below curve)
     f''(x) < 0  →  CONCAVE DOWN  ∩  (tangents above curve)
     f''(x) = 0  →  POSSIBLE INFLECTION POINT
           │
           └── concavity changes?  →  YES: INFLECTION POINT
                                       NO:  not an inflection point
```

## §10. Why the Hypotheses Matter

**The differentiability hypothesis is essential.** Consider $f(x) = |x|$ on $[-1, 1]$:

- $f'(x) = -1$ for $x < 0$ and $f'(x) = +1$ for $x > 0$
- $f'(0)$ does not exist
- $f$ has a global minimum at $x = 0$ — but Fermat's Theorem still holds! ($x=0$ is a critical number because $f'(0)$ DNE)

The theorems in this note require differentiability, but the extrema can still occur at non-differentiable points — they just can't be detected by $f' = 0$ there. **Always check where $f'$ fails to exist, not just where $f' = 0$.**
